# XtremIO Clusters

This function is only available with certain asset types. Therefore, the relevant tab may not be displayed.


Use this function to ???

- In the main window, select **ITSM > Configuration Management > CMDB**.
- Search for and select the applicable asset record.
- When the asset record window displays, click the **Components** tab.
- Click **XtremIO Clusters**.

! !

!


### Add XtremIO Cluster

- Click **Add**. The XtremIO Cluster dialog box displays.

! !

- Click **New XtremIO Cluster**
- Enter the applicable information in each field.
- When all entries are made, click **Save.**
- When all selections/entries are made, click **Add**.


### Edit XtremeIO Cluster

- Open the applicable blueprint.
- Navigate to the **Components** tab.
- Click a sub-component tab.
- In the list, select the applicable record.
- Do one of the following:
- Make the necessary modifications.
- Add new details.
- When all selections/entries are made, click **Add**.

### XtremIO Cluster Components

This component includes the following sub-components:

- **Volume**. Asset Name, GUID, Volume ID, and Created Time.

!

- **Target Group**. Asset Name and GUID.

!

- **Bricks**. Asset Name, GUID and State.

!

- **Initiator Group**. Asset Name and GUID.

!

- **Lun Map**. Asset Name, GUID and State.

!

- **Switch**. Asset Name, GUID, Cluster Name, Part Number and Hardware Model.

!

To access the functions, do the following:

- While viewing the XtremIO Clusters tab, click **Components**.
- Select an existing asset.
- Click **Components**. A subset of tabs displays.

!

- Click each applicable tab and do either of the following:
- Click **Add**, then enter all the available information.
- Click an asset record, and modify the existing information, as necessary.
- When all selections/entries are made, click **Add**.


### Delete XtremIO Cluster

Deleting is a permanent action and cannot be undone. Deleting may affect other functionality and information in the application such as data in configured reports, fields in windows, selectable options, etc. Therefore, be sure to understand the potential effects before making a deletion.


While viewing the list of existing records, select a record(s) to delete, and click the **Delete** button.
