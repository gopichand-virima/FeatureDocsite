# SLA: Tasks

 -- ----- ----------------------------------------------------------------------------
 1. In the main window, select **Admin \> Other \> Service Level Agreements**.
 -- ----- ----------------------------------------------------------------------------

 -- ----- ------------------------------
 2. Select the record to manage.
 -- ----- ------------------------------

 -- ----- ----------------------------------------------------------
 3. Follow the instructions below for the applicable action.
 -- ----- ----------------------------------------------------------

 1. Click the **Tasks** tab. The Add Task dialog box displays.\

 -- ----- -------------------------------------------------------------

 -- ----- ----------------------------------------------
 2. In the *Task* field, enter the task details.
 -- ----- ----------------------------------------------

 -- ----- ----------------------------------------------------------------------------------------------------------------------------
 3. To assign a user/group for the task, click the **Add** buttonclick the **plus + sign**. The Assignee dialog box displays.\
