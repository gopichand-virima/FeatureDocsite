# Details Tab

Displays the full configuration of the scheduled job:

- **Name**
- **Type** (Scan or Import)
- **Frequency**
- **Next Scheduled Run**
- **Last Execution Time**
- **Status** (Active / Inactive)
- **Target IP Range** (for scans) or **Credential** (for imports)
- **Scan/Import Type**
