# Release Hierarchy

Use this function to view the scheduled releases for the selected CI.

 -- ----- ---------------------------------------------------------------------------------------------------------------------------
 1. In the navigation pane, select **ITSM** **\> Configuration Management \> CMDB**. The Configuration Items window displays.
 -- ----- ---------------------------------------------------------------------------------------------------------------------------

 -- ----- ---------------------------------------------------------------------------------------------------------------
 2. In the navigation pane, select **Configuration Management \> CMDB**. The Configuration Items window displays.
 -- ----- ---------------------------------------------------------------------------------------------------------------

 -- ----- ---------------------------------------------------------------------------------------------------
 3. Select a record in the list. A new windows displays with the **Details** tab selected by default.
 -- ----- ---------------------------------------------------------------------------------------------------

 -- ----- ---------------------------------------------------------------------------------------------------------------
 4. From the *Select Actions* drop-down list, choose **Release Hierarchy**. A visual representation is displayed.
 -- ----- ---------------------------------------------------------------------------------------------------------------

 -- ----- ---------------------------------------------------------------------
 5. If there is no release hierarchy, a message displays. Click **OK**.
 -- ----- ---------------------------------------------------------------------
