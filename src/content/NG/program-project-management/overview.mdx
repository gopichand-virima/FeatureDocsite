# Program and Project Management Overview

Program and Project Management provides tools for planning, executing, and tracking IT projects and programs across your organization.

## Key Features

- Project planning and scheduling
- Resource allocation
- Task management
- Budget tracking
- Portfolio management
- AI-driven project insights

## Getting Started

Learn how to manage projects effectively:
1. Creating and configuring projects
2. Assigning resources and tasks
3. Tracking project progress
4. Managing budgets and timelines
5. Leveraging predictive analytics

## Version Information

This documentation applies to Virima **NextGen**. This version includes enhanced features and AI-powered capabilities not available in previous versions.
