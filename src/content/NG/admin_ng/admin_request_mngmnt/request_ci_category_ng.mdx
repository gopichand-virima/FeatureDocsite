# Request CI Category

Use this function to configure request-associated CI (Configuration Item) categories.

In the main window, click **Admin > Request Management > Request CI Category**. The Request CI Category window displays.

## New Request CI Category

From the *Select Actions* drop-down list, choose **New Request CI Category**. The New Request CI Category window displays.

Enter a *Name* and *Description*.

To add an image, click **Upload**, search for and select an image, then click **OK**.

Click **Add**.

## Edit Request CI Category

Navigate to the window containing the record to edit.

Click the line containing the record. The applicable window displays.

Make the necessary modifications.

Click **Save**.

## Delete Request CI Category

Click the line that contains the item to delete.

From the *Select Actions* drop-down list, choose **Delete**.
