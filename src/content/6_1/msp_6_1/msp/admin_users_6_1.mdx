# Users

Use this function to add and manage users that are accessing the vendor portal.

<div class="NumList1" madcap:autonum="1. ">

In the MSP window, click **Admin**.

</div>

<div class="NumList2" madcap:autonum="2. ">

Click **User**.

</div>

Add User

<div class="NumList1" madcap:autonum="1. ">

From the *Select Actions* drop-down list, choose **Add User**.

</div>

<div class="Ind50">

<span class="image placeholder" original-image-src="../Resources/Images/MSP/user-main-window.PNG" original-image-title="" madcap:conditions="Default.OldUI"></span>

</div>

<div class="Ind50">

The Add User dialog box displays.

</div>

<div class="Ind50">

<span class="image placeholder" original-image-src="../Resources/Images/MSP/add-user.PNG" original-image-title="" madcap:conditions="Default.OldUI"></span>

</div>

<div class="NumList2" madcap:autonum="2. ">

Enter the details for this user in the required /* fields. Note the following:

</div>

<div class="Bullet2">

When *MSP Admin* is selected, the user is considered an Admin with the applicable privileges.

</div>

<div class="Bullet2">

In the *Roles* field, click **Add**, then search for and select the roles for this user. If no role is assigned, the Admin privilege is considered the default role.

</div>

<div class="NumList2" madcap:autonum="3. ">

When all selections/entries are made, click **Add**.

</div>

Edit User Delete User Import User

<div class="NumList1" madcap:autonum="1. ">

From the *Select Actions* drop-down list, choose **Import User**. The Add User dialog box displays.

</div>

<div class="Ind50">

<span class="image placeholder" original-image-src="../Resources/Images/MSP/attach-data-files.PNG" original-image-title="" madcap:conditions="Default.OldUI"></span>

</div>

<div class="NumList2" madcap:autonum="2. ">

In the Attach Data Files dialog box, search for and select the applicable file contains the users to import.

</div>

<div class="NumList2" madcap:autonum="3. ">

Click **Upload**.

</div>

Associate User to Client

Use this function to view activities and support the associated clients on a requirement basis.

<div class="NumList1" madcap:autonum="1. ">

While viewing the list of user, select a user.

</div>

<div class="NumList2" madcap:autonum="2. ">

In the Users window, under the Clients section, click **Add**.

</div>

<div class="Ind50">

<span class="image placeholder" original-image-src="../Resources/Images/MSP/associate-user-add.PNG" original-image-title="" madcap:conditions="Default.OldUI"></span>

</div>

<div class="NumList2" madcap:autonum="3. ">

In the Client dialog, box search for and select the Client to associate with the selected user. Then click **Add**.

</div>

<div class="NumList2" madcap:autonum="4. ">

To remove the association:

</div>

<div class="NumLista" madcap:autonum="a. ">

Select the line item containing the client to remove.

</div>

<div class="numListb" madcap:autonum="b. ">

Click **Remove**.

</div>
