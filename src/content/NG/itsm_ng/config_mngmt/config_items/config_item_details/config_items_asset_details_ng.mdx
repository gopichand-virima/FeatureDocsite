# Asset Details

To view the details for a configuration item, search for and select an item from the existing list of assets. The Details window displays showing all relevant information.

The information shown is configured by the administrator. This information could include the following:

- Asset ID
- Asset Name
- Serial number
- Model
- Status
- Asset Type
- Other CI specific information
