# CI: Installed On

Use this function to view what software is installed on what system.

 -- ----- -----------------------------------------------------------------------------------------------------------------------
 1. In the navigation pane, select **ITSM \> Configuration Management \> CMDB**. The Configuration Items window displays.
 -- ----- -----------------------------------------------------------------------------------------------------------------------

 -- ----- -----------------------------------------------------------------------------------
 2. Select a record in the list. A new window opens and the **Details** tab displays.
 -- ----- -----------------------------------------------------------------------------------

 -- ----- -----------------------------------------------------------------------------------------------------
 3. Click the **Installed On** tab. A list of all settings on which the software is installed displays.
 -- ----- -----------------------------------------------------------------------------------------------------

 -- ----- ----------------------------------------------------------------------------------------------------------------
 4. To view the details for an asset, click on the asset name. You are returned to the Configuration Items window.
 -- ----- ----------------------------------------------------------------------------------------------------------------

 -- ----- ----------------------------------------------------------------
 5. To view the location for a selected asset, click **View Map**.
 -- ----- ----------------------------------------------------------------

 3. Make the necessary modifications.

 -- ----- -------------------------------------------
 4. Click **Save** or **Add**, as applicable.
 -- ----- -------------------------------------------
