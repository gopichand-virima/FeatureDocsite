# Portal Configuration

Use this function to configure and customize the self-service portal that end users access to submit tickets, track requests, and access knowledge base articles.

The self-service portal empowers users to resolve issues independently and submit tickets without contacting the service desk directly.

In the main window, click **Admin > Service Desk > Portal Configuration**. The Portal Configuration window displays.

## General Portal Settings

Configure basic portal settings:

**Portal Name**: Enter the display name for your portal (e.g., "IT Help Portal").

**Portal URL**: Configure the custom URL or subdomain.

**Portal Logo**: Upload your organization's logo.

**Favicon**: Upload a favicon for browser tabs.

**Default Language**: Select the default portal language.

**Multi-Language Support**: Enable support for multiple languages.

**Portal Theme**: Select from available themes or create custom.

Click **Save**.

## Portal Branding

Customize the visual appearance of the portal.

Click **Configure Branding**.

**Color Scheme**:

- **Primary Color**: Main brand color
- **Secondary Color**: Accent color
- **Background Color**: Page background
- **Text Color**: Default text color
- **Link Color**: Hyperlink color

**Typography**:

- **Heading Font**: Font for headings
- **Body Font**: Font for body text
- **Font Sizes**: Customize text sizes

**Header Configuration**:

- **Header Style**: Fixed, scrolling, or hidden
- **Navigation Menu**: Configure menu items and links
- **Search Bar**: Show/hide search functionality
- **User Menu**: Login, profile, logout options

**Footer Configuration**:

- **Footer Text**: Copyright and legal information
- **Footer Links**: Help, privacy policy, contact
- **Social Media**: Links to social media accounts

Click **Save**.

## Home Page Configuration

Configure what users see on the portal home page.

Click **Configure Home Page**.

**Layout Options**:

- Classic layout
- Card-based layout
- Dashboard layout
- Custom layout

**Content Sections**:

Arrange the following sections:

**Submit Ticket Section**:
- Quick ticket submission form
- Service catalog tiles
- Common request shortcuts

**My Tickets Section**:
- User's open tickets
- Recent ticket activity
- Quick status view

**Knowledge Base Section**:
- Featured articles
- Popular articles
- Recently published
- Search knowledge base

**Announcements**:
- Service desk announcements
- Planned maintenance
- System status

**Quick Links**:
- Frequently used services
- Common tasks
- External resources

Drag and drop sections to arrange layout.

Click **Save**.

## Ticket Submission Configuration

Configure how users submit tickets through the portal.

Click **Configure Ticket Submission**.

**Ticket Form Options**:

**Required Fields**: Select which fields are mandatory.

**Available Fields**: Choose which fields to display:
- Subject
- Description
- Category
- Priority (allow user selection or hide)
- Affected CI
- Attachments
- Custom fields

**Smart Forms**:

- **Conditional Fields**: Show/hide fields based on selections
- **Dynamic Categories**: Category changes based on ticket type
- **Field Dependencies**: Link related field values

**File Attachments**:

- **Allow Attachments**: Enable file uploads
- **Maximum File Size**: Set size limit per file
- **Allowed File Types**: Restrict file extensions
- **Maximum Files**: Limit number of attachments

**Guided Ticket Submission**:

- **Wizard Mode**: Multi-step ticket creation
- **Knowledge Suggestions**: Show relevant articles as user types
- **Similar Ticket Detection**: Alert if similar tickets exist

Click **Save**.

## Service Catalog Configuration

Configure the service catalog displayed in the portal.

Click **Configure Service Catalog**.

**Catalog Display**:

- **Layout**: Grid, list, or category view
- **Services per Page**: Number of items to display
- **Search and Filter**: Enable catalog search and filtering

**Service Items**:

For each service catalog item, configure:

- **Service Name**: Display name
- **Description**: Service details
- **Icon**: Visual identifier
- **Category**: Service grouping
- **Request Form**: Custom form for this service
- **Approval Required**: If approval is needed
- **Estimated Delivery**: Typical fulfillment time
- **Cost**: If cost information should be displayed

**Popular Services**:

- Highlight frequently requested services
- Display on home page
- Quick access buttons

Click **Save**.

## User Profile Configuration

Configure user profile and account management.

Click **Configure User Profile**.

**Profile Information**:

Allow users to view/edit:

- Contact information
- Department and location
- Notification preferences
- Password change

**My Tickets View**:

- Filter options
- Sort options
- Columns to display
- Ticket actions available

**My Requests View**:

- Active requests
- Request history
- Approval requests

**Notification Preferences**:

Allow users to configure:

- Email notifications
- SMS notifications
- In-app notifications
- Notification frequency

Click **Save**.

## Knowledge Base Integration

Configure how knowledge base is accessible from the portal.

Click **Configure Knowledge Base Access**.

**Knowledge Base Display**:

- **Browsing**: Category and tag browsing
- **Search**: Full-text search capability
- **Featured Content**: Highlighted articles
- **Recently Viewed**: User's article history

**Article Features**:

- **Ratings**: Allow users to rate helpfulness
- **Comments**: Enable user feedback
- **Share**: Social sharing options
- **Print**: Print-friendly view
- **Related Articles**: Show similar content

Click **Save**.

## Access Control

Configure who can access the portal and what they can see.

Click **Configure Access Control**.

**Authentication**:

- **Public Access**: Allow guest access (limited)
- **SSO Integration**: Single sign-on configuration
- **LDAP/AD**: Active Directory authentication
- **Local Accounts**: Portal-only accounts
- **Multi-Factor Authentication**: Require MFA

**User Permissions**:

Define what authenticated users can do:

- Submit tickets
- View own tickets only or all tickets
- Access knowledge base
- View announcements
- Update profile

**Guest Access**:

If enabled, configure guest capabilities:

- Browse public knowledge articles
- View system status
- Submit tickets (with email verification)

Click **Save**.

## Mobile Configuration

Optimize the portal for mobile devices.

Click **Configure Mobile Experience**.

**Responsive Design**: Enable mobile-optimized layouts.

**Mobile Features**:

- Touch-friendly interface
- Simplified navigation
- Mobile-optimized forms
- Push notifications (if supported)

**Mobile App**:

If a mobile app is available:

- App download links
- Deep linking configuration
- App-specific features

Click **Save**.

## Analytics and Reporting

Configure portal usage tracking and analytics.

Click **Configure Analytics**.

**Track Metrics**:

- Portal visits
- Page views
- Search terms
- Ticket submission source
- Knowledge article views
- User satisfaction

**Integration**:

- Google Analytics integration
- Custom analytics tools
- Heat map tracking

Click **Save**.
