# Import CIs

Use this function to upload Excel files containing CI details.

<div class="numlist1" madcap:autonum="1. ">

In the navigation pane, select **Discovery Scan /> Import Data Files**. The Import Data Files window displays.

</div>

<div class="numlist2" madcap:autonum="2. ">

In the *Select Actions* drop-down list, choose **Import CIs**. The Attach CI Data Files dialog box displays.

</div>

<div class="Ind50">

<span class="image placeholder" original-image-src="../Resources/Images/Discovery/attach-ci-data-files.PNG" original-image-title="" madcap:conditions="Default.OldUI" style="width: 379px;height: 228px;"></span>

</div>

<div class="Ind50">

<span class="image placeholder" original-image-src="../Resources/Images/Discovery/attach-ci-data-files-new.PNG" original-image-title="" madcap:conditions="Default.NewUI" style="width: 377px;height: 244px;"></span>

</div>

<div class="numlist2" madcap:autonum="3. ">

For *Select Mapping Type*, select either **Choose from Template** or **None**.

</div>

<div class="Bullet2">

If *Choose from Template* is selected, the *Choose Template* field displays. Click the drop-down list and select the applicable template. To view a sample file, select **Click here to download the sample file**.

</div>

<div class="Bullet2">

If **None** is selected, continue with the next step.

</div>

<div class="numlist2" madcap:autonum="4. ">

In the *Upload CI Data* field, click **Browse** and select an allowed file type (such as .xls, .xlsx, .csv). The name of the selected file is shown in the field.

</div>

<div class="numlist2" madcap:autonum="5. ">

Click **Upload**. The *Preview data to be imported and map columns* dialog box displays.

</div>

<div class="Ind50">

<span class="image placeholder" original-image-src="../Resources/Images/Discovery/preview-data-imported-map-columns.PNG" original-image-title="" style="width: 621px;height: 233px;" madcap:conditions="Default.OldUI"></span>

</div>

<div class="Ind50" madcap:conditions="Default.NewUI">

[<span class="image placeholder" original-image-src="../Resources/Images/Discovery/preview-data-imported-map-columns-new.PNG" original-image-title="" style="width: 623px;height: 299px;"></span>](../Resources/Images/Common%20Topics/Personalize-Columns.png)

</div>

## Preview data to be imported and map columns

The data imported from the spreadsheet populates this window. The column names and associated properties are displayed.

|     |                                                                                                                                                |
|-----|------------------------------------------------------------------------------------------------------------------------------------------------|
|     | All properties must be mapped before you can Submit the data. Properties cannot be deleted as they may be in use elsewhere in the application. |

<div class="numlist1" madcap:autonum="1. ">

To change the property for a column, click the drop-down list arrow and choose a property from the list. Note the following:

</div>

<div class="emdash1" madcap:autonum="- ">

The <span style="color: #dc143c; font-weight: bold;">red </span>and <span style="font-weight: bold; color: #008000;">green </span>column headings indicate if the data is properly mapped.

</div>

<div class="emdash1" madcap:autonum="- ">

The name of a property in this dialog box cannot be deleted or changed. The spreadsheet containing the data must be edited then reimported.

</div>

<div class="numlist2" madcap:autonum="2. ">

When all selections are made, click **Submit**.

</div>

<div class="NumList2" madcap:autonum="3. ">

<span style="background-color: #ffffff;">If a property is not mapped, an Alert message displays. Map the remainder of the properties. </span>

</div>

## Add New Property

<div class="numlist1" madcap:autonum="1. ">

While viewing the *Preview data to be imported and map columns dialog box*, click **Add New Property**. The *Add New Field dialog box* displays.

</div>

<div class="Ind50">

<span class="image placeholder" original-image-src="../Resources/Images/Discovery/add-new-field-dialog-box.PNG" original-image-title="" madcap:conditions="Default.OldUI" style="width: 377px;height: 196px;"></span>

</div>

<div class="Ind50">

[<span class="image placeholder" original-image-src="../Resources/Images/Discovery/add-new-field-dialog-box-new.PNG" original-image-title="" madcap:conditions="Default.NewUI" style="width: 376px;height: 298px;"></span>](../Resources/Images/Discovery/add-new-field-dialog-box-new.PNG)

</div>

<div class="numlist2" madcap:autonum="2. ">

In the *Property Name* field, enter a name for this property.

</div>

<div class="numlist2" madcap:autonum="3. ">

In the *Property Group* field, click the drop-down list and select from the groups shown.

</div>

<div class="numlist2" madcap:autonum="4. ">

In the *Property Type* field, click the drop-down list and select from the types shown.

</div>

<div class="numlist2" madcap:autonum="5. ">

Click **Add**. The property is now available as an option in the applicable drop-down menus.

</div>
