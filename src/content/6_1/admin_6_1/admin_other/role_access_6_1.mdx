# Role Access

Use this function to set permissions for roles to perform specific tasks for each module. The roles are:

|     |                                                                |
|:---:|----------------------------------------------------------------|
|  R  | Responsible for performing the assigned task.                  |
|  A  | Accountable for making certain work is assigned and performed. |
|  C  | Consulted about performing the task appropriately.             |
|  I  | Informed about key events regarding the task.                  |

To configure role access:

<div class="NumList1" madcap:conditions="Default.OldUI" madcap:autonum="1. ">

In the main window, click **Admin /> Others /> Role Access**. The Configuration Management window displays all the roles for the selected module (for example, SACM or Discovery) and the associated permissions (R, A, C or I).

</div>

<div class="NumList2" madcap:autonum="2. ">

To change the module to view, in the *Select Module* field, click the drop-down list and choose a module.

</div>

<div class="Ind50" madcap:conditions="Default.OldUI">

<span class="image placeholder" original-image-src="../Resources/Images/Admin%20-%20Other/role-access-config-mngmt-old.PNG" original-image-title="" style="width: 626px;height: 230px;"></span>

</div>

<div class="NumList2" madcap:autonum="3. ">

To change a permission, do the following:

</div>

<div class="NumLista" madcap:autonum="a. ">

Locate the line containing the role.

</div>

<div class="numListb" madcap:autonum="b. ">

Locate the column containing the permission.

</div>

<div class="numListb" madcap:autonum="c. ">

Click the current permission icon.

</div>

<div class="numListb" madcap:autonum="d. ">

From the drop-down list, select the new permission.

</div>

<div class="Ind75">

<span class="image placeholder" original-image-src="../Resources/Images/Admin%20-%20Other/select-permission-list.png" original-image-title=""></span>

</div>

<div class="NumList2" madcap:autonum="4. ">

When all selections are made, click **Save**.

</div>
