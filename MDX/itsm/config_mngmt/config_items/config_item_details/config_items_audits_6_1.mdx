# CI: Audits

Use this function to

- In the navigation pane, select **ITSM >Configuration Management > CMDB**. The Configuration Items window displays.
- In the navigation pane, select **Configuration Management > CMDB**. The Configuration items window displays.
- Select a record in the list. A new window opens and the **Details** tab displays.
- Click the **Audits**Â tab. A list of existing audits displays.

!


### Add Audit

- Click **New Audit**.
- Enter a *Name* and *Description*.
- In the Auditor field, click Add, then search for and select an Auditor.
- For *Recurring Frequency*, click the drop-down list and select the frequency, such as Daily, Weekly, and so forth.
- Enter the audit *Duration in days*.
- When all selections/entries are made, click **Add**.

!


### Edit Audit

- With the list of Audits displayed, select an audit in the list.
- Make the necessary changes.
- Click **Save**.


### Delete Audit

Deleting is a permanent action and cannot be undone. Deleting may affect other functionality and information in the application such as data in configured reports, fields in windows, selectable options, etc. Therefore, be sure to understand the potential effects before making a deletion.


- With the list of Audits displayed, select an audit in the list.
- Select a line item.
- Click **Delete**.
