# Technical Services

Use this function to create and manage technical services.

In the navigation pane, select **ITSM > Service Portfolio > Technical Services**. The Technical Services window displays.

New Technical Service

From the *Select Actions* drop-down list, choose **New Technical Service**.

Click **New**.

Complete the fields.

When all selections/entries are made, click **Add**.
