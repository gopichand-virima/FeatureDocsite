# Organizational Details

Use this function to update and save the organization's information.

|     |                                                                                                                                                |
|-----|------------------------------------------------------------------------------------------------------------------------------------------------|
|     | The Organization Details are created upon initial set up by Virima. While the Organization cannot be deleted, the information can be modified. |

<span class="span_1">In the main window, click </span><span class="span_2" style="font-weight: bold;">Admin</span><span class="span_1" style="font-weight: bold;"> /> </span><span class="span_2" style="font-weight: bold;">Organizational Details /> Organizational Details</span><span class="span_2" style="font-weight: bold;">.</span> The Organizational Details window displays.
