# User Entitlement

User license management gives organizations a clear, consolidated view of their licenses and deployments across the product users. This information can be incredibly valuable in the software audit, giving organizations the data they need to manage risk and remain compliant as software is consumed across their organization.

 -- ----- -------------------------------------------------------------------------------------------
 1. Navigate to ITAM \> User License Entitlement. The User License Entitlement page displays.
 -- ----- -------------------------------------------------------------------------------------------

![user entitle main 619x87](../Resources/Images/ITAM/user-entitle-main_619x87.png)

The user license entitlement page displays the following information:

::: Bullet1
Name assigned for the license entitled machine.

::: Bullet1
Date of system, and license entitled machine details configuration in the application.
