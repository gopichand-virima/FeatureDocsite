# Software Assets

Use this function to add and edit software assets. There are two components to this function: (1) adding a software asset group, and (2) adding the software asset itself.

 -- ----- ----------------------------------------------------------------------------------------------------------------------------------------------------------
 1. In the navigation pane, select **ITAM \> Software Asset Management \> Software Assets.** The window displays the list of all configured software assets.
 -- ----- ----------------------------------------------------------------------------------------------------------------------------------------------------------

 -- ----- -----------------------------------------------------------------------------------------
 2. Click the radio button for the preferred display type: **By Group** or **By Software**.
 -- ----- -----------------------------------------------------------------------------------------

[[[Closed]{.image .placeholder .MCDropDown_Image_Icon original-image-src="../../../Skins/Default/Stylesheets/Images/transparent.gif" original-image-title="" height="12" width="16" .dropDownHotspot .MCDropDownHotSpot_ .MCHotSpotImage aria-expanded="false"}[]{#SWAssetGroup}[New Software Asset By Group](javascript:void(0)){.MCDropDownHotSpot .dropDownHotspot .MCDropDownHotSpot_ aria-expanded="false"}]{.MCDropDownHead .dropDownHead}

 -- ----- --------------------------------------
 1. Click the **By Group** radio button.
 -- ----- --------------------------------------

 -- ----- --------------------------------------------------------------------------
 2. From the *Select Actions* drop-down list, choose **New Software Group**.
 -- ----- --------------------------------------------------------------------------

::: {.Ind50
![sw group new old](../../Resources/Images/ITAM/sw-group-new-old.PNG)

::: {.Ind50
The *New Software Group* window displays.

::: {.Ind50
![sw group add old](../../Resources/Images/ITAM/sw-group-add-old.PNG)

 -- ----- --------------------------------------------
 3. Enter a unique, alphanumeric *Group Name*.
 -- ----- --------------------------------------------

 -- ----- -----------------------------------------------------------------------------------------------
 4. Click **Add**, then search for and select the applicable *Vendor* and *Software License Key*.
 -- ----- -----------------------------------------------------------------------------------------------

 --- ---------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------
   Both a Vendor and Software License Key must be configured in order to choose them from the list. *Refer to* [Vendor Management](../Vendor-Mngmnt.mdx){style="font-style: italic;"} *and* [Software License Key](SW-License-Key.mdx){style="font-style: italic;"}*.*
 --- ---------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------

 -- ----- ----------------
 5. Click **Add**.
 -- ----- ----------------

[[[Closed]{.image .placeholder .MCDropDown_Image_Icon original-image-src="../../../Skins/Default/Stylesheets/Images/transparent.gif" original-image-title="" height="12" width="16" .dropDownHotspot .MCDropDownHotSpot_ .MCHotSpotImage aria-expanded="false"}[]{#SWAssetSoftware}[New Software Asset By Software](javascript:void(0)){.MCDropDownHotSpot .dropDownHotspot .MCDropDownHotSpot_ aria-expanded="false"}]{.MCDropDownHead .dropDownHead}

 -- ----- -----------------------------------------
 1. Click the **By Software** radio button.
 -- ----- -----------------------------------------

 -- ----- ---------------------------------------------------------------------------------------------------------------------
 2. From the *Select Actions* drop down, click **New**. The New dialog box displays.\
 ![SW Assets 6.1](../../Resources/Images/ITAM/SW-Assets-6.1.png)\

 -- ----- ---------------------------------------------------------------------------------------------------------------------

 -- ----- -------------------------------------------------------------
 3. Enter an *Asset Name*, *Software Name*, and *Manufacturer*.
 -- ----- -------------------------------------------------------------

 -- ----- ---------------------------------------------------------------------------------
 4. In the *Group* field, click the drop-down list and select the applicable group.
 -- ----- ---------------------------------------------------------------------------------

 --- ------------------------------------------------------------------------------------------------------------------------------------------------------
   There must be an asset Software Group created in order to have it available in the list (see previous section). Selecting a *Group* is not required.
 --- ------------------------------------------------------------------------------------------------------------------------------------------------------

 -- ----- ----------------
 5. Click **Add**.
 -- ----- ----------------

 -- ----- ----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------
 6. When returned to the list of software assets, select the asset just added. A new window displays in which to specify more information about this software asset, such as the software license key, components, relationships, SLA, etc. *Refer to Software Asset Details.*
 -- ----- ----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------

[[[Closed]{.image .placeholder .MCDropDown_Image_Icon original-image-src="../../../Skins/Default/Stylesheets/Images/transparent.gif" original-image-title="" height="12" width="16" Software Asset](javascript:void(0)){.MCDropDownHotSpot .dropDownHotspot .MCDropDownHotSpot_ .MCHotSpotImage aria-expanded="false"}]{.MCDropDownHead .dropDownHead}

 --- -----------------------------------------------------------------------------------------
   Not all functions include an option to edit record line items shown in the main window.
 --- -----------------------------------------------------------------------------------------

 -- ----- -------------------------------------------------------
 1. Navigate to the window containing the record to edit.
 -- ----- -------------------------------------------------------

 -- ----- -------------------------------------------------------------------------------------
 2. Click the line containing the record. The applicable window or dialog box displays.
 -- ----- -------------------------------------------------------------------------------------

 -- ----- -----------------------------------
 3. Make the necessary modifications.
 -- ----- -----------------------------------

 -- ----- -------------------------------------------
 4. Click **Save** or **Add**, as applicable.
 -- ----- -------------------------------------------

In the Details window, click **Related Software** to view a list of all software related to this software asset.

[[[Closed]{.image .placeholder .MCDropDown_Image_Icon original-image-src="../../../Skins/Default/Stylesheets/Images/transparent.gif" original-image-title="" height="12" width="16" Software Asset](javascript:void(0)){.MCDropDownHotSpot .dropDownHotspot .MCDropDownHotSpot_ .MCHotSpotImage aria-expanded="false"}]{.MCDropDownHead .dropDownHead}

 --- --------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------
   Deleting is a permanent action and cannot be undone. Deleting may affect other functionality and information in the application such as data in configured reports, fields in windows, selectable options, etc. Therefore, be sure to understand the potential effects before making a deletion.
 --- --------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------

 -- ----- --------------------------------------------------
 1. Click the line that contains the item to delete.
 -- ----- --------------------------------------------------

 -- ----- ------------------------------------------------------------------------------------------------------------------------------------------------------
 2. From the *Select Actions* drop-down list, choose **Delete**. If a confirmation message is displayed, type **Delete** and click **Delete**to confirm.
 -- ----- ------------------------------------------------------------------------------------------------------------------------------------------------------

[[[Closed]{.image .placeholder .MCDropDown_Image_Icon original-image-src="../../../Skins/Default/Stylesheets/Images/transparent.gif" original-image-title="" height="12" width="16" Software Asset](javascript:void(0)){.MCDropDownHotSpot .dropDownHotspot .MCDropDownHotSpot_ .MCHotSpotImage aria-expanded="false"}]{.MCDropDownHead .dropDownHead}

 -- ----- -----------------------------------------
 1. Click the **By Software** radio button.
 -- ----- -----------------------------------------

 -- ----- -------------------------------------------------------------------------------------
 2. From the *Select Actions* drop down, click **Import**. The New dialog box displays.
 -- ----- -------------------------------------------------------------------------------------

 -- ----- -----------------------------------------------------------------------------
 3. Click *Click here to download the sample file* to download the sample file.
 -- ----- -----------------------------------------------------------------------------

 -- ----- -------------------------------------------------
 4. Click **Browse** and select the file to import.
 -- ----- -------------------------------------------------

 -- ----- -------------------
 5. Click **Import**.
 -- ----- -------------------
