# Filter Recent Scans

Use the filter options to narrow down the list of recent scans.

## Filter By Status:
- **All**: Show all scans regardless of status
- **Running**: Show only scans currently in progress
- **Completed**: Show only successfully completed scans
- **Failed**: Show only scans that encountered errors
- **Stopped**: Show only scans that were manually stopped

## Filter By Date Range:
- Use the date picker to select a specific time period
- Common options: Today, Last 7 Days, Last 30 Days, Custom Range

## Filter By Client:
- Select a specific discovery client to show only scans executed by that client

## Filter By Probe Type:
- Filter scans by the type of probe used (e.g., Deep Host Scan, vCenter Scan)

## Column Filters:
- Use the filter fields at the top of each column to search for specific values
- Supports partial matching and wildcards

## Save Filters:
1. Apply your desired filters
2. Click **Save Filter**
3. Enter a name for the filter
4. The filter will be available for quick access in future sessions

Users can combine multiple filters to create precise views of scan history for reporting and analysis purposes.
