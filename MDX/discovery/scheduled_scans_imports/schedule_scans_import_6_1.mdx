# Overview

The **Scheduled Scans and Imports** feature allows administrators to configure recurring network scans, target specific IP ranges, associate clients and credentials, and manage multiple scan jobs from a single interface.

By providing detailed execution history, scan statuses, and flexible scheduling options, the feature reduces manual effort and improves data accuracy
