# New Stockroom

This feature enables to add stock with its details to form a record in the application.

<div class="NumList1" madcap:autonum="1. ">

Choose New Stock Room option from Select Actions drop down. The New Stock Room page displays.

</div>

<div class="Ind50">

<span class="image placeholder" original-image-src="../Resources/Images/ITAM/stockroom-new.png" original-image-title="" style="width: 632px;height: 99px;"></span>

</div>

<div class="NumList2" madcap:autonum="2. ">

Enter a name to identify the configured stock room in the Display Name box.

</div>

<div class="NumList2" madcap:autonum="3. ">

Click Add to update values of the following parameters:

</div>

<div class="NumLista" madcap:autonum="a. ">

Assignment Group

</div>

<div class="NumListb" madcap:autonum="b. ">

Manager

</div>

<div class="NumListb" madcap:autonum="c. ">

Location

</div>

<div class="NumListb" madcap:autonum="d. ">

Stock Type

</div>

Instructions:

<div class="NumLista" madcap:autonum="a. ">

Enter the search criteria in the given text box, and then click the search icon.

</div>

<div class="NumListb" madcap:autonum="b. ">

The results are displayed as per the given search criteria.

</div>

<div class="NumListb" madcap:autonum="c. ">

Select a required value, and then click Save.

</div>

<div class="NumListb" madcap:autonum="d. ">

Click Cancel to discard the action, and in continuation no value will be selected.

</div>

Reference : Click the link for https:/wiki.virima.com/wiki/doku.php?id=advanced_search_page for Advanced Search instructions

<div class="NumList2" madcap:autonum="4. ">
