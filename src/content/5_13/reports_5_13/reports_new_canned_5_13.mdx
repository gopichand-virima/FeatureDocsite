# New Canned Reports

Use this function to create, view and modify canned reports.


Canned reports are created by saving an [Ad Hoc report](/reports/reports_new_ad_hoc_5_13.mdx).


In the navigation pane, select **Reports > Canned Reports**. The Canned Reports window displays.

From the *Select Actions* drop-down list, choose **New Canned Report**. The Reports dialog box displays.

In the *Search* field, type the **name of the report** to search for.

Click the **Search** button. A list displays containing all reports that match the search criteria*.*

In the *Search* field, type the **name of the report** to search for. A list displays containing all reports that match the search criteria*.*

In the results list, select the applicable report(s). As you select each report, it is moved to the Selected list.

To remove all selected reports, click **Clear All**.

Click **Save**. The Canned Reports window displays and the selected canned report(s) are shown in the list. By default, the list is sorted by the *Created On* date.

To edit a canned report, refer to [Run Report](/reports/reports_run_5_13.mdx) and follow the instructions under *Generate/Re-generate Report*.
