# Problems

Use this <span style="background-color: #ffffff;">function </span>to create and manage problems.

In the main window, select **ITSM /> Problem Management /> Problems**.

<span class="image placeholder" original-image-src="../../Resources/Images/ITSM/Problem%20Management/problems-main-old.PNG" original-image-title=""></span>

<span class="image placeholder" original-image-src="../../Resources/Images/ITSM/Problem%20Management/problems-main-new.PNG" original-image-title=""></span>

New Problem

<div class="NumList1" madcap:conditions="Default.OldUI" madcap:autonum="1. ">

From the *Select Actions* drop-down list, choose **New Problem**.

</div>

<div class="NumList1" madcap:conditions="Default.NewUI" madcap:autonum="1. ">

Click **New**.

</div>

<div class="NumList2" madcap:autonum="2. ">

Complete the fields, referring to the information below.

</div>

<div class="NumList2" madcap:autonum="3. ">

When all selections/entries are made, click **Add**.

</div>

### Time Tracking

Use this function to add a record/task or generate a time spent report.

While viewing the Details window, select **Time Tracking**. The Time Tracking window displays.

<div class="NumList1" madcap:autonum="1. ">
