# Restart Client

|     |                                                                |
|-----|----------------------------------------------------------------|
|     | In order to restart a client, the state must be set to "true." |

<div class="NumList1" madcap:conditions="Default.OldUI" madcap:autonum="1. ">

From the main window, select **Admin /> Discovery /> Client.**

</div>

<div class="NumList2" madcap:conditions="Default.NewUI" madcap:autonum="2. ">

In the main window, click the **gear** icon and select **Discovery /> Client**.

</div>

<div class="NumList2" madcap:autonum="3. ">

In the Client window, select the applicable client.  

</div>

<div class="NumList2" madcap:autonum="4. ">

Click the **Discovery Agents** tab.

</div>

<div class="NumList2" madcap:conditions="Default.OldUI" madcap:autonum="5. ">

From the *Select Actions* drop-down list, choose **Restart Client**.

</div>

<div class="NumList2" madcap:conditions="Default.NewUI" madcap:autonum="6. ">

Click the **Restart Client** button.

</div>

<div class="Ind50" madcap:conditions="Default.OldUI">

<span class="image placeholder" original-image-src="../Resources/Images/Discovery/conf-msg-restart-client-old.PNG" original-image-title="" style="width: 242px;height: 105px;"></span>

</div>

<div class="Ind50" madcap:conditions="Default.NewUI">

<span class="image placeholder" original-image-src="../Resources/Images/Discovery/conf-msg-restart-client.PNG" original-image-title="" style="width: 246px;height: 86px;"></span>

</div>

<div class="NumList2" madcap:conditions="Default.OldUI" madcap:autonum="7. ">

In the Confirmation dialog box, click **Yes** (to restart) or **No** (to cancel the action).

</div>

<div class="NumList2" madcap:conditions="Default.NewUI" madcap:autonum="8. ">

In the Confirmation dialog box, click **Continue** (to restart) or **Cancel** (to cancel the action).

</div>
