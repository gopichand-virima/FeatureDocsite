# Edit or Delete Probes

Use this function to edit or delete the configured values of an existing probe.

## Edit Probe

In the main window, select **Admin > Discovery > Probes**. The Probes window displays.

In the main window, click the **gear** icon and select **Discovery > Probes**. Â The Probes window displays.

Select a record to edit. The *Details* information displays.

Make changes, as necessary, to any of the available fields.

Modify any of the following information:

- [Tasks](/itsm/release_mngmt/tasks_6_1.mdx)
- [Comments](/discovery/comments_6_1.mdx)
- [Attachments](/discovery/attachments_6_1.mdx)

When all selections/entries are made, click **Save**.
