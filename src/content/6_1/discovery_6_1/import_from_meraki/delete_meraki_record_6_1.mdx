# Deleting Meraki Record

- Deletes the currently selected imported Meraki record.
- From the **Select Actions** drop-down list, choose **Delete**.
- Confirm the deletion and click **Delete**.

**Important Notes:**
- Deleting an item is permanent and cannot be undone.
- Deleting an imported Meraki Import run will remove all corresponding discovered items.
- If any discovered items from the selected Meraki Import run have already been moved to the CMDB, those records will remain in the CMDB.
