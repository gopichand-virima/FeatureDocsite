# Viewing a discovered AZURE record

1.  In <span class="Strong">Import from AZURE</span>, open the <span class="Strong">All</span> tab. Click any row (e.g., an Asset Name).

2.  The <span class="Strong">Discovered Item</span> view for that asset opens.
