# Azure VMSS Instances

While viewing an asset record, click **Components**.

Click **Azure VMSS Instances**.

## Add a New Azure VMSS Instance

Click **Add**. The Azure VMSS Instance dialog box displays.

Click **New Azure VMSS Instance.**

Enter the applicable field information.

When all selections/entries are made, click **Save**.

## Modify Existing Azure VMSS Instance

Click **Existing Azure VMSS Instance**.

Enter the search criteria to locate the existing device.  *See also* [Advanced Search](/common_topics/advanced_search_5_13.mdx)*.*

Select the applicable device from the results list.

When all selections are made, click **Save**.

## Delete Azure VMSS Instance

While viewing the list of existing devices, select the device to delete, and click the **Delete** button.
