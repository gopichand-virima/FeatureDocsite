# View and Edit a Network

# Details Tab

Navigate to the IPAM window and select any network record to view, such as the network 192.168.48.0/24.

<span class="image placeholder" original-image-src="../../Resources/Images/Discovery/IPAM-Networks/View%20Edit%20a%20Network.png" original-image-title=""></span>

- The Details Tab provides default and static information about the selected network, which is generally not editable. The fields displayed include:

  - <span class="Strong">Location:</span> The physical or logical location of the network (e.g., Virima Technologies Bangalore).

  - <span class="Strong">Network View:</span> The view type for the network (e.g., default).

  - <span class="Strong">/_ref:</span> The reference identifier for the network record (typically blank or system-generated).

  - <span class="Strong">Comment:</span> A description for the network (e.g., Virima Dev Network).

  - <span class="Strong">Network:</span> The IP address range for the network (e.g., 192.168.48.0/24).

  - <span class="Strong">OS Manual:</span> Operating system details (if applicable).

  - <span class="Strong">Virima:</span> A field for Virima-specific information.

  - <span class="Strong">Region:</span> Geographic region of the network.

  - <span class="Strong">VLAN:</span> VLAN information, if relevant.

  - <span class="Strong">Test Value:</span> An extra customizable field.

  - <span class="Strong">Building:</span> The physical location tied to the network.

This tab displays only static data and doesn't allow editing.

# IP List Tab

The <span class="Strong">IP List</span> tab shows all IP addresses associated with the network, allowing you to view the status of each IP address.

<span class="image placeholder" original-image-src="../../Resources/Images/Discovery/IPAM-Networks/View%20Edit%20a%20Network_1.png" original-image-title=""></span>

- <span class="Strong">Example:</span>

  - <span class="Strong">100 Records per Page:</span> Number of records displayed per page.

  - <span class="Strong">Status:</span> The status of each IP (e.g., <span class="Strong">USED</span>, <span class="Strong">UNUSED</span>).

  - <span class="Strong">IP Address:</span> The actual IP address associated with the network.

  - <span class="Strong">Filter for Status:</span> Filter IP addresses based on their status (e.g., <span class="Strong">USED</span> or <span class="Strong">UNUSED</span>).

  - <span class="Strong">Filter for IP Address:</span> Filter by specific IP addresses.

# Schedule Details Tab

The <span class="Strong">Schedule Details</span> tab shows scheduled tasks for the network, such as automatic syncs or scans.

<span class="image placeholder" original-image-src="../../Resources/Images/Discovery/IPAM-Networks/View%20Edit%20a%20Network_2.png" original-image-title=""></span>

- <span class="Strong">Example:</span>

  - <span class="Strong">100 Records per Page:</span> Number of scheduled records displayed per page.

  - <span class="Strong">Client:</span> The client associated with the schedule (e.g., <span class="Strong">Virima Public App</span>).

  - <span class="Strong">Subnet:</span> The subnet associated with the schedule (e.g., <span class="Strong">192.168.48.0/24</span>).

  - <span class="Strong">Added At:</span> The date and time when the schedule was added (e.g., <span class="Strong">12/09/2021 17:04:26</span>).

  - <span class="Strong">Cron Expression:</span> The scheduling pattern (e.g., <span class="Strong">at 0, 10, 20, 30, 40 and 50 minutes</span>).

# History Tab

The History Tab provides a record of all changes made to the network, including actions such as adding or modifying configurations.

<span class="image placeholder" original-image-src="../../Resources/Images/Discovery/IPAM-Networks/View%20Edit%20a%20Network_3.png" original-image-title=""></span>

- <span class="Strong">Example:</span>

  - <span class="Strong">10 Records per Page:</span> Number of history records displayed per page.

  - <span class="Strong">Action:</span> The action performed (e.g., Subnet Add).

  - <span class="Strong">Modified On:</span> The date and time when the action was performed (e.g., 05/10/2019, 17:20:39).

  - <span class="Strong">Modified By:</span> The user who made the change (e.g., Franco Powers).

  - <span class="Strong">Changes:</span> The specific changes made in the action.
