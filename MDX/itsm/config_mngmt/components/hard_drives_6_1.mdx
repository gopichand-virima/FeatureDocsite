# Hard Drives

Displays a list of all hard drives installed on this asset and provides the functionality to update Windows patches.

## Apply Windows Update Patches

Before you can apply a patch, you must have an agent associated with the CMDB CI.


- From the navigation pane, select **ITSM > Configuration Management > Configuration Items**. The Configuration Items window displays.
- From the navigation pane, select **Configuration Management > Configuration Items.** The Configuration Items window displays.
- Select the applicable *WindowsÂ Host* record in the list. A new window displays with the **Details** tab displayed.
- Click the **Components** tab.
- Click the **Hard Drives** tab. A new set of subtabs displays.
- Click **Windows Updates**.
- Click the **Available Patches** button. The Available Patches window displays. The Add Maintenance window displays.

! !

- Select a patch(es) to apply and click the **Apply** button. The Downloadable Patches dialog box displays containing a list of all available patches.

! !

- Click **Apply**.
