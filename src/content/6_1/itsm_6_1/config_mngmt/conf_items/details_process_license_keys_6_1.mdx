# Process License Keys

Use this function to create a configuration item (e.g., Blueprint "Software License Key") for the detected license key (e.g., Windows Server/Host OS) and establish the "Component of" relationship with the Parent configuration item (e.g., Blueprint "Software").

For example:

<div class="Bullet1">

Parent CI Blueprint: **Software**

</div>

<div class="Bullet1">

Blueprint: **Software License Key**

</div>

<div class="Bullet1">

Detected license key: **Windows Server/Host Operating**

</div>

|     |                                                                 |
|-----|-----------------------------------------------------------------|
|     | The Parent CI will be created if not already found in the CMDB. |
