# Credentials Backup File

This file stores the credentials (which are encrypted) and is used by the application to scan the assets. These credentials are entered manually through one of the following ways:

<div class="Bullet1">

From the Virima application GUI.

</div>

<div class="Bullet1">

From the Discovery application GUI.

</div>

<div class="Bullet1">

An Excel import from the Discovery application.

</div>

|     |                                                                                                                                                                                                                                                |
|-----|------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
|     | Before reinstalling the application, take a backup of the existing credentials file (cat.json) to avoid re-entering the credentials in the newer version of the Discovery application. Making a backup of the credentials file is an optional. |

## Backup Credentials File

<div class="numlist1" madcap:autonum="1. ">

<span class="span_5">Navigate to the installed location of the previously installed Discovery application (usually </span><span class="Strong_3"><span style="font-family: 'Courier New'; font-size: 10pt;">C:/Program Files/Virima Discovery Application/</span>).</span>

</div>

<div class="numlist2" madcap:autonum="2. ">

<span class="span_5">Locate the </span><span class="Strong_3">**cat.json** file.</span>

</div>

<div class="numlist2" madcap:autonum="3. ">

<span class="span_5">Backup </span><span class="Strong_3" style="font-weight: bold;">cat.json</span><span class="span_5"> to a different location. </span>

</div>

## <span class="Strong_5">Reuse Backup Credentials File (cat.json)</span>

<div class="numlist1" madcap:autonum="1. ">

<span class="span_5">Replace the existing cat.json file with the backup file copy in the installed root directory<span style="font-size: 10pt;">: </span></span><span class="Strong_3" style="font-family: 'Courier New';"><span style="font-size: 10pt;">C:/Program Files/Virima Discovery Application/cat.json</span></span><span class="span_5">.</span>

</div>

<div class="numlist2" madcap:autonum="2. ">

Restart the application to apply the changes.

</div>
