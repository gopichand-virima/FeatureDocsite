# Bulk Update Scan

1.  In <span class="Strong">Scans</span> (radio button selected), Select one or more job

2.  Click on “Select Actions” and select “Update Scan”

<span class="image placeholder" original-image-src="../../Resources/Images/Discovery/Scheduled-Scans-and-Imports/Scheduled%20Scans%20and%20Imports_1_10.png" original-image-title=""></span>

3.  Limited set of attributes, common to all selected scans, that is not scan specific is displayed

    1.  <span class="Strong">Client:</span> pick the target client you want the selected scans to run on

    2.  <span class="Strong">Scan Frequency:</span> choose the frequency preset (e.g., <span class="Emphasis">Current Time Settings</span>) you want to apply.

    3.  <span class="Strong">Recurring:</span> select <span class="Strong">Yes</span> or <span class="Strong">No</span>.

    4.  <span class="Strong">Time Zone:</span> choose the correct timezone.

    5.  <span class="Strong">Active:</span> select <span class="Strong">Yes</span> (to run) or <span class="Strong">No</span> (to pause).

    6.  <span class="Strong">Schedule wheels:</span> tick the box above each wheel you want to control (<span class="Strong">Second / Minute / Hour / Day / Month / Weekday</span>), then set the values with the pickers.

4.  Click <span class="Strong">Update</span>.

5.  Updated details shall reflect in all the selected checkbox(es)
