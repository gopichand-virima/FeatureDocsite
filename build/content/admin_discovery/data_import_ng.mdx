# Data Import

Use this function to import data from external sources into Virima.

In the main window, click **Admin > Discovery > Data Import**. The Data Import window displays.

## Import Data

From the *Select Actions* drop-down list, choose **Import Data**. The Import Data window displays.

Select the import type and file format.

Upload the data file.

Map the columns to the appropriate fields.

When all selections/entries are made, click **Import**.
