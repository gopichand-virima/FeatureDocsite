# Email Actions

Use this function to configure what action to take upon receipt of an email to create an ITSM record.

In the main window, click **Admin > Event Management > Email Actions.**

!

!


### New Email Action

- From the *Select Actions* drop-down list, choose **New Action**.
- Complete the sections as described below.
- When all selections/entries are made, click **Add**.

### General Info

!

- Enter a *Name* and *Description* for the email action.
- Select *Enable* to define the corresponding email action as functional.
- Select the *Priority*.
- For Mailbox Configuration, click **Add**.

!

- Search for and select the template to associate with this action. Then click **Save**.
- Select to *Turn on Cascade Execution*.

### **Condition Details**

!

- Select *Ignore Condition* to ignore this condition for this email action.
- In the *Property* and *Operator* fields, click the drop-down list and select the property and operator.
- Enter the applicable value.
- To configure an AND condition, click the green plus + sign icon.
- To configure an OR condition, click the green | bar icon.
- To remove an existing condition, click the **red X** icon.

### **Action Details**

!

- Select *Ignore Action* to ignore this action for this email action.
- In the *Target* *Module* field, click the drop-down list and select the required module.
- In the *Create From Module* field click the drop-down list, and select the appropriate model of the selected module.
- In the *Property* field, click the drop-down list and select the property.
- Select the applicable *Value* or *Map*. Choose the option based on the requirement, then enter/select the relevant value.


The values options to define “Set Value”, and “Map” depends on the type of selected Property. The values configured in the Email Template are also displayed.


- To add another property, click the **plus + sign**.
- To remove an existing property, click the **minus + sign**.


### Edit Email Action

Not all functions include an option to edit record line items shown in the main window.


- Navigate to the window containing the record to edit.
- Click the line containing the record. The applicable window or dialog box displays.
- Make the necessary modifications.
- Click **Save** or **Add**, as applicable.


### Delete Email Action

Deleting is a permanent action and cannot be undone. Deleting may affect other functionality and information in the application such as data in configured reports, fields in windows, selectable options, etc. Therefore, be sure to understand the potential effects before making a deletion.


- Click the line that contains the item to delete.
- From the *Select Actions* drop-down list, choose **Delete**. If a confirmation message is displayed, type **Delete** and click **Delete**to confirm.
