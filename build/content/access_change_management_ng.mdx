# Access Change Management

The Change Management module provides comprehensive tools for managing changes to your IT infrastructure. This guide explains how to access the various components of Change Management.

## Accessing from the Main Menu

1. Log in to your Virima instance
2. Navigate to the **ITSM** module from the main navigation
3. Select **Change Management** from the dropdown menu

## Quick Access Options

### Dashboard Widget
You can add a Change Management widget to your dashboard for quick access to recent changes:

1. Navigate to your **Dashboard**
2. Click **Add Widget**
3. Select **Change Management** widgets
4. Choose your preferred view (Recent Changes, Pending Approvals, etc.)

### Search Bar
Use the global search bar to quickly find specific change requests:

1. Click the search icon in the top navigation
2. Enter the change request number or keywords
3. Select the relevant change from the search results

## Navigation Structure

The Change Management module is organized into the following sections:

- **Changes**: View and manage all change requests
- **Change Proposals**: Review and approve change proposals
- **Request for Changes (RFC)**: Submit and track RFCs
- **CAB Meetings**: Schedule and manage Change Advisory Board meetings
- **Outages**: Track planned and unplanned outages

## User Permissions

Access to Change Management features depends on your user role:

- **Change Managers**: Full access to all change management functions
- **Change Implementers**: Can view and update assigned changes
- **Change Reviewers**: Can review and approve changes
- **Standard Users**: Can submit change requests and view their own submissions

Contact your System Administrator if you need additional permissions.
