# New BSM View

- In the main window, select **Admin > SACM > BSM View**. The BSM View window displays.

!

- From the *Select Actions* drop-down list, choose **New BSM View**. The BSM View window displays.

!

- Enter a *Name* for the BSM view.
- From the *Include Relationships* and *Include Bluerpints* drop-down lists, select the type of required relationship and blueprint.
- From the *Exclude Relationships* and *Exclude Blueprints* drop-down lists, select the type of relationship and blueprint to be omitted.
- Check *Include External CI Communication* option to include external communications to the corresponding CI.
- When all selections/entries are made, click **Add**.
