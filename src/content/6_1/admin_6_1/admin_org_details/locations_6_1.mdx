# Locations

Use this function to associate users, facilities, or configuration items with a location.

In the main window, click **Admin /> Organizational Details /> Locations**. The Locations window displays.

<span class="image placeholder" original-image-src="../Resources/Images/Admin%20-%20Org%20Details/locations-old.PNG" original-image-title="" madcap:conditions="Default.OldUI" style="width: 621px;height: 105px;"></span>

New Location

<div class="NumList1" madcap:conditions="Default.OldUI" madcap:autonum="1. ">

From the *Select Actions* drop-down list, choose <span style="font-weight: bold;">New Location</span>. The New Location window displays.

</div>

<div class="NumList2" madcap:conditions="Default.NewUI" madcap:autonum="2. ">

Click **New Locations**. The New Location window displays.

</div>

<div class="Ind50" style="color: #dc143c;" madcap:conditions="Default.Omit">

<span class="image placeholder" original-image-src="../Resources/Images/Admin%20-%20Org%20Details/location-add.png" original-image-title="" madcap:conditions="Default.OldUI" style="width: 627px;height: 216px;"></span>

</div>

<div class="NumList2" style="font-style: normal;" madcap:autonum="3. ">

Complete the fields with the available information. The *Name*, *City*, *State* and *Country* are required fields.

</div>

<div class="NumList2" madcap:autonum="4. ">

When all entries are made, click **Add**.

</div>

Edit Location Delete Location

To delete an existing location:

<div class="NumList1" madcap:autonum="1. ">

Select the location to delete. from the *Select Actions* drop-down list, choose **Delete**.

</div>

<div class="NumList2" madcap:conditions="Default.NewUI" madcap:autonum="2. ">

From the *Select Actions* drop-down list, choose **Delete**. If a confirmation message is displayed, type **Delete** and then click **Delete**.

</div>
