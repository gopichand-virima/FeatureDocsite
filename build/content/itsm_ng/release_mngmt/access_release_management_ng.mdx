# Access Release Management

Release Management coordinates the planning, building, testing, and deployment of releases into production.

## Accessing from Main Menu

1. Log in to Virima
2. Navigate to **ITSM** > **Release Management**
3. Access release functions

## Navigation Structure

- **Releases**: All release records
- **Release Calendar**: Scheduled releases
- **Build Plans**: Release components
- **Testing**: Test plans and results
- **Deployment**: Release deployment tracking

## Quick Actions

### Create Release
1. Click **+** icon
2. Select **New Release**
3. Complete release details
4. Submit

### View Calendar
Access release calendar to see:
- Scheduled releases
- Release dependencies
- Resource allocation
- Blackout periods

## User Permissions

### Release Manager
- Create and manage releases
- Approve release plans
- Coordinate deployments
- Access all reports

### Release Coordinator
- Build release packages
- Execute test plans
- Track deployment
- Update documentation

### Release Approver
- Review release plans
- Approve deployments
- Authorize go-live
- Post-release review
