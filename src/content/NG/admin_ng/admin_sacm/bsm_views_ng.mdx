# BSM View

Use this function to customize the display properties of Business Service Management (BSM).

In the main window, click **Admin > SACM > BSM View**. The BSM View window displays.

In the main window, click the **gear** icon and select **SACM > BSM View**.

The BSM Views window displays.

New BSM View

From the *Select Actions* drop-down list, choose New BSM View.

Click **New BSM View**.

Enter a *Name* for the BSM view.

From the *Include Relationships* and *Include Blueprints* drop-down lists, select the type of required relationship and blueprint.

From the *Exclude Relationships* and *Exclude Blueprints* drop-down lists, select the type of relationship and blueprint to be omitted.

Check *Include External CI Communication* option to include external communications to the corresponding CI.

When all selections/entries are made, click **Add**.
