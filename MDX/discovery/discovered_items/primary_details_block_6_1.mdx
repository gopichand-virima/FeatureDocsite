# Primary Details Block

Displays high-level information: asset ID, blueprint, created date, IP address, host name, scan time.

- **If the asset already exists in the CMDB**:  
  The **Asset ID** field shows the existing CI ID. Clicking this ID opens the detailed CI record in the CMDB for review or editing.

- **If the asset is newly discovered**:  
  The **Asset ID** displays as **"New,"** indicating that this item has not yet been moved into the CMDB and needs validation or onboarding.
