# Cherwell Mappings

Use this function to map properties between a Virima Blueprint and a Cherwell Object.

|     |                                                                                                         |
|-----|---------------------------------------------------------------------------------------------------------|
|     | There must be a [Cherwell Credential](Cherwell-Credential.htm) configured before a mapping can be done. |

In the main window, click **Admin /> Integrations /> Cherwell Mappings**.

<span class="image placeholder" original-image-src="../Resources/Images/Admin%20-%20Integration/cherwell-mappings-main-old.PNG" original-image-title=""></span>

New Mapping

<div class="NumList2" madcap:conditions="Default.OldUI" madcap:autonum="1. ">

From the *Select Actions* drop-down list, choose **New Mapping**.

</div>

<div class="Ind50" madcap:conditions="Default.OldUI">

<span class="image placeholder" original-image-src="../Resources/Images/Admin%20-%20Integration/cherwell-map-add-old.PNG" original-image-title=""></span>

</div>

<div class="NumList2" madcap:conditions="Default.OldUI" madcap:autonum="2. ">

Click the drop-down lists and select the *Property*, *Operator* and *Blueprint*.

</div>

<div class="Bullet2" madcap:conditions="Default.OldUI">

To add a mapping and set the AND parameter, click the **plus** icon.

</div>

<div class="Bullet2" madcap:conditions="Default.OldUI">

To add a mapping and set the OR parameter, click the paperclip icon.

</div>

<div class="Bullet2" madcap:conditions="Default.OldUI">

To remove a mapping, click the red **X**.

</div>

<div class="NumList2" madcap:conditions="Default.OldUI" madcap:autonum="3. ">

In the *Cherwell Object* field, click the drop-down list and select the applicable object.

</div>

<div class="NumList2" madcap:conditions="Default.OldUI" madcap:autonum="4. ">

When all selections/entries are made, click **Add**.

</div>

<div class="NumList2" madcap:conditions="Default.OldUI" madcap:autonum="5. ">

To remove a mapping, select the line item containing the mapping, and click **Remove**.

</div>
