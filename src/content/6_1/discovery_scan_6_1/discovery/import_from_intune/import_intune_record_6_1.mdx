# Importing Intune Record

Use this function to import the required Intune resources into the Discovery application.

- Open the Import From Intune page, From the <span class="Strong" style="font-weight: bold;">Select Actions</span> drop-down list, choose <span class="Strong" style="font-weight: bold;">Import</span>.

  <span class="image placeholder" original-image-src="../../Resources/Images/Discovery/Import-from-Intune/Import%20INTUNE%20Window_3.png" original-image-title=""></span>

<!-- -->

- The **Import From Intune** dialog box displays.

  <span class="image placeholder" original-image-src="../../Resources/Images/Discovery/Import-from-Intune/Import%20INTUNE%20Window_4.png" original-image-title=""></span>

<!-- -->

- In the Name field, enter a name for the import action.

- In the Credential field, click the drop-down list and select the appropriate credential to access the Intune service.

  - If no credentials are listed in the dropdown, add credentials by navigating to <span class="Strong" style="font-weight: bold;">Admin /> Discovery /> Credentials</span>.

  - Only Active Intune credentials will appear in the dropdown. If a credential is Inactive, it will not be listed in the credential's dropdown.

- Click <span class="Strong" style="font-weight: bold;">Add</span> to import the Intune services to virima.

  <span class="image placeholder" original-image-src="../../Resources/Images/Discovery/Import-from-Intune/Import%20INTUNE%20Window_5.png" original-image-title=""></span>

The system displays an error if **Add** is clicked without selecting credentials.

<span class="image placeholder" original-image-src="../../Resources/Images/Discovery/Import-from-Intune/Import%20INTUNE%20Window_6.png" original-image-title=""></span>

- <span class="inline-comment-marker">After clicking **Add**, the system redirects to the **Import From Intune** page and displays a success message.</span>

  <span class="image placeholder" original-image-src="../../Resources/Images/Discovery/Import-from-Intune/Import%20INTUNE%20Window_7.png" original-image-title=""></span>

<!-- -->

- To view the details of a CI, click on the CI. This redirects you to the discovered item page, where you can view the imported CI details.
