# Purchase Order

Use this feature to configure the details of the purchase of products or services.

In the navigation pane, select **ITAM \> Procurement \> Purchase Order**. The Purchase Order window displays.

![purchase order main old](../../Resources/Images/ITAM/purchase-order-main-old.PNG)

![arrow border](../../Resources/Images/arrow-border.PNG)

 1. From the *Select Actions* drop-down list, select **New Purchase Order**. The New Purchase Order window displays.

 2. Complete the fields, referring to the sections below.
 -- ----- -------------------------------------------------------

 -- ----- -----------------------------------------------------
 3. When all selections/entries are made, click **Add**
 -- ----- -----------------------------------------------------

### Purchase Order Fields

![Purchase Order](../../Resources/Images/ITAM/Procurement_v6_1/Purchase Order.png)

::: Bullet2
**Purchase Order Number**. Enter the purchase order number. This is a required field.

::: Bullet2
**Purchase Order Date**. Click the **calendar icon** and select the date/time.

::: Bullet2
**Vendor**. Enter the name of the vendor.

- [Received]{style="background-color: #ffffff; color: #000000; font-weight: bold;"}.Select this checkbox only when all purchase order line items for this purchase order have been fully received. Leave it unchecked to record partial receipts by line.
::: Bullet1
**Requested Date**. Enter the date/time the items on the purchase order were requested.

::: Bullet1
**Expected Delivery Date**. Enter the date/time the purchase order items should be delivered.

::: Bullet1
**Ordered On**. Enter the date/time the order was placed.
