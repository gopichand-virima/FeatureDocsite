# Change Error Code

Use this function to define an error message to display in case of incorrect configuration scenario.

In the main window, click **Admin > Problem Management > Problem Error Code**. The Change Error Code window displays.

New Problem Error Code

From the *Select Actions* drop-down list, choose **New Problem Error Code**. The New Problem Error Code window displays.

Enter a *Name* and *Description*.

Click **Add**.
