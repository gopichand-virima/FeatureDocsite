# Discovery Agents

Use this function to view the total number of discovery agents installed for the corresponding Client, remotely install a new discovery agent, or restart the client.

In the main window, click **Admin > Discovery > Client**. The Client window displays.

Select the client for which you want manage discovery agents. A new window displays with the **Details** tab displayed.

Click the **Discovery Agents** tab.!

To trigger a scheduled scan on the remote machine where an agent is installed, refer to [Agent Schedules](/admin_discovery/client_agents_schedules_6_1_1.mdx).

From the *Select Actions* drop-down list, choose an action:

- [Build an agent](/admin_discovery/client_build_agent_6_1_1.mdx)
- [Remote Install](/admin_discovery/client_remote_install_6_1_1.mdx) to add a new Discovery Agent.
- [Restart the Client](/admin_discovery/client_restart_6_1_1.mdx)
- [Scan Discovery Agents](/admin_discovery/client_scan_6_1_1.mdx)

**Update Agents**. To update the agents with the latest version, select an agent, then click **Update Agents**.

**Schedule**. To schedule a scan on the remote client, click **Schedule** and specify a time period.

**Delete Schedule**. To delete the schedule, click **Delete Schedule**.

**Delete/Uninstall**. To uninstall or remove, select an agent, click **Uninstall**, then click **Uninstall**.
