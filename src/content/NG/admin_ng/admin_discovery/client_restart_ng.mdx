# Restart Client

Use this function to restart the Discovery Application.

In the main window, click **Admin > Discovery > Client**. The Client window displays.

Select a client, then click the **Discovery Agents** tab.

From the *Select Actions* drop-down list, choose **Restart**. The client restarts.
