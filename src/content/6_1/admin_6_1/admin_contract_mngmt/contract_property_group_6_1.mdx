# Contract Property Group

Use this function to create groups that can be used when setting up [contract properties](Contract-Properties.mdx).

In the main window, click **Admin \> Contract Management \> Contract Property Group**. The Contract Property Group window displays.

![contract property group main old](../Resources/Images/Admin - Contract/contract-property-group-main-old.PNG)

![arrow border](../Resources/Images/arrow-border.PNG)

[[[Closed]{.image .placeholder .MCDropDown_Image_Icon original-image-src="../../Skins/Default/Stylesheets/Images/transparent.gif" original-image-title="" height="12" width="16" Contract Property Group](javascript:void(0)){.MCDropDownHotSpot .dropDownHotspot .MCDropDownHotSpot_ .MCHotSpotImage aria-expanded="false"}]{.MCDropDownHead .dropDownHead}

 -- ----- ---------------------------------------------------------------------------------------------------------------------------
 1. From the *Select Actions* drop-down list, choose **New Contract Group**. The New Contract Property Group window displays.
 -- ----- ---------------------------------------------------------------------------------------------------------------------------

   Not all functions include an option to edit record line items shown in the main window.
 --- -----------------------------------------------------------------------------------------

 -- ----- -------------------------------------------------------
 1. Navigate to the window containing the record to edit.
 -- ----- -------------------------------------------------------

 -- ----- -------------------------------------------------------------------------------------
 2. Click the line containing the record. The applicable window or dialog box displays.
 -- ----- -------------------------------------------------------------------------------------

 -- ----- -----------------------------------
 3. Make the necessary modifications.
 -- ----- -----------------------------------

 -- ----- -------------------------------------------
 4. Click **Save** or **Add**, as applicable.
 -- ----- -------------------------------------------

To view the activity that has taken place for this vendor (such as changes), click the [History](/common_topics/history_6_1.mdx){.MCTopicPopup .MCTopicPopupHotSpot .a} tab.

 --- --------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------
   Deleting is a permanent action and cannot be undone. Deleting may affect other functionality and information in the application such as data in configured reports, fields in windows, selectable options, etc. Therefore, be sure to understand the potential effects before making a deletion.

 -- ----- --------------------------------------------------
