# New Canned Reports

Use this function to create, view and modify canned reports.

 --- ----------------------------------------------------------------------------------
   Canned reports are created by saving an [Ad Hoc report](Reports-New-Ad-Hoc.mdx).
 --- ----------------------------------------------------------------------------------

 -- ----- ---------------------------------------------------------------------------------------------------
 1. In the navigation pane, select **Reports \> Canned Reports**. The Canned Reports window displays.
 -- ----- ---------------------------------------------------------------------------------------------------

::: Ind50
![canned report main old](../Resources/Images/Reports/canned-report-main-old.PNG)

 -- ----- ----------------------------------------------------------------------------------------------------------
 2. From the *Select Actions* drop-down list, choose **New Canned Report**. The Reports dialog box displays.
 -- ----- ----------------------------------------------------------------------------------------------------------

::: {.Ind50
[]{.image .placeholder original-image-src="../Resources/Images/Reports/reports-dialog-box_471x211.png" original-image-title=""

 -- ----- -----------------------------------------------------------------------
 3. In the *Search* field, type the **name of the report** to search for.
 -- ----- -----------------------------------------------------------------------

 -- ----- -------------------------------------------------------------------------------------------------------
 4. Click the **Search** button. A list displays containing all reports that match the search criteria*.*
 -- ----- -------------------------------------------------------------------------------------------------------

 -- ----- ------------------------------------------------------------------------------------------------------------------------------------------------
 5. In the *Search* field, type the **name of the report** to search for. A list displays containing all reports that match the search criteria*.*
 -- ----- ------------------------------------------------------------------------------------------------------------------------------------------------

 -- ----- --------------------------------------------------------------------------------------------------------------------
 6. In the results list, select the applicable report(s). As you select each report, it is moved to the Selected list.
 -- ----- --------------------------------------------------------------------------------------------------------------------

 -- ----- ------------------------------------------------------
 7. To remove all selected reports, click **Clear All**.
 -- ----- ------------------------------------------------------

 -- ----- ----------------------------------------------------------------------------------------------------------------------------------------------------------------------
 8. Click **Save**. The Canned Reports window displays and the selected canned report(s) are shown in the list. By default, the list is sorted by the *Created On* date.
 -- ----- ----------------------------------------------------------------------------------------------------------------------------------------------------------------------

 -- ----- ----------------------------------------------------------------------------------------------------------------------------------
 9. To edit a canned report, refer to [Run Report](Reports-Run.mdx) and follow the instructions under *Generate/Re-generate Report*.
 -- ----- ----------------------------------------------------------------------------------------------------------------------------------
