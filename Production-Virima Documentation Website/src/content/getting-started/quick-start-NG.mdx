# Quick Start Guide

Get up and running with Virima in minutes. This guide will walk you through the essential 
steps to start discovering and managing your IT infrastructure.

## Prerequisites

- System requirements met (see Compatibility Matrix)
- Administrative access to your network
- Valid Virima license key
- Supported database installed (PostgreSQL, MySQL, or SQL Server)

## Installation Steps

### Step 1: Download Virima
Download the latest Virima installer from the customer portal.

### Step 2: Run the Installer
Execute the installer and follow the setup wizard. You'll need to configure database 
connection settings and provide your license key.

```bash
./virima-installer-NG.sh --install
```

### Step 3: Configure Discovery
Set up your first discovery by specifying network ranges and credentials. 
Virima will automatically start discovering devices and services.

### Step 4: Access the Dashboard
Open your browser and navigate to the Virima web interface. Default URL is:

```
https://your-server:8443/virima
```

## Next Steps

Once installation is complete, explore these resources to make the most of Virima:

- Configure additional discovery patterns for cloud resources
- Set up user accounts and role-based access control
- Create custom dashboards and reports
- Integrate with your ITSM platform
- Configure automated alerts and notifications
