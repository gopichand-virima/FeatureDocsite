# Create Scheduled Scan

From the **Select Actions** drop-down, choose **New Scheduled Scan**.

1. Complete the required fields:
   - **Name**: Identify the scheduled scan
   - **Probe**: Select the scan type
   - **Client**: Choose the discovery client
   - **IP Range**: Specify targets to scan
   - **Exclude IP Range**: (Optional) IPs to skip

2. Configure schedule settings:
   - **Scan Frequency**: How often to run
   - **Is Active**: Enable/disable the schedule
   - **Is Recurring**: Whether to repeat automatically
   - **Timezone**: Select appropriate timezone
   - **Schedule Parameters**: Set exact timing

3. Optional settings:
   - **Location**: Add location tags
   - **Send Scan Report To**: Select recipients

4. Click **Add** to save the scheduled scan.

The scheduled scan will execute automatically according to the configured frequency and will appear in Recent Scans after each execution.
