# Vendor Property Group

Use this function to create a vendor property group to select when setting up a [vendor property](Vendor-Properties.mdx).

In the main window, click **Admin \> Vendor Management \> Vendor Property Group**. The Vendor Property Group window displays.

![vendor property group main old](../Resources/Images/Admin - Vendor/vendor-property-group-main-old.PNG)

![arrow border](../Resources/Images/arrow-border.PNG)

[[[Closed]{.image .placeholder .MCDropDown_Image_Icon original-image-src="../../Skins/Default/Stylesheets/Images/transparent.gif" original-image-title="" height="12" width="16" Vendor Property Group](javascript:void(0)){.MCDropDownHotSpot .dropDownHotspot .MCDropDownHotSpot_ .MCHotSpotImage aria-expanded="false"}]{.MCDropDownHead .dropDownHead}

 -- ----- ------------------------------------------------------------------------
 1. From the *Select Actions* drop-down list, choose **New Vendor Group**.
 -- ----- ------------------------------------------------------------------------

[[[Closed]{.image .placeholder .MCDropDown_Image_Icon original-image-src="../../Skins/Default/Stylesheets/Images/transparent.gif" original-image-title="" height="12" width="16" Vendor Property Group](javascript:void(0)){.MCDropDownHotSpot .dropDownHotspot .MCDropDownHotSpot_ .MCHotSpotImage aria-expanded="false"}]{.MCDropDownHead .dropDownHead}

 --- -----------------------------------------------------------------------------------------
   Not all functions include an option to edit record line items shown in the main window.
 --- -----------------------------------------------------------------------------------------

 -- ----- -------------------------------------------------------
 1. Navigate to the window containing the record to edit.
 -- ----- -------------------------------------------------------

 -- ----- -------------------------------------------------------------------------------------
 2. Click the line containing the record. The applicable window or dialog box displays.
 -- ----- -------------------------------------------------------------------------------------

 -- ----- -----------------------------------
 3. Make the necessary modifications.
 -- ----- -----------------------------------

 -- ----- -------------------------------------------
 4. Click **Save** or **Add**, as applicable.
 -- ----- -------------------------------------------

To view the activity that has taken place for this vendor (such as changes), click the [History](/common_topics/history_6_1.mdx){.MCTopicPopup .MCTopicPopupHotSpot .a} tab.

[[[Closed]{.image .placeholder .MCDropDown_Image_Icon original-image-src="../../Skins/Default/Stylesheets/Images/transparent.gif" original-image-title="" height="12" width="16" Vendor Property Group](javascript:void(0)){.MCDropDownHotSpot .dropDownHotspot .MCDropDownHotSpot_ .MCHotSpotImage aria-expanded="false"}]{.MCDropDownHead .dropDownHead}

 --- --------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------
   Deleting is a permanent action and cannot be undone. Deleting may affect other functionality and information in the application such as data in configured reports, fields in windows, selectable options, etc. Therefore, be sure to understand the potential effects before making a deletion.
 --- --------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------

 -- ----- --------------------------------------------------
 1. Click the line that contains the item to delete.
 -- ----- --------------------------------------------------

 -- ----- -------------------------------------------------------------------------------------------------------------------------------------------------------------------------------
 2. From the *Select Actions* drop-down list, choose **Delete**. If a confirmation message is displayed, take the appropriate action (for example, click **OK** or **Continue**).
 -- ----- -------------------------------------------------------------------------------------------------------------------------------------------------------------------------------
