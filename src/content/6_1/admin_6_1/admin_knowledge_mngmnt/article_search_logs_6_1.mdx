# Article Search Logs

Use this function to define a threshold for the Knowledge topics that are not updated in the portal. The threshold automatically submits a knowledge base item if the user is unable to find the article for the defined threshold level.

<div class="NumList1" madcap:conditions="Default.OldUI" madcap:autonum="1. ">

In the main window, click **Admin /> Knowledge Management /> Article Search Logs**. The Article Search Logs window displays.

</div>

<div class="Ind50" madcap:conditions="Default.OldUI">

<span class="image placeholder" original-image-src="../Resources/Images/Admin%20-%20Knowledge/article-search-logs-main-old.PNG" original-image-title=""></span>

</div>

<div class="NumList1" madcap:conditions="Default.NewUI" madcap:autonum="1. ">

In the main window, click the **gear** icon and select **Knowledge Management /> Article Search Logs.**

</div>

<div class="Ind50" style="color: #dc143c;" madcap:conditions="Default.NewUI">

<span class="image placeholder" original-image-src="../Resources/Images/Admin%20-%20Knowledge/icon-article-search-logs.PNG" original-image-title=""></span>

</div>

<div class="Ind50" madcap:conditions="Default.NewUI">

The Article Search Logs window displays.

</div>

<div class="Ind50" madcap:conditions="Default.NewUI">

<span class="image placeholder" original-image-src="../Resources/Images/Admin%20-%20Knowledge/article-search-logs-main-new.PNG" original-image-title=""></span>

</div>

<div class="NumList2" madcap:conditions="Default.OldUI" madcap:autonum="2. ">

From the *Select Actions* drop-down list, choose **Set Threshold**. The Threshold dialog box displays.

</div>

<div class="Ind50" madcap:conditions="Default.OldUI">

<span class="image placeholder" original-image-src="../Resources/Images/Admin%20-%20Knowledge/set-threshold-old.PNG" original-image-title=""></span>

</div>

<div class="NumList2" madcap:conditions="Default.NewUI" madcap:autonum="3. ">

Click **Set Threshold**. The Threshold dialog box displays.

</div>

<div class="Ind50" madcap:conditions="Default.NewUI">

<span class="image placeholder" original-image-src="../Resources/Images/Admin%20-%20Knowledge/set-threshold-new.PNG" original-image-title=""></span>

</div>

<div class="NumList2" madcap:autonum="4. ">

In the *Search Term Threshold* field, enter the total number of search attempts by the user for the required Knowledge topic.

</div>

<div class="NumList2" madcap:autonum="5. ">

In the *Search Result Threshold* field, enter the maximum number of search attempts the user can make.

</div>
