# Reports Overview

Reports module provides comprehensive reporting and analytics capabilities across all Virima modules with customizable dashboards and visualizations.

## Key Features

- Pre-built report templates
- Custom report builder
- Interactive dashboards
- Scheduled report delivery
- Export to multiple formats
- AI-powered insights and recommendations

## Getting Started

Learn how to create and manage reports:
1. Running pre-built reports
2. Creating custom reports
3. Scheduling automated reports
4. Sharing reports with stakeholders
5. Leveraging AI-driven analytics

## Version Information

This documentation applies to Virima **NextGen**. This version includes enhanced features and AI-powered capabilities not available in previous versions.
