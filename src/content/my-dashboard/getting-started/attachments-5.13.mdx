# Attachments

Learn how to add, manage, and share file attachments within Virima records.

## Overview

The Attachments function allows you to upload and manage files associated with various records throughout the Virima platform, including incidents, assets, and configuration items.

## Adding Attachments

### Upload Files
1. Navigate to the record where you want to add an attachment
2. Click the **Attachments** tab or icon
3. Click **Add Attachment** or drag and drop files
4. Select files from your computer
5. Click **Upload** to complete the process

### Supported File Types
- Documents: PDF, DOC, DOCX, XLS, XLSX, PPT, PPTX
- Images: JPG, PNG, GIF, BMP, SVG
- Text: TXT, CSV, XML, JSON
- Archives: ZIP, RAR, 7Z
- Other: Based on system configuration

## Managing Attachments

### View Attachments
- Click on an attachment name to preview or download
- View attachment properties (name, size, upload date, uploaded by)
- Sort attachments by various criteria

### Delete Attachments
1. Locate the attachment in the attachments list
2. Click the **Delete** icon next to the attachment
3. Confirm deletion when prompted

### Download Attachments
- Click the **Download** icon to save locally
- Download multiple attachments as a ZIP archive
- Access attachment history and versions

## Best Practices

- Use descriptive file names for easy identification
- Check file size limits before uploading
- Remove outdated or duplicate attachments
- Verify file types are appropriate for the record
- Use version control for frequently updated documents

## File Size Limits

- Maximum file size: Configured by system administrator
- Total attachment limit per record: Varies by configuration
- Contact your administrator if you need to upload larger files
