# Monitoring Profile: Trigger Conditions

Use this function to define a condition to match the discovered data as configured.

<div class="NumList1" madcap:conditions="Default.OldUI" madcap:autonum="1. ">

<span class="span_1">On the main window, select </span><span class="Strong_2" style="font-weight: bold;">Admin</span><span class="span_1" style="font-weight: bold;"> /> </span><span class="Strong_2" style="font-weight: bold;">Discovery</span><span class="span_1">** /> Discovery Monitoring Profile**. The Discovery Monitoring Profile window displays.</span>

</div>

<div class="NumList2" madcap:autonum="2. ">

On the main window, click the **gear** icon and select **Discovery /> Discovery Monitoring Profile**. The Discovery Monitoring Profile window displays.

</div>

<div class="NumList2" madcap:autonum="3. ">

Select either **System Monitor** or **File Monitor**.

</div>

<div class="NumList2" madcap:autonum="4. ">

In the **Trigger Conditions** window, use the icons to do the following:

</div>

<div class="Bullet2">

Click the **plus + sign** to add an additional row.

</div>

<div class="Bullet2">

Click the **red <span style="color: #dc143c;">x</span>** to remove the row.

</div>

<div class="NumList2" madcap:autonum="5. ">

If *System Monitor* is selected:

</div>

<div class="NumLista" madcap:autonum="a. ">

In the *Property* field, click the drop-down list and choose a property from those available.

</div>

<div class="NumListb" madcap:autonum="b. ">

In the *Operator* field, click the drop-down list and choose the applicable operator. The Operators displayed are valid for the selected *Property* type.

</div>

<span class="image placeholder" original-image-src="../Resources/Images/Admin%20-%20Discovery/discovery-mon-prof-trigger-conditions.PNG" original-image-title="" madcap:conditions="Default.OldUI"></span>

</div>

<div class="Ind75" madcap:conditions="Default.NewUI">

</div>

<div class="NumList2" madcap:autonum="6. ">

If *File Monitor* is selected:

</div>

<div class="NumLista" madcap:autonum="a. ">

In the *Type* field, click the drop-down list and select the applicable type.

</div>

<div class="numListb" madcap:autonum="b. ">

In the text field, type the applicable information.

</div>

<div class="Ind75">

<span class="image placeholder" original-image-src="../Resources/Images/Admin%20-%20Discovery/discovery-mon-prof-trigger-conditions-file-mon.PNG" original-image-title="" madcap:conditions="Default.OldUI"></span>

</div>

<div class="Ind75" madcap:conditions="Default.NewUI">

<span class="image placeholder" original-image-src="../Resources/Images/Admin%20-%20Discovery/discovery-mon-prof-trigger-conditions-file-mon-new.PNG" original-image-title=""></span>

</div>

<div class="NumListb" madcap:autonum="c. ">

To add a row, click the **plus + sign**.

</div>

<div class="NumListb" madcap:autonum="d. ">

To delete a row, click the **minus - sig**n.

</div>

<div class="NumList2" madcap:autonum="7. ">

When all selections/entries are made, and a match is found, the condition is triggered.

</div>

<div class="NumList2" madcap:autonum="8. ">

When all selections/entries are made, click **Add**.

</div>
