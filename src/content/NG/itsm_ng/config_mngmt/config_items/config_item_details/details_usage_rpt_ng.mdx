# CI: Usage Report

Use this function to view software usage metric information for software configured as *major* software.

 -- ----- -----------------------------------------------------------------------------------------------------------------------
 1. In the navigation pane, select **ITSM \> Configuration Management \> CMDB**. The Configuration Items window displays.
 -- ----- -----------------------------------------------------------------------------------------------------------------------

 -- ----- ---------------------------------------------------------------------------------------------------------------
 2. In the navigation page, select **Configuration Management \> CMDB**. The Configuration Items window displays.
 -- ----- ---------------------------------------------------------------------------------------------------------------

 -- ----- ------------------------------------------------------------
 3. Select a software asset record. The asset window displays.
 -- ----- ------------------------------------------------------------

 -- ----- ---------------------------------------------------
 4. From the drop-down list, select **Usage Report**.
 -- ----- ---------------------------------------------------

 -- ----- ----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------
 5. In the Software Report window, select the time period for which you want to see the usage (such as last 24 hours or a custom range). The window updates and displays all records that match the date range criteria.
 -- ----- ----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------

 -- ----- ------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------
 6. From the drop-down list, select the time period for which you want to see usage (such as last 24 hours or a custom range). The window updates and displays all records that match the date range criteria.
 -- ----- ------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------

::: {.Ind50
![sw usage report new](../../../../Resources/Images/Admin - SACM/sw-usage-report-new.PNG)

 -- ----- -------------------------------------------------------------
 7. Select the record for which you want to view usage details.
 -- ----- -------------------------------------------------------------

 -- ----- -------------------------------------------------------------------------------------------------
 8. To export the details, click **Export**. A confirmation email is sent when the report is ready.
 -- ----- -------------------------------------------------------------------------------------------------

 -- ----- -------------------------------------------------------------------------------------------------------------------------------------------------------
 9. To view the details for a report, click the line item containing the report. The window updates and displays all the details for the selected report.
 -- ----- -------------------------------------------------------------------------------------------------------------------------------------------------------

::: {.Ind50
![sw usage report details new 622x174](../../../../Resources/Images/Admin - SACM/sw-usage-report-details-new_622x174.png)

 -- ------ ----------------------------------------------------------------------------------
 10. To view the configuration information for the report, click the CI Id hyperlink.
 -- ------ ----------------------------------------------------------------------------------
