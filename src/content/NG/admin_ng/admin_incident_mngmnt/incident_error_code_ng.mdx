# Incident Error Code

Use this function to define an error message for Incident Management to be displayed for incorrect configuration scenarios.

In the main window, click **Admin > Incident Management > Incident Error Code**. The Incident Error Code window displays.

New Incident Error Code

From the *Select Actions* drop-down list, choose **New Incident Error Code**. The New Incident Error Code window displays.

Enter a *Name* and *Description* for this error code.

Click **Add**.
