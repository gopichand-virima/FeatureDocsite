## Accessing Run a Scan

1. Go to **Discovery Scan \> Run a Scan** from the left-hand navigation menu.

2. Complete the scan configuration fields as described below, then click **Run**.

 ![Initiating and Configuring 670x249](../../Resources/Images/Discovery/Run-a-Scan/Initiating and Configuring_670x249.png)

3. The system navigates you to the **Run a Scan** page.

 ![Initiating and Configuring 1](../../Resources/Images/Discovery/Run-a-Scan/Initiating and Configuring_1.png)
