# Edit an Import from AWS

Use this function to edit the values of the configured parameters and update the information in the application.

 -- ----- -----------------------------------------------------------------------
 1. In the navigation pane, select **Discovery Scan \> Import from AWS**.
 -- ----- -----------------------------------------------------------------------

 -- ----- --------------------------------------------------------------------------------------------------------------------------------------------------------------------
 2. Select the record to edit. The Edit Report window displays the configured CI details into the following categories: **All**, **Authorized**, and **Unauthorized**.
 -- ----- --------------------------------------------------------------------------------------------------------------------------------------------------------------------

::: Bullet2
*Authorized* represents all assets that are already present inside CMDB.

::: Bullet2
*Unauthorized* represents new assets.

::: Ind50
![import aws details new 627x157](../Resources/Images/Discovery/import-aws-details-new_627x157.png)

 -- ----- ------------------------------------------
 3. To move a record CMDB, do the following:
 -- ----- ------------------------------------------

 -- ----- ------------------
 a. Select a record.
 -- ----- ------------------

 -- ----- -------------------------
 b. Click **Move to CMDB**.
 -- ----- -------------------------

 -- ----- -------------------------------------------------------------------------------------------------------------------------------------------
 c. In the Alert Message window, click **Continue** to move the record. The process runs in the background and no further action is required.
 -- ----- -------------------------------------------------------------------------------------------------------------------------------------------
