# Kubernetes Services

 --- -------------------------------------------------------------------------------------------------------------
   This function is only available with certain asset types. Therefore, the relevant tab may not be displayed.
 --- -------------------------------------------------------------------------------------------------------------

 -- ----- ------------------------------------------------------
 1. While viewing an asset record, click **Components**.
 -- ----- ------------------------------------------------------

 -- ----- --------------------------------
 2. Click **Kubernetes Services**.
 -- ----- --------------------------------

::: Ind50
![kubernetes services new 467x140](../../../Resources/Images/Components/kubernetes-services-new_467x140.png)

## Add a New Kubernetes Service

 -- ----- ----------------------------------------------------------------
 1. Click **Add**. The New Kubernetes Service dialog box displays.
 -- ----- ----------------------------------------------------------------

::: Ind50
![kubernetes services add new 384x122](../../../Resources/Images/Components/kubernetes-services-add-new_384x122.png)

 -- ----- ----------------------------------
 2. Click **New Kubernetes Service**
 -- ----- ----------------------------------

 -- ----- -------------------------------------------------
 3. Enter the applicable information in each field.
 -- ----- -------------------------------------------------

 -- ----- --------------------------------------------
 4. When all entries are made, click **Save.**
 -- ----- --------------------------------------------

## Modify Existing Kubernetes Service

 -- ----- ----------------------------------------
 1. Click **Existing Kubernetes Service**.
 -- ----- ----------------------------------------

::: Ind50
![kubernetes services existing new 372x249](../../../Resources/Images/Components/kubernetes-services-existing-new_372x249.png)

 -- ----- ----------------------------------------------------------------------------------------------------------------------------------------------------------------------
 2. Enter the search criteria to locate the existing device.  *See also* [Advanced Search](../../../Common%20Topics/Advanced-Search.mdx){style="font-style: italic;"}*.*
 -- ----- ----------------------------------------------------------------------------------------------------------------------------------------------------------------------

 -- ----- -----------------------------------------------------
 3. Select the applicable device from the results list.
 -- ----- -----------------------------------------------------

 -- ----- -------------------------------------------------------
 4. When all selections/entries are made, click **Save**.
 -- ----- -------------------------------------------------------

## Delete Kubernetes Service

While viewing the list of existing devices, select the device to delete, and click the **Delete** button.
