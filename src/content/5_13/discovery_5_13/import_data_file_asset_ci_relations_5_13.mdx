# Import Asset/CI Relations File

Use this function to upload and attach an asset/CI relations file

In the navigation pane, select **Discovery Scan > Import Data Files**. The Import Data Files window displays.

! !

In the *Select Actions* drop-down list, choose **Import Asset/CI Relations.** The Attach Asset/CI Relations File dialog box displays.

! !

To view a sample file, select **Click here to download the sample file**.

In the *Upload Data* field, click **Browse** and select an allowed file type (.xls, .xlsx and .cvs).

Click **Upload**.
