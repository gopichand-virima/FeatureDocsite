# [Scan]{style="background-color: #ffffff;"}

This action opens a configuration window that allows you to scan discovery agents.

 -- ----- -------------------------------------------------------------
 1. In the main window, click **Admin \> Discovery \> Client**.
 -- ----- -------------------------------------------------------------

 -- ----- --------------------------------------------------------------------------------------
 2. Select the applicable client. A new window opens with the **Details** tab displayed.
 -- ----- --------------------------------------------------------------------------------------

 -- ----- --------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------
 3. Click the **Discovery Agents** tab.![Discovery Agent Select Actions](../Resources/Images/Admin - Discovery/Discovery-Agent-Select-Actions.png)
 -- ----- --------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------

 -- ----- ---------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------
 4. From the *Select Actions* drop-down list, choose **Scan**. A confirmation dialog box displays.![Scan 6.1](../Resources/Images/Admin - Discovery/Scan-6.1.png)
 -- ----- ---------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------

 --- -----------------------------------------------------------------
 Â  The Scan action appears only if the discovery agent is present.
 --- -----------------------------------------------------------------

 -- ----- -----------------------------------------------
 5. Click **Yes** to confirm or **No** to cancel.
 -- ----- -----------------------------------------------

 -- ----- --------------------------------------------------------------
 6. Redirects to *Run a Scan* page, where user can perform scan.
 -- ----- --------------------------------------------------------------
