# Program and Project Management - Online Help Overview

Welcome to the Program and Project Management module documentation for Virima version 5.13.

## Overview

The Program and Project Management module enables IT teams to plan, execute, and monitor projects and programs, ensuring successful delivery of IT initiatives with proper resource allocation and tracking.

## Key Features

- **Project Planning**: Create and manage project plans
- **Task Management**: Assign and track tasks and milestones
- **Resource Allocation**: Manage team assignments and workload
- **Gantt Charts**: Visual project timeline representation
- **Budget Tracking**: Monitor project costs and expenses
- **Portfolio Management**: Oversee multiple projects

## Getting Started

To begin managing projects:
1. Navigate to Program and Project Management
2. Create a new project
3. Define tasks and milestones
4. Assign resources
5. Set project timeline and budget
