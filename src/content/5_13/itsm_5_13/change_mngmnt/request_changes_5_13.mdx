# Request for Changes

Use this function to submit a formal proposal for implementing a change. The Request for Change (RFC) provides the details for this change.

In the navigation pane, select **ITSM > Change Management > Request for Changes**. The Request for Changes window displays.


New Request for Change

From the *Select Actions* drop-down list, choose **New RFC**. The New RFC window displays.

Click **New**. The New RFC window displays.

Enter a *Title* and *Description*.

In the following fields, click **Addthe plus + sign**, then search for and select the applicable item:

- Associate CI
- Associate Problems
- Associate Known Errors

In the following fields, click the drop-down list and select the applicable choice:

- CI Category
- CI SubCategory
- Urgency
- State
- Approval State

When all selections/entries are made, click **Add**.
