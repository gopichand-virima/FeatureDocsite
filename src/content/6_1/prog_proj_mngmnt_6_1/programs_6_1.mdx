# Programs

Use this function to manage several related projects.

In the main window, select **Program and Project Management** **> Programs**.


New Program

From the *Select Actions* drop-down list, choose **New Program**.

Complete the fields, referring to the table below.

When all selections/entries are made, click **Add**.

### Program Fields

| Field | Description |

| Name | A generic name given to the program. |

| Description | Give a brief description about the program. |

| Program Manager | The role responsible for overseeing the ongoing/ completion of the program. |

| Project Template | Associate the project template. |

| Status | Indicates the current state of the program.

- **Critical**: Specifies that the program is critical in nature.
- **At Risk**: Specifies that the program is associated with some risks.
- **Normal**: Specifies that the program is not associated with risks. |

| State | Indicates the current state of the program.

- **Not Started**: The default state of the program.
- **Active**: Indicates that the program is still ongoing.
- **Suspended**: Indicates that the program was suspended or canceled due to some reasons.
- **Completed**: Indicates that the program is completed. |

| Program Start Date

Program End Date | Dates on which the program starts and ends. |

| Cost Center | The business unit for which cost of the program is charged. |

| Percentage Completion | The percentage of the program that is completed. |

| Project | Associates a project(s) to a program. |


### Program Heat Map

Use this function to view the phase status of associated project(s). If a phase status is changed, or if a new phase has been added/deleted to the associated project, then the change gets updated.

While viewing the Details tab, click the **Program Heat Map** tab.

- The color of the phases indicates their respective status as shown in the legend.
- Click on the phase to change the status of the associated project directly from the program.
- If an associated project gets suspended, it won't affect the status of the program.
