# Overview

The <span class="Strong" style="font-weight: bold;">Azure AD User Import Logs</span> in Virima provide a centralized view of user data imported from <span class="Strong" style="font-weight: bold;">Azure Active Directory (AAD)</span>. Administrators can track when imports start and finish, verify completion status, and identify issues quickly.

This feature helps ensure <span class="Strong">data accuracy, compliance, and troubleshooting</span> by maintaining a clear history of all import activities. Logs can be filtered by name or date, and detailed records are available for review.
