# Problem Error Code

Use this function to define an error message to display in case of incorrect configuration scenario.

In the main window, click **Admin > Change Management > Change Error Code**. The Change Error Code window displays.

New Change Error Code

From the *Select Actions* drop-down list, choose **New Change Error Code**. The New Change Error Code window displays.

Enter a *Name* and *Description*.

Click **Add**.
