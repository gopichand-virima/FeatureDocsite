# Relationship Types

Use this function to define the relationship between a source and target.

In the main window, click **Admin > SACM > Relationship Types**. The Relationship Types window displays.

! New Relationship Type

From the *Select Actions* drop-down list, choose an action: **New Relationship Type.**

Enter the *Primary Name* to define the relationship from the Source to the Target.

Enter the *Reverse Name* to define the relationship from the Target to the Source.

Select a *Color* to represent the relationship between the Source and the Target and vice versa.

Check if the *UserType Relation* should be applied.

Select a **Line Stroke**.

Click **Add**.

From the *Select Actions* drop-down list, choose  Import

The Attach Data Files dialog appears. Click **Browse** to upload the prepare file.

A sample template file can be download from the dialog. Click **Upload** to complete the import.!
