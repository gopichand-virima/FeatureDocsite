# Azure AD User Import Logs

Use this function to view imported logs containing details Azure AD Users.

 -- ----- --------------------------------------------------------------------------------------------------------------------------------
 1. In the navigation pane, select **Discovery Scan \> Azure AD User Import Logs**. The Azure AD User Import Logs window displays.
