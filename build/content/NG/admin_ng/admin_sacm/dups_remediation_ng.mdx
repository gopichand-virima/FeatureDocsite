# Duplicates Remediation

Use this function to view duplicate CIs present in the application.

In the main window, click **Admin > SACM > Duplicates Remediation**. The Duplicates Remediation window displays.

In the *Select Properties* field, click the drop-down list arrow and select the applicable property(ies).

Click **View Duplicates**. A list of all matching items and the number of duplicates is shown.

To view the details for a specific item, click the **number hyperlink**.

Click **All Details** to view all the details for the duplications (Grouped Values, Count, and Asset IDs).

Click **Hide Details** to close the details information.

To merge different CI contents in a single CI and delete the others, select the required CIs, then click **Retain and Delete CIs**.

To merge the content of existing CIs marked as duplicates, click the line item, then click **Merge CIs**.
