# Discovery Scan - Compatibility Matrix

Supported platforms and system requirements for Discovery Scan module.

## Supported Platforms

### Network Devices
- Cisco IOS/IOS-XE
- Juniper JUNOS
- HP/Aruba Network OS
- Dell Networking OS
- Palo Alto Networks PAN-OS

### Operating Systems
| OS | Versions Supported |
|----|-------------------|
| Windows Server | 2012 R2 - 2022 |
| Windows Desktop | 10, 11 |
| Red Hat Enterprise Linux | 7, 8, 9 |
| Ubuntu | 18.04 LTS - 22.04 LTS |
| CentOS | 7, 8 |
| Debian | 9, 10, 11 |

### Cloud Platforms
- Amazon Web Services (AWS)
- Microsoft Azure
- Google Cloud Platform (GCP)
- Oracle Cloud Infrastructure

### Virtualization
- VMware vSphere 6.5+
- Microsoft Hyper-V 2016+
- Citrix XenServer
- KVM

## Protocol Requirements

| Protocol | Port | Purpose |
|----------|------|---------|
| SNMP | 161/UDP | Network device discovery |
| WMI | 135/TCP, 445/TCP | Windows discovery |
| SSH | 22/TCP | Linux/Unix discovery |
| HTTPS | 443/TCP | Cloud API access |

## Network Requirements

- Outbound internet access for cloud discovery
- Internal network access to target systems
- DNS resolution capability
- NTP synchronization recommended
