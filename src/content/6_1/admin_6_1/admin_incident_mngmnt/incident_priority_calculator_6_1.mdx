# Priority Calculator

Use this function to set values for Urgency and Impact.

|     |                                                       |
|-----|-------------------------------------------------------|
|     | Role access must be given to perform these functions. |

In the main window, click **Admin /> Incident Management /> Incident Priority Calculator**. The Priority Calculation window displays.
