# Windows Updates

This function is only available with certain asset types. Therefore, the relevant tab may not be displayed.


- While viewing an asset record, click the **Components** option.
- Click **Windows Updates**.

!

- Click a list item to view the details for the update.

!

- Click **Available Patches** to install available [patches](/itsm/config_mngmt/config_items/config_item_components/hard_drives_6_1.mdx#WindowsPatches).
