# Imported Assets

Use this function to view the results of data imported during the data files import operation and move them (if applicable) to CMDB. An asset can also be rescanned through this function.

- In the navigation pane, select **Discovery Scan > Imported Assets**. The Imported Assets window displays.

![A screenshot of a computer Description automatically generated](../../Resources/Images/Discovery/imported-assets.png) !

- Select at least one or more records. (When rescanning, the asset must have an IPÂ address.)
- From the *Select Actions* drop-down list, choose either [Move to CMDB](/discovery/move_cmdb_6_1.mdx) or [Re-Scan](/discovery/discovered_items/re_scan_6_1.mdx).
- At the prompt, click **Yes**.
