# Software License Keys

Use this function to configure a license key for a specific software asset.

In the navigation pane, select **ITAM \> Software Asset Management \> Software License Keys**. The Software License Keys window displays.

![sw lic key main old](../../Resources/Images/ITAM/sw-lic-key-main-old.PNG)

![arrow border](../../Resources/Images/arrow-border.PNG)

 1. In the *Select Actions* drop-down list, choose **New**. The Software License Key dialog box displays.

 --- -----------------------------------------------------------------------------------------
   Not all functions include an option to edit record line items shown in the main window.
 --- -----------------------------------------------------------------------------------------

 -- ----- -------------------------------------------------------
 1. Navigate to the window containing the record to edit.
 -- ----- -------------------------------------------------------

 -- ----- -------------------------------------------------------------------------------------
 2. Click the line containing the record. The applicable window or dialog box displays.
 -- ----- -------------------------------------------------------------------------------------

 -- ----- -----------------------------------
 3. Make the necessary modifications.
 -- ----- -----------------------------------

 -- ----- -------------------------------------------
 4. Click **Save** or **Add**, as applicable.
 -- ----- -------------------------------------------

The following information and sub-functions are shown in this window:

::: Bullet1
**Used On**. Lists the assets with the corresponding Software License key.

::: Bullet1
**ITSM**. Displays and allows for the creation of [Incidents](/itsm/incident_mngmt/incidents_6_1.mdx), [Requests](/itsm/request_fulfillment/requests_6_1.mdx), [Changes](/itsm/change_mngmnt/changes_6_1.mdx), [Problems](/itsm/problem_mngmt/problems_6_1.mdx), [Known Errors](../../ITSM/Problem-Mngmt/Known-Errors.mdx), [Releases](/itsm/release_mngmt/releases_6_1.mdx), and [Knowledge](../../ITSM/Knowledge-Mngmt/About-Knowledge-Mngmnt.mdx).

::: Bullet1
**Relationships**. Relates the software asset to an existing *Source* and *Target* and view the Business Service Map.

 -- ----- ------------------------------------------------------------------------------------------
 a. Click the **Add Source** or **Add Target** button. The Add Relation dialog box displays.
 -- ----- ------------------------------------------------------------------------------------------

::: {.Ind75
![sw lic key add relation old](../../Resources/Images/ITAM/sw-lic-key-add-relation-old.PNG)

 -- ----- ------------------------------------------------------------
 b. Select the blueprint, and search for the applicable asset.
 -- ----- ------------------------------------------------------------

 -- ----- -----------------------------------------------------------------------------------
 c. In the *Choose Relation* field, click the drop-down list and choose the relation.
 -- ----- -----------------------------------------------------------------------------------

 -- ----- ----------------
 d. Click **Add**.
 -- ----- ----------------

::: Bullet1
**SLA**. Configures response and resolution times for the Software License Key if marked as Incident, Requests, Changes, or Problems for resolution. Specify the *Response* and *Resolution Time* as well as the *Support* Levels. Then, configure the *Escalate* parameters.

::: {.Ind50
![sw lic key sla old](../../Resources/Images/ITAM/sw-lic-key-sla-old.PNG)

::: Bullet1
**Maintenance**. Set of activities defined to conserve an asset in its original condition. *Refer to [CI: Maintenance](../../ITSM/Config%20Mngmt/Config%20Items/Config%20Item%20Details/Config-Items-Maintenance.mdx).*

::: Bullet1
**Audits**. Analyzes the product or its process to assess compliance with defined specifications or the standards of the deliverable.\
\
To add an Audit:

 -- ----- ---------------------------------
 a. Click the **Add Audit** button.
 -- ----- ---------------------------------

::: {.Ind50
![sw lic key audit old](../../Resources/Images/ITAM/sw-lic-key-audit-old.PNG)

 -- ----- -----------------------------------
 b. Enter a *Name* and *Description*.
 -- ----- -----------------------------------

 -- ----- ------------------------------------------------------------------------------------------------------
 c. In the *Auditor* field, click **Add,** then search for and select a user to function as the Auditor.
 -- ----- ------------------------------------------------------------------------------------------------------

 -- ----- -----------------------------------
 d. Select the *Recurring Frequency*.
 -- ----- -----------------------------------

 -- ----- ------------------------------------------------------------------------------
 e. Enter the *Duration (in days)* to indicate the length of time for the audit.
 -- ----- ------------------------------------------------------------------------------

 -- ----- ------------------------------------------------------
 f. When all selections/entries are made, click **Add**.
 -- ----- ------------------------------------------------------

When editing an existing item, other options may be available, such as adding [Tasks](/itsm/release_mngmt/tasks_6_1.mdx){.MCTopicPopup .MCTopicPopupHotSpot .a}, [Comments](/discovery/comments_6_1.mdx){.MCTopicPopup .MCTopicPopupHotSpot .a}, and [Attachments](/discovery/attachments_6_1.mdx){.MCTopicPopup .MCTopicPopupHotSpot .a} and viewing [History](/common_topics/history_6_1.mdx){.MCTopicPopup .MCTopicPopupHotSpot .a}.

 --- ---------------------------------------------------------------------------------------
   The options change based upon the function; therefore, not every option may be shown.
 --- ---------------------------------------------------------------------------------------
