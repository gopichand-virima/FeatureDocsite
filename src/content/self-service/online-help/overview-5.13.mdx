# Self Service - Online Help Overview

Welcome to the Self Service module documentation for Virima version 5.13.

## Overview

The Self Service module empowers end users to request services, access knowledge base articles, and track their requests without IT staff intervention, improving efficiency and user satisfaction.

## Key Features

- **Service Catalog**: Browse and request IT services
- **Knowledge Base**: Self-help articles and FAQs
- **Request Tracking**: Monitor request status in real-time
- **User Portal**: Personalized dashboard for end users
- **Mobile Access**: Request services on the go

## Getting Started

To begin using Self Service:
1. Access the Self Service portal
2. Browse available services in the catalog
3. Submit service requests
4. Search the knowledge base
5. Track your open requests
