# Edit Infoblox Appliance

Use this function to change the details of an Infoblox Appliance.

In the Infoblox Appliances window, select an existing Infoblox Appliance record. The New Infoblox Appliance window displays.

Modify any of the following fields:

- **Name**. Enter a name for the InfoBlox Appliance.
- **IP Address**. Enter the IP address for the InfoBlox Appliance.
- **Credential**. Click the drop-down list and select a credential.

When all selections are made, click **Save & Close**. The window closes and refreshes and shows the edited Infoblox Appliance.
