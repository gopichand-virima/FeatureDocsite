# Release Phase State

Use this function to add a new state defining the status of the Release Phase being worked by an assignee.

In the main window, click **Admin /> Release Management /> Release Phase State**. The Release Phase State window displays.

<span class="image placeholder" original-image-src="../Resources/Images/Admin%20-%20Release/release-phase-state-main-old.PNG" original-image-title=""></span>

In the main window, click the **gear** icon and select **Release Management /> Release Phase State.**

<span class="image placeholder" original-image-src="../Resources/Images/Admin%20-%20Release/icon-release-phase-state.PNG" original-image-title=""></span>

The  Release Phase State window displays.

<span class="image placeholder" original-image-src="../Resources/Images/Admin%20-%20Release/release-phase-state-main-new.PNG" original-image-title=""></span>
