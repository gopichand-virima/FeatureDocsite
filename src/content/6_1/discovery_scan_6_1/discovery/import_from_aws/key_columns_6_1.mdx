# Key columns

- <span class="Strong">Blueprint Name</span> – The discovery blueprint that matched the resource (e.g., <span class="Emphasis">AWS FARGATE</span>, <span class="Emphasis">AWS ECS</span>, <span class="Emphasis">AWS EKS</span>).

- <span class="Strong">Asset Name</span><span class="inline-comment-marker"> – The </span>discovered resource name (cluster/service/task/instance, etc.).

  - The asset name is the AWS instance ID.

- <span class="Strong">Status</span><span class="inline-comment-marker">: Indicates the current state of the CI imported from AWS. When a CI is newly imported (does not exist in CMDB), the status is shown as "New."</span>

- <span class="Strong">Asset ID</span>: A unique, sequential identification number assigned to each discovered AWS record. The ID shall have a prefix “DSC”+ 6 digits Unique sequential number.

Users can select which columns to display when importing AWS and sort the grid by any column.
