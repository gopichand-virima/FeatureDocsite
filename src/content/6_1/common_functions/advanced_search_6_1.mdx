## Advanced Search

Use this function to search assets specifically as per defined values.

When this option displays in a window (such as Adding a New Task), click **Advanced Search.** The Advanced Search window displays.

<div class="numlist1" madcap:conditions="Default.OldUI" madcap:autonum="1. ">

In /<**choose field**/> (for example, Display Name), click the drop-down list and select from the field names shown. A "field name" can be such items as Display Name, Roles, Groups, and so forth.

</div>

<div class="numlist1" madcap:conditions="Default.NewUI" madcap:autonum="1. ">

In /<**choose field**/> (for example, First Name), click the drop-down list and select from the field names shown. A "field name" can be such items as Display Name, Roles, Groups, and so forth.

</div>

<div class="numlist2" madcap:conditions="Default.OldUI" madcap:autonum="2. ">

For the /<**oper**/> (for example, Contains), click the drop-down list and select from the choices shown. An "operator" can be such items as Contains, Does Not Contain,  Starts with and so forth.

</div>

<div class="numlist2" madcap:conditions="Default.NewUI" madcap:autonum="3. ">

For the /<**choose operator**/> (for example, Does Not Contain), click the drop-down list and select from the choices shown. An "operator" can be such items as Contains, Does Not Contain, Starts with, and so forth.

</div>

<div class="numlist2" madcap:conditions="Default.OldUI" madcap:autonum="4. ">

For */<***value***/>* (for example, C), click the drop-down list and select the specific criteria to search for based on the field type. For example, if the *Field Type* selected is "Display Name," the /<value/> can be a portion of the possible display name, such as the letter C.

</div>

<div class="numlist2" madcap:conditions="Default.NewUI" madcap:autonum="5. ">

For */<***value***/>* (for example, a), click the drop-down list and select the specific criteria to search for based on the field type. For example, if the *Field Type* selected is "First Name," the /<value/> can be a portion of the possible first name, such as the letter a.

</div>

<div class="numlist2" madcap:autonum="6. ">

To add an additional row with a field type, operator, and value, click **AND** or **OR**. A new row is added.

</div>

<div class="numlist2" madcap:autonum="7. ">

Repeat the above steps for each field.

</div>

<div class="numlist2" madcap:autonum="8. ">

To remove a row, click the **X** at the end of the row.

</div>
