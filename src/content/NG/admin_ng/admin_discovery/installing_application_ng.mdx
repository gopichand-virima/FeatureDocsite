# Installing the Application

1. **Run the Installer:**

 - Right-click the installer and select **Run as administrator**.

2. **Windows SmartScreen Warning:**

 - If prompted, select **More info**, and then select **Run anyway** to continue the installation.

![Installing the Application](../Resources/Images/Admin - Discovery/Installing the Application.png)

![Installing the Application 1 549x249](../Resources/Images/Admin - Discovery/Installing the Application_1_549x249.png)

3. Click **Next** to proceed.

![Installing the Application 2](../Resources/Images/Admin - Discovery/Installing the Application_2.png)

4. Click **Install** to begin installation.

![Installing the Application 3](../Resources/Images/Admin - Discovery/Installing the Application_3.png)

5. Click **Skip** if prompted.

![Installing the Application 4](../Resources/Images/Admin - Discovery/Installing the Application_4.png)

6. Click **Finish** to complete the installation.

![Installing the Application 5](../Resources/Images/Admin - Discovery/Installing the Application_5.png)

7. **Launch the Application:**

 - Find the **Virima Discovery Application** icon on your desktop.

 - Right-click and select **Run as administrator**.

![Installing the Application 6](../Resources/Images/Admin - Discovery/Installing the Application_6.png)
