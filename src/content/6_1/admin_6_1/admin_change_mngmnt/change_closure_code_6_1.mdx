# Change Closure Code

Use this function to configure a code with its description for the Closed state in the Change Management function.

In the main window, click **Admin /> Change Management /> Change Closure Code**. The Change Closure Code window displays.

<span class="image placeholder" original-image-src="../Resources/Images/Admin%20-%20Change/closure-code-main-old.PNG" original-image-title=""></span>

In the main window, click the **gear** icon and select **Change Management /> Change Closure Code**.

<span class="image placeholder" original-image-src="../Resources/Images/Admin%20-%20Change/icon-change-closure-code.PNG" original-image-title=""></span>

The Change Closure Code window displays.

<span class="image placeholder" original-image-src="../Resources/Images/Admin%20-%20Change/closure-code-main-new.PNG" original-image-title=""></span>
