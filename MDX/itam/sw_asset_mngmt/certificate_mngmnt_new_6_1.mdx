# New Certificate Management

This feature enables to configure certificate license details by entering values for the parameters given in the application page.

- Choose New Certificate Details option from Select Actions drop-down. The New Certificate Details page displays.

!

- Enter the IP address of the software /hardware for which certificate details to be configured in the IP Address.
- Enter the name of the software /hardware installed host in the Host Name.
- Enter software / hardware installed host type in the Host Type box.
- Enter the name of the host type operating system in the OS box.
- Enter the name of the certificate configured for the corresponding software /hardware in the Certificate Name box.
- Enter the location of the configured /saved certificate in the path box.
- Enter the name of the company /issuer of the corresponding certificate in the Issuer box.
- Click Start Date, and Expiry Date text box to select the duration of the certificate accordingly.
- Click Add to update CI(s) to be associated with the corresponding certificate.
- Click Add to add update configured Certificate Details in the application.

Instructions For Add:

- Enter the search criteria in the given text box, and then click Search icon.
- The results are displayed as per the given search criteria.
- Select a required value, and then click Save.
- Click Cancel to discard the action, and in continuation no value will be selected.
