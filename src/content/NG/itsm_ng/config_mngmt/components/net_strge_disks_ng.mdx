# Network Storage Disks

 --- -------------------------------------------------------------------------------------------------------------
   This function is only available with certain asset types. Therefore, the relevant tab may not be displayed.
 --- -------------------------------------------------------------------------------------------------------------

 -- ----- ------------------------------------------------------
 1. While viewing an asset record, click **Components**.
 -- ----- ------------------------------------------------------

 -- ----- ----------------------------------
 2. Click **Network Storage Disks**.
 -- ----- ----------------------------------

::: Ind50
![network storage disks new 623x163](../../../Resources/Images/Components/network-storage-disks-new_623x163.png)

## Add a New Network Storage Disk

 -- ----- --------------------------------------------------------------
 1. Click **Add**. The Network Storage Disk dialog box displays.
 -- ----- --------------------------------------------------------------

::: Ind50
![network storage disks add new 482x217](../../../Resources/Images/Components/network-storage-disks-add-new_482x217.png)

 -- ----- -------------------------------------
 2. Click **New Network Storage Disk.**
 -- ----- -------------------------------------

 -- ----- -----------------------------------------
 3. Enter the applicable field information.
 -- ----- -----------------------------------------

 -- ----- -------------------------------------------------------
 4. When all selections/entries are made, click **Save**.
 -- ----- -------------------------------------------------------

## Modify Existing Network Storage Disk

 -- ----- ------------------------------------------
 1. Click **Existing Network Storage Disk**.
 -- ----- ------------------------------------------

::: Ind50
![network storage disks existing new 479x328](../../../Resources/Images/Components/network-storage-disks-existing-new_479x328.png)

 -- ----- ----------------------------------------------------------------------------------------------------------------------------------------------------------------------
 2. Enter the search criteria to locate the existing device.  *See also* [Advanced Search](../../../Common%20Topics/Advanced-Search.mdx){style="font-style: italic;"}*.*
 -- ----- ----------------------------------------------------------------------------------------------------------------------------------------------------------------------

 -- ----- -----------------------------------------------------
 3. Select the applicable device from the results list.
 -- ----- -----------------------------------------------------

 -- ----- -----------------------------------------------
 4. When all selections are made, click **Save**.
 -- ----- -----------------------------------------------

## Delete Network Storage Disk

While viewing the list of existing devices, select the device to delete, and click the **Delete** button.
