# Hardware Properties

Use this function to configure the properties for a hardware asset.

In the navigation pane, select **Admin > Hardware Asset Management > Hardware Properties**. The Hardware Properties window displays.


### New Hardware Properties

- From the *Select Actions* drop-down list, choose **New Hardware Property**. The Add Hardware Properties window displays.

- Enter the *Property Name*.
- In the *Property Type*, *Property Style*, and *Property Group* fields, click the drop-down list and select the applicable item.
- In the *UI Order* field, type the number for the order in which this property is to display.
- In the *Is Visible*, *Is Mandatory*, and *Is Multiple* fields, select if these are required.
- When all selections/entries are made, click **Save**.


### Edit Hardware Properties

Not all functions include an option to edit record line items shown in the main window.


- Navigate to the window containing the record to edit.
- Click the line containing the record. The applicable window or dialog box displays.
- Make the necessary modifications.
- Click **Save** or **Add**, as applicable.

When editing an existing item, other options may be available, such as adding [Tasks](/itsm/release_mngmt/tasks_ng.mdx), [Comments](/discovery/comments_ng.mdx), and [Attachments](/discovery/attachments_ng.mdx) and viewing [History](/common_topics/history_ng.mdx).


The options change based upon the function; therefore, not every option may be shown.


### Delete Hardware Properties

Deleting is a permanent action and cannot be undone. Deleting may affect other functionality and information in the application such as data in configured reports, fields in windows, selectable options, etc. Therefore, be sure to understand the potential effects before making a deletion.


- Click the line that contains the item to delete.
- From the *Select Actions* drop-down list, choose **Delete**. If a confirmation message is displayed, Type **Delete** and click **Delete**.
