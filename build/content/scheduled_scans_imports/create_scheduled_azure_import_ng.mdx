# Creating a Scheduled Azure Import

1. From the **Select Actions** dropdown, choose **New Scheduled Azure Import**.

2. A configuration dialog opens.

**Name:** Enter a descriptive name.

**Credential:** Choose the Azure credential.

**Select Services:** Check the Azure services to import (e.g., Virtual Machines, Storage Accounts, Databases).

**Import Frequency:** Define the frequency (e.g., Every Day, Weekly).

**Active:** Enable the schedule.

**Timezone and Timing:** Configure as needed.

1. Click **Add**.
