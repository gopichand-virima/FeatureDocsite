# ITSM - Compatibility Matrix

System requirements and compatibility information for the ITSM module.

## Browser Compatibility

| Browser | Minimum Version | Recommended |
|---------|----------------|-------------|
| Chrome | 90+ | Latest |
| Firefox | 88+ | Latest |
| Safari | 14+ | Latest |
| Edge | 90+ | Latest |

## User Permissions

| Role | Permissions |
|------|------------|
| End User | Create tickets, view own tickets |
| Technician | Work on assigned tickets, update status |
| Team Lead | Assign tickets, view team workload |
| Administrator | Full ITSM configuration access |

## Email Integration

Supported email protocols:
- IMAP
- POP3
- Microsoft Exchange (EWS)
- Office 365 (Graph API)
- Gmail API

## Third-Party Integrations

Compatible with:
- Slack (notifications)
- Microsoft Teams (notifications)
- Jira (ticket sync)
- ServiceNow (data exchange)
- Zendesk (migration)

## Mobile App Support

- iOS 13+
- Android 9+
- Progressive Web App (PWA) support
