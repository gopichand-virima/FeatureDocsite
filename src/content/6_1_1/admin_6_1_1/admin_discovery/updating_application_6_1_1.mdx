# Updating the Discovery Application

Use this function to update the Discovery Application to the latest version.

## Automatic Updates

The Discovery Application can be configured to automatically update when new versions are available.

In the [Client](/admin_discovery/client_6_1_1.mdx) configuration, enable **App Auto Update Enabled**.

## Manual Updates

Download the latest version from **Admin > Discovery > Download Application**.

Stop the existing Discovery Application service.

Run the new installer.

Follow the installation prompts.

Restart the Discovery Application service.

## Update Agents

To update agents deployed across your environment:

Navigate to **Admin > Discovery > Client**.

Select the client.

Click the **Discovery Agents** tab.

Select the agents to update.

Click **Update Agents**.
