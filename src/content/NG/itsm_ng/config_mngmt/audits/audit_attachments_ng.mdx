# Audit: Attachments

Use this function to attach, download and delete attachments.

 -- ----- -------------------------------------------------------------------
 1. In the main window, click **Configuration Management \> Audits**.
 -- ----- -------------------------------------------------------------------

 -- ----- ------------------------------
 2. Select the record to manage.
 -- ----- ------------------------------

 -- ----- ----------------------------------------------------------
 3. Follow the instructions below for the applicable action.
 -- ----- ----------------------------------------------------------

![arrow border](../../../Resources/Images/arrow-border.PNG)

[[[Closed]{.image .placeholder .MCDropDown_Image_Icon original-image-src="../../../../Skins/Default/Stylesheets/Images/transparent.gif" original-image-title="" height="12" width="16" Attachment](javascript:void(0)){.MCDropDownHotSpot .dropDownHotspot .MCDropDownHotSpot_ .MCHotSpotImage aria-expanded="false"}]{.MCDropDownHead .dropDownHead}

 -- ----- ------------------------------------------------------------------------------------------
 1. [Click the ]{.span_1}[Attachments]{.Strong_2}[ tab.]{.span_1}
 -- ----- ------------------------------------------------------------------------------------------

 -- ----- ------------------------------------------------------------------------------------------
 2. [Click ]{.span_1}[**New Attachment**. The *Attach File dialog box* displays.]{.Strong_2}
 -- ----- ------------------------------------------------------------------------------------------

::: {.Ind50
![attach file dialog box old 442x268](../../../Resources/Images/Admin - Discovery/attach-file-dialog-box-old_442x268.png)

 -- ----- -------------------------------------------------------------------------------------------------------------------------------------------
 3. [In the *Upload Document* field, click ]{.span_1}[Browse]{.Strong_2}[ to select the applicable file.]{.span_1}
 -- ----- -------------------------------------------------------------------------------------------------------------------------------------------

 -- ----- ---------------------------------------------------------------------
 4. In the *Description* field, enter information about the attachment.
 -- ----- ---------------------------------------------------------------------

 -- ----- ---------------------------------------------------------------------------------------------------------------
 5. [Click ]{.span_1}[**Upload**. When you return to the Attachment window, the added item displays. ]{.Strong_2}
 -- ----- ---------------------------------------------------------------------------------------------------------------

[[[Closed]{.image .placeholder .MCDropDown_Image_Icon original-image-src="../../../../Skins/Default/Stylesheets/Images/transparent.gif" original-image-title="" height="12" width="16" Attachment](javascript:void(0)){.MCDropDownHotSpot .dropDownHotspot .MCDropDownHotSpot_ .MCHotSpotImage aria-expanded="false"}]{.MCDropDownHead .dropDownHead}

 -- ----- ------------------------------------------------------------------------------------------
 1. [Click the ]{.span_1}[Attachments]{.Strong_2}[ tab.]{.span_1}
 -- ----- ------------------------------------------------------------------------------------------

 -- ----- -------------------------------------------------------
 2. Locate the row containing the attachment to download.
 -- ----- -------------------------------------------------------

 -- ----- --------------------------------------------------------
 3. Click the **Download** icon. The attachment downloads.
 -- ----- --------------------------------------------------------

 --- -------------------------------------------------------------------------------------------------------------------------
   The process for downloading the attachment is based upon your individual system settings, such as the selected browser.
 --- -------------------------------------------------------------------------------------------------------------------------

[[[Closed]{.image .placeholder .MCDropDown_Image_Icon original-image-src="../../../../Skins/Default/Stylesheets/Images/transparent.gif" original-image-title="" height="12" width="16" Attachment](javascript:void(0)){.MCDropDownHotSpot .dropDownHotspot .MCDropDownHotSpot_ .MCHotSpotImage aria-expanded="false"}]{.MCDropDownHead .dropDownHead}

 --- --------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------
   Deleting is a permanent action and cannot be undone. Deleting may affect other functionality and information in the application such as data in configured reports, fields in windows, selectable options, etc. Therefore, be sure to understand the potential effects before making a deletion.
 --- --------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------

 -- ----- ------------------------------------------------------------------------------------------
 1. [Click the ]{.span_1}[Attachments]{.Strong_2}[ tab.]{.span_1}
 -- ----- ------------------------------------------------------------------------------------------

 -- ----- ----------------------------
 2. Click the **Delete** icon.
 -- ----- ----------------------------

 -- ----- -------------------------------------------------------------------
 3. In the Confirmation window, Type **Delete** and click **Delete**.
 -- ----- -------------------------------------------------------------------
