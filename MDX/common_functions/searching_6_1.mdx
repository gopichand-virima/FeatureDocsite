## Searching

Adding new or updating existing information can be done in may fields. The following is a sample.

- While viewing a Details window, locate the information to change.
- Next to the field, click either the **Add** button (to add a new item) or **Update** button (to change an existing item). The applicable dialog box displays (such as the Includes Blueprint dialog box in this example).

!

- Next to the field, click the **plus + sign**. The applicable dialog box displays (such as the Includes Blueprint dialog box in this example).

[!](../Resources/Images/Admin%20-%20SACM/blueprints-includes-dialog-box-new.PNG)

- In the *Search* field, type the name of the item to locate and click the search icon, press **Enter**. *See also* [Advanced Search](/common_topics/advanced_search_6_1.mdx)*.*
- Do either of the following:
- To add an item, select it under the *Search Results* list.
- To remove an item, select it under the *Selected Items* list.
- Click Clear All to remove the selections.
- Click **Save**.
- Click **Add**.
