# User Groups

Use this function to configure groups among the existing application users by defining their precise roles.

The following information should be configured before creating a new user group:

::: Bullet1
Manager (required)

::: Bullet1
Department (optional, but recommended)

::: Bullet1
Role (optional, but recommended)

 --- ---------------------------------------------------------------------------------------------------------
   [Role Access](../Admin%20-%20Other/Role-Access.mdx) permissions must be set before adding a user group.
 --- ---------------------------------------------------------------------------------------------------------

In the main window, click **Admin \> Users \> User Groups**. The User Groups window displays.

Once you add a group, you can add one or more members to it.

 -- ----- -------------------------------
 1. Click on a group in the list.
 -- ----- -------------------------------

 -- ----- --------------------------------------------------------------------------------
 2. In the Details window, click the **Members** tab. The Members window displays.
 -- ----- --------------------------------------------------------------------------------

 -- ----- --------------------------------------------------------------------------
 3. Click **New User** and search for and select the applicable member(s)*.*
 -- ----- --------------------------------------------------------------------------

 -- ----- -----------------------------------------------
 4. When all selections are made, click **Save**.
 -- ----- -----------------------------------------------

 --- --------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------

To delete an existing user group:

 -- ----- ----------------------------------------------
 1. Click the checkbox for the applicable group.
 -- ----- ----------------------------------------------

 -- ----- --------------------------------------------------------------
 2. From the *Select Actions* drop-down list, choose **Delete**.
 -- ----- --------------------------------------------------------------

 -- ----- -------------------------------------------------------------------
 3. In the confirmation window, Type **Delete** and click **Delete**.
 -- ----- -------------------------------------------------------------------
