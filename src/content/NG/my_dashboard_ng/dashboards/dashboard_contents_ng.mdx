# Dashboard Contents

The Discovery Scan Dashboard can be configured to show a variety of reports. The reports can be added, removed, and placed in various locations on the Dashboard. Each report has options for generating or saving the report.

 -- ----- ------------------------------------------------------------------------------------------------
 1. In the navigation pane, select **Discovery Scan \> Dashboard**. The Dashboard window displays.
 -- ----- ------------------------------------------------------------------------------------------------
