# Program Dashboard

<div class="NumList1" madcap:autonum="1. ">

Click /*<span style="color: #ff4500;">program dashboard icon</span>/* present in All Modules page. Program Dashboard page is displayed.

</div>

<div class="Ind50">

<span class="image placeholder" original-image-src="../Resources/Images/Discovery/program-dashboard-5.png" original-image-title="" style="width: 476px;height: 200px;"></span>

</div>

The page displays following information:

|            |                                                     |
|:----------:|:---------------------------------------------------:|
| Parameter  |                        Value                        |
|    Name    | It is the name given for the corresponding Program  |
|   State    | It is the defined state for the corresponding State |
|   Status   |  It is the present status defining program status   |
| Created On |                                                     |

<div class="NumList2" madcap:autonum="2. ">

Click Add Contents to update required Reports to be displayed in the Dashboard page.

</div>

<div class="Ind50">

<span class="image placeholder" original-image-src="../Resources/Images/Discovery/program-dashboard-add-contents-5.png" original-image-title="" style="width: 466px;height: 174px;"></span>

</div>

<div class="NumList2" madcap:autonum="3. ">

Select required type of the report from the drop-down.

</div>

<div class="NumList2" madcap:autonum="4. ">

Enter the search criteria in the given text box, and then click /*<span style="color: #ff4500;">Search icon</span>/*

</div>

<div class="NumList2" madcap:autonum="5. ">

The results are displayed as per given search criteria.

</div>

<div class="Ind50">

<span class="image placeholder" original-image-src="../Resources/Images/Discovery/program-dashboard-search-5.png" original-image-title="" style="width: 477px;height: 278px;"></span>

</div>

Report Alignment in Dashboard page:

<div class="NumList1" madcap:autonum="1. ">

Select required report, and then click Add to the Top to align selected report at the top of the Dashboard web page.

</div>

<div class="NumList2" madcap:autonum="2. ">

Select required report, and then click Add to the left to align selected report to the left side of the Dashboard web page.

</div>

<div class="NumList2" madcap:autonum="3. ">

Select required report, and then click Add to the right to align selected report to the right side of the Dashboard web page.

</div>

<div class="NumList2" madcap:autonum="4. ">

Select required report, and then click Add to the bottom to align selected report at the bottom of the Dashboard web page.

</div>
