# Accessing the Import Meraki Window

1. In the navigation pane, select **Discovery Scan › Import from Meraki**.

2. The **Import From Meraki** window appears, displaying any records you've already imported.

The page displays a list of **imported runs** from your Meraki networks. Each row represents a single import run.

- **Key columns:**
  - **Id**: Unique identification ID of the import run (e.g., MERAKI000559), records imported from Meraki will have the prefix "MERAKI" followed by a six-digit sequential number.
  - **Name**: User-provided name used to identify the import run.
  - **Organization Id**: The unique identifier for the Meraki organization.
  - **Imported On**: The date and time when the import was completed.
  - **Status**: Current state (e.g., Importing, Completed, Failed).
  - **Imported By**: The logged-in user who initiated the import.
  - **Credential**: Virima credential profile used to connect to Meraki.
  - **Select Actions**: Perform bulk operations on selected or filtered rows.
