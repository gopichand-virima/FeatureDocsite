# Time Track Reports

Use this function to schedule a time and date to send a notification report on the user(s) or user group working on raised requests (for example, assigned Incidents, problems, etc.)

In the main window, click **Admin \> Users \> Time Track Reports**. The Time Track Reports window displays.

![time track report main old 662x138](../Resources/Images/Admin - Users/time-track-report-main-old_662x138.png)

![arrow border](../Resources/Images/arrow-border.PNG)

 -- ----- -----------------------------------------------------------------------------------------------------------------------------------------------------------------
 1. While viewing the Time Track Reports window, from the *Select Actions* drop-down list, choose **New Schedule Report**. The New Schedule Report window displays.
 -- ----- -----------------------------------------------------------------------------------------------------------------------------------------------------------------

[[[Closed]{.image .placeholder .MCDropDown_Image_Icon original-image-src="../../Skins/Default/Stylesheets/Images/transparent.gif" original-image-title="" height="12" width="16" Report](javascript:void(0)){.MCDropDownHotSpot .dropDownHotspot .MCDropDownHotSpot_ .MCHotSpotImage aria-expanded="false"}]{.MCDropDownHead .dropDownHead}

To generate a report:

 -- ----- ----------------------------------------------------------------------------------------------------------------------------------------------------------------
 1. While viewing the Time Track Reports window, from the *Select Actions* drop-down list, choose **Generate Report**. The Generate Report dialog window displays.
 -- ----- ----------------------------------------------------------------------------------------------------------------------------------------------------------------

 --- -----------------------------------------------------------------------------------------
   Not all functions include an option to edit record line items shown in the main window.
 --- -----------------------------------------------------------------------------------------

 -- ----- -------------------------------------------------------
 1. Navigate to the window containing the record to edit.
 -- ----- -------------------------------------------------------

 -- ----- -------------------------------------------------------------------------------------
 2. Click the line containing the record. The applicable window or dialog box displays.
 -- ----- -------------------------------------------------------------------------------------

 -- ----- -----------------------------------
 3. Make the necessary modifications.
 -- ----- -----------------------------------

 -- ----- -------------------------------------------
 4. Click **Save** or **Add**, as applicable.
 -- ----- -------------------------------------------
