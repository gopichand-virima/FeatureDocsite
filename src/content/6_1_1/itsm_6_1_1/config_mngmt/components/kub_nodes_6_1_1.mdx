# Kubernetes Nodes

While viewing an asset record, click **Components**.

Click **Kubernetes Nodes**.

## Add a New Kubernetes Node

Click **Add**. The Kubernetes Nodes dialog box displays.

Click **New Kubernetes Node**

Enter the applicable information in each field.

When all entries are made, click **Save.**

## Modify Existing Kubernetes Node

Click **Existing Kubernetes Node**.

Enter the search criteria to locate the existing device.  *See also* [Advanced Search](/common_topics/advanced_search_6_1_1.mdx)*.*

Select the applicable device from the results list.

When all selections/entries are made, click **Save**.

## Delete Kubernetes Node

While viewing the list of existing devices, select the device to delete, and click the **Delete** button.
