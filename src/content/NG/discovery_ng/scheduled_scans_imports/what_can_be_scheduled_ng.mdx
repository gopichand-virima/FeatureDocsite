# What Can Be Scheduled

# **Discovery Scans**

Network-based discovery targeting specific IP ranges. Scan types include:

- Deep Host Scan
- Basic Host Scan
- Port Scan
- Certificate Scan
- Website Discovery
- Custom scan types configured by administrators

# **Cloud Imports**

Automated imports from cloud platforms:

- **AWS** (EC2, RDS, S3, Lambda, VPC, etc.)
- **Azure** (Virtual Machines, Databases, Storage, etc.)
- **Intune** (managed mobile and desktop devices)
- **Meraki** (Access Points, Switches, Firewalls, Routers)
