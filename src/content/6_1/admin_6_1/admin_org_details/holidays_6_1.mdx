# Holidays

Use this function to specify holidays, which are used to auto-calculate resolution dates as they relate to incidents, changes, and problems.

- In the main window, click **Admin > Organizational Details > Holiday**. The Holiday window displays.

!

- To scroll forward and backward between years, click the left and right arrows next to the date.
- From the *Select Actions* drop-down list, choose **New Holiday**. The Add Holiday dialog box displays.

!

- In the main window, click the **gear** icon and select **Organizational Details > Holidays**.

!

The Holidays window displays and contains a list of all defined holidays.

!

- To scroll forward and backward between years, click the left and right arrows next to the date.

!


### Add Holiday

- Click **New Holiday**. The Add Holiday dialog box displays.

!

- In the *Date* field, use the calendar icon to select a holiday.
- In the *Description* field, type a description.
- If the holiday occurs on the same date each year, check **Is Recurring**.
- Click **Add**.


### Edit Holiday

To edit the holiday details:

- Select a holiday in the list.
- In the Update Holiday dialog box, make the necessary changes.
- Click **Save**.


### Delete Holiday

To delete an existing holiday:

- Select a holiday in the list.
- From the *Select Actions* drop-down list, choose **Delete**. If a confirmation message is displayed, type **Delete** and then click **Delete**.
