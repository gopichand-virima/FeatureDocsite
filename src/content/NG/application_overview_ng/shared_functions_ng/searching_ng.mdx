## Searching

Adding new or updating existing information can be done in may fields. The following is a sample.

 -- ----- -------------------------------------------------------------------
 1. While viewing a Details window, locate the information to change.
 -- ----- -------------------------------------------------------------------

 -- ----- ---------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------
 2. Next to the field, click either the **Add** button (to add a new item) or **Update** button (to change an existing item). The applicable dialog box displays (such as the Includes Blueprint dialog box in this example).
 -- ----- ---------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------

::: {.Ind50 .step-media
![blueprints includes dialog box 469x284](../Resources/Images/Admin - SACM/blueprints-includes-dialog-box_469x284.png)

 -- ----- -----------------------------------------------------------------------------------------------------------------------------------------------
 3. Next to the field, click the **plus + sign**. The applicable dialog box displays (such as the Includes Blueprint dialog box in this example).
 -- ----- -----------------------------------------------------------------------------------------------------------------------------------------------

::: {.Ind50 .step-media
[![blueprints includes dialog box new 471x314](../Resources/Images/Admin - SACM/blueprints-includes-dialog-box-new_471x314.png)](../Resources/Images/Admin%20-%20SACM/blueprints-includes-dialog-box-new.PNG)

 -- ----- ----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------
 4. In the *Search* field, type the name of the item to locate and click the search iconpress **Enter**. *See also* [Advanced Search](Advanced-Search.mdx)*.*
 -- ----- ----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------

 -- ----- -----------------------------
 5. Do either of the following:
 -- ----- -----------------------------

::: Bullet2
To add an item, select it under the *Search Results* list.

::: Bullet2
To remove an item, select it under the *Selected Items* list.

 -- ----- -------------------------------------------
 6. Click Clear All to remove the selections.
 -- ----- -------------------------------------------

 -- ----- -----------------
 7. Click **Save**.
 -- ----- -----------------

 -- ----- ----------------
 8. Click **Add**.
 -- ----- ----------------
