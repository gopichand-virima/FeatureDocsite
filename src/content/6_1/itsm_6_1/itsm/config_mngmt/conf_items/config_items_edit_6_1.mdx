# Edit Configuration Item (CI)

Use this function to edit configuration item details.

<div class="NumList1" madcap:conditions="Default.OldUI" madcap:autonum="1. ">

In the navigation pane, select **ITSM /> Configuration Management /> Configuration Items**. The Configuration Items window displays.

</div>

<div class="NumList1" madcap:conditions="Default.NewUI" madcap:autonum="1. ">

In the navigation pane, select **Configuration Management /> Configuration Items**. The Configuration Items window displays.

</div>

<div class="NumList2" madcap:autonum="2. ">

Select a record to edit and make the necessary modifications. *Refer to* <a href="Config-Items-Options.htm" style="font-style: italic;">Configuration Item Options</a> *for information on each tab.*

</div>

<div class="NumList2" madcap:autonum="3. ">

When all selections/entries are made, click **Save**.

</div>
