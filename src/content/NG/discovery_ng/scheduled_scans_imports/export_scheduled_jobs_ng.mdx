# Exporting Scheduled Jobs

1. Select one or more scheduled jobs.

2. From the **Select Actions** dropdown, choose **Export**.

1. A confirmation dialog appears. Click **Continue**.

2. The system generates an **Excel (.xlsx) file** containing details of the selected scheduled jobs.

3. An **email** with a download link is sent to the logged-in user.

**Exported Data Includes:**

- Job Name
- Type (Scan / Import)
- Frequency
- Active Status
- Last Run Time
- Next Scheduled Run
