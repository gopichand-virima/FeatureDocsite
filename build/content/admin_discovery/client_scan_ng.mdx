# Scan Discovery Agents

Use this function to initiate an immediate scan of the selected agent.

In the main window, click **Admin > Discovery > Client**. The Client window displays.

Select a client, then click the **Discovery Agents** tab.

Select the agent(s) to scan.

From the *Select Actions* drop-down list, choose **Scan Discovery Agents**. The scan is initiated.
