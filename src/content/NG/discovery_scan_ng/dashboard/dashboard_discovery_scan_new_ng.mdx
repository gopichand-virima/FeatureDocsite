# Discovery Dashboard

Dashboards in **Virima** provide a centralized, user-friendly interface for quick access to key information across the platform. They offer a visual overview to help users monitor system activity, scan results, and other critical metrics at a glance.

**Tip:** Use the dashboard to enhance navigation and productivity by displaying relevant data widgets in a customizable layout.
