# Release Notes - Version NG

**Released: November 10, 2025**

## New Features

### Enhanced Service Mapping
Improved algorithm for automatic service dependency mapping with 40% faster discovery times.

### Cloud Integration Enhancements
Added support for Azure Government Cloud and expanded AWS GovCloud capabilities.

### Advanced Analytics Dashboard
New customizable analytics dashboard with real-time insights and predictive analytics.

## Improvements

- Performance optimization for large-scale deployments (10,000+ assets)
- Enhanced UI/UX with improved navigation and search functionality
- Faster report generation with new caching mechanism
- Improved API response times (up to 60% faster)
- Enhanced security with support for OAuth 2.0 and SAML 2.0

## Bug Fixes

- Fixed issue with discovery agent failing on certain Linux distributions
- Resolved memory leak in continuous monitoring service
- Fixed data synchronization issues with ServiceNow integration
- Corrected timezone handling in scheduled reports
- Fixed UI rendering issues in Safari browser

## Upgrade Notes

⚠️ **Important:** This version requires database schema updates. Please review the upgrade guide and backup your data before proceeding.
