# Incident Error Code

Use this function to define error codes and messages to display in case of incorrect configuration scenarios.

In the main window, click **Admin > Incident Management > Incident Error Code**. The Incident Error Code window displays.

## New Incident Error Code

From the *Select Actions* drop-down list, choose **New Incident Error Code**. The New Incident Error Code window displays.

Enter a *Name* and *Description*.

Enter the **Error Message** that should be displayed to users.

Specify the **Severity Level**:

- Critical
- High
- Medium
- Low

Click **Add**.

## Edit Incident Error Code

Navigate to the window containing the record to edit.

Click the line containing the record. The applicable window displays.

Make the necessary modifications.

Click **Save**.

## Delete Incident Error Code

Click the line that contains the item to delete.

From the *Select Actions* drop-down list, choose **Delete**.
