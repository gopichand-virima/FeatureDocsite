# Request Priority Calculator

Use this function to set the value for Urgency and Impact for a request.

|     |                                                       |
|-----|-------------------------------------------------------|
|     | Role access must be given to perform these functions. |

In the main window, click **Admin /> Request Management /> Request Priority Calculator**. The Request Priority Calculator window displays.

<span class="image placeholder" original-image-src="../Resources/Images/Admin%20-%20Request/request-priority-calculation-old.PNG" original-image-title=""></span>

In the main window, click the **gear** icon and select **Request Management /> Request Priority Calculator**.

<span class="image placeholder" original-image-src="../Resources/Images/Admin%20-%20Request/icon-req-prior-calc.PNG" original-image-title=""></span>

The Request Priority Calculator window displays.

<span class="image placeholder" original-image-src="../Resources/Images/Admin%20-%20Request/request-priority-calculator-main-new.PNG" original-image-title=""></span>

New Request Priority Calculator

<div class="NumList1" madcap:autonum="1. ">

Set the values according to the table below. Once the **Impact** and **Urgency** are defined as shown in the figure above, the **Priority** is set automatically when creating a request.

</div>

|     |                                                               |
|-----|---------------------------------------------------------------|
|     | Priority is always set to 1-Very High if the Reporter is VIP. |

<div class="NumList2" madcap:autonum="2. ">

When all selections are made, click **Save**.

</div>

### Priority Calculator Values

| Impact | Urgency | Priority    |
|--------|---------|-------------|
| High   | High    | 1-Very High |
| High   | Medium  | 2- High     |
| High   | Low     | 3- Medium   |
| Medium | High    | 2- High     |
| Medium | Medium  | 3- Medium   |
| Medium | Low     | 4-Low       |
| Low    | High    | 3- Medium   |
| Low    | Medium  | 4-Low       |
| Low    | Low     | 5-Very Low  |
