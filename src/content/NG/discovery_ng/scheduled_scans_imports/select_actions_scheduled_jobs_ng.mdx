# Select Actions

The **Select Actions** dropdown allows bulk operations on scheduled jobs:

- **New Scan Schedule** – Create a new scheduled scan.
- **New Scheduled AWS Import** – Schedule AWS resource imports.
- **New Scheduled Azure Import** – Schedule Azure resource imports.
- **New Scheduled Intune Import** – Schedule Intune device imports.
- **New Scheduled Meraki Import** – Schedule Meraki device imports.
- **Export** – Export selected scheduled jobs to Excel.
- **Delete** – Remove selected scheduled jobs.
