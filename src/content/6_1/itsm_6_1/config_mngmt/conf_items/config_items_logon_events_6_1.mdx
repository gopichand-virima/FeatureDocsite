# CI: Logon Events

Use this function to view the log on events for a specified period of time. Applies to both Windows Server and Windows Host.

<div class="NumList1" madcap:conditions="Default.OldUI" madcap:autonum="1. ">

From the navigation pane, select **ITSM /> Configuration Management /> Configuration Items**. The Configuration Items window displays.

</div>

<div class="NumList1" madcap:conditions="Default.NewUI" madcap:autonum="1. ">

From the navigation pane, select **Configuration Management /> Configuration Items**. The Configuration Items window displays.

</div>

<div class="NumList2" madcap:autonum="2. ">

Select a record in the list. A new window opens and the **Details** tab displays.

</div>

<div class="NumList2" madcap:autonum="3. ">

Click the **Logon Events** tab.

</div>

<div class="NumList2" madcap:autonum="4. ">

Click the *Client* drop-down list and select the applicable client.

</div>

<div class="NumList2" madcap:conditions="Default.OldUI" madcap:autonum="5. ">

Click the *Duration* drop-down list and select the duration of time for which you want to find the logon events.

</div>

<div class="NumList2" madcap:conditions="Default.NewUI" madcap:autonum="6. ">

Click the *Auto Refresh* drop-down list and select the duration of time for which you want to find the logon events.

</div>

<div class="Bullet2" madcap:conditions="Default.NewUI,Default.OldUI">

To specify a duration other than the options shown, select **Custom** in the drop-down list, and enter a value in the blank field.

</div>

<div class="NumList2" madcap:conditions="Default.OldUI,Default.NewUI" madcap:autonum="7. ">

Click **Refresh**. The list refreshes and displays all log on events for the specified duration.

</div>
