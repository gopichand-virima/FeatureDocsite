# New Asset License Entitlement

This feature enables to configure license entitlement details in the application.

<div class="NumList1" madcap:autonum="1. ">

Choose New Asset License Entitlements from the Select Actions drop-down. The New Asset License Entitlements window displayeds.

</div>

<div class="Ind50">

<span class="image placeholder" original-image-src="../Resources/Images/ITAM/asset-lic-entitlement-new.png" original-image-title="" style="width: 632px;height: 107px;"></span>

</div>

<div class="NumList2" madcap:autonum="2. ">

Enter a text/name to be displayed for the license entitled machine in the Display Name box.

</div>

<div class="NumList2" madcap:autonum="3. ">

Click Add to select the CI to assign the license. The Configuration Item dialog box displays.

</div>

<div class="Ind50">

<span class="image placeholder" original-image-src="../Resources/Images/ITAM/asset-lic-entitlement-new-search.png" original-image-title="" style="width: 473px;height: 187px;"></span>

</div>

<div class="NumList2" madcap:autonum="4. ">

Select the required blueprint from the given drop-down.

</div>

<div class="NumList2" madcap:autonum="5. ">

Enter the search criteria, and then click search icon.

</div>

The results are displayed based on the search criteria:

<div class="NumList1" madcap:autonum="1. ">

Select the required CI, and then click Save.

</div>

<div class="NumList2" madcap:autonum="2. ">

Click Cancel to discard the action, and in continuation, no CI will be updated in the application.

</div>
