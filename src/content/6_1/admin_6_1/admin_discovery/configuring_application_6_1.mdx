# Configuring the Application

1. Open the **Virima Discovery Application**.

2. Go to **Settings** \> **App Configuration**.

![Configuring the Application](../Resources/Images/Admin - Discovery/Configuring the Application.jpg)

3. Enter the following details:

 - **Username:** Your Web App login email

 - **Password:** Your Web App password

 - **Customer ID:** Found in your organization details in the Web App

 - **Application Name:** Any preferred name

 - **Your IP Address:** IP of the machine where the app is installed

![Configuring the Application](../Resources/Images/Admin - Discovery/Configuring the Application.png)

4. Click **Save** to apply the configuration.

5. The application will launch and open the Virima Discovery interface.

6. Click **Start** to begin discovery Virima Discovery Application.

![Configuring the Application 1](../Resources/Images/Admin - Discovery/Configuring the Application_1.png)

![Configuring the Application 1](../Resources/Images/Admin - Discovery/Configuring the Application_1.jpg)

The application will now begin running based on the configured settings.
