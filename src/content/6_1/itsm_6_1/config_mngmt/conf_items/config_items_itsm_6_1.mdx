# CI: ITSM

[This content is currently under development.]{style="color: #dc143c;"}

All the text below is conditioned out until the \"New\" buttons start working.

Use this function to create ITSM records.

 -- ----- ----------------------------------------------------------------------------------------------------------------------------------------
 1. From the navigation pane, select **ITSM \> Configuration Management \> Configuration Items**. The Configuration Items window displays.
 -- ----- ----------------------------------------------------------------------------------------------------------------------------------------

 -- ----- ----------------------------------------------------------------------------------------------------------------------------------------
 2. From the navigation pane, select **ITAM \> Configuration Management \> Configuration Items**. The Configuration Items window displays.
 -- ----- ----------------------------------------------------------------------------------------------------------------------------------------

 -- ----- ---------------------------------------------------------------------------------------------------
 3. Select a record in the list. A new windows displays with the **Details** tab selected by default.
 -- ----- ---------------------------------------------------------------------------------------------------

 -- ----- -----------------
 4. Click **ITSM**.
 -- ----- -----------------

::: {.Ind50
![itsm list 127x202](../../../Resources/Images/ITSM/itsm-list_127x202.png)

 -- ----- ---------------------------------------------------------------------------------
 5. Click an item in the list. The main window updates with the appropriate fields.
 -- ----- ---------------------------------------------------------------------------------

 -- ----- --------------------------------------------------------------------------------------------------------
 6. Click the \"New\" button for the selected item (for example, New Incident, New Request, and so forth).
 -- ----- --------------------------------------------------------------------------------------------------------
