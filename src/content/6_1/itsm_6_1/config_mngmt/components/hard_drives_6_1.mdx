# Hard Drives

Displays a list of all hard drives installed on this asset and provides the functionality to update Windows patches.

## <span id="WindowsPatches"></span>Apply Windows Update Patches

|     |                                                                                   |
|-----|-----------------------------------------------------------------------------------|
|     | Before you can apply a patch, you must have an agent associated with the CMDB CI. |

<div class="NumList1" madcap:conditions="Default.OldUI" madcap:autonum="1. ">

From the navigation pane, select **ITSM /> Configuration Management /> Configuration Items**. The Configuration Items window displays.

</div>

<div class="NumList2" madcap:conditions="Default.NewUI" style="font-weight: normal;" madcap:autonum="2. ">

From the navigation pane, select **Configuration Management /> Configuration Items.** The Configuration Items window displays.

</div>

<div class="NumList2" madcap:autonum="3. ">

Select the applicable *Windows Host* record in the list. A new window displays with the **Details** tab displayed.

</div>

<div class="NumList2" madcap:autonum="4. ">

Click the **Components** tab.

</div>

<div class="NumList2" madcap:autonum="5. ">

Click the **Hard Drives** tab. A new set of subtabs displays.

</div>

<div class="NumList2" madcap:autonum="6. ">

Click **Windows Updates**.

</div>

<div class="NumList2" madcap:autonum="7. ">

Click the **Available Patches** button. The Available Patches window displays. The Add Maintenance window displays.

</div>

<div class="Ind50" madcap:conditions="Default.OldUI">

<span class="image placeholder" original-image-src="../../../Resources/Images/ITSM/available-patches-dialog-box.PNG" original-image-title=""></span>

</div>

<div class="Ind50" madcap:conditions="Default.NewUI">

<span class="image placeholder" original-image-src="../../../Resources/Images/ITSM/add-maintenance-window-new.PNG" original-image-title=""></span>

</div>

<div class="NumList2" madcap:autonum="8. ">

Select a patch(es) to apply and click the **Apply** button. The Downloadable Patches dialog box displays containing a list of all available patches.

</div>

<div class="Ind50" madcap:conditions="Default.OldUI">

<span class="image placeholder" original-image-src="../../../Resources/Images/ITSM/downloadable-patches-dialog-box.PNG" original-image-title="" madcap:conditions="Default.OldUI"></span>

</div>

<div class="Ind50">

<span class="image placeholder" original-image-src="../../../Resources/Images/ITSM/downloadable-patches-dialog-box-new.PNG" original-image-title="" madcap:conditions="Default.NewUI"></span>

</div>

<div class="NumList2" madcap:autonum="9. ">

Click **Apply**.

</div>
