# Viewing AZURE Import Record

Use this function to view the imported AZURE resources (Regardless of manual import or scheduled import).

1. Select an import record to open its details.

2. Report window displays imported CI(s) in these categories:

 - All

 - Authorized

 - Unauthorized

 ![Import From AZURE 1 6](../../Resources/Images/Discovery/Import-From-Azure/Import From AZURE_1_6.png)

- **All** -- Every resource returned by the import.

- **Authorized** -- Assets that are already present in the CMDB.

 ![Import From AZURE 1 7](../../Resources/Images/Discovery/Import-From-Azure/Import From AZURE_1_7.png)

- **Unauthorized** -- New assets not yet listed in the CMDB.

 ![Import From AZURE 1 8 1074x247](../../Resources/Images/Discovery/Import-From-Azure/Import From AZURE_1_8_1074x247.png)
