# Software Usage Report

Use this function to view software usage metric information for software configured as *major* software.

<div class="NumList1" madcap:conditions="Default.OldUI" madcap:autonum="1. ">

From the navigation pane, select **ITSM /> Configuration Management /> Configuration Items**. The Configuration Items window displays. <span style="color: #dc143c;" madcap:conditions="Default.OldUI">MISSING v5 DETAILS AS THERE IS NO DATA IN THE UI.</span>

</div>

<div class="NumList2" madcap:conditions="Default.NewUI" madcap:autonum="2. ">

From the navigation page, select **Configuration Management /> Configuration Items**. The Configuration Items window displays.

</div>

<div class="Ind50" madcap:conditions="Default.NewUI" madcap:autonum="">

<span class="image placeholder" original-image-src="../../../Resources/Images/Admin%20-%20SACM/sw-usage-report-new.PNG" original-image-title="" style="width: 634px;height: 156px;"></span>

</div>

<div class="NumList2" madcap:conditions="Default.NewUI" madcap:autonum="3. ">

From the drop-down list, select the time period for which you want to see usage (such as last 24 hours or a custom range). The window updates and displays all records that match the date range criteria.

</div>

<div class="NumList2" madcap:autonum="4. " madcap:conditions="Default.NewUI">

Select the record for which you want to view usage details.

</div>

<div class="NumList2" madcap:conditions="Default.NewUI" madcap:autonum="5. ">

To export the details, click **Export**. A confirmation email is sent when the report is ready.

</div>

<div class="NumList2" madcap:conditions="Default.NewUI" madcap:autonum="6. ">

To view the details for a report, click the line item containing the report. The window updates and displays all the details for the selected report.

</div>

<div class="Ind50" madcap:conditions="Default.NewUI">

<span class="image placeholder" original-image-src="../../../Resources/Images/Admin%20-%20SACM/sw-usage-report-details-new.PNG" original-image-title="" style="width: 622px;height: 174px;"></span>

</div>

<div class="NumList2" madcap:autonum="7. " madcap:conditions="Default.NewUI">

To view the configuration information for the report, click the CI Id hyperlink.

</div>
