# Kubernetes Namespaces

 --- -------------------------------------------------------------------------------------------------------------
   This function is only available with certain asset types. Therefore, the relevant tab may not be displayed.
 --- -------------------------------------------------------------------------------------------------------------

 -- ----- ------------------------------------------------------
 1. While viewing an asset record, click **Components**.
 -- ----- ------------------------------------------------------

 -- ----- ----------------------------------
 2. Click **Kubernetes Namespaces**.
 -- ----- ----------------------------------

::: Ind50
![kubernetes namespaces new 482x159](../../../Resources/Images/Components/kubernetes-namespaces-new_482x159.png)

## Add a New Kubernetes Namespace

 -- ----- --------------------------------------------------------------
 1. Click **Add**. The Kubernetes Namespace dialog box displays.
 -- ----- --------------------------------------------------------------

::: Ind50
![kubernetes namespaces add new 381x119](../../../Resources/Images/Components/kubernetes-namespaces-add-new_381x119.png)

 -- ----- ------------------------------------
 2. Click **New Kubernetes Namespace**
 -- ----- ------------------------------------

 -- ----- -------------------------------------------------
 3. Enter the applicable information in each field.
 -- ----- -------------------------------------------------

 -- ----- --------------------------------------------
 4. When all entries are made, click **Save.**
 -- ----- --------------------------------------------

## Modify Existing Kubernetes Namespace

 -- ----- ------------------------------------------
 1. Click **Existing Kubernetes Namespace**.
 -- ----- ------------------------------------------

::: Ind50
![kubernetes namespaces existing new 378x258](../../../Resources/Images/Components/kubernetes-namespaces-existing-new_378x258.png)

 -- ----- ----------------------------------------------------------------------------------------------------------------------------------------------------------------------
 2. Enter the search criteria to locate the existing device.  *See also* [Advanced Search](../../../Common%20Topics/Advanced-Search.mdx){style="font-style: italic;"}*.*
 -- ----- ----------------------------------------------------------------------------------------------------------------------------------------------------------------------

 -- ----- -----------------------------------------------------
 3. Select the applicable device from the results list.
 -- ----- -----------------------------------------------------

 -- ----- -------------------------------------------------------
 4. When all selections/entries are made, click **Save**.
 -- ----- -------------------------------------------------------

## Delete Kubernetes Namespace

While viewing the list of existing devices, select the device to delete, and click the **Delete** button.
