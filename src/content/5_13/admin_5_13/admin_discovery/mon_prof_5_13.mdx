# Discovery Monitoring Profile

Use this function to configure monitoring profiles for scheduled monitoring based on a Cron Expression.

In the main window, click **Admin > Discovery > Discovery Monitoring Profile**. The Monitoring Profiles window displays.

From the *Select Actions* drop-down list, choose an action: **New Monitoring Profile**, [Delete](/common_topics/delete_remove_5_13.mdx), or [Export](/discovery/import_data_files/export_5_13.mdx).

In the *Name* field, type the name for this profile.

In the *Cron Expression* field, enter a valid cron expression (e.g., 0 0 12 \* \* ? for daily at noon).

Under *Select Scope*, choose the configuration items to monitor:
- Click **Select** to add items.
- Use the search function to find and select items.
- Click **Done**.

When all selections/entries are made, click **Add**.
