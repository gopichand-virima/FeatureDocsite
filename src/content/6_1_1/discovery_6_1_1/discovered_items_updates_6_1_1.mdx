# Discovered Items: Updates

Use this function to view updates to discovered items.


This information is view only.


In the navigation pane, select **Discovery Scan > Discovered Items**. The Discovered Items window displays.

Click **Updates**. The list of all updated actions displays.


The number shown after "Updates" indicates the number of changes made (described under the *Changes* column).
