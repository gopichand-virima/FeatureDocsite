# Tabs for Extended Information

- <span class="Strong" style="font-weight: bold;">Tasks</span>: Displays or allows creation of tasks linked to the import job.

- <span class="Strong" style="font-weight: bold;">Comments</span>: Provides space for administrators to add notes or review remarks from others.

- <span class="Strong" style="font-weight: bold;">Attachments</span>: Lets users upload or view related files, such as screenshots, reports, or documentation tied to the import.

Clicking on the "<span class="Strong" style="font-weight: bold;">New User Count</span>" will navigate you to the User Record page, where the Azure AD User Import Logs are automatically filtered and the relevant results are displayed.

<span class="image placeholder" original-image-src="../../Resources/Images/Discovery/Azure-AD-Users-Import-Logs/Azure%20AD%20User%20Import%20Logs_14.png" original-image-title=""></span>

<span class="Strong" style="font-weight: bold;">Key Information displayed</span>:

- <span class="Strong" style="font-weight: bold;">Id</span>: The unique identifier assigned to each user record.

- <span class="Strong" style="font-weight: bold;">Email</span>: The email address associated with the user.

- <span class="Strong" style="font-weight: bold;">User Name</span>: The Azure AD user’s principal name or login name.

- <span class="Strong" style="font-weight: bold;">Display Name</span>: The display name of the user as shown in Azure AD.

- <span class="Strong" style="font-weight: bold;">Phone</span>: The phone number associated with the user (empty in the current data).

- <span class="Strong" style="font-weight: bold;">Roles</span>: The assigned role(s) for the user (e.g., End User).
