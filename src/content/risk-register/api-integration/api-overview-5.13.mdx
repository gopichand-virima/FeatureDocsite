# Risk Register - API Integration

API documentation for risk management operations.

## API Endpoints

### Create Risk
```
POST /api/v1/risk/register
```

Create a new risk entry.

### Update Risk Assessment
```
PUT /api/v1/risk/{riskId}/assessment
```

Update risk probability and impact scores.

### Get Risk Heat Map Data
```
GET /api/v1/risk/heatmap
```

Retrieve data for risk heat map visualization.

### Create Mitigation Plan
```
POST /api/v1/risk/{riskId}/mitigation
```

Add a mitigation plan to a risk.

## Authentication

```
Authorization: Bearer YOUR_API_TOKEN
```

## Request Example - Create Risk

```json
{
  "title": "Cloud Service Outage Risk",
  "description": "Potential disruption due to cloud provider outage",
  "category": "operational",
  "subcategory": "infrastructure",
  "owner": "john.doe@company.com",
  "probability": 3,
  "impact": "high",
  "affectedAssets": ["ci-12345", "ci-67890"]
}
```

## Response Example

```json
{
  "status": "success",
  "data": {
    "riskId": "risk-12345",
    "riskScore": 12,
    "riskLevel": "high",
    "createdAt": "2025-11-10T10:30:00Z",
    "reviewDate": "2025-12-10"
  }
}
```

## Webhook Events

- Risk threshold exceeded
