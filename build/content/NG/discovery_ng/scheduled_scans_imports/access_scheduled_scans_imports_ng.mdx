# Accessing Scheduled Scans and Imports

In the navigation pane, select **Discovery Scan > Scheduled Scans/Imports**. The Scheduled Scans/Imports window displays.

This centralized page shows all automated discovery operations including:
- Scheduled network scans
- Scheduled cloud imports (AWS, Azure, Intune, Meraki)
- Scheduled IPAM syncs
- Recurring discovery jobs

Administrators can view, edit, enable/disable, or delete scheduled operations from this interface.
