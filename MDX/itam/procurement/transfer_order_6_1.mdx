# Transfer Order

Use this function to move materials from a source storage bin to a destination storage bin in a warehouse complex at a specified time. A transfer order consists of items that contain the quantity of the material to move, and specifies the source and destination storage bins.

In the navigation pane, select **ITAM > Procurement > Transfer Order**. The Transfer Order window displays

!

!


### New Transfer Order

- From the *Select Actions* drop-down list, choose **New Transfer Order**.  The Transfer Order window displays.

!

- Complete the fields, referring to the information below.
- **Transfer Order Number**. Enter a number for this transfer order.
- **From Stockroom/To Stockroom**. Click **Add**, then search for and select the applicable stockroom.
- **From Location/To Location**. Click **Add**, then search for and select the applicable location.
- **Requested Date**. Select the date/time the transfer order was made.
- **Deliver By Date**. Select the date/time the transferred items are to be delivered.
- **Stage**. Click the drop-down list and select the *Stage* of the transfer order, such as Requested, Shipment Preparation, etc.
- **Is Drop-Off**. Select to decline the transfer order.
- When all selections/entries are made, click **Add**.


### Edit Transfer Order

Not all functions include an option to edit record line items shown in the main window.


- Navigate to the window containing the record to edit.
- Click the line containing the record. The applicable window or dialog box displays.
- Make the necessary modifications.
- Click **Save** or **Add**, as applicable.


### Delete Transfer Order

Deleting is a permanent action and cannot be undone. Deleting may affect other functionality and information in the application such as data in configured reports, fields in windows, selectable options, etc. Therefore, be sure to understand the potential effects before making a deletion.


- Click the line that contains the item to delete.
- From the *Select Actions* drop-down list, choose **Delete**. If a confirmation message is displayed, type **Delete** and click **Delete**to confirm.
