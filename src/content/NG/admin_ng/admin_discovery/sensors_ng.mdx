# Sensors

Use this function to configure the discovery information in a readable format. Sensors are the scripts that act as per probes output.


This function requires special privileges. Contact your System Administrator for more information.


In the main window, clickAdmin>Discovery** > Sensors**. The Sensors window displays.

New Sensor

From the *Select Actions* drop-down list, choose **New Sensor**.

Enter a *Name* and *Description*.

Enter a *Script*. A few lines from a sample script are shown below.

When all entries are made, click **Add**.
