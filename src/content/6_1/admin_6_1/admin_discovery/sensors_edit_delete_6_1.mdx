# Edit or Delete Sensors

Use this function to edit or delete the sensor details and scripting.

## Edit Sensor

<div class="numlist1" madcap:conditions="Default.OldUI" madcap:autonum="1. ">

In the main window, select **Admin /> Discovery /> Sensors**. The Sensors window displays.

</div>

<div class="NumList2" madcap:conditions="Default.NewUI" madcap:autonum="2. ">

In the main window, click the **gear** icon and select **Discovery /> Sensors**. The Sensors window displays.

</div>

<div class="NumList2" madcap:autonum="3. ">

Select a record to edit. The *Details* information displays.

</div>

<div class="NumList2" madcap:autonum="4. ">

Make changes, as necessary, to any of the available fields.

</div>
