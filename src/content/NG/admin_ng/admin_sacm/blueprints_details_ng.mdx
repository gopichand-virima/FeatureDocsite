# Blueprint: Details

Use this function to add details for a blueprint.

 -- ----- -----------------------------------------------------------------------------------------------
 1. In the Blueprints window, from the *Select Actions* drop-down list, choose **New Blueprint**.
 -- ----- -----------------------------------------------------------------------------------------------

 -- ----- -------------------------------------------------------------------------------------
 2. Select an existing record. A new window displays with the **Details** tab selected.
 -- ----- -------------------------------------------------------------------------------------

 -- ----- ---------------------------------------------------------
 3. Complete the fields according to the information below.
 -- ----- ---------------------------------------------------------

 -- ----- ---------------------------------------------------------------------------------------
 4. Refer to [Searching](/common_topics/searching_6_1.mdx) for details on locating items.
 -- ----- ---------------------------------------------------------------------------------------

 -- ----- -------------------------------------------------------
 5. When all selections/entries are made, click **Save**.
 -- ----- -------------------------------------------------------
