# Other Asset Info

Access supplementary asset information and extended attributes for configuration items.

## Overview

The Other Asset Info function provides access to additional asset details that complement the primary asset record, including financial information, warranty details, maintenance history, and custom attributes.

## Accessing Other Asset Info

### View Additional Information
1. Open an asset or configuration item record
2. Click the **Other Asset Info** tab or section
3. View extended attributes and details
4. Navigate between different info categories
5. Edit information if you have permissions

### Information Categories
Other Asset Info is organized into sections:
- Financial Information
- Warranty and Support
- Maintenance History
- Compliance and Licensing
- Custom Attributes
- Related Documentation

## Financial Information

### Cost Details
- **Purchase Price**: Original acquisition cost
- **Current Value**: Depreciated or market value
- **Depreciation Rate**: Annual depreciation percentage
- **Salvage Value**: Estimated end-of-life value
- **Total Cost of Ownership (TCO)**: Lifetime cost including maintenance

### Procurement Information
- Purchase order number
- Vendor/supplier details
- Purchase date
- Payment terms
- Delivery information
- Invoice references

### Budget and Accounting
- Cost center allocation
- Budget code
- Accounting category
- Capitalization status
- Amortization schedule

## Warranty and Support

### Warranty Details
- **Warranty Start Date**: When coverage began
- **Warranty End Date**: Expiration date
- **Warranty Type**: Standard, extended, or premium
- **Coverage Level**: What's included
- **Warranty Provider**: Manufacturer or third party

### Support Contracts
- Support level (gold, silver, bronze)
- Support hours (24x7, business hours)
- Response time SLAs
- Contract renewal dates
- Support contact information

## Editing Other Asset Info

### Update Information
1. Click **Edit** in the Other Asset Info section
2. Modify fields as needed
3. Add new custom attributes if configured
4. Upload relevant documents
5. Click **Save** to commit changes

### Permissions
- View permissions: See other asset info
- Edit permissions: Modify information
- Financial permissions: Edit cost data
- Admin permissions: Configure custom fields
