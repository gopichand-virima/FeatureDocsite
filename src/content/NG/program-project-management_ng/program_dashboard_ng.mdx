# Program Dashboard

 -- ----- -------------------------------------------------------------------------------------------------------------------------------
 1. Click \*[program dashboard icon]{style="color: #ff4500;"}\* present in All Modules page. Program Dashboard page is displayed.
 -- ----- -------------------------------------------------------------------------------------------------------------------------------

::: Ind50
![program dashboard 5 476x200](../Resources/Images/Discovery/program-dashboard-5_476x200.png)

The page displays following information:

 ------------ -----------------------------------------------------
 Parameter Value
 Name It is the name given for the corresponding Program
 State It is the defined state for the corresponding State
 Status It is the present status defining program status
 Created On Â 
 ------------ -----------------------------------------------------

 -- ----- --------------------------------------------------------------------------------------
 2. Click Add Contents to update required Reports to be displayed in the Dashboard page.
 -- ----- --------------------------------------------------------------------------------------

::: Ind50
![program dashboard add contents 5 466x174](../Resources/Images/Discovery/program-dashboard-add-contents-5_466x174.png)

 -- ----- --------------------------------------------------------
 3. Select required type of the report from the drop-down.
 -- ----- --------------------------------------------------------

 -- ----- ------------------------------------------------------------------------------------------------------------
 4. Enter the search criteria in the given text box, and then click \*[Search icon]{style="color: #ff4500;"}\*
 -- ----- ------------------------------------------------------------------------------------------------------------

 -- ----- ---------------------------------------------------------
 5. The results are displayed as per given search criteria.
 -- ----- ---------------------------------------------------------

::: Ind50
![program dashboard search 5 477x278](../Resources/Images/Discovery/program-dashboard-search-5_477x278.png)

Report Alignment in Dashboard page:

 -- ----- ----------------------------------------------------------------------------------------------------------------------
 1. Select required report, and then click Add to the Top to align selected report at the top of the Dashboard web page.
 -- ----- ----------------------------------------------------------------------------------------------------------------------

 -- ----- -----------------------------------------------------------------------------------------------------------------------------
 2. Select required report, and then click Add to the left to align selected report to the left side of the Dashboard web page.
 -- ----- -----------------------------------------------------------------------------------------------------------------------------

 -- ----- -------------------------------------------------------------------------------------------------------------------------------
 3. Select required report, and then click Add to the right to align selected report to the right side of the Dashboard web page.
 -- ----- -------------------------------------------------------------------------------------------------------------------------------

 -- ----- ----------------------------------------------------------------------------------------------------------------------------
 4. Select required report, and then click Add to the bottom to align selected report at the bottom of the Dashboard web page.
 -- ----- ----------------------------------------------------------------------------------------------------------------------------
