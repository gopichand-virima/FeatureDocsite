# Virtual Machines

This function is only available with certain asset types. Therefore, the relevant tab may not be displayed.


- While viewing an asset record, click **Components**.
- Click **Virtual Machines**.

!

## Add a New Virtual Machine

- Click **Add**. The Virtual Machines dialog box displays.

!

- Click **New Virtual Machine.**
- Enter the applicable field information.
- When all selections/entries are made, click **Save**

## Modify Existing Virtual Machine

- Click **Existing Virtual Machine**.

!

- Enter the search criteria to locate the existing device. Â *See also* [Advanced Search](/common_topics/advanced_search_6_1.mdx)*.*
- Select the applicable device from the results list.
- When all selections are made, click **Save**.

## Delete Virtual Machine

While viewing the list of existing devices, select the device to delete, and click the **Delete** button.
