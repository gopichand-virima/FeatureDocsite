# Software Instances

 --- -------------------------------------------------------------------------------------------------------------
   This function is only available with certain asset types. Therefore, the relevant tab may not be displayed.
 --- -------------------------------------------------------------------------------------------------------------

 -- ----- -----------------------------------------------------------------
 1. While viewing an asset record, click the **Components** option.
 -- ----- -----------------------------------------------------------------

 -- ----- -------------------------------
 2. Click **Software Instances**.
 -- ----- -------------------------------

::: {.Ind50
![software instances new 673x161](../../../Resources/Images/Components/software-instances-new_673x161.png)

## Add New Software Instances

 -- ----- -----------------------------------------------------------------------------------
 1. To add a new instance, click **Add**. The Software Instances dialog box displays.
 -- ----- -----------------------------------------------------------------------------------

::: {.Ind50
![software instance new add 475x207](../../../Resources/Images/Components/software-instance-new-add_475x207.png)

 -- ----- ----------------------------------
 2. Click **New Software Instance**.
 -- ----- ----------------------------------

 -- ----- -------------------------------------------------
 3. Enter the applicable information in each field.
 -- ----- -------------------------------------------------

 -- ----- -----------------------------------------------
 4. When all selections are made, click **Save**.
 -- ----- -----------------------------------------------

## Modify Existing Software Instance

 -- ----- ----------------------------------------------------------------------------
 1. In the Software Instance dialog box, click **Existing Software Instance**.
 -- ----- ----------------------------------------------------------------------------

::: {.Ind50
![software instances new existing 472x321](../../../Resources/Images/Components/software-instances-new-existing_472x321.png)

 -- ----- ---------------------------------------------------------------------------------------------------------------------------------------------------------------------------------
 2. Enter the search criteria to locate the existing software instance.  *See also* [Advanced Search](../../../Common%20Topics/Advanced-Search.mdx){style="font-style: italic;"}*.*
 -- ----- ---------------------------------------------------------------------------------------------------------------------------------------------------------------------------------

 -- ----- -----------------------------------------------------------------------------------------
 3. Select the applicable item from the results list to move it to the selected items list.
 -- ----- -----------------------------------------------------------------------------------------

 -- ----- -----------------------------------------------
 4. When all selections are made, click **Save**.
 -- ----- -----------------------------------------------

## Delete Existing Software Instance

While viewing the list of existing software instances, select the item to delete, and click the **Delete** button.
