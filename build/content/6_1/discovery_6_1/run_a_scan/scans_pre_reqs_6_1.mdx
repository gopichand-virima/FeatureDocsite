# Prerequisites for Scan

Credentials are required to scan theTarget Hostand can be entered using the methods described below.


Added and imported credentials are encrypted and stored on the machine where the Discovery Application is installed.


### Method 1:Through Virima Discovery Admin

*Refer to* [Credentials](/admin_discovery/credentials_6_1.mdx)*.*

### Method 2:Through the Discovery Application using the Application Interface

Right-click the**Virima Discovery Application**.

Click**Run as Administrator**.

ClickStartand wait until it showsthat the Discovery Agent is Running.

From the Credentials drop-down list, select **Add Credential**.

The Add new credential window displays.

Complete the fields.

When all selections/entries are made, click **Save**.

### Method 3: Load the Credentials from an Excel File

To load the credentials through this method, credentials must be configured in an Excel file, as shown in the figure below.

Store the credential file in this location:C:\Program Files\Virima Discovery Application\ConfigasCredentials.xlsorCredentials.xlsx

Right-click the**Virima Discovery Application**.

Click**Run as administrator**.

ClickStartand wait for the indication that theDiscovery agent is Running.

In the Discovery Agent window, click the **Credentials** drop-down list and select **Import Credentials**.


The added/imported credentials are encrypted and stored on the machine where the Discovery Application is installed.
