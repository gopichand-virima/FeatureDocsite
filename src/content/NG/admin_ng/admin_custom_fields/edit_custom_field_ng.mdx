# Edit Custom Field

Use this function to modify existing custom field configurations.

In the main window, click **Admin > Custom Fields**, select a field, and click **Edit**.

## Edit Field Properties

**Editable Properties**:

- Field Label
- Description
- Help Text
- Required status
- Default value
- Validation rules
- Permissions
- Display settings

**Non-Editable**:

- Field Name (technical name)
- Field Type
- Applied To (table)

## Caution

Editing custom fields affects existing data. Test changes in non-production environments first.
