# Request State

Use this function to add a new state that defines status of the request being handled by an Assignee.

In the main window, click **Admin /> Request Management /> Request State**. The Request State window displays.

<span class="image placeholder" original-image-src="../Resources/Images/Admin%20-%20Request/request-state-main-old.PNG" original-image-title=""></span>

In the main window, click the **gear** icon and select **Request Management /> Request State**.

<span class="image placeholder" original-image-src="../Resources/Images/Admin%20-%20Request/icon-req-state.PNG" original-image-title=""></span>

The Request State window displays.

<span class="image placeholder" original-image-src="../Resources/Images/Admin%20-%20Request/request-state-main-new.PNG" original-image-title=""></span>

</div>

</div>

</div>

<div class="NumList2" madcap:conditions="Default.NewUI" madcap:autonum="11. ">

Search for and select the applicable role(s) with permissions to approve the defined state. Then click **Add**.

</div>

<div class="NumList2" madcap:autonum="12. ">

When all selections/entries are made, click **Add**.

</div>
