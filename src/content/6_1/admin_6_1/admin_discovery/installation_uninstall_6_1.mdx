# Uninstall the Discovery Application

The best and safest way to uninstall the existing application is by using a Windows uninstaller.

 -- ----- --------------------------------------------------------------------------------------------------------------------------------------------------------------
 1. Go to Windows [Control Panel]{.Strong_4}[ ]{.span_5}and select [**Programs and Features**.]{.Strong_4}
 -- ----- --------------------------------------------------------------------------------------------------------------------------------------------------------------

 -- ----- -----------------------------------------------------------------------------------------
 2. [Select ]{.span_5}[Virima Discovery Application.]{.Strong_4}
 -- ----- -----------------------------------------------------------------------------------------

 -- ----- ----------------------------------------------
 3. [Click ]{.span_5}[**Uninstall**.]{.Strong_4}
 -- ----- ----------------------------------------------

 -- ----- -------------------------------------------------------------------------------------------------------------------------------------------------------------------
 4. [Navigate to the installed location and delete the directory ]{.span_5}[Virima Discovery Application]{.Strong_3}[ to remove all the temp and log files.]{.span_5}
 -- ----- -------------------------------------------------------------------------------------------------------------------------------------------------------------------
