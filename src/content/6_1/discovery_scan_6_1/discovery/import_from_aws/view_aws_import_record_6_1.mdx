# View AWS Import Record

Use this function to view the imported AWS resources (Regardless of manual import or scheduled import).

1.  Select an import record to open its details.

2.  The report window displays imported CI(s) in these categories:

    - All

    - Authorized

    - Unauthorized

    - Logs

    <span class="image placeholder" original-image-src="../../Resources/Images/Discovery/Import-From-AWS/Import%20From%20AWS_1_5.png" original-image-title=""></span>

<span class="Strong">All</span> – Every resource returned by the import.

<span class="Strong">Authorized</span> – Assets that are already present in the CMDB.

<span class="image placeholder" original-image-src="../../Resources/Images/Discovery/Import-From-AWS/Import%20From%20AWS_1_6.png" original-image-title=""></span>

<span class="Strong">Unauthorized</span> – New assets not yet listed in the CMDB.

<span class="image placeholder" original-image-src="../../Resources/Images/Discovery/Import-From-AWS/Import%20From%20AWS_1_7.png" original-image-title=""></span>

<span class="Strong">Logs</span> – The import run’s messages (timings, successes, errors).

<span class="Strong">Common controls (top-right of the grid)</span>

- <span class="Strong">Move to CMDB</span> – Promote the selected records to CMDB (Refer Move to CMBD).

- <span class="Strong">Auto Refresh</span><span class="inline-comment-marker"> – Refresh the grid on set periodic interval. Use this when import is still in progress so that the imported items show up without manual refresh.</span>
