# Kubernetes Pods

 --- -------------------------------------------------------------------------------------------------------------
   This function is only available with certain asset types. Therefore, the relevant tab may not be displayed.
 --- -------------------------------------------------------------------------------------------------------------

 -- ----- ------------------------------------------------------
 1. While viewing an asset record, click **Components**.
 -- ----- ------------------------------------------------------

 -- ----- ----------------------------
 2. Click **Kubernetes Pods**.
 -- ----- ----------------------------

::: Ind50
![kubernetes pods new 463x122](../../../Resources/Images/Components/kubernetes-pods-new_463x122.png)

## Add a New Kubernetes Pod

 -- ----- ---------------------------------------------------------
 1. Click **Add**. The Kubernetes Pods dialog box displays.
 -- ----- ---------------------------------------------------------

::: Ind50
![kubernetes pods add new 380x121](../../../Resources/Images/Components/kubernetes-pods-add-new_380x121.png)

 -- ----- ------------------------------
 2. Click **New Kubernetes Pod**
 -- ----- ------------------------------

 -- ----- -------------------------------------------------
 3. Enter the applicable information in each field.
 -- ----- -------------------------------------------------

 -- ----- --------------------------------------------
 4. When all entries are made, click **Save.**
 -- ----- --------------------------------------------

## Modify Existing Kubernetes Pod

 -- ----- ------------------------------------
 1. Click **Existing Kubernetes Pod**.
 -- ----- ------------------------------------

::: Ind50
![kubernetes pods existing new 379x259](../../../Resources/Images/Components/kubernetes-pods-existing-new_379x259.png)

 -- ----- ----------------------------------------------------------------------------------------------------------------------------------------------------------------------
 2. Enter the search criteria to locate the existing device.  *See also* [Advanced Search](../../../Common%20Topics/Advanced-Search.mdx){style="font-style: italic;"}*.*
 -- ----- ----------------------------------------------------------------------------------------------------------------------------------------------------------------------

 -- ----- -----------------------------------------------------
 3. Select the applicable device from the results list.
 -- ----- -----------------------------------------------------

 -- ----- -------------------------------------------------------
 4. When all selections/entries are made, click **Save**.
 -- ----- -------------------------------------------------------

## Delete Kubernetes Pod

While viewing the list of existing devices, select the device to delete, and click the **Delete** button.
