# Edit User Entitlement

This feature enables editing the configured values of the existing contract, updating, and then saving the same in the application.

Click to open required User License Management. Edit User License Entitlement displays.

The Edit User License Entitlement displays the following information:

- Tasks
- Comments
- Attachment
- History
