# Edit Blueprints

Use this function to edit the details for a blueprint.

- In the Blueprints window, select a record to edit. The applicable record displays.
- Modify any of the following information:
- [Details](/admin_sacm/blueprints_details_6_1.mdx)
- [SLA](/admin_sacm/blueprints_sla_6_1.mdx)
- [Tasks](/admin_sacm/blueprints_tasks_6_1.mdx)
- [Comments](/admin_sacm/blueprints_comments_6_1.mdx)
- [Attachments](/admin_sacm/blueprints_attachments_6_1.mdx)
- When all selections/entries are made, click **Save**. 
- In the Blueprints window, select a record to edit. The applicable record displays.
- Modify the fields according to the information provided under [Blueprint Details > Field Descriptions](/admin_sacm/blueprints_details_6_1.mdx#Blueprint-Fields).
- Modify any of the following information:
- [Details](/admin_sacm/blueprints_details_6_1.mdx)
- [SLA](/admin_sacm/blueprints_sla_6_1.mdx)
- [Tasks](/admin_sacm/blueprints_tasks_6_1.mdx)
- [Comments](/admin_sacm/blueprints_comments_6_1.mdx)
- [Attachments](/admin_sacm/blueprints_attachments_6_1.mdx)
- When all selections/entries are made, click **Add**.
