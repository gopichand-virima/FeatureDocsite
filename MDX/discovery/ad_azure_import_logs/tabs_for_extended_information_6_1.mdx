# Tabs for Extended Information

- **Tasks**: Displays or allows creation of tasks linked to the import job.
- **Comments**: Provides space for administrators to add notes or review remarks from others.
- **Attachments**: Lets users upload or view related files, such as screenshots, reports, or documentation tied to the import.

Clicking on the "**New User Count**" will navigate you to the User Record page, where the AD User Import Logs are automatically filtered and the relevant results are displayed.
