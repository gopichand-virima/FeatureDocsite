# Viewing IPAM Networks

Use this function to view IPAM network information.

In the main window, select **Discovery Scan > IPAM Networks**. The IPAM Networks window displays.

Click any record to view its details. A new window displays showing the information configured for the currently selected record. For example, **IP Address**, **Host Name**, **Location**, **Network**, **Network Address**, and **Network Mask** are displayed.
