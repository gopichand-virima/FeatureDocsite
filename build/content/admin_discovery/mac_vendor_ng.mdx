# MAC Vendor

Use this function to view and manage MAC address vendor mappings.

In the main window, click **Admin > Discovery > MAC Vendor**. The MAC Vendor window displays.

The MAC Vendor database maps MAC address prefixes to their respective manufacturers. This information is automatically used during discovery to identify device manufacturers.

## Update MAC Vendor Database

From the *Select Actions* drop-down list, choose **Update MAC Vendor Database**. The system will download and update the latest MAC vendor mappings.

## Search MAC Vendors

Use the search functionality to find specific MAC vendor information:

Enter a MAC address prefix (e.g., "00:1A:2B") or vendor name in the search field.

The results will display matching MAC prefixes and their associated vendors.
