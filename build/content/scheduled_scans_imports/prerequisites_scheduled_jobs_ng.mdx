# Prerequisites

To successfully create and run scheduled scans or imports:

- **Discovery Client Setup:** At least one active client must be configured (Admin > Discovery > Clients).
- **Credentials Configured:** Valid credentials for scans or imports must exist (Admin > Discovery > Credentials).
- **User Permissions:** The logged-in user must have permissions to schedule scans/imports.
- **Network Access:** The discovery client must be able to reach the target IP ranges or cloud endpoints.
