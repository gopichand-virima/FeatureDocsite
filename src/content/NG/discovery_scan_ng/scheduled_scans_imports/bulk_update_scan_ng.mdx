# Bulk Update Scan

1. In **Scans** (radio button selected), Select one or more job

2. Click on "Select Actions" and select "Update Scan"

![Scheduled Scans and Imports 1 10](../../Resources/Images/Discovery/Scheduled-Scans-and-Imports/Scheduled Scans and Imports_1_10.png)

3. Limited set of attributes, common to all selected scans, that is not scan specific is displayed

 1. **Client:** pick the target client you want the selected scans to run on

 2. **Scan Frequency:** choose the frequency preset (e.g., *Current Time Settings*) you want to apply.

 3. **Recurring:** select **Yes** or **No**.

 4. **Time Zone:** choose the correct timezone.

 5. **Active:** select **Yes** (to run) or **No** (to pause).

 6. **Schedule wheels:** tick the box above each wheel you want to control (**Second / Minute / Hour / Day / Month / Weekday**), then set the values with the pickers.

4. Click **Update**.

5. Updated details shall reflect in all the selected checkbox(es)
