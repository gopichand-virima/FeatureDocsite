# Viewing Meraki Import Record

Use this function to view the imported Meraki resources (Regardless of manual import or scheduled import).

1.  Select an import record to open its details.

2.  Report window displays imported CI(s) in these categories:

    - All

    - Authorized

    - Unauthorized

    - Logs

    <span class="image placeholder" original-image-src="../../Resources/Images/Discovery/Import-From-Meraki/Import%20Meraki%20Window_7.png" original-image-title=""></span>

- <span class="Strong">All</span> – Every resource returned by the import.

- <span class="Strong">Authorized</span> – Assets that are already present in the CMDB.

  <span class="image placeholder" original-image-src="../../Resources/Images/Discovery/Import-From-Meraki/Import%20Meraki%20Window_8.png" original-image-title=""></span>

<!-- -->

- <span class="Strong">Unauthorized</span> – New assets not yet listed in the CMDB.

  <span class="image placeholder" original-image-src="../../Resources/Images/Discovery/Import-From-Meraki/Import%20Meraki%20Window_9.png" original-image-title=""></span>

<!-- -->

- <span class="Strong">Logs</span>: The import run's messages (timings, successes, errors).
