# Edit Service Level Agreement

Use this function to edit the details for a service level agreement.

In the main window, select **Admin > Other > Service Level Agreement.** The Service Level Agreements window displays.

Select a record to edit.

Modify any of the following information:

- [Details](/admin_other/sla_details_6_1.mdx)
- [Tasks](/admin_other/sla_tasks_6_1.mdx)
- [Comments](/admin_other/sla_comments_6_1.mdx)
- [Attachments](/admin_other/sla_attachments_6_1.mdx)

When all selections/entries are made, click **Save**.
