# Exporting Records Without Selecting Any Record

1.  <span class="Strong">Open the Export Wizard</span>

    - Click on <span class="Strong">Select Actions</span> and select <span class="Strong">Export</span> from the dropdown menu.

    - The system will display the Export wizard.

    <span class="image placeholder" original-image-src="../../Resources/Images/Discovery/Discovered-Items/Discovered%20Items_1_7.png" original-image-title=""></span>

<!-- -->

2.  <span class="Strong">Select Blueprints</span>

    - In the Blueprints step, use the search bar to find and tick the blueprint(s) you want to export (e.g., AWS, Windows, etc.).

    - Click <span class="Strong">Next</span> to proceed.

3.  <span class="Strong">Select Components</span>

    - If the selected blueprints have component types, choose the components you need.

    - Click <span class="Strong">Next</span> to continue.

    <span class="image placeholder" original-image-src="../../Resources/Images/Discovery/Discovered-Items/Discovered%20Items_1_8.png" original-image-title=""></span>

<!-- -->

4.  <span class="Strong">Select Properties</span>

    - Start with <span class="Strong">Main Properties</span>.

    - If you see the message: “No main properties are found for selected blueprints,” switch to <span class="Strong">Other Properties</span>.

    - In <span class="Strong">Other Properties</span>, search for and check the fields you want to include (move them to <span class="Strong">Selected Items</span> if required by your UI).

    - Click <span class="Strong">Next</span> or proceed as per the UI.

    <span class="image placeholder" original-image-src="../../Resources/Images/Discovery/Discovered-Items/Discovered%20Items_1_9.png" original-image-title=""></span>

<!-- -->

5.  <span class="Strong">Export the Data</span>

    - Click <span class="Strong">Export</span>.

    - A dialog will appear with the message:  
      <span class="Emphasis">“The complete CMDB information will be generated to an Excel spreadsheet. You will receive an email when it is ready to be downloaded.”</span>

    - Click <span class="Strong">Continue</span> to confirm.

6.  <span class="Strong">Download the Exported File</span>

    - Wait for the email or notification.

    - Follow the link provided in the email/notification to download the Excel file.
