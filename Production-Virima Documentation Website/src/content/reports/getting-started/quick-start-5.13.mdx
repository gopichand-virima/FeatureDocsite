# Reports - Quick Start Guide

Start generating insights with the Reports module.

## Prerequisites

- Reports module access
- Understanding of data sources
- Familiarity with reporting requirements

## Quick Start Steps

### Step 1: Browse Report Library
1. Navigate to Reports > Report Library
2. Browse reports by category:
   - Asset Reports
   - Incident Reports
   - Change Reports
   - Compliance Reports
   - Custom Reports
3. Preview report samples

### Step 2: Run a Pre-built Report
1. Select a report from the library
2. Configure report parameters:
   - Date range
   - Filters (module, category, status)
   - Grouping options
3. Click "Run Report"
4. Review results

### Step 3: Create a Custom Report
1. Click "Create New Report"
2. Select data source(s)
3. Choose fields to include
4. Add filters and conditions
5. Configure visualization (table, chart, graph)
6. Save report with a descriptive name

### Step 4: Schedule Report Delivery
1. Open saved report
2. Click "Schedule"
3. Set delivery frequency (daily, weekly, monthly)
4. Add email recipients
5. Choose export format
6. Activate schedule
