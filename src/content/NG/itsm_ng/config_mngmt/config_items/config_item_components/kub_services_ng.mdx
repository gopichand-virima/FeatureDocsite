# Kubernetes Services

 --- -------------------------------------------------------------------------------------------------------------
   This function is only available with certain asset types. Therefore, the relevant tab may not be displayed.
 --- -------------------------------------------------------------------------------------------------------------

 -- ----- ------------------------------------------------------
 1. While viewing an asset record, click **Components**.
 -- ----- ------------------------------------------------------

 -- ----- --------------------------------
 2. Click **Kubernetes Services**.
 -- ----- --------------------------------

::: Ind50
![kubernetes services new 467x140](../../../../Resources/Images/Components/kubernetes-services-new_467x140.png)

![arrow border](../../../../Resources/Images/arrow-border.PNG)

[[[Closed]{.image .placeholder .MCDropDown_Image_Icon original-image-src="../../../../../Skins/Default/Stylesheets/Images/transparent.gif" original-image-title="" height="12" width="16" Kubernetes Service](javascript:void(0)){.MCDropDownHotSpot .dropDownHotspot .MCDropDownHotSpot_ .MCHotSpotImage aria-expanded="false"}]{.MCDropDownHead .dropDownHead}

 -- ----- ----------------------------------------------------------------
 1. Click **Add**. The New Kubernetes Service dialog box displays.
 -- ----- ----------------------------------------------------------------

::: Ind50
![kubernetes services add new 384x122](../../../../Resources/Images/Components/kubernetes-services-add-new_384x122.png)

 -- ----- ----------------------------------
 2. Click **New Kubernetes Service**
 -- ----- ----------------------------------

 -- ----- -------------------------------------------------
 3. Enter the applicable information in each field.
 -- ----- -------------------------------------------------

 -- ----- --------------------------------------------
 4. When all entries are made, click **Save.**
 -- ----- --------------------------------------------

[[[Closed]{.image .placeholder .MCDropDown_Image_Icon original-image-src="../../../../../Skins/Default/Stylesheets/Images/transparent.gif" original-image-title="" height="12" width="16" Kubernetes Service](javascript:void(0)){.MCDropDownHotSpot .dropDownHotspot .MCDropDownHotSpot_ .MCHotSpotImage aria-expanded="false"}]{.MCDropDownHead .dropDownHead}

 -- ----- ----------------------------------------
 1. Click **Existing Kubernetes Service**.
 -- ----- ----------------------------------------

::: Ind50
![kubernetes services existing new 372x249](../../../../Resources/Images/Components/kubernetes-services-existing-new_372x249.png)

 -- ----- ------------------------------------------------------------
 2. Enter the search criteria to locate the existing device*.*
 -- ----- ------------------------------------------------------------

 -- ----- -----------------------------------------------------
 3. Select the applicable device from the results list.
 -- ----- -----------------------------------------------------

 -- ----- -------------------------------------------------------
 4. When all selections/entries are made, click **Save**.
 -- ----- -------------------------------------------------------

[[[Closed]{.image .placeholder .MCDropDown_Image_Icon original-image-src="../../../../../Skins/Default/Stylesheets/Images/transparent.gif" original-image-title="" height="12" width="16" Kubernetes Service](javascript:void(0)){.MCDropDownHotSpot .dropDownHotspot .MCDropDownHotSpot_ .MCHotSpotImage aria-expanded="false"}]{.MCDropDownHead .dropDownHead}

 --- --------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------
   Deleting is a permanent action and cannot be undone. Deleting may affect other functionality and information in the application such as data in configured reports, fields in windows, selectable options, etc. Therefore, be sure to understand the potential effects before making a deletion.
 --- --------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------

While viewing the list of existing records, select a record(s) to delete, and click the **Delete** button.
