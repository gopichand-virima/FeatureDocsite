# SLA: Details

<div class="NumList2" madcap:autonum="1. ">

In the Service Level Agreement window, from the *Select Actions* drop-down list, choose **New**.

</div>

<div class="NumList2" madcap:autonum="2. ">

Select an existing record. A new window displays with the **Details** tab displayed, by default.

</div>

<div class="Ind50">

<span class="image placeholder" original-image-src="../Resources/Images/Admin%20-%20Other/sla-details.PNG" original-image-title="" style="width: 620px;height: 235px;"></span>

</div>

|     |                                                                        |
|-----|------------------------------------------------------------------------|
|     | By default, the **Id** has already been created and cannot be changed. |

<div class="NumList2" madcap:autonum="3. ">

In the *Name* field, enter a name to identify this SLA.

</div>

<div class="NumList2" madcap:autonum="4. ">

In the *Description* field, enter a description for this SLA.

</div>

<div class="NumList2" madcap:autonum="5. ">

In the *Breach Penalty Amount (/$)* field, type the dollar amount that will be accrued should the SLA agreement is breached.

</div>

<div class="NumList2" madcap:autonum="6. ">

In the *Breach Penalty* Per field, type the duration at which point the penalty amount is accrued.

</div>

<div class="NumList2" madcap:autonum="7. ">

In the *Response Time* field, enter the days/hours/minutes that must be maintained before the penalty applies.

</div>

<div class="NumList2" madcap:autonum="8. ">

In the *Resolution Time* field, enter the days/hours/minutes that must be maintained before the penalty applies.

</div>

<div class="NumList2" madcap:autonum="9. ">

When all selections/entries are made, click **Save**.

</div>
