# Surveys

NEED v6 IMAGES

Use this function to enable a Reporter to send a survey when the Incident/Request/Change/Problem is being closed. The surveys are usually conducted by a pre-defined questionnaire. The data in the Survey tab will be displayed only if the Incident/Request/Change/Problem is closed.

In the main window, click **Admin \> Others \> Surveys**. The Surveys window displays.

![survey main old](../Resources/Images/Admin - Other/survey-main-old.PNG)

In the main window, click the **gear** icon and select **Others \> Surveys**.

[ICON]{style="color: #dc143c; background-color: #ffff00;"}

The Surveys window displays.

[IMAGE]{style="background-color: #ffff00; color: #dc143c;"}

![arrow border](../Resources/Images/arrow-border.PNG)

 -- ----- --------------------------------------------------------------------------------------------------
 1. From the *Select Actions* drop-down list, choose **New Survey**. The New Survey window displays.
 -- ----- --------------------------------------------------------------------------------------------------

 -- ----- ---------------------------------------------------
 2. Enter a *Name* and *Introduction* for the survey.
 -- ----- ---------------------------------------------------

 -- ----- ------------------------
 3. Enter any *End Notes*.
 -- ----- ------------------------

 -- ----- --------------------------------------------------------------------------------------
 4. In the *Threshold* fields, specify the frequency at which the survey should be sent.
 -- ----- --------------------------------------------------------------------------------------

 -- ----- ---------------------------------------------------
 5. To add survey questions, click **Add Questions**.
 -- ----- ---------------------------------------------------

 --- -----------------------------------------------------------------------------------------
   Not all functions include an option to edit record line items shown in the main window.
 --- -----------------------------------------------------------------------------------------

 -- ----- -------------------------------------------------------
 1. Navigate to the window containing the record to edit.
 -- ----- -------------------------------------------------------

 -- ----- -------------------------------------------------------------------------------------
 2. Click the line containing the record. The applicable window or dialog box displays.
 -- ----- -------------------------------------------------------------------------------------

 -- ----- -----------------------------------
 3. Make the necessary modifications.
 -- ----- -----------------------------------

 -- ----- -------------------------------------------
 4. Click **Save** or **Add**, as applicable.
 -- ----- -------------------------------------------

To view a list of all functions where this survey is specified:

 -- ----- -----------------------------
 1. Open an existing line item.
 -- ----- -----------------------------

 -- ----- ----------------------------
 2. Click the **Used By** tab.
 -- ----- ----------------------------
