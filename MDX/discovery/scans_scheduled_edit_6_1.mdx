# Edit or Delete Scheduled Scan

Use this function to manage a scheduled scan.


The **Related Scans** information is view only. It displays all previously triggered scans.


## Edit Scheduled Scan

- In the Scheduled Scans window, select a record to edit.
- Modify any of the following information:
- [Details](/discovery/scans_scheduled_new_6_1.mdx)
- [Tasks](/discovery/ad_user_import_tasks_6_1.mdx)
- [Comments](/discovery/ad_user_import_comments_6_1.mdx)
- [Attachments](/discovery/ad_user_import_attachments_6_1.mdx)
- When all selections/entries are made, click **Save**.Â 

## Delete Scheduled Scan

Deleting is a permanent action and cannot be undone. Deleting may affect other functionality and information in the application such as data in configured reports, fields in windows, selectable options, etc. Therefore, be sure to understand the potential effects before making a deletion.


- Click the line that contains the item to delete.
- From the *Select Actions* drop-down list, choose **Delete**. If a confirmation message is displayed, type **Delete** and click **Delete**to confirm.
