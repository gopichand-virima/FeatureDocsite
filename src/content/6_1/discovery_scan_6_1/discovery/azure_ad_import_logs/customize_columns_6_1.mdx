# Customizing Columns

1.  In the <span class="Strong" style="font-weight: bold;">Azure AD User Import Logs</span> page, click the <span class="Strong" style="font-weight: bold;">Personalize Column icon</span> in the top-right corner.

<span class="image placeholder" original-image-src="../../Resources/Images/Discovery/Azure-AD-Users-Import-Logs/Azure%20AD%20User%20Import%20Logs_15.png" original-image-title=""></span>

2.  The <span class="Strong" style="font-weight: bold;">Personalize List Columns</span> window will appear.

    - The <span class="Strong" style="font-weight: bold;">left panel</span> shows available fields.

    - The <span class="Strong" style="font-weight: bold;">right panel</span> shows currently displayed fields.

3.  Check or uncheck fields as needed.

4.  Click <span class="Strong" style="font-weight: bold;">Save</span> to apply changes.

    - <span class="Strong" style="font-weight: bold;">Default View</span> resets columns to system default.

    - <span class="Strong" style="font-weight: bold;">Clear All</span> removes all selections for a fresh start.

<span class="image placeholder" original-image-src="../../Resources/Images/Discovery/Azure-AD-Users-Import-Logs/Azure%20AD%20User%20Import%20Logs_16.png" original-image-title=""></span>
