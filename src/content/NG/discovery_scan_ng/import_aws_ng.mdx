# [Import from AWS]{style="background-color: #ffffff;"}

Use this function to identify information existing in Amazon Web Services (AWS). The discovery of the Amazon Web Services cloud is based on account information rather than an IP range. Discovery Servers are not used in this type of discovery.

 -- ----- -------------------------------------------------------------------------------------------------------------------------------------------------------------------------
 1. In the navigation pane, select **Discovery Scan \> Import from AWS**. The [Import AWS window displays and contains a list of all currently imported records. ]{.span_5}
