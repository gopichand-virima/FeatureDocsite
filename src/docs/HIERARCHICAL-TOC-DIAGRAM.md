## Summary

The hierarchical system organizes navigation in three clear levels:
1. **Version level**: Lists modules
2. **Module level**: Lists sections
3. **Section level**: Lists pages

Each level has its own small index.mdx file that's easy to find and update. The system loads only what's needed (lazy loading) and caches everything for performance.
