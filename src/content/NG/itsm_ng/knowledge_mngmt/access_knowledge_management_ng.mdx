# Access Knowledge Management

Knowledge Management provides a centralized repository for organizational knowledge, best practices, and solutions.

## Accessing from Main Menu

1. Log in to Virima
2. Navigate to **ITSM** > **Knowledge Management**
3. Browse or search knowledge articles

## Quick Access

### Search Bar
Global knowledge search:
1. Click search icon
2. Enter keywords
3. View matching articles
4. Select article to read

### From Incidents
Access knowledge while resolving incidents:
1. Open incident
2. Click **Search Knowledge**
3. System suggests relevant articles
4. Apply solution from knowledge

## Navigation Structure

- **Knowledge Base**: All articles
- **Categories**: Browse by topic
- **My Articles**: Articles you created
- **Popular Articles**: Most viewed
- **Recent Articles**: Latest additions
- **Favorites**: Bookmarked articles

## User Permissions

### Knowledge Consumer
- Search and view articles
- Rate articles
- Provide feedback
- Bookmark favorites

### Knowledge Contributor
- Create articles
- Edit own articles
- Submit for approval
- View analytics

### Knowledge Manager
- Approve articles
- Manage categories
- Archive outdated content
- Access full analytics
