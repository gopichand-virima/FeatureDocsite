# Problem Category

Use this function to configure a category for the change being associated, such as CI, Application Software, Hardware or any aspect of the business that a change affects.

In the main window, click **Admin /> Problem Management /> Problem Category**. The Problem Category window displays.

<span class="image placeholder" original-image-src="../Resources/Images/Admin%20-%20Problem/problem-category-main-old.PNG" original-image-title=""></span>

In the main window, click the **gear** icon and select **Problem Management /> Problem Category**.

<span class="image placeholder" original-image-src="../Resources/Images/Admin%20-%20Problem/icon-problem-category.PNG" original-image-title=""></span>

The Problem Category window displays.

<span class="image placeholder" original-image-src="../Resources/Images/Admin%20-%20Problem/problem-category-main-new.PNG" original-image-title=""></span>

</div>
