# Designations

Use this function to categorize users.

In the main window, click **Admin > Organizational Details > Designations**. The Designations Window displays.

### New Designation

To create a new designation:

- From the _Select Actions_ drop-down list, choose **New Designation**. The New Designation window displays.

- Enter a _Name_ and _Description_. The _Name_ field is required.
- Click **Add**.

### Edit Designation

To edit an existing designation:

- Click a record in the list. The Details tab displays.
- Modify the _Name_ and/or _Description_.
- View, add, or modify any of the following:
- [Tasks](/itsm/release_mngmt/tasks_5_13.mdx)
- [Comments](/discovery/comments_5_13.mdx)
- [Attachments](/discovery/attachments_5_13.mdx)
- When all selections/entries are made, click **Save**.

### Delete Designation

To delete an existing designation:

- Highlight the designation to delete. The Details tab displays.
- FFrom the _Select Actions_ drop-down list, choose **Delete**. If a confirmation message is displayed, type **Delete** and then click **Delete**.
