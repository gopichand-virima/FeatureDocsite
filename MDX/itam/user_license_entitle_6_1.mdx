# User License Entitlements

Use this function to obtain a clear, consolidated view of an organization's licenses and deployments across the product users. This information can be valuable in the software audit, giving organizations the data needed to manage risk and remain compliant as software is consumed across their organization.

In the navigation pane, select **ITAM > User License Entitlement**. The User License Entitlement page displays.

!

!

!


### New User License Entitlement

- From the *Select Actions* drop-down list, choose **New User License Entitlement**. The New User License Entitlement window displays.

!

- Click **New**. The New User License Entitlement window displays.

!

- Enter a *Display Name*.
- For *Configuration item*, click **Add,** then search for and select the CI (typically an asset) that goes with this license.
- For *Configuration item*, click the **plus + sign**, then search for and select the CI (typically an asset) that goes with this license.
- For *User*, click **Add**, then search for an add the user who is entitled to this license.
- For *User*, click the **plus + sign**, then search for and add the user entitled to this license.
- When all selections/entries are made, click **Add**.


### Edit User License Entitlement

Not all functions include an option to edit record line items shown in the main window.


- Navigate to the window containing the record to edit.
- Click the line containing the record. The applicable window or dialog box displays.
- Make the necessary modifications.
- Click **Save** or **Add**, as applicable.

When editing an existing item, other options may be available, such as adding [Tasks](/itsm/release_mngmt/tasks_6_1.mdx), [Comments](/discovery/comments_6_1.mdx), and [Attachments](/discovery/attachments_6_1.mdx) and viewing [History](/common_topics/history_6_1.mdx).


The options change based upon the function; therefore, not every option may be shown.


### Delete User License Entitlement

Deleting is a permanent action and cannot be undone. Deleting may affect other functionality and information in the application such as data in configured reports, fields in windows, selectable options, etc. Therefore, be sure to understand the potential effects before making a deletion.


- Click the line that contains the item to delete.
- From the *Select Actions* drop-down list, choose **Delete**. If a confirmation message is displayed, type **Delete** and click **Delete**to confirm.
