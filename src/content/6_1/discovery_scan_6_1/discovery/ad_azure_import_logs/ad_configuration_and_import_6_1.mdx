# AD Configuration and Import

Active Directory (AD) is a Microsoft directory service for Windows domain networks. It authenticates and authorizes users and computers, enforces security policies, and manages software installations and updates.

In the main window, click <span class="Strong" style="font-weight: bold;">Admin /> Users /> AD Configuration</span>. **AD Configuration** window displays.
