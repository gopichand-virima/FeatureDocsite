# Email Actions

Use this function to configure what action to take upon receipt of an email to create an ITSM record.

In the main window, click **Admin \> Event Management \> Email Actions.**

![email actions old](../Resources/Images/Admin - Event/email-actions-old.PNG)

![arrow border](../Resources/Images/arrow-border.PNG)

 1. From the *Select Actions* drop-down list, choose **New Action**.

 2. Complete the sections as described below.
 -- ----- -------------------------------------------

 -- ----- ------------------------------------------------------
 3. When all selections/entries are made, click **Add**.
