# Uninstall the Discovery Application

The best and safest way to uninstall the existing application is by using a Windows uninstaller.

- Go to Windows Control Paneland select **Programs and Features**.
- SelectVirima Discovery Application.
- Click**Uninstall**.
- Navigate to the installed location and delete the directoryVirima Discovery Applicationto remove all the temp and log files.
