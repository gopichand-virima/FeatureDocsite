# Sensors

Use this function to configure sensors for continuous monitoring and data collection.

In the main window, click **Admin > Discovery > Sensors**. The Sensors window displays.

From the *Select Actions* drop-down list, choose an action: **New Sensor**, [Delete](/common_topics/delete_remove_5_13.mdx), or [Export](/discovery/import_data_files/export_5_13.mdx).
