# Release State

Use this function to add a new state defining the status for the Release being worked by an assignee.

In the main window, click **Admin > Release Management > Release State**. The Release State window displays.

New Release State

From the *Select Actions* drop-down list, choose **New Release State**. The New Release State window displays.

Enter a *Name* and *Description*.

In the *State* field, click the drop-down list and select the applicable state.

Click **Add**.
