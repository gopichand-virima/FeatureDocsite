# Accessing the Import Data Files Page

1.  From the left navigation panel, navigate to <span class="Strong" style="font-weight: bold;">Discovery Scan</span>. Click <span class="Strong" style="font-weight: bold;">Import Data Files</span>.

    <span class="image placeholder" original-image-src="../../Resources/Images/Discovery/Import-Data-Files/Import%20Data%20Files_1.png" original-image-title=""></span>

<!-- -->

2.  The **Import Data Files** page opens.

    <span class="image placeholder" original-image-src="../../Resources/Images/Discovery/Import-Data-Files/Import%20Data%20Files_1_1.png" original-image-title=""></span>

<!-- -->

3.  Display the list of uploaded files with these columns:

    - <span class="Strong" style="font-weight: bold;">File Name</span> – The name of the uploaded file.

    - <span class="Strong" style="font-weight: bold;">Status</span><span class="inline-comment-marker"> – Indicates processing state (e.g., </span><span class="Emphasis">Processed</span><span class="inline-comment-marker">, </span><span class="Emphasis">Processing</span><span class="inline-comment-marker">).</span>

      - <span class="Strong" style="font-weight: bold;">Processing</span>: The CI(s) in the uploaded file are currently being imported. (Import in Progress)

      - <span class="Strong" style="font-weight: bold;">Processed</span>: The system successfully imported the CIs from the uploaded file.

    - <span class="Strong" style="font-weight: bold;">Uploaded By</span> – Identifies the user who uploaded the file.

    - <span class="Strong" style="font-weight: bold;">Uploaded On</span> – The date and time when the file was uploaded.

    <span class="image placeholder" original-image-src="../../Resources/Images/Discovery/Import-Data-Files/Import%20Data%20Files_1_2.png" original-image-title=""></span>

<!-- -->

4.  Use the checkboxes to select one or more files for bulk actions.
