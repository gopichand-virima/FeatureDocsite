## Personalize Columns

|                                                                                                                                                            |                                                                              |
|------------------------------------------------------------------------------------------------------------------------------------------------------------|------------------------------------------------------------------------------|
| <span class="image placeholder" original-image-src="../Resources/Images/Common%20Topics/icon-personalize-list-columns.png" original-image-title=""></span> | Use this function to personalize the columns displayed in the record window. |

<div class="NumList1" madcap:autonum="1. ">

While viewing a record list, click the **Personalize Columns** icon.

</div>

<div class="Ind50">

<span class="image placeholder" original-image-src="../Resources/Images/Discovery/personalize-list-columns.PNG" original-image-title="" style="width: 332px;height: 238px;" madcap:conditions="Default.OldUI"></span>

</div>

<div class="Ind50">

<span class="image placeholder" original-image-src="../Resources/Images/Discovery/personalize-columns.PNG" original-image-title="" madcap:conditions="Default.NewUI" style="width: 331px;height: 314px;"></span>

</div>

<div class="numlist2" madcap:conditions="Default.OldUI" madcap:autonum="2. ">

In the *Search for properties* field, type the name of any properties to search for, then click the **Search** icon.

</div>

<div class="numlist2" madcap:conditions="Default.NewUI" madcap:autonum="3. ">

In the *Search* field, type the name of any properties to search for.

</div>

<div class="numlist2" madcap:conditions="Default.OldUI" madcap:autonum="4. ">

Under the *Search Results* column, a list of all available columns is shown. A check mark indicates the column has been selected.

</div>

<div class="numlist2" madcap:conditions="Default.NewUI" madcap:autonum="5. ">

Under the *AVAILABLE FIELDS* column, a list of all available columns is shown. A check mark indicates the column has been selected.

</div>

|     |                                                                                                              |
|-----|--------------------------------------------------------------------------------------------------------------|
|     | The list of available columns are predefined in the system and cannot be changed through the user interface. |

<div class="numlist2" madcap:conditions="Default.OldUI" madcap:autonum="6. ">

Under the *SELECTED ITEMS* column, a list of all currently selected columns is shown. To add an item to this list, click on the column name in the *Search Results* list. The selected item is now available in the *Selected Items* list.

</div>

<div class="numlist2" madcap:conditions="Default.NewUI" madcap:autonum="7. ">

Under the *SELECTED FIELDS* column, a list of all currently selected columns is shown. To add an item to this list, check the box for the field name in the *Available Fields* list. The selected item is now available in the *Selected Fields* list.

</div>

<div class="numlist2" madcap:conditions="Default.NewUI,Default.OldUI" madcap:autonum="8. ">

Click **Default View** to load the default settings.

</div>

<div class="numlist2" madcap:conditions="Default.OldUI,Default.NewUI" madcap:autonum="9. ">

Click **Clear All** to remove all selections and return to the default view.

</div>

<div class="numlist2" madcap:conditions="Default.OldUI,Default.NewUI" madcap:autonum="10. ">

Click **Save** to retain the current selections.

</div>

<div class="numlist2" madcap:conditions="Default.OldUI,Default.NewUI" madcap:autonum="11. ">

Click **Cancel** to discard any changes.

</div>
