## Personalize Columns

|  |  |

| --- | --- |

| ![icon personalize list columns](../Resources/Images/Common%20Topics/icon-personalize-list-columns.png) | Use this function to personalize the columns displayed in the record window. |


- While viewing a record list, click the **Personalize Columns** icon.

! !

- In the *Search for properties* field, type the name of any properties to search for, then click the **Search** icon.
- In the *Search* field, type the name of any properties to search for.
- Under the *Search Results* column, a list of all available columns is shown. A check mark indicates the column has been selected.
- Under the *AVAILABLE FIELDS* column, a list of all available columns is shown. A check mark indicates the column has been selected.


The list of available columns are predefined in the system and cannot be changed through the user interface.


- Under the *SELECTED ITEMS* column, a list of all currently selected columns is shown. To add an item to this list, click on the column name in the *Search Results* list. The selected item is now available in the *Selected Items* list.
- Under the *SELECTED FIELDS* column, a list of all currently selected columns is shown. To add an item to this list, check the box for the field name in the *Available Fields* list. The selected item is now available in the *Selected Fields* list.
- Click **Default View** to load the default settings.
- Click **Clear All** to remove all selections and return to the default view.
- Click **Save** to retain the current selections.
- Click **Cancel** to discard any changes.
