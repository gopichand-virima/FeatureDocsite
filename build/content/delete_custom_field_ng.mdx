# Delete Custom Field

Use this function to remove custom fields that are no longer needed.

In the main window, click **Admin > Custom Fields**, select a field, and click **Delete**.

## Delete Considerations

**Before Deleting**:

- Backup existing data
- Check field usage in reports
- Verify workflow dependencies
- Review API integrations
- Check import/export templates

**Deletion Impact**:

- All field data will be permanently deleted
- Reports using this field will fail
- Workflows may break
- API calls will return errors
- Cannot be undone

**Alternative**: Instead of deleting, consider making the field inactive or hidden.
