# IT Asset Management - Online Help Overview

Welcome to the IT Asset Management (ITAM) module documentation for Virima version 5.13.

## Overview

The ITAM module provides comprehensive lifecycle management for IT assets, from procurement through retirement, including software license management, hardware tracking, and financial management.

## Key Features

- **Asset Lifecycle Management**: Track assets from cradle to grave
- **Software License Management**: Optimize license usage and compliance
- **Contract Management**: Manage vendor contracts and warranties
- **Financial Tracking**: Monitor costs, depreciation, and TCO
- **Procurement Integration**: Streamline asset acquisition process

## Getting Started

To begin managing IT assets:
1. Navigate to ITAM from the main menu
2. Import or add existing assets
3. Configure asset categories and statuses
4. Set up software license tracking
5. Create asset management policies
