# Edit Discovered Items

Use this function to edit the details for discovered items.

In the navigation pane, select **Discovery Scan > Discovered Items**. The Discovered Items window displays.

Select an item to edit. The window refreshes and the Configuration Items window displays.


The options that display change based on the type of asset selected in the previous window.
