## Filters

|                                                                                                                                                 |                                      |
|-------------------------------------------------------------------------------------------------------------------------------------------------|--------------------------------------|
| <span class="image placeholder" original-image-src="../Resources/Images/Common%20Topics/icon-saved-filters.png" original-image-title=""></span> | Use this function to manage filters. |

As the amount of data accumulates, it is often necessary to sort it to display specific data that is needed for a particular task. A filter is a set of conditions applied to a table in order to find and work with a subset of the data in that table.

|     |                             |
|-----|-----------------------------|
|     | Filters are users specific. |

## Apply and Save Filters

<div class="numlist1" madcap:conditions="Default.OldUI" madcap:autonum="1. ">

Under the column heading, click in the *Filter for Name* field.

</div>

<div class="NumList2" madcap:conditions="Default.NewUI" madcap:autonum="2. ">

Under the column heading, click in the *Search for Name* field.

</div>

<div class="numlist2" madcap:autonum="3. ">

Type the search text.

</div>

<div class="numlist2" madcap:autonum="4. ">

Press **Enter**. The window updates, and all matching records (if any) are displayed.

</div>

<div class="NumList2" madcap:autonum="5. ">

Click **SAVE FILTER**.

</div>

<div class="numlist2" madcap:autonum="6. ">

In the Save Filter dialog box, type a *Name* to identify this filter, and click **Save**.

</div>

## Apply and Delete Filters

<div class="numlist1" madcap:autonum="1. ">

In the results window, move to the toolbar and click the **Saved Filter** icon. The Filters window displays.

</div>

<div class="Ind50" madcap:conditions="Default.OldUI">

<span class="image placeholder" original-image-src="../Resources/Images/Common%20Topics/saved-filter-list.PNG" original-image-title=""></span>

</div>

<div class="Ind50" madcap:conditions="Default.NewUI">

<span class="image placeholder" original-image-src="../Resources/Images/Common%20Topics/saved-filters-new-list-delete.PNG" original-image-title="" style="width: 363px;height: 134px;"></span>

</div>

<div class="numlist2" madcap:autonum="2. ">

Select the filter you want to apply from the list.

</div>

<div class="numlist2" madcap:autonum="3. ">

To save a filter as the default, click **Set as Default**. Only one filter can be the default.

</div>

<div class="numlist2" madcap:autonum="4. ">

To delete the filter, click the **trashcan** icon.

</div>
