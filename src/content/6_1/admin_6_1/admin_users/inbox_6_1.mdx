# Inbox

Use this function to create and send new messages, and view and manage incoming messages.

<div class="NumList1" madcap:autonum="1. ">

In the Access Management window, select a user to edit. The Details window displays.

</div>

<div class="NumList2" madcap:autonum="2. ">

Click **Inbox**.

</div>

<div class="Ind50" madcap:conditions="Default.NewUI">

<span class="image placeholder" original-image-src="../Resources/Images/Admin%20-%20Users/inbox_6.1.png" original-image-title=""></span>

</div>

<div class="NumList2" madcap:autonum="3. ">

The following functions are available:

</div>

<div class="Bullet2">

**Delete**. Removes the selected message(s) from the Inbox.

</div>

<div class="Bullet2">

**Compose**. Opens a new window in which a new message is written and sent. Attachments can be included with the message.

</div>

<div class="Bullet2">

**Inbox**. Displays the list of all messages.

</div>

<div class="Bullet2">

**Sent**. Displays a list of sent items.

</div>

<div class="Bullet2">

**Starred**. Displays a list of all mail that has been flagged to show greater significance.

</div>

<div class="Bullet2">

**Trash**. Shows all mail that has been deleted. In this view, you can also permanently delete messages.

</div>

<div class="NumList2" madcap:autonum="4. ">

To search for a message, enter the criteria in the *Search* field and press **Enter**.

</div>
