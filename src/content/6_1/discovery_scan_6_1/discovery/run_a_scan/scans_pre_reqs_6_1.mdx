# Prerequisites for Scan

<span class="span_5">Credentials are required to scan the </span><span class="Strong_4">Target Host</span><span class="span_5"> and can be entered using the methods described below.</span>

|     |                                                                                                                      |
|-----|----------------------------------------------------------------------------------------------------------------------|
|     | Added and imported credentials are encrypted and stored on the machine where the Discovery Application is installed. |

### <span class="Strong_4">Method 1</span><span class="span_5">: </span><span class="Strong_4">Through Virima Discovery Admin</span>

*Refer to* <a href="../../Admin%20-%20Discovery/Credentials.htm" style="font-style: italic;">Credentials</a>*.*

### <span class="Strong_4">Method 2</span><span class="span_5">: </span><span class="Strong_4">Through the Discovery Application using the Application Interface</span>

<div class="NumList1" madcap:autonum="1. ">

<span class="span_5">Right-click the </span><span class="Strong_4">**Virima Discovery Application**.</span>

</div>

<div class="NumList2" madcap:autonum="2. ">

<span class="span_5">Click </span><span class="Strong_4">**Run as Administrator**.</span>

</div>

<div class="NumList2" madcap:autonum="3. ">

<span class="span_5">Click </span><span class="Strong_4" style="font-weight: bold;">Start</span><span class="span_5"> and wait until it shows </span><span class="Strong_4">that the Discovery Agent is Running.</span>

</div>

<div class="NumList2" madcap:autonum="4. ">

<span class="span_5">From the Credentials drop-down list, select **Add Credential**. </span>

</div>

<div class="Ind50 step-media">

<span class="image placeholder" original-image-src="../../Resources/Images/Discovery/scan-prereq-credentials.png" original-image-title="" style="width: 471px;height: 216px;"></span>

</div>

<div class="Ind50">

The Add new credential window displays.

</div>

<div class="Ind50 step-media">

<span class="image placeholder" original-image-src="../../Resources/Images/Admin%20-%20Discovery/discovery-agent-add-credential.PNG" original-image-title="" style="width: 472px;height: 416px;"></span>

</div>

<div class="NumList2" madcap:autonum="5. ">

<span class="span_5">Complete the fields.</span>

</div>

<div class="NumList2" madcap:autonum="6. ">

When all selections/entries are made, click **Save**.

</div>

### Method 3: Load the Credentials from an Excel File

To load the credentials through this method, credentials must be configured in an Excel file, as shown in the figure below.

<span class="image placeholder" original-image-src="../../Resources/Images/Admin%20-%20Discovery/credential-inventory-excel-sample.PNG" original-image-title="" style="width: 671px;height: 252px;"></span>

<div class="NumList1" madcap:autonum="1. ">

<span class="span_5">Store the credential file in this location: </span><span class="Strong_3"><span style="font-family: 'Courier New';"><span style="font-size: 10pt;">  
C:/Program Files/Virima Discovery Application/Config</span></span></span><span class="span_5"><span style="font-family: 'Courier New';"><span style="font-size: 10pt;"> as </span></span></span><span class="Strong_3"><span style="font-family: 'Courier New';"><span style="font-size: 10pt;">Credentials.xls</span></span></span><span class="span_5"><span style="font-family: 'Courier New';"><span style="font-size: 10pt;"> or </span></span></span><span class="Strong_3"><span style="font-family: 'Courier New';"><span style="font-size: 10pt;">Credentials.xlsx</span></span></span>

</div>

<div class="NumList2" madcap:autonum="2. ">

<span class="span_5">Right-click the </span><span class="Strong_4">**Virima Discovery Application**.</span>

</div>

<div class="NumList2" madcap:autonum="3. ">

<span class="span_5">Click </span><span class="Strong_4">**Run as administrator**.</span>

</div>

<div class="NumList2" madcap:autonum="4. ">

<span class="span_5">Click </span><span class="Strong_4" style="font-weight: bold;">Start</span><span class="span_5"> and wait for the indication that the </span><span class="Strong_4">Discovery agent is Running.</span>

</div>

<div class="NumList2" madcap:autonum="5. ">

<span class="span_5">In the Discovery Agent window, click the **Credentials** drop-down list and select **Import Credentials**.</span>

</div>

|     |                                                                                                                      |
|-----|----------------------------------------------------------------------------------------------------------------------|
|     | The added/imported credentials are encrypted and stored on the machine where the Discovery Application is installed. |
