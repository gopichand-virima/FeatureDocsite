# My Requests

Use this function to displays the status of requests that are created by the currently logged in user.

</div>

In the navigation pane, select **Self-Service /> My Requests**. The resulting window displays a list of all service catalog requests (if any).

<span class="image placeholder" original-image-src="../Resources/Images/Self-Service/my-requests-main-old.PNG" original-image-title=""></span>

To view the related request information, under Request ID, click the **hyperlink**. The related [Request](../ITSM/Request-Fulfillment/Requests.htm) window displays.
