# Tasks

Use this function to view tasks that have been created when working with [activities](Activities.htm).

In the main window, select **ITSM /> Release Management /> Tasks.**

<span class="image placeholder" original-image-src="../../Resources/Images/ITSM/Release%20Management/tasks-main-old.PNG" original-image-title=""></span>

<span class="image placeholder" original-image-src="../../Resources/Images/ITSM/Release%20Management/tasks-main-new.PNG" original-image-title=""></span>

New Tasks

New tasks cannot be created in this window. They must be added through the [Activities](Activities.htm) function.

<div class="NumList1" madcap:autonum="1. ">

In the main Activities window, do either of the following:

</div>

<div class="Bullet2">

Select a line containing an existing activity.

</div>

<div class="Bullet2">

From the *Select Actions* drop-down list, choose **New Release Activity**.

</div>

<div class="NumList2" madcap:autonum="2. ">

Under *Release Tasks*, click the **plus + sign**. The Release Tasks dialog box displays.

</div>

<div class="Ind50" madcap:conditions="Default.OldUI">

<span class="image placeholder" original-image-src="../../Resources/Images/ITSM/Release%20Management/release-tasks-old.PNG" original-image-title=""></span>

</div>

<div class="Ind50" madcap:conditions="Default.NewUI">

<span class="image placeholder" original-image-src="../../Resources/Images/ITSM/Release%20Management/release-tasks-new.PNG" original-image-title=""></span>

</div>

<div class="NumList2" madcap:autonum="3. ">

Click **Add New Release Task**. The New Release Task window displays.

</div>

<div class="Ind50">

<span class="image placeholder" original-image-src="../../Resources/Images/ITSM/Release%20Management/new-release-task-old.PNG" original-image-title="" madcap:conditions="Default.OldUI"></span>

</div>

<div class="Ind50" madcap:conditions="Default.NewUI">

<span class="image placeholder" original-image-src="../../Resources/Images/ITSM/Release%20Management/new-release-task-new.PNG" original-image-title=""></span>

</div>

<div class="NumList2" madcap:autonum="4. ">

Enter the task information.

</div>

<div class="NumList2" madcap:autonum="5. ">

Click **Add**. This task now displays in the Tasks main window.

</div>
