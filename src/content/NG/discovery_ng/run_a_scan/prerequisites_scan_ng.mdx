# Prerequisites

Before running a scan, ensure the following are configured:

- **Discovery Client Setup**: At least one active client must be configured in **Admin > Discovery > Clients**.
- **Credentials Configured**: Required credentials (SNMP, SSH, WMI, etc.) must be available in **Admin > Discovery > Credentials**.
- **Network Access**: The discovery client must have network connectivity to the target IP range.
- **User Permissions**: The logged-in user must have **Run Scan** permissions.
