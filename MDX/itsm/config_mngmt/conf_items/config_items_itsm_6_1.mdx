# CI: ITSM

This content is currently under development.

All the text below is conditioned out until the "New" buttons start working.

Use this function to create ITSM records.

- From the navigation pane, select **ITSM > Configuration Management > Configuration Items**. The Configuration Items window displays.
- From the navigation pane, select **ITAM > Configuration Management > Configuration Items**. The Configuration Items window displays.
- Select a record in the list. A new windows displays with the **Details** tab selected by default.
- Click **ITSM**.

!

- Click an item in the list. The main window updates with the appropriate fields.
- Click the "New" button for the selected item (for example, New Incident, New Request, and so forth).

NEED TO INSERT INSTRUCTIONS ON WHAT THEY DO ON EACH "NEW" WINDOW.
