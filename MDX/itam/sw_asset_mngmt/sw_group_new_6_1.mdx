# New Software Group

This feature enables to create group of software assets present in the monitored network environment.

- Choose New Software Group option from the Select Actions dropdown. The New Software Group page displays.


- Enter the name for the software group in the Group Name.
- Click Add and update details of the Vendor, and Software License Key parameters.

Vendor:

!

- Enter the search criteria, and then click search icon.
- Select the required vendor from the displayed list, and then click Apply.
