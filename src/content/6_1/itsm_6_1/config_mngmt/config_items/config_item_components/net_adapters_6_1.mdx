Add Network Adapter

<div class="NumList1" madcap:autonum="1. ">

Click **Add**. The Network Adapter dialog box displays.

</div>

<div class="Ind50" madcap:conditions="Default.OldUI">

<span class="image placeholder" original-image-src="../../../../Resources/Images/Discovery/network-adapter-add-new.PNG" original-image-title=""></span>

<div class="Ind50" madcap:conditions="Default.NewUI">

</div>

<div class="NumList2" madcap:conditions="Default.NewUI" madcap:autonum="2. ">

Click **New Network Adapter.**

</div>

<div class="NumList2" madcap:autonum="3. ">

Enter the *MAC Address*, *Network Adapter Name*, *IP Address*, and *Asset Names*. These are all required fields.

</div>

<div class="NumList2" madcap:conditions="Default.NewUI" madcap:autonum="4. ">

When all selections/entries are made, click **Save**.

</div>

<div class="NumList2" madcap:conditions="Default.OldUI" madcap:autonum="5. ">

When all selections/entries are made, click **Add**.

</div>

Edit Network Adapter

<div class="NumList1" madcap:autonum="1. ">

Click **Existing Network Adapter**.

</div>

<div class="Ind50" madcap:conditions="Default.OldUI">

<span class="image placeholder" original-image-src="../../../../Resources/Images/Discovery/network-adapter-modify-existing.PNG" original-image-title=""></span>

<div class="Ind50" madcap:conditions="Default.NewUI">

</div>

<div class="NumList2" madcap:autonum="2. ">

Enter the search criteria to locate the existing adapter*.*

</div>

<div class="NumList2" madcap:autonum="3. ">

Select the applicable adapter from the results list.

</div>

<div class="NumList2" madcap:autonum="4. ">

When all selections are made, click **Save**.

</div>

Edit Network Adapter Details

<div class="NumList1" madcap:autonum="1. ">

While viewing the list of existing adapters, select the adapter to edit. The windows refreshes and the details for the adapter displays.

</div>

<div class="NumList2" madcap:autonum="2. ">

Make any necessary changes to the existing information. The changes are automatically saved. *Refer to* <a href="Component-Tab-Options.htm" style="font-style: italic;">CI Components</a>*.*

</div>

<div class="NumList2" madcap:autonum="3. ">
