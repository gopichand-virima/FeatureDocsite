# Catalog Categories

Use this function to organize catalog items into logical categories and subcategories.

Catalog Categories help users find services easily and provide a structured navigation experience.

In the main window, click **Admin > Service Catalog > Catalog Categories**. The Catalog Categories window displays.

## Category List

View all categories:

**Category Columns**:

- **Category Name**: Category display name
- **Parent Category**: Parent if subcategory
- **Items**: Number of catalog items
- **Order**: Display order
- **Status**: Active, Inactive
- **Icon**: Category icon

## Create Category

From the *Select Actions* drop-down list, choose **New Category**.

**Category Details**:

- **Category Name**: Hardware Requests
- **Description**: Request IT hardware equipment
- **Parent Category**: None (top-level) or select parent
- **Icon**: Upload category icon
- **Display Order**: 1 (appears first)
- **Status**: ☑ Active

**Example Category Structure**:

```
Hardware Requests
  ├── End User Devices
  │   ├── Laptops
  │   ├── Desktops
  │   └── Monitors
  ├── Mobile Devices
  │   ├── Smartphones
  │   └── Tablets
  └── Accessories
      ├── Keyboards
      ├── Mice
      └── Headsets

Software Requests
  ├── Productivity Software
  ├── Development Tools
  └── Design Software

Access Requests
  ├── System Access
  ├── Application Access
  └── Network Access
```

Click **Save**.
