# New Probes

- In the main window, select **Admin > Discovery > Probes**. The Probes window displays.

!

- From the *Select Actions* drop-down list, choose **New Probes**. The New Probes window displays.
- In the main window, click the **gear** icon and select **Discovery > Probes**. The Probes window displays.

!

- Click **New Probe**. The New Probe window displays.

!

- Enter a *Name* for the probe.
- From the drop-down lists, select the *Type* and *Sensor*.
- Enter a *Description*.
- When all selections/entries are made, click **Add**.
