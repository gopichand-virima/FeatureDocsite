# Self Service - User Manual

Complete guide for the Self Service portal in Virima 5.13.

## Service Catalog

### Service Categories
Common service categories:
- Hardware Requests (laptops, monitors, peripherals)
- Software Access (application licenses)
- Account Management (password resets, access requests)
- Office Services (workspace setup, moves)
- Training Requests

### Service Request Process
1. **Selection**: Choose service from catalog
2. **Form Completion**: Provide required information
3. **Approval**: Automatic routing for approvals
4. **Fulfillment**: IT team processes request
5. **Completion**: Service delivered and closed

### Viewing Requests
- My Open Requests
- Request History
- Request Details
- Status updates

### Request Status
- **Submitted**: Awaiting review
- **Pending Approval**: Waiting for approval
- **In Progress**: Being worked on
- **Awaiting Info**: Requires user input
- **Resolved**: Completed
- **Closed**: Finalized
