# Databases

 --- -------------------------------------------------------------------------------------------------------------
   This function is only available with certain asset types. Therefore, the relevant tab may not be displayed.
 --- -------------------------------------------------------------------------------------------------------------

 -- ----- ------------------------------------------------------
 1. While viewing an asset record, click **Components**.
 -- ----- ------------------------------------------------------

 -- ----- ----------------------
 2. Click **Databases**.
 -- ----- ----------------------

::: Ind50
![databases new 482x140](../../../Resources/Images/Components/databases-new_482x140.png)

## Add a New Database

 -- ----- --------------------------------------------------
 1. Click **Add**. The Database dialog box displays.
 -- ----- --------------------------------------------------

::: Ind50
![databases add new 382x221](../../../Resources/Images/Components/databases-add-new_382x221.png)

 -- ----- -------------------------
 2. Click **New Database.**
 -- ----- -------------------------

 -- ----- -----------------------------------------
 3. Enter the applicable field information.
 -- ----- -----------------------------------------

 -- ----- --------------------------------------------
 4. When all entries are made, click **Save.**
 -- ----- --------------------------------------------

## Modify Existing Database

 -- ----- ------------------------------
 1. Click **Existing Database**.
 -- ----- ------------------------------

::: Ind50
![databases existing new 371x253](../../../Resources/Images/Components/databases-existing-new_371x253.png)

 -- ----- ----------------------------------------------------------------------------------------------------------------------------------------------------------------------
 2. Enter the search criteria to locate the existing device.  *See also* [Advanced Search](../../../Common%20Topics/Advanced-Search.mdx){style="font-style: italic;"}*.*
 -- ----- ----------------------------------------------------------------------------------------------------------------------------------------------------------------------

 -- ----- -----------------------------------------------------
 3. Select the applicable device from the results list.
 -- ----- -----------------------------------------------------

 -- ----- -----------------------------------------------
 4. When all selections are made, click **Save**.
 -- ----- -----------------------------------------------
