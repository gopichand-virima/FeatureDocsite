# Dashboard Customization

The Dashboard can be customized to display various reports.


Refer to [Add Contents to the Discovery Dashboard](/dashboards/dashboard_contents_5_13.mdx) for details on managing reports.


Refer to [Add Contents to the Discovery Dashboard](/dashboards/dashboard_contents_5_13.mdx) for details on managing reports.

This is not a complete list of reports available, only selected samples.

- [Discovered assets based on blueprints](/discovery/popups/example_disc_assets_blueprints_5_13.mdx)

For dashboard reports, the following options may be available:

- **Click for a Complete Report**. Displays a new window that contains additional details for the selected report.
- **Export**. Generates an Excel spreadsheet and sends an email to the logged-in user when the spreadsheet is ready for download.
- **Save Image**. Saves a copy of the current image. The image-saving process varies based on the browser and defined settings.
- **Collapse/Expand**. Closes or opens the report.
- **Maximize/Minimize**. When *Maximize* is selected, the data displays in a new window. When *Minimize* is selected in the data window, the window is closed and returns to the Dashboard.
- **Remove**. Removes the report from the display area. Prompts to confirm the removal of the currently selected report. Click **Yes** or **No**.
