# View Scan Details

Click on any scan record to open its detailed view.

The scan details page provides comprehensive information organized in tabs:

## Details Tab
- **Scan Name**: The name assigned to this scan
- **Status**: Current status (Running, Completed, Failed, Stopped)
- **Probe Type**: The type of scan that was executed
- **Client**: Discovery client that performed the scan
- **IP Range**: Target IP addresses
- **Exclude IP Range**: Excluded IP addresses (if any)
- **Started On**: Start timestamp
- **Completed On**: Completion timestamp
- **Duration**: Total execution time
- **Location**: Associated location tags
- **Scan Report Recipients**: Users who received the scan report

## Discovered Items Tab
- List of all assets discovered during this scan
- Quick access to view or manage discovered items
- Filter and sort capabilities
- Bulk actions (Move to CMDB, Rescan, etc.)

## Log Tab
- Detailed scan execution logs
- Step-by-step process information
- Success and error messages
- Timing information for each scan phase
- Troubleshooting data for failed scans

## Tasks Tab
- View and manage tasks associated with this scan
- Create new tasks related to scan follow-up actions

## Comments Tab
- View and add comments about the scan
- Collaboration notes from team members

## Attachments Tab
- Upload and view files related to the scan
- Store screenshots, reports, or supporting documentation
