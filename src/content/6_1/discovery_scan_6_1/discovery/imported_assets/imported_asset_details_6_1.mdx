# Imported Asset Details Page

When viewing an imported asset in Virima, the page displays structured information across multiple sections, allowing administrators to review, validate, and take further actions such as moving the asset to the CMDB or rescanning.

- Clicking the hyperlinked CI ID under CI column will display the corresponding record in CMDB.

<span class="image placeholder" original-image-src="../../Resources/Images/Discovery/Imported-Assets/Imported%20Assets_11.png" original-image-title=""></span>

- If we click on any other column values regardless of whether CI exists in CMDB or not, the user is redirected to Imported Assets view to view the imported asset data.

<span class="image placeholder" original-image-src="../../Resources/Images/Discovery/Imported-Assets/Imported%20Assets_12.png" original-image-title=""></span>

# <span class="Strong">Header Section</span>

- <span class="Strong" style="font-weight: bold;">Asset ID</span>: Unique identifier assigned to the imported asset (e.g., <span class="Strong">IMP001277</span>).

- <span class="Strong" style="font-weight: bold;">Move to CMDB Button</span>: Pushes the asset record into the Configuration Management Database (CMDB).

  - The record will appear greyed out while the imported asset is being moved to the CMDB. Once the move is complete, the asset will be removed from the Imported Assets list.

  - During this process, the system displays the progress message “import item is being processed” and automatically redirects the page to the imported asset’s details.

<span class="image placeholder" original-image-src="../../Resources/Images/Discovery/Imported-Assets/Imported%20Assets_13.png" original-image-title=""></span>

- <span class="Strong" style="font-weight: bold;">Rescan Now Button</span>: Triggers an immediate re-scan of the asset to validate or update its details.

  - Select the <span class="Strong" style="font-weight: bold;">Client</span> and <span class="Strong" style="font-weight: bold;">Type of scan</span> from the dropdown menus and then Click <span class="Strong" style="font-weight: bold;">Scan</span> to initiate the rescan.

  - The system will display a success message and redirect you to the Recent Scan page.

<span class="image placeholder" original-image-src="../../Resources/Images/Discovery/Imported-Assets/Imported%20Assets_14.png" original-image-title=""></span>

# <span class="Strong">Tabs Section</span>

Multiple tabs are available to provide deeper insights into the asset:

- <span class="Strong" style="font-weight: bold;">Details</span> (default view) – Displays primary information, hardware, lifecycle, and network adapter details.

- <span class="Strong" style="font-weight: bold;">Components</span> – Shows associated components for the asset.

- <span class="Strong" style="font-weight: bold;">Log On Events</span> – Captures and displays login activity related to the asset.

- <span class="Strong" style="font-weight: bold;">Tasks</span> – Tracks assigned tasks or actions linked to this asset.

- <span class="Strong" style="font-weight: bold;">Comments</span> – Allows administrators to add or view comments about the asset.

- <span class="Strong" style="font-weight: bold;">Attachments</span> – Stores supporting files such as documents, screenshots, or reports.

- <span class="Strong" style="font-weight: bold;">Updates</span> – Displays historical updates or changes to the asset record.

For more information on Tabs refer the [Asset Tab Details](https:/virima.atlassian.net/wiki/spaces/PM/pages/231112748/Asset+tab+Details?atlOrigin=eyJpIjoiN2Q3OTdiMDU4NjhlNDEyYTk2OWUyN2UzYjQ0ZmRjOTgiLCJwIjoiYyJ9) document.

# Main panel – Details tab sections

- <span class="Strong" style="font-weight: bold;">Asset Primary Information</span>

  - <span class="Strong" style="font-weight: bold;">Asset ID</span> – the imported ID (IMP prefix).

  - <span class="Strong" style="font-weight: bold;">Asset Name</span> – Asset name from the file.

  - <span class="Strong" style="font-weight: bold;">Host Name</span> – The hostname of the asset.

- <span class="Strong" style="font-weight: bold;">Hardware and Network</span>

  - <span class="Strong" style="font-weight: bold;">IP Address</span> – last/primary IP from the import (e.g., <span class="Emphasis">1472.2.25</span>).

- <span class="Strong" style="font-weight: bold;">Life Cycle Information</span>

  - <span class="Strong" style="font-weight: bold;">Acquisition Date</span> – date/time of acquisition provided during import.

- <span class="Strong" style="font-weight: bold;">Network Adapter Details</span>

  - <span class="Strong" style="font-weight: bold;">MAC Address</span> – network adapter MAC captured in the import.

# <span class="Strong">Left-Side Panels</span>

1.  <span class="Strong" style="font-weight: bold;">Primary Details</span>

    - <span class="Strong" style="font-weight: bold;">Asset ID</span>: Unique system identifier (e.g., <span class="Emphasis">IMP001277</span>).

    - <span class="Strong" style="font-weight: bold;">Blueprint</span>: Type of asset (e.g., <span class="Emphasis">Windows Server</span>).

    - <span class="Strong" style="font-weight: bold;">Created On</span>: Timestamp when the asset was imported (e.g., <span class="Emphasis">08/11/2025, 3:52:51 PM</span>).

    - <span class="Strong" style="font-weight: bold;">Last Modified On</span>: Timestamp of the most recent update.

    - <span class="Strong" style="font-weight: bold;">Authorization Status</span>: Indicates whether the CI is authorized (in this case, <span class="Emphasis">unauthorized</span>).

    - <span class="Strong" style="font-weight: bold;">Tags</span>: Optional labels for categorization (empty if not provided).

2.  <span class="Strong" style="font-weight: bold;">Owner</span>

    - <span class="Strong" style="font-weight: bold;">Name</span>: Designated owner of the asset (blank if not assigned).

    - <span class="Strong" style="font-weight: bold;">Email</span>: Contact email of the owner.

    - <span class="Strong" style="font-weight: bold;">Phone</span>: Contact number of the owner.

3.  <span class="Strong" style="font-weight: bold;">Import Details</span>

    - <span class="Strong" style="font-weight: bold;">File Name</span>: Source file used for import (e.g., <span class="Emphasis">Import+CI’ Test 2.xlsx</span>).

    - <span class="Strong" style="font-weight: bold;">Status</span>: Processing state of the file (e.g., <span class="Emphasis">Processed</span>).

    - <span class="Strong" style="font-weight: bold;">Uploaded By</span>: User who uploaded the file (e.g., <span class="Emphasis">Admin Virima</span>).

    - <span class="Strong" style="font-weight: bold;">Uploaded On</span>: Timestamp of the file upload (e.g., <span class="Emphasis">08/11/2025, 3:52:29 PM</span>).

    - <span class="Strong" style="font-weight: bold;">Authorization Status</span>: Reflects whether the CI is authorized (<span class="Emphasis">unauthorized</span>).
