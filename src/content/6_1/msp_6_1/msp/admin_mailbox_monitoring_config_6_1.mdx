# Mailbox Monitoring Configuration

Use this function to configure the mailbox for monitoring ITSM emails.

<div class="NumList1" madcap:autonum="1. ">

In the MSP window, click **Admin**.

</div>

<div class="NumList2" madcap:autonum="2. ">

Click **Mailbox Monitoring Configuration**.

</div>

New Mailbox Configuration

<div class="NumList1" madcap:autonum="1. ">

From the *Select Actions* drop-down list, choose **Add New Mailbox Configuration**.

</div>

</div>

<div class="NumList2" madcap:autonum="2. ">

Complete the fields, entering the applicable information.

</div>

<div class="Ind50">

<span class="image placeholder" original-image-src="../Resources/Images/MSP/mailbox-config-new.PNG" original-image-title="" madcap:conditions="Default.OldUI"></span>

</div>
