# LDAP User Import Scans

<span style="color: #000000;">Use this function to view scanned results and the details for imported LDAP Users.</span>

In the navigation pane, select **Discovery Scans /> LDAP User Import Scans**. The LDAP User Import Scan window displays and shows a list of scanned data including *Name*, *Started On* and *Completed On*.
