# Edit Transfer Order

This feature enables editing the configured values of the existing contract, and update, and then save the same in the application.

- Click to open the required Transfer Order. The Transfer Order page displays.

!

The Transfer Order displays the following information:

- Details
- Tasks
- Comments
- Attachment
- History
