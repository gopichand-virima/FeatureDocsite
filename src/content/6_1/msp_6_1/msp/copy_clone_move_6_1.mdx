# Copy Template and Clone/Move Incident

The first client configured by the Vendor in the portal is considered the "Primary Client."

The Primary Client can perform the following actions:

<div class="Bullet1">

Copy Template

</div>

<div class="Bullet1">

Clone Incident

</div>

<div class="Bullet1">

Move Incident

</div>

Clone Incident

Use this function to make an identical copy of the properties for an existing incident.

<div class="NumList1" madcap:autonum="1. ">

In the navigation pane, select **ITSM /> Incident Management /> Incident**.

</div>

<div class="NumList2" madcap:autonum="2. ">

Select the incident record to be cloned.

</div>

<div class="NumList2" madcap:autonum="3. ">

From the *Select Actions* drop-down list, choose **Clone**.

</div>

<div class="NumList2" madcap:autonum="4. ">

At the confirmation prompt, click **OK**.

</div>

Move Incident

Use this function to move and incident from one client to another.

<div class="NumList1" madcap:autonum="1. ">

In the navigation pane, select **ITSM /> Incident Management /> Incident**.

</div>

<div class="NumList2" madcap:autonum="2. ">

Select the incident to be moved.

</div>

<div class="NumList2" madcap:autonum="3. ">

From the *Select Actions* drop-down list, choose **Move to Different Client**.

</div>

<div class="NumList2" madcap:autonum="4. ">

In the Clients window, search for and select the applicable client.

</div>
