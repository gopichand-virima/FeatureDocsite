# Sync Logs

Use this function to view the status of an integration. When the record is moved from CMDB to the integrated application, the sync log shows the success or failure of said move.

In the main window, select **ITSM > Configuration Management > Sync Logs**. The Sync Logs window displays and shows the details for all moves.

To view the specifics for a move, click the line item. A window displays and contains relevant details, for example: 

- Move ID
- Start/End Time
- Status
- Created By
- CI information
