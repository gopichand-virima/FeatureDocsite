# Generate Installed Software Report

You can generate an <span class="Strong">Installed/Uninstalled Software Report</span> for a specific date range in exportable format when software is installed or uninstalled.

1.  Click <span class="Strong" style="font-weight: bold;">CMDB</span>, then select <span class="Strong" style="font-weight: bold;">Generate Installed Software Report</span> from the <span class="Strong" style="font-weight: bold;">Select Actions</span> list to export the data.
