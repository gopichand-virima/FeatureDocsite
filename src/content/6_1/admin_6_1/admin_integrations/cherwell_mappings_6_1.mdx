# Cherwell Mappings

Use this function to map properties between a Virima Blueprint and a Cherwell Object.


There must be a [Cherwell Credential](/admin_integrations/cherwell_credential_6_1.mdx) configured before a mapping can be done.


In the main window, click **Admin > Integrations > Cherwell Mappings**.

!

!


### New Mapping

- From the *Select Actions* drop-down list, choose **New Mapping**.

!

- Click the drop-down lists and select the *Property*, *Operator* and *Blueprint*.
- To add a mapping and set the AND parameter, click the **plus** icon.
- To add a mapping and set the OR parameter, click the paperclip icon.
- To remove a mapping, click the red **X**.
- In the *Cherwell Object* field, click the drop-down list and select the applicable object.
- When all selections/entries are made, click **Add**.
- To remove a mapping, select the line item containing the mapping, and click **Remove**.


### Other Functions

!

- **Clone Mapping**. Creates a duplicate of the mappings for the selected record.
- **Re-sync Data**. Re-syncs all the data with Cherwell.
- **Reset Default Mappings**. Resets all the mappings to the default.
- **Re-Sync Data**. Re-syncs all the data with Cherwell.
- **Reset Default Mapping**. Resets all the mappings to the default.


### Edit Mapping

Not all functions include an option to edit record line items shown in the main window.


- Navigate to the window containing the record to edit.
- Click the line containing the record. The applicable window or dialog box displays.
- Make the necessary modifications.
- Click **Save** or **Add**, as applicable.


### Delete Mapping

Deleting is a permanent action and cannot be undone. Deleting may affect other functionality and information in the application such as data in configured reports, fields in windows, selectable options, etc. Therefore, be sure to understand the potential effects before making a deletion.


- Click the line that contains the item to delete.
- From the *Select Actions* drop-down list, choose **Delete**. If a confirmation message is displayed, type **Delete** and click **Delete**to confirm.
