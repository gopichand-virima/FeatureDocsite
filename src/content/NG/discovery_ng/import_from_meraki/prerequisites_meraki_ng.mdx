# Prerequisites

- Ensure valid Cisco Meraki API credentials are created and saved under Admin > Discovery > Credentials**.**
- The user must have appropriate permissions to access the Discovery module and perform import actions.
- Required Meraki services (Access Points, Switches, Firewalls, Routers) should be enabled in the Meraki dashboard.
- The Virima platform must have network access to the Meraki cloud API endpoints.
- Verify that selected services and credentials are correctly mapped during the import setup.
