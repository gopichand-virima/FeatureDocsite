# Azure AD Configuration and Import

<span class="Strong" style="font-weight: bold;">Azure Active Directory (Azure AD)</span> is Microsoft's cloud-based identity and access management service. It authenticates and authorizes users and devices, manages user identities, and controls access to cloud and on-premises applications and resources.

In the main window, click <span class="Strong" style="font-weight: bold;">Admin /> Users /> Azure AD Configuration. </span>The Azure AD Configuration window displays.

<span class="image placeholder" original-image-src="../../Resources/Images/Discovery/Azure-AD-Users-Import-Logs/Azure%20AD%20User%20Import%20Logs.png" original-image-title=""></span>

1.  From the **Select Actions** drop-down list, choose <span class="Strong" style="font-weight: bold;">New Azure AD Configuration.</span>

<span class="image placeholder" original-image-src="../../Resources/Images/Discovery/Azure-AD-Users-Import-Logs/Azure%20AD%20User%20Import%20Logs_1.png" original-image-title=""></span>

2.  On the Configuration tab, fill in the values for the attributes.

- <span class="Strong" style="font-weight: bold;">Name</span> – Provide a configuration name.

- <span class="Strong" style="font-weight: bold;">Client ID</span> – Enter the Azure application client ID.

- <span class="Strong" style="font-weight: bold;">Tenant ID</span> – Enter the Azure tenant ID.

- <span class="Strong" style="font-weight: bold;">Secret Key </span>– Provide the authentication method (secret key).

<span class="image placeholder" original-image-src="../../Resources/Images/Discovery/Azure-AD-Users-Import-Logs/Azure%20AD%20User%20Import%20Logs_2.png" original-image-title=""></span>

- <span class="Strong" style="font-weight: bold;">Certificate</span> – Provide the authentication method (certificate with private key).

<span class="image placeholder" original-image-src="../../Resources/Images/Discovery/Azure-AD-Users-Import-Logs/Azure%20AD%20User%20Import%20Logs_3.png" original-image-title=""></span>
