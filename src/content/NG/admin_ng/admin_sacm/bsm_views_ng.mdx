# BSM View

Use this function to customize the display properties of Business Service Management (BSM).

In the main window, click **Admin \> SACM \> BSM View**. The BSM View window displays.

![bsm view old 680x114](../Resources/Images/Admin - SACM/bsm-view-old_680x114.png)

In the main window, click the **gear** icon and select **SACM \> BSM View**.

![icon custom bsm views](../Resources/Images/Admin - SACM/icon-custom-bsm-views.PNG)

The BSM Views window displays.

![bsm view new 613x296](../Resources/Images/Admin - SACM/bsm-view-new_613x296.png)

![arrow border](../Resources/Images/arrow-border.PNG)

[[[Closed]{.image .placeholder .MCDropDown_Image_Icon original-image-src="../../Skins/Default/Stylesheets/Images/transparent.gif" original-image-title="" height="12" width="16" BSM View](javascript:void(0)){.MCDropDownHotSpot .dropDownHotspot .MCDropDownHotSpot_ .MCHotSpotImage aria-expanded="false"}]{.MCDropDownHead .dropDownHead}

 -- ----- -------------------------------------------------------------------------------------------------------------------------
 1. From the *Select Actions* drop-down list, choose [New BSM View]{style="background-color: #ffffff; font-weight: bold;"}.
 -- ----- -------------------------------------------------------------------------------------------------------------------------

 -- ----- -------------------------
 1. Click **New BSM View**.
 -- ----- -------------------------

 -- ----- ----------------------------------
 2. Enter a *Name* for the BSM view.
 -- ----- ----------------------------------

 -- ----- ------------------------------------------------------------------------------------------------------------------------------------
 3. From the *Include Relationships* and *Include Blueprints* drop-down lists, select the type of required relationship and blueprint.
 -- ----- ------------------------------------------------------------------------------------------------------------------------------------

 -- ----- -----------------------------------------------------------------------------------------------------------------------------------------
 4. From the *Exclude Relationships* and *Exclude Blueprints* drop-down lists, select the type of relationship and blueprint to be omitted.
 -- ----- -----------------------------------------------------------------------------------------------------------------------------------------

 -- ----- --------------------------------------------------------------------------------------------------------------
 5. Check *Include External CI Communication* option to include external communications to the corresponding CI.
 -- ----- --------------------------------------------------------------------------------------------------------------

 -- ----- ------------------------------------------------------
 6. When all selections/entries are made, click **Add**.
 -- ----- ------------------------------------------------------

[[[Closed]{.image .placeholder .MCDropDown_Image_Icon original-image-src="../../Skins/Default/Stylesheets/Images/transparent.gif" original-image-title="" height="12" width="16" BSM View](javascript:void(0)){.MCDropDownHotSpot .dropDownHotspot .MCDropDownHotSpot_ .MCHotSpotImage aria-expanded="false"}]{.MCDropDownHead .dropDownHead}

 --- -----------------------------------------------------------------------------------------
   Not all functions include an option to edit record line items shown in the main window.
 --- -----------------------------------------------------------------------------------------

 -- ----- -------------------------------------------------------
 1. Navigate to the window containing the record to edit.
 -- ----- -------------------------------------------------------

 -- ----- -------------------------------------------------------------------------------------
 2. Click the line containing the record. The applicable window or dialog box displays.
 -- ----- -------------------------------------------------------------------------------------

 -- ----- -----------------------------------
 3. Make the necessary modifications.
 -- ----- -----------------------------------

 -- ----- -------------------------------------------
 4. Click **Save** or **Add**, as applicable.
 -- ----- -------------------------------------------

When editing an existing item, other options may be available, such as adding [Tasks](/itsm/release_mngmt/tasks_6_1.mdx){.MCTopicPopup .MCTopicPopupHotSpot .a}, [Comments](/discovery/comments_6_1.mdx){.MCTopicPopup .MCTopicPopupHotSpot .a}, and [Attachments](/discovery/attachments_6_1.mdx){.MCTopicPopup .MCTopicPopupHotSpot .a} and viewing [History](/common_topics/history_6_1.mdx){.MCTopicPopup .MCTopicPopupHotSpot .a}.

 --- ---------------------------------------------------------------------------------------
   The options change based upon the function; therefore, not every option may be shown.
 --- ---------------------------------------------------------------------------------------

[[[Closed]{.image .placeholder .MCDropDown_Image_Icon original-image-src="../../Skins/Default/Stylesheets/Images/transparent.gif" original-image-title="" height="12" width="16" BSM View](javascript:void(0)){.MCDropDownHotSpot .dropDownHotspot .MCDropDownHotSpot_ .MCHotSpotImage aria-expanded="false"}]{.MCDropDownHead .dropDownHead}

 --- --------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------
   Deleting is a permanent action and cannot be undone. Deleting may affect other functionality and information in the application such as data in configured reports, fields in windows, selectable options, etc. Therefore, be sure to understand the potential effects before making a deletion.
 --- --------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------

 -- ----- --------------------------------------------------
 1. Click the line that contains the item to delete.
 -- ----- --------------------------------------------------

 -- ----- ------------------------------------------------------------------------------------------------------------------------------------------------------
 2. From the *Select Actions* drop-down list, choose **Delete**. If a confirmation message is displayed, type **Delete** and click **Delete**to confirm.
 -- ----- ------------------------------------------------------------------------------------------------------------------------------------------------------
