# Discovered Items: Details

Use this function to view details for discovered items.

In the navigation pane, select **Discovery Scan > Discovered Items**. The Discovered Items window displays.

Select an existing record. A new window displays with the **Details** tab displayed, by default.

View the details configured for the currently selected record.
