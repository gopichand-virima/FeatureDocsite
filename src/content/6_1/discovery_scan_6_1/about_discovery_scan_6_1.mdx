# About Discovery Scan

This module contains the following functions:

::: Bullet1
[Dashboard](../Dashboards/Dashboard-Overview.mdx) contains a list of all the **recent** scans and any reports that have been added to the window. In this window, [new content](../Dashboards/Dashboard-Contents.mdx) can be added and [customized](../Dashboards/Dashboard-Customization.mdx).

- [AD User Import Logs](AD-User-Import-Logs.mdx) [displays scanned results and the details for imported AD users.]{style="color: #000000;"}
- [Azure AD User Import Logs](Azure-AD-User-Import-Logs.mdx) displays scanned results and the details for imported Azure AD user.
