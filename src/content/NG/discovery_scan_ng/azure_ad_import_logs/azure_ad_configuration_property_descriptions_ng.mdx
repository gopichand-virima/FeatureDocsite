# Azure AD Configuration Property Descriptions

 ------------------------------------------------ -------------------------------------------------------------------------------------------------------------------
 [Property]{.Strong} [Description]{.Strong}
 **Name** A user-defined label to identify this Azure AD configuration among multiple setups.
 **Client ID** The unique identifier assigned to your Azure application. Used by Azure AD to recognize the application.
 **Tenant ID** The unique identifier of your Azure Active Directory tenant. Specifies the directory where authentication occurs.
 **Secret Key** A confidential string generated in Azure, used as a password for application authentication.
 **Certificate** A digital certificate (with private key) uploaded to Azure, used as a secure authentication method for the app.
 ------------------------------------------------ -------------------------------------------------------------------------------------------------------------------
