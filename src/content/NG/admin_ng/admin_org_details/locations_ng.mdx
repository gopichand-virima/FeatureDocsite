# Locations

Use this function to associate users, facilities, or configuration items with a location.

In the main window, click **Admin \> Organizational Details \> Locations**. The Locations window displays.

![arrow border](../Resources/Images/arrow-border.PNG)

 1. From the *Select Actions* drop-down list, choose [New Location]{style="font-weight: bold;"}. The New Location window displays.

 -- ----- ------------------------------------------------------------
 2. Click **New Locations**. The New Location window displays.

 --- ---------------------------------------------------------------------------------------

To delete an existing location:

 -- ----- ---------------------------------------------------------------------------------------------
 1. Select the location to delete. from the *Select Actions* drop-down list, choose **Delete**.
 -- ----- ---------------------------------------------------------------------------------------------

 -- ----- -------------------------------------------------------------------------------------------------------------------------------------------------
 2. From the *Select Actions* drop-down list, choose **Delete**. If a confirmation message is displayed, type **Delete** and then click **Delete**.
 -- ----- -------------------------------------------------------------------------------------------------------------------------------------------------
