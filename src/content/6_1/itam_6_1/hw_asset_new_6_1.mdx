# New Hardware Asset

This feature enables to configure a Hardware Asset and its details in the application.

<div class="NumList1" madcap:autonum="1. ">

Choose New Hardware Asset option from Select Actions drop-down. The New Hardware Assets page displays.

</div>

<div class="Ind50">

<span class="image placeholder" original-image-src="../Resources/Images/ITAM/HW-asset-new.png" original-image-title="" style="width: 624px;height: 162px;"></span>

</div>

<div class="NumList2" madcap:autonum="2. ">

Enter the name of the Hardware in the Name box.

</div>

<div class="NumList2" madcap:autonum="3. ">

Enter additional information to describe hardware assets in the Description box.

</div>

<div class="NumList2" madcap:autonum="4. ">

Enter the total number of the hardware assets in the Quantity page.

</div>

<div class="NumList2" madcap:autonum="5. ">

Click the plus sign to associate the required Configuration Items to the corresponding hardware. Configuration Items window displays.

</div>

<div class="Ind50">

<span class="image placeholder" original-image-src="../Resources/Images/ITAM/HW-asset-new-config.png" original-image-title="" style="width: 471px;height: 209px;"></span>

</div>

<div class="NumList2" madcap:autonum="6. ">

Select required Blueprint from the drop down or enter search criteria, and then click.

</div>

<div class="NumList2" madcap:autonum="7. ">

Check to select the required blueprint.

</div>

<div class="NumList2" madcap:autonum="8. ">

Click Save.

</div>

Select the current state of the Asset from the State drop-down.

Department

</div>

Parent Asset

</div>

<div class="NumListb" madcap:autonum="h. ">

Supported By

</div>

<div class="NumList2" madcap:autonum="2. ">

Click Assigned On text box to select a date for assigning corresponding Hardware asset to the user /department.

</div>

<div class="NumList2" madcap:autonum="3. ">

Click Installed On text box to block a date to install corresponding Hardware.

</div>

<div class="NumList2" madcap:autonum="4. ">

Enter the name of the hardware item manufacturer company name in the Company box.

</div>

Depreciation:

Enter the depreciated amount of the hardware asset in the Depreciation box.

</div>

<div class="NumList2" madcap:autonum="2. ">

</div>

Contracts:

<div class="NumList1" madcap:autonum="1. ">

Click plus sign to select the required contracts for the corresponding Hardware Asset.

</div>

<div class="Ind50">

<span class="image placeholder" original-image-src="../Resources/Images/ITAM/HW-asset-new-contract.png" original-image-title="" style="width: 602px;height: 218px;"></span>

</div>

<div class="NumList2" madcap:autonum="2. ">

Enter text to search required contracts, and then click search icon.

</div>

<div class="NumList2" madcap:autonum="3. ">

Check to select a required contract.

</div>

<div class="NumList2" madcap:autonum="4. ">

Click Save to update the selected contract in the application.

</div>
