# Installing and Running the MAC Application

This function is not operational.

[The Mac application can perform ]{.span_1}[Unix System Discovery]{.span_2}[ and ]{.span_1}[Network Device Discovery]{.span_2}[ in a Network. ]{.span_1}

Once the application is downloaded, follow the below installation steps.

 -- ----- ------------------------------------------------------------------
 1. [Unzip ]{.span_1}[the **DiscoveryAgent_MAC.zip** file.]{.span_2}
 -- ----- ------------------------------------------------------------------

 -- ----- -------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------
 2. [On the terminal, ]{.span_1}run[ [[:]{style="font-size: 10pt;"}]{style="font-family: 'Courier New';"}]{.span_1}[[[sudo bash install.sh]{style="font-size: 10pt;"}]{style="font-family: 'Courier New';"}]{.span_4}
 -- ----- -------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------

 -- ----- ---------------------------------------------------------------------------------------------------------------------
 3. [Double-click ]{.span_1}[[virimamacagent.workflow]{style="font-family: 'Courier New'; font-size: 10pt;"}.]{.span_2}
 -- ----- ---------------------------------------------------------------------------------------------------------------------

 -- ----- ----------------------------------------------------------------------------------------------------------
 4. [In the *Service Installer dialog box*, click ]{.span_1}[Install]{.span_2}[.\
 ]{.span_1}

 -- ----- ----------------------------------------------------------------------------------------------------------

 -- ----- ------------------------------------------------------------------
 5. From the menu, select **Finder \> Services \> virimamacagent**.\

 -- ----- ------------------------------------------------------------------
