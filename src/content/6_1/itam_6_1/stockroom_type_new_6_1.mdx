# New Stockroom Type

This feature enables to creation new stockroom type with its details in the application.

<div class="NumList1" madcap:autonum="1. ">

Choose New Stockroom Type option from Select Actions drop down. The New Stockroom Type page displays.

</div>

<div class="Ind50">

<span class="image placeholder" original-image-src="../Resources/Images/ITAM/stockroom-type-new.png" original-image-title="" style="width: 793px;height: 193px;"></span>

</div>

<div class="NumList2" madcap:autonum="2. ">

Enter a name to identify the stockroom Type in the Display Name box.

</div>

<div class="NumList2" madcap:autonum="3. ">

Enter necessary information/definition of the corresponding Stockroom Type in the Description box.

</div>

<div class="NumList2" madcap:autonum="4. ">

Enter the value (price) of the stockroom type in the Value box.

</div>

<div class="NumList2" madcap:autonum="5. ">

Enter the priority of the stockroom type in the Priority box.

</div>

<div class="NumList2" madcap:autonum="6. ">

Check “Is Shipment Required” if the corresponding stockroom type require shipping to other location.

</div>

<div class="NumList2" madcap:autonum="7. ">

Check “Is External Stockroom” if the corresponding stockroom is from an external vendor/place.

</div>

<div class="NumList2" madcap:autonum="8. ">

Click Add to update Stockroom Type in the application.

</div>
