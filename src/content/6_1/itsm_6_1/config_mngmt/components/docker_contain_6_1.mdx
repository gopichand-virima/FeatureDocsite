# Docker Containers

 --- -------------------------------------------------------------------------------------------------------------
   This function is only available with certain asset types. Therefore, the relevant tab may not be displayed.
 --- -------------------------------------------------------------------------------------------------------------

 -- ----- ------------------------------------------------------
 1. While viewing an asset record, click **Components**.
 -- ----- ------------------------------------------------------

 -- ----- ------------------------------
 2. Click **Docker Containers**.
 -- ----- ------------------------------

::: Ind50
![docker containers new 472x140](../../../Resources/Images/Components/docker-containers-new_472x140.png)

## Add a New Docker Container

 -- ----- -----------------------------------------------------------
 1. Click **Add**. The Docker Containers dialog box displays.
 -- ----- -----------------------------------------------------------

::: Ind50
![docker containers add new 384x171](../../../Resources/Images/Components/docker-containers-add-new_384x171.png)

 -- ----- ---------------------------------
 2. Click **New Docker Container.**
 -- ----- ---------------------------------

 -- ----- ------------------------------
 3. Enter the field information.
 -- ----- ------------------------------

 -- ----- --------------------------------------------
 4. When all entries are made, click **Save**.
 -- ----- --------------------------------------------

## Modify Existing Docker Container

 -- ----- --------------------------------------
 1. Click **Existing Docker Container**.
 -- ----- --------------------------------------

::: Ind50
![docker container existing new 373x254](../../../Resources/Images/Components/docker-container-existing-new_373x254.png)

 -- ----- ----------------------------------------------------------------------------------------------------------------------------------------------------------------------
 2. Enter the search criteria to locate the existing device.  *See also* [Advanced Search](../../../Common%20Topics/Advanced-Search.mdx){style="font-style: italic;"}*.*
 -- ----- ----------------------------------------------------------------------------------------------------------------------------------------------------------------------

 -- ----- -----------------------------------------------------
 3. Select the applicable device from the results list.
 -- ----- -----------------------------------------------------

 -- ----- -----------------------------------------------
 4. When all selections are made, click **Save**.
 -- ----- -----------------------------------------------

## Delete Docker Container

While viewing the list of existing devices, select the device to delete, and click the **Delete** button.
