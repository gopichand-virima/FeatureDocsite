# Kubernetes Pods

 --- -------------------------------------------------------------------------------------------------------------
   This function is only available with certain asset types. Therefore, the relevant tab may not be displayed.
 --- -------------------------------------------------------------------------------------------------------------

 -- ----- ------------------------------------------------------
 1. While viewing an asset record, click **Components**.
 -- ----- ------------------------------------------------------

 -- ----- ----------------------------
 2. Click **Kubernetes Pods**.
 -- ----- ----------------------------

::: Ind50
![kubernetes pods new 463x122](../../../../Resources/Images/Components/kubernetes-pods-new_463x122.png)

![arrow border](../../../../Resources/Images/arrow-border.PNG)

[[[Closed]{.image .placeholder .MCDropDown_Image_Icon original-image-src="../../../../../Skins/Default/Stylesheets/Images/transparent.gif" original-image-title="" height="12" width="16" Kubernetes Pod](javascript:void(0)){.MCDropDownHotSpot .dropDownHotspot .MCDropDownHotSpot_ .MCHotSpotImage aria-expanded="false"}]{.MCDropDownHead .dropDownHead}

 -- ----- ---------------------------------------------------------
 1. Click **Add**. The Kubernetes Pods dialog box displays.
 -- ----- ---------------------------------------------------------

::: Ind50
![kubernetes pods add new 380x121](../../../../Resources/Images/Components/kubernetes-pods-add-new_380x121.png)

 -- ----- ------------------------------
 2. Click **New Kubernetes Pod**
 -- ----- ------------------------------

 -- ----- -------------------------------------------------
 3. Enter the applicable information in each field.
 -- ----- -------------------------------------------------

 -- ----- --------------------------------------------
 4. When all entries are made, click **Save.**
 -- ----- --------------------------------------------

[[[Closed]{.image .placeholder .MCDropDown_Image_Icon original-image-src="../../../../../Skins/Default/Stylesheets/Images/transparent.gif" original-image-title="" height="12" width="16" Kubernetes Pod](javascript:void(0)){.MCDropDownHotSpot .dropDownHotspot .MCDropDownHotSpot_ .MCHotSpotImage aria-expanded="false"}]{.MCDropDownHead .dropDownHead}

 -- ----- ------------------------------------
 1. Click **Existing Kubernetes Pod**.
 -- ----- ------------------------------------

::: Ind50
![kubernetes pods existing new 379x259](../../../../Resources/Images/Components/kubernetes-pods-existing-new_379x259.png)

 -- ----- ------------------------------------------------------------
 2. Enter the search criteria to locate the existing device*.*
 -- ----- ------------------------------------------------------------

 -- ----- -----------------------------------------------------
 3. Select the applicable device from the results list.
 -- ----- -----------------------------------------------------

 -- ----- -------------------------------------------------------
 4. When all selections/entries are made, click **Save**.
 -- ----- -------------------------------------------------------

[[[Closed]{.image .placeholder .MCDropDown_Image_Icon original-image-src="../../../../../Skins/Default/Stylesheets/Images/transparent.gif" original-image-title="" height="12" width="16" Kubernetes Pod](javascript:void(0)){.MCDropDownHotSpot .dropDownHotspot .MCDropDownHotSpot_ .MCHotSpotImage aria-expanded="false"}]{.MCDropDownHead .dropDownHead}

 --- --------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------
   Deleting is a permanent action and cannot be undone. Deleting may affect other functionality and information in the application such as data in configured reports, fields in windows, selectable options, etc. Therefore, be sure to understand the potential effects before making a deletion.
 --- --------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------

While viewing the list of existing records, select a record(s) to delete, and click the **Delete** button.
