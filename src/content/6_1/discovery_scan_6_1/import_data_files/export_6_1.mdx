# Export

To export uploaded file records,

1. Select the file(s) to export.

2. From the [Select Actions]{.Strong} menu, click [Export]{.Strong}.

 ![Import Data Files 1 26](../../Resources/Images/Discovery/Import-Data-Files/Import Data Files_1_26.png)

3. A confirmation dialog appears, indicating that the selected records will be exported to an Excel file.

 ![Import Data Files 1 27](../../Resources/Images/Discovery/Import-Data-Files/Import Data Files_1_27.png)

4. [Click ]{.inline-comment-marker}[Continue]{.Strong}[.]{.inline-comment-marker}

5. An email is sent to the requesting user's email address (the one currently signed in).

 1. [Download Link: ]{.Strong}The email contains a download link to a ZIP file with the exported data.

 2. The exported ZIP file follows this naming format:\
 ImportDataFiles\<hh.mm.AM/PM\>-\<MM.DD.YYYY\>.zip\
 (For example: ImportDataFiles10.15.AM-10.08.2025.zip)

 3. [Columns Included in the Export:]{.Strong}\
 The exported file contains the following columns:

 1. [File Name:]{.Strong} The name of the uploaded file (e.g., Import+CI\'s.xlsx)

 2. ****Status**:** The processing state of that upload (Processed, Processing)

 3. [Uploaded By:]{.Strong} The display name of the user who uploaded the file

 4. [Uploaded On:]{.Strong} The local timestamp when the file was uploaded (e.g., 10/08/2025, 10:15:47 AM)
