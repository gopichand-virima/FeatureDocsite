## **Delete/Remove**

Deletes the currently selected record or removes the displayed item.


|  |  |

| --- | --- |

| ![icon delete new 29x31](../Resources/Images/Common%20Topics/icon-delete-new_29x31.png) | Deletes the currently selected record or removes the displayed item. |


Deleting an item is a permanent action and cannot be undone. Deleting items may affect other functionality in the application such as data in configured reports, removal of fields in windows, etc. Therefore, be sure to understand the potential effects before deleting the item.
