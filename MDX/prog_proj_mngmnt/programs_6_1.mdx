# Programs

Use this function to manage several related projects.

In the main window, select **Program and Project Management** **> Programs**.

!

!

!


### New Program

- From the *Select Actions* drop-down list, choose **New Program**.
- Complete the fields, referring to the table below.
- When all selections/entries are made, click **Add**.

### Program Fields

| Field | Description |

| --- | --- |

| Name | A generic name given to the program. |

| Description | Give a brief description about the program. |

| Program Manager | The role responsible for overseeing the ongoing/ completion of the program. |

| Project Template | Associate the project template. |

| Status | Indicates the current state of the program. - **Critical**: Specifies that the program is critical in nature. - **At Risk**: Specifies that the program is associated with some risks. - **Normal**: Specifies that the program is not associated with risks. |

| State | Indicates the current state of the program. - **Not Started**: The default state of the program. - **Active**: Indicates that the program is still ongoing. - **Suspended**: Indicates that the program was suspended or canceled due to some reasons. - **Completed**: Indicates that the program is completed. |

| Program Start Date Program End Date | Dates on which the program starts and ends. |

| Cost Center | The business unit for which cost of the program is charged. |

| Percentage Completion | The percentage of the program that is completed. |

| Project | Associates a project(s) to a program. |


### Program Heat Map

Use this function to view the phase status of associated project(s). If a phase status is changed, or if a new phase has been added/deleted to the associated project, then the change gets updated.

While viewing the Details tab, click the **Program Heat Map** tab.

- The color of the phases indicates their respective status as shown in the legend.
- Click on the phase to change the status of the associated project directly from the program.
- If an associated project gets suspended, it won’t affect the status of the program.

### Edit Program

Not all functions include an option to edit record line items shown in the main window.


- Navigate to the window containing the record to edit.
- Click the line containing the record. The applicable window or dialog box displays.
- Make the necessary modifications.
- Click **Save** or **Add**, as applicable.

When editing an existing item, other options may be available, such as adding [Tasks](/itsm/release_mngmt/tasks_6_1.mdx), [Comments](/discovery/comments_6_1.mdx), and [Attachments](/discovery/attachments_6_1.mdx) and viewing [History](/common_topics/history_6_1.mdx).


The options change based upon the function; therefore, not every option may be shown.


### Delete Program

Deleting is a permanent action and cannot be undone. Deleting may affect other functionality and information in the application such as data in configured reports, fields in windows, selectable options, etc. Therefore, be sure to understand the potential effects before making a deletion.


- Click the line that contains the item to delete.
- From the *Select Actions* drop-down list, choose **Delete**. If a confirmation message is displayed, type **Delete** and click **Delete**to confirm.
