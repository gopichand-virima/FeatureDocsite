# Key Information Displayed

- <span class="Strong" style="font-weight: bold;">Name</span>: A user-defined name for the Azure AD configuration.

- <span class="Strong" style="font-weight: bold;">Triggered At</span>: The exact date and time when the import process started.

- <span class="Strong" style="font-weight: bold;">Completed At</span>: Timestamp showing when the import was successfully finished (blank if still <span class="Strong">in progress</span>).

- <span class="Strong" style="font-weight: bold;">Time Taken</span>: Total duration of the import job.

- <span class="Strong" style="font-weight: bold;">Edited Users</span>: Number of existing user records updated during this import.

- <span class="Strong" style="font-weight: bold;">New Users</span>: Number of users added as part of the import.

- <span class="Strong" style="font-weight: bold;">Total Users</span>: The complete number of user records processed.

- <span class="Strong" style="font-weight: bold;">Description</span>: A text field for system notes or administrator comments about the import (e.g., “Imported successfully” or “No changes detected”).
