# Viewing Meraki Import Record

Use this function to view the imported Meraki resources (Regardless of manual import or scheduled import).

1. Select an import record to open its details.

2. Report window displays imported CI(s) in these categories:

 - All

 - Authorized

 - Unauthorized

 - Logs

 ![Import Meraki Window 7](../../Resources/Images/Discovery/Import-From-Meraki/Import Meraki Window_7.png)

- **All** -- Every resource returned by the import.

- **Authorized** -- Assets that are already present in the CMDB.

 ![Import Meraki Window 8](../../Resources/Images/Discovery/Import-From-Meraki/Import Meraki Window_8.png)

- **Unauthorized** -- New assets not yet listed in the CMDB.

 ![Import Meraki Window 9](../../Resources/Images/Discovery/Import-From-Meraki/Import Meraki Window_9.png)

- **Logs**: The import run\'s messages (timings, successes, errors).
