# IPAM

Use this function to view IPAM records that have been scanned or downloaded from the InfoBlox Appliance.

In the main window, select **Discovery Scan > IPAM Networks**. The IPAM Networks window displays.

View the records displayed. Each *hyperlinked* record can be viewed by clicking on the *IP Address* or *Network ID*.


To view or edit the information, click an applicable record in the list.


Edit IPAM Records


The IPAM data displayed in the list is scanned (or pulled) data which cannot be edited.
