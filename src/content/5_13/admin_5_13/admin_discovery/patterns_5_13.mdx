# Patterns

Use this function to configure patterns for application identification during discovery.

In the main window, click **Admin > Discovery > Patterns**. The Patterns window displays.

From the *Select Actions* drop-down list, choose an action: **New Pattern**, [Delete](/common_topics/delete_remove_5_13.mdx), or [Export](/discovery/import_data_files/export_5_13.mdx).
