# <span class="Strong">Authorized Tab</span>

- Shows CI(s) that match existing entries in the CMDB - Correlator Match.

- Fields in this view:

  - <span class="Strong" style="font-weight: bold;">Blueprint Name</span>: Name of the Blueprint.

  - <span class="Strong" style="font-weight: bold;">CI</span>: CMDB CI ID matching the imported CI.

  - <span class="Strong" style="font-weight: bold;">Asset Name</span>: Provided Asset name.

  - <span class="Strong" style="font-weight: bold;">Asset ID</span>: New Asset ID is created with prefix of “IMP” to denote that this record was imported. The format is “IMP” + 6 digits (auto generated sequential number.

  - <span class="Strong" style="font-weight: bold;">Status</span><span class="inline-comment-marker">: Indicates the current status of the imported asset.</span>

    - <span class="Strong">What it is:</span> CI lifecycle state (e.g., <span class="Emphasis">In Use, Retired, Decommissioned</span>).

    - <span class="Strong">Where it comes from:</span> The Status dropdown is <span class="Strong">driven by</span> <span class="Strong">Admin → SACM → CMDB Properties → Status</span>. The same picklist is used by <span class="Strong">Business Rules</span>.

  - <span class="Strong" style="font-weight: bold;">Host Name</span><span class="inline-comment-marker">: The hostname of the asset or device.</span>

  - <span class="Strong" style="font-weight: bold;">Terminal ID</span><span class="inline-comment-marker">: Optional, stored exactly as provided; not used for correlation unless your blueprint explicitly defines it.</span>

  - <span class="Strong" style="font-weight: bold;">Operating System</span><span class="inline-comment-marker">: The operating system detected or specified for the asset.</span>

  - <span class="Strong" style="font-weight: bold;">Missing Components</span><span class="inline-comment-marker">: Lists any components that are missing from the imported asset.</span>

    - <span class="Strong">Missing Components:</span> <span class="Emphasis">Not applicable</span> for <span class="Strong">Imported Items</span>. We don’t import component lists, so this field is <span class="Strong">left blank</span> (or treated as <span class="Strong">N/A</span>) on import. It will only be evaluated <span class="Strong">after</span> a CI is <span class="Strong">moved to CMDB</span> and the <span class="Strong">Process-Missing-Components</span> job runs.

# <span class="Strong">Layout and Sections</span>

1.  <span class="Strong" style="font-weight: bold;">Left Sidebar</span>

- <span class="Strong" style="font-weight: bold;">Primary Details:</span> Shows key identifiers and metadata for the asset, including:

  - Asset ID (e.g., IMP001255)

  - Blueprint (e.g., Windows Server)

  - Created On and Last Modified On timestamps

  - Update To (which refers to the Asset ID from CMDB associated with this record).

  - Tags if any (with an editable pencil icon)

- <span class="Strong">**Owner**:</span> The system assigns the logged-in user as the CI owner when importing a CI.

- <span class="Strong" style="font-weight: bold;">Import Details:</span> Information about the import process, such as:

  - File Name (e.g., Import+CI's.xlsx)

  - Status (e.g., Processed)

  - Uploaded By (e.g., Lakshmi Virima)

  - Uploaded On (timestamp)

  - Update To (again referencing AST003156)

2.  <span class="Strong" style="font-weight: bold;">Main Content Area</span>

- <span class="Strong">**Header**:</span> Displays the asset identifier (e.g., IMP001255).

- <span class="Strong" style="font-weight: bold;">Action Buttons:</span>

  - <span class="Strong" style="font-weight: bold;">Move to CMDB</span> (to promote the asset to the configuration database)

  - <span class="Strong" style="font-weight: bold;">Rescan Now</span> (to trigger a new scan of the asset)

- <span class="Strong" style="font-weight: bold;">Tabbed Navigation:</span> Provides access to different information categories:

  - Details (active tab)

  - Components

  - Log On Events

  - Tasks

  - Comments

  - Attachments

  - Updates

3.  <span class="Strong" style="font-weight: bold;">Details Tab (Main Content)</span>

- <span class="Strong" style="font-weight: bold;">Asset Primary Information:</span>

  - Asset ID, Host Name, Asset Name

- <span class="Strong" style="font-weight: bold;">Hardware and Network:</span>

  - IP Address

- <span class="Strong" style="font-weight: bold;">Life Cycle Information:</span>

  - Acquisition Date (repeated, possibly for multiple records or a display issue)

- <span class="Strong" style="font-weight: bold;">Network Adapter Details:</span>

  - MAC Address
