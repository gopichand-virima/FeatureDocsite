# Request for Changes

Use this function to submit a formal proposal for implementing a change. The Request for Change (RFC) provides the details for this change.

In the navigation pane, select **ITSM > Change Management > Request for Changes**. The Request for Changes window displays.

!

!

!


### New Request for Change

- From the *Select Actions* drop-down list, choose **New RFC**. The New RFC window displays.

!

- Click **New**. The New RFC window displays.

!

- Enter a *Title* and *Description*.
- In the following fields, click **Addthe plus + sign**, then search for and select the applicable item:
- Associate CI
- Associate Problems
- Associate Known Errors
- In the following fields, click the drop-down list and select the applicable choice:
- CI Category
- CI SubCategory
- Urgency
- State
- Approval State
- When all selections/entries are made, click **Add**.

### Time Tracking

Use this function to add a record/task or generate a time spent report.

While viewing the Details window, select **Time Tracking**. The Time Tracking window displays.

!

#### Add Record or Tasks

- To add a new item, click **Add**. The Add Time Tracking dialog box displays.

! !

- Select either **Record** or **Tasks**.
- If **Record** is selected, In the *User* field, click **Add**, then search for and select the applicable user.
- If **Record** is selected, In the *User* field, click the **plus + sign**, then search for and select the applicable user.
- If **Tasks** is selected, in the *Tasks* field, click the drop-down list and select the applicable task. (This is a task already created for this change request under the Tasks tab.)
- Enter the *Date* and *Time* spent.
- Enter a *Description*.
- When all selections/entries are made, click **Add**.

#### Time Spent Report

To generate a report for the time spent on this change, click **Time Spent Report**. A new window opens and displays the applicable time spent, sorted by User.

!


### Edit Request for Change

Not all functions include an option to edit record line items shown in the main window.


- Navigate to the window containing the record to edit.
- Click the line containing the record. The applicable window or dialog box displays.
- Make the necessary modifications.
- Click **Save** or **Add**, as applicable.

When editing an existing item, other options may be available, such as adding [Tasks](/itsm/release_mngmt/tasks_6_1.mdx), [Comments](/discovery/comments_6_1.mdx), and [Attachments](/discovery/attachments_6_1.mdx) and viewing [History](/common_topics/history_6_1.mdx).


The options change based upon the function; therefore, not every option may be shown.


### Delete Request for Change

Deleting is a permanent action and cannot be undone. Deleting may affect other functionality and information in the application such as data in configured reports, fields in windows, selectable options, etc. Therefore, be sure to understand the potential effects before making a deletion.


- Click the line that contains the item to delete.
- From the *Select Actions* drop-down list, choose **Delete**. If a confirmation message is displayed, type **Delete** and click **Delete**to confirm.
