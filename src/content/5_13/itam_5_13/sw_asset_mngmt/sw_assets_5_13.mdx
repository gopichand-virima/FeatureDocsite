# Software Assets

Use this function to add and edit software assets. There are two components to this function: (1) adding a software asset group, and (2) adding the software asset itself.

In the navigation pane, select **ITAM > Software Asset Management > Software Assets.** The window displays the list of all configured software assets.

Click the radio button for the preferred display type: **By Group** or **By Software**.

! !


The *Ungrouped Software* item cannot be deleted.


New Software Asset By Group

Click the **By Group** radio button.

From the *Select Actions* drop-down list, choose **New Software Group**.

The *New Software Group* window displays.

Enter a unique, alphanumeric *Group Name*.

Click **Add**, then search for and select the applicable *Vendor* and *Software License Key*.


Both a Vendor and Software License Key must be configured in order to choose them from the list. *Refer to* [Vendor Management](/itam/vendor_mngmnt_5_13.mdx) *and* [Software License Key](/itam/sw_asset_mngmt/sw_license_key_5_13.mdx)*.*


Click **Add**.

New Software Asset By Software

Click the **By Software** radio button.

From the *Select Actions* drop down, click **New**. The New dialog box displays.  

Enter an *Asset Name*, *Software Name*, and *Manufacturer*.

In the *Group* field, click the drop-down list and select the applicable group.


There must be an asset Software Group created in order to have it available in the list (see previous section). Selecting a *Group* is not required.


Click **Add**.

When returned to the list of software assets, select the asset just added. A new window displays in which to specify more information about this software asset, such as the software license key, components, relationships, SLA, etc. *Refer to Software Asset Details.*

Edit Software Asset

In the Details window, click **Related Software** to view a list of all software related to this software asset.

Delete Software Asset Import Software Asset

Click the **By Software** radio button.

From the *Select Actions* drop down, click **Import**. The New dialog box displays.

Click *Click here to download the sample file* to download the sample file.

Click **Browse** and select the file to import.

Click **Import**.
