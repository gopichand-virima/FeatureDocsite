# Import Templates

The Import Templates page serves as the central location where predefined templates for Configuration Item (CI) data imports are managed and displayed. These templates act as structured formats that guide users in preparing data files for uploading assets into the Configuration Management Database (CMDB).

Use this function to view the data as provided on an imported Excel spreadsheet.


You cannot modify this information.


- On the main window, click **Admin > Discovery > Import Templates**. The Import Templates window displays.

!

- Column description:  
  **Id**: Unique template identifier  
  **Name**: Template name (e.g., "Azure VMs", "Stg Srvr")  
  **Created On**: Date of creation.  

- From the *Select Actions* drop-down list, choose either [Delete](/common_topics/delete_remove_6_1.mdx) or [Export](/discovery/import_data_files/export_6_1.mdx).
