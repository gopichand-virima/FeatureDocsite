# Incident Category

Use this function to create and manage incident categories that help classify and organize incidents.

In the main window, click **Admin > Incident Management > Incident Category**. The Incident Category window displays.

## New Incident Category

From the *Select Actions* drop-down list, choose **New Incident Category**. The New Incident Category window displays.

Enter a *Name* and *Description* for this category.

Select a **Parent Category** if this is a sub-category (optional).

To add an image/icon for this category:

- Click **Upload**
- Search for and select an image
- Click **OK**

When all selections/entries are made, click **Add**.

## Edit Incident Category

Navigate to the window containing the record to edit.

Click the line containing the record. The applicable window displays.

Make the necessary modifications.

Click **Save**.

## Delete Incident Category

Deleting is a permanent action and cannot be undone. Ensure no incidents are currently using this category before deletion.

Click the line that contains the item to delete.

From the *Select Actions* drop-down list, choose **Delete**. If a confirmation message is displayed, take the appropriate action.
