# Network Adapters

 --- -------------------------------------------------------------------------------------------------------------
   This function is only available with certain asset types. Therefore, the relevant tab may not be displayed.
 --- -------------------------------------------------------------------------------------------------------------

 -- ----- ------------------------------------------------------
 1. While viewing an asset record, click **Components**.
 -- ----- ------------------------------------------------------

 -- ----- -----------------------------
 2. Click **Network Adapters**.
 -- ----- -----------------------------

::: Ind50
![network adapters new 671x161](../../../Resources/Images/Components/network-adapters-new_671x161.png)

[[[Closed]{.image .placeholder .MCDropDown_Image_Icon original-image-src="../../../../Skins/Default/Stylesheets/Images/transparent.gif" original-image-title="" height="12" width="16" a New Network Adapter](javascript:void(0)){.MCDropDownHotSpot .dropDownHotspot .MCDropDownHotSpot_ .MCHotSpotImage aria-expanded="false"}]{.MCDropDownHead .dropDownHead}

 -- ----- ---------------------------------------------------------
 1. Click **Add**. The Network Adapter dialog box displays.
 -- ----- ---------------------------------------------------------

[[[Closed]{.image .placeholder .MCDropDown_Image_Icon original-image-src="../../../../Skins/Default/Stylesheets/Images/transparent.gif" original-image-title="" height="12" width="16" Existing Network Adapter](javascript:void(0)){.MCDropDownHotSpot .dropDownHotspot .MCDropDownHotSpot_ .MCHotSpotImage aria-expanded="false"}]{.MCDropDownHead .dropDownHead}

 -- ----- -------------------------------------
 1. Click **Existing Network Adapter**.
 -- ----- -------------------------------------

::: {.Ind50
![network adapter modify existing 452x154](../../../Resources/Images/Discovery/network-adapter-modify-existing_452x154.png)

::: {.Ind50
![network adapters new existing 458x310](../../../Resources/Images/Components/network-adapters-new-existing_458x310.png)

 -- ----- -----------------------------------------------------------------------------------------------------------------------------------------------------------------------
 2. Enter the search criteria to locate the existing adapter.  *See also* [Advanced Search](../../../Common%20Topics/Advanced-Search.mdx){style="font-style: italic;"}*.*
 -- ----- -----------------------------------------------------------------------------------------------------------------------------------------------------------------------

 -- ----- ------------------------------------------------------
 3. Select the applicable adapter from the results list.
 -- ----- ------------------------------------------------------

 -- ----- -----------------------------------------------
 4. When all selections are made, click **Save**.
 -- ----- -----------------------------------------------

[[[Closed]{.image .placeholder .MCDropDown_Image_Icon original-image-src="../../../../Skins/Default/Stylesheets/Images/transparent.gif" original-image-title="" height="12" width="16" the Network Adapter Details](javascript:void(0)){.MCDropDownHotSpot .dropDownHotspot .MCDropDownHotSpot_ .MCHotSpotImage aria-expanded="false"}]{.MCDropDownHead .dropDownHead}

 -- ----- ------------------------------------------------------------------------------------------------------------------------------------------
 1. While viewing the list of existing adapters, select the adapter to edit. The windows refreshes and the details for the adapter displays.
 -- ----- ------------------------------------------------------------------------------------------------------------------------------------------

 -- ----- ---------------------------------------------------------------------------------------------------------------------------------------------------------------------------
 2. Make any necessary changes to the existing information. The changes are automatically saved. *Refer to* [CI Components](CI-Comp-List.mdx){style="font-style: italic;"}*.*
 -- ----- ---------------------------------------------------------------------------------------------------------------------------------------------------------------------------

 -- ----- ---------------------------------------------------------------------------------------------------------
 3. To add a contact, navigate to the Contacts section and click **Add**. The Contacts dialog box displays.
 -- ----- ---------------------------------------------------------------------------------------------------------

 --- --------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------
   Deleting is a permanent action and cannot be undone. Deleting may affect other functionality and information in the application such as data in configured reports, fields in windows, selectable options, etc. Therefore, be sure to understand the potential effects before making a deletion.
 --- --------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------

 -- ----- --------------------------------------------------
 1. Click the line that contains the item to delete.
 -- ----- --------------------------------------------------

 -- ----- ------------------------------------------------------------------------------------------------------------------------------------------------------
 2. From the *Select Actions* drop-down list, choose **Delete**. If a confirmation message is displayed, type **Delete** and click **Delete**to confirm.
 -- ----- ------------------------------------------------------------------------------------------------------------------------------------------------------
