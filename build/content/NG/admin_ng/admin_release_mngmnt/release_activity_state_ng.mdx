# Release Activity State

Use this function to add a new state defining the status of the activity being worked by an assignee.

In the main window, click **Admin > Release Management > Release Activity State**. The Release Activity State window displays.

New Release Activity State

From the *Select Actions* drop-down list, choose **New Release Activity State**. The New Release Activity State window displays.

Enter a *Name* and *Description*.

In the *State* field, click the drop-down list and select the applicable state.

Click **Add**.
