# <span class="Strong">Re-scan</span>

- Select one or more checkboxes

  - Only CI(s) with IP address can be rescanned AST006004.

- Click on <span class="Strong">Select Actions</span> and then Select <span class="Strong">Re-Scan.</span>

- A pop-up window shall be displayed as below

  - Choose the discovery application (Client) which will execute the scan.

    - If not selected and clicked on <span class="Strong">Scan</span>, an error <span class="Strong">Please select a client</span> is displayed.

  - Choose the type of scan

    - If not selected and clicked on <span class="Strong">Scan</span>, an error <span class="Strong">Please select a scan type </span>is displayed.

  - Click on Scan

    - A pop up window showing a scan is in progress is displayed for few seconds and user is redirected to <span class="Strong">Recent Scans </span>page.
