# Viewing AZURE Import Record

Use this function to view the imported AZURE resources (Regardless of manual import or scheduled import).

1.  Select an import record to open its details.

2.  Report window displays imported CI(s) in these categories:

    - All

    - Authorized

    - Unauthorized

    <span class="image placeholder" original-image-src="../../Resources/Images/Discovery/Import-From-Azure/Import%20From%20AZURE_1_6.png" original-image-title=""></span>

- <span class="Strong">All</span> – Every resource returned by the import.

- <span class="Strong">Authorized</span> – Assets that are already present in the CMDB.

  <span class="image placeholder" original-image-src="../../Resources/Images/Discovery/Import-From-Azure/Import%20From%20AZURE_1_7.png" original-image-title=""></span>

<!-- -->

- <span class="Strong">Unauthorized</span> – New assets not yet listed in the CMDB.

  <span class="image placeholder" original-image-src="../../Resources/Images/Discovery/Import-From-Azure/Import%20From%20AZURE_1_8.png" original-image-title=""></span>
