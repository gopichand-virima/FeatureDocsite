# Comments

Add comments and notes to records for collaboration and documentation purposes.

## Overview

The Comments function enables users to add contextual notes, updates, and discussions to various records within Virima. Comments facilitate team collaboration and provide an audit trail of communications related to specific items.

## Adding Comments

### Create a Comment
1. Navigate to the record where you want to add a comment
2. Locate the **Comments** section or tab
3. Click in the comment text field
4. Type your comment
5. Click **Add Comment** or **Post** to save

### Comment Formatting
- Use rich text formatting for emphasis
- Add bullet points and numbered lists
- Include hyperlinks to related resources
- Mention other users with @ notation

## Managing Comments

### View Comments
- Comments display in chronological order
- See comment author and timestamp
- View complete comment history
- Filter comments by date or author

### Edit Comments
1. Locate your comment in the comments list
2. Click the **Edit** icon
3. Make your changes
4. Click **Save** to update

### Delete Comments
1. Find the comment you want to remove
2. Click the **Delete** icon
3. Confirm deletion when prompted

> **Note**: You can only edit or delete your own comments unless you have administrator privileges.

## Comment Features

### @Mentions
- Tag other users in comments using @username
- Mentioned users receive notifications
- Creates links to user profiles

### Notifications
- Receive alerts when comments are added to records you follow
- Get notified when you're mentioned in a comment
- Configure notification preferences in settings

### Comment Privacy
- Mark comments as internal or public
- Control visibility based on user roles
- Add private notes for personal reference

## Best Practices

- Be clear and concise in your comments
- Use professional language
- Tag relevant team members for visibility
- Include action items when appropriate
- Review comments before posting
- Update comments if information changes
- Use comments to document decisions and rationale

## Comment History

All comments are tracked and timestamped:
- View who added each comment
- See when comments were posted
- Access edit history for modified comments
- Export comments for reporting purposes
