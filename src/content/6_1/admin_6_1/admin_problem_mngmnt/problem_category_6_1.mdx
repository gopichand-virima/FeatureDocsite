# Problem Category

Use this function to configure a category for the change being associated, such as CI, Application Software, Hardware or any aspect of the business that a change affects.

In the main window, click **Admin > Problem Management > Problem Category**. The Problem Category window displays.

!

!


### New Problem Category

- From the *Select Actions* drop-down list, choose **New Problem Category**. The New Problem Category window displays.

!

- Enter a *Name* and *Description* for this category.
- To upload an image, next to *Category image*, click **Upload**, search for and select an image, then click **OK**.
- When all selections/entries are made, click **Add**.


### Add a Problem SubCategory

- Click the line item to open the category type window.
- Click **SubCategory**. A list of all existing subcategories displays.

!

- Click **New Problem SubCategory**.
- Enter a *Name* and *Description*.
- When all entries are made, click **Add**.
- To delete the SubCategory, click the **Delete** button.


In the SubCategory window, you can also personalize the columns. save filters, export the details, or delete the information (see warning about deleting).


Deleting is a permanent action and cannot be undone. Deleting may affect other functionality and information in the application such as data in configured reports, fields in windows, selectable options, etc. Therefore, be sure to understand the potential effects before making a deletion.


### Edit Problem Category

Not all functions include an option to edit record line items shown in the main window.


- Navigate to the window containing the record to edit.
- Click the line containing the record. The applicable window or dialog box displays.
- Make the necessary modifications.
- Click **Save** or **Add**, as applicable.


### Delete Problem Category

Deleting is a permanent action and cannot be undone. Deleting may affect other functionality and information in the application such as data in configured reports, fields in windows, selectable options, etc. Therefore, be sure to understand the potential effects before making a deletion.


- Click the line that contains the item to delete.
- From the *Select Actions* drop-down list, choose **Delete**. If a confirmation message is displayed, type **Delete** and click **Delete**to confirm.
