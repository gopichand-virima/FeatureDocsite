# AD Configuration and Import

Active Directory (AD) is a Microsoft directory service for Windows domain networks. It authenticates and authorizes users and computers, enforces security policies, and manages software installations and updates.

In the main window, click [Admin \> Users \> AD Configuration]{.Strong}. **AD Configuration** window displays.

::: step-media
![AD User Import Logs 1](../../Resources/Images/Discovery/AD-User-Import-Logs/AD User Import Logs_1.png)

1. From the **Select Actions** drop-down list, choose [New AD Configuration.]{.Strong}

 ::: step-media
 ![AD User Import Logs 1 1](../../Resources/Images/Discovery/AD-User-Import-Logs/AD User Import Logs_1_1.png)
 
2. On the AD Configuration tab, fill in the values for the attributes.

 ::: step-media
 ![AD User Import Logs 1 2](../../Resources/Images/Discovery/AD-User-Import-Logs/AD User Import Logs_1_2.png)
