# [Background Information on Scans]{style="background-color: #ffffff;"}

There are three components related to scanning:

::: Bullet1
[Probes ]{.span_2}[executes a set of scripts on the target hosts to retrieve information. Each host type will have a different probe.]{.span_1}

::: Bullet1
[**Probe Workflows** executes a]{.span_2}[ script when a scan is created. The workflow calls the set of the probes required to discover the information about the hosts on which the scan is performed.]{.span_1}

::: Bullet1
[Sensors. ]{.span_2}[Scripts that are applied to the output of the respective probes. The sensors give the data ]{.span_1}[JSON]{.span_2}[ output.]{.span_1}

The basic work flow is as follows:

::: Bullet1
Find up and running Hosts.

::: Bullet1
Get Host Types for each host.

::: Bullet2
[If Host Type is ]{.span_1}[**Unix**, find the Unix OS flavor and, depending on the OS, send the probes to the Discovery Server, which retrieves the information of the host.]{.span_2}

::: Bullet2
[If Host Type is ]{.span_1}[Windows]{.span_2}[ or ]{.span_1}[**Network device**, send probes to the the Discovery Server that retrieves the information of the host.]{.span_2}

[For discovering Windows machines in a workgroup, the admin shared folder should be remotely accessible. Enable this folder by following these steps:]{.span_1}

 -- ----- ----------------------------------------
 1. [Run ]{.span_1}[**regedit**.]{.span_6}
 -- ----- ----------------------------------------

 -- ----- --------------------------------------------------------------------------------------------------------------------------------------------------------------
 2. [Navigate to this directory]{.span_1}[:\
 ]{.span_2}[HKEY_LOCAL_MACHINE\\SOFTWARE\\Microsoft\\Windows\\CurrentVersion\\Policies\\System]{.span_6}

 -- ----- --------------------------------------------------------------------------------------------------------------------------------------------------------------

 -- ----- -------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------
 3. [Right-click on the right pane and choose ]{.span_1}[New]{.span_2}[ \> ]{.span_1}[DWORD]{.span_2}[** (32-bit) Value**.]{.span_1}
 -- ----- -------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------

 -- ----- ----------------------------------------------------------------------------------------------------------------------------------------
 4. [Fill ]{.span_1}[LocalAccountTokenFilterPolicy]{.span_6}[ for the Value name and press **Enter**.]{.span_1}
 -- ----- ----------------------------------------------------------------------------------------------------------------------------------------

 -- ----- ------------------------------------------------------------------------------------------------------------------------------------------------------------
 5. [Double-click ]{.span_1}[LocalAccountTokenFilterPolicy]{.span_6}[ value and modify the "Value Data" from "0" to "1\".]{.span_1}
 -- ----- ------------------------------------------------------------------------------------------------------------------------------------------------------------

 -- ----- -----------------------------------------------------------------------
 6. [Click ]{.span_1}[OK]{.span_2}[.]{.span_1}
 -- ----- -----------------------------------------------------------------------
