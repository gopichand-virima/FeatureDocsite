# Incident Model

Use this function to configure and populate Incident  Models that can be reused for defining Incidents. It can be used to define commonly occurring incidents raised by Service Desk Members, thus enabling to skip and reduce the time required to complete forms.

|     |                                                                    |
|-----|--------------------------------------------------------------------|
|     | Incident Models can be configured with relevant Role based access. |

In the main window, click **Admin /> Incident Management /> Incident Model**. The Incident Model window displays.
