# Knowledge Categories

Knowledge categories organize articles into logical groups making them easier to browse and find.

## Standard Categories

### Hardware
Articles about physical equipment:
- Desktops and laptops
- Servers
- Printers
- Mobile devices
- Peripherals
- Network equipment

### Software
Application-related articles:
- Operating systems
- Business applications
- Productivity tools
- Custom applications
- Software installation
- Licensing

### Network
Network and connectivity:
- Wi-Fi connectivity
- VPN access
- Network drives
- Internet access
- Email configuration
- Remote access

### Security
Security-related topics:
- Password management
- Access controls
- Security policies
- Phishing awareness
- Data protection
- Compliance

### Procedures
Process documentation:
- Standard procedures
- Policy guidelines
- Workflow instructions
- Compliance procedures
- Best practices

### Troubleshooting
Problem resolution:
- Common errors
- System issues
- Performance problems
- Error messages
- Diagnostic steps

## Creating Categories

### Add New Category
1. Navigate to **Admin** > **Knowledge Management**
2. Select **Categories**
3. Click **New Category**
4. Enter category details:
   - Name
   - Description
   - Parent category (if subcategory)
   - Icon (optional)
5. Set permissions
6. Save

### Subcategories
Organize within main categories:

**Example Structure:**
```
Hardware
├── Desktops
│   ├── Windows PCs
│   ├── Mac Computers
│   └── Thin Clients
├── Laptops
│   ├── Windows Laptops
│   └── MacBooks
├── Printers
│   ├── Network Printers
│   └── Local Printers
└── Mobile Devices
    ├── Smartphones
    └── Tablets
```

## Managing Categories

### Edit Category
1. Select category
2. Click **Edit**
3. Update details
4. Save changes

### Move Articles
Recategorize articles:
1. Select articles
2. Click **Move**
3. Choose new category
4. Confirm move

### Merge Categories
Combine similar categories:
1. Select categories to merge
2. Choose target category
3. Move all articles
4. Delete empty category

### Delete Category
Remove unused categories:
1. Ensure category is empty
2. Move articles if needed
3. Select category
4. Click **Delete**
5. Confirm deletion

## Category Permissions

### View Permissions
Control who can see:
- All users
- Specific groups
- IT staff only
- Management

### Contribute Permissions
Who can create articles:
- Knowledge contributors
- Subject matter experts
- Specific teams
- All staff

## Category Analytics

### Usage Metrics
Track category performance:
- Articles per category
- Views by category
- Search hits
- Most active categories

### Gap Analysis
Identify needs:
- Categories with few articles
- High search, low content
- Emerging topics
- Knowledge gaps

## Best Practices

### Logical Organization
- Clear hierarchy
- Intuitive structure
- Consistent naming
- Limit depth (3-4 levels max)

### User-Friendly
- Use familiar terms
- Avoid jargon
- Clear descriptions
- Logical grouping

### Maintenance
- Regular review
- Merge similar categories
- Remove unused
- Update descriptions

### Balance
- Not too many categories
- Not too few
- Right level of detail
- Flexible for growth
