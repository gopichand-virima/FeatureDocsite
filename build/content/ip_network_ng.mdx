# IP Network

Use this function to define IP networks for discovery scans.

In the main window, click **Admin > Discovery > IP Network**. The IP Network window displays.

## New IP Network

From the *Select Actions* drop-down list, choose **New IP Network**. The New IP Network window displays.

Enter a *Name* and *Description* for this network.

Enter the network details:

- **Network Address**: Enter the network IP address (e.g., 192.168.1.0)
- **Subnet Mask**: Enter the subnet mask (e.g., 255.255.255.0) or CIDR notation (e.g., /24)
- **Gateway**: (Optional) Enter the default gateway
- **VLAN**: (Optional) Enter the VLAN ID

Associate this network with a location if applicable.

When all selections/entries are made, click **Add**.

## Edit IP Network

Navigate to the window containing the record to edit.

Click the line containing the record. The applicable window displays.

Make the necessary modifications.

Click **Save**.

## Delete IP Network

Click the line that contains the item to delete.

From the *Select Actions* drop-down list, choose **Delete**.
