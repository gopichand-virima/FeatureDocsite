# Release Management: Dashboard

Use this function to view different types of data. The dashboard can be customized.

In the navigation pane, select **Release Management > Dashboard**. The Release Management Dashboard window displays.

Click **View All** to open the applicable functions for the selected *items*.

Click **Show All** to open the applicable function for the selected *record*.

To add content to the dashboard, click [Add Contents](/my_dashboard/dashboard_add_contents_new_5_13.mdx).
