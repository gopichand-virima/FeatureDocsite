# New Major Software

Use this function to add information for a new software.

<div class="numlist1" madcap:conditions="Default.OldUI" madcap:autonum="1. ">

In the main window, select **Admin /> Discovery /> Major Software**. The Major Software window displays.

</div>

<div class="NumList2" madcap:conditions="Default.NewUI" madcap:autonum="2. ">

In the main window, click the gear icon and select **Discovery /> Major Software**.

</div>

<div class="NumList2" madcap:conditions="Default.OldUI" madcap:autonum="3. ">

From the *Select Actions* drop-down list, choose **New Major Software**. A new window displays with the **Details** tab selected.

</div>

<div class="NumList2" madcap:conditions="Default.NewUI" madcap:autonum="4. ">

Click the **New Major Software** button. A new window displays with the **Details** tab selected.

</div>

<div class="NumList2" madcap:autonum="5. ">

Complete the tabs according to the following:

</div>

<div class="Bullet2">

<a href="Major-Software-Details.htm" target="_popup">Details</a>

</div>

<div class="Bullet2">

<a href="../Common%20Topics/Tasks.htm" target="_popup">Tasks</a>

</div>

<div class="Bullet2">

<a href="../Common%20Topics/Comments.htm" target="_popup">Comments</a>

</div>

<div class="Bullet2">

<a href="../Common%20Topics/Attachments.htm" target="_popup">Attachments</a>

</div>

<div class="NumList2" madcap:autonum="6. ">

When all selections/entries are made, click **Add**.

</div>
