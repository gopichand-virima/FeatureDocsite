# Accessing Recent Scans

1. From the left navigation pane, go to **Discovery Scan > Recent Scans**.

2. The **Recent Scans** page opens.

1. This page contains a list of all scans that have been executed

**Key Information Displayed:**

- **Scan Name** – The name or ID assigned to each scan run.
- **Scan Type** – The type of scan performed (e.g., Deep Host Scan, Basic Host Scan, Network Scan).
- **Status** – Indicates the scan's current state: Running, Completed, or Failed.
- **Client** – The discovery server or agent that performed the scan.
- **Started On** – Timestamp showing when the scan was initiated.
- **Completed On** – Timestamp showing when the scan finished (empty if still in progress).
- **Duration** – The amount of time the scan took from start to finish.
- **Schedule** – Indicates whether this scan was a one-time run (On Demand) or part of a Scheduled Scan job.
