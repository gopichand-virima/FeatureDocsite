# Customizing Columns

1. In the [AD User Import Logs]{.Strong} page, click the [Personalize Column icon]{.Strong} in the top-right corner.

 ![AD User Import Logs 1 14](../../Resources/Images/Discovery/AD-User-Import-Logs/AD User Import Logs_1_14.png)

2. The [Personalize List Columns]{.Strong} window will appear.

 - The [left panel]{.Strong} shows available fields.

 - The [right panel]{.Strong} shows currently displayed fields.

3. Check or uncheck fields as needed.

4. Click [Save]{.Strong} to apply changes.

 - [Default View]{.Strong} resets columns to system default.

 - [Clear All]{.Strong} removes all selections for a fresh start.

 ![AD User Import Logs 1 15](../../Resources/Images/Discovery/AD-User-Import-Logs/AD User Import Logs_1_15.png)
