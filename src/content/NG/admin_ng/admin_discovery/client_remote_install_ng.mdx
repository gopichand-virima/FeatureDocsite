# Remote Install

Use this function to remotely install a Discovery Agent.

In the main window, click **Admin > Discovery > Client**. The Client window displays.

Select the client on which to install a new discovery agent. A new window displays with the **Details** tab displayed.

Click the **Discovery Agents** tab.

From the *Select Actions* drop-down list, choose **Remote Install**. The Remote Install Discovery Agent window displays.

Enter or select the information in the following fields:

- **Host Type**: Select either Windows or Linux.
- **IP Address/Host Name**: Enter the IP address or host name.
- **Port**: Enter the port number.
- **Domain**: Enter the domain name (Windows only).
- **User Name**: Enter the username for authentication.
- **Password**: Enter the password.

Click **Install**.
