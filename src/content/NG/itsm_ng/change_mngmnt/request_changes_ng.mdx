# Request for Changes

Use this function to submit a formal proposal for implementing a change. The Request for Change (RFC) provides the details for this change.

In the navigation pane, select **ITSM \> Change Management \> Request for Changes**. The Request for Changes window displays.

![request for changes main old](../../Resources/Images/ITSM/Change Management/request-for-changes-main-old.PNG)

![request for changes main new](../../Resources/Images/ITSM/Change Management/request-for-changes-main-new.PNG)

![arrow border](../../Resources/Images/arrow-border.PNG)

### Time Tracking

Use this function to add a record/task or generate a time spent report.

While viewing the Details window, select **Time Tracking**. The Time Tracking window displays.

![changes time tracking main old](../../Resources/Images/ITSM/Change Management/changes-time-tracking-main-old.PNG)

#### Add Record or Tasks

 -- ----- ------------------------------------------------------------------------------
 1. To add a new item, click **Add**. The Add Time Tracking dialog box displays.
 -- ----- ------------------------------------------------------------------------------

::: {.Ind50
![time tracking add record old](../../Resources/Images/ITSM/Change Management/time-tracking-add-record-old.PNG)

::: {.Ind50
![time tracking add task old](../../Resources/Images/ITSM/Change Management/time-tracking-add-task-old.PNG)

 -- ----- ----------------------------------------
 2. Select either **Record** or **Tasks**.
 -- ----- ----------------------------------------

- [[[Closed]{.image .placeholder .MCDropDown_Image_Icon original-image-src="../../../Skins/Default/Stylesheets/Images/transparent.gif" original-image-title="" height="12" width="16" Request for Change](javascript:void(0)){.MCDropDownHotSpot .dropDownHotspot .MCDropDownHotSpot_ .MCHotSpotImage aria-expanded="false"}]{.MCDropDownHead .dropDownHead}
 --- -----------------------------------------------------------------------------------------
   Not all functions include an option to edit record line items shown in the main window.
 --- -----------------------------------------------------------------------------------------

 -- ----- -------------------------------------------------------
 1. Navigate to the window containing the record to edit.
 -- ----- -------------------------------------------------------

 -- ----- -------------------------------------------------------------------------------------
 2. Click the line containing the record. The applicable window or dialog box displays.
 -- ----- -------------------------------------------------------------------------------------

 -- ----- -----------------------------------
 3. Make the necessary modifications.
 -- ----- -----------------------------------

 -- ----- -------------------------------------------
 4. Click **Save** or **Add**, as applicable.
 -- ----- -------------------------------------------

When editing an existing item, other options may be available, such as adding [Tasks](/itsm/release_mngmt/tasks_6_1.mdx){.MCTopicPopup .MCTopicPopupHotSpot .a}, [Comments](/discovery/comments_6_1.mdx){.MCTopicPopup .MCTopicPopupHotSpot .a}, and [Attachments](/discovery/attachments_6_1.mdx){.MCTopicPopup .MCTopicPopupHotSpot .a} and viewing [History](/common_topics/history_6_1.mdx){.MCTopicPopup .MCTopicPopupHotSpot .a}.

 --- ---------------------------------------------------------------------------------------
   The options change based upon the function; therefore, not every option may be shown.
 --- ---------------------------------------------------------------------------------------

 --- --------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------
   Deleting is a permanent action and cannot be undone. Deleting may affect other functionality and information in the application such as data in configured reports, fields in windows, selectable options, etc. Therefore, be sure to understand the potential effects before making a deletion.
 --- --------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------

 -- ----- --------------------------------------------------
 1. Click the line that contains the item to delete.
 -- ----- --------------------------------------------------

 -- ----- ------------------------------------------------------------------------------------------------------------------------------------------------------
 2. From the *Select Actions* drop-down list, choose **Delete**. If a confirmation message is displayed, type **Delete** and click **Delete**to confirm.
 -- ----- ------------------------------------------------------------------------------------------------------------------------------------------------------
