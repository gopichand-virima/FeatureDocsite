# Viewing a discovered Meraki record

1. In **Import from Meraki**, open the **All** tab and **click any row** (e.g., an Asset Name).

2. The **Discovered Item** view for that asset opens.

This view provides detailed information about the selected Meraki device, including:
- Primary device details
- Network configuration
- Hardware specifications
- Status and health information
- Associated components and connections
- Historical data and change tracking
