# Monitoring Profile: Details

Use this function to add details for a monitoring profile.

On the main window, selectAdmin>Discovery** > Discovery Monitoring Profile**. The Discovery Monitoring Profile window displays.

From the *Select Actions* drop-down list, choose an action: **New**.  

In the *Name* field, enter a name for this profile.

In the *Description* field, enter information to identify this profile.

In the *Client* field, click the drop-down list and select the applicable client.

- If the applicable client is not shown, click the link to check the client's status.

In the *IP Address* field, enter the IP Address for the client. The IP range can be a single IP, multiple IPs (separated by a space), CIDR Notation or IP with Range.

When all selections/entries are made, click **Add**.

When all selections/entries are made, click **Save**.
