# Edit Service Level Agreement

Use this function to edit the details for a service level agreement.

 -- ----- -------------------------------------------------------------------------------------------------------------------------
 1. In the main window, select **Admin \> Other \> Service Level Agreement.** The Service Level Agreements window displays.
 -- ----- -------------------------------------------------------------------------------------------------------------------------

 -- ----- --------------------------
 2. Select a record to edit.
 -- ----- --------------------------

 -- ----- ------------------------------------------
 3. Modify any of the following information:
 -- ----- ------------------------------------------

::: Bullet2
[Details](SLA-Details.mdx){.MCTopicPopup .MCTopicPopupHotSpot .a}

::: Bullet2
[Tasks](SLA-Tasks.mdx){.MCTopicPopup .MCTopicPopupHotSpot .a}

::: Bullet2
[Comments](SLA-Comments.mdx){.MCTopicPopup .MCTopicPopupHotSpot .a}

::: Bullet2
[Attachments](SLA-Attachments.mdx){.MCTopicPopup .MCTopicPopupHotSpot .a}

 -- ----- -------------------------------------------------------
 4. When all selections/entries are made, click **Save**.
 -- ----- -------------------------------------------------------
