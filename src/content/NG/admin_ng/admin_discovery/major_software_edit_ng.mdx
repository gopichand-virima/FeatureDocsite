# Edit or Delete Major Software

Use this function to edit or delete the details for an installed major software application.

## Edit Major Software

In the Major Software window, select a record to edit. The details for the software display.

Modify any of the following information:

- [Details](/admin_discovery/major_software_details_ng.mdx)
- [Tasks](/itsm/release_mngmt/tasks_ng.mdx)
- [Comments](/discovery/comments_ng.mdx)
- [Attachments](/discovery/attachments_ng.mdx)

When all selections/entries are made, click **Save**.
