# Knowledge Model

Use this function to add a model for defining knowledge articles.

In the main window, click **Admin /> Knowledge Management /> Knowledge Model**.

<span class="image placeholder" original-image-src="../Resources/Images/Admin%20-%20Knowledge/knowledge-model-main-old.PNG" original-image-title=""></span>

In the main window, click the **gear** icon and select **Knowledge Management /> Knowledge Model**.

<span class="image placeholder" original-image-src="../Resources/Images/Admin%20-%20Knowledge/icon-knowledge-model.PNG" original-image-title=""></span>

The Knowledge Model window displays.

<span class="image placeholder" original-image-src="../Resources/Images/Admin%20-%20Knowledge/knowledge-model-main-new.PNG" original-image-title=""></span>
