# Software Instances

<div class="NumList1" madcap:autonum="1. ">

While viewing an asset record, click the **Components** option.

</div>

<div class="NumList2" madcap:autonum="2. ">

Click **Software Instances**.

</div>

<div class="Ind50" madcap:conditions="Default.NewUI">

<span class="image placeholder" original-image-src="../../../Resources/Images/Components/software-instances-new.PNG" original-image-title="" style="width: 673px;height: 161px;"></span>

</div>

## Add New Software Instances

<div class="NumList1" madcap:autonum="1. ">

To add a new instance, click **Add**. The Software Instances dialog box displays.

</div>

<div class="Ind50" madcap:conditions="Default.NewUI">

<span class="image placeholder" original-image-src="../../../Resources/Images/Components/software-instance-new-add.PNG" original-image-title="" style="width: 475px;height: 207px;"></span>

</div>

<div class="NumList2" madcap:autonum="2. ">

Click **New Software Instance**.

</div>

<div class="NumList2" madcap:autonum="3. ">

Enter the applicable information in each field.

</div>

<div class="NumList2" madcap:autonum="4. ">

When all selections are made, click **Save**.

</div>

## Modify Existing Software Instance

<div class="NumList1" madcap:autonum="1. ">

In the Software Instance dialog box, click **Existing Software Instance**.

</div>

<div class="Ind50" madcap:conditions="Default.NewUI">

<span class="image placeholder" original-image-src="../../../Resources/Images/Components/software-instances-new-existing.PNG" original-image-title="" style="width: 472px;height: 321px;"></span>

</div>

<div class="NumList2" madcap:autonum="2. ">

Enter the search criteria to locate the existing software instance.  *See also* <a href="../../../Common%20Topics/Advanced-Search.htm" style="font-style: italic;">Advanced Search</a>*.*

</div>

<div class="NumList2" madcap:autonum="3. ">

Select the applicable item from the results list to move it to the selected items list.

</div>

<div class="NumList2" madcap:autonum="4. ">

When all selections are made, click **Save**.

</div>

## Delete Existing Software Instance

While viewing the list of existing software instances, select the item to delete, and click the **Delete** button.
