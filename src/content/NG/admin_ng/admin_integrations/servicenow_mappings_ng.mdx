# ServiceNow Mappings

Use this function to view and add ServiceNow Property mapping information.

 --- --------------------------------------------------------------------------------------------------------------
   There must be a [ServiceNow Credential](ServiceNow-Credentials.mdx) configured before a mapping can be done.
 --- --------------------------------------------------------------------------------------------------------------

In the main window, click **Admin \> Integrations \> ServiceNow Mappings** . The Service Now Mapping window displays.

![servicenow map main old](../Resources/Images/Admin - Integration/servicenow-map-main-old.PNG)

![arrow border](../Resources/Images/arrow-border.PNG)

[[[Closed]{.image .placeholder .MCDropDown_Image_Icon original-image-src="../../Skins/Default/Stylesheets/Images/transparent.gif" original-image-title="" height="12" width="16" ServiceNow Mapping](javascript:void(0)){.MCDropDownHotSpot .dropDownHotspot .MCDropDownHotSpot_ .MCHotSpotImage aria-expanded="false"}]{.MCDropDownHead .dropDownHead}

 -- ----- ---------------------------------------------------------------------------------
 1. In the main window, select the applicable record. The mappings window displays.
 -- ----- ---------------------------------------------------------------------------------

 -- ----- -------------------------------------------------------------------------------------------------------------------------
 2. Choose a *Virima Blueprint* (e.g., Windows Server, Linux Server). This determines which CI type the mapping applies to.
 -- ----- -------------------------------------------------------------------------------------------------------------------------

 -- ----- -----------------------------------------------------------------
 3. Pick the *ServiceNow Table* where the CI data should be stored.
 -- ----- -----------------------------------------------------------------

[[[Closed]{.image .placeholder .MCDropDown_Image_Icon original-image-src="../../Skins/Default/Stylesheets/Images/transparent.gif" original-image-title="" height="12" width="16" ServiceNow Mapping](javascript:void(0)){.MCDropDownHotSpot .dropDownHotspot .MCDropDownHotSpot_ .MCHotSpotImage aria-expanded="false"}]{.MCDropDownHead .dropDownHead}

 -- ----- ---------------------------------------------------------------------------------------------------------------------------------------
 1. In the main window, select the applicable record. The mappings window displays, and contains a list of all current property mappings.
 -- ----- ---------------------------------------------------------------------------------------------------------------------------------------

 -- ----- ----------------------------------------------------------
 2. Hover over the line item containing the mapping to edit.
 -- ----- ----------------------------------------------------------

 -- ----- ----------------------------------------------------------------
 3. Make the necessary changes in the *ServiceNow Property* field.
 -- ----- ----------------------------------------------------------------

 -- ----- -------------------------------------------------------
 4. When all selections/entries are made, click **Save**.
 -- ----- -------------------------------------------------------

[[[Closed]{.image .placeholder .MCDropDown_Image_Icon original-image-src="../../Skins/Default/Stylesheets/Images/transparent.gif" original-image-title="" height="12" width="16" ServiceNow Mapping](javascript:void(0)){.MCDropDownHotSpot .dropDownHotspot .MCDropDownHotSpot_ .MCHotSpotImage aria-expanded="false"}]{.MCDropDownHead .dropDownHead}

 -- ----- ---------------------------------------------------------------------------------------------------------------------------------------
 1. In the main window, select the applicable record. The mappings window displays, and contains a list of all current property mappings.
 -- ----- ---------------------------------------------------------------------------------------------------------------------------------------

 -- ----- -----------------------------------------------------
 2. Select the applicable mapping and click **Remove**.
 -- ----- -----------------------------------------------------

[[[Closed]{.image .placeholder .MCDropDown_Image_Icon original-image-src="../../Skins/Default/Stylesheets/Images/transparent.gif" original-image-title="" height="12" width="16" ServiceNow Mapping](javascript:void(0)){.MCDropDownHotSpot .dropDownHotspot .MCDropDownHotSpot_ .MCHotSpotImage aria-expanded="false"}]{.MCDropDownHead .dropDownHead}

 --- --------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------
   Deleting is a permanent action and cannot be undone. Deleting may affect other functionality and information in the application such as data in configured reports, fields in windows, selectable options, etc. Therefore, be sure to understand the potential effects before making a deletion.
 --- --------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------

 -- ----- --------------------------------------------------
 1. Click the line that contains the item to delete.
 -- ----- --------------------------------------------------

 -- ----- ------------------------------------------------------------------------------------------------------------------------------------------------------
 2. From the *Select Actions* drop-down list, choose **Delete**. If a confirmation message is displayed, type **Delete** and click **Delete**to confirm.
 -- ----- ------------------------------------------------------------------------------------------------------------------------------------------------------
