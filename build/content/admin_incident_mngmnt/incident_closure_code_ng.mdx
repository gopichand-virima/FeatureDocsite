# Incident Closure Code

Use this function to configure closure codes with descriptions for the Closed state in the Incident Management function.

In the main window, click **Admin > Incident Management > Incident Closure Code**. The Incident Closure Code window displays.

## New Incident Closure Code

From the *Select Actions* drop-down list, choose **New Incident Closure Code**. The New Incident Closure Code window displays.

Enter a *Name* and *Description*.

Select the **Resolution Type**:

- Resolved
- Workaround
- Not Resolved
- Duplicate
- User Error

Click **Add**.

## Edit Incident Closure Code

Navigate to the window containing the record to edit.

Click the line containing the record. The applicable window displays.

Make the necessary modifications.

Click **Save**.

## Delete Incident Closure Code

Click the line that contains the item to delete.

From the *Select Actions* drop-down list, choose **Delete**.
