# Configuring the Application

1. Open the **Virima Discovery Application**.

2. Go to **Settings** > **App Configuration**.

!

1. Enter the following details:

  - **Username:** Your Web App login email
  - **Password:** Your Web App password
  - **Customer ID:** Found in your organization details in the Web App
  - **Application Name:** Any preferred name
  - **Your IP Address:** IP of the machine where the app is installed

!

1. Click **Save** to apply the configuration.

2. The application will launch and open the Virima Discovery interface.

3. Click **Start** to begin discovery Virima Discovery Application.

!

!

The application will now begin running based on the configured settings.
