# Port Configuration for Process

Use this function to configure port-process mappings for application discovery.

In the main window, click **Admin > Discovery > Port Configuration for Process**. The Port Configuration window displays.

From the *Select Actions* drop-down list, choose an action: **New Port Configuration**, [Delete](/common_topics/delete_remove_6_1.mdx), or [Export](/discovery/import_data_files/export_6_1.mdx).

In the *Port Number* field, enter the port number.

In the *Process Name* field, enter the associated process name.

In the *Description* field, provide details about this configuration.

When all entries are made, click **Add**.
