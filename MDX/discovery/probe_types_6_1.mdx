# Probe Types

| Type | Description |

| --- | --- |

| Deep Host Scan | Details of the host including IP address, OS type, MAC address, machine basic details, and components. |

| Host Type Scan | Type of host present in the network environment. |

| Basic Host Scan | Basic details of the machine. |

| Windows Basic AD Scan | Basic details of the computer present under the AD server. |

| Windows Deep AD Scan | Basic details of every computer present under the AD server with AD structures present under diverse types of Hosts present in the network environment. |

| IP Connection Scan | IP connection details. |

| Website Discovery | Web URL input and relevant information. |

| Port Scan | Pings existing Ports in the network environment. |

| Medical Device Scan | Details of medical devices present in a monitored network environment. |

| Install Software | Downloadable link of the software that needs to be installed on the remote machine. |

| Certificate Scan | Details of the certificate for the selected certificate path as defined in the *Certificate Path* field. |

| Registry Scan | Details of the Registry for the selected Registry path as defined in the *Registry Path* field. |

| Storage Server Discovery Scan XtreamIO | Details of the XtreamIO storage server present in a monitored network environment. |

| Kubernetes Discovery | Details of the Kubernetes. |

| Paexec Cleanup | Removes Paexec Services and Paexec .exe files. |

| Windows Major Component Discovery | Discovers Windows components installed in the network environment. |

| Windows all Info with Powershell | Discovers all the Windows details in the network environment using Powershell cmd. |

| VCenter Scan | Discovers ESXi host details on VMware present in a monitored network environment. |

| .Net Framework Audit Scan | Details of the .Net software present in a monitored network environment. |

| Java Audit Scan | Details of the Java software present in a monitored network environment. |

| SQL Server Query Scan | Executes SQL query on a host and performs the following: - Writes data to an Excel sheet only if the executed queries return successfully. - Reads data fetched from the executed query and can be used to (a) create a discovery asset and (b) export data through mail. |

| MSSQL Maintenance Connection Import Scan | Executes query (CRUD) Operation On Host with static query and data mapping. |
