# Admin Graphical Workflows

Use this function to create visual workflows that automate actions based on trigger conditions.

In the main window, click **Admin > \[Module\] > Graphical Workflows**. The Graphical Workflows window displays.

New Graphical Workflow

From the *Select Actions* drop-down list, choose **New Graphical Workflow**. The Graphical Workflow Builder window displays.

Enter a *Workflow Name* and *Description*.

Configure the workflow trigger conditions and actions using the visual workflow builder.

When all configurations are complete, click **Save**.
