# Tabs for Extended Information

- [Tasks]{.Strong}: Displays or allows creation of tasks linked to the import job.

- [Comments]{.Strong}: Provides space for administrators to add notes or review remarks from others.

- [Attachments]{.Strong}: Lets users upload or view related files, such as screenshots, reports, or documentation tied to the import.

Clicking on the \"[New User Count]{.Strong}\" will navigate you to the User Record page, where the AD User Import Logs are automatically filtered and the relevant results are displayed.
