# Request Error Code

Use this function to define an error message to display in case of incorrect configuration scenario.

In the main window, click **Admin > Request Management > Request Error Code**.

New Request Error Code

From the *Select Actions* drop-down list, choose **New Request Error Code**. The New Request Error Code window displays.

Enter a *Name* and *Description*.

Click **Add**.
