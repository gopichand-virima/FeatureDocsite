# Advanced Search

The Advanced Search function allows you to perform complex queries across your Virima data.

## Overview

Advanced Search provides powerful filtering and search capabilities to help you locate specific records, analyze data patterns, and create custom views based on multiple criteria.

## Using Advanced Search

### Basic Search
1. Click the **Search** icon in the toolbar
2. Enter your search term in the search field
3. Press Enter or click the search button
4. Results will display in real-time

### Advanced Filters
Access advanced filtering options:
1. Click the **Advanced Search** link or icon
2. Select fields to search within
3. Choose comparison operators (equals, contains, greater than, etc.)
4. Add multiple criteria using AND/OR logic
5. Apply filters to view results

## Search Criteria

### Available Operators
- **Equals**: Exact match
- **Contains**: Partial match
- **Starts with**: Matches beginning of field
- **Ends with**: Matches end of field
- **Greater than/Less than**: Numeric and date comparisons
- **Is empty/Is not empty**: Null value checks

### Field Selection
Search across multiple field types:
- Text fields
- Date and time fields
- Numeric fields
- Dropdown selections
- Custom fields

## Saving Searches

### Save Custom Queries
- Create and name custom search queries
- Save frequently used searches for quick access
- Share saved searches with team members
- Set default searches for specific views

## Search Tips

- Use wildcards (*) for broader matches
- Combine multiple criteria for precise results
- Save complex searches to avoid rebuilding them
- Use date ranges for time-based queries
- Clear filters to reset your view
