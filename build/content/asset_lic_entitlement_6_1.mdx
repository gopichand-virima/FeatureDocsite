# Asset License Entitlement

Use this function to enable the asset manager to entitle a user to use the machine having a corresponding license.

The benefits of Asset License Entitlement are that it:

- Facilitates tracking and management of license agreements.
- Maintains a record of software deployment globally.
- Ensures legal compliance of software in on-premises, mobile, and cloud environments.

In the navigation pane, select **ITAM > Asset License Entitlements**.

New Asset License Entitlements

From the *Select Actions* drop-down list, choose **New Asset License Entitlements**. The New Asset License Entitlements window displays.

Enter a *Display Name*.

In the *Configuration* Item field, click **Add**, then search for select the applicable configuration item.

In the *User* field, click the **Add** button, then search for select the applicable User.

Click **Add**.
