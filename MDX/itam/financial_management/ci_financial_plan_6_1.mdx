# CI Financial Plan

In the navigation pane, select **ITAM > Financial Management** then select **CI Financial Plan** .

!  

!

!


### New CI Financial Plan

- From the *Select Actions* drop-down list, choose **New CI Financial Plan**. The applicable window displays.
- Complete the fields referring to the table below.
- When all selections/entries are made, click **Add**.

### CI Financial Plan Fields

| Fields | Description |

| --- | --- |

| Primary Details | **Name**. Enter a name for thisÂ Financial Plan. **Description**. Enter information that describes this financial plan. |

| Record Details | **Start Date/End Date**. Select the dates between which this financial plan is active. **Status**. The status of the financial plan such as Operational, Non-operational, and so forth. Click the drop-down list and select the relevant status. **Interval**. The frequency at which the cost recurs. Options include Daily, Weekly, and Monthly. **Last Processed On**. The date and time this cost was last processed. **Next Process On**. The next date on which new expenses will be processed. **Order**. The Order of processing the expense. |

| Financial Details | *Base Cost*. The amount that must be paid before taxes. *Support Cost*. Expenses for maintaining or supporting an asset *Tax Cost*. Total cost of the tax. *Total Cost*. Total base cost, including taxes. *Tax Rate*. The tax rate to apply to the cost. *Recurring*. When selected, indicates whether the cost is a repeating cost. *Sales Tax*. When selected, indicates whether to apply sales tax to the cost. *Associated CI*.Click **Add**, then in the Associate CI dialog box, select a Blueprint, then search for and select the applicable CI. Click **Save**. |


### Edit CI Financial Plan

Not all functions include an option to edit record line items shown in the main window.


- Navigate to the window containing the record to edit.
- Click the line containing the record. The applicable window or dialog box displays.
- Make the necessary modifications.
- Click **Save** or **Add**, as applicable.

When editing an existing item, other options may be available, such as adding [Tasks](/itsm/release_mngmt/tasks_6_1.mdx), [Comments](/discovery/comments_6_1.mdx), and [Attachments](/discovery/attachments_6_1.mdx) and viewing [History](/common_topics/history_6_1.mdx).


The options change based upon the function; therefore, not every option may be shown.


### Delete CI Financial Plan

Deleting is a permanent action and cannot be undone. Deleting may affect other functionality and information in the application such as data in configured reports, fields in windows, selectable options, etc. Therefore, be sure to understand the potential effects before making a deletion.


- Click the line that contains the item to delete.
- From the *Select Actions* drop-down list, choose **Delete**. If a confirmation message is displayed, type **Delete** and click **Delete**to confirm.
