# Monitoring Profile: Trigger Conditions

Use this function to define a condition to match the discovered data as configured.

On the main window, selectAdmin>Discovery** > Discovery Monitoring Profile**. The Discovery Monitoring Profile window displays.

Select either **System Monitor** or **File Monitor**.

In the **Trigger Conditions** window, use the icons to do the following:

- Click the **plus + sign** to add an additional row.
- Click the **red x** to remove the row.

If *System Monitor* is selected:

In the *Property* field, click the drop-down list and choose a property from those available.

In the *Operator* field, click the drop-down list and choose the applicable operator.

The Operators displayed are valid for the selected *Property* type, and then enter the value.  

If *File Monitor* is selected:

In the *Type* field, click the drop-down list and select the applicable type.

In the text field, type the applicable information.

To add a row, click the **plus + sign**.

To delete a row, click the **minus - sig**n.

When all selections/entries are made, and a match is found, the condition is triggered.

When all selections/entries are made, click **Add**.
