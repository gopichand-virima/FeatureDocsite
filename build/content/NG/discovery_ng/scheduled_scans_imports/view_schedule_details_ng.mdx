# View Schedule Details

Click on any scheduled operation to view its complete configuration and history.

## Details Tab

### For Scheduled Scans:
- **Name**: Schedule identifier
- **Probe**: Type of scan configured
- **Client**: Discovery client executing the scan
- **IP Range**: Target IP addresses
- **Exclude IP Range**: Excluded IP addresses
- **Scan Frequency**: How often it runs
- **Is Active**: Whether enabled
- **Is Recurring**: Whether it repeats
- **Timezone**: Configured timezone
- **Schedule Parameters**: Exact timing configuration
- **Location**: Associated location tags
- **Send Scan Report To**: Email recipients
- **Created By**: User who created the schedule
- **Created On**: Creation timestamp
- **Modified By**: User who last modified
- **Modified On**: Last modification timestamp

### For Scheduled Imports:
- **Name**: Schedule identifier
- **Source**: Cloud provider (AWS, Azure, Intune, Meraki)
- **Credential**: Cloud credential used
- **Services to Import**: Selected resources
- **Import Frequency**: How often it runs
- **Active**: Whether enabled
- **Timezone**: Configured timezone
- **Schedule Parameters**: Exact timing configuration
- **Created By**: User who created the schedule
- **Created On**: Creation timestamp
- **Modified By**: User who last modified
- **Modified On**: Last modification timestamp

## Related Scans Tab
- Complete list of all executions
- Historical results and status
- Click any execution to view detailed results

## Tasks Tab
- Manage tasks associated with the schedule
- Create follow-up actions
- Track task completion

## Comments Tab
- View and add notes about the schedule
- Team collaboration on schedule management
- Document changes and decisions

## Attachments Tab
- Upload supporting documentation
- Store configuration details
- Reference materials

## Log Tab
- View execution logs for recent runs
- Troubleshoot failed executions
- Monitor performance trends

All information in the Details view provides comprehensive visibility into scheduled operation configuration, execution history, and management activities.
