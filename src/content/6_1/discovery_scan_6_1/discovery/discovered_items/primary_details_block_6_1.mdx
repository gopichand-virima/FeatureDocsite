# Primary Details Block

Displays high-level information: asset ID, blueprint, created date, IP address, host name, scan time.

- <span class="Strong">If the asset already exists in the CMDB</span>:  
  The <span class="Strong">Asset ID</span> field shows the existing CI ID. Clicking this ID opens the detailed CI record in the CMDB for review or editing.

- <span class="Strong">If the asset is newly discovered</span>:  
  The <span class="Strong">Asset ID</span> displays as <span class="Strong">"New,"</span> indicating that this item has not yet been moved into the CMDB and needs validation or onboarding.
