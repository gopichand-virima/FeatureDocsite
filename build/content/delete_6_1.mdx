# Delete

To delete uploaded file records,

1. Select one or more files from the list.

2. From the **Select Actions** menu, click **Delete**.

1. If no files are selected, an alert appears: "Please select at least one record."

1. Confirm the deletion and Click on**Delete**.

  1.  Deleting a file from Import Data Files will also remove the associated imported assets only if they have not yet been promoted to the CMDB.

2. Click**Cancel **to cancel the delete.
