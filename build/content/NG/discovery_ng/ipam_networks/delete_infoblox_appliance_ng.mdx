# Delete Infoblox Appliance

Use this function to remove an Infoblox Appliance and its associated information.

In the Infoblox Appliances window, select an existing Infoblox Appliance record.

Click the **Delete** icon on the toolbar.

In the popup, type **Delete** in the text field.

Click **Delete**. The Infoblox Appliance is removed from the list.
