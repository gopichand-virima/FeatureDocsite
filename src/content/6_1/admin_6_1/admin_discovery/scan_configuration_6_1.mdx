# Discovery Scan Configuration

Use this function to configure scan settings for discovery operations.

In the main window, click **Admin > Discovery > Discovery Scan Configuration**. The Scan Configuration window displays.

## Configuration Options

- **Scan Timeout** - Set the timeout period for scans
- **Thread Pool Size** - Configure concurrent scan threads
- **Retry Attempts** - Number of retry attempts for failed scans
- **Scan Schedule** - Configure automated scan schedules

From the *Select Actions* drop-down list, choose an action: **New Scan Configuration**, [Delete](/common_topics/delete_remove_6_1.mdx), or [Export](/discovery/import_data_files/export_6_1.mdx).
