# Organizational Details

Use this function to update and save the organization\'s information.

 --- ------------------------------------------------------------------------------------------------------------------------------------------------
   The Organization Details are created upon initial set up by Virima. While the Organization cannot be deleted, the information can be modified.
 --- ------------------------------------------------------------------------------------------------------------------------------------------------

[In the main window, click ]{.span_1}[Admin]{.span_2}[ \> ]{.span_1}[Organizational Details \> Organizational Details]{.span_2}[.]{.span_2} The Organizational Details window displays.

![org details New](../Resources/Images/Admin - Org Details/org-details_New.png)

![arrow border](../Resources/Images/arrow-border.PNG)

[[[Closed]{.image .placeholder .MCDropDown_Image_Icon original-image-src="../../Skins/Default/Stylesheets/Images/transparent.gif" original-image-title="" height="12" width="16" Organizational Details](javascript:void(0)){.MCDropDownHotSpot .dropDownHotspot .MCDropDownHotSpot_ .MCHotSpotImage aria-expanded="false"}]{.MCDropDownHead .dropDownHead}

 -- ----- ----------------------------------------------------------------------------------------
 1. Enter the applicable information in the provided fields, referring to the table below.
 -- ----- ----------------------------------------------------------------------------------------

 -- ----- -------------------------------------------------------
 2. When all selections/entries are made, click **Save**.
 -- ----- -------------------------------------------------------

### Organizational Details Fields

+---------------------------------+--------------------------------------------------------------------------------------------------------------------------------------------------------------+
| Field | Description |
+=================================+==============================================================================================================================================================+
| Name | Name of this organization. |
+---------------------------------+--------------------------------------------------------------------------------------------------------------------------------------------------------------+
| Customer ID | An ID number assigned by the system, which cannot be edited. |
+---------------------------------+--------------------------------------------------------------------------------------------------------------------------------------------------------------+
| Description | Details about the organization. |
+---------------------------------+--------------------------------------------------------------------------------------------------------------------------------------------------------------+
| Location Info | Details about the organization location such as Address, Country, State, City, Postal Code, Phone, Fax, URL, and Email. |
+---------------------------------+--------------------------------------------------------------------------------------------------------------------------------------------------------------+
| Logo | Configures the logo displays on reports, email messages, application log in page, password page, and in web page header. |
| --- | --- |
| | -- ----- ------------------------------------------------------------------- |
| | 1. Click the **Upload** button. The Attach File dialog box displays. |
| | -- ----- ------------------------------------------------------------------- |
| | |
| | -- ----- -------------------------------------------------------------------------- |
| | 2. Click the **Browse** button, and locate and select the applicable image. |
| | -- ----- -------------------------------------------------------------------------- |
| | |
| | -- ----- --------------- |
| | 3. Click **OK**. |
| | -- ----- --------------- |
+---------------------------------+--------------------------------------------------------------------------------------------------------------------------------------------------------------+
| Currency | Specifies the currency type used for cost center information. Click the drop-down list and select a currency from those shown. |
+---------------------------------+--------------------------------------------------------------------------------------------------------------------------------------------------------------+
| Default Landing Page | Configures a default landing page. |
| --- | --- |
| | -- ----- -------------------------------------------------------------- |
| | 1. Click **Add**. The Default Landing Page dialog box displays. |
| | -- ----- -------------------------------------------------------------- |
| | |
| | -- ----- ----------------------------------------------------------------------------- |
| | 2. In the *Search* field, type one or more characters for the search criteria. |
| | -- ----- ----------------------------------------------------------------------------- |
| | |
| | -- ----- --------------------------------------------------- |
| | 3. Select the applicable page from the results list. |
| | -- ----- --------------------------------------------------- |
| | |
| | -- ----- ----------------- |
| | 4. Click **Save**. |
| | -- ----- ----------------- |
+---------------------------------+--------------------------------------------------------------------------------------------------------------------------------------------------------------+
| Time Zone | Specifies the time zone for this organization.  Click the drop-down list and select a time zone from those shown. |
+---------------------------------+--------------------------------------------------------------------------------------------------------------------------------------------------------------+
| Is Default Notification Enabled | Disables/enables the global functionality to turn email notifications on/off. When enabled, Virima notifications can be sent by users throughout the system. |
| --- | --- |
| | --- --------------------------------------------------------------------------------------------------------------------- |
| | Users can control email notifications through [Email Preferences](/admin_users/users_6_1.mdx) in their profile. |
| | --- --------------------------------------------------------------------------------------------------------------------- |
+---------------------------------+--------------------------------------------------------------------------------------------------------------------------------------------------------------+
