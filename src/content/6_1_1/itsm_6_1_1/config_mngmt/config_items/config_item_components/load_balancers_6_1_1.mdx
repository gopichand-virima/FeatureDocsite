# Load Balancers

Click **Load Balancers**.

! ! Add Load Balancer

Click **Add**. The Load Balancer dialog box displays.

Click **New Load Balancer.**

Enter the applicable field information.

When all selections/entries are made, click **Save**.

Edit Load Balancer

Click **Existing Load Balancer**.

Enter the search criteria to locate the existing device.  *See also* [Advanced Search](/common_topics/advanced_search_6_1_1.mdx)*.*

Select the applicable device from the results list.

When all selections are made, click **Save**.
