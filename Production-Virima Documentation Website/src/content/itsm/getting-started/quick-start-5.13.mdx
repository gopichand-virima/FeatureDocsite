# ITSM - Quick Start Guide

Get started with IT Service Management quickly and effectively.

## Prerequisites

- ITSM module access permissions
- Email configuration (for notifications)
- Understanding of your service desk processes

## Quick Start Steps

### Step 1: Access ITSM Dashboard
1. Log in to Virima
2. Navigate to ITSM from the main menu
3. Review the ticket dashboard

### Step 2: Create an Incident
1. Click "New Incident" button
2. Fill in required fields:
   - Title and description
   - Affected user/asset
   - Priority and category
3. Assign to appropriate team/agent
4. Save the incident

### Step 3: Configure SLA Rules
1. Go to Settings > SLA Management
2. Create SLA policy
3. Define response and resolution times
4. Set escalation rules
5. Save and activate

### Step 4: Set Up Email Notifications
1. Navigate to Settings > Notifications
2. Configure email templates
3. Set notification triggers
4. Test email delivery
