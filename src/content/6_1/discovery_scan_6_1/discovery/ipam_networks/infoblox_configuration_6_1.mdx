# Infoblox Configuration

Navigate to <span class="Strong">Admin /> Integration /> Infoblox Configuration</span>

<span class="image placeholder" original-image-src="../../Resources/Images/Discovery/IPAM-Networks/Infoblox%20Configuration.png" original-image-title=""></span>

You have two options for adding Infoblox credentials. Please choose one of the following methods:

<span class="image placeholder" original-image-src="../../Resources/Images/Discovery/IPAM-Networks/Infoblox%20Configuration_1.png" original-image-title=""></span>

<span class="Strong">Option 1: Add New Infoblox Credential</span>

Manually add a new Infoblox credential for authenticating the connection between the system and Infoblox. Enter the Infoblox username, password, host IP, and select the applicable client.

<span class="image placeholder" original-image-src="../../Resources/Images/Discovery/IPAM-Networks/Infoblox%20Configuration_2.png" original-image-title=""></span>

1.  In the <span class="Strong">Admin</span> section, navigate to <span class="Strong">Integration /> Infoblox Configuration</span>.

2.  In the Infoblox Credential dialog box, click <span class="Strong">Add New Infoblox Credential</span>.

3.  Enter the following details:

    1.  <span class="Strong">Username:</span> Your Infoblox username.

    2.  <span class="Strong">Password:</span> Your password.

    3.  <span class="Strong">Host:</span> The IP address of the Infoblox machine.

    4.  <span class="Strong">Client:</span> Select the client from the drop-down list.

4.  Click <span class="Strong">Save</span> to complete the setup.

<span class="Strong">Option 2: Auto Pick Credential from Discovery Application</span>

Automatically pick credentials from the Discovery application, streamlining the process. The system will fill in the required details, including host IP and client information.

<span class="image placeholder" original-image-src="../../Resources/Images/Discovery/IPAM-Networks/Infoblox%20Configuration_3.png" original-image-title=""></span>

1.  In the <span class="Strong">Admin</span> section, navigate to <span class="Strong">Integration /> Infoblox Configuration</span>.

2.  In the Infoblox Credential dialog box, select <span class="Strong">Auto pick credential from Discovery application</span>.

3.  Enter the following details:

    - <span class="Strong">Host:</span> The IP address of the Infoblox machine.

    - <span class="Strong">Client:</span> Select the client from the drop-down list.

4.  Click <span class="Strong">Save</span>.
