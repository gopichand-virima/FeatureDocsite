# Dashboard Customization

- <span class="Strong">Add/Remove Reports:</span> Easily add, remove, or rearrange reports at any time.

- <span class="Strong">Personalize Layout:</span> Tailor the dashboard to your needs or department.

  <span class="image placeholder" original-image-src="../../Resources/Images/Discovery/Discovery-Dashboard/Discovery%20Dashboard_1_9.png" original-image-title=""></span> <span class="image placeholder" original-image-src="../../Resources/Images/Discovery/Discovery-Dashboard/Discovery%20Dashboard_1_10.png" original-image-title=""></span>

The Virima <span class="Strong">Dashboard</span> can be personalized to display a wide range of reports based on user needs and preferences. Reports can be added, removed, or rearranged at any time.

# Report Widget Controls

Each report widget includes controls in the upper-right corner:

- <span class="Strong">Click for Complete Report:</span> Located in the top-right corner, this button opens a full, detailed version of the report in a new browser window.  
  <span class="Emphasis">(For example, clicking this on a software report will open a full Software Query Report view.)</span>

- <span class="Strong">Export:</span> Exports the report as an Excel file and emails a download link to the logged-in user.

- <span class="Strong">Save Image:</span> Downloads a snapshot of the current report view.

- <span class="Strong">Collapse/Expand:</span> Minimizes or restores the report’s visibility.

- <span class="Strong">Maximize/Minimize:</span> Opens the report in a larger window or returns it to the dashboard.

- <span class="Strong">Remove:</span> Removes the report from the dashboard (confirmation required).

# Auto Refresh

The <span class="Strong">Auto Refresh</span> feature keeps dashboard data current by updating at set intervals.

<span class="image placeholder" original-image-src="../../Resources/Images/Discovery/Discovery-Dashboard/Discovery%20Dashboard_1_11.png" original-image-title=""></span>

<span class="Strong">To use:</span>

- Click the <span class="Strong">Auto Refresh</span> drop-down menu.

- Choose from:

  - None (disable auto-refresh)

  - 20 seconds

  - 40 seconds

  - 60 seconds

  - Custom (specify your own interval)

<span class="Strong">Use Case:</span> Auto Refresh is especially useful for monitoring real-time scan activity or system changes.
