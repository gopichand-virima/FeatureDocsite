# Scheduled AD Import

After AD configuration, follow these steps:

1.  Click the **Schedule Import** icon to the right of the selected line item.

<span class="image placeholder" original-image-src="../../Resources/Images/Discovery/AD-User-Import-Logs/AD%20User%20Import%20Logs_1_8.png" original-image-title=""></span>

2.  The **Schedule AD Import** dialog box displays.

<span class="image placeholder" original-image-src="../../Resources/Images/Discovery/AD-User-Import-Logs/AD%20User%20Import%20Logs_1_9.png" original-image-title=""></span>

3.  Select <span class="Strong" style="font-weight: bold;">Enable Schedule Import</span> to turn on the import function.

4.  Select <span class="Strong" style="font-weight: bold;">Activate User Account</span> to activate all users.

5.  Select <span class="Strong" style="font-weight: bold;">Send Activation notification</span> to the applicable users when the import is complete.

- <span class="Strong" style="font-weight: bold;">Reflect AD Account Status</span> – Enable to match user login status with Azure AD.

- <span class="Strong" style="font-weight: bold;">Activate User Account</span> – Enable to make imported users active by default.

- <span class="Strong" style="font-weight: bold;">Send Activation Notification</span> – Enable to email users upon account activation.

6.  In the <span class="Strong" style="font-weight: bold;">AD Import Frequency</span> field, click the drop-down list and select the applicable import frequency, such as <span class="Strong" style="font-weight: bold;">Every Day, Every 2 days</span>, and so forth.

7.  In the <span class="Strong" style="font-weight: bold;">Type of Import</span> field, click the drop-down list and select the type of import, such as <span class="Strong" style="font-weight: bold;">All Users, New & Modified Users and New Users</span>

8.  In the schedule area, select <span class="Strong" style="font-weight: bold;">Timezone</span> and the appropriate schedule timing.

9.  When all selections/entries are made, click <span class="Strong" style="font-weight: bold;">Add</span>. (If you are editing the existing information, the Add button changes Update.)

10. After import completion, view import records in the <span class="Strong" style="font-weight: bold;">AD User Import Logs</span>.
