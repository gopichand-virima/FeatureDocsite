# Customizing Columns

1. In the [Azure AD User Import Logs]{.Strong} page, click the [Personalize Column icon]{.Strong} in the top-right corner.

![Azure AD User Import Logs 15](../../Resources/Images/Discovery/Azure-AD-Users-Import-Logs/Azure AD User Import Logs_15.png)

2. The [Personalize List Columns]{.Strong} window will appear.

 - The [left panel]{.Strong} shows available fields.

 - The [right panel]{.Strong} shows currently displayed fields.

3. Check or uncheck fields as needed.

4. Click [Save]{.Strong} to apply changes.

 - [Default View]{.Strong} resets columns to system default.

 - [Clear All]{.Strong} removes all selections for a fresh start.

![Azure AD User Import Logs 16](../../Resources/Images/Discovery/Azure-AD-Users-Import-Logs/Azure AD User Import Logs_16.png)
