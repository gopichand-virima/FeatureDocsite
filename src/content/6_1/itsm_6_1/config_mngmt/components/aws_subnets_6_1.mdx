# AWS Subnets

<div class="NumList1" madcap:autonum="1. ">

While viewing an asset record, click **Components**.

</div>

<div class="NumList2" madcap:autonum="2. ">

Click **AWS Subnets**.

</div>

<div class="Ind50">

<span class="image placeholder" original-image-src="../../../Resources/Images/Components/aws-subnets-new.png" original-image-title="" style="width: 621px;height: 164px;"></span>

</div>

## Add a New AWS Subnet

<div class="NumList1" madcap:autonum="1. ">

Click **Add**. The AWS Subnet dialog box displays.

</div>

<div class="Ind50">

<span class="image placeholder" original-image-src="../../../Resources/Images/Components/aws-subnets-add-new.png" original-image-title="" style="width: 472px;height: 209px;"></span>

</div>

<div class="NumList2" madcap:autonum="2. ">

Click **New AWS Subnet**

</div>

<div class="NumList2" style="font-style: normal;" madcap:autonum="3. ">

Enter the applicable information in each field.

</div>

<div class="NumList2" madcap:autonum="4. ">

When all entries are made, click **Save.**

</div>

## Modify Existing AWS Subnet

<div class="NumList1" madcap:autonum="1. ">

Click **Existing AWS Subnet**.

</div>

<div class="Ind50">

<span class="image placeholder" original-image-src="../../../Resources/Images/Components/aws-subnets-existing-new.png" original-image-title="" style="width: 471px;height: 319px;"></span>

</div>

<div class="NumList2" madcap:autonum="2. ">

Enter the search criteria to locate the existing device.  *See also* <a href="../../../Common%20Topics/Advanced-Search.htm" style="font-style: italic;">Advanced Search</a>*.*

</div>

<div class="NumList2" madcap:autonum="3. ">

Select the applicable device from the results list.

</div>

<div class="NumList2" madcap:autonum="4. ">

When all selections/entries are made, click **Save**.

</div>

## Delete AWS Subnet

While viewing the list of existing devices, select the device to delete, and click the **Delete** button.
