# Edit or Delete Probes

Use this function to edit or delete the configured values of an existing probe.

## Edit Probe

 -- ----- ------------------------------------------------------------------------------------------
 1. In the main window, select **Admin \> Discovery \> Probes**. The Probes window displays.
 -- ----- ------------------------------------------------------------------------------------------

 -- ----- --------------------------------------------------------------------------------------------------------------
 2. In the main window, click the **gear** icon and select **Discovery \> Probes**.  The Probes window displays.
 -- ----- --------------------------------------------------------------------------------------------------------------

 -- ----- --------------------------------------------------------------
 3. Select a record to edit. The *Details* information displays.
 -- ----- --------------------------------------------------------------

 -- ----- -------------------------------------------------------------
 4. Make changes, as necessary, to any of the available fields.
 -- ----- -------------------------------------------------------------

 -- ----- ------------------------------------------
 5. Modify any of the following information:
 -- ----- ------------------------------------------

::: Bullet2
[Tasks](/itsm/release_mngmt/tasks_6_1.mdx){.MCTopicPopup .MCTopicPopupHotSpot .a}

::: Bullet2
[Comments](/discovery/comments_6_1.mdx){.MCTopicPopup .MCTopicPopupHotSpot .a}

::: Bullet2
[Attachments](/discovery/attachments_6_1.mdx){.MCTopicPopup .MCTopicPopupHotSpot .a}

 -- ----- -------------------------------------------------------
 6. When all selections/entries are made, click **Save**.
 -- ----- -------------------------------------------------------

## Delete Probe

 --- --------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------
   Deleting is a permanent action and cannot be undone. Deleting may affect other functionality and information in the application such as data in configured reports, fields in windows, selectable options, etc. Therefore, be sure to understand the potential effects before making a deletion.
 --- --------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------

 -- ----- --------------------------------------------------
 1. Click the line that contains the item to delete.
 -- ----- --------------------------------------------------

 -- ----- ------------------------------------------------------------------------------------------------------------------------------------------------------
 2. From the *Select Actions* drop-down list, choose **Delete**. If a confirmation message is displayed, type **Delete** and click **Delete**to confirm.
 -- ----- ------------------------------------------------------------------------------------------------------------------------------------------------------
