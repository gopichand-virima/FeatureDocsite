# Accessing the Azure AD User Import Logs

1.  From the left navigation pane, go to <span class="Strong">Discovery Scan /> Azure AD User Import Logs</span>.

    <span class="image placeholder" original-image-src="../../Resources/Images/Discovery/Azure-AD-Users-Import-Logs/Azure%20AD%20User%20Import%20Logs_11.png" original-image-title=""></span>

<!-- -->

2.  The <span class="Strong" style="font-weight: bold;">AD User Import Logs</span> page opens.

    <span class="image placeholder" original-image-src="../../Resources/Images/Discovery/Azure-AD-Users-Import-Logs/Azure%20AD%20User%20Import%20Logs_12.png" original-image-title=""></span>

<!-- -->

3.  This shows a list of records.

    - <span class="Strong" style="font-weight: bold;">Key Information displayed</span>:

      - <span class="Strong" style="font-weight: bold;">Name</span> – The name used to identify the import (for example, Azure AD).

      - <span class="Strong" style="font-weight: bold;">Started On</span> – The date and time the import process began.

      - <span class="Strong" style="font-weight: bold;">Completed On</span> – The date and time the import process completed.

4.  Use the <span class="Strong" style="font-weight: bold;">filter fields</span> at the top of each column to search by name or narrow results by date range.
