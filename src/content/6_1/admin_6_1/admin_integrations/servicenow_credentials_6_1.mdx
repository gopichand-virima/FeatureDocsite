# ServiceNow Credentials

Virima provides multiple options for bi-directional exchange of CI data via the API with ServiceNow. Current API integrations occur between the following Virima and ServiceNow functions/processes:

- Virima Discovery to ServiceNow CMDB
- Virima CMDB to ServiceNow Incident Management
- Virima CMDB to ServiceNow Change Management
- Virima CMDB to ServiceNow CMDB

Use this function to configure ServiceNow credentials for integration to the application.

In the main window, click **Admin > Integrations > ServiceNow Credentials**.

In the *Username* field, type the login username.

In the *Password* field, type the password to associate with this login.

In the *Password* field, click **Change Password** and enter the password to associate with this login.

In the *Domain* field, enter the domain.

Click **Save**.
