# Tabs for Extended Information

- <span class="Strong" style="font-weight: bold;">Tasks</span>: Displays or allows creation of tasks linked to the import job.

- <span class="Strong" style="font-weight: bold;">Comments</span>: Provides space for administrators to add notes or review remarks from others.

- <span class="Strong" style="font-weight: bold;">Attachments</span>: Lets users upload or view related files, such as screenshots, reports, or documentation tied to the import.

Clicking on the "<span class="Strong" style="font-weight: bold;">New User Count</span>" will navigate you to the User Record page, where the AD User Import Logs are automatically filtered and the relevant results are displayed.
