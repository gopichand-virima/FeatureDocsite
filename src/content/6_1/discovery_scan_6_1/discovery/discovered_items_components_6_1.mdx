# Discovered Items: Components

Every configuration item has software and hardware associated with it. These components vary based on the [blueprint](../Admin%20-%20SACM/Blueprints.htm).

When a new CI component is added, an email notification is sent to those Roles configured as per the SACM RACI matrix and email preferences.

To access the functions, do the following:

<div class="NumList1" madcap:autonum="1. ">

From the navigation pane, select **Discovery Scan /> Discovered Items.** The Discovered Items window displays.

</div>

<div class="NumList2" madcap:autonum="2. ">

Select a record in the list. A new windows displays with the Details tab selected by default.

</div>

<div class="NumList2" madcap:autonum="3. ">

Click the **Components** tab (if displayed). The applicable tabs and subtabs for the selected asset display.

</div>

|     |                                                                                                 |
|-----|-------------------------------------------------------------------------------------------------|
|     | Some tabs can be edited while others are view only. In addition, some tabs have sub-components. |

To view a list of some available components, refer to [CI: Components List](../ITSM/Config%20Mngmt/Components/CI-Comp-List.htm).
