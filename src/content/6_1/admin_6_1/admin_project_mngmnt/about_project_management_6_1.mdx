# About Project Management

|     |                                                              |
|-----|--------------------------------------------------------------|
|     | Admin functions are only available to System Administrators. |

The Project Management application is a suite of tools that aids in managing projects, tasks, and resources. It provides the ability to create and manage projects of all sizes, from small projects with a few tasks to large portfolios of projects that contain complex tasks.

It also includes features to achieve project goals in alignment with the other activities your organization, such as integration with other modules in the Virima application, such as Change Management, Request Management and Release Management.
