# About Discovery Scan

This module contains the following functions:

<div class="Bullet1">

[Dashboard](../Dashboards/Dashboard-Overview.htm) contains a list of all the **recent** scans and any reports that have been added to the window. In this window, [new content](../Dashboards/Dashboard-Contents.htm) can be added and [customized](../Dashboards/Dashboard-Customization.htm).

</div>

<div class="Bullet1" madcap:conditions="Default.OldUI">

[Run a Scan](Run-a-Scan/Run-a-Scan.htm) identifies the devices connected to and communicating in a network and allows for the configuration of a [scan frequency](Scans-Frequency.htm).

</div>

<div class="Bullet1">

[Recent Scans](Recent-Scans/Scans-Recent.htm) displays successfully completed scans and creates a new scan to run.

</div>

<div class="Bullet1">

[Scheduled Scans](Scans-Scheduled.htm) enables the execution of scan functionality according to a configured date and time.

</div>

<div class="Bullet1">

[IPAM Networks](IPAM-Networks.htm) enables the planning, tracking, and managing the information associated with an Internet Protocol address existing in the monitored network environment.

</div>

<div class="Bullet1">

[Discovered Items](Discovered-Items.htm) displays the details for a successful scan.

</div>

<div class="Bullet1">

[Import from AWS](Import-AWS.htm) identifies information in Amazon Web Services (AWS).

</div>

<div class="Bullet1">

[Import from AZURE](Import-Azure.htm) identifies information in AZURE, a cloud service owned by Microsoft.

</div>

<div class="Bullet1">

[Import Data Files](Import-Data-File.htm) imports details of CI, CI relations, and asset details from an Excel spreadsheet.

</div>

<div class="Bullet1">

[Imported Assets](Imported-Assets.htm) displays the results of data imported during the data files import operation.

</div>

<div class="Bullet1" style="color: #dc143c;">

[AD User Import Logs](AD-User-Import-Logs.htm) <span style="color: #000000;">displays scanned results and the details for imported AD users.</span>

</div>

<div class="Bullet1" style="color: #000000;">

[Azure AD User Import Logs](Azure-AD-User-Import-Logs.htm) displays scanned results and the details for imported Azure AD user.

</div>

<div class="Bullet1" style="color: #dc143c;" madcap:conditions="Default.Omit">

[LDAP User Import Scans](LDAP-User-Import-Scans.htm) <span style="color: #000000;">displays scanned results and the details for imported LDAP Users.</span>

</div>
