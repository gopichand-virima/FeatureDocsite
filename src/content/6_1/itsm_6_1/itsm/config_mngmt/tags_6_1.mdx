# Tags

Use this function to tag the CIs (Parent CIs) during the scan when an instance(s) of these applications/software are found in the CI. This helps to easily search a CI/ Discovered asset in the CMDB.

In the navigation pane, select **ITSM /> Configuration Management /> Tags**. The Tags window displays.

<span class="image placeholder" original-image-src="../../Resources/Images/ITAM/Config-Mngmt/tags-main-old.PNG" original-image-title=""></span>

In the navigation pane, select <span class="Strong_4" style="font-weight: bold;">Configuration Management /> Tags. </span>The Tags window displays.

<span class="image placeholder" original-image-src="../../Resources/Images/ITAM/Config-Mngmt/tags-main-new.PNG" original-image-title=""></span>

New Tag

<div class="NumList2" madcap:conditions="Default.OldUI" madcap:autonum="1. ">

From the *Select Actions* drop-down list, choose an action: **New Tags**.

</div>

<div class="NumList2" madcap:conditions="Default.NewUI" madcap:autonum="2. ">

Click **New Tag**.

</div>

<div class="NumList2" madcap:autonum="3. ">

In the Add Tag window, enter a *Name* for the tag and select a *Color* from the color picker.

</div>

</div>

Edit Tag

To view a list of the CIs that are tagged, click **Tagged CIs**. Any listed record can be *Removed* or *Exported*.
