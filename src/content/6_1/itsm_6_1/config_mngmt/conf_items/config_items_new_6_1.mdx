# New Configuration Item (CI)

Use this function to configure a new CI by entering values for the parameters present.

 -- ----- ---------------------------------------------------------------------------------------------------------------------------------------
 1. In the navigation pane, select **ITSM \> Configuration Management \> Configuration Items**. The Configuration Items window displays.
 -- ----- ---------------------------------------------------------------------------------------------------------------------------------------

 -- ----- -----------------------------------------------------------------------------------------------------------------------
 1. In the navigation pane, select **ITAM \> Configuration Management \> CMDB**. The Configuration Items window displays.
