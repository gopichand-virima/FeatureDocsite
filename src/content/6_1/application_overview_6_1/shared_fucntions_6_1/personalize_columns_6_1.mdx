## Personalize Columns

| ![icon personalize list columns](../Resources/Images/Common%20Topics/icon-personalize-list-columns.png) | Use this function to personalize the columns displayed in the record window. |


1. While viewing a record list, click the **Personalize Columns** icon.

2. In the **Search for properties** field, type the name of any properties to search for, then click the **Search** icon.

3. Under the **Search Results** column, a list of all available columns is shown. A check mark indicates the column has been selected.


The list of available columns are predefined in the system and cannot be changed through the user interface.


1. Under the **SelectedÂ Items** column, a list of all currently selected columns is shown. To add an item to this list, click on the column name in the **Search Results** list. The selected item is now available in the **Selected Items** list.

2. Click **Default View** to load the default settings.

3. Click **Clear All** to remove all selections and return to the default view.

4. Click **Save** to retain the current selections.

5. Click **Cancel** to discard any changes.
