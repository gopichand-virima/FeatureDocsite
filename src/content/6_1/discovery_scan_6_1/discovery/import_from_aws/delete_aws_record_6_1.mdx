# Deleting AWS Record

- Deletes the currently selected imported AWS record.

- From the <span class="Strong">Select Actions</span> drop-down list, choose <span class="Strong">Delete</span>.

  <span class="image placeholder" original-image-src="../../Resources/Images/Discovery/Import-From-AWS/Import%20From%20AWS_1_10.png" original-image-title=""></span>

<!-- -->

- <span class="inline-comment-marker">Confirm the deletion and click </span><span class="Strong">Delete</span><span class="inline-comment-marker">.</span>

  <span class="image placeholder" original-image-src="../../Resources/Images/Discovery/Import-From-AWS/Import%20From%20AWS_1_11.png" original-image-title=""></span>

<!-- -->

- Deleting an item is permanent and cannot be undone. Understand the potential effects before deleting an item.

- Deleting an imported AWS Import run will deleted all the corresponding discovered items. if any discovered items through the selected AWS Import run is moved to CMDB, such records will not be deleted.
