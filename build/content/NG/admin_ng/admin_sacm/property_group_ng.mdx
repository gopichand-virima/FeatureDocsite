# Property Group

Use this function to add a new blueprint property group.

In the main window, click **Admin > SACM > Property Group**. The Property Group window displays.

Add Property Group

From the *Select Actions* drop-down list, choose **New Property Group**.

Enter a *Group Name* and *Group Order*.

When all selections/entries are made, click **Add**.
