# Edit Client

Use this function to edit details for an existing client.

In the main window, click **Admin > Discovery > Client**. The Client window displays.

Select an existing client record. A new window displays with the **Details** tab selected.

Modify the applicable data.

When all selections/entries are made, click **Save**.
