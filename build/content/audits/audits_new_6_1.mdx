# New Audit

Use this function to add an audit process.

In the navigation pane, select **ITSM > Configuration Management > Audits**. The Audits window displays.

From the *Select Actions* drop-down list, choose **New**. The New Audits window displays.

In the navigation pane, select **Configuration Management > Audits**. The Audits window displays.

Click **New Audit** to add a new audits process. The New Audit window displays.

In the *Name* field, type a name for this audit process.

In the *Description* field, type a description.

In the *Auditor* field, click **Add**. The Auditor dialog box displays.

In the *Auditor* field, click **+** to add an auditor. The Auditor dialog box displays.

In the *Search* field, type all or part of the name to search for. *See also* [Advanced Search](/common_topics/advanced_search_6_1.mdx)

When the applicable auditor displays, click the radio button next to the auditor's name.

Click **Save**.

Click **Add**.

In the *Checked By* field, click **Add+**. The Checked By dialog box displays.

! !

In the *Search* field, type all or part of the name to search for. *See also* [Advanced Search](/common_topics/advanced_search_6_1.mdx)*.*

Click the **Search** icon.

When the applicable auditor displays, click the radio button next to the auditor's name.

Click **Save**.

Click **Add**.

Select **Run a Scan** to trigger a validation type probe scan on a device. Based on the result of the scan, the audit is marked as passed or failed.

From the *Audit Result* drop-down list, choose **Pass**, **Fail** or **In Progress**.

From the *Select Probe* drop-down list, choose the applicable option.

From the *Scan Frequency* drop-down list, choose the applicable frequency of the scan to be performed.

To add configuration items for this audit, in the Configuration Items section, click **Add**. The Configuration Items dialog box displays.

! !

In the *Blueprint* field, click the drop-down list and select the applicable blueprint.

In the *Search* field, enter the search criteria. *See also* [Advanced Search](/common_topics/advanced_search_6_1.mdx)*.*

Under the *Search Results*, select the applicable search result(s) to move it to the *Selected Items* list.

Click **Save**. The added configuration item displays.

Add any additional configuration items.

When all selections/entries are made, click **Add**.
