# Designations

Use this function to categorize users.

In the main window, click **Admin > Organizational Details > Designations**. The Designations Window displays.

!

!


### New Designation

To create a new designation:

- From the *Select Actions* drop-down list, choose **New Designation**. The New Designation window displays.

!

- Enter a *Name* and *Description*. The *Name* field is required.
- Click **Add**.


### Edit Designation

To edit an existing designation:

- Click a record in the list. The Details tab displays.
- Modify the *Name* and/or *Description*.
- View, add, or modify any of the following:
- [Tasks](/itsm/release_mngmt/tasks_6_1.mdx)
- [Comments](/discovery/comments_6_1.mdx)
- [Attachments](/discovery/attachments_6_1.mdx)
- When all selections/entries are made, click **Save**.


### Delete Designation

To delete an existing designation:

- Highlight the designation to delete. The Details tab displays.
- FFrom the *Select Actions* drop-down list, choose **Delete**. If a confirmation message is displayed, type **Delete** and then click **Delete**.
