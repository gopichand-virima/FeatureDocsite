# Agent Schedules

Use this function to trigger a scheduled scan on the remote machine where an agent is installed.

<div class="NumList1" madcap:autonum="1. ">

While viewing the client window, select a line item.

</div>

<div class="NumList2" madcap:autonum="2. ">

In the Details window, click **Agent Schedules**.

</div>

<div class="NumList2" madcap:autonum="3. ">

To add an expression, click **Add Cron Expression**. The Add Cron for Agents Schedules window displays.

</div>

<div class="Ind50" madcap:autonum="">

<span class="image placeholder" original-image-src="../Resources/Images/Admin%20-%20Discovery/add-cron-agent-schedules-old.PNG" original-image-title=""></span>

</div>

<div class="NumLista" madcap:autonum="a. ">

Select a pre-defined *Scan Frequency* (every hour, every 2 hours, and so forth).

</div>

<div class="numListb" madcap:autonum="b. ">

Specify if this schedule **Is Recurring**.

</div>

<div class="numListb" madcap:autonum="c. ">

Enter the time components such as second, minute, hour, and so forth.

</div>

<div class="numListb" madcap:autonum="d. ">

When all selections/entries are made, click **Add Cron**.

</div>

<div class="NumList2" madcap:autonum="4. ">

To remove an item, select an existing line, and click **Delete Cron Expression**.

</div>
