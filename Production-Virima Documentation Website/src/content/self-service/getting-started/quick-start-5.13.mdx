# Self Service - Quick Start Guide

Get started with the Self Service portal quickly.

## Prerequisites

- Self Service portal access (typically all users)
- Valid Virima account credentials
- Modern web browser

## Quick Start Steps

### Step 1: Access the Portal
1. Navigate to the Self Service portal URL
2. Log in with your credentials
3. Review the portal dashboard

### Step 2: Browse Service Catalog
1. Click on "Service Catalog"
2. Browse available services by category
3. Use search to find specific services
4. Review service details and requirements

### Step 3: Submit a Request
1. Select desired service
2. Fill out the request form
3. Add attachments if needed
4. Submit the request
5. Note the request number for tracking

### Step 4: Search Knowledge Base
1. Click on "Knowledge Base"
2. Enter search terms
3. Browse relevant articles
4. Rate articles for helpfulness
5. Provide feedback
