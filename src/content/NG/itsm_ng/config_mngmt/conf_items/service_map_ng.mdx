# Business Service Map

Use the Business Service Map to graphically display the configuration items that support a business service. For example, delivering email to employees of an organization is supported by exchange servers and web servers.

The following lists the available views.

::: Bullet1
**[ADM View](BSM%20Examples/BSM-ADM-View.mdx){style="font-weight: normal;"}** displays the graphical connection between the applications or between applications and configuration items based on the relationship *Communicates With*.

- **Affected Business Application View** displays the list of services/applications impacted due to an affected Business Service.
::: Bullet1
**[Communication View](BSM%20Examples/BSM-Comm-View.mdx){.MCTopicPopup .MCTopicPopupHotSpot .a}** displays the graphical connection between the configuration items based on the relationship *Communicates With*.

- [[**Infrastructure View** displays the end-to-end visualization of a Server/Application infrastructure.]{style="color: #000000;"}]{style="color: #dc143c;"}
::: Bullet1
**[Network View](BSM%20Examples/BSM-Network-View.mdx){style="font-weight: normal;"}** displays the physical and non-physical connections among the assets and/or business services based on relationships.

::: Bullet1
**Service Topology View** displays the end-to-end visualization of a running service.

The following actions can be performed in this window.

![sm menu top 665x44](../../../Resources/Images/ITSM/Biz-Svc-Map/sm-menu-top_665x44.png)

![sm menu bottom 379x38](../../../Resources/Images/ITSM/Biz-Svc-Map/sm-menu-bottom_379x38.png)

**Type.** Click the drop-down list and select the type of view to display.

**Rescan.** When selected. displays the Rescan Now dialog box. Choose a *Client* and the *Type of Scan*. Then click **Scan**.

![rescan now dialog box 476x212](../../../Resources/Images/ITSM/rescan-now-dialog-box_476x212.png)

**Host Check**. When selected, displays the Check Host Reachability dialog box. Enter the *IP Address* and *Choose a client*, then click **Scan**.

![check host reachability 473x216](../../../Resources/Images/ITSM/Biz-Svc-Map/check-host-reachability_473x216.png)

**Advanced Settings**. When selected, displays a new window for selecting the direction of the view (horizontal or vertical) and the number of levels you want to display.\
![icon adv settings](../../../Resources/Images/ITSM/Biz-Svc-Map/icon-adv-settings.png)\

![advanced settings 251x221](../../../Resources/Images/ITSM/Biz-Svc-Map/advanced-settings_251x221.png)

**Minimize** and **Maximize**. Toggles between making the screen full screen or normal view.

![icon maximize](../../../Resources/Images/ITSM/Biz-Svc-Map/icon-maximize.png) ![icon minimize](../../../Resources/Images/ITSM/Biz-Svc-Map/icon-minimize.png)

**Load View**. When selected, loads any saved views.

![icon load view](../../../Resources/Images/ITSM/Biz-Svc-Map/icon-load-view.png)

**Save View**. When selected, saves the current view, which can be reloaded for reference at another time.

![icon save view](../../../Resources/Images/ITSM/Biz-Svc-Map/icon-save-view.png)

**Download BSM**. When selected, displays the Download BSM View dialog box in which you specify what type of file to download. Click either **Save as PNG** (image) or **Save Data as Excel** (spreadsheet), then click **Download**.

 --- --------------------------------------------------------
   There must be data in order for the download to occur.
 --- --------------------------------------------------------

![icon download view](../../../Resources/Images/ITSM/Biz-Svc-Map/icon-download-view.png)

![download bsm view 476x156](../../../Resources/Images/ITSM/Biz-Svc-Map/download-bsm-view_476x156.png)

**Show/Hide Legends**. Toggles between showing and hiding the onscreen data legends.

![icon show legends](../../../Resources/Images/ITSM/Biz-Svc-Map/icon-show-legends.png)

![icon hide legends](../../../Resources/Images/ITSM/Biz-Svc-Map/icon-hide-legends.png)

**Zoom**. Changes the size of the data view when moving the slider. Use the plus + and minus - signs to change the slider location.

![zoom slider](../../../Resources/Images/ITSM/Biz-Svc-Map/zoom-slider.png)
