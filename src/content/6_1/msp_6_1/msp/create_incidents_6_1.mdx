# Create Incidents

Use this function to create an incident.

<div class="NumList1" madcap:autonum="1. ">

In the MSP window, locate the applicable client, and click **Create Incident**. The Incidents window displays.

</div>

<div class="Ind50">

<span class="image placeholder" original-image-src="../Resources/Images/MSP/create-incident-main.PNG" original-image-title=""></span>

</div>

<div class="NumList2" madcap:autonum="2. ">

On the *Primary Details* tab, enter the applicable information for each section. *Refer to* <a href="../ITSM/Incident-Mngmt/Incidents.htm" style="font-style: italic;">ITSM &gt; Incident Management &gt; Incidents</a> *for more information*.

</div>

<div class="NumList2" madcap:autonum="3. ">

If applicable, choose a template from the *Select* <a href="../Admin%20-%20Incident%20Mngmnt/Incident-Model.htm" style="font-style: italic;">Admin &gt; Incident Management &gt; Incident Model</a> drop-down list.

</div>

|     |                                                                    |
|-----|--------------------------------------------------------------------|
|     | Only users with the Admin permission can create an Incident Model. |

<div class="NumList2" madcap:autonum="4. ">

When all selections/entries are made, click **Add**.

</div>

## Select Action Options

The following actions can be performed for this incident:

### **Change Attributes**

<div class="NumList1" madcap:autonum="1. ">

Select a record in the list.

</div>

<div class="NumList2" madcap:autonum="2. ">

From the Select Actions drop-down list, choose **Change Attributes**.

</div>

<div class="NumList2" madcap:autonum="3. ">

At the confirmation prompt, click **OK**.

</div>

<div class="NumList2" madcap:autonum="4. ">

In the Change Attributes window, make the necessary changes.

</div>

<div class="NumList2" madcap:autonum="5. ">

When all selections/entries are made, click **Update**.

</div>

### Move to Different Client

<div class="NumList1" madcap:autonum="1. ">

Select a record in the list.

</div>

<div class="NumList2" madcap:autonum="2. ">

From the Select Actions drop-down list, choose **Move to Different Client**.

</div>

<div class="NumList2" madcap:autonum="3. ">

In the Client dialog box, search for a select a client. Then click **Save**. The record is moved to the different client.

</div>
