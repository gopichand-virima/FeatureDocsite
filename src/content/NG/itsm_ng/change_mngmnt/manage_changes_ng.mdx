# Manage Changes

The Manage Changes interface provides a centralized location for viewing, tracking, and managing all change requests in your organization.

## Main Change List View

The main change list displays all change requests with the following key information:

### Column Descriptions

- **Change Number**: Unique identifier for the change request
- **Title**: Short description of the change
- **Type**: Normal, Standard, or Emergency change
- **Status**: Current state (Draft, Pending Approval, Scheduled, Implementing, Completed, etc.)
- **Priority**: Impact and urgency combination (Critical, High, Medium, Low)
- **Requester**: Person who submitted the change
- **Assigned To**: Person or group responsible for implementing the change
- **Scheduled Start**: Planned start date and time
- **Scheduled End**: Planned completion date and time
- **Category**: Change category (Infrastructure, Application, Network, etc.)

## Filtering Changes

Use the filter panel to narrow down the change list:

1. Click the **Filter** button in the toolbar
2. Select filter criteria:
   - Status
   - Type
   - Priority
   - Date range
   - Assigned group
   - Category
3. Click **Apply** to update the list

### Saved Filters

Create and save custom filters for frequently used views:

1. Configure your desired filters
2. Click **Save Filter**
3. Enter a name for the filter
4. The filter will appear in the Quick Filters dropdown

## Sorting and Grouping

### Sort Options
Click on any column header to sort by that field. Click again to reverse the sort order.

### Group By
Group changes by common attributes:

1. Click the **Group By** dropdown
2. Select grouping criteria (Status, Priority, Assigned Group, etc.)
3. The list will reorganize into collapsible groups

## Bulk Actions

Perform actions on multiple changes simultaneously:

1. Select changes using the checkboxes
2. Click the **Actions** dropdown
3. Choose from available options:
   - Update Status
   - Reassign
   - Change Priority
   - Add to CAB Meeting
   - Export Selected

## Quick Actions

From the change details screen, use quick action buttons:

- **Edit**: Modify change details
- **Submit for Approval**: Send to approvers
- **Approve/Reject**: For authorized approvers
- **Schedule**: Set implementation date/time
- **Implement**: Begin implementation
- **Complete**: Mark change as completed
- **Close**: Close the change record
- **Cancel**: Cancel the change request

## Change Lifecycle Management

### Draft State
- Initial creation
- Information gathering
- Planning phase

### Pending Approval
- Submitted for review
- Awaiting approvals
- CAB review

### Scheduled
- Approved for implementation
- Date and time set
- Resources allocated

### Implementing
- In progress
- Tasks being executed
- Monitoring progress

### Review
- Implementation completed
- Post-implementation review
- Verification

### Closed
- Change completed successfully
- Documentation updated
- Lessons learned captured

## Search Capabilities

### Quick Search
Use the search bar to find changes by:
- Change number
- Title keywords
- Requester name
- CI name

### Advanced Search
Access advanced search for complex queries:
1. Click **Advanced Search**
2. Build search criteria using multiple fields
3. Combine with AND/OR logic
4. Save search for future use

## Export Options

Export change data for reporting:

1. Select changes or apply filters
2. Click **Export**
3. Choose format:
   - Excel (XLSX)
   - CSV
   - PDF Report
4. Select fields to include
5. Click **Generate Export**
