# Active Integrations

Use this function to manage, monitor, and troubleshoot active integrations with external systems.

Active Integrations provides centralized visibility and control over all configured integrations, their status, performance, and sync activity.

In the main window, click **Admin > Integration > Active Integrations**. The Active Integrations window displays.

## Integrations Overview

View all active integrations:

**Integration Cards**:

- **Integration Name**: Connected system
- **Status**: Active, Paused, Error, Disconnected
- **Last Sync**: Last successful sync time
- **Sync Status**: Success, Warning, Failed
- **Records Synced**: Count of synced records
- **Health**: Overall health indicator

**List View Columns**:

- **Name**: Integration name
- **Type**: Integration type
- **Status**: Current status
- **Direction**: One-way or Two-way
- **Last Sync**: Timestamp
- **Success Rate**: Percentage
- **Errors**: Error count
- **Actions**: Quick actions

**Pause Options**:

- **Notify**: Notify integration stakeholders
