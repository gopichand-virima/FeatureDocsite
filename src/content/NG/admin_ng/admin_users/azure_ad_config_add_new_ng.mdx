# New Azure AD Configuration

While viewing the ADÂ Configuration window, click **New AD Configuration**. The Export dialog box displays.

Enter the *Name*, *Client Id*, *Tenant Id* and *Secret Key*.

Click **Next**. The Map Property dialog box displays.

Select required values to import (for example, First Name, Last Name and so forth).

Click the drop-down list arrow and select the property to map to.

When all selections/entires are made, click **Finish**.
