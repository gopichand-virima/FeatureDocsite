# Service Requests

Use this function to create and manage service requests.

In the navigation pane, select **ITSM > Request Fulfillment > Service Requests**. The Service Requests window displays.

New Service Request

From the *Select Actions* drop-down list, choose **New Service Request**.

Click **New**.

Complete the fields.

When all selections/entries are made, click **Add**.
