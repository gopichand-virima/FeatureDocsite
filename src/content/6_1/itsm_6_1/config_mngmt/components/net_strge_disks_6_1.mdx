# Network Storage Disks

<div class="NumList1" madcap:autonum="1. ">

While viewing an asset record, click **Components**.

</div>

<div class="NumList2" madcap:autonum="2. ">

Click **Network Storage Disks**.

</div>

## Add a New Network Storage Disk

<div class="NumList1" madcap:autonum="1. ">

Click **Add**. The Network Storage Disk dialog box displays.

</div>

<div class="Ind50">

<span class="image placeholder" original-image-src="../../../Resources/Images/Components/network-storage-disks-add-new.png" original-image-title="" style="width: 482px;height: 217px;"></span>

</div>

<div class="NumList2" madcap:autonum="2. ">

Click **New Network Storage Disk.**

</div>

<div class="NumList2" style="font-style: normal;" madcap:autonum="3. ">

Enter the applicable field information.

</div>

<div class="NumList2" madcap:autonum="4. ">

When all selections/entries are made, click **Save**.

</div>

## Modify Existing Network Storage Disk

<div class="NumList1" madcap:autonum="1. ">

Click **Existing Network Storage Disk**.

</div>

<div class="Ind50">

<span class="image placeholder" original-image-src="../../../Resources/Images/Components/network-storage-disks-existing-new.png" original-image-title="" style="width: 479px;height: 328px;"></span>

</div>

<div class="NumList2" madcap:autonum="2. ">

Enter the search criteria to locate the existing device.  *See also* <a href="../../../Common%20Topics/Advanced-Search.htm" style="font-style: italic;">Advanced Search</a>*.*

</div>

<div class="NumList2" madcap:autonum="3. ">

Select the applicable device from the results list.

</div>

<div class="NumList2" madcap:autonum="4. ">

When all selections are made, click **Save**.

</div>

## Delete Network Storage Disk

While viewing the list of existing devices, select the device to delete, and click the **Delete** button.
