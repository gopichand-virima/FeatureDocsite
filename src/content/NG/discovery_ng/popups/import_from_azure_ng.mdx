# Import from Azure

Use this function to identify information existing in AZURE, a cloud service owned by Microsoft.

In the navigation pane, select **Discovery Scan > Import from Azure**. The Import from AZURE window displays.

From the *Select Options* drop-down box, choose **Import**. The Import from AZURE dialog box displays.

Click on the **Import** icon. The Import from Azure dialog box displays.

In the *Name* field, enter a name for the credential.

In the *Credential* field, click the drop-down list and select the type of credential, such as established in Azure.

In the *Credential* field, click the drop-down list and select the type of credential, such as Azure Subscription.

Select the information to import.

Click **Add** to import the information. Otherwise, click **Cancel** to cancel the operation.
