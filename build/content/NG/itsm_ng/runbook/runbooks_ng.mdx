# Runbooks

Use this function to create and manage runbooks.

In the navigation pane, select **ITSM > Runbook > Runbooks**. The Runbooks window displays.

New Runbook

From the *Select Actions* drop-down list, choose **New Runbook**.

Click **New**.

Complete the fields.

When all selections/entries are made, click **Add**.
