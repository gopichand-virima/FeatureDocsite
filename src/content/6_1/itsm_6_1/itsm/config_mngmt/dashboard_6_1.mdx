# Configuration Management: Dashboard

Use this function to view different types of data. The dashboard can be customized.

<div class="NumList2" madcap:autonum="1. ">

In the navigation pane, select **Configuration Management /> Dashboard**. The Configuration Management Dashboard window displays.

</div>

<div class="NumList2" madcap:conditions="Default.OldUI" madcap:autonum="2. ">

Click **View All** to open the applicable functions for the selected *items*.

</div>

<div class="NumList2" madcap:conditions="Default.NewUI" madcap:autonum="3. ">

Click **Show All** to open the applicable function for the selected *record*.

</div>

<div class="NumList2" madcap:autonum="4. ">

To add content to the dashboard, click [Add Contents](../../My%20Dashboard/Dashboard-Add-Contents-New.htm).

</div>

<div class="NumList2" madcap:autonum="5. ">

To add a new configuration item, click **New CI**. The Add New CI dialog box displays.

</div>

<div class="Ind50" madcap:conditions="Default.NewUI">

Any of the following options are available (and determined by report type):

</div>

<div class="Bullet2" madcap:conditions="Default.NewUI">

**Save Image**. Saves the selected report as an image file.

</div>

<div class="Bullet2" madcap:conditions="Default.NewUI">

**Export**. The information is automatically downloaded. An email confirmation is sent when the download is complete.

</div>

<div class="Bullet2" madcap:conditions="Default.NewUI">

**Complete Report**. *This function is under development.*

</div>

<div class="Bullet2" madcap:conditions="Default.NewUI">

**Delete**. Removes the current report from the dashboard.

</div>
