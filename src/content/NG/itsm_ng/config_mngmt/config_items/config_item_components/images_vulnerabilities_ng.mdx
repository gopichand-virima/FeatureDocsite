# Images and Vulnerabilities

 --- -------------------------------------------------------------------------------------------------------------
   This function is only available with certain asset types. Therefore, the relevant tab may not be displayed.
 --- -------------------------------------------------------------------------------------------------------------

[[Use this function to]{style="color: #000000;"} [retrieve all the docker images inside AWS ECR ( Elastic Container Registry) repositories. These images can contain vulnerabilities, which can be scanned and shown on the Vulnerabilities tab.]{style="color: #000000;"}]{style="color: #dc143c;"}

 -- ----- -------------------------------------------------------------------------
 1. In the main window, select **ITSM \> Confirmation Management \> CMDB**.
 -- ----- -------------------------------------------------------------------------

 -- ----- ----------------------------------------------------
 2. Search for and select the applicable asset record.
 -- ----- ----------------------------------------------------

 -- ----- ------------------------------------------------------------------------
 3. When the asset record window displays, click the **Repositories** tab.
 -- ----- ------------------------------------------------------------------------

 -- ----- ----------------------------------
 4. Select a repository in the list.
 -- ----- ----------------------------------

 -- ----- -------------------
 5. Click **Images**.
 -- ----- -------------------

::: Ind50
![repositories main old](../../../../Resources/Images/Components/repositories-main-old.PNG)

 -- ----- -----------------------------------------------------------
 6. Chick in the checkbox to select a repository in the list.
 -- ----- -----------------------------------------------------------

 --- ---------------------------------------------------------------------------------------------------------------------------------
   If you click on a line item, a new window displays and provides details for the selected image, along with any Vulnerabilities.
 --- ---------------------------------------------------------------------------------------------------------------------------------

 -- ----- -------------------------------------------------------------------------------
 7. Click **Scan**. A confirmation displays if the scan was started successfully.
 -- ----- -------------------------------------------------------------------------------

 -- ----- ----------------------------------------------------------------------------------------------
 8. While viewing the list of images, from the drop-down list, and select any of the following:
