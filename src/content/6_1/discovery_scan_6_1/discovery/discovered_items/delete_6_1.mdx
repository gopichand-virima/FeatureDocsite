# <span class="Strong">Delete</span>

- Select one or more checkboxes

- Click on <span class="Strong">Select Actions </span>and then Select <span class="Strong">Delete.</span>

- System will display a confirmation message as below. Read the Message.

  - Type <span class="Strong">Delete </span>in the text box.

  - Click on <span class="Strong">Delete </span>to permanently delete the selected record(s).

    - Selected records will be permanently deleted.

    - User will be redirected to the Discovered Items list page.

  - Click on <span class="Strong">Cancel </span>to abort.

    - Selected records will be retained.
