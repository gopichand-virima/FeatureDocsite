# Configure Services Account

Use this function to configure service accounts with admin privileges on both the host being scanned and the server where the application is being installed.

 -- ----- ---------------------------------
 1. Open the **Services.msc** file.
 -- ----- ---------------------------------

 -- ----- -------------------------------------------------------------------------------------------------------------------------------------------------
 2. [Double-click ]{.span_1}[on **Virima Discovery Application**]{.span_2}[. The Virima *Discovery Agent Properties dialog box* displays.]{.span_1}
 -- ----- -------------------------------------------------------------------------------------------------------------------------------------------------

::: Ind50
![discovery agent properties](../Resources/Images/Discovery/discovery-agent-properties.png)

 -- ----- ------------------------------------------------------------
 3. Provide the Administrator credential information as shown.
 -- ----- ------------------------------------------------------------

 -- ----- --------------------------------------------------
 4. Click **OK** to restart the application service.
 -- ----- --------------------------------------------------

 --- --------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------
   [The Discovery Application is packed within a Java runtime environment. However, Discovery can be pointed to a different JRE through ]{.span_1}[Settings]{.span_2}[ \> ]{.span_1}[Java Configuration.]{.span_2}
 --- --------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------
