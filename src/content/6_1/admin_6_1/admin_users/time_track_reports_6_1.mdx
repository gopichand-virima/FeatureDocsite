# Time Track Reports

Use this function to schedule a time and date to send a notification report on the user(s) or user group working on raised requests (for example, assigned Incidents, problems, etc.)

In the main window, click **Admin /> Users /> Time Track Reports**. The Time Track Reports window displays.

<span class="image placeholder" original-image-src="../Resources/Images/Admin%20-%20Users/time-track-report-main-old.PNG" original-image-title="" style="width: 662px;height: 138px;"></span>

New Schedule Report

<div class="NumList1" madcap:conditions="Default.OldUI" madcap:autonum="1. ">

While viewing the Time Track Reports window, from the *Select Actions* drop-down list, choose **New Schedule Report**. The New Schedule Report window displays.

</div>

<div class="Ind50" madcap:conditions="Default.OldUI">

<span class="image placeholder" original-image-src="../Resources/Images/Admin%20-%20Users/time-track-rpt-old.PNG" original-image-title="" style="width: 357px;height: 176px;"></span>

</div>

<div class="NumList2" madcap:autonum="2. ">

In the *Report Name* field, type a name to identify this report.

</div>

<div class="NumList2" madcap:autonum="3. ">

In the *Module* field, click the drop-down list and select the applicable module. <span style="color: #dc143c;" madcap:conditions="Default.Omit"> </span>

</div>

<div class="NumList2" madcap:conditions="Default.OldUI" madcap:autonum="4. ">

In the *Select User* field, click **Add**, search for and select a user. Then click **Save**.

</div>

<div class="emdash1" madcap:conditions="Default.OldUI" madcap:autonum="- ">

To remove a user, click the **x** next to the user's name.

</div>

<div class="NumList2" madcap:autonum="5. ">

In the *Frequency* field, click the drop-down list, and select when the time tracking report should be generated.

</div>

<div class="NumList2" madcap:conditions="Default.OldUI" madcap:autonum="6. ">

In the *Send Report To* field, click **Add**. The Send Report to dialog box displays.

</div>

<div class="NumLista" madcap:conditions="Default.OldUI" madcap:autonum="a. ">

Select the user, role, group, etc. to which the report is to be sent.

</div>

<div class="NumListb" madcap:conditions="Default.OldUI" madcap:autonum="b. ">

Click **Save**

</div>

<div class="NumList2" madcap:autonum="7. ">

When all selections/entries are made, click **Add**.

</div>

Generate Report

To generate a report:

<div class="NumList1" madcap:conditions="Default.OldUI" madcap:autonum="1. ">

While viewing the Time Track Reports window, from the *Select Actions* drop-down list, choose **Generate Report**. The Generate Report dialog window displays.

</div>

<div class="Ind50" madcap:conditions="Default.OldUI">

<span class="image placeholder" original-image-src="../Resources/Images/Admin%20-%20Users/time-track-gen-rpt-old.PNG" original-image-title=""></span>

</div>

<div class="NumList2" madcap:conditions="Default.NewUI" madcap:autonum="2. ">

Click the *Module* field drop-down list and select the applicable module.

</div>

<div class="NumList2" madcap:conditions="Default.OldUI" madcap:autonum="3. ">

To select a user, click **Add,** search for and select the applicable user. Then click **Save**.

</div>

<div class="NumList2" madcap:autonum="4. ">

Enter the *Start Date* and *End Date*.

</div>

<div class="NumList2" madcap:autonum="5. ">

When all selections/entries are made, click **Report Details**. The Report window displays and shows the applicable results.

</div>
