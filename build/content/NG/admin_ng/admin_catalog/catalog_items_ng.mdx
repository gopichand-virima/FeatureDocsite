# Catalog Items

Use this function to create and manage service catalog items available to end users.

Catalog Items represent the services that users can request through the self-service portal.

In the main window, click **Admin > Service Catalog > Catalog Items**. The Catalog Items window displays.

## Catalog Item List

View all catalog items:

**Item Columns**:

- **Item Name**: Service name
- **Category**: Service category
- **Status**: Active, Draft, Retired
- **Price**: Cost (if applicable)
- **Requests**: Total requests
- **Avg Fulfillment**: Average completion time
- **Rating**: User satisfaction rating

## Create Catalog Item

From the *Select Actions* drop-down list, choose **New Catalog Item**.

### Basic Information

**Item Details**:

- **Item Name**: New Laptop Request
- **Short Description**: Request a new laptop for employee
- **Category**: Hardware > End User Devices
- **Status**: ☑ Active
- **Item Image**: Upload service icon/image

**Availability**:

- **Available To**: All Users, Specific Roles, Specific Departments
- **Available Locations**: All, Specific Offices
- **Business Hours Only**: ☐ No

### Request Form

**Form Fields**:

| Field | Type | Required |
|-------|------|----------|
| Laptop Type | Dropdown | Yes |
| Operating System | Radio | Yes |
| RAM | Dropdown | Yes |
| Hard Drive | Dropdown | Yes |
| Business Justification | Text Area | Yes |
| Delivery Location | Text | Yes |
| Preferred Delivery Date | Date | No |

**Laptop Type Options**:

- Standard Business Laptop - $1,200
- High Performance Laptop - $2,500
- Developer Workstation - $3,000

### Pricing

**Cost Information**:

- **Base Price**: $1,200
- **Recurring Cost**: $0/month
- **Setup Fee**: $50
- **Variable Pricing**: Based on selections

**Budget Code**:

- Require budget/cost center code: ☑ Yes
- Validate against available budget: ☑ Yes

### Approval Workflow

**Approval Process**:

1. **Manager Approval**: User's direct manager
2. **Budget Approval**: If cost > $1,000
3. **IT Approval**: IT asset manager
4. **Auto-Approve**: If cost < $500

**Approval Timeout**:

- Reminder after: 2 days
- Escalate after: 5 days
- Auto-reject after: 10 days (no response)

### Fulfillment

**Assignment**:

- **Fulfillment Group**: IT Hardware Team
- **Assignment Rule**: Round robin, Load balanced

**Fulfillment Tasks**:

1. **Procure Device** (3 days)
2. **Image and Configure** (1 day)
3. **Quality Check** (0.5 days)
4. **Deliver to User** (0.5 days)

**SLA**:

- Target fulfillment: 5 business days
- Maximum time: 10 business days

### Notifications

**User Notifications**:

- Request submitted confirmation
- Approval status updates
- Request assigned notification
- Fulfillment status updates
- Completion notification

**Fulfiller Notifications**:

- New request assigned
- Request status changes
- SLA warnings

Click **Save**.

## Catalog Item Templates

**Common Service Templates**:

- Hardware Requests (laptop, monitor, phone)
- Software Access (application licenses)
- Account Requests (system access, email)
- Onboarding/Offboarding
- Conference Room Booking
- Travel Arrangements
