# XtremIO Clusters

Click **XtremIO Clusters**.

! Add XtremIO Cluster

Click **Add**. The XtremIO Cluster dialog box displays.

Click **New XtremIO Cluster**

Enter the applicable information in each field.

When all entries are made, click **Save.**

Edit XtremIO Cluster

Click **Existing XtremIO Cluster**.

Enter the search criteria to locate the existing device.  *See also* [Advanced Search](/common_topics/advanced_search_ng.mdx)*.*

Select the applicable device from the results list.

When all selections/entries are made, click **Save**.
