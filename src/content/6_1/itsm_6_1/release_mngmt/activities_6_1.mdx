# Activities

Use this function to configure conditions to perform set of actions in measurable amount of work for the Release.

In the main window, select **ITSM /> Release Management /> Activities**.

<span class="image placeholder" original-image-src="../../Resources/Images/ITSM/Release%20Management/activities-main-old.PNG" original-image-title=""></span>

<span class="image placeholder" original-image-src="../../Resources/Images/ITSM/Release%20Management/activities-main-new.PNG" original-image-title=""></span>

New Activities

<div class="NumList1" madcap:conditions="Default.OldUI" madcap:autonum="1. ">

From the *Select Actions* drop-down list, choose **New Release Activity**.

</div>

<div class="Ind50" madcap:conditions="Default.OldUI">

<span class="image placeholder" original-image-src="../../Resources/Images/ITSM/Release%20Management/activities-new-old.PNG" original-image-title=""></span>

</div>

<div class="NumList2" madcap:conditions="Default.NewUI" madcap:autonum="2. ">

Click **New**.

</div>

<div class="Ind50" madcap:conditions="Default.NewUI">

<span class="image placeholder" original-image-src="../../Resources/Images/ITSM/Release%20Management/activities-new-new.PNG" original-image-title=""></span>

</div>

<div class="NumList2" madcap:autonum="3. ">

Complete the fields, referring to the information below.

</div>

<div class="Bullet2">

**Title**. Enter the name for the activity.

</div>

<div class="Bullet2">

**Description**. Enter a description.

</div>

<div class="Bullet2">

**Assignee/Group Assignee**. Click **Add**the **plus + sign**, then search for and select the applicable user.

</div>

<div class="Bullet2">

**Status.**<span style="font-weight: normal;">Select a status for this activity. </span>

</div>

<div class="Bullet2">

**Priority**. Select a priority for this activity.

</div>

<div class="Bullet2">

**Start/End Dates**. Enter the start and end dates of this activity.

</div>

<div class="Bullet2">

**Work Notes**. Enter any notes associated with this activity.

</div>

<div class="Bullet2">

**Release Tasks**. Click **Add**the **plus + sign**, then search for and select the applicable task. If the applicable task cannot be located, click **New Release Task** and add the necessary information.

</div>

<div class="NumList2" madcap:autonum="4. ">

When all selections/entries are made, click **Add**.

</div>
