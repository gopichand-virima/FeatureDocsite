# Flush Credentials

Use this function to disassociate a set of credentials and a device on the network which was scanned previously will be removed.

 -- ----- -------------------------------------------------------------------------------------------
 1. In the [Credentials](/admin_discovery/credentials_6_1.mdx) window, select the credential record to be flushed.
 -- ----- -------------------------------------------------------------------------------------------

 --- --------------------------------------------------------------------------------------------
 Â  At least one record must be selected. If no record is selected, an Alert message displays.
 --- --------------------------------------------------------------------------------------------

 -- ----- -------------------------------------------------------------
 2. From the *Select Actions* drop-down list, choose **Flush**.
 -- ----- -------------------------------------------------------------

 -- ----- ---------------------------
 3. Click the **Flush** icon.
 -- ----- ---------------------------

 -- ----- ---------------------------------------------------------------------------
 4. At the *Confirmation* prompt, click **Yes.** No other action is required.
 -- ----- ---------------------------------------------------------------------------

 -- ----- --------------------------------------------------------------------------------
 5. At the *Confirmation* prompt, click **Continue**. No other action is required.
 -- ----- --------------------------------------------------------------------------------
