# Edit or Delete Credentials

Use this function to edit or delete the configured values for an existing credential.

## Edit a Credential

In the main window, select **Admin > Discovery > Credentials**. The Credentials window displays.

In the main window, click the **gear** icon and select **Discovery > Credentials**. The Credentials window displays.

Select a record to edit. A new window displays with the **Details** tab selected.

Modify any of the following information:

- [Details](/admin_discovery/credentials_details_6_1.mdx)
- [Tasks](/itsm/release_mngmt/tasks_6_1.mdx)
- [Comments](/discovery/comments_6_1.mdx)
- [Attachments](/discovery/attachments_6_1.mdx)

When all selections/entries are made, click **Save**.
