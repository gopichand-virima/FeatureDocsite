# Export

To export uploaded file records,

1. Select the file(s) to export.

2. From the **Select Actions** menu, click **Export**.

1. A confirmation dialog appears, indicating that the selected records will be exported to an Excel file.

1. Click**Continue**.

2. An email is sent to the requesting userâ€™s email address (the one currently signed in).

  1.  **Download Link: **The email contains a download link to a ZIP file with the exported data.

  2.  The exported ZIP file follows this naming format:  
  ImportDataFiles-.zip  
  (For example: ImportDataFiles10.15.AM-10.08.2025.zip)

  3.  **Columns Included in the Export:**  
  The exported file contains the following columns:

  1.  **File Name:** The name of the uploaded file (e.g., Import+CI's.xlsx)

  2.  ****Status**:** The processing state of that upload (Processed, Processing)

  3.  **Uploaded By:** The display name of the user who uploaded the file

  4.  **Uploaded On:** The local timestamp when the file was uploaded (e.g., 10/08/2025, 10:15:47 AM)
