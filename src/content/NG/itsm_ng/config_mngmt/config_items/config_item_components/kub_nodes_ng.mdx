# Kubernetes Nodes

 --- -------------------------------------------------------------------------------------------------------------
   This function is only available with certain asset types. Therefore, the relevant tab may not be displayed.
 --- -------------------------------------------------------------------------------------------------------------

 -- ----- ------------------------------------------------------
 1. While viewing an asset record, click **Components**.
 -- ----- ------------------------------------------------------

 -- ----- -----------------------------
 2. Click **Kubernetes Nodes**.
 -- ----- -----------------------------

::: Ind50
![kubernetes nodes new 487x128](../../../../Resources/Images/Components/kubernetes-nodes-new_487x128.png)

![arrow border](../../../../Resources/Images/arrow-border.PNG)

 -- ----- ----------------------------------------------------------
 1. Click **Add**. The Kubernetes Nodes dialog box displays.
 -- ----- ----------------------------------------------------------

::: Ind50
![kubernetes nodes add new 374x118](../../../../Resources/Images/Components/kubernetes-nodes-add-new_374x118.png)

 -- ----- -------------------------------
 2. Click **New Kubernetes Node**
 -- ----- -------------------------------

 -- ----- -------------------------------------------------
 3. Enter the applicable information in each field.
 -- ----- -------------------------------------------------

 -- ----- --------------------------------------------
 4. When all entries are made, click **Save.**
 -- ----- --------------------------------------------

[[[Closed]{.image .placeholder .MCDropDown_Image_Icon original-image-src="../../../../../Skins/Default/Stylesheets/Images/transparent.gif" original-image-title="" height="12" width="16" Kubernetes Node](javascript:void(0)){.MCDropDownHotSpot .dropDownHotspot .MCDropDownHotSpot_ .MCHotSpotImage aria-expanded="false"}]{.MCDropDownHead .dropDownHead}

 -- ----- -------------------------------------
 1. Click **Existing Kubernetes Node**.
 -- ----- -------------------------------------

::: Ind50
![kubernetes namespaces existing new 377x257](../../../../Resources/Images/Components/kubernetes-namespaces-existing-new_377x257.png)

 -- ----- ------------------------------------------------------------
 2. Enter the search criteria to locate the existing device*.*
 -- ----- ------------------------------------------------------------

 -- ----- -----------------------------------------------------
 3. Select the applicable device from the results list.
 -- ----- -----------------------------------------------------

 -- ----- -------------------------------------------------------
 4. When all selections/entries are made, click **Save**.
 -- ----- -------------------------------------------------------

[[[Closed]{.image .placeholder .MCDropDown_Image_Icon original-image-src="../../../../../Skins/Default/Stylesheets/Images/transparent.gif" original-image-title="" height="12" width="16" Kubernetes Node](javascript:void(0)){.MCDropDownHotSpot .dropDownHotspot .MCDropDownHotSpot_ .MCHotSpotImage aria-expanded="false"}]{.MCDropDownHead .dropDownHead}

 --- --------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------
   Deleting is a permanent action and cannot be undone. Deleting may affect other functionality and information in the application such as data in configured reports, fields in windows, selectable options, etc. Therefore, be sure to understand the potential effects before making a deletion.
 --- --------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------

While viewing the list of existing records, select a record(s) to delete, and click the **Delete** button.
