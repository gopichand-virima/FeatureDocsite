# [Client]

The Discovery Application is installed in data centers to discover the host\'s communication in the Enterprise network. In the case of multiple data centers in different locations, the Discovery Application must be installed on one machine at every data center.

Every machine on which the Discovery Application is installed is referred to as a \"Client,\" and each machine is added during the Virima installation process.

![arrow border](../Resources/Images/arrow-border.PNG)

In the main window, click **Admin \> Discovery \> Client**. The Client window displays.

![client main old](../Resources/Images/Admin - Discovery/client-main-old.PNG)

### Search Options

The following search functionality is available when working in this window.

::: Bullet1
The Client list, by default, displays the most recently added client as the first row in the results.

::: Bullet1
You can search in any available column.

::: Bullet1
The columns selected are saved by the application when you exit and then return to the Client window.
