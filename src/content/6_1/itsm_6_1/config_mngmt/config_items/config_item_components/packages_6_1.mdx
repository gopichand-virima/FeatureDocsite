# Packages

 --- -------------------------------------------------------------------------------------------------------------
 Â  This function is only available with certain asset types. Therefore, the relevant tab may not be displayed.
 --- -------------------------------------------------------------------------------------------------------------

Use this function to view the packages installed on a Unix server.

 -- ----- --------------------------------------------------------------------------
 1. In the main window, select **ITSM \> Configuration Management \> CMDB**.
 -- ----- --------------------------------------------------------------------------

 -- ----- ----------------------------------------------------
 2. Search for and select the applicable asset record.
 -- ----- ----------------------------------------------------

 -- ----- ----------------------------------------------------------------------
 3. When the asset record window displays, click the **Components** tab.
 -- ----- ----------------------------------------------------------------------

 -- ----- ---------------------
 4. Click **Packages**.
 -- ----- ---------------------
