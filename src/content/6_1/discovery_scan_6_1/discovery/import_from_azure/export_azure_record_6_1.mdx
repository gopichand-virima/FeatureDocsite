# Exporting AZURE Records

Exports data for the selected imported AZURE record as an Excel spreadsheet and sends an email notification when the spreadsheet is ready for download.

- From the <span class="Strong">Select Actions</span> drop-down list, choose <span class="Strong">Export</span>.

  <span class="image placeholder" original-image-src="../../Resources/Images/Discovery/Import-From-Azure/Import%20From%20AZURE_1_12.png" original-image-title=""></span>

<!-- -->

- Click <span class="Strong">Continue.</span>

  <span class="image placeholder" original-image-src="../../Resources/Images/Discovery/Import-From-Azure/Import%20From%20AZURE_1_13.png" original-image-title=""></span>

<!-- -->

- The system downloads an Excel (.xlsx) file of the list with the details,

  - <span class="Strong">Unique ID</span> – The Virima import record ID (e.g., AZURE000578).

  - <span class="Strong">Subscription Id</span> – "Subscription Id" column displays the value you entered for either a Subscription Id or a Management Group Id when adding Azure credentials.

  - <span class="Strong">Credential</span> – The credential set used (e.g., AZURE, AZURE_ACCOUNT).

  - <span class="Strong">Imported By</span> – The user who initiated the import (e.g., Admin Virima).

    - Manual Import: The logged-in user who initiated a manual import.

    - Scheduled Import: The logged-in user who initiated a scheduled import.

  - <span class="Strong">Name</span> – The AZURE service or resource type imported (e.g., nat, load, AZURE Batch, ECS, AZURE EKS6).

  - <span class="Strong">Imported On</span> – The timestamp when that record was imported (as displayed in the UI).

  <span class="image placeholder" original-image-src="../../Resources/Images/Discovery/Import-From-Azure/Import%20From%20AZURE_1_14.png" original-image-title=""></span>
