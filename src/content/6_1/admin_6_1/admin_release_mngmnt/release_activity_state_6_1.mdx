# Release Activity State

Use this function to add a new state defining the status of the activity being worked by an assignee.

In the main window, click **Admin /> Release Management /> Release Activity State**. The Release Activity State window displays.

<span class="image placeholder" original-image-src="../Resources/Images/Admin%20-%20Release/release-activity-state-main-old.PNG" original-image-title=""></span>

In the main window, click the **gear** icon and select **Release Management /> Release Activity State**.

<span class="image placeholder" original-image-src="../Resources/Images/Admin%20-%20Release/icon-release-activity-state.PNG" original-image-title=""></span>

The Release Activity State window displays.

<span class="image placeholder" original-image-src="../Resources/Images/Admin%20-%20Release/release-activity-state-main-new.PNG" original-image-title=""></span>
