# Edit AD User Import Scans

Use this function to edit the details for an AD User Import Scans record.

 --- -----------------------------------
   The **Details** cannot be edited.
 --- -----------------------------------

 -- ----- --------------------------------------------------------------
 1. In the AD User Import Scans window, select a record to edit.
 -- ----- --------------------------------------------------------------

 -- ----- ------------------------------------------
 2. Modify any of the following information:
 -- ----- ------------------------------------------

::: Bullet2
[Tasks](AD-User-Import-Tasks.mdx){.MCTopicPopup .MCTopicPopupHotSpot .a}

::: Bullet2
[Comments](AD-User-Import-Comments.mdx){.MCTopicPopup .MCTopicPopupHotSpot .a}

::: Bullet2
[Attachments](AD-User-Import-Attachments.mdx){.MCTopicPopup .MCTopicPopupHotSpot .a}

 -- ----- --------------------------------------------------------
 3. When all selections/entries are made, click **Save**. 
 -- ----- --------------------------------------------------------
