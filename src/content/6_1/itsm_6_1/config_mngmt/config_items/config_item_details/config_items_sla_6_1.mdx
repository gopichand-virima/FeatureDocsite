# CI: SLA

Use this function to assign an SLA to the selected asset.

<div class="NumList1" madcap:conditions="Default.OldUI" madcap:autonum="1. ">

In the navigation pane, select **ITSM /> Configuration Management /> CMDB**. The Configuration Items window displays.

</div>

<div class="NumList1" madcap:conditions="Default.NewUI" madcap:autonum="1. ">

In the navigation pane, select **Configuration Management /> CMDB**. The Configuration Items window displays.

</div>

<div class="NumList2" madcap:conditions="Default.NewUI,Default.OldUI" madcap:autonum="2. ">

Select a record in the list. A new window opens and the **Details** tab displays.

</div>

<div class="NumList2" madcap:conditions="Default.NewUI,Default.OldUI" madcap:autonum="3. ">

Click the **SLA**Â tab.

</div>

<div class="Ind50">

<span class="image placeholder" original-image-src="../../../../Resources/Images/ITSM/add-sla.PNG" original-image-title="" madcap:conditions="Default.OldUI" style="width: 621px;height: 256px;"></span>

</div>

<div class="Ind50">

<span class="image placeholder" original-image-src="../../../../Resources/Images/ITSM/add-sla-new.PNG" original-image-title="" madcap:conditions="Default.NewUI" style="width: 623px;height: 249px;"></span>

</div>

<div class="NumList2" style="font-style: normal;" madcap:autonum="4. ">

In the *Response Time* field, specify the maximum amount of time that can pass before a response must occur.

</div>

<div class="NumList2" style="font-style: normal;" madcap:autonum="5. ">

In the *Resolution Time* field, specify the total time allowed to find a resolution. The *Resolution Time* cannot be smaller than the total of all the other times (such as Response Time plus 1st Level Support).

</div>

<div class="NumList2" style="font-style: normal;" madcap:autonum="6. ">

In the *Support Level* fields, specify the amount of time allows for each support level to respond.

</div>

<div class="NumList2" style="font-style: normal;" madcap:autonum="7. ">

In the **Escalate** section, do the following:

</div>

<div class="NumLista" madcap:autonum="a. ">

For the fields in the first column, specify when an action should take place, either **Before** or **After** the specified time duration has occurred.

</div>

<div class="numListb" madcap:autonum="b. ">

For the fields in the second column, select the amount of time that can elapse before a notification is sent to the users, roles, groups, and departments specified. Multiple options can be selected.

</div>

<div class="numListb" madcap:conditions="Default.OldUI" madcap:autonum="c. ">

For the actions in the third column, click **Add** (new) or **Update** (existing) to indicate who receives a notification that activity has been escalated.

</div>

<div class="numListb" madcap:conditions="Default.NewUI" madcap:autonum="d. ">

For the actions in the third column, click the **plus + sign** to indicate who receives a notification that activity has been escalated.

</div>

<div class="NumList2" madcap:autonum="8. ">

When all selections/entries are made, click **Save**.

</div>
