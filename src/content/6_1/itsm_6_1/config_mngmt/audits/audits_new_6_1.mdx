# New Audit

Use this function to add an audit process.

 -- ----- ------------------------------------------------------------------------------------------------------------
 1. In the navigation pane, select **ITSM \> Configuration Management \> Audits**. The Audits window displays.
 -- ----- ------------------------------------------------------------------------------------------------------------

 -- ----- -------------------------------------------------------------------------------------------
 2. From the *Select Actions* drop-down list, choose **New**. The New Audits window displays.
 -- ----- -------------------------------------------------------------------------------------------

::: {.Ind50
![audits old add 573x177](../../../Resources/Images/ITSM/audits-old-add_573x177.png)

 -- ----- ----------------------------------------------------------------------------------------------------
 1. In the navigation pane, select **Configuration Management \> Audits**. The Audits window displays.
 -- ----- ----------------------------------------------------------------------------------------------------

 -- ----- ---------------------------------------------------------------------------------
 2. Click **New Audit** to add a new audits process. The New Audit window displays.
 -- ----- ---------------------------------------------------------------------------------

::: {.Ind50
![audits new add 579x236](../../../Resources/Images/ITSM/audits-new-add_579x236.png)

 -- ----- ----------------------------------------------------------
 3. In the *Name* field, type a name for this audit process.
 -- ----- ----------------------------------------------------------

 -- ----- -------------------------------------------------
 4. In the *Description* field, type a description.
 -- ----- -------------------------------------------------

 -- ----- -------------------------------------------------------------------------
 5. In the *Auditor* field, click **Add**. The Auditor dialog box displays.
 -- ----- -------------------------------------------------------------------------

::: {.Ind50
![auditor search new 326x300](../../../Resources/Images/ITSM/auditor-search-new_326x300.png)
 -- ----- ------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------

 -- ----- -------------------------------------------------------------------------------------------
 b. When the applicable auditor displays, click the radio button next to the auditor\'s name.
 -- ----- -------------------------------------------------------------------------------------------

 -- ----- -----------------
 c. Click **Save**.
 -- ----- -----------------

 -- ----- ----------------
 d. Click **Add**.
 -- ----- ----------------

 -- ----- --------------------------------------------------------------------------------
 7. In the *Checked By* field, click **Add+**. The Checked By dialog box displays.
 -- ----- --------------------------------------------------------------------------------

::: {.Ind50
![checked by search new 471x434](../../../Resources/Images/ITSM/checked-by-search-new_471x434.png)*.*
 -- ----- ---------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------

 -- ----- ----------------------------
 b. Click the **Search** icon.
 -- ----- ----------------------------

 -- ----- -------------------------------------------------------------------------------------------
 c. When the applicable auditor displays, click the radio button next to the auditor\'s name.
 -- ----- -------------------------------------------------------------------------------------------

 -- ----- -----------------
 d. Click **Save**.
 -- ----- -----------------

 -- ----- ----------------
 e. Click **Add**.
 -- ----- ----------------

 -- ----- ------------------------------------------------------------------------------------------------------------------------------------------------------
 8. Select **Run a Scan** to trigger a validation type probe scan on a device. Based on the result of the scan, the audit is marked as passed or failed.
 -- ----- ------------------------------------------------------------------------------------------------------------------------------------------------------

 -- ----- ---------------------------------------------------------------------------------------
 9. From the *Audit Result* drop-down list, choose **Pass**, **Fail** or **In Progress**.
 -- ----- ---------------------------------------------------------------------------------------

 -- ------ -----------------------------------------------------------------------
 10. From the *Select Probe* drop-down list, choose the applicable option.
 -- ------ -----------------------------------------------------------------------

 -- ------ --------------------------------------------------------------------------------------------------------
 11. From the *Scan Frequency* drop-down list, choose the applicable frequency of the scan to be performed.
 -- ------ --------------------------------------------------------------------------------------------------------

 -- ------ --------------------------------------------------------------------------------------------------------------------------------------------
 12. To add configuration items for this audit, in the Configuration Items section, click **Add**. The Configuration Items dialog box displays.
 -- ------ --------------------------------------------------------------------------------------------------------------------------------------------

::: Ind50
![config items audit new 470x380](../../../Resources/Images/ITSM/config-items-audit-new_470x380.png)*.*
 -- ------ ----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------

 -- ------ -------------------------------------------------------------------------------------------------------------
 15. Under the *Search Results*, select the applicable search result(s) to move it to the *Selected Items* list.
 -- ------ -------------------------------------------------------------------------------------------------------------

 -- ------ --------------------------------------------------------
 16. Click **Save**. The added configuration item displays.
 -- ------ --------------------------------------------------------

 -- ------ -----------------------------------------
 17. Add any additional configuration items.
 -- ------ -----------------------------------------

 -- ------ ------------------------------------------------------
 18. When all selections/entries are made, click **Add**.
 -- ------ ------------------------------------------------------
