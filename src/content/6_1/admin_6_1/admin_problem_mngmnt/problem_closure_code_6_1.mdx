# Problem Closure Code

Use this function to configure a code with its description for the Closed state in the Problem Management.

In the main window, click **Admin /> Problem Management /> Problem Closure Code**. The Problem Closure Code window displays.

<span class="image placeholder" original-image-src="../Resources/Images/Admin%20-%20Problem/problem-closure-code-main-old.PNG" original-image-title=""></span>

In the main window, click the **gear** icon and select **Problem Management /> Problem Closure Code**.

<span class="image placeholder" original-image-src="../Resources/Images/Admin%20-%20Problem/icon-problem-closure-code.PNG" original-image-title=""></span>

The Problem Closure Code window displays.

<span class="image placeholder" original-image-src="../Resources/Images/Admin%20-%20Problem/problem-closure-code-main-new.PNG" original-image-title=""></span>
