# <span class="Strong">Common controls (top-right of the grid)</span>

- <span class="Strong">Move to CMDB</span> – Promote the selected records to CMDB (Refer Move to CMBD).

- <span class="Strong">Auto Refresh</span> – Refresh the grid on set periodic interval. Use this when import is still in progress so that the imported items show up without manual refresh.
