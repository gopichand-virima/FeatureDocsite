# Operational Hours

Use this function to define working or company’s operational timings, and then save the same in the application.

|     |                                                     |
|-----|-----------------------------------------------------|
|     | Operational hours cannot be deleted, only modified. |

In the main window, click **Admin /> Organization Details /> Operational Hours**. The Work Schedules window displays.
