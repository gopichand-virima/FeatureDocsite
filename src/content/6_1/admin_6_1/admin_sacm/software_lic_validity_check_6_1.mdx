# Software License Validity Check

Use this function to configure notifications sent to the assigned user before the software expiration date and the software's and assets' availability in the monitored network environment.

<div class="NumList1" madcap:conditions="Default.OldUI" madcap:autonum="1. ">

In the main window, click **Admin /> SACM /> Software License Validity Check**. The Software Notifications dialog box displays.

</div>

<div class="Ind50" madcap:conditions="Default.OldUI">

<span class="image placeholder" original-image-src="../Resources/Images/Admin%20-%20SACM/software-notification-details-old.PNG" original-image-title=""></span>

</div>

<div class="NumList1" madcap:conditions="Default.NewUI" madcap:autonum="1. ">

In the main window, click the **gear** icon and select **SACM /> Software License Validity Check**.

</div>

<div class="Ind50" madcap:conditions="Default.NewUI">

<span class="image placeholder" original-image-src="../Resources/Images/Admin%20-%20SACM/icon-sw-lic-valid-check.PNG" original-image-title=""></span>

</div>

<div class="Ind50" madcap:conditions="Default.NewUI">

The Software Notification Details dialog box displays.

</div>

<div class="Ind50" madcap:conditions="Default.NewUI">

<span class="image placeholder" original-image-src="../Resources/Images/Admin%20-%20SACM/software-notification-details-new.PNG" original-image-title=""></span>

</div>

<div class="NumList2" madcap:autonum="2. ">

In the *Number of days before expiry* field, enter the number of days to receive notification before the software expiration date. For example, if ”1” is entered, a notification is sent to the assigned user/user group one day before the software expiry date.

</div>

<div class="NumList2" madcap:autonum="3. ">

In the *Number of availability becomes* field, enter the amount of software that remains in stock. For example, if ”2” is entered, a notification is sent to the assigned user/user group when the software count indicates there are only two of the items in the stock.

</div>

<div class="NumList2" madcap:autonum="4. ">

In the *Send Notification* field, click the **plus + sign**. The Send Report To dialog box displays.

</div>

<div class="Ind50" madcap:conditions="Default.OldUI">

<span class="image placeholder" original-image-src="../Resources/Images/Admin%20-%20SACM/send-report-to-old.PNG" original-image-title=""></span>

</div>

<div class="Ind50" madcap:conditions="Default.NewUI">

<span class="image placeholder" original-image-src="../Resources/Images/Admin%20-%20SACM/send-report-to-new.PNG" original-image-title=""></span>

</div>

<div class="NumLista" madcap:autonum="a. ">

Click a tab to specify what *Users*, *Roles*, *Groups*, or *Departments* or *From Record* should receive a notification. You can add as many of each as required. For example, you can specify a User and a Department.

</div>

<div class="NumListb" madcap:autonum="b. ">

Enter the criteria in the search field.

</div>

<div class="NumListb" madcap:autonum="c. ">

In the results list, select the applicable recipient(s). This automatically moves the selection to the SELECTED list.

</div>

<div class="NumListb" madcap:autonum="d. ">

To remove a recipient from the SELECTED list, select the recipient.

</div>

<div class="NumListb" madcap:autonum="e. ">

To remove all the selections on the displayed tab, click **Clear All**.

</div>

<div class="NumListb" madcap:autonum="f. ">

When all selections are made, click **Add**.

</div>

<div class="NumList2" madcap:autonum="5. ">

When returning to the Software Notifications Details dialog box, click **Save**.

</div>
