# IT Asset Management - Compatibility Matrix

System requirements and compatibility information for ITAM module.

## Browser Compatibility

| Browser | Minimum Version | Recommended |
|---------|----------------|-------------|
| Chrome | 90+ | Latest |
| Firefox | 88+ | Latest |
| Safari | 14+ | Latest |
| Edge | 90+ | Latest |

## User Permissions

| Role | Permissions |
|------|------------|
| Asset Viewer | View asset records |
| Asset Manager | Create and edit assets |
| License Manager | Manage software licenses |
| Financial Manager | Access cost and financial data |
| ITAM Administrator | Full module access |

## Integration Capabilities

### Procurement Systems
- SAP
- Oracle Financials
- Coupa
- Custom ERP via API

### Discovery Integration
- Automatic asset population from Discovery Scan
- Real-time software installation tracking
- Hardware specification updates

### Mobile App Support
- iOS 13+ (barcode scanning)
- Android 9+ (barcode scanning)
- Offline asset tracking

## Supported Import/Export Formats

- CSV
- Excel (XLSX)
- JSON
- XML
- PDF (reports)

## Barcode Standards

Supported barcode formats:
- QR Code
- Code 128
- Code 39
- EAN-13
- UPC-A
