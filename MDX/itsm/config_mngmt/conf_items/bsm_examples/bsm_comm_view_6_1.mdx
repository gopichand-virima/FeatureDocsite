# Service Map: Communication View

In the example figure below:

- 7 CI’s of type Windows Server
- 5 CI’s of type Windows Host
- 12 CI’s of type Unix Server are communicating with Windows Host (Virima-010), which in turn communicates with 16 Unix servers, 2 Windows Hosts, 2 Windows Servers and 15 Undiscovered machines/devices.

To view the list of Windows Hosts, click on the number.


Grayed-out machines or devices can be of different networks or devices that are not discovered by the Discovery application.
