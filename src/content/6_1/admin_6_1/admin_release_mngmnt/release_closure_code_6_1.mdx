# Release Closure Code

Use this function to configure a code for a closed state.

In the main window, click **Admin /> Release Management /> Release Closure Code**. The Release Closure Code window displays.

<span class="image placeholder" original-image-src="../Resources/Images/Admin%20-%20Release/release-closure-code-main-old.PNG" original-image-title=""></span>

In the main window, click the **gear** icon and select **Release Management /> Release Closure Code**.

<span class="image placeholder" original-image-src="../Resources/Images/Admin%20-%20Release/icon-release-closure-code.PNG" original-image-title=""></span>
