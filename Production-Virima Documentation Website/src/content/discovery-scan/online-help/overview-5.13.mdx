# Discovery Scan - Online Help Overview

Welcome to the Discovery Scan module documentation for Virima version 5.13.

## Overview

The Discovery Scan module automates the detection and inventory of IT assets across your infrastructure, including on-premises, cloud, and hybrid environments.

## Key Features

- **Agentless Discovery**: Scan networks without installing agents
- **Multi-Cloud Support**: Discover assets in AWS, Azure, GCP
- **Scheduled Scans**: Automate regular discovery tasks
- **Comprehensive Detection**: Hardware, software, network devices, and virtual machines

## Getting Started

To begin discovering assets:
1. Navigate to Discovery Scan module
2. Configure scan credentials
3. Define scan scope (IP ranges, cloud accounts)
4. Run initial discovery scan
5. Review discovered assets
