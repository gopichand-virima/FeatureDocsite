# Major Software

Use this function to assign major software applications to minor software applications.

In the main window, click **Admin > Discovery > Major Software**. The Major Software window displays.

From the *Select Actions* drop-down list, choose an action: **New Major Software**, [Delete](/common_topics/delete_remove_5_13.mdx), or [Export](/discovery/import_data_files/export_5_13.mdx).

In the *Name* field, type the name for this major software.

Under *Minor Softwares*, click **Select**. The Select Minor Softwares dialog box displays.

Use the Search function to find and select the applicable minor software, or click the check box next to all applicable minor software.

Click **Done**.

When all selections/entries are made, click **Add**.
