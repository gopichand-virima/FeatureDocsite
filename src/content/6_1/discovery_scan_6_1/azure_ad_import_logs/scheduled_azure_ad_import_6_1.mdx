# Scheduled Azure AD Import

After Azure AD configuration, follow these steps:

1. Click the **Schedule Import icon** to the right of the selected line item.

2. The **Schedule Azure AD Import** dialog box displays.

![Azure AD User Import Logs 10](../../Resources/Images/Discovery/Azure-AD-Users-Import-Logs/Azure AD User Import Logs_10.png)

3. Select [Enable Schedule Import]{.Strong} to turn on the import function.

4. [Select ]{.inline-comment-marker}[Reflect AD Account Status]{.Strong}[ to match user login status with Azure AD.]{.inline-comment-marker}

5. Select [Activate User Account]{.Strong} to activate all users.

6. Select [Send Activation notification]{.Strong} to the applicable users when the import is complete.

- [Reflect AD Account Status]{.Strong} -- Enable to match user login status with Azure AD.

- [Activate User Account]{.Strong} -- Enable to make imported users active by default.

- [Send Activation Notification]{.Strong} -- Enable to email users upon account activation.

6. In the [Azure AD Import Frequency]{.Strong} field, click the drop-down list and select the applicable import frequency, such as [Every Day, Every 2 days]{.Strong}, and so forth.

7. In the schedule area, select [Timezone]{.Strong} and the appropriate schedule timing.

8. When all selections/entries are made, click [Add]{.Strong}. (If you are editing the existing information, the Add button changes Update.)

9. After import completion, view import records in the [Azure AD User Import Logs]{.Strong}.
