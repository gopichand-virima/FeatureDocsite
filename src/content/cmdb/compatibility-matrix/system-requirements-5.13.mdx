# CMDB - Compatibility Matrix

System requirements and compatibility information for the CMDB module.

## Browser Compatibility

| Browser | Minimum Version | Recommended |
|---------|----------------|-------------|
| Chrome | 90+ | Latest |
| Firefox | 88+ | Latest |
| Safari | 14+ | Latest |
| Edge | 90+ | Latest |

## User Permissions

| Permission | Description |
|-----------|-------------|
| CMDB View | Read-only access to CIs |
| CMDB Edit | Create and modify CIs |
| CMDB Manage Relationships | Create and edit CI relationships |
| CMDB Admin | Full administrative access |

## Integration Dependencies

The CMDB module integrates with:
- Discovery Scan (automatic CI population)
- ITSM (incident and change management)
- ITAM (asset lifecycle management)
- Reports (CMDB analytics and reporting)

## Data Import/Export

Supported formats:
- CSV
- Excel (XLSX)
- JSON
- XML
- RESTful API
