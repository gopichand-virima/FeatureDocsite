# Login and Password Recovery

Use this function to login and recover a lost password for the currently logged in user.

<div class="NumList1" madcap:autonum="1. ">

Open the browser.

</div>

<div class="NumList2" madcap:autonum="2. ">

Enter the applicable URL in the following format: http:/111.111.11.111:port number/em/live/pages/signinMSP/

</div>

<div class="NumList2" madcap:autonum="3. ">

Enter the configured/assigned *User Name*.

</div>

<div class="NumList2" madcap:autonum="4. ">

Enter the corresponding *Password*.

</div>

<div class="NumList2" madcap:autonum="5. ">

Click **Sign in**. The Virima MSP Vendor page displays and contains the following:

</div>

<div class="Ind50">

<span class="image placeholder" original-image-src="../Resources/Images/MSP/main-window.PNG" original-image-title=""></span>

</div>

<div class="Bullet2">

[Dashboard](Dashboard.htm) displays all configured Vendors including Name and Client ID.

</div>

<div class="Bullet2">

[Reports](Reports.htm) launches a new window containing a list of all current reports and enables creating of new MSP reports.

</div>

<div class="Bullet2">

[My Incidents](My-Incidents.htm) launches a new window containing all incidents reported by the logged in Vendor.

</div>

<div class="Bullet2">

The **Search** field accepts criteria to search the list for configured Vendors.

</div>

<div class="Bullet2">

[Admin](Admin.htm) displays a window for creating **Clients**, **Users**, **Mailbox Monitoring Configuration** and **Email Templates**.

</div>

<div class="Bullet2">

[Create Incident](Create-Incidents.htm) launches a new window for the configured Customers and enables creating a new Incident.

</div>

<div class="Bullet2">

**Login** launches the Home page for the Virima application interfaced with MSP.

</div>

<div class="Bullet2">

**Logged In** displays the Home page for the selected Client.

</div>

<div class="Bullet2">

**Logout** logs out the current user.

</div>

<div class="Bullet2">

[Show Graph](Show-Graph.htm) displays a graphical format for the Customer.

</div>

<div class="Bullet2">

**Admin** provides the following options:

</div>

<div class="Ind75">

<span class="image placeholder" original-image-src="../Resources/Images/MSP/admin-virima.PNG" original-image-title=""></span>

</div>

<div class="emdash1" madcap:autonum="- ">

**My Profile**. Details for the logged in user. Edit the available fields, as necessary. Also allows the function to Add and Remove Clients.

</div>

<div class="emdash1" madcap:autonum="- ">

**Virima Tech**. Displays identifying information such as Organization Name, Country, Address and so forth. Edit the details, as necessary, and click **Save**.

</div>

<div class="emdash1" madcap:autonum="- ">

**Home**. Displays the Home page.

</div>

<div class="emdash1" madcap:autonum="- ">

**Logout**. Logs out the current user and closes MSP.

</div>

## Password Recovery

Use this function to recover a lost password for the configured user name.

<div class="NumList1" madcap:autonum="1. ">

Open the browser and enter the applicable URL.

</div>

<div class="Ind50">

<span class="image placeholder" original-image-src="../Resources/Images/MSP/login-window.PNG" original-image-title=""></span>

</div>

<div class="NumList2" madcap:autonum="2. ">

On the Login window, click **Lost Password**. The Recover Password window displays.

</div>

<div class="Ind50">

<span class="image placeholder" original-image-src="../Resources/Images/MSP/recover-password.PNG" original-image-title=""></span>

</div>

<div class="NumList2" madcap:autonum="3. ">

Enter the *Username* (valid email ID).

</div>

<div class="NumList2" madcap:autonum="4. ">

Click **Reset**. A link to reset the password is sent to applicable email ID.

</div>
