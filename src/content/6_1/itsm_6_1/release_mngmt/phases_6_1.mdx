# Phases

Use this function to set a distinct period or stage in a process.

In the main window, select **ITSM /> Release Management /> Phases**.

<span class="image placeholder" original-image-src="../../Resources/Images/ITSM/Release%20Management/phases-main-old.PNG" original-image-title=""></span>

<span class="image placeholder" original-image-src="../../Resources/Images/ITSM/Release%20Management/phases-main-new.PNG" original-image-title=""></span>

New Phase

<div class="NumList1" madcap:conditions="Default.OldUI" madcap:autonum="1. ">

From the *Select Actions* drop-down list, choose **New Release Phase**.

</div>

<div class="Ind50" madcap:conditions="Default.OldUI">

<span class="image placeholder" original-image-src="../../Resources/Images/ITSM/Release%20Management/phases-new-old.PNG" original-image-title=""></span>

</div>

<div class="NumList1" madcap:conditions="Default.NewUI" madcap:autonum="1. ">

Click **New**.

</div>

<div class="Ind50" madcap:conditions="Default.NewUI">

<span class="image placeholder" original-image-src="../../Resources/Images/ITSM/Release%20Management/phase-new-new.PNG" original-image-title=""></span>

</div>

<div class="NumList2" madcap:autonum="2. ">

Complete the fields, referring to the information below.

</div>

<div class="Bullet2">

**Title**. Enter the name for the for the phase.

</div>

<div class="Bullet2">

**Description**. Enter the details for this phase.

</div>

<div class="Bullet2">

**Assignee/Group Assignee**. click **Add**the **plus + sign**, then search for and select the applicable user.

</div>

<div class="Bullet2">

**Status.** <span style="font-weight: normal;">Select a status for this phase. </span>

</div>

<div class="Bullet2">

**Priority**. Select a priority for this phase.

</div>

<div class="Bullet2">

**Start/End Dates**. Enter the start and end dates of this phase.

</div>

<div class="Bullet2">

**Work Notes**. Enter any notes associated with this phase.

</div>

<div class="Bullet2">

**Release Activities**. Select and add relevant release activities for this phase. If the applicable activity cannot be located, click **Add New Release Activity** and enter the necessary information.

</div>

<div class="NumList2" madcap:autonum="3. ">

When all selections/entries are made, click **Add**.

</div>
