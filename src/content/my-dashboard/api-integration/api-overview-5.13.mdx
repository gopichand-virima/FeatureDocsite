# My Dashboard - API Integration

API documentation for integrating with the My Dashboard module.

## API Endpoints

### Get Dashboard Configuration
```
GET /api/v1/dashboard/config
```

Retrieve the current user's dashboard configuration.

### Update Dashboard Layout
```
PUT /api/v1/dashboard/layout
```

Update widget positions and configurations.

### Get Widget Data
```
GET /api/v1/dashboard/widget/{widgetId}/data
```

Fetch data for a specific widget.

## Authentication

All API requests require authentication using API tokens:

```
Authorization: Bearer YOUR_API_TOKEN
```

## Response Format

All API responses follow this structure:

```json
{
  "status": "success",
  "data": {},
  "message": "Operation completed successfully"
}
```

## Rate Limits

- 1000 requests per hour per user
- 100 requests per minute per endpoint

## Example Usage

### JavaScript
```javascript
fetch('/api/v1/dashboard/config', {
  headers: {
    'Authorization': 'Bearer YOUR_API_TOKEN'
  }
})
.then(response => response.json())
.then(data => console.log(data));
```
