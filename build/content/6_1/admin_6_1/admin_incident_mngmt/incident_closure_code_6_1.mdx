# Incident Closure Code

Use this function to configure a code with its description for the Closed state in the Incident Management function.

In the main window, click **Admin > Incident Management > Incident Closure Code**. The Incident Closure Code window displays.

## New Incident Closure Code

From the *Select Actions* drop-down list, choose **New Incident Closure Code**. The New Incident Closure Code window displays.

Enter a *Name* and *Description*.

Click **Add**.
