# System Icons

This section describes the system icons used throughout the Virima application interface.

## Overview

System icons provide visual cues for common actions and status indicators within the Virima platform. Understanding these icons helps you navigate the application more efficiently.

## Common System Icons

### Navigation Icons
- **Home Icon**: Returns you to the main dashboard
- **Menu Icon**: Opens the navigation menu
- **Search Icon**: Activates the search functionality

### Action Icons
- **Edit**: Modify existing records
- **Delete**: Remove items from the system
- **Save**: Commit changes to the database
- **Cancel**: Discard changes and return to previous view

### Status Indicators
- **Success**: Green checkmark indicating successful completion
- **Warning**: Yellow triangle for cautionary alerts
- **Error**: Red X for critical issues
- **Info**: Blue information symbol for helpful tips

## Usage Guidelines

System icons are consistently placed throughout the interface to ensure a familiar user experience. Hover over any icon to display its tooltip with a brief description of its function.
