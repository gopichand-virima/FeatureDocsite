# <span style="background-color: #ffffff;">Import from AWS</span>

Use this function to identify information existing in Amazon Web Services (AWS). The discovery of the Amazon Web Services cloud is based on account information rather than an IP range. Discovery Servers are not used in this type of discovery.

<div class="NumList1" madcap:autonum="1. ">

In the navigation pane, select **Discovery Scan /> Import from AWS**. The <span class="span_5">Import AWS window displays and contains a list of all currently imported records. </span>

</div>

<div class="Ind50">

<span class="image placeholder" original-image-src="../Resources/Images/Discovery/import-aws.PNG" original-image-title="" madcap:conditions="Default.OldUI"></span>

</div>

<div class="Ind50" madcap:conditions="Default.NewUI">

<span class="image placeholder" original-image-src="../Resources/Images/Discovery/import-aws-new.PNG" original-image-title="" madcap:conditions="Default.NewUI"></span>

</div>

<div class="NumList2" madcap:conditions="Default.OldUI" madcap:autonum="2. ">

From the *Select Actions* drop-down list, choose [Import](Import-AWS-Record.htm).

</div>

<div class="NumList2" madcap:conditions="Default.NewUI" madcap:autonum="3. ">

Click the **<a href="Import-AWS-Record.htm" style="font-weight: normal;">Import</a>** icon.

</div>

Edit AWS Import Record

Use this function to edit the values of the configured parameters and update the information in the application.

<div class="NumList1" madcap:autonum="1. ">

Select the record to edit. The Edit Report window displays the configured CI details into the following categories: **All**, **Authorized**, and **Unauthorized**.

</div>

<div class="Bullet2">

*Authorized* represents all assets that **are present** inside CMDB.

</div>

<div class="Bullet2">

*Unauthorized* represents new assets **are *not* present** inside CMDB.

</div>

<div class="NumList2" style="color: #000000;" madcap:autonum="2. ">

To move a record CMDB, do the following:

</div>

<div class="NumLista" madcap:autonum="a. ">

Select a record.

</div>

<div class="NumListb" madcap:autonum="b. ">

Click **Move to CMDB**.

</div>

<div class="NumListb" madcap:autonum="c. ">

In the Alert Message window, click **Continue** to move the record. The process runs in the background and no further action is required.

</div>
