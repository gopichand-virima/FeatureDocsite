# Import Users

While importing users into Virima, the system checks if the user already exists. If not, Virima creates a new user.

For consecutive Azure AD imports, the value of a user gets updated if the value has been modified in the Azure AD.

After configuring Azure AD, follow these steps:

1. Click the **Import** **icon** located to the right of the selected entry.

![Azure AD User Import Logs 8](../../Resources/Images/Discovery/Azure-AD-Users-Import-Logs/Azure AD User Import Logs_8.png)

2. The [Azure AD Import]{.Strong} dialog box displays.

 1. In the [Azure AD Import]{.Strong} dialog box:

 1. [Reflect AD Account Status]{.Strong} -- Enable to match user login status with Azure AD.

 2. [Activate User Account]{.Strong} -- Enable to make imported users active by default.

 3. [Send Activation Notification]{.Strong} -- Enable to email users upon account activation.

![Azure AD User Import Logs 9](../../Resources/Images/Discovery/Azure-AD-Users-Import-Logs/Azure AD User Import Logs_9.png)

3. When all selections/entries are made, click [Import]{.Strong}.

4. A confirmation message displays. Click [OK]{.Strong}.
