# CI: Components

Any changes need to be made in Discovered Items Components as well.

Every configuration item has software and hardware associated with it. These components vary based on the [blueprint](../../../Admin%20-%20SACM/Blueprints.htm).

When a new CI component is added, an email notification is sent to those Roles configured as per the SACM RACI matrix and email preferences. <span style="color: #dc143c;"> </span>

To access the functions, do the following:

<div class="NumList1" madcap:conditions="Default.OldUI" madcap:autonum="1. ">

From the navigation pane, select **ITSM /> Configuration Management /> Configuration Items**. The Configuration Items window displays.

</div>

<div class="NumList1" madcap:conditions="Default.NewUI" madcap:autonum="1. ">

From the navigation pane, select **Configuration Management /> Configuration Items**. The Configuration Items window displays.

</div>

<div class="NumList2" madcap:autonum="2. ">

Select a record in the list. A new window opens and the **Details** tab displays.

</div>

<div class="NumList2" madcap:conditions="Default.NewUI" madcap:autonum="3. ">

Click the **<a href="../Components/CI-Comp-List.htm" style="font-weight: normal;">Components</a>** tab. The applicable tabs and subtabs for the selected asset display.

</div>
