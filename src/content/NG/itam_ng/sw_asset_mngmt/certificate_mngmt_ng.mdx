# Certificate Management

Use this function to configure digital certificate protocols of the software and hardware monitored in a network environment to keep the records valid along with the attributes line validity period, owner, issuer, etc.

In the navigation pane, select **ITAM > Software Asset Management > Certificate Management**. The Certification Management window displays.

New Certificate

From the *Select Actions* drop-down list, choose **New Certificate** . The New Certificate window displays.

Enter a *Certificate Name*, *Port*, and *Issuer*.

In the *State Date* and *Expiry Date* fields, click the calendar icon and select the date and time.

In the *Associate CI* field, click **Add**, then search for and select the relevant asset.

Click **Add**.
