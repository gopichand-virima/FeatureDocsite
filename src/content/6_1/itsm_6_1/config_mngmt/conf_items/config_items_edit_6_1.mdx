# Edit Configuration Item (CI)

Use this function to edit configuration item details.

 -- ----- --------------------------------------------------------------------------------------------------------------------------------------
 1. In the navigation pane, select **ITSM \> Configuration Management \> Configuration Items**. The Configuration Items window displays.
 -- ----- --------------------------------------------------------------------------------------------------------------------------------------

 -- ----- ---------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------

 -- ----- -------------------------------------------------------
 3. When all selections/entries are made, click **Save**.
