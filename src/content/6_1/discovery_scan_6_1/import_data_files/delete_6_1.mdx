# Delete

To delete uploaded file records,

1. Select one or more files from the list.

2. From the [Select Actions]{.Strong} menu, click [Delete]{.Strong}.

 ![Import Data Files 1 24](../../Resources/Images/Discovery/Import-Data-Files/Import Data Files_1_24.png)

3. If no files are selected, an alert appears: [\"Please select at least one record.\"]{.Emphasis}

 ![Import Data Files 1 25](../../Resources/Images/Discovery/Import-Data-Files/Import Data Files_1_25.png)

4. [Confirm the deletion and Click on ]{.inline-comment-marker}[Delete]{.Strong}[.]{.inline-comment-marker}

 1. [Deleting a file from Import Data Files will also remove the associated imported assets only if they have not yet been promoted to the CMDB.]{.inline-comment-marker}

5. [Click ]{.inline-comment-marker}[Cancel ]{.Strong}[to cancel the delete.]{.inline-comment-marker}
