# User Roles

Use this function to create roles to which you can assign users.

|     |                                                                                                        |
|-----|--------------------------------------------------------------------------------------------------------|
|     | [Role Access](../Admin%20-%20Other/Role-Access.htm) permissions must be set before adding a user role. |

In the main window, click **Admin /> Users /> User Roles**. The User Roles window displays.

<span class="image placeholder" original-image-src="../Resources/Images/Admin%20-%20Users/user-roles-main-old.PNG" original-image-title=""></span>

New User Role

<div class="NumList1" madcap:conditions="Default.OldUI" madcap:autonum="1. ">

From the *Select Actions* drop-down list, choose **New Role**.

</div>

<div class="Ind50" madcap:conditions="Default.OldUI">

<span class="image placeholder" original-image-src="../Resources/Images/Admin%20-%20Users/user-roles-add-old.PNG" original-image-title=""></span>

</div>

<div class="NumList2" madcap:autonum="2. ">

Enter a *Name* and *Description*.

</div>

<div class="NumList2" madcap:autonum="3. ">

Click the drop-down list and select the *Group* to which the role is associated.

</div>

<div class="NumList2" madcap:conditions="Default.NewUI" madcap:autonum="4. ">

Click the drop-down list and select a *Module*. Multiple modules can be selected.

</div>

<div class="NumList2" madcap:autonum="5. ">

When all selections/entries are made, click **Add**.

</div>

Edit User Role

To view users who are associated with the role, click the **Users** tab.

Delete User Role

To delete an existing user role:

<div class="NumList1" madcap:conditions="Default.OldUI" madcap:autonum="1. ">

Click the checkbox for the applicable role.

</div>

<div class="NumList2" madcap:conditions="Default.OldUI" madcap:autonum="2. ">

From the *Select Actions* drop-down list, choose **Delete**.

</div>

<div class="NumList1" madcap:conditions="Default.NewUI" madcap:autonum="1. ">

Click on a role in the list.

</div>

<div class="NumList2" madcap:conditions="Default.NewUI" madcap:autonum="2. ">

Click **Delete**.

</div>
