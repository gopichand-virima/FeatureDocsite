# User Groups

Use this function to configure groups among the existing application users by defining their precise roles.

The following information should be configured before creating a new user group:

<div class="Bullet1">

Manager (required)

</div>

<div class="Bullet1">

Department (optional, but recommended)

</div>

<div class="Bullet1">

Role (optional, but recommended)

</div>

|     |                                                                                                         |
|-----|---------------------------------------------------------------------------------------------------------|
|     | [Role Access](../Admin%20-%20Other/Role-Access.htm) permissions must be set before adding a user group. |

In the main window, click **Admin /> Users /> User Groups**. The User Groups window displays.

<span class="image placeholder" original-image-src="../Resources/Images/Admin%20-%20Users/user-groups-main-old.PNG" original-image-title="" style="width: 522px;height: 169px;"></span>

New User Group

<div class="NumList2" madcap:conditions="Default.OldUI" madcap:autonum="1. ">

From the *Select Actions* drop-down list, choose **New Group**. The New User Group window displays.

</div>

<div class="Ind50" madcap:conditions="Default.OldUI">

<span class="image placeholder" original-image-src="../Resources/Images/Admin%20-%20Users/user-group-add-old.PNG" original-image-title="" style="width: 613px;height: 228px;"></span>

</div>

<div class="NumList2" madcap:autonum="2. ">

Complete the *Name*, *Group Email*, and *Description* fields.

</div>

<div class="NumList2" madcap:autonum="3. ">

For the *Department*, *Manager*, and *Role* fields, click the **Add** button, then search for and make the applicable selections.

</div>

|     |                                                                                                                                                                           |
|-----|---------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
|     | In order to add a User Group, the Manager must be a member of the [Department](../Admin%20-%20Org%20Details/Departments.htm) being selected or an error message displays. |

<div class="NumList2" madcap:conditions="Default.OldUI" madcap:autonum="4. ">

To add an image, click the **Upload** button and select the applicable image.

</div>

<div class="NumList2" madcap:autonum="5. ">

When all selections/entries are made, click **Add**.

</div>

Member Information for a Group

Once you add a group, you can add one or more members to it.

<div class="NumList1" madcap:autonum="1. ">

Click on a group in the list.

</div>

<div class="NumList2" madcap:autonum="2. ">

In the Details window, click the **Members** tab. The Members window displays.

</div>

<div class="NumList2" madcap:autonum="3. ">

Click **New User** and search for and select the applicable member(s)*.*

</div>

<div class="NumList2" madcap:autonum="4. ">

When all selections are made, click **Save**.

</div>

Edit User Group Delete User Group

To delete an existing user group:

<div class="NumList1" madcap:conditions="Default.OldUI" madcap:autonum="1. ">

Click the checkbox for the applicable group.

</div>

<div class="NumList2" madcap:conditions="Default.OldUI" madcap:autonum="2. ">

From the *Select Actions* drop-down list, choose **Delete**.

</div>

<div class="NumList2" madcap:conditions="Default.OldUI" madcap:autonum="3. ">

In the confirmation window, Type **Delete** and click **Delete**.

</div>
