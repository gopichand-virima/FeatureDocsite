# Hardware Assets

Use this function to manage the physical components of computers, computer networks, and systems. This begins with the acquisition and continues through maintenance until the hardware\'s ultimate disposal.

 --- ---------------------------------------------------------------
   Before adding a new hardware asset, a correlator must be set.
 --- ---------------------------------------------------------------

In the navigation pane, select **ITAM \> Hardware Assets.** The Hardware Assets window displays the list of all configured hardware assets.
