# Hard Drives

## 

Displays a list of all hard drives installed on this asset and provides the functionality to update Windows patches.

Click the **Hard Drives** tab.

To view the hard drive component details, click a line item. The Component Details information box displays. This information is view only.

## Apply Windows Update Patches

Before you can apply a patch, you must have an agent associated with the CMDB CI.


Click **Windows Updates**.

Click **Available Patches**. The Available Patches window displays. The Add Maintenance window displays.

! !

Select a patch(es) to apply and click the **Apply** button. The Downloadable Patches dialog box displays containing a list of all downloadable patches.

! !

Click **Apply**.
