# Change attributes

The <span class="Strong">Change Attributes</span> action in the Virima CMDB enables users to update one or more field values across selected Configuration Items (CIs) in bulk.

- Select one or more blueprints.

- Click the <span class="Strong">Select Actions</span> dropdown at the top-right of the CMDB list.

- Choose <span class="Strong">Change Attributes </span>from the dropdown.

  <span class="image placeholder" original-image-src="../Resources/Images/CMDB%20-%20Left-Nav/Configuration%20Management%20Database_6.png" original-image-title=""></span>

<!-- -->

- After selecting blueprints and triggering the <span class="Strong">Change Attribute </span>action, a confirmation prompt appears.

  <span class="image placeholder" original-image-src="../Resources/Images/CMDB%20-%20Left-Nav/Configuration%20Management%20Database_7.png" original-image-title=""></span>

<!-- -->

- Click <span class="Strong">"Yes"</span> to proceed or <span class="Strong">"No"</span> to cancel. A screen then displays all <span class="Strong">editable fields</span>, grouped by category.

<!-- -->

- Editable fields are grouped by category based on the blueprint example (with a detailed list of categories and example fields).

- Properties common to all blueprints, if edited, are reflected in all CIs associated with the selected blueprints.

- Properties specific to a particular blueprint, if edited, are reflected only in CIs associated with that blueprint.

# <span class="Strong">Modify CI Properties</span>

The Change Attributes form is organized into the following sections:

The following fields may appear (depending on CI type, user role, and blueprint design) and can be edited in bulk:

<span class="image placeholder" original-image-src="../Resources/Images/CMDB%20-%20Left-Nav/Configuration%20Management%20Database_8.png" original-image-title=""></span> <span class="image placeholder" original-image-src="../Resources/Images/CMDB%20-%20Left-Nav/Configuration%20Management%20Database_9.png" original-image-title=""></span>

<span class="Strong">Application Details</span>

- <span class="Strong">Date Properties</span>: Various date fields with calendar pickers

- <span class="Strong">Dropdown Properties</span>: Configurable dropdown selections

- <span class="Strong">Custom Fields</span>: Application-specific properties

<span class="Strong">Asset Primary Information</span>

- <span class="Strong">Manufacturer</span>: Asset manufacturer details

- <span class="Strong">Asset Name</span>: Descriptive name for the asset

- <span class="Strong">Subscription ID</span>: Unique subscription identifier

- <span class="Strong">Server Port</span>: Network port information

- <span class="Strong">Condition</span>: Asset condition (Normal, etc.)

- <span class="Strong">Business Criticality</span>: Impact level (High, Medium, Low)

- <span class="Strong">Type</span>: Asset categorization

- <span class="Strong">Description</span>: Detailed asset description

- <span class="Strong">Model Number</span>: Manufacturer model identifier

- <span class="Strong">Serial Number</span>: Unique serial identifier

- <span class="Strong">Resource Group</span>: Organizational grouping

- <span class="Strong">Company</span>: Associated company/organization

- <span class="Strong">Local FQDN</span>: Fully qualified domain name

- <span class="Strong">Host Name</span>: Network host identifier

- <span class="Strong">Contracts</span>: Associated contract information

- <span class="Strong">TestToDelete</span>: Testing/validation fields

<span class="Strong">Device Details</span>

- <span class="Strong">Device Location</span>: Physical/logical location

- <span class="Strong">Hardware Type</span>: Device classification

<span class="Strong">Hardware and Network</span>

- <span class="Strong">IP Address</span>: Primary IP configuration

- <span class="Strong">Virtual IP Address</span>: Virtual network addressing

- <span class="Strong">Uptime</span>: System uptime information

- <span class="Strong">Domain</span>: Network domain membership

- <span class="Strong">Network Properties</span>: Various network-related integer and string properties

- <span class="Strong">Boolean Properties</span>: Network configuration flags

<span class="Strong">Maintenance and Backup Information</span>

- <span class="Strong">Backup Configurations</span>: Dropdown selections for backup settings

- <span class="Strong">Maintenance Schedules</span>: Maintenance-related properties

<span class="Strong">Network Adapter Details</span>

- <span class="Strong">MAC Address</span>: Physical network adapter address

- <span class="Strong">Default Gateway</span>: Network gateway configuration

<span class="Strong">Operating System Details</span>

- <span class="Strong">Operating System</span>: OS identification

- <span class="Strong">OS Build Number</span>: Specific build information

- <span class="Strong">OS Architecture</span>: System architecture (32-bit/64-bit)

- <span class="Strong">OS Kernel</span>: Kernel version information

<span class="Strong">Processor Details</span>

- <span class="Strong">Number of Processors</span>: CPU count information

<span class="Strong">Software Instance Details</span>

- <span class="Strong">Build Version</span>: Software build information

- <span class="Strong">Display Version</span>: User-visible version strings

<span class="Strong">Virtual Machine Details</span>

- <span class="Strong">UUID</span>: Unique virtual machine identifier

<span class="Strong">Other Information</span>

- <span class="Strong">Owner</span>: Asset owner assignment with <span class="Strong">"Add"</span> button

- <span class="Strong">Second Level Support</span>: Support contact with <span class="Strong">"Add"</span> button

- <span class="Strong">Third Level Support</span>: Escalation contact with <span class="Strong">"Add"</span> button

- <span class="Strong">Tags</span>: Asset categorization tags with <span class="Strong">"Add"</span> button

- <span class="Strong">Location</span>: Physical/logical location with <span class="Strong">"Add"</span> button

- <span class="Strong">Hardware Asset</span>: Associated hardware with <span class="Strong">"Add"</span> button

- <span class="Strong">OS License Key</span>: Operating system licensing with <span class="Strong">"Add"</span> button

- <span class="Strong">Vendor</span>: Manufacturer/vendor information with <span class="Strong">"Add"</span> button

<span class="Strong">Contacts</span>

- <span class="Strong">Contact Management</span>: Add/manage CI-related contacts with <span class="Strong">"Add"</span> button

After configuring property values, click <span class="Strong">"Save"</span>

Changes apply to all selected CIs. You will then return to the main CMDB view.

<span class="Strong">Verify changes</span>: Confirm updated properties reflect in CI records. Open specific CIs to confirm detailed changes
