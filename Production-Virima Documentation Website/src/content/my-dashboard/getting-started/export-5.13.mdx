# Export

Export data from Virima to various file formats for analysis, reporting, and external use.

## Overview

The Export function allows you to extract data from Virima in multiple formats, enabling offline analysis, backup, integration with other tools, and sharing with stakeholders.

## Export Formats

### Available Formats
- **Excel (XLSX)**: Spreadsheet format with formatting and formulas
- **CSV**: Comma-separated values for data portability
- **PDF**: Portable document format for sharing and printing
- **JSON**: Structured data format for API integration
- **XML**: Extensible markup language for data exchange

## Exporting Data

### Basic Export
1. Navigate to the list or report you want to export
2. Click the **Export** button or icon
3. Select your preferred format
4. Choose which columns to include
5. Click **Export** to download the file

### Advanced Export Options
- Select specific fields/columns
- Apply filters before exporting
- Choose sort order
- Set date range for export
- Include or exclude headers
- Configure formatting options

## Export Scope

### Current View
- Exports only visible records
- Respects applied filters and search
- Includes current sort order
- Limited to displayed columns

### All Records
- Export all records regardless of current view
- Option to override filters
- May require additional permissions
- Subject to export size limits

### Selected Records
- Choose specific records to export
- Use checkboxes to select items
- Useful for partial exports
- Combine with filters for precision

## Export Limits

### Record Limits
- Standard users: Up to 10,000 records per export
- Power users: Up to 50,000 records per export
- Administrators: Up to 100,000 records per export
- Contact administrator for larger exports

### File Size Limits
- Maximum file size varies by format
- Large exports may be split into multiple files
- Consider using filters to reduce export size

## Scheduled Exports

### Automated Exports
- Schedule regular exports (daily, weekly, monthly)
- Email export files automatically
- Save to network location
- Configure export parameters

### Setting Up Scheduled Exports
1. Create and save your desired view/filter
2. Click **Schedule Export**
3. Set frequency and timing
4. Configure delivery method
5. Save schedule

## Export Templates

### Custom Templates
- Create reusable export configurations
- Save column selections
- Store formatting preferences
- Share templates with team members

### Using Templates
1. Click **Export**
2. Select **Use Template**
3. Choose your saved template
4. Review settings
5. Click **Export**

## Data Formatting

### Excel Exports
- Maintain column formatting
- Include conditional formatting
- Preserve formulas (optional)
- Add pivot tables (advanced)

### CSV Exports
- Choose delimiter (comma, semicolon, tab)
- Configure text qualifier
- Set date format
- Choose encoding (UTF-8, ASCII)

### PDF Exports
- Select page orientation (portrait/landscape)
- Configure page size
- Include headers and footers
- Add page numbers

## Security Considerations

### Export Permissions
- Users can only export data they can view
- Sensitive fields may be excluded
- Audit logging of export activities
- Compliance with data policies

### Data Protection
- Exported files may contain sensitive data
- Secure file transmission and storage
- Follow organizational data handling policies
- Consider encryption for sensitive exports

## Best Practices

- Filter data before exporting to reduce file size
- Use appropriate format for intended use
- Schedule regular exports for backups
- Verify export data accuracy
- Clean up old export files regularly
- Document export procedures
- Test scheduled exports
- Review exported data before sharing

## Troubleshooting

### Common Issues
- **Export fails**: Check record count limits
- **Missing data**: Verify filters and permissions
- **Format errors**: Ensure correct format selection
- **Timeout errors**: Reduce export size or use filters
