# New Service Level Agreements (SLA)

Use this function to create a new service level agreement.

 -- ----- --------------------------------------------------------------------------------------------------------------------------
 1. In the main window, select **Admin \> Other \> Service Level Agreements**. The Service Level Agreements window displays.
 -- ----- --------------------------------------------------------------------------------------------------------------------------

::: Ind50
![sla window 626x114](../Resources/Images/Admin - Other/sla-window_626x114.png)

 -- ----- ---------------------------------------------------------------------------------------
 2. In the *Select Actions* field drop-down list, choose **New Service Level Agreement**.
 -- ----- ---------------------------------------------------------------------------------------

 -- ----- -----------------------------------------------
 3. Complete the tabs according to the following:
 -- ----- -----------------------------------------------

::: Bullet2
[Details](SLA-Details.mdx){.MCTopicPopup .MCTopicPopupHotSpot .a}

::: Bullet2
[Tasks](SLA-Tasks.mdx){.MCTopicPopup .MCTopicPopupHotSpot .a}

::: Bullet2
[Comments](SLA-Comments.mdx){.MCTopicPopup .MCTopicPopupHotSpot .a}

::: Bullet2
[Attachments](SLA-Attachments.mdx){.MCTopicPopup .MCTopicPopupHotSpot .a}

 -- ----- ------------------------------------------------------
 4. When all selections/entries are made, click **Add**.
 -- ----- ------------------------------------------------------
