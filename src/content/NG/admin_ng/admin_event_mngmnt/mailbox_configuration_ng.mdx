# Mailbox Configuration

Use this function to configure functional account set-up for valid email addresses to exchange information and further to be used for receiving alerts.

 -- ----- --------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------
 1. In the main window, click **Admin \> Event Management \>Â Mailbox Configuration**. The Mailbox Configuration window displays. ![inbox ITSM Configuration 6.1](../Resources/Images/Admin - Other/inbox-ITSM-Configuration-6.1.PNG)
 -- ----- --------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------

 -- ----- -----------------------------------------------
 2. Complete the fields, referring the following:
 -- ----- -----------------------------------------------

::: Bullet2
**Authentication**: Select an authentication type from the drop-down list.

::: Bullet2
**Email ID**: Enter the email ID of the receiver.

- Use Proxy : [When selected, enables use of a proxy server. Enter the Host, Username, Password, and Port for the proxy.]{style="font-weight: normal;"}
 -- ----- -----------------------------------------------------
 3. When all selections/entries are made, click **Add**
