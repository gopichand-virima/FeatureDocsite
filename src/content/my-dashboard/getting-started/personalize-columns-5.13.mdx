# Personalize Columns

Customize which columns display in list views and adjust their order and width to optimize your workflow.

## Overview

The Personalize Columns function allows you to customize list views by selecting which columns to display, arranging their order, and adjusting their width. This helps you focus on the most relevant information and create efficient, personalized views.

## Accessing Column Personalization

### Open Column Settings
1. Navigate to any list view
2. Click the **Personalize Columns** icon or button (usually a column/grid icon)
3. View available columns panel
4. Make your customization choices
5. Click **Apply** or **Save** to update the view

### Quick Access
Alternative methods to personalize columns:
- Right-click on column headers
- Use column header dropdown menus
- Drag column headers to reorder
- Double-click column dividers to auto-size

## Column Selection

### Show/Hide Columns
Control which columns display:
1. Open column personalization panel
2. Check boxes to show columns
3. Uncheck boxes to hide columns
4. See preview of changes
5. Apply selections

### Available Columns
Columns typically include:
- **Default Columns**: Standard fields for the record type
- **Custom Columns**: Organization-specific fields
- **System Columns**: Metadata (created date, modified by, etc.)
- **Calculated Columns**: Computed values
- **Related Columns**: Fields from related records

### Required Columns
Some columns cannot be hidden:
- Primary identifier (ID, name)
- Critical status fields
- Mandatory regulatory fields
- Key reference fields

## Column Ordering

### Rearrange Columns
Change the order columns appear:
- **Drag and Drop**: Click and drag column headers
- **Column List**: Use up/down arrows in personalization panel
- **Keyboard**: Use Alt + Arrow keys to move columns
- **Quick Reset**: Return to default order with one click

### Common Arrangements
Recommended column ordering:
- Most important fields on the left
- Related fields grouped together
- Action columns on the right
- Timestamp columns together
- Status fields near beginning

## Column Width

### Resize Columns
Adjust column width to fit content:
- **Manual Resize**: Drag column divider
- **Auto-Fit**: Double-click column divider
- **Set Width**: Enter specific pixel width
- **Percentage Width**: Relative sizing
- **Min/Max Width**: Set boundaries

### Width Presets
- **Narrow**: For status indicators, icons
- **Medium**: For IDs, short text
- **Wide**: For descriptions, comments
- **Auto**: Adjusts to content

## Saving Column Preferences

### Personal Views
Save your column customizations:
1. Customize columns as desired
2. Click **Save View** or **Save Preferences**
3. Provide a name for your view
4. Choose scope (personal or shared)
5. Set as default (optional)

### Multiple Views
Create different column configurations:
- **Daily Tasks View**: Columns for routine work
- **Detailed Analysis View**: All available columns
- **Quick Review View**: Minimal essential columns
- **Export View**: Optimized for data export
- **Management View**: Summary and status columns

## Shared Views

### Share with Team
Share your column configuration:
1. Create your personalized view
2. Click **Share View**
3. Select users or groups to share with
4. Set permissions (view only or can edit)
5. Save and distribute

### Team Templates
Administrators can create:
- Department-specific column sets
- Role-based default views
- Industry standard layouts
- Compliance-required columns

## Column Grouping

### Group by Column
Organize records by column values:
1. Drag column header to grouping area
2. Records automatically group by that field
3. Expand/collapse groups as needed
4. Add multiple grouping levels
5. Sort within groups

### Common Groupings
- Group by status
- Group by priority
- Group by assigned user
- Group by category
- Group by date ranges

## Column Sorting

### Sort Columns
Order records by column values:
- **Single Sort**: Click column header
- **Multi-Sort**: Shift + click additional columns
- **Ascending/Descending**: Toggle sort direction
- **Clear Sort**: Reset to default order
- **Custom Sort**: Define sort rules

### Sort Indicators
Visual cues for sorting:
- Up arrow: Ascending sort
- Down arrow: Descending sort
- Number badge: Multi-sort priority

## Column Filtering

### Filter Individual Columns
Apply filters directly from columns:
1. Click filter icon in column header
2. Select or enter filter criteria
3. Apply filter
4. Multiple column filters combine

### Quick Filters
- Search box in column header
- Select from unique values
- Date range pickers
- Numeric range sliders

## Advanced Features

### Conditional Formatting
Apply visual formatting based on values:
- Color-code by priority
- Highlight overdue items
- Bold critical status
- Icons for status indicators

### Column Formulas
Calculate values in custom columns:
- Sum, average, count
- Date calculations
- String concatenation
- Conditional expressions

### Frozen Columns
Pin columns to remain visible:
- Freeze left columns while scrolling
- Keep identifier columns visible
- Pin up to 3 columns
- Unfreeze as needed

## Reset and Restore

### Reset to Default
Return to standard column configuration:
1. Click **Reset to Default** or **Restore**
2. Confirm reset action
3. All customizations cleared
4. View returns to system default

### Restore Previous View
Undo recent column changes:
- System keeps recent view history
- Restore previous configuration
- Compare different views
- Recover accidentally changed settings

## Export with Column Settings

### Export Current View
When exporting data:
- Export uses current column selection
- Column order preserved in export
- Hidden columns excluded
- Width settings applied (PDF exports)

## Mobile Column Display

### Responsive Columns
On mobile devices:
- Automatically reduces visible columns
- Shows most important fields first
- Swipe to see additional columns
- Optimized for small screens

## Keyboard Shortcuts

### Quick Column Actions
- **Ctrl + K**: Open personalize columns
- **Alt + Left/Right**: Move column
- **Ctrl + R**: Reset columns
- **Ctrl + H**: Hide selected column
- **Ctrl + Shift + A**: Auto-fit all columns

## Best Practices

### Column Selection
- Show only necessary columns
- Keep critical fields visible
- Group related columns together
- Use consistent naming
- Remove redundant columns

### Performance
- Limit number of visible columns
- Hide computed columns if not needed
- Use column filters instead of showing all data
- Consider page size with many columns

### Organization
- Place action columns consistently
- Keep ID/name columns leftmost
- Group dates together
- Order by workflow sequence
- Use logical left-to-right flow

## Troubleshooting

### Common Issues
- **Changes not saving**: Check browser cookies
- **Columns disappeared**: Check column selection settings
- **Can't resize column**: May be fixed-width column
- **View reset unexpectedly**: Cache cleared or session expired
