# New Azure AD Configuration

 -- ----- ------------------------------------------------------------------------------------------------------------
 1. While viewing the ADÂ Configuration window, click **New AD Configuration**. The Export dialog box displays.
 -- ----- ------------------------------------------------------------------------------------------------------------

::: Ind50
![azure ad config export dialog box 472x278](../Resources/Images/Admin - Users/azure-ad-config-export-dialog-box_472x278.png)

 -- ----- --------------------------------------------------------------
 2. Enter the *Name*, *Client Id*, *Tenant Id* and *Secret Key*.
 -- ----- --------------------------------------------------------------

 -- ----- -------------------------------------------------------
 3. Click **Next**. The Map Property dialog box displays.
 -- ----- -------------------------------------------------------

::: Ind50
![azure ad config export map prop 472x448](../Resources/Images/Admin - Users/azure-ad-config-export-map-prop_472x448.png)

 -- ----- -------------------------------------------------------------------------------------
 4. Select required values to import (for example, First Name, Last Name and so forth).
 -- ----- -------------------------------------------------------------------------------------

 -- ----- -------------------------------------------------------------------
 5. Click the drop-down list arrow and select the property to map to.
 -- ----- -------------------------------------------------------------------

::: Ind50
![first name example](../Resources/Images/Admin - Users/first-name-example.PNG)

 -- ----- ---------------------------------------------------------
 6. When all selections/entires are made, click **Finish**.
 -- ----- ---------------------------------------------------------
