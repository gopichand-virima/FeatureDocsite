# IPAM Features

- **Sync**: Import subnet and IP address data from Infoblox into Virima.
- **Status Update**: Ping IP addresses to update their usage status (Used/Unused).
- **Regular Scan**: Perform full discovery scans on selected networks.
- **Schedule**: Configure automated sync and scan operations.
- **Export**: Generate reports of IP address allocation and usage.
