# New Ivanti Mappings

Use this function to enter the details for mapping properties between Virima and Ivanti.

- In the main window, click the **gear** icon and select **Integrations > Ivanti Mappings**.

!

The Ivanti Mappings window displays.

!

- To create a new mapping, click **New Mapping**. The New Ivanti Mapping window displays.

!

- Click the drop-down lists and select the **Ivanti CI Type**, **CI SubType** and **CI Category**.
- Under Virima Blueprint, do the following:
- Select the **Property**, **Operator**, and **Value**.
- Select the type of condition (and, or).
- Select the **Property**, **Operator**, and **Value**.
- To delete a condition line, click the trashcan icon.
- Under Ivanti Properties, do the following:
- Click the Add button. A new line displays.
- In the *Virima Property* field, click the drop-down list and select an existing property.
- In the *Ivanti Property* field, click the drop-down list and select an existing property.
- In the *Value* field, enter the value to associate with the property.
- To use a correlator, check Correlator.
- To delete an item, select the property line and click the trashcan icon.
