# Scheduled AD Import

After AD configuration, follow these steps:

1. Click the **Schedule Import** icon to the right of the selected line item.

![AD User Import Logs 1 8](../../Resources/Images/Discovery/AD-User-Import-Logs/AD User Import Logs_1_8.png)

2. The **Schedule AD Import** dialog box displays.

![AD User Import Logs 1 9](../../Resources/Images/Discovery/AD-User-Import-Logs/AD User Import Logs_1_9.png)

3. Select [Enable Schedule Import]{.Strong} to turn on the import function.

4. Select [Activate User Account]{.Strong} to activate all users.

5. Select [Send Activation notification]{.Strong} to the applicable users when the import is complete.

- [Reflect AD Account Status]{.Strong} -- Enable to match user login status with Azure AD.

- [Activate User Account]{.Strong} -- Enable to make imported users active by default.

- [Send Activation Notification]{.Strong} -- Enable to email users upon account activation.

6. In the [AD Import Frequency]{.Strong} field, click the drop-down list and select the applicable import frequency, such as [Every Day, Every 2 days]{.Strong}, and so forth.

7. In the [Type of Import]{.Strong} field, click the drop-down list and select the type of import, such as [All Users, New & Modified Users and New Users]{.Strong}

8. In the schedule area, select [Timezone]{.Strong} and the appropriate schedule timing.

9. When all selections/entries are made, click [Add]{.Strong}. (If you are editing the existing information, the Add button changes Update.)

10. After import completion, view import records in the [AD User Import Logs]{.Strong}.
