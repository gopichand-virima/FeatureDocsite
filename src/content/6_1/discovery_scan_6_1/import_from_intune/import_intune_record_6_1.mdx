# Importing Intune Record

Use this function to import the required Intune resources into the Discovery application.

- Open the Import From Intune page, From the [Select Actions]{.Strong} drop-down list, choose [Import]{.Strong}.

 ![Import INTUNE Window 3](../../Resources/Images/Discovery/Import-From-Intune/Import INTUNE Window_3.png)

- The **Import From Intune** dialog box displays.

 ![Import INTUNE Window 4](../../Resources/Images/Discovery/Import-From-Intune/Import INTUNE Window_4.png)

- In the Name field, enter a name for the import action.

- In the Credential field, click the drop-down list and select the appropriate credential to access the Intune service.

 - If no credentials are listed in the dropdown, add credentials by navigating to [Admin \> Discovery \> Credentials]{.Strong}.

 - Only Active Intune credentials will appear in the dropdown. If a credential is Inactive, it will not be listed in the credential\'s dropdown.

- Click [Add]{.Strong} to import the Intune services to virima.

 ![Import INTUNE Window 5](../../Resources/Images/Discovery/Import-From-Intune/Import INTUNE Window_5.png)

The system displays an error if **Add** is clicked without selecting credentials.

![Import INTUNE Window 6](../../Resources/Images/Discovery/Import-From-Intune/Import INTUNE Window_6.png)

- [After clicking **Add**, the system redirects to the **Import From Intune** page and displays a success message.]{.inline-comment-marker}

 ![Import INTUNE Window 7](../../Resources/Images/Discovery/Import-From-Intune/Import INTUNE Window_7.png)

- To view the details of a CI, click on the CI. This redirects you to the discovered item page, where you can view the imported CI details.
