# Holidays

Use this function to specify holidays, which are used to auto-calculate resolution dates as they relate to incidents, changes, and problems.

<div class="NumList1" madcap:conditions="Default.OldUI" madcap:autonum="1. ">

In the main window, click **Admin /> Organizational Details /> Holiday**. The Holiday window displays.

</div>

<div class="Ind50" madcap:conditions="Default.OldUI">

<span class="image placeholder" original-image-src="../Resources/Images/Admin%20-%20Org%20Details/holiday-old.PNG" original-image-title="" style="width: 533px;height: 58px;"></span>

</div>

<div class="NumList2" madcap:conditions="Default.OldUI" madcap:autonum="2. ">

To scroll forward and backward between years, click the left and right arrows next to the date.

</div>

<div class="NumList2" madcap:conditions="Default.OldUI" madcap:autonum="3. ">

From the *Select Actions* drop-down list, choose **New Holiday**. The Add Holiday dialog box displays.

</div>

<div class="Ind50" madcap:conditions="Default.OldUI">

<span class="image placeholder" original-image-src="../Resources/Images/Admin%20-%20Org%20Details/holiday-add-old.PNG" original-image-title=""></span>

</div>

<div class="NumList1" madcap:conditions="Default.NewUI" madcap:autonum="1. ">

In the main window, click the **gear** icon and select **Organizational Details /> Holidays**.

</div>

<div class="Ind50" madcap:conditions="Default.NewUI">

<span class="image placeholder" original-image-src="../Resources/Images/Admin%20-%20Org%20Details/icon-holidays.PNG" original-image-title=""></span>

</div>

<div class="Ind50" madcap:conditions="Default.NewUI">

The Holidays window displays and contains a list of all defined holidays.

</div>

<div class="Ind50" madcap:conditions="Default.NewUI">

<span class="image placeholder" original-image-src="../Resources/Images/Admin%20-%20Org%20Details/holiday-main-new.PNG" original-image-title=""></span>

</div>

<div class="NumList2" madcap:conditions="Default.NewUI" madcap:autonum="2. ">

To scroll forward and backward between years, click the left and right arrows next to the date.

</div>

Add Holiday

<div class="NumList1" madcap:conditions="Default.NewUI" madcap:autonum="1. ">

Click **New Holiday**. The Add Holiday dialog box displays.

</div>

<div class="Ind50" madcap:conditions="Default.NewUI">

<span class="image placeholder" original-image-src="../Resources/Images/Admin%20-%20Org%20Details/add-holiday-new.PNG" original-image-title="" style="width: 347px;height: 211px;"></span>

</div>

<div class="NumList2" madcap:autonum="2. ">

In the *Date* field, use the calendar icon to select a holiday.

</div>

<div class="NumList2" madcap:autonum="3. ">

In the *Description* field, type a description.

</div>

<div class="NumList2" madcap:autonum="4. ">

If the holiday occurs on the same date each year, check **Is Recurring**.

</div>

<div class="NumList2" madcap:autonum="5. ">

Click **Add**.

</div>

Edit Holiday

To edit the holiday details:

<div class="NumList1" madcap:autonum="1. ">

Select a holiday in the list.

</div>

<div class="NumList2" madcap:autonum="2. ">

In the Update Holiday dialog box, make the necessary changes.

</div>

<div class="NumList2" madcap:autonum="3. ">

Click **Save**.

</div>

Delete Holiday  

To delete an existing holiday:

<div class="NumList1" madcap:autonum="1. ">

Select a holiday in the list.

</div>

<div class="NumList2" madcap:autonum="2. ">

From the *Select Actions* drop-down list, choose **Delete**. If a confirmation message is displayed, type **Delete** and then click **Delete**.

</div>
