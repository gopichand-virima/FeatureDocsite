# Accessing Discovered Items

1.  In the navigation pane, select <span class="Strong">Discovery Scan</span> /> <span class="Strong">Discovered Items</span>.

    <span class="image placeholder" original-image-src="../../Resources/Images/Discovery/Discovered-Items/Discovered%20Items_1.png" original-image-title=""></span>

<!-- -->

2.  The Discovered Items window will display.

    <span class="image placeholder" original-image-src="../../Resources/Images/Discovery/Discovered-Items/Discovered%20Items_1_1.png" original-image-title=""></span>

- <span class="Strong">Blueprint</span>

  - Type or category of the discovered item.

- <span class="Strong">CI (Configuration Item)</span>

  - Displays the existing CI ID if the asset is already in the CMDB; otherwise, shows "New".

- <span class="Strong">Host Name</span>

  - Name of the system or device detected during the scan.

- <span class="Strong">IP Address</span>

  - Network IP address assigned to the device or host.

- <span class="Strong">Operating System</span>

  - Operating system running on the host device.

- <span class="Strong">Location</span>

  - Geographical or logical location of the asset.

- <span class="Strong">Created On</span>

  - Timestamp when the item added to the Discovered Items list.
