# CI: Usage Report

Use this function to view software usage metric information for software configured as *major* software.

<div class="NumList1" madcap:conditions="Default.OldUI" madcap:autonum="1. ">

In the navigation pane, select **ITSM /> Configuration Management /> CMDB**. The Configuration Items window displays.

</div>

<div class="NumList2" madcap:conditions="Default.NewUI" madcap:autonum="2. ">

In the navigation page, select **Configuration Management /> CMDB**. The Configuration Items window displays.

</div>

<div class="NumList2" madcap:conditions="Default.OldUI" madcap:autonum="3. ">

Select a software asset record. The asset window displays.

</div>

<div class="NumList2" madcap:conditions="Default.OldUI" madcap:autonum="4. ">

From the drop-down list, select **Usage Report**.

</div>

<div class="NumList2" madcap:conditions="Default.OldUI" madcap:autonum="5. ">

In the Software Report window, select the time period for which you want to see the usage (such as last 24 hours or a custom range). The window updates and displays all records that match the date range criteria.

</div>

<div class="NumList2" madcap:conditions="Default.NewUI" madcap:autonum="6. ">

From the drop-down list, select the time period for which you want to see usage (such as last 24 hours or a custom range). The window updates and displays all records that match the date range criteria.

</div>

<div class="Ind50" madcap:conditions="Default.NewUI" madcap:autonum="">

<span class="image placeholder" original-image-src="../../../../Resources/Images/Admin%20-%20SACM/sw-usage-report-new.PNG" original-image-title=""></span>

</div>

<div class="NumList2" madcap:autonum="7. ">

Select the record for which you want to view usage details.

</div>

<div class="NumList2" madcap:autonum="8. ">

To export the details, click **Export**. A confirmation email is sent when the report is ready.

</div>

<div class="NumList2" madcap:autonum="9. ">

To view the details for a report, click the line item containing the report. The window updates and displays all the details for the selected report.

</div>

</div>

<div class="NumList2" madcap:autonum="10. ">

To view the configuration information for the report, click the CI Id hyperlink.

</div>
