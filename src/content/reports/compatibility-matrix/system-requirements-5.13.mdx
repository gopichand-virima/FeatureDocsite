# Reports - Compatibility Matrix

System requirements and compatibility for Reports module.

## Browser Compatibility

| Browser | Minimum Version | Recommended |
|---------|----------------|-------------|
| Chrome | 90+ | Latest |
| Firefox | 88+ | Latest |
| Safari | 14+ | Latest |
| Edge | 90+ | Latest |

## User Permissions

| Role | Permissions |
|------|------------|
| Report Viewer | View shared reports |
| Report Creator | Create and run custom reports |
| Dashboard Manager | Create and manage dashboards |
| Report Administrator | Full reporting configuration |

## Data Sources

The Reports module can access data from:
- All Virima modules (CMDB, ITSM, Discovery, etc.)
- External databases (via JDBC)
- REST APIs
- CSV/Excel imports
- Cloud data sources

## Export Formats

| Format | Use Case |
|--------|----------|
| PDF | Executive reports, printing |
| Excel (XLSX) | Data analysis, pivot tables |
| CSV | Data import to other systems |
| PowerPoint (PPTX) | Presentations |
| JSON | API integration, data exchange |

## Visualization Libraries

Built on industry-standard libraries:
- Chart.js (charts and graphs)
- D3.js (advanced visualizations)
- Leaflet (geographic maps)
- Recharts (React-based charts)

## Performance Specifications

### Report Generation Times
| Dataset Size | Typical Generation Time |
|-------------|------------------------|
| < 1,000 records | < 5 seconds |
| 1,000 - 10,000 | 5-15 seconds |
| 10,000 - 100,000 | 15-60 seconds |
| > 100,000 | 1-5 minutes |

### Dashboard Refresh
- Real-time widgets: 5-30 second intervals
- Standard widgets: 1-15 minute intervals
- Custom refresh intervals supported

## Integration Capabilities

### Business Intelligence Tools
- Power BI (connector available)
- Tableau (data export)
- Qlik Sense (API integration)
- Google Data Studio (connector)

### Email Integration
- SMTP support
- Office 365
- Gmail
- SendGrid
- Custom mail servers

## Mobile Support

- Responsive dashboards
- Mobile-optimized reports
- Touch-friendly interactions
- Offline report viewing (cached)
