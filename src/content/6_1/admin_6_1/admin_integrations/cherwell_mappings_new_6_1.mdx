# Cherwell New Mapping

Use this function to enter the details for mapping properties between Virima and Cherwell.

 -- ----- -----------------------------------------------------------------------------------------------
 1. In the main window, click the **gear** icon and select **Integrations \> Cherwell Mappings**.
 -- ----- -----------------------------------------------------------------------------------------------

::: Ind50
![icon cherwell](../Resources/Images/Admin - Integration/icon-cherwell.PNG)

::: Ind50
The Cherwell Mappings window displays.

::: Ind50
![cherwell mappings 623x116](../Resources/Images/Admin - Integration/cherwell-mappings_623x116.png)

 -- ----- -------------------------------------------------------------------------------------------
 2. To create a new mapping, click **New Mapping**. The New Cherwell Mapping window displays.
 -- ----- -------------------------------------------------------------------------------------------

::: Ind50
![new cherwell mapping new 624x193](../Resources/Images/Admin - Integration/new-cherwell-mapping-new_624x193.png)

 -- ----- ------------------------------------------------------------------------------------------------
 3. In the *Virima Blueprint* field, click the drop-down list and select the applicable blueprint.
 -- ----- ------------------------------------------------------------------------------------------------

 -- ----- --------------------------------------------------------------------------------------------
 4. In the *Cherwall Object* field, click the drop-down list and select the applicable object.
 -- ----- --------------------------------------------------------------------------------------------

 -- ----- ----------------------------------------------
 5. When the selections are made, click **Add**.
 -- ----- ----------------------------------------------

 -- ----- ------------------------------------------------------------------------------------------------------------------
 6. In the *Virima Property* field, click the drop-down list and select the property to map to the Cherwell Propery.
 -- ----- ------------------------------------------------------------------------------------------------------------------

 -- ----- -------------------------------------------------------------------------------------------------------------------
 7. In the *Cherwell Property* field, click the drop-down list and select the property to map to the Virima Property.
 -- ----- -------------------------------------------------------------------------------------------------------------------

 -- ----- ---------------------------------------------------------------------
 8. In the Value field, type the value that applies to both properties.
 -- ----- ---------------------------------------------------------------------

 -- ----- -------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------
 9. [[To use a correlator, select ]{style="color: #000000;"}]{style="color: #dc143c;"}**Correlator**[[.]{style="color: #000000;"}]{style="color: #dc143c;"}[ ]{style="color: #000000;"}
 -- ----- -------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------

 -- ------ -----------------------------------------------------------------------------------------------
 10. To remove a mapping, select the item in the list of mappings and click the **trashcan icon**.
 -- ------ -----------------------------------------------------------------------------------------------
