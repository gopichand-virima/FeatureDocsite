# New User Entitlement

This feature enables to configure license entitlement details in the application.

 -- ----- --------------------------------------------------------------------------------------------------------------------------
 1. Choose New Asset License Entitlements from the select actions dropdown. The New User License Entitlements page displays.
 -- ----- --------------------------------------------------------------------------------------------------------------------------

::: Ind50
![user entitle new 789x104](../Resources/Images/ITAM/user-entitle-new_789x104.png)

 -- ----- ---------------------------------------------------------------------------------------------
 2. Enter a text/name to be displayed for the license entitled machine in the Display Name box.
 -- ----- ---------------------------------------------------------------------------------------------

 -- ----- ---------------------------------------------------------------------------------------
 3. Click Add to select the CI to assign the license. Configuration Item window displays.
 -- ----- ---------------------------------------------------------------------------------------

::: Ind50
![user entitle new config 600x332](../Resources/Images/ITAM/user-entitle-new-config_600x332.png)

 -- ----- --------------------------------------------------------
 a. Select the required blueprint from the given dropdown.
 -- ----- --------------------------------------------------------

 -- ----- --------------------------------------------------------
 b. Enter the search criteria, and then click search icon.
 -- ----- --------------------------------------------------------

 -- ----- ---------------------------------------------------------
 c. The results are displayed based on the search criteria.
 -- ----- ---------------------------------------------------------

 -- ----- ----------------------------------------------
 d. Select the required CI, and then click Save.
 -- ----- ----------------------------------------------

 -- ----- ----------------------------------------------------------------------------------------------------
 e. Click Cancel to discard the action, and in continuation, no CI will be updated in the application.
 -- ----- ----------------------------------------------------------------------------------------------------
