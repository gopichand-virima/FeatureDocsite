# Port Configuration

Use this function toÂ create a relationship between a process and host if, during the Discovery process, it is found running on a specific port.

On the main window, select **Admin > Discovery > Port Configuration for Process**. The Port Configuration window displays.

New Port Configuration Process

In the *Select Actions* drop-down list, choose **New Port Configuration for Process**.

Enter the *Process Name* and *Port*.

In the *Relationship Type* field, click the drop-down list and select the applicable relation type.

Click **Add**.

If the user clicks Add without filling any required field (Process Name, Port, or Relationship Type) in the Port Configuration for Process window, the system displays an error.
