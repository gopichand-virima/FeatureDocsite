# Icons

The following icons are used in the Virima application.

 ---------------------------------------------------------------------------------------------------------------------------------------------------------------- -----------------------------------------------------------------------------------------------------------
 ![icon inbox 36x30](Resources/Images/Common Topics/icon-inbox_36x30.png) Opens the user\'s Inbox, which contains notifications the application sends when certain events occur.
 ![icon chat 37x27](Resources/Images/Common Topics/icon-chat_37x27.png) Opens the chat window.
 ![icon cart 38x33](Resources/Images/Common Topics/icon-cart_38x33.png) Opens the self-service portal through which end users can acquire assets, such as laptops, monitors, etc.
 ![icon suggestions 32x38](Resources/Images/Common Topics/icon-suggestions_32x38.png) Opens the Suggestions feedback form.
 ---------------------------------------------------------------------------------------------------------------------------------------------------------------- -----------------------------------------------------------------------------------------------------------

 ----------------------------------------------------------------------------------------------------------------------------- -------------------------------------------------------
 ![icon v6 nav pane](Resources/Images/Other Topics/icon-v6-nav-pane.PNG) Expands or collapses the left navigation pane.
 ![icon v6 admin](Resources/Images/Other Topics/icon-v6-admin.PNG) Displays the Virima Admin user account information.
 ![icon v6 gear admin](Resources/Images/Other Topics/icon-v6-gear-admin.PNG) Displays the Administration modules.
 ![icon v6 logout](Resources/Images/Other Topics/icon-v6-logout.PNG) Closes the application and logs off the current user.
 ----------------------------------------------------------------------------------------------------------------------------- -------------------------------------------------------

 ------------------------------------------------------------------------------------------------------------------------------------------------------------------------- -----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------
 ![icon outage calendar 32x33](Resources/Images/Common Topics/icon-outage-calendar_32x33.png) Changes or reschedules an outage period.
 ![icon personalize list columns](Resources/Images/Common Topics/icon-personalize-list-columns.png) Personalizes the columns displayed in the record window.
 ![icon print](Resources/Images/Common Topics/icon-print.png) Prints the asset details.
 ![icon proc miss components 31x31](Resources/Images/Common Topics/icon-proc-miss-components_31x31.png) The Unlink icon represents the action of disassociating or removing a relationship between two linked records in the application without deleting either of them.
 ![icon saved filters](Resources/Images/Common Topics/icon-saved-filters.png) Updates the results shown in the columns. The filter can be saved and used in future searches.
 ![icon v5 help](Resources/Images/Other Topics/icon-v5-help.PNG) Launches the online help documentation as it relates to the current function.
 ![icon export new](Resources/Images/Common Topics/icon-export-new.png) Exports data for the selected record as an Excel spreadsheet and sends an email notification when the spreadsheet is ready for download.
 ![icon resource chart](Resources/Images/Common Topics/icon-resource-chart.png) Displays a listing of all Resources and the status of assigned tasks, such as Active, Started, and Not Started. *Refer to* [Users](/admin_users/users_6_1.mdx)*.*
 ![icon delete new](Resources/Images/Common Topics/icon-delete-new.png) Deletes the currently selected record or removes the displayed item.
 ![icon v6 update scan](Resources/Images/Other Topics/icon-v6-update-scan.png) Launches the Update Scheduled Scan dialog box in which details for the currently selected scan record can be modified. *Refer to* [Scan Frequency](Discovery/Scans-Frequency.mdx)*.*
 ![icon resync data](Resources/Images/Admin - Integration/icon-resync-data.png) Resyncs the displayed data.
 ![icon reload default mapping](Resources/Images/Admin - Integration/icon-reload-default-mapping.png) Reloads the default mappings.
 ![icon primary details](Resources/Images/Common Topics/icon-primary-details.png) Displays information about the selected item, such as the associated Blueprint and Owner.
 ------------------------------------------------------------------------------------------------------------------------------------------------------------------------- -----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------                                                                                                                                                      |
