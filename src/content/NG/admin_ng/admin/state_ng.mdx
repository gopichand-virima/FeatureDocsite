# State

Use this function to configure states for various modules.

In the main window, click **Admin > [Module] > State**. The State window displays.

New State

From the *Select Actions* drop-down list, choose **New State**. The New State window displays.

Enter a *Name* and *Description* for this state.

Configure additional state properties as needed.

When all selections/entries are made, click **Add**.
