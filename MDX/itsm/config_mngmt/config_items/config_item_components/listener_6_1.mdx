# Listeners

This function is only available with certain asset types. Therefore, the relevant tab may not be displayed.


Use this function to check for connection requests, using the configured protocol and port. The rules defined for a listener determine how the load balancer routes requests to its registered targets.

- In the main window, select **ITSM > Configuration Management > CMDB**.
- Search for and select the applicable asset record.
- When the asset record window displays, click the **Components** tab.
- Click **Listeners**.

!

!


### Add Listener

- Click **Listeners**. The Listener dialog box displays.
- Enter the *Load Balancer Protocol*, *Load Balancer ARN*, and *Listener ARN*.
- Click **Add.**

!


### Edit Listener

Not all functions include an option to edit record line items shown in the main window.


- Navigate to the window containing the record to edit.
- Click the line containing the record. The applicable window or dialog box displays.
- Make the necessary modifications.
- Click **Save** or **Add**, as applicable.


### Delete Listener

While viewing the list of existing records, select a record(s) to delete, and click the **Delete** button.
