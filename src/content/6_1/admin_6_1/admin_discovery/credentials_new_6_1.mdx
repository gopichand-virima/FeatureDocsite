# New Credentials

This function enables the addition of credentials for SSH/SNMP/Windows components by entering values for the parameters present in the New Credentials window. Credentials are encrypted and stored in the Virima customer database.

## <span class="Strong_5">Credential Types</span>

The following are the types of available credentials.<span style="color: #dc143c;"> </span>

<div class="Bullet1" style="color: #000000;">

**AWS_Account.** Used for importing AWS Resources.

</div>

<div class="Bullet1" style="color: #000000;">

**AZURE_Account**. Used for importing Azure Resources.

</div>

<div class="Bullet1" style="color: #000000;">

<span class="Strong_4" style="font-weight: bold;">EMC_VNX.</span><span class="span_5"> Used to scan the EMC storage server. </span>

</div>

<div class="Bullet1" style="color: #000000;">

**Kubernetes**. Used for scanning Kubernetes Clusters.

</div>

<div class="Bullet1" style="color: #000000;">

**Meraki**. Used to import Meraki devices such as Access Points, Switches, Firewalls, and Routers.

</div>

<div class="Bullet1">

<span class="Strong_4" style="font-weight: bold;">SNMP </span>and<span class="Strong_4" style="font-weight: bold;"> SNMP_v3.</span><span class="span_5"> Used to scan SNMP device. </span>

</div>

<div class="Bullet1" style="color: #000000;">

**SQL_Server.** Used for scanning the databases running inside MSSQL Server.

</div>

<div class="Bullet1">

<span class="Strong_4" style="font-weight: bold;">SSH.</span><span class="span_5"> Provided for SSH Probes (mainly </span><span class="Strong_4">Unix</span><span class="span_5"> Hosts and</span><span class="Strong_4"> ESXi</span><span class="span_5"> Hosts). Specify the </span><span class="Strong_4">Username</span><span class="span_5">, </span><span class="Strong_4">Password</span><span class="span_5"> and </span><span class="Strong_4">Priority</span><span class="span_5">.</span>

</div>

<div class="Bullet1">

<span class="Strong_4" style="font-weight: bold;">SSH_PRIVATEKEY.</span><span class="span_5"> Provided SSH Probes (for Hosts with a PRIVATE KEY). </span>

</div>

<div class="Bullet1" style="color: #000000;">

<span class="Strong_4" style="font-weight: bold;">VCenter</span><span class="span_5">. Used to scan Vcenter's through API. </span>

</div>

<div class="Bullet1" style="color: #000000;">

<span style="color: #000000;">**VMWare**. Used for scanning VMWare vCenter & vSphere servers.</span>

</div>

<div class="Bullet1">

<span class="Strong_4" style="font-weight: bold;">Windows.</span><span class="span_5"> Used for application dependency Discovery in windows hosts. </span>

</div>

<div class="Bullet1">

<span class="Strong_4" style="font-weight: bold;">Windows_AD</span><span class="span_5">**/_Domain_Controller.** Used to scan domain networks. It is recommended to provide domain controller credentials.</span>

</div>

<div class="Bullet1" style="color: #000000;">

**XtremIO**. Used for scanning XtremIO storage servers.

</div>

## New Credential

<div class="NumList1" madcap:conditions="Default.OldUI" madcap:autonum="1. ">

In the main window, click **Admin /> Discovery /> Credentials**. The Credentials window displays.

</div>

<div class="NumList2" madcap:conditions="Default.OldUI" madcap:autonum="2. ">

<span class="span_1">From the *Select Actions* drop-down list, choose </span><span class="Strong_2" style="font-weight: bold;">New Credentials</span>. A new window displays.

</div>

<div class="Ind50" madcap:conditions="Default.OldUI">

<span class="image placeholder" original-image-src="../Resources/Images/Admin%20-%20Discovery/credentials-new-old.PNG" original-image-title=""></span>

</div>

<div class="NumList1" madcap:conditions="Default.NewUI" madcap:autonum="1. ">

In the main window, click the **gear** icon and select **Discovery /> Credentials**.

</div>

<div class="Ind50" madcap:conditions="Default.NewUI">

<span class="image placeholder" original-image-src="../Resources/Images/Admin%20-%20Discovery/icon-credentials.PNG" original-image-title=""></span>

</div>

<div class="Ind50" madcap:conditions="Default.NewUI">

The Credentials window displays.

</div>

<div class="Ind50" madcap:conditions="Default.NewUI">

<span class="image placeholder" original-image-src="../Resources/Images/Admin%20-%20Discovery/credentials-main-new.PNG" original-image-title="" style="width: 625px;height: 102px;"></span>

</div>

<div class="NumList2" madcap:conditions="Default.NewUI" madcap:autonum="2. ">

Click **New Credentials**.

</div>

<div class="Ind50" madcap:conditions="Default.NewUI">

<span class="image placeholder" original-image-src="../Resources/Images/Admin%20-%20Discovery/credentials-new-new.PNG" original-image-title="" style="width: 623px;height: 112px;"></span>

</div>

<div class="numlist2" madcap:autonum="3. ">

<span class="span_1">In the *Name* field, enter a name for this credential.</span>

</div>

<div class="numlist2" madcap:autonum="4. ">

In the *Credential Type* field, click the drop-down list select an available type.  

</div>

|     |                                                                            |
|-----|----------------------------------------------------------------------------|
|     | Based on the type of credential selected, the fields in the window change. |

<div class="NumList2" madcap:autonum="5. ">

Provide the required details for the type selected (refer to the list above).

</div>

<div class="NumList2" madcap:autonum="6. ">

If an *IP Range* field is shown, enter the IP Address information in one of the below formats. (This field is not required.)

</div>

<div class="Bullet2">

192.168.48.1 (Single IP)

</div>

<div class="Bullet2">

192.168.48.1 192.168.49.2 (Multiple IPs of Different Subnets)

</div>

<div class="Bullet2">

192.168.48.1,2,3,4 (Multiple IPs of Same Subnet)

</div>

<div class="Bullet2">

192.168.48.1-50 (A wide Range of IPs in a Subnet)

</div>

<div class="Bullet2">

192.168.48.0/24 (Single Subnet/CIDR Notation)

</div>

<div class="Bullet2">

192.168.48.0/24 192.168.49.0/24 (Multiple Subnets/CIDR Notation)

</div>

<div class="numlist2" madcap:autonum="7. ">

When all selections/entries are made, click **Add**.

</div>
