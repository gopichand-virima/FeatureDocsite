# Blueprints: History

Use this function to view changes, modifications and actions for the selected blueprint.


This information can only be viewed.


In the main window, select **Admin > SACM > Blueprints**. The Blueprints window displays.

Select the record for the blueprint record you want to view. A new window displays.
