# Imported Assets

Use this function to view the results of data imported during the data files import operation and move them (if applicable) to CMDB. An asset can also be rescanned through this function.

 -- ----- ------------------------------------------------------------------------------------------------------------
 1. In the navigation pane, select **Discovery Scan \> Imported Assets**. The Imported Assets window displays.
 -- ----- ------------------------------------------------------------------------------------------------------------

[A screenshot of a computer Description automatically generated]{.image .placeholder original-image-src="../../Resources/Images/Discovery/imported-assets.png" original-image-title=""

![imported assets main new](../../Resources/Images/Discovery/imported-assets-main-new.png)

 -- ----- --------------------------------------------------------------------------------------------
 2. Select at least one or more records. (When rescanning, the asset must have an IP address.)
 -- ----- --------------------------------------------------------------------------------------------

 -- ----- -------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------
 3. From the *Select Actions* drop-down list, choose either [Move to CMDB](../Move-CMDB.mdx) or [Re-Scan](../../Common%20Functions/Re-scan.mdx){.MCTopicPopup .MCTopicPopupHotSpot .a}.
 -- ----- -------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------

 -- ----- -------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------
 4. From the *Select Actions* drop-down list, choose either [Move to CMDB](../Move-CMDB.mdx) or [Re-Scan](../../Common%20Functions/Re-scan.mdx){.MCTopicPopup .MCTopicPopupHotSpot .a}.
 -- ----- -------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------

 -- ----- -------------------------------
 5. At the prompt, click **Yes**.
 -- ----- -------------------------------
