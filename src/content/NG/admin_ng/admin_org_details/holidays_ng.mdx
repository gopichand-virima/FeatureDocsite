# Holidays

Use this function to specify holidays, which are used to auto-calculate resolution dates as they relate to incidents, changes, and problems.

 -- ----- --------------------------------------------------------------------------------------------------------
 1. In the main window, click **Admin \> Organizational Details \> Holiday**. The Holiday window displays.
 -- ----- --------------------------------------------------------------------------------------------------------

[[[Closed]{.image .placeholder .MCDropDown_Image_Icon original-image-src="../../Skins/Default/Stylesheets/Images/transparent.gif" original-image-title="" height="12" width="16" Holiday](javascript:void(0)){.MCDropDownHotSpot .dropDownHotspot .MCDropDownHotSpot_ .MCHotSpotImage aria-expanded="false"}]{.MCDropDownHead .dropDownHead}

 -- ----- -------------------------------------------------------------
 1. Click **New Holiday**. The Add Holiday dialog box displays.
 -- ----- -------------------------------------------------------------

[[[Closed]{.image .placeholder .MCDropDown_Image_Icon original-image-src="../../Skins/Default/Stylesheets/Images/transparent.gif" original-image-title="" height="12" width="16" Holiday](javascript:void(0)){.MCDropDownHotSpot .dropDownHotspot .MCDropDownHotSpot_ .MCHotSpotImage aria-expanded="false"}]{.MCDropDownHead .dropDownHead}

To edit the holiday details:

 -- ----- -------------------------------
 1. Select a holiday in the list.
 -- ----- -------------------------------

 -- ----- ---------------------------------------------------------------
 2. In the Update Holiday dialog box, make the necessary changes.
 -- ----- ---------------------------------------------------------------

 -- ----- -----------------
 3. Click **Save**.
 -- ----- -----------------

[[[Closed]{.image .placeholder .MCDropDown_Image_Icon original-image-src="../../Skins/Default/Stylesheets/Images/transparent.gif" original-image-title="" height="12" width="16" Holiday\
](javascript:void(0)){.MCDropDownHotSpot .dropDownHotspot .MCDropDownHotSpot_ .MCHotSpotImage aria-expanded="false"}]{.MCDropDownHead .dropDownHead}

To delete an existing holiday:

 -- ----- -------------------------------
 1. Select a holiday in the list.
 -- ----- -------------------------------

 -- ----- -------------------------------------------------------------------------------------------------------------------------------------------------
 2. From the *Select Actions* drop-down list, choose **Delete**. If a confirmation message is displayed, type **Delete** and then click **Delete**.
 -- ----- -------------------------------------------------------------------------------------------------------------------------------------------------
