# Kubernetes Namespaces

 --- -------------------------------------------------------------------------------------------------------------
   This function is only available with certain asset types. Therefore, the relevant tab may not be displayed.
 --- -------------------------------------------------------------------------------------------------------------

 -- ----- ------------------------------------------------------
 1. While viewing an asset record, click **Components**.
 -- ----- ------------------------------------------------------

 -- ----- ----------------------------------
 2. Click **Kubernetes Namespaces**.
 -- ----- ----------------------------------

::: Ind50
![kubernetes namespaces new 482x159](../../../../Resources/Images/Components/kubernetes-namespaces-new_482x159.png)

![arrow border](../../../../Resources/Images/arrow-border.PNG)

 -- ----- --------------------------------------------------------------
 1. Click **Add**. The Kubernetes Namespace dialog box displays.
 -- ----- --------------------------------------------------------------

::: Ind50
![kubernetes namespaces add new 381x119](../../../../Resources/Images/Components/kubernetes-namespaces-add-new_381x119.png)

 -- ----- ------------------------------------
 2. Click **New Kubernetes Namespace**
 -- ----- ------------------------------------

 -- ----- -------------------------------------------------
 3. Enter the applicable information in each field.
 -- ----- -------------------------------------------------

 -- ----- --------------------------------------------
 4. When all entries are made, click **Save.**
 -- ----- --------------------------------------------

[[[Closed]{.image .placeholder .MCDropDown_Image_Icon original-image-src="../../../../../Skins/Default/Stylesheets/Images/transparent.gif" original-image-title="" height="12" width="16" Kubernetes Namespace](javascript:void(0)){.MCDropDownHotSpot .dropDownHotspot .MCDropDownHotSpot_ .MCHotSpotImage aria-expanded="false"}]{.MCDropDownHead .dropDownHead}

 -- ----- --------------------------------
 1. Open the applicable blueprint.
 -- ----- --------------------------------

 -- ----- -------------------------------------
 2. Navigate to the **Components** tab.
 -- ----- -------------------------------------

 -- ----- ----------------------------
 3. Click a sub-component tab.
 -- ----- ----------------------------

 -- ----- --------------------------------------------
 4. In the list, select the applicable record.
 -- ----- --------------------------------------------

 -- ----- --------------------------
 5. Do one of the following:
 -- ----- --------------------------

- [[[Closed]{.image .placeholder .MCDropDown_Image_Icon original-image-src="../../../../../Skins/Default/Stylesheets/Images/transparent.gif" original-image-title="" height="12" width="16" Kubernetes Namespace](javascript:void(0)){.MCDropDownHotSpot .dropDownHotspot .MCDropDownHotSpot_ .MCHotSpotImage aria-expanded="false"}]{.MCDropDownHead .dropDownHead}
 --- --------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------
   Deleting is a permanent action and cannot be undone. Deleting may affect other functionality and information in the application such as data in configured reports, fields in windows, selectable options, etc. Therefore, be sure to understand the potential effects before making a deletion.
 --- --------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------

While viewing the list of existing records, select a record(s) to delete, and click the **Delete** button.
