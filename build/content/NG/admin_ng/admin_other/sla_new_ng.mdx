# New Service Level Agreements (SLA)

Use this function to create a new service level agreement.

In the main window, select **Admin > Other > Service Level Agreements**. The Service Level Agreements window displays.

In the *Select Actions* field drop-down list, choose **New Service Level Agreement**.

Complete the tabs according to the following:

- [Details](/admin_other/sla_details_ng.mdx)
- [Tasks](/admin_other/sla_tasks_ng.mdx)
- [Comments](/admin_other/sla_comments_ng.mdx)
- [Attachments](/admin_other/sla_attachments_ng.mdx)

When all selections/entries are made, click **Add**.
