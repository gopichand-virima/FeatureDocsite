# Credential Details

Use this function to edit details for credentials.


Not all fields are enabled to accept changes as the fields vary based on the Credential Type configured.


- In the main window, click **Admin > Discovery > Credentials**. The Credential window displays.
- Select an existing record. A new window displays with the **Details** tab selected. !
- Make the necessary modifications.
- Modify the tabs according to the following:
- [Components](/itsm/config_mngmt/config_items/config_item_components/component_tab_options_6_1.mdx)
- [Relationships](/itsm/config_mngmt/config_items/config_item_details/config_items_relationships_6_1.mdx)
- [SLA](/itsm/config_mngmt/config_items/config_item_details/config_items_sla_6_1.mdx)
- [Maintenance](/itsm/config_mngmt/config_items/config_item_details/config_items_maintenance_6_1.mdx)
- [Tasks](/itsm/release_mngmt/tasks_6_1.mdx)
- [Comments](/discovery/comments_6_1.mdx)
- [Attachments](/discovery/attachments_6_1.mdx)
- [Audits](/itsm/config_mngmt/audits/audits_6_1.mdx)
- [History](/itsm/config_mngmt/config_items/config_item_details/config_items_history_6_1.mdx)
- When all selections/entries are made, click **Save**.
