# Probe Workflow

Use this function to configure workflows for probe execution during discovery.

In the main window, click **Admin > Discovery > Probe Workflow**. The Probe Workflow window displays.

From the *Select Actions* drop-down list, choose an action: **New Probe Workflow**, [Delete](/common_topics/delete_remove_5_13.mdx), or [Export](/discovery/import_data_files/export_5_13.mdx).
