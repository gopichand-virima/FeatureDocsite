# Discovery Scan Overview

Discovery Scan automatically identifies and catalogs IT assets across your network infrastructure, providing real-time visibility into your environment.

## Key Features

- Automated network discovery
- Agentless and agent-based scanning
- Cloud resource discovery
- Real-time asset updates
- Integration with CMDB
- AI-powered asset classification
