# CI Details and tabs

When you open a CI (e.g., AST000001), the following key tabs are available:

<span class="image placeholder" original-image-src="../Resources/Images/CMDB%20-%20Left-Nav/Configuration%20Management%20Database_36.png" original-image-title=""></span>

|                                                            |                                                                                     |
|------------------------------------------------------------|-------------------------------------------------------------------------------------|
| <span class="Strong" style="font-weight: bold;">Tab</span> | <span class="Strong" style="font-weight: bold;">Purpose</span>                      |
| <span class="Strong">Details</span>                        | Displays asset information such as name, status, IP address, OS, and hardware info  |
| <span class="Strong">Components</span>                     | Shows associated components based on the Blueprint                                  |
| <span class="Strong">Log On Events</span>                  | Lists login activity and access events for the device or system                     |
| <span class="Strong">ITSM</span>                           | Links to incidents, requests, problems, or changes associated with the CI           |
| <span class="Strong">Relationships</span>                  | Shows connections between this CI and others                                        |
| <span class="Strong">SLA</span>                            | Defines response and resolution timelines based on ITSM priorities                  |
| <span class="Strong">Maintenance</span>                    | Schedule maintenance tasks for a CI, such as rebooting or running specific commands |
| <span class="Strong">Attachments</span>                    | Upload and view files like screenshots, reports, or manuals                         |
| <span class="Strong">Audits</span>                         | Displays audits the CI is part of, including results and history                    |
| <span class="Strong">History</span>                        | Shows modification history, including timestamps and users                          |
| <span class="Strong">Vulnerability</span>                  | Lists known vulnerabilities, threat severity, and risk details                      |
| <span class="Strong">Tasks</span>                          | Tracks tasks or actions assigned to this CI                                         |
| <span class="Strong">Comments</span>                       | Used for collaboration and notes by users                                           |

Use these tabs to <span class="Strong">view or edit data</span>, understand dependencies, and track the CI's role in services and infrastructure.
