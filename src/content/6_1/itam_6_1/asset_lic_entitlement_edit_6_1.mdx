# Edit Asset License Entitlement

This feature enables editing the configured values of the existing contract, and update, and then save the same in the application.

Click to open required Asset License Management. The Edit Asset License Entitlement window displays.

The Edit Asset License Entitlement displays the following information:

- Tasks
- Comments
- Attachment
- History
