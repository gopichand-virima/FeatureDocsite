# Probe Workflow

Use this function to define a workflow for a probe. A Probe Workflow is a sequence of steps executed while discovering a host by entering appropriate scripts.

In the main window, click **Admin \> Probe Workflow**. The Probe Workflow window displays.

![Probe Workflow 6.1](../Resources/Images/Admin - Discovery/Probe-Workflow-6.1.png)

![arrow border](../Resources/Images/arrow-border.PNG)

[[[Closed]{.image .placeholder .MCDropDown_Image_Icon original-image-src="../../Skins/Default/Stylesheets/Images/transparent.gif" original-image-title="" height="12" width="16" GroupProbe](javascript:void(0)){.MCDropDownHotSpot .dropDownHotspot .MCDropDownHotSpot_ .MCHotSpotImage aria-expanded="false"}]{.MCDropDownHead .dropDownHead}

 -- ----- ---------------------------------------------------------------------------------------------------------
 1. From the *Select Actions* drop-down list, choose **New GroupProbe**.The New GroupProbe window displays.
 -- ----- ---------------------------------------------------------------------------------------------------------

 -- ----- -------------------------------------
 2. Enter the *Name* and *Description*.
 -- ----- -------------------------------------

 -- ----- -----------------------------------------------------------------------------------
 3. Enter a *Script* program that invokes the probe. A partial sample is shown below.
 -- ----- -----------------------------------------------------------------------------------

::: Ind50
![probe workflow script example 546x179](../Resources/Images/Admin - Discovery/probe-workflow-script-example_546x179.png)

 -- ----- -------------------------------------------
 4. When all entries are made, click **Add**.
 -- ----- -------------------------------------------

 --- -----------------------------------------------------------------------------------------
   Not all functions include an option to edit record line items shown in the main window.
 --- -----------------------------------------------------------------------------------------

 -- ----- -------------------------------------------------------
 1. Navigate to the window containing the record to edit.
 -- ----- -------------------------------------------------------

 -- ----- -------------------------------------------------------------------------------------
 2. Click the line containing the record. The applicable window or dialog box displays.
 -- ----- -------------------------------------------------------------------------------------

 -- ----- -----------------------------------
 3. Make the necessary modifications.
 -- ----- -----------------------------------

 -- ----- -------------------------------------------
 4. Click **Save** or **Add**, as applicable.
 -- ----- -------------------------------------------

When editing an existing item, other options may be available, such as adding [Tasks](/itsm/release_mngmt/tasks_6_1.mdx){.MCTopicPopup .MCTopicPopupHotSpot .a}, [Comments](/discovery/comments_6_1.mdx){.MCTopicPopup .MCTopicPopupHotSpot .a}, and [Attachments](/discovery/attachments_6_1.mdx){.MCTopicPopup .MCTopicPopupHotSpot .a} and viewing [History](/common_topics/history_6_1.mdx){.MCTopicPopup .MCTopicPopupHotSpot .a}.

 --- ---------------------------------------------------------------------------------------
   The options change based upon the function; therefore, not every option may be shown.
 --- ---------------------------------------------------------------------------------------
