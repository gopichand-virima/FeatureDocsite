# Software Usage Report

Use this function to view software usage metric information for a software deemed a “Major Software.”

|     |                                                                                                                                      |
|-----|--------------------------------------------------------------------------------------------------------------------------------------|
|     | Software usage monitoring is configured through [Admin /> Discovery /> Major Software](../Admin%20-%20Discovery/Major-Software.htm). |

<div class="NumList1" madcap:conditions="Default.OldUI" madcap:autonum="1. ">

In the main window, click **Admin /> SACM /> Software Usage Report**. The Report window displays.

</div>

<div class="NumList2" madcap:conditions="Default.OldUI" madcap:autonum="2. ">

From the drop-down list, select a predefined time period (for example, last 30 days) or specific a custom date range. When entering a custom date range, enter the from and to dates, then click **Apply**.

</div>

<div class="Ind50" madcap:conditions="Default.OldUI">

<span class="image placeholder" original-image-src="../Resources/Images/Admin%20-%20SACM/sw-usage-report-list-old.PNG" original-image-title=""></span>

</div>

<div class="NumList2" madcap:conditions="Default.NewUI" madcap:autonum="3. ">

To change the report duration, click the drop-down list and select or specify a different time period. After you make a selection, the list updates.

</div>

<div class="NumList2" madcap:autonum="4. ">

To view additional information for the report, click the line item. The window updates and more information about the report is shown, including a link to the corresponding configuration item.

</div>

<div class="Ind50" madcap:conditions="Default.OldUI">

<span class="image placeholder" original-image-src="../Resources/Images/Admin%20-%20SACM/sw-usage-report-details-old.PNG" original-image-title=""></span>

</div>

<div class="NumList2" madcap:autonum="5. ">

To export the report, click **Export**. The Export Confirmation window displays.

</div>

<div class="NumList2" madcap:autonum="6. ">

Click **Continue**.

</div>
