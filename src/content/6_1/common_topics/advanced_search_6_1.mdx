## Advanced Search

Use this function to search assets specifically as per defined values.

When this option displays in a window (such as Adding a New Task), click **Advanced Search.** The Advanced Search window displays.

<div class="numlist1" madcap:conditions="Default.OldUI" madcap:autonum="1. ">

In /<**choose field**/> (for example, Display Name), click the drop-down list and select from the field names shown. A "field name" can be such items as Display Name, Roles, Groups, and so forth.

</div>

<div class="numlist2" madcap:conditions="Default.NewUI" madcap:autonum="2. ">

For the /<**choose operator**/> (for example, Does Not Contain), click the drop-down list and select from the choices shown. An "operator" can be such items as Contains, Does Not Contain, Starts with, and so forth.

</div>

<div class="numlist2" madcap:conditions="Default.OldUI" madcap:autonum="3. ">

For */<***value***/>*, click the drop-down list and select the specific criteria to search for based on the field type. For example, if the *Field Type* selected is "Display Name," the /<value/> can be a portion of the possible display name.

</div>

<div class="numlist2" madcap:autonum="4. ">

To add an additional row with a field type, operator, and value, click **AND** or **OR**. A new row is added.

</div>

<div class="numlist2" madcap:autonum="5. ">

Repeat the above steps for each field.

</div>

<div class="numlist2" madcap:autonum="6. ">

To remove a row, click the **X** at the end of the row.

</div>

<div class="Ind50 step-media">

<span class="image placeholder" original-image-src="../Resources/Images/Common%20Topics/Advanced_Search_6.1.png" original-image-title=""></span>

</div>

<div class="numlist2" madcap:autonum="7. ">

When all selections/entries are made, click **Run**. The matching search results are displayed.

</div>

<div class="Bullet2">

If no results are found, a note displays in the results window.

</div>

<div class="Bullet2">

If there are multiple results that exceed the length of the results window.

</div>

<div class="numlist2" madcap:autonum="8. ">

Select the applicable result and click **Add**.

</div>

<div class="Ind50 step-media">

The selected items display in the applicable fields on the window from which the search request originated.

</div>
