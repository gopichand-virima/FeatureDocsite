# Problem Closure Code

Use this function to configure a code with its description for the Closed state in the Problem Management.

In the main window, click **Admin \> Problem Management \> Problem Closure Code**. The Problem Closure Code window displays.

![problem closure code main old](../Resources/Images/Admin - Problem/problem-closure-code-main-old.PNG)

![arrow border](../Resources/Images/arrow-border.PNG)

[[[Closed]{.image .placeholder .MCDropDown_Image_Icon original-image-src="../../Skins/Default/Stylesheets/Images/transparent.gif" original-image-title="" height="12" width="16" Problem Closure Code](javascript:void(0)){.MCDropDownHotSpot .dropDownHotspot .MCDropDownHotSpot_ .MCHotSpotImage aria-expanded="false"}]{.MCDropDownHead .dropDownHead}

 -- ----- ------------------------------------------------------------------------------------------------------------------------------
 1. From the *Select Actions* drop-down list, choose **New Problem Closure Code**. The New Problem Closure Code window displays.
 -- ----- ------------------------------------------------------------------------------------------------------------------------------

 -- ----- -----------------------------------
 2. Enter a *Name* and *Description*.
 -- ----- -----------------------------------

 -- ----- ----------------
 3. Click **Add**.
 -- ----- ----------------

[[[Closed]{.image .placeholder .MCDropDown_Image_Icon original-image-src="../../Skins/Default/Stylesheets/Images/transparent.gif" original-image-title="" height="12" width="16" Problem Closure Code](javascript:void(0)){.MCDropDownHotSpot .dropDownHotspot .MCDropDownHotSpot_ .MCHotSpotImage aria-expanded="false"}]{.MCDropDownHead .dropDownHead}

 --- -----------------------------------------------------------------------------------------
   Not all functions include an option to edit record line items shown in the main window.
 --- -----------------------------------------------------------------------------------------

 -- ----- -------------------------------------------------------
 1. Navigate to the window containing the record to edit.
 -- ----- -------------------------------------------------------

 -- ----- -------------------------------------------------------------------------------------
 2. Click the line containing the record. The applicable window or dialog box displays.
 -- ----- -------------------------------------------------------------------------------------

 -- ----- -----------------------------------
 3. Make the necessary modifications.
 -- ----- -----------------------------------

 -- ----- -------------------------------------------
 4. Click **Save** or **Add**, as applicable.
 -- ----- -------------------------------------------
