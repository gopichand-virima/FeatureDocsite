# Asset Types

Use this function to define and manage asset types and their specific attributes.

Asset Types categorize assets and determine which fields and properties are applicable to each category.

In the main window, click **Admin > Asset Management > Asset Types**. The Asset Types window displays.

## Asset Type List

View all asset types:

**Type Columns**:

- **Type Name**: Asset type name
- **Category**: Hardware, Software, Other
- **Assets**: Count of assets
- **Custom Fields**: Number of type-specific fields
- **Status**: Active, Inactive

## Standard Asset Types

**Hardware Assets**:

- Server
- Network Device (Router, Switch, Firewall)
- Desktop Computer
- Laptop
- Monitor
- Printer
- Scanner
- Mobile Device
- Tablet
- Storage Device
- Peripheral (Keyboard, Mouse, etc.)

**Software Assets**:

- Operating System
- Application
- License
- Cloud Subscription
- Development Tool
- Database

**Other Assets**:

- Furniture
- Vehicle
- Office Equipment
- Facility

## Create Asset Type

From the *Select Actions* drop-down list, choose **New Asset Type**.

**Type Details**:

- **Type Name**: Laptop
- **Category**: Hardware
- **Parent Type**: Computer
- **Description**: Portable computing device
- **Icon**: Upload icon
- **Status**: â˜‘ Active

**Standard Fields**:

All asset types include:

- Asset Tag
- Serial Number
- Manufacturer
- Model
- Purchase Date
- Purchase Cost
- Assigned To
- Location
- Status

**Type-Specific Fields**:

For Laptop type:

- Processor
- RAM
- Storage Capacity
- Operating System
- Screen Size
- Battery Health
- Warranty Expiration

**Lifecycle States**:

- Ordered
- In Stock
- Deployed
- In Use
- In Repair
- Retired
- Disposed

Click **Save**.
