# Ivanti Credential

Use this function to specify a credential for the Ivanti domain.

 -- ----- ----------------------------------------------------------------------------
 1. In the main window, select **Admin \> Integrations \> Ivanti Credential**.
 -- ----- ----------------------------------------------------------------------------

::: {.Ind50
![ivanti credential old](../Resources/Images/Admin - Integration/ivanti-credential-old.PNG)

 -- ----- ---------------------------------
 2. Enter a *Domain* and *API Key*.
 -- ----- ---------------------------------

 -- ----- ---------------------------------------------
 3. When both entries are made, click **Save**.
 -- ----- ---------------------------------------------
