# Priority Calculator

Use this function to set values for Urgency and Impact.

 --- -------------------------------------------------------
   Role access must be given to perform these functions.
 --- -------------------------------------------------------

In the main window, click **Admin \> Problem Management \> Problem Priority Calculator**. The Priority Calculation window displays.

![problem priority calculator main old](../Resources/Images/Admin - Problem/problem-priority-calculator-main-old.PNG)

![arrow border](../Resources/Images/arrow-border.PNG)

[[[Closed]{.image .placeholder .MCDropDown_Image_Icon original-image-src="../../Skins/Default/Stylesheets/Images/transparent.gif" original-image-title="" height="12" width="16" Priority Calculator](javascript:void(0)){.MCDropDownHotSpot .dropDownHotspot .MCDropDownHotSpot_ .MCHotSpotImage aria-expanded="false"}]{.MCDropDownHead .dropDownHead}

 -- ----- --------------------------------------------------------------------------------------------------------------------------------------
 1. Set the values according to the table below. Once the **Impact** and **Urgency** are defined, the **Priority** is set automatically.
 -- ----- --------------------------------------------------------------------------------------------------------------------------------------

 --- --------------------------------------------------------------------------------------------------------------
   Priority is always set to 1-Very High if the Reporter is set up as a VIP (done through the user\'s profile).
 --- --------------------------------------------------------------------------------------------------------------

 -- ----- -----------------------------------------------
 2. When all selections are made, click **Save**.
 -- ----- -----------------------------------------------

### Priority Calculator Values

 Impact Urgency Priority
 -------- --------- ---------------
 High High 1 - Very High
 High Medium 2 - High
 High Low 3 - Medium
 Medium High 2 - High
 Medium Medium 3 - Medium
 Medium Low 4 - Low
 Low High 3 - Medium
 Low Medium 4 - Low
 Low Low 5 - Very Low
