# Edit AD User Import Scans

Use this function to edit the details for an AD User Import Scans record.


The **Details** cannot be edited.


In the AD User Import Scans window, select a record to edit.

Modify any of the following information:

- [Details](/discovery/ad_user_import_details_6_1.mdx)
- [Tasks](/discovery/ad_user_import_tasks_6_1.mdx)
- [Comments](/discovery/ad_user_import_comments_6_1.mdx)
- [Attachments](/discovery/ad_user_import_attachments_6_1.mdx)

When all selections/entries are made, click **Save**.
