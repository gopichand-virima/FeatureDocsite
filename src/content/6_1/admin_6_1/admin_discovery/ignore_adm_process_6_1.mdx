# [Ignore ADM Process]{style="background-color: #ffffff;"}

Use this function to ignore ADM back-end processes when creating relationships.

 -- ----- --------------------------------------------------------------------------------------------------------------------------
 1. In the main window, click **Admin \> Discovery \> Ignore ADM Process**. The Ignore ADÂ Process Names dialog box displays.
 -- ----- --------------------------------------------------------------------------------------------------------------------------

::: Ind50
[]{.image .placeholder original-image-src="../Resources/Images/Admin%20-%20Discovery/ignore-adm-process-names_381x119.png" original-image-title=""

 -- ----- -------------------------------------------------------------------------------------------
 2. Enter the name(s) of the process to ignore. Separate multiple process names with a comma.
 -- ----- -------------------------------------------------------------------------------------------

 -- ----- ----------------
 3. Click **Add**.
 -- ----- ----------------

 -- ----- -----------------------------------------
 4. System displays the successful message.
 -- ----- -----------------------------------------

 -- ----- ---------------
 5. Click **OK**.
