# New Software License Key

Delete this text and replace it with your own content.

<div class="NumList1" madcap:autonum="1. ">

From the navigation pane, select **ITAM /> Software Asset Management /> Software License Keys**. The Software License Keys window displays.

</div>

<div class="NumList2" madcap:autonum="2. ">

From the *Select Actions* drop-down list, choose **New**. The Software License Key dialog box displays.

</div>

<div class="Ind50">

<span class="image placeholder" original-image-src="../../Resources/Images/ITAM/sw-lic-key-dialog-box.PNG" original-image-title="" style="width: 468px;height: 123px;"></span>

<div class="NumList2" madcap:autonum="3. ">

In the *License Key* field, enter the license key.

</div>

<div class="NumList2" madcap:autonum="4. ">

In the *Asset Name* field, enter the name of the software for which the key applies.

</div>

<div class="NumList2" madcap:autonum="5. ">

</div>
