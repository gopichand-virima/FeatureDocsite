# New SAML Configuration

 -- ----- --------------------------------------------------------------------------------
 1. While viewing the SAML Configuration window, click **New SAML Configuration**.
 -- ----- --------------------------------------------------------------------------------

::: {.Ind50
![saml config add new 483x76](../Resources/Images/Admin - Users/saml-config-add-new_483x76.png)

 -- ----- --------------------------------------------------
 2. In the *Name* field, enter the name of the SAML.
 -- ----- --------------------------------------------------

 -- ----- --------------------------------------------------------------------------------------------
 3. For the *Signature Algorithm*, click the drop-down list and select the required signature.
 -- ----- --------------------------------------------------------------------------------------------

 -- ----- ----------------
 4. Click **Add**.
 -- ----- ----------------
