# Import AWS Record

Use this function to enable users to import the required AWS into the Discovery application.

<div class="NumList1" madcap:autonum="1. ">

From the *Select Actions* drop-down list, choose **Import**. The Import AWS dialog box displays.

</div>

<div class="Ind50">

<span class="image placeholder" original-image-src="../Resources/Images/Discovery/import-from-aws.PNG" original-image-title="" madcap:conditions="Default.OldUI" style="width: 472px;height: 344px;"></span>

</div>

<div class="Ind50">

<span class="image placeholder" original-image-src="../Resources/Images/Discovery/import-from-aws-new.PNG" original-image-title="" madcap:conditions="Default.NewUI" style="width: 470px;height: 404px;"></span>

</div>

<div class="NumList2" madcap:autonum="2. ">

In the *Name* field, enter a name for the import action.

</div>

<div class="NumList2" madcap:conditions="Default.OldUI,Default.NewUI" madcap:autonum="3. ">

In the *Credential* field, click the drop-down list and choose the appropriate credential to use to leverage the AWS service.

</div>

<div class="NumList2" madcap:autonum="4. ">

Select all the required services present to be imported from the AWS label.

</div>

<div class="NumList2" madcap:autonum="5. ">

Click **Add** to update the AWS import in the application.

</div>

<div class="NumList2" madcap:autonum="2. ">

**Name**. Enter a *Name* to identify the Import.

</div>

<div class="NumList2" madcap:autonum="3. ">

**Credential**. Click the drop-down list and select the applicable *Credential* to be used to leverage AWS service.

</div>

<div class="NumList2" madcap:autonum="4. ">

Select the required services to import.

</div>

<div class="NumList2" madcap:autonum="5. ">

**Import Frequency**. Click the drop-down list and select the *Import Frequency*.

</div>

<div class="NumList2" madcap:autonum="6. ">

**Active**. If this import is active, select *Active*.

</div>

<div class="NumList2" madcap:autonum="7. ">

Click the drop-down list and select the *Timezone*.

</div>

<div class="NumList2" madcap:autonum="8. ">

Configure the time parameters (such as Seconds, Minutes, Hours and so forth.)

</div>

<div class="NumList2" madcap:autonum="9. ">

When all selections/entries are made, click **Add**.

</div>
