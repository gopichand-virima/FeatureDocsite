# Request Model

Use this function to configure and populate request templates that can be reused for defining service requests. These models can be used for commonly occurring requests to skip or reduce form completion time.

In the main window, click **Admin > Request Management > Request Model**. The Request Model window displays.

## New Request Model

From the *Select Actions* drop-down list, choose **New Request Model**. The New Request Model window displays.

Enter the request details that should be pre-populated:

- **Name**: Template name
- **Description**: Detailed description of the request type
- **Category**: Select request category
- **Subcategory**: Select request subcategory
- **Priority**: Set default priority
- **Impact**: Set default impact level
- **Urgency**: Set default urgency level
- **Assignment Group**: Default assignment group
- **Assignee**: Default assignee
- **Fulfillment Instructions**: Standard fulfillment steps
- **Approval Required**: Check if approval is needed

Configure additional pre-filled fields as needed.

When all selections/entries are made, click **Add**.

## Common Request Models

Examples of request models you might configure:

- **New Employee Onboarding**: Standard new hire IT setup
- **Software Installation**: Common software deployment requests
- **Access Request**: Standard access permission requests
- **Equipment Request**: Hardware provisioning
- **Password Reset**: User password assistance
- **Account Creation**: New user account setup
- **License Request**: Software license provisioning

## Edit Request Model

Navigate to the window containing the record to edit.

Click the line containing the record. The applicable window displays.

Make the necessary modifications.

Click **Save**.

## Delete Request Model

Click the line that contains the item to delete.

From the *Select Actions* drop-down list, choose **Delete**.
