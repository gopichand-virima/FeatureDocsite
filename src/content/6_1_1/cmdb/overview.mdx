# CMDB Overview

The Configuration Management Database (CMDB) is a central repository that stores information about configuration items (CIs) and their relationships within your IT infrastructure.

## Key Features

- Comprehensive CI management
- Relationship mapping and visualization
- Change tracking and audit trails
- Integration with discovery tools
- Automated updates and synchronization

## Getting Started

Learn how to manage your CMDB effectively:
1. Understanding configuration items
2. Creating and editing CIs
3. Managing relationships
4. Running reports and analytics

## Version Information

This documentation applies to Virima version **6.1.1**.
