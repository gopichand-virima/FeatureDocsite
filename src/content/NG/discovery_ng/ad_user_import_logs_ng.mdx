# AD User Import Logs

Use this function to view scanned results and the details for imported AD Users.

In the navigation pane, select **Discovery Scan > AD User Import Logs**. The AD User Import Logs window displays.

To view or edit the information, click on the applicable record in the list.

Edit AD User Import Logs Delete AD User Import Logs

AD User Import Logs cannot be deleted.
