# New Azure AD Configuration

<div class="NumList1" madcap:autonum="1. ">

While viewing the AD Configuration window, click **New AD Configuration**. The Export dialog box displays.

</div>

<div class="Ind50">

<span class="image placeholder" original-image-src="../Resources/Images/Admin%20-%20Users/azure-ad-config-export-dialog-box.PNG" original-image-title="" style="width: 472px;height: 278px;"></span>

</div>

<div class="NumList2" madcap:autonum="2. ">

Enter the *Name*, *Client Id*, *Tenant Id* and *Secret Key*.

</div>

<div class="NumList2" madcap:autonum="3. ">

Click **Next**. The Map Property dialog box displays.

</div>

<div class="Ind50">

<span class="image placeholder" original-image-src="../Resources/Images/Admin%20-%20Users/azure-ad-config-export-map-prop.PNG" original-image-title="" style="width: 472px;height: 448px;"></span>

</div>

<div class="NumList2" madcap:autonum="4. ">

Select required values to import (for example, First Name, Last Name and so forth).

</div>

<div class="NumList2" madcap:autonum="5. ">

Click the drop-down list arrow and select the property to map to.

</div>

<div class="Ind50">

<span class="image placeholder" original-image-src="../Resources/Images/Admin%20-%20Users/first-name-example.PNG" original-image-title=""></span>

</div>

<div class="NumList2" madcap:autonum="6. ">

When all selections/entires are made, click **Finish**.

</div>
