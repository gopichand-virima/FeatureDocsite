# Installing and Running the Windows Application

<span class="span_1">The Windows application can perform </span><span class="span_2">Unix System Discovery</span><span class="span_1">, </span><span class="span_2">Network Device Discovery</span><span class="span_1">, </span><span class="span_2">Pattern Discovery</span><span class="span_1">, and </span><span class="span_2">Windows System Discovery</span><span class="span_1">.</span>

<div class="NumList1" madcap:autonum="1. ">

<span class="span_1">In the main window, click </span><span class="span_2" style="font-weight: bold;">Admin</span><span class="span_1" style="font-weight: bold;"> /> </span><span class="span_2" style="font-weight: bold;">Discovery</span><span class="span_1" style="font-weight: bold;"> /> </span><span class="span_2" style="font-weight: bold;">Download Application. </span>The Download Application window displays (as shown below).

</div>

<div class="Ind50">

<span class="span_2" style="font-weight: bold;"><span class="image placeholder" original-image-src="../Resources/Images/Discovery/admin_discovery_download_app.png" original-image-title="" madcap:conditions="Default.OldUI" style="width: 407px;height: 115px;"></span></span>

</div>

<div class="Ind50">

<span class="image placeholder" original-image-src="../Resources/Images/Discovery/admin_discovery_download_app-new.png" original-image-title="" madcap:conditions="Default.NewUI" style="width: 401px;height: 168px;"></span>

</div>

<div class="NumList2" madcap:autonum="2. ">

<span class="span_1">Click </span><span class="span_2">**Download**. </span>

</div>

<div class="NumList2" madcap:autonum="3. ">

Double-click the **Discovery icon**.

</div>

<div class="NumList2" madcap:autonum="4. ">

On the User *Account Control* screen, click **Yes**. Once the setup is complete, the InstallShield Wizard launches.  

</div>

<div class="NumList2" madcap:autonum="5. ">

<span class="span_1">On the *Virima Login Credential* screen, enter the Virima </span><span class="span_2" style="font-weight: bold;">User Name</span><span class="span_1">, </span><span class="span_2" style="font-weight: bold;">Password</span><span class="span_1">, </span><span class="span_2" style="font-weight: bold;">Customer ID</span><span class="span_1">, </span><span class="span_2" style="font-weight: bold;">Client Name</span><span class="span_1">, and </span><span class="span_2" style="font-weight: bold;">IP Address</span>.

</div>

<div class="Bullet2">

<span class="span_2" style="font-weight: bold;">Customer ID</span><span class="span_1"> can be found in Virima under [Admin /> Organizational Details](../Admin%20-%20Org%20Details/Organizational-Details.htm). </span>

</div>

<div class="Bullet2">

**Client Name** is a generic name to assign to the client.

</div>

<div class="Ind50">

<span class="image placeholder" original-image-src="../Resources/Images/Discovery/admin-discovery-login-credential.png" original-image-title="" style="width: 443px;height: 335px;"></span> 

</div>

## Entering Credentials

<div class="NumList1" madcap:autonum="1. ">

To bypass entering the credential, click **Skip**. 

</div>

|     |                                                                                                                                                                                                                     |
|-----|---------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
|     | If this step is bypassed during the installation process, it must be configured after launching the Discovery Application in Virima. *See* <a href="Credentials.htm" style="font-style: italic;">Credentials</a>*.* |

<div class="NumList2" madcap:autonum="2. ">

<span class="span_1">Click </span><span class="span_2" style="font-weight: bold;">Next</span><span class="span_1">, wait for the information to be validated, then click **Next**. </span>

</div>

<div class="NumList2" madcap:autonum="3. ">

<span class="span_1">Click </span><span class="span_2" style="font-weight: bold;">Install</span><span class="span_1">. The installation process begins.</span>

</div>

<div class="NumList2" madcap:autonum="4. ">

When prompted, enter the <span class="span_2">Windows Service Log in</span><span class="span_1"> </span><span class="span_2" style="font-weight: normal; font-style: italic;">User Name</span><span class="span_1"> and </span><span class="span_2" style="font-weight: normal; font-style: italic;">Password</span><span class="span_1">. </span>

</div>

<div class="NumList2" madcap:autonum="5. ">

To bypass entering the credential information, click **Skip**. 

</div>

|     |                                                                                                                                                                                                                     |
|-----|---------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
|     | If this step is bypassed during the installation process, it must be configured after launching the Discovery Application in Virima. *See* <a href="Credentials.htm" style="font-style: italic;">Credentials</a>*.* |

<div class="NumList2" madcap:autonum="6. ">

<span class="span_1">Click </span><span class="span_2" style="font-weight: bold;">Next</span><span class="span_1">. Once the login credential is verified, a confirmation window displays.</span>

</div>

<div class="NumList2" madcap:autonum="7. ">

<span class="span_1">Click </span><span class="span_2">**Finish**.</span>

</div>

<div class="NumList2" madcap:autonum="8. ">

<span class="span_1">Launch the Discovery application. After a successful start, it displays in Virima under </span><span class="span_2">[<span class="span_2">Admin</span><span class="span_1"> /> </span><span class="span_2">Client</span>](Client.htm)</span><span class="span_1">.</span>

</div>
