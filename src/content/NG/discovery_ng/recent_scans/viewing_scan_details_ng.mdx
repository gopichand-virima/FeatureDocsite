# Viewing Scan Details

Clicking on a scan from the **Recent Scans** list opens a detailed view with multiple tabs:

# Details Tab

Displays high-level information about the scan:

- **Scan Name / ID**
- **Type of Scan** (e.g., Deep Host Scan, Basic Host Scan)
- **Status** (Running, Completed, Failed)
- **Client Used** (the discovery server that ran the scan)
- **Start Time** and **End Time**
- **Total Duration**
- **Schedule Type** (On Demand or Scheduled Scan)
- **Target IP Range or Hosts**

# **Tasks Tab**

Allows administrators to:

- Create and assign **tasks** related to the scan (e.g., "Review scan failures," "Re-credential Device XYZ").
- Track follow-up activities tied to the scan.

# Comments Tab

Enables team collaboration through **notes and annotations** on the scan.

# Attachments Tab

Supports uploading or linking files related to the scan:

- Screenshots
- Logs from external systems
- Diagnostic reports
- Configuration backups
