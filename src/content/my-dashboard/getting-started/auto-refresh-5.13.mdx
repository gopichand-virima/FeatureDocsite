# Auto Refresh

Configure automatic data refresh intervals to keep your views up-to-date with the latest information.

## Overview

The Auto Refresh function automatically updates data displayed in lists, dashboards, and reports at specified intervals, ensuring you always have current information without manual page reloads.

## Enabling Auto Refresh

### Activate Auto Refresh
1. Locate the **Auto Refresh** icon in the toolbar
2. Click to toggle auto refresh on or off
3. Select your preferred refresh interval
4. The view will now update automatically

### Refresh Intervals
Available refresh interval options:
- 30 seconds
- 1 minute
- 5 minutes
- 10 minutes
- 15 minutes
- 30 minutes
- Custom interval (if configured)

## Configuration

### Set Default Intervals
Configure your default auto-refresh settings:
1. Go to **User Preferences**
2. Navigate to **Display Settings**
3. Set your preferred default refresh interval
4. Save your preferences

### Module-Specific Settings
Different refresh intervals can be set for:
- Dashboard widgets
- List views
- Report displays
- Chart visualizations

## Managing Auto Refresh

### Pause Auto Refresh
- Click the pause icon to temporarily stop auto refresh
- Resume by clicking the refresh icon again
- Useful when reviewing or editing data

### Manual Refresh
- Click the **Refresh** button to update immediately
- Keyboard shortcut: F5 or Ctrl+R
- Bypasses the scheduled auto refresh timing

## Best Practices

- Set shorter intervals for critical monitoring views
- Use longer intervals for historical or static data
- Pause auto refresh when editing records
- Consider server load when setting very short intervals
- Disable auto refresh on slower connections

## Performance Considerations

- Shorter refresh intervals may impact browser performance
- Large datasets may take longer to refresh
- Network latency affects refresh speed
- Adjust intervals based on data update frequency
