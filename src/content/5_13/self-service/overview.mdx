# Self Service Overview

Self Service portal empowers end users to submit requests, access knowledge base articles, and resolve common issues without IT intervention.

## Key Features

- User-friendly service catalog
- Knowledge base access
- Request tracking
- Automated workflows
- Mobile accessibility

## Getting Started

Learn how to use the Self Service portal:
1. Submitting service requests
2. Searching the knowledge base
3. Tracking request status
4. Providing feedback

## Version Information

This documentation applies to Virima version **5.13**.
