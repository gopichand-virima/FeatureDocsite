# Probes

Use this function to configure probes for discovering different types of devices and systems.

In the main window, click **Admin > Discovery > Probes**. The Probes window displays.

From the *Select Actions* drop-down list, choose an action: **New Probe**, [Delete](/common_topics/delete_remove_5_13.mdx), or [Export](/discovery/import_data_files/export_5_13.mdx).

## Probe Types

The following probe types are available:

- **SNMP Probe** - For network devices
- **SSH Probe** - For Unix/Linux systems
- **Windows Probe** - For Windows systems
- **VMware Probe** - For VMware infrastructure
- **Cloud Probes** - For AWS, Azure, etc.
