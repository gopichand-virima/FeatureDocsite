# Images and Vulnerabilities

<span style="color: #dc143c;"><span style="color: #000000;">Use this function to</span> <span style="color: #000000;">retrieve all the docker images inside AWS ECR ( Elastic Container Registry) repositories. These images can contain vulnerabilities, which can be scanned and shown on the Vulnerabilities tab.</span></span>

<div class="NumList2" madcap:autonum="1. ">

In the main window, select **ITSM /> Confirmation Management /> CMDB**.

</div>

<div class="NumList2" madcap:autonum="2. ">

Search for and select the applicable asset record.

</div>

<div class="NumList2" madcap:autonum="3. ">

When the asset record window displays, click the **Repositories** tab.

</div>

<div class="NumList2" madcap:autonum="4. ">

Select a repository in the list.

</div>

<div class="NumList2" madcap:autonum="5. ">

Click **Images**.

</div>

<div class="Ind50">

<span class="image placeholder" original-image-src="../../../../Resources/Images/Components/repositories-main-old.PNG" original-image-title=""></span>

</div>

<div class="NumList2" madcap:autonum="6. ">

Chick in the checkbox to select a repository in the list.

</div>

|     |                                                                                                                                 |
|-----|---------------------------------------------------------------------------------------------------------------------------------|
|     | If you click on a line item, a new window displays and provides details for the selected image, along with any Vulnerabilities. |

<div class="NumList2" madcap:autonum="7. ">

Click **Scan**. A confirmation displays if the scan was started successfully.

</div>

<div class="NumList2" madcap:conditions="Default.Omit" madcap:autonum="8. ">

While viewing the list of images, from the drop-down list, and select any of the following: 

</div>

<div class="Bullet2" madcap:conditions="Default.Omit">

AWS Credentials

</div>

<div class="Bullet2" madcap:conditions="Default.Omit">

AWS New Account

</div>

<div class="Bullet2" madcap:conditions="Default.Omit">

New AWS Subscription

</div>
