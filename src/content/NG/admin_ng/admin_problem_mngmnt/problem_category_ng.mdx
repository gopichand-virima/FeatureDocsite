# Problem Category

Use this function to create and manage problem categories that help classify and organize problems.

In the main window, click **Admin > Problem Management > Problem Category**. The Problem Category window displays.

## New Problem Category

From the *Select Actions* drop-down list, choose **New Problem Category**. The New Problem Category window displays.

Enter a *Name* and *Description* for this category.

Select a **Parent Category** if this is a sub-category (optional).

To add an image/icon for this category:

- Click **Upload**
- Search for and select an image
- Click **OK**

Configure category properties:

**Requires RCA**: Check if problems in this category require formal Root Cause Analysis.

**Default Assignment Group**: Select a group that typically handles this category.

**Estimated Resolution Time**: Set typical resolution timeframe for planning purposes.

When all selections/entries are made, click **Add**.

## Edit Problem Category

Navigate to the window containing the record to edit.

Click the line containing the record. The applicable window displays.

Make the necessary modifications.

Click **Save**.

## Delete Problem Category

Deleting is a permanent action and cannot be undone. Ensure no problems are currently using this category before deletion.

Click the line that contains the item to delete.

From the *Select Actions* drop-down list, choose **Delete**. If a confirmation message is displayed, take the appropriate action.
