# Cherwell Credential

Use this function to specify credential details for Cherwell.

 -- ----- -----------------------------------------------------------------------------
 1. In the main window, click **Admin \> Integrations \> Cherwell Credential**.
 -- ----- -----------------------------------------------------------------------------

::: {.Ind50
![cherwell credential new old](../Resources/Images/Admin - Integration/cherwell-credential-new-old.PNG)

 -- ----- -------------------------------------------
 2. Enter the *Username* for this credential.
 -- ----- -------------------------------------------

 -- ----- -----------------------------------------------------
 3. Click **Change Password** and enter a new password.
 -- ----- -----------------------------------------------------

::: Bullet2
To view the password, click the **eye** icon.

::: Bullet2
To cancel the password change operation, click the **x** icon.

 -- ----- ---------------------------------
 4. Enter the *URL* for the domain.
 -- ----- ---------------------------------

 -- ----- --------------------------------------------
 5. Enter the *API Key* to use for the domain.
 -- ----- --------------------------------------------

 -- ----- -----------------------------------------------------------------------------------------------------------------------
 6. [[Select to enable the ability to integrate with the application.]{style="color: #000000;"}]{style="color: #dc143c;"}
 -- ----- -----------------------------------------------------------------------------------------------------------------------

 -- ----- -------------------------------------------------------
 7. When all selections/entries are made, click **Save**.
 -- ----- -------------------------------------------------------
