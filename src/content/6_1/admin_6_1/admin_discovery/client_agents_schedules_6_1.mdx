# Agent Schedules

Use this function to trigger a scheduled scan on the remote machine where an agent is installed.

 -- ----- ------------------------------------------------------
 1. While viewing the client window, select a line item.
 -- ----- ------------------------------------------------------

 -- ----- ------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------
 2. In the Details window, click **Agent Schedules**. ![Agent Schedules 6.1](../Resources/Images/Admin - Discovery/Agent-Schedules-6.1.png)
 -- ----- ------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------

 -- ----- ---------------------------------------------------------------------------------------------------------
 3. To add an expression, click **Add Cron Expression**. The Add Cron for Agents Schedules window displays.
 -- ----- ---------------------------------------------------------------------------------------------------------

::: Ind50
![add cron agent schedules old](../Resources/Images/Admin - Discovery/add-cron-agent-schedules-old.PNG)

 -- ----- ----------------------------------------------------------------------------------
 a. Select a pre-defined *Scan Frequency* (every hour, every 2 hours, and so forth).
 -- ----- ----------------------------------------------------------------------------------

 -- ----- --------------------------------------------
 b. Specify if this schedule **Is Recurring**.
 -- ----- --------------------------------------------

 -- ----- -----------------------------------------------------------------------
 c. Enter the time components such as second, minute, hour, and so forth.
 -- ----- -----------------------------------------------------------------------

 -- ----- -----------------------------------------------------------
 d. When all selections/entries are made, click **Add Cron**.
 -- ----- -----------------------------------------------------------

 -- ----- -----------------------------------------------------------------------------------
 4. To remove an item, select an existing line, and click **Delete Cron Expression**.
 -- ----- -----------------------------------------------------------------------------------
