# KPIs

A Key Performance Indicator (KPI) is a measurable value that demonstrates the defined objectives effectively. It provides a reporting dashboard reflecting insights for the current performance.

Use this function to view the currently defined KPIs.

- In the main window, click **Admin > Others > KPIs**. The KPIs window displays.

!

- From the *Select Actions* drop-down list, choose an action: [Delete](/common_topics/delete_remove_6_1.mdx) or [Export](/discovery/import_data_files/export_6_1.mdx).
- To edit an existing KPI, click the applicable line item.

!

- Click the **Save** button.
