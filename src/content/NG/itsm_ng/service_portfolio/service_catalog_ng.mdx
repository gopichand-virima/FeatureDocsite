# Service Catalog

Use this function to manage the service catalog.

The Service Catalog is a subset of the Service Portfolio that includes only those services that are available to customers. It contains information about all operational IT services.

In the navigation pane, select **ITSM > Service Portfolio > Service Catalog**. The Service Catalog window displays.

New Catalog Item

From the *Select Actions* drop-down list, choose **New Catalog Item**.

Click **New**.

Complete the fields.

When all selections/entries are made, click **Add**.
