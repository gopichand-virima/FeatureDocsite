# Surveys

NEEDÂ v6 IMAGES

Use this function to enable a Reporter to send a survey when the Incident/Request/Change/Problem is being closed. The surveys are usually conducted by a pre-defined questionnaire. The data in the Survey tab will be displayed only if the Incident/Request/Change/Problem is closed.

In the main window, click **Admin > Others > Surveys**. The Surveys window displays.

In the main window, click the **gear** icon and select **Others > Surveys**.

ICON

The Surveys window displays.

IMAGE

New Survey

From the *Select Actions* drop-down list, choose **New Survey**. The New Survey window displays.

Click **New**. The New Survey window displays.

IMAGE

Enter a *Name* and *Introduction* for the survey.

Enter any *End Notes*.

In the *Threshold* fields, specify the frequency at which the survey should be sent.

To add survey questions, click **Add Questions**.

In the *Question* field, type the question.

In the *Type* field, click the drop-down list and select the type of question, such as yes/no, multiple choice, and so forth.

Click **Add**.

To add existing questions to the survey, click **Add Existing Questions**. The Questions dialog box displays.

Enter the search criteria.

From the list of results, select an existing question(s) to add to this survey.

When all selections are made, click **Apply**.

When all selections/entries are made, click **Save**.

Edit Surveys

To view a list of all functions where this survey is specified:

Open an existing line item.

Click the **Used By** tab.
