# Monitoring Profile: Details

Use this function to add details for a monitoring profile.

 -- ----- ----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------
 1. [On the main window, select ]{.span_1}[Admin]{.Strong_2}[ \> ]{.span_1}[Discovery]{.Strong_2}[** \> Discovery Monitoring Profile**. The Discovery Monitoring Profile window displays.]{.span_1}
 -- ----- ----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------

 -- ----- ----------------------------------------------------------------------------------------------------------------------------------------------------
 2. From the *Select Actions* drop-down list, choose an action: **New**.\
 ![discovery mon prof details 6.1](../Resources/Images/Admin - Discovery/discovery-mon-prof-details-6.1.PNG)\

 -- ----- ----------------------------------------------------------------------------------------------------------------------------------------------------

 -- ----- -----------------------------------------------------
 3. In the *Name* field, enter a name for this profile.
 -- ----- -----------------------------------------------------

 -- ----- -------------------------------------------------------------------------
 4. In the *Description* field, enter information to identify this profile.
 -- ----- -------------------------------------------------------------------------

 -- ----- -----------------------------------------------------------------------------------
 5. In the *Client* field, click the drop-down list and select the applicable client.
 -- ----- -----------------------------------------------------------------------------------

::: Bullet2
If the applicable client is not shown, click the link to check the client\'s status.

 -- ----- -----------------------------------------------------------------------------------------------------------------------------------------------------------------------
 6. In the *IP Address* field, enter the IP Address for the client. The IP range can be a single IP, multiple IPs (separated by a space), CIDR Notation or IP with Range.
 -- ----- -----------------------------------------------------------------------------------------------------------------------------------------------------------------------

 -- ----- ------------------------------------------------------
 7. When all selections/entries are made, click **Add**.
 -- ----- ------------------------------------------------------

 -- ----- -------------------------------------------------------
 8. When all selections/entries are made, click **Save**.
 -- ----- -------------------------------------------------------
