# User-Specific Functions

Located in the upper-right corner of the window, a series of buttons provides access to the user-specific functions.

![inbox chat cart suggestions](Resources/Images/Common Topics/inbox-chat-cart-suggestions.PNG)

 Icon Description
 ------------- -------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------
 Inbox \[n\] Displays the number of unread Inbox messages. When clicked, the window updates and displays all messages\--both read and unread.
 Chat Opens a chat window through which communication can occur between users. The Chat server must be running for this option to be enabled.
 Cart \[n\] Displays the number of items in the cart. When clicked, the window updates and displays the list of items. Refer to [Self-Service:Â Service Catalog](Self%20Service/Service-Catalog.mdx) for more information on adding items to the cart.
 Suggestions Allows the user to submit suggestions or provide feedback.
