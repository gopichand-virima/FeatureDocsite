# CI: Details - DML (Digital Media Library)

Use this function to upload an executable (.exe) file for a software or add a link to the location where the software can be downloaded.

|     |                                                                                                   |
|-----|---------------------------------------------------------------------------------------------------|
|     | Configuring a download link is useful when software is required for installation on many clients. |

<div class="NumList1" madcap:conditions="Default.OldUI" madcap:autonum="1. ">

From the navigation pane, select **ITSM />** **Configuration Management /> Configuration Items**. The Configuration Items window displays.

</div>

<div class="NumList2" madcap:conditions="Default.NewUI" madcap:autonum="2. ">

From the navigation pane, select **Configuration Management /> Configuration Items**. The Configuration items window displays.

</div>

<div class="NumList2" madcap:autonum="3. ">

Select a record in the list. A new window opens and the **Details** tab displays.

</div>

<div class="NumList2" madcap:autonum="4. ">

Click the **DML** tab.

</div>

<div class="Ind50" madcap:conditions="Default.OldUI">

<span class="image placeholder" original-image-src="../../../Resources/Images/ITSM/dml-software.PNG" original-image-title="" style="width: 475px;height: 126px;" madcap:conditions="Default.OldUI"></span>

</div>

<div class="Ind50" madcap:conditions="Default.NewUI">

<span class="image placeholder" original-image-src="../../../Resources/Images/ITSM/dml-software-new.png" original-image-title="" madcap:conditions="Default.NewUI" style="width: 478px;height: 120px;"></span>

</div>

<div class="NumList2" madcap:autonum="5. ">

Perform any of the following actions.

</div>

## Add Software

<div class="NumList1" madcap:autonum="1. ">

Click **Add Software**. The Attach Software File dialog box displays.

</div>

<div class="Ind50" madcap:conditions="Default.OldUI">

<span class="image placeholder" original-image-src="../../../Resources/Images/ITSM/attach-software-file.PNG" original-image-title="" style="width: 381px;height: 185px;" madcap:conditions="Default.OldUI"></span>

</div>

<div class="Ind50" madcap:conditions="Default.NewUI">

[<span class="image placeholder" original-image-src="../../../Resources/Images/ITSM/attach-software-file-new.png" original-image-title="" madcap:conditions="Default.NewUI" style="width: 380px;height: 212px;"></span>](../../../Resources/Images/ITSM/attach-software-file-new.png)

</div>

<div class="NumList2" madcap:autonum="2. ">

In the *Upload Document* field, click the **Browse** button and select the applicable file.

</div>

<div class="NumList2" madcap:autonum="3. ">

In the *Description* field, enter details to describe the software.

</div>

<div class="NumList2" madcap:autonum="4. ">

Click **Upload**. If the upload is successful, the software is shown on the DML tab. By default, an ID is assigned.

</div>

## Add Link

Only one link at a time can be configured.

|     |                                                                                                                                                                                                |
|-----|------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
|     | A version of the software must be configured before the link can be saved. Any changes to the version can be made under the Software License Key Details on the **Details** tab for the asset. |

<div class="NumList1" madcap:autonum="1. ">

Click **Add Link**. The Add Link dialog box displays.

</div>

<div class="Ind50" madcap:conditions="Default.OldUI">

<span class="image placeholder" original-image-src="../../../Resources/Images/ITSM/link-add.PNG" original-image-title="" style="width: 375px;height: 190px;"></span>

</div>

<div class="Ind50" madcap:conditions="Default.NewUI">

<span class="image placeholder" original-image-src="../../../Resources/Images/ITSM/link-add-new.png" original-image-title="" style="width: 378px;height: 252px;"></span>

</div>

<div class="NumList2" madcap:autonum="2. ">

In the *Name* field, type the name of the software.

</div>

<div class="NumList2" madcap:autonum="3. ">

In the *Link* field, enter the path where the software is located and from where it can be downloaded.

</div>

<div class="NumList2" madcap:autonum="4. ">

In the *Description* field, enter details to describe the software.

</div>

<div class="NumList2" madcap:autonum="5. ">

Click **Add**. The software is shown on the DML tab. By default, an ID is assigned.

</div>

## Downloading the Software

To download the software, locate it in the list, and click the **Download** icon.

## Deleting the Software

|     |                                                                                                                                                       |
|-----|-------------------------------------------------------------------------------------------------------------------------------------------------------|
|     | Once you have updated a version of a specific software, the older version cannot be deleted.<span style="color: #dc143c; font-weight: bold;"> </span> |

To delete a software, locate it in the list, and click the **Delete** icon.
