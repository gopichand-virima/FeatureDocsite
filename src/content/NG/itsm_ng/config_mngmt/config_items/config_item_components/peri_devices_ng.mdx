# Peripheral Devices

Click **Peripheral Devices**.

! ! Add Peripheral Device

Click **Add**. The Peripheral Device dialog box displays.

! !

Click **New Peripheral Device.**

Enter the applicable field information.

When all selections/entries are made, click **Save**.

Edit Peripheral Device

Click **Existing Peripheral Device**.

! !

Enter the search criteria to locate the existing device.  *See also* [Advanced Search](/common_topics/advanced_search_ng.mdx)*.*

Select the applicable device from the results list.

When all selections are made, click **Save**.

View and Edit Peripheral Device Details

While viewing the list of existing devices, select the device to edit. The windows refreshes and the details for the device displays. These details are view only.

To view the supporting CIs, click the CI item. The Configuration Items window displays.

Click **Components**.

To add a contact, navigate to the Contacts section and click **Add**. The Contacts dialog box displays.

! !

In the *Choose Relation* field, click the drop-down list and select how this contact should be related to this asset.

In the search field, enter the search criteria.

Select a contact from the results list, moving it to the selected list.

When all selections are made, click **Save**.
