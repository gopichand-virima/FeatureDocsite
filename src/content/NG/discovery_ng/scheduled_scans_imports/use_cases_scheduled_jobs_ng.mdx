# Use Cases

# **Nightly Discovery Scans**

Schedule a **Deep Host Scan** every night at 2:00 AM to discover and update all devices in the corporate IP range.

# **Weekly Cloud Imports**

Schedule AWS and Azure imports every **Sunday at midnight** to sync cloud resources into the CMDB.

# **Daily Meraki Device Refresh**

Schedule a Meraki import every **24 hours** to keep network device inventory current.

# **Quarterly Compliance Audits**

Schedule scans or imports to run **every 3 months** to support audit and compliance reporting.
