# Dashboard

Use this function to view user information for the raised Incidents. In this window the user can view a complete report and export an incident.

<div class="NumList1" madcap:autonum="1. ">

Login to MSP.

</div>

<div class="NumList2" madcap:autonum="2. ">

On the Home page, click **Dashboard**. The Total Number of Incidents window displays.

</div>

<div class="Ind50">

<span class="image placeholder" original-image-src="../Resources/Images/MSP/total-number-incidents.png" original-image-title=""></span>

</div>

<div class="NumList2" madcap:autonum="3. ">

In this window, perform any of the following actions:

</div>

<div class="Bullet2">

To view the incident reports, select **Click for complete report**.

</div>

<div class="Bullet2">

To export the web information in Excel sheet format for the configured user Email ID, click the **Export** icon.

</div>

<div class="Bullet2">

To delete an incident record, highlight the record and click the **Delete** icon.

</div>

<div class="Bullet2">

Use the minus (-) and plus (+) signs to **Minimize** or **Maximize** the window.

</div>
