# Testing AD Configuration

1. Open the AD Configuration line item.

2. From the **Select Actions** drop-down list, Select **Test AD Configuration.**


1. !


1. If you have set the values of the attributes correctly as per your organization's AD structure, then the connection will be a success. Otherwise, it will fail.


1. !
