# Self Service - Compatibility Matrix

System requirements and compatibility for Self Service portal.

## Browser Compatibility

| Browser | Minimum Version | Recommended |
|---------|----------------|-------------|
| Chrome | 90+ | Latest |
| Firefox | 88+ | Latest |
| Safari | 14+ | Latest |
| Edge | 90+ | Latest |

## Mobile Support

### Mobile Browsers
- iOS Safari 13+
- Chrome Mobile (Android)
- Samsung Internet

### Mobile App
- iOS 13+ (native app)
- Android 9+ (native app)
- Progressive Web App (PWA)

## User Access

| User Type | Access Level |
|-----------|-------------|
| End Users | Submit requests, view own requests |
| Service Owners | Manage service catalog items |
| Approvers | Review and approve requests |
| Portal Administrators | Full configuration access |

## Integration Capabilities

The Self Service portal integrates with:
- ITSM (request fulfillment)
- CMDB (asset selection)
- Active Directory/LDAP (authentication)
- SSO providers (SAML, OAuth)
- Microsoft Teams (notifications)
- Slack (notifications)

## Accessibility

Compliance with:
- WCAG 2.1 Level AA
- Section 508
- ADA accessibility standards

## Supported Languages

- English
- Spanish
- French
- German
- Japanese
- Chinese (Simplified)
