# Ignore Process

Use this function to ignore back-end processes when creating relationships.

On the main window, select **Admin > Discovery > Ignore Process**. The Ignore Process Names dialog box displays.

Enter the name(s) of the process to ignore. Separate multiple process names with a comma.

Click **Add**.

In the confirmation window, click **OK**.
