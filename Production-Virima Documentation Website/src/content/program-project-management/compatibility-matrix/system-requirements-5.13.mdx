# Program and Project Management - Compatibility Matrix

System requirements and compatibility for Program and Project Management module.

## Browser Compatibility

| Browser | Minimum Version | Recommended |
|---------|----------------|-------------|
| Chrome | 90+ | Latest |
| Firefox | 88+ | Latest |
| Safari | 14+ | Latest |
| Edge | 90+ | Latest |

## User Permissions

| Role | Permissions |
|------|------------|
| Project Viewer | View project details |
| Team Member | Update assigned tasks |
| Project Manager | Manage project, tasks, resources |
| Portfolio Manager | View and manage all projects |
| PMO Administrator | Full module configuration |

## Integration Capabilities

### Third-Party Tools
- Microsoft Project (import/export)
- Jira (sync tasks and issues)
- Smartsheet (data exchange)
- Asana (integration)
- Monday.com (integration)

### Virima Modules
- ITSM (change management integration)
- ITAM (resource allocation)
- CMDB (infrastructure dependencies)
- Reports (project analytics)

## Import/Export Formats

Supported formats:
- Microsoft Project (MPP, XML)
- Excel (XLSX)
- CSV
- JSON
- PDF (reports and Gantt charts)

## Mobile Support

- Responsive web interface
- iOS app (task updates)
- Android app (task updates)
- Offline task management

## Performance Requirements

For optimal performance with large projects (500+ tasks):
- 16GB RAM recommended
- Modern processor
- Stable internet connection
- Latest browser version
