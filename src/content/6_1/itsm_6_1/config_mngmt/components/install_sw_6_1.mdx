# Installed Software

Use this function to add and manage the software installed on a host.

Components are categorized based on publishers. When the Discovery application detects more than one application from the same publisher on a host, they are grouped under Installed Software. A single instance of a publisher is grouped as **Uncategorized**.

<span class="image placeholder" original-image-src="../../../Resources/Images/ITSM/grouped-installed-software.PNG" original-image-title=""></span>

## Add Installed Software

<div class="NumList1" madcap:autonum="1. ">

In the selected record configuration window, click **Components**.

</div>

<div class="NumList2" madcap:autonum="2. ">

In the secondary group of tabs, click **Installed Software**.

</div>

<div class="Ind50" madcap:conditions="Default.OldUI">

<span class="image placeholder" original-image-src="../../../Resources/Images/ITSM/installed-software.PNG" original-image-title="" madcap:conditions="Default.OldUI" style="width: 477px;height: 84px;"></span>

</div>

<div class="Ind50" madcap:conditions="Default.NewUI">

[<span class="image placeholder" original-image-src="../../../Resources/Images/ITSM/installed-software-new.png" original-image-title="" madcap:conditions="Default.NewUI" style="width: 476px;height: 106px;"></span>](../../../Resources/Images/ITSM/installed-software-new.png)

</div>

<div class="NumList2" madcap:autonum="3. ">

Click **Add**. The Installed Software dialog box displays.

</div>

<div class="Ind50" madcap:conditions="Default.OldUI">

<span class="image placeholder" original-image-src="../../../Resources/Images/ITSM/installed-software-add.PNG" original-image-title="" madcap:conditions="Default.OldUI" style="width: 373px;height: 154px;"></span>

</div>

<div class="Ind50">

<span class="image placeholder" original-image-src="../../../Resources/Images/ITSM/installed-software-add-new.png" original-image-title="" madcap:conditions="Default.NewUI" style="width: 376px;height: 169px;"></span>

</div>

<div class="NumList2" style="font-weight: normal;" madcap:autonum="4. ">

Do one of the following:

</div>

<div class="Bullet2">

Click **New Installed Software** and complete all the fields. All fields are required.

</div>

<div class="Bullet2">

Click **Existing Installed Software** and make any applicable changes.

</div>

<div class="NumList2" style="font-weight: normal;" madcap:autonum="5. ">

When all selections/entries are made, click **Add**. A unique ID is assigned.

</div>

## Edit Installed Software

<div class="NumList1" madcap:autonum="1. ">

In the selected record configuration window, click **Components**.

</div>

<div class="NumList2" madcap:autonum="2. ">

In the secondary group of tabs, click **Installed Software**.

</div>

<div class="NumList2" madcap:autonum="3. ">

From the list of records, select the applicable software to edit.

</div>

<div class="NumList2" madcap:autonum="4. ">

Make the applicable edits on the appropriate tab.

</div>

<div class="NumList2" madcap:autonum="5. ">

To add a Software License Key, click the **Add** button.

</div>
