# RunBook

Use this function to create a RunBook, which is a documented procedure that is carried out regularly to maintain performance within monitored IT environment. RunBook automation is a system whereby this procedure is able to be completed without manual input, therefore increasing efficiency and reliability.

In the navigation pane, select **ITSM > RunBook**. The RunBook window displays.

New RunBook


A RunBook must be added before tasks can be created.


From the *Select Actions* drop-down list, choose **New RunBook**. The New RunBook window displays.

Enter a *RunBook Name* and *Description*.

Click **Add**.

### Add a RunBook Task

In the main window, select the RunBook to which a task is to be added.

In the RunBook window, click **New Task**. The Add Task dialog box displays. Note: The dialog box changes based on whether the "*User Defined Command*" or "*Select Probe*" option is selected.

Complete the fields, referring to the table below.

When all selections/entries are made, click **Add**.

#### Add Task Fields

| Field | Description |

| Name | Enter the name for this task. |

| Order | Enter a number for the order in which the command will be executed. |

| Command Type | Click the drop-down list and select the applicable command (such as Windows or Unix) to be executed. |

| Timeout (sec) | Enter the time interval in seconds to execute the command. |

| User Defined Command | Select if a command will be entered in the Command box. |

| Command | Enter a command. This field is only displayed if "User Defined Command" is selected. |

| Select Probe | Select if a probe will be specified. |

| Select Probe Type | From the drop-down list, select the required probe. This field is only displayed if "Select Probe" is selected. |

| Cascade on Error

Cascade on Success | If *Cascade on Error* is **Yes**, and *Cascade on Success* is **Yes**, the command is executed.

If *Cascade on Error* is **No** and *Cascade on Success* is **Yes**, the command execution will be stopped. |

| Wait Time (sec) | Enter the time in seconds to wait before the next command is executed. |

| Host | Click **Add** then search for select the applicable host.

Click the **plus + sign**, then search for select the applicable host. |

| Client | From the drop-down list, select the applicable Client. The update to the Runbook occurs. |
