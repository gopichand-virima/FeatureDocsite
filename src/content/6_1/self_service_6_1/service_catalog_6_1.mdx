# Service Catalog

Use this function to create a "catalog" that includes information as it relates to services and deliverables. These items are classified based on the category.

There are three items of catalog items:

<div class="Bullet1">

Product

</div>

<div class="Bullet1">

Request

</div>

<div class="Bullet1">

Issue

</div>

Depending on the catalog item you select, the workflow is somewhat different. *Refer to each of the sections below for details.*

Product

<div class="NumList1" madcap:autonum="1. ">

In the navigation pane, select **Self Service /> Service Catalog**. The Service Catalog window displays.

</div>

<div class="Ind50" madcap:conditions="Default.OldUI">

<span class="image placeholder" original-image-src="../Resources/Images/Self-Service/service-catalog-main-old.PNG" original-image-title=""></span>

</div>

<div class="NumList2" madcap:autonum="2. ">

Click the **Category** containing the applicable item. The new window displays with all the available catalog items in that category.

</div>

<div class="NumList2" madcap:autonum="3. ">

Click a catalog item. The resulting window displays the details for the selected catalog item. Depending on the type of item, one of following actions can be performed:

</div>

<div class="Bullet2">

Submitting a request

</div>

<div class="Bullet2">

Ordering an item

</div>

<div class="Bullet2">

Reporting an issue

</div>

Request

<div class="NumList1" madcap:autonum="1. ">

With the catalog item selected, under Order this item, click **Request**.

</div>

<div class="Ind50" madcap:conditions="Default.OldUI">

<span class="image placeholder" original-image-src="../Resources/Images/Self-Service/request-order-request-old.PNG" original-image-title=""></span>

</div>

<div class="Ind50" madcap:conditions="Default.NewUI">

<span class="image placeholder" original-image-src="../Resources/Images/Self-Service/request-order-request-new.PNG" original-image-title=""></span>

</div>

<div class="Ind50">

The Request Details dialog box displays.

</div>

<div class="NumList2" madcap:autonum="2. ">

Complete the dialog box as per the instructions below.

</div>

### Request Details Dialog Box

Order

<div class="NumList1" madcap:autonum="1. ">

Select an item to order. A new window displays with the details for that item.

</div>

<div class="NumList2" madcap:autonum="2. ">

Move to the *Order this item* area.

</div>

<div class="NumList2" madcap:autonum="3. ">

Do either of the following: click **Order Now** or **Add to Cart** (see below).

</div>

### Order Now

If this is the only item to order, click **Order Now**. The Request Details dialog box displays. *Refer to the Request Details Dialog Box instructions below.*

### Add to Cart

<div class="NumList1" style="font-weight: normal;" madcap:autonum="1. ">

If there are additional items to add, click **Add to Cart.** A Confirmation dialog box displays.

</div>

<div class="NumList2" style="font-weight: normal;" madcap:autonum="2. ">

Select one of the following:

</div>

<div class="Bullet2">

Proceed to Checkout

</div>

<div class="Bullet2">

Continue Shopping

</div>

<div class="NumList2" madcap:autonum="3. ">

If *Proceed to Checkout* is selected, the item is added to the cart. Click **Order Now** to place the order.

</div>

<div class="NumList2" madcap:autonum="4. ">

If *Continue Shopping* is selected, the window returns to the catalog item selected. Search for and repeat the process for any additional catalog items.

</div>

Issue

<div class="NumList1" madcap:autonum="1. ">

From the catalog window, select an item.

</div>

<div class="NumList2" madcap:autonum="2. ">

From the list of items, click the item related to the issue. The issue window shows a list of *Similar Incidents*. This is the avoid the possible duplication of an issue.

</div>

<div class="Ind50" madcap:conditions="Default.OldUI">

<span class="image placeholder" original-image-src="../Resources/Images/Self-Service/issue-old.PNG" original-image-title=""></span>

</div>

<div class="NumList2" madcap:autonum="3. ">

If the issue doesn't seem to exist, click **Report this issue**. The Issue Details dialog box displays.

</div>

<div class="Ind50" madcap:conditions="Default.OldUI">

<span class="image placeholder" original-image-src="../Resources/Images/Self-Service/issue-details-old.PNG" original-image-title=""></span>

</div>

<div class="NumList2" madcap:autonum="4. ">

Enter a *Description*.

</div>

<div class="NumList2" madcap:autonum="5. ">

To include an *Attachment*, click **Upload**, then **Browse** for and select the applicable document.  

</div>

<div class="NumList2" madcap:autonum="6. ">

If there are additional attachments, click the **plus + sign**, and repeat the above steps.

</div>

<div class="NumList2" madcap:autonum="7. ">

Click **Submit Issue**. A Success Message window displays.

</div>

<div class="Ind50" madcap:conditions="Default.OldUI">

<span class="image placeholder" original-image-src="../Resources/Images/Self-Service/issue-success-message-old.PNG" original-image-title=""></span>

</div>

<div class="NumList2" madcap:autonum="8. ">

Click **OK**.

</div>
