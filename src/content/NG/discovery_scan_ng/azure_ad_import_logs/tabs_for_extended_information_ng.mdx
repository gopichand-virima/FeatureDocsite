# Tabs for Extended Information

- [Tasks]{.Strong}: Displays or allows creation of tasks linked to the import job.

- [Comments]{.Strong}: Provides space for administrators to add notes or review remarks from others.

- [Attachments]{.Strong}: Lets users upload or view related files, such as screenshots, reports, or documentation tied to the import.

Clicking on the \"[New User Count]{.Strong}\" will navigate you to the User Record page, where the Azure AD User Import Logs are automatically filtered and the relevant results are displayed.

![Azure AD User Import Logs 14](../../Resources/Images/Discovery/Azure-AD-Users-Import-Logs/Azure AD User Import Logs_14.png)

[Key Information displayed]{.Strong}:

- [Id]{.Strong}: The unique identifier assigned to each user record.

- [Email]{.Strong}: The email address associated with the user.

- [User Name]{.Strong}: The Azure AD user's principal name or login name.

- [Display Name]{.Strong}: The display name of the user as shown in Azure AD.

- [Phone]{.Strong}: The phone number associated with the user (empty in the current data).

- [Roles]{.Strong}: The assigned role(s) for the user (e.g., End User).
