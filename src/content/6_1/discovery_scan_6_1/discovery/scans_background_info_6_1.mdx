# <span style="background-color: #ffffff;">Background Information on Scans</span>

There are three components related to scanning:

<div class="Bullet1">

<span class="span_2" style="font-weight: bold;">Probes </span> <span class="span_1">executes a set of scripts on the target hosts to retrieve information. Each host type will have a different probe.</span>

</div>

<div class="Bullet1">

<span class="span_2"> **Probe Workflows** executes a</span> <span class="span_1"> script when a scan is created. The workflow calls the set of the probes required to discover the information about the hosts on which the scan is performed.</span>

</div>

<div class="Bullet1">

<span class="span_2" style="font-weight: bold;">Sensors. </span> <span class="span_1">Scripts that are applied to the output of the respective probes. The sensors give the data </span> <span class="span_2">JSON</span> <span class="span_1"> output.</span>

</div>

The basic work flow is as follows:

<div class="Bullet1">

Find up and running Hosts.

</div>

<div class="Bullet1">

Get Host Types for each host.

</div>

<div class="Bullet2">

<span class="span_1">If Host Type is </span> <span class="span_2"> **Unix**, find the Unix OS flavor and, depending on the OS, send the probes to the Discovery Server, which retrieves the information of the host.</span>

</div>

<div class="Bullet2">

<span class="span_1">If Host Type is </span> <span class="span_2" style="font-weight: bold;">Windows</span> <span class="span_1"> or </span> <span class="span_2"> **Network device**, send probes to the the Discovery Server that retrieves the information of the host.</span>

</div>

<span class="span_1">For discovering Windows machines in a workgroup, the admin shared folder should be remotely accessible. Enable this folder by following these steps:</span>

<div class="NumList1" madcap:autonum="1. ">

<span class="span_1">Run </span> <span class="span_6"> **regedit**.</span>

</div>

<div class="NumList2" madcap:autonum="2. ">

<span class="span_1">Navigate to this directory</span> <span class="span_2">:  
</span> <span class="span_6" style="font-family: 'Courier New'; font-size: 10pt;">HKEY_LOCAL_MACHINE/SOFTWARE/Microsoft/Windows/CurrentVersion/Policies/System</span>

</div>

<div class="NumList2" madcap:autonum="3. ">

<span class="span_1">Right-click on the right pane and choose </span> <span class="span_2" style="font-weight: bold;">New</span> <span class="span_1" style="font-weight: bold;"> /> </span> <span class="span_2" style="font-weight: bold;">DWORD</span> <span class="span_1"> ** (32-bit) Value**.</span>

</div>

<div class="NumList2" madcap:autonum="4. ">

<span class="span_1">Fill </span> <span class="span_6" style="font-weight: bold;">LocalAccountTokenFilterPolicy</span> <span class="span_1"> for the Value name and press **Enter**.</span>

</div>

<div class="NumList2" madcap:autonum="5. ">

<span class="span_1">Double-click </span> <span class="span_6" style="font-weight: bold;">LocalAccountTokenFilterPolicy</span> <span class="span_1"> value and modify the “Value Data” from “0” to “1".</span>

</div>

<div class="NumList2" madcap:autonum="6. ">

<span class="span_1">Click </span> <span class="span_2" style="font-weight: bold;">OK</span> <span class="span_1">.</span>

</div>

<span class="span_1">To verify whether the admin share is accessible, run the following on the command line:   
</span>

<span class="span_6"> <span style="font-family: 'Courier New';"> <span style="font-size: 10pt;">net use </span> </span> </span> <span class="span_7"> <span style="font-family: 'Courier New';"> <span style="font-size: 10pt;">/remoteIP/admin/$</span> </span> </span> <span class="span_6"> <span style="font-family: 'Courier New';"> <span style="font-size: 10pt;"> /user:/<username/> </span> </span> </span> <span class="span_4"> <span style="font-family: 'Courier New';"> <span style="font-size: 10pt;">/*</span> </span> </span> <span class="span_1"> <span style="font-family: 'Courier New';"> <span style="font-size: 10pt;"> </span> </span>
