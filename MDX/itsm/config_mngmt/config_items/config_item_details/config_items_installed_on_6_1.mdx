# CI: Installed On

Use this function to view what software is installed on what system.

- In the navigation pane, select **ITSM > Configuration Management > CMDB**. The Configuration Items window displays.
- Select a record in the list. A new window opens and the **Details** tab displays.
- Click the **Installed On**Â tab. A list of all settings on which the software is installed displays.
- To view the details for an asset, click on the asset name. You are returned to the Configuration Items window.
- To view the location for a selected asset, click **View Map**.


### Edit Installed On

Not all functions include an option to edit record line items shown in the main window.


- Navigate to the window containing the record to edit.
- Click the line containing the record. The applicable window or dialog box displays.
- Make the necessary modifications.
- Click **Save** or **Add**, as applicable.

When editing an existing item, other options may be available, such as adding [Tasks](/itsm/release_mngmt/tasks_6_1.mdx), [Comments](/discovery/comments_6_1.mdx), and [Attachments](/discovery/attachments_6_1.mdx) and viewing [History](/common_topics/history_6_1.mdx).


The options change based upon the function; therefore, not every option may be shown.


### Delete Installed On

To delete an existing record, select the record in the list. Then, from the *Select Actions* drop-down list, choose **Delete**.
