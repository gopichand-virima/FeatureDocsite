# Catalog Items

Use this function to create a structured document for IT live Services reflecting items that are ready for deployment. It is the only part that is published to end-users, and includes support for sales and delivery, as well information on deliverables, processes, and available quantities.

In the navigation pane, select **ITSM \> Service Portfolio \> Service Catalog \> Catalog Items**.

![catalog items main old](../../Resources/Images/ITSM/catalog-items-main-old.PNG)

![arrow border](../../Resources/Images/arrow-border.PNG)

 1. From the *Select Actions* drop-down list, choose **New Catalog Items**. The New Catalog Item window displays.

 1. In the main window, from the *Select Actions* drop-down list, choose **Generate Billing Report**. The Generate Billing Report dialog box displays.
 -- ----- ----------------------------------------------------------------------------------------------------------------------------------------------------

[[[Closed]{.image .placeholder .MCDropDown_Image_Icon original-image-src="../../../Skins/Default/Stylesheets/Images/transparent.gif" original-image-title="" height="12" width="16" Image](javascript:void(0)){.MCDropDownHotSpot .dropDownHotspot .MCDropDownHotSpot_ .MCHotSpotImage aria-expanded="false"}]{.MCDropDownHead .dropDownHead}

This is conditioned out as you can add an image in the new window. Want to save the information, however.

Once a catalog item has been entered, it can be edited and an image uploaded. This image then displays in the Catalog ([ITSM \> Service Portfolio \> Service Catalog](#)).

 -- ----- ---------------------------------------------------------------------
 1. Open the line containing the item to which an image is to be added.
 -- ----- ---------------------------------------------------------------------

 -- ----- -----------------------------------------------------------------------------------------------
 2. Move to the upper right corner and click the three bars. The Primary Details window displays.
 -- ----- -----------------------------------------------------------------------------------------------

 -- ----- -------------------------------------------------------------------
 3. Click **Update Picture**. The Update Picture dialog box displays.
 -- ----- -------------------------------------------------------------------

::: Ind50
![update picture new](../../Resources/Images/Self-Service/update-picture-new.PNG)

 -- ----- --------------------------------------
 4. Select one of the following options:
 -- ----- --------------------------------------

::: Bullet2
*Blueprint Images* and *Service Catalog Images* display a list of all available images. Click on an image to select.

::: Bullet2
*Upload New Picture*, which allows you to **Browse** for and select an image to upload.

 -- ----- ---------------------------------------------------------------------------------------------------------------------------------------------------------------------------------
 5. When the applicable selection is made, click **Update**. The image is now associated with this catalog item, and is available for future selection under Service Catalog Items.
 -- ----- ---------------------------------------------------------------------------------------------------------------------------------------------------------------------------------

[[[Closed]{.image .placeholder .MCDropDown_Image_Icon original-image-src="../../../Skins/Default/Stylesheets/Images/transparent.gif" original-image-title="" height="12" width="16" Service Catalog Item](javascript:void(0)){.MCDropDownHotSpot .dropDownHotspot .MCDropDownHotSpot_ .MCHotSpotImage aria-expanded="false"}]{.MCDropDownHead .dropDownHead}

 --- -----------------------------------------------------------------------------------------
   Not all functions include an option to edit record line items shown in the main window.
 --- -----------------------------------------------------------------------------------------

 -- ----- -------------------------------------------------------
 1. Navigate to the window containing the record to edit.
 -- ----- -------------------------------------------------------

 -- ----- -------------------------------------------------------------------------------------
 2. Click the line containing the record. The applicable window or dialog box displays.
 -- ----- -------------------------------------------------------------------------------------

 -- ----- -----------------------------------
 3. Make the necessary modifications.
 -- ----- -----------------------------------

 -- ----- -------------------------------------------
 4. Click **Save** or **Add**, as applicable.
 -- ----- -------------------------------------------

When editing an existing item, other options may be available, such as adding [Tasks](/itsm/release_mngmt/tasks_6_1.mdx){.MCTopicPopup .MCTopicPopupHotSpot .a}, [Comments](/discovery/comments_6_1.mdx){.MCTopicPopup .MCTopicPopupHotSpot .a}, and [Attachments](/discovery/attachments_6_1.mdx){.MCTopicPopup .MCTopicPopupHotSpot .a} and viewing [History](/common_topics/history_6_1.mdx){.MCTopicPopup .MCTopicPopupHotSpot .a}.

 --- ---------------------------------------------------------------------------------------
   The options change based upon the function; therefore, not every option may be shown.
 --- ---------------------------------------------------------------------------------------

[[[Closed]{.image .placeholder .MCDropDown_Image_Icon original-image-src="../../../Skins/Default/Stylesheets/Images/transparent.gif" original-image-title="" height="12" width="16" Service Catalog Item](javascript:void(0)){.MCDropDownHotSpot .dropDownHotspot .MCDropDownHotSpot_ .MCHotSpotImage aria-expanded="false"}]{.MCDropDownHead .dropDownHead}

 --- --------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------
   Deleting is a permanent action and cannot be undone. Deleting may affect other functionality and information in the application such as data in configured reports, fields in windows, selectable options, etc. Therefore, be sure to understand the potential effects before making a deletion.
 --- --------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------

 -- ----- --------------------------------------------------
 1. Click the line that contains the item to delete.
 -- ----- --------------------------------------------------

 -- ----- ------------------------------------------------------------------------------------------------------------------------------------------------------
 2. From the *Select Actions* drop-down list, choose **Delete**. If a confirmation message is displayed, type **Delete** and click **Delete**to confirm.
 -- ----- ------------------------------------------------------------------------------------------------------------------------------------------------------
