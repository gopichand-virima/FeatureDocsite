# Monitoring Profile: Action Details

Use this function to specify what how the system handles changes found for an item being monitored. For example, if a configuration file is being monitored, and a change is detected, the action may be to create an incident report or send a notification to a user.

In the main window, click **Admin > Discovery > Discovery Monitoring Profile**.

From the *Select Actions* drop-down list, choose **New**. The window updates.

Navigate to the **Action Details** section.

In the *Target Module* field, click the drop-down list and select the required ITSM module.

In the *Create From Model* field, select the template to be used for the selected module.

In the *Property* fields, click the drop-down list, select the applicable property, and then select a matching value (required).


Depending on the type of property, an additional step may be required. For example, if the Property **Assigned User from Group** is selected, a user must be selected in the *User* dialog box.

![user dialog box old](../Resources/Images/Admin%20-%20Discovery/user-dialog-box-old.PNG)


To add a new property field row, click the **plus + sign**.

To remove a property field row, click the **minus - sign**.

When all selections/entries are made, click **Add**.


If required information is missing (such as a property value), an Alert Message displays. Resolve the error and then attempt to add or save the information again
