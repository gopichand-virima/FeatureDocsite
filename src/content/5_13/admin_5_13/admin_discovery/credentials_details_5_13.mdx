# Credential Details

Use this function to edit details for credentials.

Not all fields are enabled to accept changes as the fields vary based on the Credential Type configured.

In the main window, click **Admin > Discovery > Credentials**. The Credential window displays.

Select an existing record. A new window displays with the **Details** tab selected. !

Make the necessary modifications.

Modify the tabs according to the following:

- [Components](/itsm/config_mngmt/config_items/config_item_components/component_tab_options_5_13.mdx)
- [Relationships](/itsm/config_mngmt/config_items/config_item_details/config_items_relationships_5_13.mdx)
- [SLA](/itsm/config_mngmt/config_items/config_item_details/config_items_sla_5_13.mdx)
- [Maintenance](/itsm/config_mngmt/config_items/config_item_details/config_items_maintenance_5_13.mdx)
- [Tasks](/itsm/release_mngmt/tasks_5_13.mdx)
- [Comments](/discovery/comments_5_13.mdx)
- [Attachments](/discovery/attachments_5_13.mdx)
- [Audits](/itsm/config_mngmt/audits/audits_5_13.mdx)
- [History](/itsm/config_mngmt/config_items/config_item_details/config_items_history_5_13.mdx)

When all selections/entries are made, click **Save**.
