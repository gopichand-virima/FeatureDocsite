# Ignore ADM Process

Use this function to ignore ADM back-end processes when creating relationships.

In the main window, click **Admin > Discovery > Ignore ADM Process**. The Ignore ADÂ Process Names dialog box displays.

Enter the name(s) of the process to ignore. Separate multiple process names with a comma.

Click **Add**.

System displays the successful message.

Click **OK**.
