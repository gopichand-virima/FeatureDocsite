# Import Data Files

Use this function to import details of CI, CI relations, and asset details in the form of an Excel spreadsheet.

In the navigation pane, select **Discovery Scan \> Import Data Files**. The Import Data Files window displays.

![arrow border](../../Resources/Images/arrow-border.PNG)

[[[Closed]{.image .placeholder .MCDropDown_Image_Icon original-image-src="../../../Skins/Default/Stylesheets/Images/transparent.gif" original-image-title="" height="12" width="16" Imported Data Files](javascript:void(0)){.MCDropDownHotSpot .dropDownHotspot .MCDropDownHotSpot_ .MCHotSpotImage aria-expanded="false"}]{.MCDropDownHead .dropDownHead}

When an asset record is selected, the Details window displays. While the content cannot be edited, the following can be done:

::: Bullet1
[Move to CMDB](../Move-CMDB.mdx)

::: Bullet1
[Rescan Now](../Rescan-Now.mdx)

::: Bullet1
View [Tasks](/itsm/release_mngmt/tasks_6_1.mdx), [Comments](/discovery/comments_6_1.mdx), and [History](/common_topics/history_6_1.mdx)

::: Bullet1
View [Updates](/common_topics/updates_6_1.mdx){.MCTopicPopup .MCTopicPopupHotSpot .a}

Other information may be shown, and varies by asset. For example:

::: Bullet1
If the asset is *Software*, a *Components* option (if shown) may contain sub-components, such as *Software License Keys* and *IP Connections*. This information is view only.

::: Bullet1
If the asset is *Windows Host*, a *Components* option (if shown) may contain sub-components, such as Network Adapters and IP Connections. This information is view only.

[[[Closed]{.image .placeholder .MCDropDown_Image_Icon original-image-src="../../../Skins/Default/Stylesheets/Images/transparent.gif" original-image-title="" height="12" width="16" Imported Data Files](javascript:void(0)){.MCDropDownHotSpot .dropDownHotspot .MCDropDownHotSpot_ .MCHotSpotImage aria-expanded="false"}]{.MCDropDownHead .dropDownHead}

The Imported Data Files window is a list of assets pulled in by an import process. Therefore, these asset records cannot be deleted.
