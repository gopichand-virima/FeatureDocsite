# CMDB - Online Help Overview

Welcome to the Configuration Management Database (CMDB) module documentation for Virima version 5.13.

## Overview

The CMDB module provides a centralized repository for all IT assets and their relationships, enabling comprehensive IT infrastructure management and dependency mapping.

## Key Features

- **Asset Repository**: Centralized storage for all IT configuration items
- **Relationship Mapping**: Visual representation of asset dependencies and connections
- **Change Tracking**: Audit trail for all configuration changes
- **Advanced Search**: Powerful filtering and query capabilities

## Getting Started

To begin using CMDB:
1. Navigate to the CMDB module from the main menu
2. Explore existing configuration items
3. Add new CIs or update existing ones
4. Map relationships between assets
5. Generate dependency reports
