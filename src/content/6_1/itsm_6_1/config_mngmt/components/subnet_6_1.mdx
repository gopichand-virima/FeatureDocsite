# Subnets

 --- -------------------------------------------------------------------------------------------------------------
   This function is only available with certain asset types. Therefore, the relevant tab may not be displayed.
 --- -------------------------------------------------------------------------------------------------------------

 -- ----- ------------------------------------------------------
 1. While viewing an asset record, click **Components**.
 -- ----- ------------------------------------------------------

 -- ----- --------------------
 2. Click **Subnets**.
 -- ----- --------------------

::: Ind50
![subnets new 475x172](../../../Resources/Images/Components/subnets-new_475x172.png)

## Add New Subnet

 -- ----- ------------------------------------------------
 1. Click **Add**. The Subnet dialog box displays.
 -- ----- ------------------------------------------------

::: Ind50
![subnets add new 478x157](../../../Resources/Images/Components/subnets-add-new_478x157.png)

 -- ----- -------------------------------------------------
 2. Click **New Subnet** (if not already selected).
 -- ----- -------------------------------------------------

 -- ----- -------------------------------------------------
 3. Enter the applicable information in each field.
 -- ----- -------------------------------------------------

 -- ----- --------------------------------------------
 4. When all entries are made, click **Save**.
 -- ----- --------------------------------------------

## Modify Existing Subnet

 -- ----- ----------------------------
 1. Click **Existing Subnet**.
 -- ----- ----------------------------

::: Ind50
![subnets existing new 478x329](../../../Resources/Images/Components/subnets-existing-new_478x329.png)

 -- ----- -----------------------------------------------------------------------------------------------------------------------------------------------------------------------------
 2. Enter the search criteria to locate the existing device. *See also [Advanced Search](../../../Common%20Topics/Advanced-Search.mdx){.MCTopicPopup .MCTopicPopupHotSpot .a}.*
 -- ----- -----------------------------------------------------------------------------------------------------------------------------------------------------------------------------

 -- ----- -----------------------------------------------------
 3. Select the applicable device from the results list.
 -- ----- -----------------------------------------------------

 -- ----- -------------------------------------------------------
 4. When all selections/entries are made, click **Save**.
 -- ----- -------------------------------------------------------

## Delete Subnet

While viewing the list of existing subnets, select the device to delete, and click the **Delete** button.
