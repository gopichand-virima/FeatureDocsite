# Tabs (main panel)

**Details:** A complete inventory of the Configuration Item (CI) is captured, with the specific details shown depending on the blueprint associated with the asset.

Common details for Meraki devices include:
- Device identification (serial number, model, MAC address)
- Network assignment and location
- Firmware version and status
- IP addressing information
- Device-specific configuration details
- Last seen timestamp
- Tags and custom attributes from Meraki
