# About Organizational Details

The Organizational Details section provides administrative functions to manage the organizational structure and configuration of your Virima instance.

## Overview

Use these functions to define and manage:

- **Cost Centers** - Define financial cost centers for budget tracking and allocation
- **Departments** - Organize users and resources by department structure
- **Designations** - Define job titles and roles within the organization
- **Holidays** - Configure organizational holidays and non-working days
- **Locations** - Manage physical locations and office sites
- **Operational Hours** - Set business hours and working schedules

## Accessing Organizational Details

To access Organizational Details:

1. Navigate to **Admin** from the main menu
2. Select **Organizational Details**
3. Choose the specific function you want to manage

Each organizational detail function provides options to create, edit, delete, and export records as needed for your organizational configuration.
