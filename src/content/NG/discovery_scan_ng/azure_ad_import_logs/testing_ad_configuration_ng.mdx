# Testing AD Configuration

1. Open the newly added Azure AD Configuration line item.

2. From the Select Actions drop-down list, Select [Test Azure AD Configuration.]{.Strong}

![Azure AD User Import Logs 6](../../Resources/Images/Discovery/Azure-AD-Users-Import-Logs/Azure AD User Import Logs_6.png)

3. If you have set the values of the attributes correctly as per your organization\'s Azure AD structure, then the connection will be a success. Otherwise, it will fail.

![Azure AD User Import Logs 7](../../Resources/Images/Discovery/Azure-AD-Users-Import-Logs/Azure AD User Import Logs_7.png)
