# Search Knowledge

Effective knowledge search helps find solutions quickly and improve incident resolution times.

## Search Methods

### Quick Search
1. Enter keywords in search bar
2. View instant results
3. Click article to read
4. Apply solution

### Advanced Search
1. Click **Advanced Search**
2. Add multiple criteria:
   - Keywords
   - Category
   - Date range
   - Author
   - Rating
3. Combine with AND/OR logic
4. Search

### Browse by Category
1. Navigate to **Knowledge Base**
2. Select category
3. View articles in category
4. Subcategories available

## Search Tips

### Use Specific Keywords
- Include error messages
- Product names
- Specific symptoms
- Technical terms

### Filter Results
- By rating (highest first)
- By views (most popular)
- By date (newest first)
- By relevance

### Save Searches
1. Create search criteria
2. Click **Save Search**
3. Name the search
4. Access from saved searches

## Using Search Results

### Article Preview
- Title and description
- Relevance score
- Views and rating
- Last updated date

### Open Article
- Click to view full content
- Follow instructions
- Apply solution
- Rate article

### Related Articles
System suggests:
- Similar articles
- Related topics
- Frequently viewed together

## Context-Sensitive Search

### From Incident
1. Open incident
2. Click **Search Knowledge**
3. System suggests articles based on:
   - Incident category
   - Description keywords
   - Error messages
   - Affected CI
4. Select relevant article
5. Apply solution

### From Problem
1. Open problem record
2. Search for related knowledge
3. Find similar issues
4. Apply documented solutions

## Mobile Knowledge Search

Access knowledge on mobile:
- Mobile-optimized interface
- Voice search
- Offline access to favorites
- Quick article access
