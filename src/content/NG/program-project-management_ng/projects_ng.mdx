# Projects

Use this function to create a project, which is the first step in Project Management process. A project can be created manually (by selecting a project template) or importing a project file.

 --- -------------------------------------------------------------------
   To create/edit projects, users must have appropriate role access.
 --- -------------------------------------------------------------------

In the navigation pane, select **Program and Project Management \> Projects**. The Projects window displays.

![project main old](../Resources/Images/Prog - Proj Mngmnt/project-main-old.PNG)

![arrow border](../Resources/Images/arrow-border.PNG)

 -- ----- -------------------------------------------------------------------------------------------------------
 1. From the *Select Actions* drop-down list, choose **New Project**. The Add Project dialog box displays
 -- ----- -------------------------------------------------------------------------------------------------------

[[[Closed]{.image .placeholder .MCDropDown_Image_Icon original-image-src="../../Skins/Default/Stylesheets/Images/transparent.gif" original-image-title="" height="12" width="16" and Tasks](javascript:void(0)){.MCDropDownHotSpot .dropDownHotspot .MCDropDownHotSpot_ .MCHotSpotImage aria-expanded="false"}]{.MCDropDownHead .dropDownHead}

 -- ----- -------------------------------------------------------------
 1. While viewing the Details window, click the **Phases** Tab.
 -- ----- -------------------------------------------------------------

 -- ----- -------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------
 2. To add a new **Phase**, from the Select Actions drop-down list, choose **Add Phase**. [Refer to Project Template: Phase Information Fields.](../Admin%20-%20Project%20Mngmnt/Project-Template.mdx#PhaseInfoFields){style="font-style: italic;"}
 -- ----- -------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------

 -- ----- --------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------
 3. To add a new **Task**, move to a line item containing the applicable phase, and click the **plus + sign**. [Refer to Project Template: Task Information Fields.](../Admin%20-%20Project%20Mngmnt/Project-Template.mdx#TaskInfoFields){style="font-style: italic;"}
 -- ----- --------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------

 -- ----- -------------------------------------------------------
 4. When all selections/entries are made, click **Save**.
 -- ----- -------------------------------------------------------

[[[Closed]{.image .placeholder .MCDropDown_Image_Icon original-image-src="../../Skins/Default/Stylesheets/Images/transparent.gif" original-image-title="" height="12" width="16" Gantt Chart](javascript:void(0)){.MCDropDownHotSpot .dropDownHotspot .MCDropDownHotSpot_ .MCHotSpotImage aria-expanded="false"}]{.MCDropDownHead .dropDownHead}

Use this option to display a visual representation of a project timeline that shows start and end dates of phases and tasks. Use this chart to add/delete a phase/task, change phases/task dates, and assess the progress of overall project.

To view the chart, do the following:

 -- ----- -----------------------------------------------------
 1. If not already displayed, click the **Phases** tab.
 -- ----- -----------------------------------------------------

 -- ----- ---------------------------------------
 2. Move to the right side of the window.
 -- ----- ---------------------------------------

 -- ----- ------------------------------
 3. Click the left-facing arrow.
 -- ----- ------------------------------

::: Ind50
![resource chart arrow old](../Resources/Images/Prog - Proj Mngmnt/resource-chart-arrow-old.PNG)

A window slides open and the Project Chart is displayed. It includes the status of each phase/task.

In this window, the icons provide the following functionality:

::: Bullet1
The **plus + sign** displays either the [Phase Information](../Admin%20-%20Project%20Mngmnt/Project-Template.mdx#PhaseInfoFields) dialog box or the [Task Information](../Admin%20-%20Project%20Mngmnt/Project-Template.mdx#TaskInfoFields) dialog box, depending on which type is selected.

::: Bullet1
The **up and down arrows** move the task up or down in the hierarchy.

::: Bullet1
The **trash can** deletes the task.

 --- -----------------------------------------------------------------------------------------
   Not all functions include an option to edit record line items shown in the main window.
 --- -----------------------------------------------------------------------------------------

 -- ----- -------------------------------------------------------
 1. Navigate to the window containing the record to edit.
 -- ----- -------------------------------------------------------

 -- ----- -------------------------------------------------------------------------------------
 2. Click the line containing the record. The applicable window or dialog box displays.
 -- ----- -------------------------------------------------------------------------------------

 -- ----- -----------------------------------
 3. Make the necessary modifications.
 -- ----- -----------------------------------

 -- ----- -------------------------------------------
 4. Click **Save** or **Add**, as applicable.
 -- ----- -------------------------------------------

When editing an existing item, other options may be available, such as adding [Tasks](/itsm/release_mngmt/tasks_6_1.mdx){.MCTopicPopup .MCTopicPopupHotSpot .a}, [Comments](/discovery/comments_6_1.mdx){.MCTopicPopup .MCTopicPopupHotSpot .a}, and [Attachments](/discovery/attachments_6_1.mdx){.MCTopicPopup .MCTopicPopupHotSpot .a} and viewing [History](/common_topics/history_6_1.mdx){.MCTopicPopup .MCTopicPopupHotSpot .a}.

 --- ---------------------------------------------------------------------------------------
   The options change based upon the function; therefore, not every option may be shown.
 --- ---------------------------------------------------------------------------------------

 --- --------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------

 -- ----- --------------------------------------------------
 1. Click the line that contains the item to delete.
 -- ----- --------------------------------------------------

 -- ----- ------------------------------------------------------------------------------------------------------------------------------------------------------
 2. From the *Select Actions* drop-down list, choose **Delete**. If a confirmation message is displayed, type **Delete** and click **Delete**to confirm.
