# Edit Receiving Slip Line

This feature enables to edit the configured values of the existing contract, and update,and then save the same in the application.

Click to open required RS Line.The RS Lines page displays.

The RS Lines displays the following information:

- Details
- Tasks
- Comments
- Attachment
- History
