# Edit or Delete Audits

Use this function to edit or delete audits.

## Edit an Audit

- In the navigation pane, select **ITSM > Configuration Management > Audits**. The Audits window displays.
- In the navigation pane, select **Configuration Management > Audits**. The Audits window displays.
- Select a record to edit and make the necessary modifications on the applicable tab.
- [Details](/itsm/config_mngmt/audits/audit_details_6_1.mdx)
- [Tasks](/itsm/release_mngmt/tasks_6_1.mdx)
- [Comments](/discovery/comments_6_1.mdx)
- [Attachments](/discovery/attachments_6_1.mdx)
- [History](/common_topics/history_6_1.mdx)
- When all selections/entries are made, click **Save**.

## Delete an Audit

Deleting is a permanent action and cannot be undone. Deleting may affect other functionality and information in the application such as data in configured reports, fields in windows, selectable options, etc. Therefore, be sure to understand the potential effects before making a deletion.


- Click the line that contains the item to delete.
- From the *Select Actions* drop-down list, choose **Delete**. If a confirmation message is displayed, type **Delete** and click **Delete**to confirm.
