# Schedule Import from AWS

Use this function to schedule when an import should occur.

Click the **Schedule Import AWS** icon. The Scheduled Import from AWS dialog box displays.

**Name**. Enter a *Name* to identify the Import.

**Credential**. Click the drop-down list and select the applicable *Credential* to be used to leverage AWS service.

Select the required services to import.

**Import Frequency**. Click the drop-down list and select the *Import Frequency*.

**Active**. If this import is active, select *Active*.

Click the drop-down list and select the *Timezone*.

Configure the time parameters (such as Seconds, Minutes, Hours and so forth.)

When all selections/entries are made, click **Add**.
