# System Requirements

Review the hardware, software, and network requirements needed to deploy Virima successfully.

## Server Requirements

| Component | Minimum | Recommended |
|-----------|---------|-------------|
| CPU | 4 cores, 2.5 GHz | 8+ cores, 3.0 GHz |
| RAM | 16 GB | 32+ GB |
| Storage | 250 GB SSD | 500+ GB SSD (NVMe) |
| Network | 1 Gbps | 10 Gbps |

## Supported Operating Systems

| OS Family | Supported Versions |
|-----------|--------------------|
| Linux | Ubuntu 20.04+, RHEL 8+, CentOS 8+, Debian 11+ |
| Windows Server | Windows Server 2019, 2022 |

## Database Support

| Database | Versions | Notes |
|----------|----------|-------|
| PostgreSQL | 13.x, 14.x, 15.x | Recommended |
| MySQL | 8.0+ | Supported |
| SQL Server | 2019, 2022 | Enterprise only |

## Browser Compatibility

| Browser | Minimum Version |
|---------|----------------|
| Google Chrome | 100+ |
| Mozilla Firefox | 100+ |
| Microsoft Edge | 100+ |
| Safari | 15+ |

## Network Requirements

- HTTPS (443) for web interface access
- SSH (22) for Linux device discovery
- WMI (135, 445) for Windows device discovery
- SNMP (161) for network device discovery
- Outbound internet access for cloud discovery (AWS, Azure, GCP)

⚠️ **Note:** Requirements may vary based on the number of managed assets and discovery frequency. 
Contact Virima support for sizing guidance for large deployments (10,000+ assets).
