# Flush Credentials

Use this function to disassociate a set of credentials and a device on the network which was scanned previously will be removed.

<div class="NumList1" madcap:autonum="1. ">

In the [Credentials](Credentials.htm) window, select the credential record to be flushed.

</div>

|     |                                                                                            |
|-----|--------------------------------------------------------------------------------------------|
| Â    | At least one record must be selected. If no record is selected, an Alert message displays. |

<div class="NumList2" madcap:conditions="Default.OldUI" madcap:autonum="2. ">

From the *Select Actions* drop-down list, choose **Flush**.

</div>

<div class="NumList2" madcap:conditions="Default.NewUI" madcap:autonum="3. ">

Click the **Flush** icon.

</div>

<div class="NumList2" madcap:conditions="Default.OldUI" madcap:autonum="4. ">

At the *Confirmation* prompt, click **Yes.** No other action is required.

</div>

<div class="NumList2" madcap:conditions="Default.NewUI" madcap:autonum="5. ">

At the *Confirmation* prompt, click **Continue**. No other action is required.

</div>
