# Infoblox Appliances

Use this function to add, edit, or remove an Infoblox Appliance.

In the main window, select **Discovery Scan > IPAM Networks**. The IPAM Networks window displays.

At the top of the window, click **Infoblox Appliances**. A list of all currently configured Infoblox Appliances displays.

To create a new InfoBlox Appliance, click the **New Infoblox Appliance** icon (shown to the right of the title "Infoblox Appliances").
