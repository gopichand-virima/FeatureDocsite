# CI: Private Properties

Use this function to display asset primary details, device details, hardware/network information, and operating system details.

|     |                                                                                                                  |
|-----|------------------------------------------------------------------------------------------------------------------|
|     | Properties that are added/marked as private when the Blueprint is created can be viewed only by specified Roles. |

<div class="NumList1" madcap:conditions="Default.OldUI" madcap:autonum="1. ">

In the navigation pane, select **ITSM /> Configuration Management /> CMDB**. The Configuration Items window displays.

</div>

<div class="NumList1" madcap:conditions="Default.NewUI" madcap:autonum="1. ">

In the navigation pane, select **Configuration Management /> CMDB**. The Configuration Items window displays.

</div>

<div class="NumList2" madcap:autonum="2. ">

Select a record in the list. A new window opens and the **Details** tab displays.

</div>

<div class="NumList2" madcap:autonum="3. ">

Click the **Private Properties** tab.

</div>

<div class="NumList2" madcap:autonum="4. ">

Modify the information, if necessary.

</div>

<div class="NumList2" madcap:autonum="5. ">

When all selections/entries are made, click **Save**.

</div>
