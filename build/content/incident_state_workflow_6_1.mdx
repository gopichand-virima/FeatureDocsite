# Incident State Workflow

Use this function to create a workflow for incidents.

In the main window, click **Admin > Incident Management > Incident State Workflow**. The Incident State Workflow window displays.

## New Incident State Workflow

From the *Select Actions* drop-down list, choose **New Incident Workflow**. The Incident State Workflow window displays.

Enter a *Name* and *Description* for this workflow.

Click **Add**.
