# Exporting Intune Records

Exports data for the selected imported Intune record as an Excel spreadsheet and sends an email notification when the spreadsheet is ready for download.

- From the [Select Actions]{.Strong} drop-down list, choose [Export]{.Strong}.

 ![Import INTUNE Window 14](../../Resources/Images/Discovery/Import-From-Intune/Import INTUNE Window_14.png)

- Click ****Continue**.**

 ![Import INTUNE Window 15 807x249](../../Resources/Images/Discovery/Import-From-Intune/Import INTUNE Window_15_807x249.png)

- The system downloads an Excel (.xlsx) file of the list with the details,

 - [Unique ID]{.Strong} -- The Virima import record ID (e.g., Intune000578).

 - [Credential]{.Strong} -- The credential set used (e.g., Intune, Intune_Credential).

 - [Imported By]{.Strong} -- The legged-in user who initiated the import (e.g., Admin Virima).

 - [Name]{.Strong} -- The Intune service or resource type imported (e.g., Intune).

 - [Imported On]{.Strong} -- The timestamp when that record was imported (as displayed in the UI).

 ![Import INTUNE Window 16](../../Resources/Images/Discovery/Import-From-Intune/Import INTUNE Window_16.png)
