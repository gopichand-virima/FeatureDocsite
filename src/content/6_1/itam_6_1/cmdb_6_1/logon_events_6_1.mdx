# Log On Events

<span class="image placeholder" original-image-src="../Resources/Images/CMDB%20-%20Left-Nav/Configuration%20Management%20Database_60.png" original-image-title=""></span>

- The <span class="Strong" style="font-weight: bold;">Log On Events</span> tab displays detailed records of user login activity associated with a Configuration Item (CI).

- This feature is useful for <span class="Strong">monitoring security access</span>, <span class="Strong">troubleshooting logon issues</span>, and <span class="Strong">tracking usage patterns</span> over a defined time frame.

- This tab is available for both <span class="Strong">Windows Server</span> and <span class="Strong">Windows Host</span> blueprints.

<span class="Strong" style="font-weight: bold;">How to view Log On Events:</span>

1.  Click the <span class="Strong" style="font-weight: bold;">Log On Events</span> tab.

2.  Use the <span class="Strong" style="font-weight: bold;">Client</span> dropdown to select the applicable system/clientâ€™s name.

3.  Choose a time range from the <span class="Strong" style="font-weight: bold;">Duration</span> dropdown (e.g., 1 day, 7 days, 30 days).

    a/) To enter a specific time range, choose <span class="Strong" style="font-weight: bold;">Custom</span> and input a manual duration.

4.  Click <span class="Strong" style="font-weight: bold;">Refresh</span> to load the login data for the selected period.
