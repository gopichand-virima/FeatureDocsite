# New Ad Hoc Reports

Use this function to create view and modify Ad Hoc reports and save [Canned Reports](Reports-Canned.htm).

There are three steps in the process:

<div class="Ind25">

Step 1. Specify the **Identifying Information** such as visibility and frequency. *See below*.

</div>

<div class="Ind25">

Step 2. Select the **Report Properties** and set the **Conditions**. *Refer to* <a href="Reports-Props-Conds.htm" style="font-style: italic;">Report Properties and Conditions</a>*.*

</div>

<div class="Ind25">

Step 3. **Run** (or Generate) the report. *Refer to* <a href="Reports-Run.htm" style="font-style: italic;">Run Report</a>*.*

</div>

|     |                                                                 |
|-----|-----------------------------------------------------------------|
|     | To *edit* a report, following the same steps as outlined above. |
