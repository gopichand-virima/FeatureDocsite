# Sync (Instant and Scheduled)

Use this function to plan, track, and manage information related to IP addresses in the monitored network.

1.  In the <span class="Strong">IPAM Networks</span> window, click <span class="Strong">Sync</span>. The <span class="Strong">Sync from Infoblox</span> window will display.

<span class="image placeholder" original-image-src="../../Resources/Images/Discovery/IPAM-Networks/IPAM%20Functions%20Overview_3.png" original-image-title=""></span>

2.  <span class="Strong">Select Sync Type</span>:

    - <span class="Strong">Instant Sync:</span> Click to immediately import records from the Infoblox server.

    - <span class="Strong">Schedule Sync:</span> Choose this option to import records at a scheduled time.

      - In the <span class="Strong">Schedule Report Frequency</span> drop-down list, select the desired frequency.

      - Select one or more options based on the schedule: <span class="Strong">Second</span>, <span class="Strong">Minute</span>, <span class="Strong">Hour</span>, <span class="Strong">Day</span>, <span class="Strong">Month</span>, <span class="Strong">Weekday</span>.

      - Click <span class="Strong">Add</span> to confirm the scheduling details.

3.  Click <span class="Strong">Sync Now</span> to execute the operation and sync records immediately.
