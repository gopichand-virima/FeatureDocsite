# Accessing the Azure AD User Import Logs

1. From the left navigation pane, go to **Discovery Scan \> Azure AD User Import Logs**.

 ![Azure AD User Import Logs 11](../../Resources/Images/Discovery/Azure-AD-Users-Import-Logs/Azure AD User Import Logs_11.png)

2. The [AD User Import Logs]{.Strong} page opens.

 ![Azure AD User Import Logs 12](../../Resources/Images/Discovery/Azure-AD-Users-Import-Logs/Azure AD User Import Logs_12.png)

3. This shows a list of records.

 - [Key Information displayed]{.Strong}:

 - [Name]{.Strong} -- The name used to identify the import (for example, Azure AD).

 - [Started On]{.Strong} -- The date and time the import process began.

 - [Completed On]{.Strong} -- The date and time the import process completed.

4. Use the [filter fields]{.Strong} at the top of each column to search by name or narrow results by date range.
