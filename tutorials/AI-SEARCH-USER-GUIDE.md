## üìö Three Ways to Search

### **1. Search Docs** üü¢
**Best for**: Finding specific documentation pages

**How to use**:
1. Type your search query
2. Choose search scope:
   - **This Page**: Search only current page
   - **All Docs**: Search current version (default)
   - **All Versions**: Search across all versions
3. Click any result to navigate

**Example**: "SNMP configuration" ‚Üí Shows all SNMP-related pages

---

### **2. AI Assistant** üü£
**Best for**: Getting answers to complex questions

**How to use**:
1. Switch to "AI Assistant" tab
2. Type your question naturally
3. AI provides comprehensive answer with sources
4. Click sources to read more
5. Ask follow-up questions

**Example Questions**:
- "How do I configure SNMP discovery?"
- "What are the best practices for cloud discovery?"
- "Explain the CMDB relationship mapping"

**Pro Tips**:
- Ask complete questions, not single words
- Use follow-ups for deeper understanding
- Click sources to verify information
- Copy answers using the "Copy" button

---

### **3. Search Web** üîµ
**Best for**: Finding external resources and community help

**How to use**:
1. Switch to "Search Web" tab
2. Type your query
3. View results from:
   - Virima official website
   - Community forums
   - Support portal
   - Knowledge bases
4. Click to open in new tab

**Popular Resources**:
- Virima Official Website
- Community Forum
- Support Portal

## ‚ùì FAQ

### **Q: When should I use AI Assistant vs Search Docs?**

**A**: 
- **AI Assistant**: Complex questions, need explanations, want multiple sources
- **Search Docs**: Quick navigation, finding specific pages, keyword searches

### **Q: Does voice search work on mobile?**

**A**: Yes! Voice search works on all devices with microphone access.

### **Q: Are AI answers always accurate?**

**A**: AI answers synthesize from documentation and web sources. Always check the cited sources for verification. Documentation sources are most reliable.

### **Q: Can I search across all versions?**

**A**: Yes! Select "All Versions" scope in Search Docs tab.

### **Q: How do I save or bookmark answers?**

**A**: Click the "Copy" button to copy AI answers, then paste into your notes. Source links remain accessible.

### **Q: Why no results?**

**A**: Try:
- Rephrasing your query
- Using different keywords
- Switching to AI Assistant
- Checking Search Web tab
- Using All Versions scope

### **Q: Can I search while on the home page?**

**A**: Yes! Search works from anywhere in the documentation.
