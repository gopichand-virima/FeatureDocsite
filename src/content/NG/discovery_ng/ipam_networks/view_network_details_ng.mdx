# View Network Details

Select a network from the IPAM Networks list to view its details.

The network details page provides comprehensive subnet information including:

## Summary Information
- Network address and CIDR notation
- Subnet mask
- Network class (A, B, or C)
- Gateway address
- Broadcast address
- Total available IP addresses
- Used IP addresses
- Free IP addresses
- Utilization percentage

## Configuration Details
- VLAN assignment
- Location and site information
- Network purpose and description
- DHCP configuration details
- DNS server associations
- Custom attributes and tags

## IP Address List
A complete table of all IP addresses in the subnet showing:
- IP address
- Status (Used/Unused/Reserved)
- Associated hostname
- MAC address
- Last seen timestamp
- Device information

Users can perform actions such as status updates, regular scans, or export reports directly from this view.
