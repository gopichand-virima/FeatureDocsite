# Cherwell New Mapping

Use this function to enter the details for mapping properties between Virima and Cherwell.

- In the main window, click the **gear** icon and select **Integrations > Cherwell Mappings**.

!

The Cherwell Mappings window displays.

!

- To create a new mapping, click **New Mapping**. The New Cherwell Mapping window displays.

!

- In the *Virima Blueprint* field, click the drop-down list and select the applicable blueprint.
- In the *Cherwall Object* field, click the drop-down list and select the applicable object.
- When the selections are made, click **Add**.
- In the *Virima Property* field, click the drop-down list and select the property to map to the Cherwell Propery.
- In the *Cherwell Property* field, click the drop-down list and select the property to map to the Virima Property.
- In the Value field, type the value that applies to both properties.
- To use a correlator, select **Correlator**.
- To remove a mapping, select the item in the list of mappings and click the **trashcan icon**.
