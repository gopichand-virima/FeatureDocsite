# Custom BSM Views

Use this function to customize the display properties of Business Service Management (BSM).

 -- ----- -----------------------------------------------------------------------------------------
 1. In the main window, select **Admin \> SACM \> BSM View**. The BSM View window displays.
 -- ----- -----------------------------------------------------------------------------------------

::: {.Ind50
![bsm view old 680x114](../Resources/Images/Admin - SACM/bsm-view-old_680x114.png)

 -- ----- ----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------
 2. From the *Select Actions* drop-down list, choose an action: [[New BSM View](Custom-BSM-VIews-New.mdx)]{style="background-color: #ffffff;"}, [Delete](../Common%20Topics/Delete-Remove.mdx){.MCTopicPopup .MCTopicPopupHotSpot .a} or [Export](/discovery/import_data_files/export_6_1.mdx){.MCTopicPopup .MCTopicPopupHotSpot .a}.
 -- ----- ----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------

 -- ----- -----------------------------------------------------------------------
 3. To create a new view, click [New BSM View](Custom-BSM-VIews-New.mdx).
 -- ----- -----------------------------------------------------------------------

## Edit BSM View

 --- -----------------------------------------------------------------------------------------
   Not all functions include an option to edit record line items shown in the main window.
 --- -----------------------------------------------------------------------------------------

 -- ----- -------------------------------------------------------
 1. Navigate to the window containing the record to edit.
 -- ----- -------------------------------------------------------

 -- ----- -------------------------------------------------------------------------------------
 2. Click the line containing the record. The applicable window or dialog box displays.
 -- ----- -------------------------------------------------------------------------------------

 -- ----- -----------------------------------
 3. Make the necessary modifications.
 -- ----- -----------------------------------

 -- ----- -------------------------------------------
 4. Click **Save** or **Add**, as applicable.
 -- ----- -------------------------------------------
