# User License Entitlements

Use this function to obtain a clear, consolidated view of an organization's licenses and deployments across the product users. This information can be valuable in the software audit, giving organizations the data needed to manage risk and remain compliant as software is consumed across their organization.

In the navigation pane, select **ITAM > User License Entitlement**. The User License Entitlement page displays.


New User License Entitlement

From the *Select Actions* drop-down list, choose **New User License Entitlement**. The New User License Entitlement window displays.

Click **New**. The New User License Entitlement window displays.

Enter a *Display Name*.

For *Configuration item*, click **Add,** then search for and select the CI (typically an asset) that goes with this license.

For *Configuration item*, click the **plus + sign**, then search for and select the CI (typically an asset) that goes with this license.

For *User*, click **Add**, then search for an add the user who is entitled to this license.

For *User*, click the **plus + sign**, then search for and add the user entitled to this license.

When all selections/entries are made, click **Add**.
