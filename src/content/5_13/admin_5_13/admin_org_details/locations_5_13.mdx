# Locations

Use this function to associate users, facilities, or configuration items with a location.

In the main window, click **Admin > Organizational Details > Locations**. The Locations window displays.

New Location

From the _Select Actions_ drop-down list, choose New Location. The New Location window displays.

Click **New Locations**. The New Location window displays.

Complete the fields with the available information. The _Name_, _City_, _State_ and _Country_ are required fields.

When all entries are made, click **Add**.

Edit Location Delete Location

To delete an existing location:

Select the location to delete. from the _Select Actions_ drop-down list, choose **Delete**.

From the _Select Actions_ drop-down list, choose **Delete**. If a confirmation message is displayed, type **Delete** and then click **Delete**.
