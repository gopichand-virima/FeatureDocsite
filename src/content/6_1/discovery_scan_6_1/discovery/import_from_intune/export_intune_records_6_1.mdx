# Exporting Intune Records

Exports data for the selected imported Intune record as an Excel spreadsheet and sends an email notification when the spreadsheet is ready for download.

- From the <span class="Strong" style="font-weight: bold;">Select Actions</span> drop-down list, choose <span class="Strong" style="font-weight: bold;">Export</span>.

  <span class="image placeholder" original-image-src="../../Resources/Images/Discovery/Import-from-Intune/Import%20INTUNE%20Window_14.png" original-image-title=""></span>

<!-- -->

- Click <span class="Strong">**Continue**.</span>

  <span class="image placeholder" original-image-src="../../Resources/Images/Discovery/Import-from-Intune/Import%20INTUNE%20Window_15.png" original-image-title=""></span>

<!-- -->

- The system downloads an Excel (.xlsx) file of the list with the details,

  - <span class="Strong" style="font-weight: bold;">Unique ID</span> – The Virima import record ID (e.g., Intune000578).

  - <span class="Strong" style="font-weight: bold;">Credential</span> – The credential set used (e.g., Intune, Intune_Credential).

  - <span class="Strong" style="font-weight: bold;">Imported By</span> – The legged-in user who initiated the import (e.g., Admin Virima).

  - <span class="Strong" style="font-weight: bold;">Name</span> – The Intune service or resource type imported (e.g., Intune).

  - <span class="Strong" style="font-weight: bold;">Imported On</span> – The timestamp when that record was imported (as displayed in the UI).
