# Monitoring Profile: Trigger Conditions

Use this function to define a condition to match the discovered data as configured.

 -- ----- ----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------
 1. [On the main window, select ]{.span_1}[Admin]{.Strong_2}[ \> ]{.span_1}[Discovery]{.Strong_2}[** \> Discovery Monitoring Profile**. The Discovery Monitoring Profile window displays.]{.span_1}
 -- ----- ----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------

 -- ----- -------------------------------------------------------
 2. Select either **System Monitor** or **File Monitor**.
 -- ----- -------------------------------------------------------

 -- ----- --------------------------------------------------------------------------
 3. In the **Trigger Conditions** window, use the icons to do the following:
 -- ----- --------------------------------------------------------------------------

::: Bullet2
Click the **plus + sign** to add an additional row.

::: Bullet2
Click the **red [x]{style="color: #dc143c;"}** to remove the row.

 -- ----- ----------------------------------
 4. If *System Monitor* is selected:
 -- ----- ----------------------------------

 -- ----- ------------------------------------------------------------------------------------------------
 a. In the *Property* field, click the drop-down list and choose a property from those available.
 -- ----- ------------------------------------------------------------------------------------------------

 -- ----- ---------------------------------------------------------------------------------------
 b. In the *Operator* field, click the drop-down list and choose the applicable operator.
 -- ----- ---------------------------------------------------------------------------------------

 -- ----- ------------------------------------------------------------------------------------------------------------------------------------------------
 c. The Operators displayed are valid for the selected *Property* type, and then enter the value.\
 ![Monitoring profile new 6.1](../Resources/Images/Admin - Discovery/Monitoring-profile-new-6.1.png)\

 -- ----- ------------------------------------------------------------------------------------------------------------------------------------------------

 -- ----- --------------------------------
 5. If *File Monitor* is selected:
 -- ----- --------------------------------

 -- ----- -------------------------------------------------------------------------------
 a. In the *Type* field, click the drop-down list and select the applicable type.
 -- ----- -------------------------------------------------------------------------------

 -- ----- -----------------------------------------------------
 b. In the text field, type the applicable information.
 -- ----- -----------------------------------------------------

::: Ind75
[]{.image .placeholder original-image-src="../Resources/Images/Admin%20-%20Discovery/discovery-mon-prof-trigger-conditions-file-mon.PNG" original-image-title=""

 -- ----- ------------------------------------------
 c. To add a row, click the **plus + sign**.
 -- ----- ------------------------------------------

 -- ----- ----------------------------------------------
 d. To delete a row, click the **minus - sig**n.
 -- ----- ----------------------------------------------

 -- ----- -----------------------------------------------------------------------------------------
 6. When all selections/entries are made, and a match is found, the condition is triggered.
 -- ----- -----------------------------------------------------------------------------------------

 -- ----- ------------------------------------------------------
 7. When all selections/entries are made, click **Add**.
 -- ----- ------------------------------------------------------
