# IT Asset Management - Quick Start Guide

Start managing your IT assets effectively.

## Prerequisites

- ITAM module access permissions
- Asset inventory data (optional)
- Procurement and contract information
- Software license details

## Quick Start Steps

### Step 1: Set Up Asset Categories
1. Navigate to ITAM > Settings > Categories
2. Define asset categories (laptops, servers, etc.)
3. Configure custom fields for each category
4. Set lifecycle stages
5. Save configurations

### Step 2: Add Assets
1. Click "Add Asset" or use bulk import
2. Select asset category
3. Fill in asset details:
   - Serial number, model
   - Owner, location
   - Purchase date, cost
4. Upload documentation
5. Save asset record

### Step 3: Configure Software Licenses
1. Go to ITAM > Software Licenses
2. Add license agreements
3. Enter license count and cost
4. Track installations
5. Set renewal reminders

### Step 4: Set Up Contracts
1. Navigate to Contracts section
2. Add vendor contracts
3. Configure renewal alerts
4. Link assets to contracts
5. Upload contract documents
