# New Canned Reports

Use this function to create, view and modify canned reports.

|     |                                                                                  |
|-----|----------------------------------------------------------------------------------|
|     | Canned reports are created by saving an [Ad Hoc report](Reports-New-Ad-Hoc.htm). |

<div class="NumList1" madcap:autonum="1. ">

In the navigation pane, select **Reports /> Canned Reports**. The Canned Reports window displays.

</div>

<div class="Ind50">

<span class="image placeholder" original-image-src="../Resources/Images/Reports/canned-report-main-old.PNG" original-image-title=""></span>

</div>

<div class="NumList2" madcap:conditions="Default.OldUI" madcap:autonum="2. ">

From the *Select Actions* drop-down list, choose **New Canned Report**. The Reports dialog box displays.

</div>

<div class="Ind50" madcap:conditions="Default.OldUI">

<span class="image placeholder" original-image-src="../Resources/Images/Reports/reports-dialog-box.PNG" original-image-title="" style="width: 471px;height: 211px;" madcap:conditions="Default.OldUI"></span>

</div>

<div class="Ind50" madcap:conditions="Default.NewUI">

<span class="image placeholder" original-image-src="../Resources/Images/Reports/canned-report-new.PNG" original-image-title="" style="width: 481px;height: 89px;" madcap:conditions="Default.NewUI"></span>

</div>

<div class="NumList2" madcap:autonum="3. ">

Click **New Canned Report**. The Reports dialog box displays.

</div>

<div class="Ind50" madcap:conditions="Default.NewUI">

<span class="image placeholder" original-image-src="../Resources/Images/Reports/canned-reports-search-new.PNG" original-image-title="" madcap:conditions="Default.NewUI" style="width: 471px;height: 312px;"></span>

</div>

<div class="NumList2" madcap:conditions="Default.OldUI" madcap:autonum="4. ">

In the *Search* field, type the **name of the report** to search for.

</div>

<div class="NumList2" madcap:conditions="Default.OldUI" madcap:autonum="5. ">

Click the **Search** button. A list displays containing all reports that match the search criteria*.*

</div>

<div class="NumList2" madcap:conditions="Default.NewUI" madcap:autonum="6. ">

In the *Search* field, type the **name of the report** to search for. A list displays containing all reports that match the search criteria*.*

</div>

<div class="NumList2" madcap:autonum="7. ">

In the results list, select the applicable report(s). As you select each report, it is moved to the Selected list.

</div>

<div class="NumList2" madcap:conditions="Default.NewUI" madcap:autonum="8. ">

To remove all selected reports, click **Clear All**.

</div>

<div class="NumList2" madcap:autonum="9. ">

Click **Save**. The Canned Reports window displays and the selected canned report(s) are shown in the list. By default, the list is sorted by the *Created On* date.

</div>

<div class="NumList2" madcap:autonum="10. ">

To edit a canned report, refer to [Run Report](Reports-Run.htm) and follow the instructions under *Generate/Re-generate Report*.

</div>
