# Request Model

Use this function to configure and populate Request Models that can be reused for defining Incidents. It can be used to define commonly occurring incidents raised by Service Desk Members, thus enabling to skip and reduce the time required to complete forms.


Request Models can be configured with relevant Role based access.


In the main window, click **Admin > Request Management > Request Model**. The Request Model window displays.

New Request Model

From the *Select Actions* drop-down list, choose **New Request Model**. The Add Request Model window displays.

Click **New**. The New Request Model window displays.
