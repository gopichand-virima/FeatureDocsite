# Blueprints: Comment

Use this function to add, edit and delete comments.

![arrow border](../Resources/Images/arrow-border.PNG)

[[[Closed]{.image .placeholder .MCDropDown_Image_Icon original-image-src="../../Skins/Default/Stylesheets/Images/transparent.gif" original-image-title="" height="12" width="16" a Comment](javascript:void(0)){.MCDropDownHotSpot .dropDownHotspot .MCDropDownHotSpot_ .MCHotSpotImage aria-expanded="false"}]{.MCDropDownHead .dropDownHead}

 -- ----- -----------------------------
 1. Click the **Comments** tab.
 -- ----- -----------------------------

 -- ----- ------------------------------------------------
 2. In the Comments window, click **New Comment**.
 -- ----- ------------------------------------------------

[[[Closed]{.image .placeholder .MCDropDown_Image_Icon original-image-src="../../Skins/Default/Stylesheets/Images/transparent.gif" original-image-title="" height="12" width="16" a Comment](javascript:void(0)){.MCDropDownHotSpot .dropDownHotspot .MCDropDownHotSpot_ .MCHotSpotImage aria-expanded="false"}]{.MCDropDownHead .dropDownHead}

 -- ----- -----------------------------
 1. Click the **Comments** tab.
 -- ----- -----------------------------

 -- ----- -----------------------------
 2. Locate the comment to edit.
 -- ----- -----------------------------

 -- ----- --------------------------------------------------------
 3. Click the **pencil** icon. The comment field displays.
 -- ----- --------------------------------------------------------
