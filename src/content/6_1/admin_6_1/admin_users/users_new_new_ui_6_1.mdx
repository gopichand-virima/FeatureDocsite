# New Users

ADD INFO FOR V5.

Use this function to add new users to the application.

|     |                                                                                  |
|-----|----------------------------------------------------------------------------------|
|     | Through this function Multi-factor Authentication (MFA) can be enabled/disabled. |

<div class="numlist1" madcap:autonum="1. ">

In the main window, click the **gear** icon and select **Users /> Users**.

</div>

<div class="Ind50">

<span class="image placeholder" original-image-src="../Resources/Images/Admin%20-%20Users/icon-users.PNG" original-image-title=""></span>

</div>

<div class="Ind50">

The Access Management window displays.

</div>

<div class="Ind50">

<span class="image placeholder" original-image-src="../Resources/Images/Admin%20-%20Users/access-management-new.PNG" original-image-title="" style="width: 624px;height: 141px;"></span>

</div>

<div class="NumList2" madcap:autonum="2. ">

Click **New User**.

</div>

<div class="Ind50">

<span class="image placeholder" original-image-src="../Resources/Images/Admin%20-%20Users/new-user-upper-new.PNG" original-image-title="" style="width: 616px;height: 273px;"></span>  
<span class="image placeholder" original-image-src="../Resources/Images/Admin%20-%20Users/new-user-lower-new.PNG" original-image-title="" style="width: 624px;height: 179px;"></span>  

</div>

<div class="NumList2" madcap:autonum="3. ">

To complete the fields, refer to the information shown in the table below.

</div>

<div class="NumList2" madcap:autonum="4. ">

When all selections/entries are made, click **Add**.

</div>
