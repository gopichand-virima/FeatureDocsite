# Creating a Scheduled Meraki Import

1. From the **Select Actions** dropdown, choose **New Scheduled Meraki Import**.

2. Configure:

**Name:** Enter a descriptive name.

**Credential:** Choose the Meraki credential.

**Import Frequency:** Specify the frequency.

**Active:** Enable the schedule.

**Timezone and Timing:** Set the execution time.

1. Click **Add**.
