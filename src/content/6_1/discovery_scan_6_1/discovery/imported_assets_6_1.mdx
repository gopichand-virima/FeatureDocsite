# Imported Assets

Use this function to view the results of data imported during the data files import operation and move them (if applicable) to CMDB. An asset can also be rescanned through this function.

<div class="NumList1" madcap:autonum="1. ">

In the navigation pane, select **Discovery Scan /> Imported Assets**. The Imported Assets window displays.

</div>

<div class="Ind50">

<span class="image placeholder" original-image-src="../Resources/Images/Discovery/imported-assets.png" original-image-title="" madcap:conditions="Default.OldUI">A screenshot of a computer Description automatically generated</span>

</div>

<div class="Ind50">

<span class="image placeholder" original-image-src="../Resources/Images/Discovery/imported-assets-main-new.png" original-image-title=""></span>

</div>

<div class="NumList2" madcap:autonum="2. ">

Select at least one or more records. (When rescanning, the asset must have an IP address.)

</div>

<div class="NumList2" madcap:conditions="Default.OldUI" madcap:autonum="3. ">

From the *Select Actions* drop-down list, choose either [Move to CMDB](Move-CMDB.htm) or <a href="../Common%20Functions/Re-scan.htm" target="_popup">Re-Scan</a>.

</div>

<div class="NumList2" madcap:conditions="Default.NewUI" madcap:autonum="4. ">

From the *Select Actions* drop-down list, choose either [Move to CMDB](Move-CMDB.htm) or <a href="../Common%20Functions/Re-scan.htm" target="_popup">Re-Scan</a>.

</div>

<div class="NumList2" madcap:conditions="Default.NewUI" madcap:autonum="5. ">

At the prompt, click **Yes**.

</div>
