# FAQs

Use this function to view a list of FAQs entered through the Knowledge Records function.

In the main window, select **ITSM \> Knowledge Management \> FAQs**. The FAQ\'s window displays.

![faqs main old](../../Resources/Images/ITSM/Knowledge Management/faqs-main-old.PNG)

When a line item is selected, the FAQ window displays and shows details about the FAQ.
