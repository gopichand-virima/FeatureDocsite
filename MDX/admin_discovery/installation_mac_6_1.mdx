# Installing and Running the MAC Application

This function is not operational.

The Mac application can performUnix System DiscoveryandNetwork Device Discoveryin a Network.

Once the application is downloaded, follow the below installation steps.

- Unzipthe **DiscoveryAgent\_MAC.zip** file.
- On the terminal,run:sudo bash install.sh
- Double-clickvirimamacagent.workflow.
- In the *Service Installer dialog box*, clickInstall.
- From the menu, select **Finder > Services > virimamacagent**.
