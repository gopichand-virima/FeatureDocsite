# About Asset Management

The Asset Management module provides comprehensive tools for tracking and managing IT and non-IT assets throughout their lifecycle. This section covers the administrative configuration options available for asset management.

Asset Management in Virima enables organizations to:

- Configure asset types and classifications
- Define asset lifecycle states
- Manage asset relationships and dependencies
- Set up depreciation and financial tracking
- Configure asset compliance rules
- Manage software licenses
- Track asset maintenance and warranties
- Control asset disposal processes

## Key Configuration Areas

The following administrative functions are available for Asset Management:

- **Asset Types**: Define hardware, software, and other asset categories
- **Asset States**: Configure lifecycle states
- **Asset Fields**: Customize asset attributes
- **Depreciation Rules**: Set up financial depreciation
- **License Management**: Track software licenses
- **Asset Relationships**: Define CI relationships
- **Compliance Rules**: Configure compliance checks
- **Disposal Policies**: Manage asset retirement

Each of these areas can be configured to match your organization's asset management strategy and compliance requirements.

## Asset Categories

**Hardware Assets**:

- Servers and infrastructure
- Network devices
- End-user devices
- Peripherals

**Software Assets**:

- Applications and licenses
- Operating systems
- Development tools
- Cloud subscriptions

**Non-IT Assets**:

- Office equipment
- Furniture
- Vehicles
- Facilities

## Asset Lifecycle

**Ordered**: Asset requested or purchased

**Received**: Asset delivered

**In Stock**: Asset in inventory

**Deployed**: Asset assigned to user

**In Use**: Asset actively used

**In Repair**: Asset being serviced

**Retired**: Asset decommissioned

**Disposed**: Asset removed from inventory
