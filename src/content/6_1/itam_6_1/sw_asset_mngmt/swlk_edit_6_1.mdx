# Edit Software License Key

Use this function to edit the configured values for a software license key.

 -- ----- ---------------------------------------------------------------------------------------------------------------------------------------------
 1. From the navigation pane, select **ITAM \> Software Asset Management \> Software License Keys**. The Software License Keys window displays.
 -- ----- ---------------------------------------------------------------------------------------------------------------------------------------------

 -- ----- --------------------------
 2. Select a record to edit.
 -- ----- --------------------------

 -- ----- ------------------------------------------
 3. Modify any of the following information:
 -- ----- ------------------------------------------

::: Bullet2
[Details](SWLK-Details.mdx){.MCTopicPopup .MCTopicPopupHotSpot .a}

::: Bullet2
[Tasks](/itsm/release_mngmt/tasks_6_1.mdx){.MCTopicPopup .MCTopicPopupHotSpot .a}

::: Bullet2
[Comments](/discovery/comments_6_1.mdx){.MCTopicPopup .MCTopicPopupHotSpot .a}

::: Bullet2
[Attachments](/discovery/attachments_6_1.mdx){.MCTopicPopup .MCTopicPopupHotSpot .a}

 -- ----- -------------------------------------------------------
 4. When all selections/entries are made, click **Save**.
 -- ----- -------------------------------------------------------
