# Azure AD Configuration and Import

[Azure Active Directory (Azure AD)]{.Strong} is Microsoft\'s cloud-based identity and access management service. It authenticates and authorizes users and devices, manages user identities, and controls access to cloud and on-premises applications and resources.

In the main window, click [Admin \> Users \> Azure AD Configuration. ]{.Strong}The Azure AD Configuration window displays.

![Azure AD User Import Logs](../../Resources/Images/Discovery/Azure-AD-Users-Import-Logs/Azure AD User Import Logs.png)

1. From the **Select Actions** drop-down list, choose [New Azure AD Configuration.]{.Strong}\
 ![Azure AD User Import Logs 1](../../Resources/Images/Discovery/Azure-AD-Users-Import-Logs/Azure AD User Import Logs_1.png)\

2. On the Configuration tab, fill in the values for the attributes.

- [Name]{.Strong} -- Provide a configuration name.

- [Client ID]{.Strong} -- Enter the Azure application client ID.

- [Tenant ID]{.Strong} -- Enter the Azure tenant ID.

- [Secret Key ]{.Strong}-- Provide the authentication method (secret key).\
 ![Azure AD User Import Logs 2](../../Resources/Images/Discovery/Azure-AD-Users-Import-Logs/Azure AD User Import Logs_2.png)\

- [Certificate]{.Strong} -- Provide the authentication method (certificate with private key).\
 ![Azure AD User Import Logs 3](../../Resources/Images/Discovery/Azure-AD-Users-Import-Logs/Azure AD User Import Logs_3.png)
