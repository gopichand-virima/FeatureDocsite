# Accessing the Import Data Files Page

1. From the left navigation panel, navigate to [Discovery Scan]{.Strong}. Click [Import Data Files]{.Strong}.

2. The **Import Data Files** page opens.

 ![Import Data Files 1 1](../../Resources/Images/Discovery/Import-Data-Files/Import Data Files_1_1.png)

3. Display the list of uploaded files with these columns:

 - [File Name]{.Strong} -- The name of the uploaded file.

 - [Status]{.Strong}[ -- Indicates processing state (e.g., ]{.inline-comment-marker}*Processed*[, ]{.inline-comment-marker}*Processing*[).]{.inline-comment-marker}

 - [Processing]{.Strong}: The CI(s) in the uploaded file are currently being imported. (Import in Progress)

 - [Processed]{.Strong}: The system successfully imported the CIs from the uploaded file.

 - [Uploaded By]{.Strong} -- Identifies the user who uploaded the file.

 - [Uploaded On]{.Strong} -- The date and time when the file was uploaded.

4. Use the checkboxes to select one or more files for bulk actions.
