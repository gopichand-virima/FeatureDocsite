# Priority Calculator

Use this function to set values for **Urgency** and **Impact**.

|     |                                                       |
|-----|-------------------------------------------------------|
|     | Role access must be given to perform these functions. |

In the main window, click **Admin /> Release Management /> Release Priority Calculator**. The Release Priority Calculator window displays.

In the main window, click the **gear** icon and select **Release Management /> Release Priority Calculator.**
