# Add New Property

Use this function to add a new field under a selected property group. The added property is added to the selected CI's blueprint. *Refer to [Details - Add Property](../ITSM/Config%20Mngmt/Config%20Items/Config%20Item%20Details/Details-Add-Property.htm).*

|     |                                                                                                                             |
|-----|-----------------------------------------------------------------------------------------------------------------------------|
|     | Depending on which option you select--*Add Manually* or *Import from Excel*--the dialog box displays with different fields. |

<div class="numlist1" madcap:conditions="Default.OldUI" madcap:autonum="1. ">

In the main window, click **Admin /> SACM /> Add New Property**. The Add New Property dialog box displays. By default, *Add Manually* is selected.

</div>

<div class="Ind50" madcap:conditions="Default.OldUI">

<span class="image placeholder" original-image-src="../Resources/Images/Admin%20-%20SACM/add-new-property.png" original-image-title="" style="width: 428px;height: 369px;"></span>

</div>

<div class="NumList1" madcap:conditions="Default.NewUI" madcap:autonum="1. ">

In the main window, click the **gear** icon and select **SACM /> Add New Property**. The Add New Property dialog box displays. By default, *Add Manually* is selected.

</div>

<div class="Ind50" madcap:conditions="Default.NewUI">

<span class="image placeholder" original-image-src="../Resources/Images/Admin%20-%20SACM/add-new-property-new-ui.PNG" original-image-title="" style="width: 378px;height: 495px;"></span>

</div>

<div class="NumList2" madcap:autonum="2. ">

Refer to the instructions below for manually adding or importing a property.

</div>

Add Manually

<div class="NumList1" madcap:autonum="1. ">

In the Add New Property dialog box, click **Add Manually** and complete the fields according to the table below.

</div>

<div class="NumList2" madcap:autonum="2. ">

When all selections/entries are made, click **Add**.

</div>

<div class="NumList2" madcap:autonum="3. ">

If the information is valid, a success message displays. Click **OK**.

</div>

Import from Excel
