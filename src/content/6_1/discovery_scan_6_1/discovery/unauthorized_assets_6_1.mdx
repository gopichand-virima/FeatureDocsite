# Unauthorized Assets

Use this function to view assets that were scanned, discovered or imported but are not present inside the CMDB.

<div class="NumList2" madcap:autonum="1. ">

In the navigation pane, select **Discovery Scan /> Discovered Items**. The Discovered Items window displays.

</div>

<div class="Ind50" madcap:conditions="Default.NewUI">

<span class="image placeholder" original-image-src="../Resources/Images/Discovery/discovered-items-main-new.PNG" original-image-title=""></span>

</div>

<div class="Ind50" madcap:conditions="Default.OldUI">

<span class="image placeholder" original-image-src="../Resources/Images/Discovery/discovered-items-main-old.PNG" original-image-title=""></span>

</div>

<div class="NumList2" madcap:autonum="2. ">

Click the *Filter By* drop-down and select **New**. The list updates and lists all assets that can be moved to the CMDB.

</div>

<div class="Ind50" madcap:conditions="Default.OldUI">

<span class="image placeholder" original-image-src="../Resources/Images/Discovery/discovered-items-main-unauthorized-old.PNG" original-image-title=""></span>

</div>

<div class="Ind50">

<span class="image placeholder" original-image-src="../Resources/Images/Discovery/discovered-items-main-unauthorized-new.PNG" original-image-title="" madcap:conditions="Default.NewUI"></span>

</div>

<div class="NumList2" madcap:autonum="3. ">

Select a record, then from the *Select Actions* drop-down list, select **Move to CMDB**.

</div>

<div class="NumList2" madcap:autonum="4. ">

When the Alert Message displays, click **Continue**. A message displays indicating the record is being moved, and the line item for the record being moved is grayed out. Once the record is moved, it no longer displays in the list.

</div>
