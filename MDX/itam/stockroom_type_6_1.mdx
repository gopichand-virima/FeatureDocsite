# Stockroom Type

Use this function to record and group different stock types available across the organization. The type of stock may be physical, raw materials, software, or virtual items.

In the navigation pane, select **ITAM > Stockroom Type**. The Stockroom Type page displays.

!

!


### New Stockroom Type

- From the *Select Actions* drop-down list, choose **New Stockroom Type**.

!

- Enter a *Display Name* and *Description*.
- Enter the *Value* (price) of the stockroom inventory.


The value should be updated on a set interval so that the value is as accurate as possible.


- Enter the *Priority* of the stockroom. This helps determine when the stockroom should be cleared of inventory.
- If shipping is required to another location, select *Is Shipment Required*.
- If the corresponding stockroom is from an external vendor/place, select *Is External Stockroom*.
- Click **Add**.


### Edit Stockroom Type

Not all functions include an option to edit record line items shown in the main window.


- Navigate to the window containing the record to edit.
- Click the line containing the record. The applicable window or dialog box displays.
- Make the necessary modifications.
- Click **Save** or **Add**, as applicable.


### Delete Stockroom Type

Deleting is a permanent action and cannot be undone. Deleting may affect other functionality and information in the application such as data in configured reports, fields in windows, selectable options, etc. Therefore, be sure to understand the potential effects before making a deletion.


- Click the line that contains the item to delete.
- From the *Select Actions* drop-down list, choose **Delete**. If a confirmation message is displayed, type **Delete** and click **Delete**to confirm.
