# Edit Blueprints

Use this function to edit the details for a blueprint.

In the Blueprints window, select a record to edit. The applicable record displays.

Modify any of the following information:

- [Details](/admin_sacm/blueprints_details_ng.mdx)
- [SLA](/admin_sacm/blueprints_sla_ng.mdx)
- [Tasks](/admin_sacm/blueprints_tasks_ng.mdx)
- [Comments](/admin_sacm/blueprints_comments_ng.mdx)
- [Attachments](/admin_sacm/blueprints_attachments_ng.mdx)

When all selections/entries are made, click **Save**.Â
