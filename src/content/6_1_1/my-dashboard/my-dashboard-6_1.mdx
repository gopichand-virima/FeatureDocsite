# My Dashboard

My Dashboard is your personalized workspace within Virima that provides quick access to items assigned to you, recent activities, notifications, and frequently used actions. It serves as your daily operational hub for managing work and staying informed.

## Overview

Unlike the broader Dashboards module that allows creation of multiple custom dashboards, "My Dashboard" is a dedicated personal view that automatically aggregates information relevant to you based on your user profile, role, and current assignments.

## Key Features

- **Personal Work Queue**: View all items assigned to you across modules
- **Activity Feed**: Track recent changes and updates to items you're watching
- **Quick Actions**: Access frequently used functions without navigation
- **Notifications Center**: See important alerts and system notifications
- **Recent Items**: Quick access to recently viewed or modified records
- **Favorites**: Pin frequently accessed items for quick reference

## Configuring My Dashboard

### Personalization Options

**Layout Configuration**
1. Click **Customize** button on My Dashboard
2. Enable/disable sections as needed:
   - Show/hide My Work Items
   - Show/hide Team Activity
   - Show/hide Notifications
   - Show/hide Quick Actions
   - Show/hide Recent Activity
3. Arrange sections in preferred order
4. Save configuration

**Widget Preferences**
Configure individual sections:
- Set number of items to display
- Define default sort order
- Configure filters
- Set refresh intervals

**Display Options**
- Compact view (more items, less detail)
- Standard view (balanced)
- Detailed view (fewer items, more information)
- Card view or list view

### Filter Settings

**My Work Items Filters**
- Priority: Critical, High, Medium, Low, All
- Status: Open, In Progress, Pending, All
- Time: Today, This Week, This Month, All
- Module: ITSM, CMDB, Asset Management, All

**Activity Feed Filters**
- Activity type: Created, Updated, Commented, Closed
- Time range: Last hour, Today, This week, Custom
- Users: Me only, My team, All
- Modules: Filter by specific modules

**Notification Filters**
- Notification type: Assignments, Updates, Alerts, All
- Read status: Unread only, All
- Priority: High priority, All
- Time: Today, This week, All time

## Troubleshooting

### Work Items Not Displaying
**Problem**: My work queue is empty but you have assignments  
**Solution**:
- Check filter settings (may be filtering out items)
- Verify you're logged in with correct user account
- Refresh the dashboard
- Check module permissions
- Contact administrator if issue persists

### Notifications Not Appearing
**Problem**: Not receiving expected notifications  
**Solution**:
- Review notification preferences
- Check notification filters
- Verify email notification settings
- Check spam/junk folders for emails
- Ensure browser notifications are enabled

### Dashboard Loading Slowly
**Problem**: My Dashboard takes long time to load  
**Solution**:
- Reduce number of items displayed per section
- Increase auto-refresh interval
- Clear browser cache
- Disable unused dashboard sections
- Check network connectivity

### Missing Recent Items
**Problem**: Recently accessed items not showing  
**Solution**:
- Check recent items time filter
- Verify browser isn't blocking cookies
- Clear and rebuild recent items cache
- Check privacy/incognito mode settings
