# [Ignore Process]{style="background-color: #ffffff;"}

Use this function to ignore back-end processes when creating relationships.

 -- ----- --------------------------------------------------------------------------------------------------------------------
 1. On the main window, select **Admin \> Discovery \> Ignore Process**. The Ignore Process Names dialog box displays.
 -- ----- --------------------------------------------------------------------------------------------------------------------

::: Ind50
[]{.image .placeholder original-image-src="../Resources/Images/Admin%20-%20Discovery/ignore-process-names_387x121.png" original-image-title=""

 -- ----- -------------------------------------------------------------------------------------------
 2. Enter the name(s) of the process to ignore. Separate multiple process names with a comma.
 -- ----- -------------------------------------------------------------------------------------------

 -- ----- ----------------
 3. Click **Add**.
 -- ----- ----------------

 -- ----- -------------------------------------------
 4. In the confirmation window, click **OK**.
 -- ----- -------------------------------------------
