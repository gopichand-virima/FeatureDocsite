# CI: Relationships

Use this function to manage the relationships between sources and targets. The Discovery application detects the Connected To, Runs On, Communicates With, Installed On, Component Of, Physically Connected To, Attaches and Used By relationships during a scan.

 --- -----------------------------------------------------------------------
   Refer to [CMDB Process Jobs](CMDB-Process-Jobs.mdx) for more details.
 --- -----------------------------------------------------------------------

 -- ----- ----------------------------------------------------------------------------------------------------------------------------------------
 1. From the navigation pane, select **ITSM \> Configuration Management \> Configuration Items**. The Configuration Items window displays.
 -- ----- ----------------------------------------------------------------------------------------------------------------------------------------

 -- ----- --------------------------------------------------------------------------------------------------------------------------------
 1. From the navigation pane, select **Configuration Management \> Configuration Items**. The Configuration Items window displays.
 -- ----- --------------------------------------------------------------------------------------------------------------------------------

 -- ----- -----------------------------------------------------------------------------------
 2. Select a record in the list. A new window opens and the **Details** tab displays.
 -- ----- -----------------------------------------------------------------------------------

 -- ----- ---------------------------------------------------------------------------------
 3. Click the **Relationships** tab. A list of all existing relationships displays.
 -- ----- ---------------------------------------------------------------------------------

::: Ind50
![add relationship new 620x167](../../../Resources/Images/ITSM/add-relationship-new_620x167.png)**.
 -- ----- -----------------------------------------------------------------------------------------------------------------------------------

 -- ----- --------------------------------------------------------------------------
 5. To remove any relationship, select it in the list, and click **Delete**.
 -- ----- --------------------------------------------------------------------------

## Add Source or Add Target

 -- ----- ------------------------------------------------------------------------------------------
 1. Click the **Add Source** or **Add Target button**. The Add Relation dialog box displays.
 -- ----- ------------------------------------------------------------------------------------------

::: Ind50
![add relation dialog box new 421x390](../../../Resources/Images/ITSM/add-relation-dialog-box-new_421x390.png).
 -- ----- ------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------

 -- ----- ------------------------------------------------------------------------------------------------------------------------------------------------------
 5. From the *Choose Relationship* drop-down list, select the relationship the Source or Target will have such as Assigned To, Belongs To, and so forth.
 -- ----- ------------------------------------------------------------------------------------------------------------------------------------------------------

 -- ----- -------------------------------------------------------------------------------------------------------------------------------------------
 6. From the *Relation* drop-down list, select the relationship the Source or Target will have such as Assigned To, Belongs To, and so forth.
 -- ----- -------------------------------------------------------------------------------------------------------------------------------------------

 -- ----- ----------------------------------------------
 7. When all selections are made, click **Add**.
 -- ----- ----------------------------------------------
