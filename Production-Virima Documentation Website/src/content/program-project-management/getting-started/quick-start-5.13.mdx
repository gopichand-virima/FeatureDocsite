# Program and Project Management - Quick Start Guide

Start managing IT projects effectively.

## Prerequisites

- Project management module access
- Understanding of project scope
- Resource availability information
- Budget allocation details

## Quick Start Steps

### Step 1: Create a Project
1. Navigate to Program and Project Management
2. Click "New Project"
3. Enter project details:
   - Project name and description
   - Start and end dates
   - Project manager
   - Budget allocation
4. Save the project

### Step 2: Define Project Structure
1. Create project phases
2. Add tasks and subtasks
3. Set task dependencies
4. Define milestones
5. Set task durations

### Step 3: Assign Resources
1. Add team members to project
2. Assign tasks to resources
3. Set resource allocation percentages
4. Review resource workload
5. Adjust assignments as needed

### Step 4: Track Progress
1. Update task completion status
2. Log actual hours worked
3. Update task progress percentages
4. Review project health dashboard
5. Generate status reports
