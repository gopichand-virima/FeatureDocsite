# Edit or Delete Probes

Use this function to edit or delete the configured values of an existing probe.

## Edit Probe

<div class="numlist1" madcap:conditions="Default.Omit" madcap:autonum="1. ">

In the main window, select **Admin /> Discovery /> Probes**. The Probes window displays.

</div>

<div class="NumList2" madcap:conditions="Default.NewUI" madcap:autonum="2. ">

In the main window, click the **gear** icon and select **Discovery /> Probes**.  The Probes window displays.

</div>

<div class="NumList2" madcap:autonum="3. ">

Select a record to edit. The *Details* information displays.

</div>

<div class="NumList2" madcap:autonum="4. ">

Make changes, as necessary, to any of the available fields.

</div>

<div class="NumList2" madcap:autonum="5. ">

Modify any of the following information:

</div>

</div>

<div class="Bullet2">

<a href="../Common%20Topics/Comments.htm" target="_popup">Comments</a>

</div>

<div class="Bullet2">

</div>

<div class="NumList2" madcap:autonum="6. ">

</div>
