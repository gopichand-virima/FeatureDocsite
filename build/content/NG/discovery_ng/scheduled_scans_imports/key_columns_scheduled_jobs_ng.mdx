# Key Columns in the Scheduled Jobs List

- **Name** – The name assigned to the scheduled job.
- **Type** – Indicates whether it's a Scan (e.g., Deep Host Scan) or an Import (e.g., AWS Import, Azure Import).
- **Frequency** – How often the job runs (e.g., Every Day, Weekly, Monthly).
- **Active** – Whether the job is currently enabled (checked = active).
- **Last Run** – Timestamp of the most recent execution.
- **Next Run** – Scheduled date and time for the next execution.
- **Created By** – The user who created the schedule.
- **Created On** – Date and time when the schedule was created.
