# About Release Management

Use this function to manage the release and deployment of IT services.

Release Management aims to plan, schedule and control the build, test and deployment of releases, and to deliver new functionality required by the business while protecting the integrity of existing services.

Release Management includes:

- Planning releases
- Building and configuring release packages
- Testing releases
- Deploying releases
- Reviewing and closing releases
