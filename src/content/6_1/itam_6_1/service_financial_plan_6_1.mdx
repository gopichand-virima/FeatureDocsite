# Service Financial Plan

[Use this function to]{style="background-color: #ffff00;"}

In the navigation pane, select **ITAM \> Financial Management \> Service Financial Plan**.

![service financial plan main old](../Resources/Images/ITAM/service-financial-plan-main-old.PNG)

[[[Closed]{.image .placeholder .MCDropDown_Image_Icon original-image-src="../../Skins/Default/Stylesheets/Images/transparent.gif" original-image-title="" height="12" width="16" Service Financial Plan](javascript:void(0)){.MCDropDownHotSpot .dropDownHotspot .MCDropDownHotSpot_ .MCHotSpotImage aria-expanded="false"}]{.MCDropDownHead .dropDownHead}

 -- ----- ---------------------------------------------------------------------------------------------
 1. From the *Select Actions* drop-down list, choose **New Service Financial Management Plan.**
 -- ----- ---------------------------------------------------------------------------------------------

 -- ----- -----------------------------------
 2. Enter a *Name* and *Description*.
 -- ----- -----------------------------------

 -- ----- -----------------------------
 3. Enter the *Record Details*.
 -- ----- -----------------------------

::: Ind50
![sfp record details old](../Resources/Images/ITAM/sfp-record-details-old.PNG)

 -- ----- --------------------------------
 4. Enter the *Financial Details*.
 -- ----- --------------------------------

::: Ind50
![sfp financial details](../Resources/Images/ITAM/sfp-financial-details.PNG)

 -- ----- ----------------------------------------------------------------------------------------------------------------------------------------------------
 5. To enter a *Service Catalog Item*, click **Add**. Then, in the Service Catalog item dialog box, search for and select the applicable service item.
 -- ----- ----------------------------------------------------------------------------------------------------------------------------------------------------

 -- ----- ------------------------------------------------------
 6. When all selections/entries are made, click **Add**.
 -- ----- ------------------------------------------------------

[[[Closed]{.image .placeholder .MCDropDown_Image_Icon original-image-src="../../Skins/Default/Stylesheets/Images/transparent.gif" original-image-title="" height="12" width="16" Service Financial Plan](javascript:void(0)){.MCDropDownHotSpot .dropDownHotspot .MCDropDownHotSpot_ .MCHotSpotImage aria-expanded="false"}]{.MCDropDownHead .dropDownHead}

 --- -----------------------------------------------------------------------------------------
   Not all functions include an option to edit record line items shown in the main window.
 --- -----------------------------------------------------------------------------------------

 -- ----- -------------------------------------------------------
 1. Navigate to the window containing the record to edit.
 -- ----- -------------------------------------------------------

 -- ----- -------------------------------------------------------------------------------------
 2. Click the line containing the record. The applicable window or dialog box displays.
 -- ----- -------------------------------------------------------------------------------------

 -- ----- -----------------------------------
 3. Make the necessary modifications.
 -- ----- -----------------------------------

 -- ----- -------------------------------------------
 4. Click **Save** or **Add**, as applicable.
 -- ----- -------------------------------------------

[[[Closed]{.image .placeholder .MCDropDown_Image_Icon original-image-src="../../Skins/Default/Stylesheets/Images/transparent.gif" original-image-title="" height="12" width="16" Service Financial Plan](javascript:void(0)){.MCDropDownHotSpot .dropDownHotspot .MCDropDownHotSpot_ .MCHotSpotImage aria-expanded="false"}]{.MCDropDownHead .dropDownHead}

 --- --------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------
   Deleting is a permanent action and cannot be undone. Deleting may affect other functionality and information in the application such as data in configured reports, fields in windows, selectable options, etc. Therefore, be sure to understand the potential effects before making a deletion.
 --- --------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------

 -- ----- --------------------------------------------------
 1. Click the line that contains the item to delete.
 -- ----- --------------------------------------------------

 -- ----- ------------------------------------------------------------------------------------------------------------------------------------------------------
 2. From the *Select Actions* drop-down list, choose **Delete**. If a confirmation message is displayed, type **Delete** and click **Delete**to confirm.
 -- ----- ------------------------------------------------------------------------------------------------------------------------------------------------------
