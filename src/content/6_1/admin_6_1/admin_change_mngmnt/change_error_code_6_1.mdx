# Change Error Code

Use this function to define an error message to display in case of incorrect configuration scenario.

In the main window, click **Admin /> Problem Management /> Problem Error Code**. The Change Error Code window displays.

<span class="image placeholder" original-image-src="../Resources/Images/Admin%20-%20Problem/problem-error-code-main-old.PNG" original-image-title=""></span>

In the main window, click the **gear** icon and select **Problem Management /> Problem Error Code**.

<span class="image placeholder" original-image-src="../Resources/Images/Admin%20-%20Change/icon-change-error-code.PNG" original-image-title=""></span>

The Problem Error Code window displays.

<span class="image placeholder" original-image-src="../Resources/Images/Admin%20-%20Problem/problem-error-code-main-new.PNG" original-image-title=""></span>

New Problem Error Code

<div class="NumList1" madcap:conditions="Default.OldUI" madcap:autonum="1. ">

<div class="NumList1" madcap:conditions="Default.NewUI" madcap:autonum="1. ">

</div>
