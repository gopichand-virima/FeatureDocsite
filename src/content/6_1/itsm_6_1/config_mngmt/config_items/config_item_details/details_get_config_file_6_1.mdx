# Get Config File

Use this function to extract the configuration file of an Installed Software or Software Instance. The extracted configuration file is stored under the **Attachments** tab of the Installed Software or Software Instance configuration item.

|     |                                                                                                                                                                                                                   |
|:---:|-------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
|     | To extract the configuration file: (1) the Discovery application must be running; (2) the configuration file name should be specified in the Major Software; and (3) a Parent configuration item must be defined. |

<div class="NumList1" madcap:autonum="1. ">

In the navigation pane, select **ITSM /> Configuration Management /> CMDB**. The Configuration Items window displays.

</div>

<div class="NumList2" madcap:autonum="2. ">

Select the record for which you want to add a detail property. A new window displays.

</div>

<div class="NumList2" madcap:autonum="3. ">

From the *Select Actions* drop-down list, select **Get Config File**. The Get Config File dialog box displays.

</div>

<div class="NumList2" madcap:autonum="4. ">

In the *Choose Client* field, click the drop-down list, and select the client from which you want to obtain the configuration file.

</div>

<div class="NumList2" madcap:autonum="5. ">

To view the client's status, click the **check here** link is selected. The Client window displays and shows the client's state (either true or false).

</div>
