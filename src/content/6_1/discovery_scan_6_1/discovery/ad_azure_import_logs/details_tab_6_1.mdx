# Details Tab

When you select a specific record from the <span class="Strong" style="font-weight: bold;">AD User Import Logs</span>, the system opens the <span class="Strong" style="font-weight: bold;">Details view</span>. This page provides a full summary of the chosen AD import activity.

<span class="Strong" style="font-weight: bold;">Key Information displayed</span>:

- <span class="Strong" style="font-weight: bold;">Name</span>: A user-defined name for the AD configuration.

- <span class="Strong" style="font-weight: bold;">Type of Import</span>: Specifies whether the import was for <span class="Emphasis">All Users</span> or a specific filtered group.

- <span class="Strong" style="font-weight: bold;">Triggered At</span>: The exact date and time when the import process started.

- <span class="Strong" style="font-weight: bold;">Completed At</span>: Timestamp showing when the import was successfully finished (blank if still in progress).

- <span class="Strong" style="font-weight: bold;">Time Taken</span>: Total duration of the import job.

- <span class="Strong" style="font-weight: bold;">Edited Users</span>: Number of existing user records updated during this import.

- <span class="Strong" style="font-weight: bold;">New Users</span>: Number of users added as part of the import.

- <span class="Strong" style="font-weight: bold;">Total Users</span>: The complete number of user records processed.

- <span class="Strong" style="font-weight: bold;">Description</span>: A text field for system notes or administrator comments about the import (e.g., “Imported successfully” or “No changes detected”).
