# Discovery Schedule

Use this function to create and manage automated discovery schedules.

In the main window, click **Admin > Discovery > Discovery Schedule**. The Discovery Schedule window displays.

## New Discovery Schedule

From the *Select Actions* drop-down list, choose **New Discovery Schedule**. The New Discovery Schedule window displays.

Enter a *Name* and *Description* for this schedule.

Configure the schedule settings:

- **Frequency**: Select how often the discovery should run (Hourly, Daily, Weekly, Monthly, or Custom Cron Expression)
- **Start Date/Time**: Specify when the schedule should begin
- **End Date/Time**: (Optional) Specify when the schedule should end
- **Is Active**: Enable or disable the schedule

Select the discovery profile(s) to run on this schedule.

When all selections/entries are made, click **Add**.

## Edit Discovery Schedule

Navigate to the window containing the record to edit.

Click the line containing the record. The applicable window displays.

Make the necessary modifications.

Click **Save**.

## Delete Discovery Schedule

Click the line that contains the item to delete.

From the *Select Actions* drop-down list, choose **Delete**.
