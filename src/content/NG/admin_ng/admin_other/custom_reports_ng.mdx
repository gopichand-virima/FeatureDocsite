# Custom Reports

Use this function to create reports using required customized options of DB by the user. It also allows for publishing of reports to other user types.

In the main window, click **Admin \> Others \> Custom Reports**.

![custom reports main old](../Resources/Images/Admin - Other/custom-reports-main-old.PNG)

![arrow border](../Resources/Images/arrow-border.PNG)

[[[Closed]{.image .placeholder .MCDropDown_Image_Icon original-image-src="../../Skins/Default/Stylesheets/Images/transparent.gif" original-image-title="" height="12" width="16" Custom Report](javascript:void(0)){.MCDropDownHotSpot .dropDownHotspot .MCDropDownHotSpot_ .MCHotSpotImage aria-expanded="false"}]{.MCDropDownHead .dropDownHead}

 -- ----- ---------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------
 1. From the *Select Actions* drop-down list, choose **New Custom Report**.![custom reports main 6.1](../Resources/Images/Admin - Other/custom-reports-main-6.1.PNG)
 -- ----- ---------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------

 -- ----- ----------------------------------------------------
 2. Complete the fields, referring to the table below.
 -- ----- ----------------------------------------------------

 -- ----- -----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------
 3. To view the report results based upon the parameters and values configured, click **Run Query and Get Sample Results**. The results are displayed in the lower portion of the window. ![custom reports run query 6.1](../Resources/Images/Admin - Other/custom-reports-run-query-6.1.png)
 -- ----- -----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------

 -- ----- ------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------
 4. To format the reports, click **Configure Presentation Details**. The lower portion of the window updates and the presentation parameters and values can be entered. ![custom reports Configure presentation 6.1](../Resources/Images/Admin - Other/custom-reports-Configure-presentation-6.1.png)
 -- ----- ------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------

 -- ----- ---------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------
 5. To define the values for a graph, click **Select Graph**. ![custom reports graph 6.1](../Resources/Images/Admin - Other/custom-reports-graph-6.1.png)
 -- ----- ---------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------

 -- ----- ----------------------------------------------------------------------------------
 a. Enter the presentation details. Fields with a drop-down list show valid options.
 -- ----- ----------------------------------------------------------------------------------

 -- ----- -----------------------------------------------------------------------------
 b. Click **View Sample Graph**. The window expands and the graph is displayed.
 -- ----- -----------------------------------------------------------------------------

[[[Closed]{.image .placeholder .MCDropDown_Image_Icon original-image-src="../../Skins/Default/Stylesheets/Images/transparent.gif" original-image-title="" height="12" width="16" Custom Report](javascript:void(0)){.MCDropDownHotSpot .dropDownHotspot .MCDropDownHotSpot_ .MCHotSpotImage aria-expanded="false"}]{.MCDropDownHead .dropDownHead}

 --- -----------------------------------------------------------------------------------------
   Not all functions include an option to edit record line items shown in the main window.
 --- -----------------------------------------------------------------------------------------

 -- ----- -------------------------------------------------------
 1. Navigate to the window containing the record to edit.
 -- ----- -------------------------------------------------------

 -- ----- -------------------------------------------------------------------------------------
 2. Click the line containing the record. The applicable window or dialog box displays.
 -- ----- -------------------------------------------------------------------------------------

 -- ----- -----------------------------------
 3. Make the necessary modifications.
 -- ----- -----------------------------------

 -- ----- -------------------------------------------
 4. Click **Save** or **Add**, as applicable.
 -- ----- -------------------------------------------

[[[Closed]{.image .placeholder .MCDropDown_Image_Icon original-image-src="../../Skins/Default/Stylesheets/Images/transparent.gif" original-image-title="" height="12" width="16" Custom Report](javascript:void(0)){.MCDropDownHotSpot .dropDownHotspot .MCDropDownHotSpot_ .MCHotSpotImage aria-expanded="false"}]{.MCDropDownHead .dropDownHead}

 --- --------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------
   Deleting is a permanent action and cannot be undone. Deleting may affect other functionality and information in the application such as data in configured reports, fields in windows, selectable options, etc. Therefore, be sure to understand the potential effects before making a deletion.
 --- --------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------

 -- ----- --------------------------------------------------
 1. Click the line that contains the item to delete.
 -- ----- --------------------------------------------------

 -- ----- ------------------------------------------------------------------------------------------------------------------------------------------------------
 2. From the *Select Actions* drop-down list, choose **Delete**. If a confirmation message is displayed, type **Delete** and click **Delete**to confirm.
 -- ----- ------------------------------------------------------------------------------------------------------------------------------------------------------
