# Azure VMSS Instances

<div class="NumList1" madcap:autonum="1. ">

While viewing an asset record, click **Components**.

</div>

<div class="NumList2" madcap:autonum="2. ">

Click **Azure VMSS Instances**.

</div>

<div class="Ind50">

<span class="image placeholder" original-image-src="../../../Resources/Images/Components/azure-vmss-instances-new.png" original-image-title="" style="width: 489px;height: 145px;"></span>

## Add a New Azure VMSS Instance

<div class="NumList1" madcap:autonum="1. ">

Click **Add**. The Azure VMSS Instance dialog box displays.

</div>

<div class="Ind50">

<span class="image placeholder" original-image-src="../../../Resources/Images/Components/azure-vmss-instances-add-new.png" original-image-title="" style="width: 429px;height: 136px;"></span>

</div>

<div class="NumList2" madcap:autonum="2. ">

Click **New Azure VMSS Instance.**

</div>

<div class="NumList2" style="font-style: normal;" madcap:autonum="3. ">

Enter the applicable field information.

</div>

<div class="NumList2" madcap:autonum="4. ">

When all selections/entries are made, click **Save**.

</div>

## Modify Existing Azure VMSS Instance

<div class="NumList1" madcap:autonum="1. ">

Click **Existing Azure VMSS Instance**.

</div>

<div class="Ind50">

<span class="image placeholder" original-image-src="../../../Resources/Images/Components/azure-vmss-instances-existing-new.png" original-image-title="" style="width: 382px;height: 260px;"></span>

</div>

<div class="NumList2" madcap:autonum="2. ">

Enter the search criteria to locate the existing device.  *See also* <a href="../../../Common%20Topics/Advanced-Search.htm" style="font-style: italic;">Advanced Search</a>*.*

</div>

<div class="NumList2" madcap:autonum="3. ">

Select the applicable device from the results list.

</div>

<div class="NumList2" madcap:autonum="4. ">

When all selections are made, click **Save**.

</div>

## Delete Azure VMSS Instance

While viewing the list of existing devices, select the device to delete, and click the **Delete** button.
