# Add Dashboard Contents

Use this function to add content (reports) to the dashboard such as Ad Hoc Reports, Calendars (change or outage), KPIs, and so forth. There is no limit on the number of reports that can be added.

|     |                                                                                                                                                                                                                    |
|-----|--------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
|     | The list at the top of the window cannot be changed or removed as this information displays data collected throughout the application. However, the details for a line item can be viewed by clicking on the item. |

## Add Contents

<div class="NumList2" madcap:autonum="1. ">

On the currently displayed dashboard, click **Add Contents**. The Add Contents pane displays.

</div>

<div class="Ind50">

<span class="image placeholder" original-image-src="../Resources/Images/My%20Dashboard/add-contents-search-window-new.png" original-image-title="" style="width: 199px;height: 273px;"></span>

</div>

<div class="NumList2" madcap:autonum="2. ">

Click the drop-down list and select a content type--for example, Ad hoc Reports, KPIs, and so forth.

</div>

<div class="NumList2" madcap:autonum="3. ">

If additional fields are displayed, provide the details for those fields. For example, if Ad hoc Reports is selected, enter search criteria in the field (such as a few letters of the report name), then select the applicable report from the list.

</div>

<div class="Ind50">

<span class="image placeholder" original-image-src="../Resources/Images/My%20Dashboard/example-ad-hoc-rpt.PNG" original-image-title=""></span>

</div>

<div class="NumList2" madcap:autonum="4. ">

After a selection is made, click **Add Now**. The dashboard window updates and the new content displays.

</div>

## Delete Contents

<div class="NumList1" madcap:autonum="1. ">

In the upper right corner of the item, click the three dots (...).

</div>

<div class="Ind50">

<span class="image placeholder" original-image-src="../Resources/Images/My%20Dashboard/example-delete-contents.PNG" original-image-title="" style="width: 676px;height: 343px;"></span>

</div>

<div class="NumList2" madcap:autonum="2. ">

From the drop-down list, select **Delete**.

</div>

<div class="NumList2" madcap:autonum="3. ">

At the prompt, click **Continue**.

</div>
