# Azure VMSS Instances

 --- -------------------------------------------------------------------------------------------------------------
   This function is only available with certain asset types. Therefore, the relevant tab may not be displayed.
 --- -------------------------------------------------------------------------------------------------------------

 -- ----- ------------------------------------------------------
 1. While viewing an asset record, click **Components**.
 -- ----- ------------------------------------------------------

 -- ----- ---------------------------------
 2. Click **Azure VMSS Instances**.
 -- ----- ---------------------------------

::: Ind50
![azure vmss instances new 489x145](../../../Resources/Images/Components/azure-vmss-instances-new_489x145.png)

## Add a New Azure VMSS Instance

 -- ----- -------------------------------------------------------------
 1. Click **Add**. The Azure VMSS Instance dialog box displays.
 -- ----- -------------------------------------------------------------

::: Ind50
![azure vmss instances add new 429x136](../../../Resources/Images/Components/azure-vmss-instances-add-new_429x136.png)

 -- ----- ------------------------------------
 2. Click **New Azure VMSS Instance.**
 -- ----- ------------------------------------

 -- ----- -----------------------------------------
 3. Enter the applicable field information.
 -- ----- -----------------------------------------

 -- ----- -------------------------------------------------------
 4. When all selections/entries are made, click **Save**.
 -- ----- -------------------------------------------------------

## Modify Existing Azure VMSS Instance

 -- ----- -----------------------------------------
 1. Click **Existing Azure VMSS Instance**.
 -- ----- -----------------------------------------

::: Ind50
![azure vmss instances existing new 382x260](../../../Resources/Images/Components/azure-vmss-instances-existing-new_382x260.png)

 -- ----- ----------------------------------------------------------------------------------------------------------------------------------------------------------------------
 2. Enter the search criteria to locate the existing device.  *See also* [Advanced Search](../../../Common%20Topics/Advanced-Search.mdx){style="font-style: italic;"}*.*
 -- ----- ----------------------------------------------------------------------------------------------------------------------------------------------------------------------

 -- ----- -----------------------------------------------------
 3. Select the applicable device from the results list.
 -- ----- -----------------------------------------------------

 -- ----- -----------------------------------------------
 4. When all selections are made, click **Save**.
 -- ----- -----------------------------------------------

## Delete Azure VMSS Instance

While viewing the list of existing devices, select the device to delete, and click the **Delete** button.
