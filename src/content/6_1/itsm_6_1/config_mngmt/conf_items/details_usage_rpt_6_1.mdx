# Software Usage Report

Use this function to view software usage metric information for software configured as *major* software.

 -- ----- ---------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------
 1. From the navigation pane, select **ITSM \> Configuration Management \> Configuration Items**. The Configuration Items window displays. [MISSING v5 DETAILS AS THERE IS NO DATA IN THE UI.]{style="color: #dc143c;"
 -- ----- ---------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------

 -- ----- --------------------------------------------------------------------------------------------------------------------------------
 2. From the navigation page, select **Configuration Management \> Configuration Items**. The Configuration Items window displays.
 -- ----- --------------------------------------------------------------------------------------------------------------------------------

::: {.Ind50
![sw usage report new 634x156](../../../Resources/Images/Admin - SACM/sw-usage-report-new_634x156.png)

 -- ----- ------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------
 3. From the drop-down list, select the time period for which you want to see usage (such as last 24 hours or a custom range). The window updates and displays all records that match the date range criteria.
 -- ----- ------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------

 -- ----- -------------------------------------------------------------
 4. Select the record for which you want to view usage details.
 -- ----- -------------------------------------------------------------

 -- ----- -------------------------------------------------------------------------------------------------
 5. To export the details, click **Export**. A confirmation email is sent when the report is ready.
 -- ----- -------------------------------------------------------------------------------------------------

 -- ----- -------------------------------------------------------------------------------------------------------------------------------------------------------
 6. To view the details for a report, click the line item containing the report. The window updates and displays all the details for the selected report.
 -- ----- -------------------------------------------------------------------------------------------------------------------------------------------------------

::: {.Ind50
![sw usage report details new 622x174](../../../Resources/Images/Admin - SACM/sw-usage-report-details-new_622x174.png)

 -- ----- ----------------------------------------------------------------------------------
 7. To view the configuration information for the report, click the CI Id hyperlink.
 -- ----- ----------------------------------------------------------------------------------
