# Installing the Application

1. **Run the Installer:**

  - Right-click the installer and select **Run as administrator**.

2. **Windows SmartScreen Warning:**

  - If prompted, select **More info**, and then select **Run anyway** to continue the installation.

![Installing%20the%20Application 1](../Resources/Images/Admin%20-%20Discovery/Installing%20the%20Application_1.png)

1. Click **Next** to proceed.

![Installing%20the%20Application 2](../Resources/Images/Admin%20-%20Discovery/Installing%20the%20Application_2.png)

1. Click **Install** to begin installation.

![Installing%20the%20Application 3](../Resources/Images/Admin%20-%20Discovery/Installing%20the%20Application_3.png)

1. Click **Skip** if prompted.

![Installing%20the%20Application 4](../Resources/Images/Admin%20-%20Discovery/Installing%20the%20Application_4.png)

1. Click **Finish** to complete the installation.

![Installing%20the%20Application 5](../Resources/Images/Admin%20-%20Discovery/Installing%20the%20Application_5.png)

1. **Launch the Application:**

  - Find the **Virima Discovery Application** icon on your desktop.
  - Right-click and select **Run as administrator**.

![Installing%20the%20Application 6](../Resources/Images/Admin%20-%20Discovery/Installing%20the%20Application_6.png)
