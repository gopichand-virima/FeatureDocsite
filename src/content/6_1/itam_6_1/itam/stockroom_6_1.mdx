# Stockroom

Use this function to define a place and create a record for the existing stock across the organization. It also facilitates designated users to check on the stock and its type.

The benefits of stockroom configuration include the following:

<div class="Bullet1">

Recording and ordering off of stock and its information.

</div>

<div class="Bullet1">

Easy integration with the finance department.

</div>

<div class="Bullet1">

Better forecasting and planning.

</div>

<div class="Bullet1">

Easy report generation.

</div>

|     |                                                                                     |
|-----|-------------------------------------------------------------------------------------|
|     | A [Stockroom Type](Stockroom-Type.htm) should be created before adding a stockroom. |
