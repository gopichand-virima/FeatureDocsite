# Discovery Agents

<span class="span_1">Use this function to view the total number of discovery agents installed for the corresponding Client, remotely install a new discovery agent, or restart the client.</span>

<div class="NumList1" madcap:conditions="Default.OldUI" madcap:autonum="1. ">

In the main window, click **Admin /> Discovery /> Client**. The Client window displays.

</div>

<div class="NumList1" madcap:conditions="Default.NewUI" madcap:autonum="1. ">

In the main window, click the **gear** icon and select **Discovery /> Client**. The Client window displays.

</div>

<div class="NumList2" madcap:autonum="2. ">

Select the client for which you want manage discovery agents. A new window displays with the **Details** tab displayed.

</div>

<div class="NumList2" madcap:autonum="3. ">

Click the **Discovery Agents** tab.

</div>

</div>

<div class="emdash1" madcap:conditions="Default.NewUI" madcap:autonum="- ">

**Schedule**. To schedule a scan on the remote client, click **Schedule** and specify a time period.

</div>

<div class="emdash1" madcap:conditions="Default.NewUI" madcap:autonum="- ">

**Delete Schedule**. To delete the schedule, click **Delete Schedule**.

</div>

<div class="emdash1" madcap:conditions="Default.NewUI" madcap:autonum="- ">

**Uninstall**. To uninstall or remove, select an agent, click **Uninstall**, then click **Continue**.

</div>
