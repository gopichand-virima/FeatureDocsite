# Custom Reports

Use this function to create reports using required customized options of DB by the user. It also allows for publishing of reports to other user types.

In the main window, click **Admin /> Others /> Custom Reports**.

<span class="image placeholder" original-image-src="../Resources/Images/Admin%20-%20Other/custom-reports-main-old.PNG" original-image-title=""></span>

New Custom Report

<div class="NumList1" madcap:autonum="1. ">

From the *Select Actions* drop-down list, choose **New Custom Report**.<span class="image placeholder" original-image-src="../Resources/Images/Admin%20-%20Other/custom-reports-main-6.1.PNG" original-image-title=""></span>

</div>

<div class="NumList2" madcap:autonum="2. ">

Complete the fields, referring to the table below.

</div>

<div class="NumList2" madcap:autonum="3. ">

To view the report results based upon the parameters and values configured, click **Run Query and Get Sample Results**. The results are displayed in the lower portion of the window. <span class="image placeholder" original-image-src="../Resources/Images/Admin%20-%20Other/custom-reports-run-query-6.1.png" original-image-title=""></span>

</div>

<div class="NumList2" madcap:autonum="4. ">

To format the reports, click **Configure Presentation Details**. The lower portion of the window updates and the presentation parameters and values can be entered. <span class="image placeholder" original-image-src="../Resources/Images/Admin%20-%20Other/custom-reports-Configure-presentation-6.1.png" original-image-title=""></span>

</div>

<div class="NumList2" madcap:autonum="5. ">

To define the values for a graph, click **Select Graph**. <span class="image placeholder" original-image-src="../Resources/Images/Admin%20-%20Other/custom-reports-graph-6.1.png" original-image-title=""></span>

</div>

<div class="NumLista" madcap:autonum="a. ">

Enter the presentation details. Fields with a drop-down list show valid options.

</div>

<div class="numListb" madcap:autonum="b. ">

Click **View Sample Graph**. The window expands and the graph is displayed.

</div>

<div class="Ind75" madcap:conditions="Default.NewUI">

<span class="image placeholder" original-image-src="../Resources/Images/Admin%20-%20Other/custom-reports-sample-graph-old.PNG" original-image-title=""></span>

</div>

<div class="NumList2" madcap:autonum="6. ">

When all actions are complete, click **Add** (when creating a new report) or **Save** (when modifying an existing report).

</div>
