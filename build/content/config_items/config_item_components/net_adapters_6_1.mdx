# Network Adapters

Click **Network Adapters**.

! Add Network Adapter

Click **Add**. The Network Adapter dialog box displays.

! !

Click **New Network Adapter.**

Enter the *MAC Address*, *Network Adapter Name*, *IP Address*, and *Asset Names*. These are all required fields.

Click **Add**.

Edit Network Adapter

Click **Existing Network Adapter**.

! !

Enter the search criteria to locate the existing adapter.  *See also* [Advanced Search](/common_topics/advanced_search_6_1.mdx)*.*

Select the applicable adapter from the results list.

When all selections are made, click **Save**.

Edit the Network Adapter Details

While viewing the list of existing adapters, select the adapter to edit. The windows refreshes and the details for the adapter displays.

Make any necessary changes to the existing information. The changes are automatically saved.

To add a contact, navigate to the Contacts section and click **Add**. The Contacts dialog box displays.

! !

In the *Choose Relation* field, click the drop-down list and select how this contact should be related to this asset.

In the search field, enter the search criteria.

Select a contact from the results list, moving it to the selected list.

When all selections are made, click **Save**.
