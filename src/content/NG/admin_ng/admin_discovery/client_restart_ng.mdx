# Restart Client

In order to restart a client, the state must be set to "true."


From the main window, select **Admin > Discovery > Client.**

In the Client window, select the applicable client.  

Click the **Discovery Agents** tab.

From the *Select Actions* drop-down list, choose **Restart Client**.

In the Confirmation dialog box, click **Yes** (to restart) or **No** (to cancel the action).
