# Virima SaaS Compatibility Matrix

This document outlines the system requirements, supported browsers, database compatibility, and integration support for Virima SaaS deployments.

## Overview

Virima is delivered as a cloud-based SaaS solution. The information below details the compatibility requirements for optimal performance and functionality.

---

## Supported Browsers

Virima SaaS supports the following modern web browsers. We recommend using the latest stable versions for the best experience.

| Browser | Minimum Version | Recommended |
|---------|-----------------|-------------|
| Google Chrome | 90+ | Latest |
| Mozilla Firefox | 88+ | Latest |
| Microsoft Edge | 90+ | Latest |
| Safari | 14+ | Latest |

> **Note:** Internet Explorer is not supported. For optimal performance, we recommend using Google Chrome.

---

## Operating System Support

### Client Operating Systems

| Operating System | Supported Versions |
|------------------|-------------------|
| Windows | Windows 10, Windows 11 |
| macOS | 11.0 (Big Sur) and later |
| Linux | Ubuntu 20.04+, RHEL 8+ |

### Server Operating Systems (For Discovery Probes)

| Operating System | Supported Versions |
|------------------|-------------------|
| Windows Server | 2016, 2019, 2022 |
| Linux | RHEL 7/8/9, CentOS 7/8, Ubuntu 18.04/20.04/22.04 |

## Discovery Probe Requirements

### Hardware Requirements

| Component | Minimum | Recommended |
|-----------|---------|-------------|
| CPU | 2 cores | 4 cores |
| RAM | 4 GB | 8 GB |
| Disk Space | 20 GB | 50 GB |

### Network Requirements

| Requirement | Details |
|-------------|---------|
| Outbound HTTPS | Port 443 to Virima SaaS endpoints |
| DNS Resolution | Required for target discovery |
| Network Access | Access to target subnets for scanning |
