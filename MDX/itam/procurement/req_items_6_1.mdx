# Requested Items

Use this function to fetch the requested item the requester raises to complete a required task or process.

In the navigation pane, select **ITAM > Procurement > Requested Item**. The Requested Items page displays.

!

!


### New Requested Items

- From the *Select Actions* drop-down list, choose **New Requested Item**. The Requested Items window displays.

!

- Complete the fields, referring to the information below.
- **Requested Item Number**. Enter the number of this item.
- **Configuration Item**. Click **Add**, then search for and select the applicable requested item.
- **Opened By**. Click **Add**, then search for and select the applicable user.
- **Requested For**. Click **Add**, then search for and select the user requesting the item.
- **Opened on Date**. Enter the date and time when the ITAM requested item should be opened.
- **Opened on Due Date**. Enter the date and time by which the ITAM requested item must be delivered.
- **Stage**. Click the drop-down list and select the stage for the request, such as Awaiting Delivery, Approved, etc.
- **Quantity**. Enter the number of items for this request.
- **Catalog Item**. Enter the catalog number for the requested item.
- **Estimated Delivery Date**. Enter the date/time by which the item is to be delivered.
- **Back ordered**. Select if the item is on back order.
- When all selections/entries are made, click **Add**.


### Edit Requested Item

Not all functions include an option to edit record line items shown in the main window.


- Navigate to the window containing the record to edit.
- Click the line containing the record. The applicable window or dialog box displays.
- Make the necessary modifications.
- Click **Save** or **Add**, as applicable.

When editing an existing item, other options may be available, such as adding [Tasks](/itsm/release_mngmt/tasks_6_1.mdx), [Comments](/discovery/comments_6_1.mdx), and [Attachments](/discovery/attachments_6_1.mdx) and viewing [History](/common_topics/history_6_1.mdx).


The options change based upon the function; therefore, not every option may be shown.


### Delete Requested Item

Deleting is a permanent action and cannot be undone. Deleting may affect other functionality and information in the application such as data in configured reports, fields in windows, selectable options, etc. Therefore, be sure to understand the potential effects before making a deletion.


- Click the line that contains the item to delete.
- From the *Select Actions* drop-down list, choose **Delete**. If a confirmation message is displayed, type **Delete** and click **Delete**to confirm.
