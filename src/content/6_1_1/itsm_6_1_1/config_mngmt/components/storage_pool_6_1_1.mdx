# Storage Pool

While viewing an asset record, click **Components**.

Click **Storage Pool**

## Add a New Storage Pool

Click **Add**. The Storage Pool dialog box displays.

Click **New Storage Pool**

Enter the applicable information.

When all selections/entries are made, click **Save.**

## Modify Existing Storage Pool

Click **Existing Storage Pool**.

Enter the search criteria to locate the existing device.  *See also* [Advanced Search](/common_topics/advanced_search_6_1_1_1.mdx)*.*

Select the applicable device from the results list.

When all selections are made, click **Save**.

## Delete Storage Pool

While viewing the list of existing devices, select the device to delete, and click the **Delete** button.
