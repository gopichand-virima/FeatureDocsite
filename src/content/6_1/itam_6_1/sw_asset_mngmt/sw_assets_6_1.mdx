# Software Assets

Use this function to add and edit software assets. There are two components to this function: (1) adding a software asset group, and (2) adding the software asset itself.

<div class="NumList1" madcap:autonum="1. ">

In the navigation pane, select **ITAM /> Software Asset Management /> Software Assets.** The window displays the list of all configured software assets.

</div>

<div class="NumList2" madcap:autonum="2. ">

Click the radio button for the preferred display type: **By Group** or **By Software**.

</div>

<span id="SWAssetGroup"></span>New Software Asset By Group

<div class="NumList1" madcap:autonum="1. ">

Click the **By Group** radio button.

</div>

<div class="NumList2" madcap:conditions="Default.OldUI" madcap:autonum="2. ">

From the *Select Actions* drop-down list, choose **New Software Group**.

</div>

<div class="Ind50" madcap:conditions="Default.OldUI">

<span class="image placeholder" original-image-src="../../Resources/Images/ITAM/sw-group-new-old.PNG" original-image-title=""></span>

The New Software Group window displays.

</div>

<div class="Ind50" madcap:conditions="Default.OldUI">

<span class="image placeholder" original-image-src="../../Resources/Images/ITAM/sw-group-add-old.PNG" original-image-title=""></span>

</div>

<div class="NumList2" madcap:conditions="Default.NewUI" madcap:autonum="3. ">

Click **New**. The New Software Group window displays.

</div>

<div class="Ind50" madcap:conditions="Default.NewUI">

<span class="image placeholder" original-image-src="../../Resources/Images/ITAM/sw-group-new-new.PNG" original-image-title="" style="width: 802px;height: 202px;"></span>

</div>

<div class="NumList2" madcap:autonum="4. ">

Enter a unique, alphanumeric *Group Name*.

</div>

<div class="NumList2" madcap:conditions="Default.OldUI" madcap:autonum="5. ">

Click **Add**, then search for and select the applicable *Vendor* and *Software License Key*.

</div>

<div class="NumList2" madcap:conditions="Default.NewUI" madcap:autonum="6. ">

Click the **plus + sign**, then search for and select the applicable *Vendor* and *Software License Key*.

</div>

|     |                                                                                                                                                                                                                                                                                         |
|-----|-----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
|     | Both a Vendor and Software License Key must be configured in order to choose them from the list. *Refer to* <a href="../Vendor-Mngmnt.htm" style="font-style: italic;">Vendor Management</a> *and* <a href="SW-License-Key.htm" style="font-style: italic;">Software License Key</a>*.* |

<div class="NumList2" madcap:autonum="7. ">

Click **Add**.

</div>

<span id="SWAssetSoftware"></span>New Software Asset By Software

<div class="NumList1" madcap:autonum="1. ">

Click the **By Software** radio button.

</div>

<div class="NumList2" madcap:autonum="2. ">

Click **New**. The New dialog box displays.

</div>

<div class="Ind50" madcap:conditions="Default.OldUI">

<span class="image placeholder" original-image-src="../../Resources/Images/ITAM/sw-asset-new-by-software-old.PNG" original-image-title=""></span>

</div>

<div class="Ind50" madcap:conditions="Default.NewUI">

<span class="image placeholder" original-image-src="../../Resources/Images/ITAM/sw-asset-new-by-software-new.PNG" original-image-title=""></span>

</div>

<div class="NumList2" madcap:autonum="3. ">

Enter an *Asset Name*, *Software Name*, and *Manufacturer*.

</div>

<div class="NumList2" madcap:autonum="4. ">

In the *Group* field, click the drop-down list and select the applicable group.

</div>

|     |                                                                                                                                                      |
|-----|------------------------------------------------------------------------------------------------------------------------------------------------------|
|     | There must be an asset Software Group created in order to have it available in the list (see previous section). Selecting a *Group* is not required. |

<div class="NumList2" madcap:autonum="5. ">

Click **Add**.

</div>

<div class="NumList2" madcap:autonum="6. ">

When returned to the list of software assets, select the asset just added. A new window displays in which to specify more information about this software asset, such as the software license key, components, relationships, SLA, etc. *Refer to Software Asset Details.*

</div>

Edit Software Asset

In the Details window, click **Related Software** to view a list of all software related to this software asset.
