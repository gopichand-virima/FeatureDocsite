# <span class="Strong">Business Service Map</span>

- The <span class="Strong">Business Service Map</span> provides a <span class="Strong">pictorial representation</span> of Configuration Items (CIs) and their interrelationships.

- It complements the <span class="Strong">Relationships</span> tab by offering a <span class="Strong">visual insight</span> into how CIs connect and contribute to service delivery.

  <span class="image placeholder" original-image-src="../Resources/Images/CMDB%20-%20Left-Nav/Configuration%20Management%20Database_56.png" original-image-title=""></span>

<!-- -->

- Upon clicking the BSM view:

  <span class="image placeholder" original-image-src="../Resources/Images/CMDB%20-%20Left-Nav/Configuration%20Management%20Database_57.png" original-image-title=""></span>

<span class="Strong" style="font-weight: bold;">Available Views:</span>

|                                                                  |                                                                                                                                      |
|------------------------------------------------------------------|--------------------------------------------------------------------------------------------------------------------------------------|
| <span class="Strong" style="font-weight: bold;">View Type</span> | <span class="Strong" style="font-weight: bold;">Purpose</span>                                                                       |
| Infrastructure View                                              | Visualizes servers and applications, showing their relationships.                                                                    |
| Service Topology View                                            | Displays the complete runtime structure of a business service and its supporting components.                                         |
| Communication View                                               | Shows CIs connected through <span class="Emphasis">Communicates With</span> relationships, ideal for mapping traffic paths.          |
| ADM View                                                         | Visualizes application interactions via <span class="Emphasis">Communicates With</span> links, typically used in ADM-based analysis. |
| Network View                                                     | Presents physical and virtual network relationships among devices and services.                                                      |
| Affected Business Application View                               | Highlights business services or applications affected by an impacted CI, helping assess service impact.                              |
| Cloud Topology View                                              | Focuses on representing cloud infrastructure, if available.                                                                          |

At the bottom of the view, <span class="Strong">color-coded legends</span> distinguish relationship types such as:

- Installed On

- Assigned To

# <span class="Strong">BSM Actions Tab:</span>

The <span class="Strong" style="font-weight: bold;">Actions</span> dropdown in the Business Service Map (BSM) provides on-demand operations to enhance real-time visibility and management of the selected Configuration Item (CI). These options are located at the top right of the BSM interface.

<span class="Strong" style="font-weight: bold;">Available Options:</span>

|                                            |                                                                              |
|--------------------------------------------|------------------------------------------------------------------------------|
| <span class="Strong">Action</span>         | <span class="Strong" style="font-weight: bold;">Purpose</span>               |
| <span class="Strong">Rescan</span>         | Initiates a discovery rescan for the selected CI to collect the latest data. |
| <span class="Strong">Host Check</span>     | Performs a basic connectivity and credential validation check on the host.   |
| <span class="Strong">View Snapshots</span> | (If enabled) Opens previous snapshots of the BSM to view historical changes. |
