# Access CMDB

- From the ITSM module, open <span class="Strong">Configuration Management</span>, then select <span class="Strong">CMDB</span>.

  <span class="image placeholder" original-image-src="../Resources/Images/CMDB%20-%20Left-Nav/Configuration%20Management%20Database.png" original-image-title=""></span>

<!-- -->

- This section provides access to the Configuration Management Database, where you can view, manage, and update configuration items and their relationships.

  <span class="image placeholder" original-image-src="../Resources/Images/CMDB%20-%20Left-Nav/Configuration%20Management%20Database_1.png" original-image-title=""></span>

<span class="Strong">Virima CMDB Column Details </span>

|                                                |                                                                                               |
|------------------------------------------------|-----------------------------------------------------------------------------------------------|
| Column Name                                    | Short Description                                                                             |
| <span class="Strong">Blueprint Name</span>     | CI type or template (e.g., Windows Server, Network Device) that defines available properties. |
| <span class="Strong">Asset Name</span>         | Name for easy identification of the asset.                                                    |
| <span class="Strong">Status</span>             | Current state of the CI (e.g., New, Proposed, Ordered, Purchased).                            |
| <span class="Strong">Asset ID</span>           | Unique identifier for the CI within the CMDB.                                                 |
| <span class="Strong">Host Name</span>          | Network host name (DNS/NetBIOS) of the asset.                                                 |
| <span class="Strong">Operating System</span>   | OS installed on the asset (e.g., Windows 10).                                                 |
| <span class="Strong">Missing Components</span> | Indicates if the CI has Missing Components (e.g., true/false).                                |
| <span class="Strong">IP Address</span>         | Primary IP address assigned to the asset.                                                     |
| <span class="Strong">Created On</span>         | Date and time when the CI was added to the CMDB.                                              |
| <span class="Strong">Jira Sync</span>          | Indicates if the CI has been synced with Jira (e.g., true/false).                             |
