# Request Closure Code

Use this function to configure closure codes with descriptions for the Closed state in the Request Management function.

In the main window, click **Admin > Request Management > Request Closure Code**. The Request Closure Code window displays.

## New Request Closure Code

From the *Select Actions* drop-down list, choose **New Request Closure Code**. The New Request Closure Code window displays.

Enter a *Name* and *Description*.

Select the **Fulfillment Type**:

- Fulfilled
- Completed
- Approved and Delivered
- Rejected
- Cancelled
- Duplicate
- Unable to Fulfill

Click **Add**.

## Edit Request Closure Code

Navigate to the window containing the record to edit.

Click the line containing the record. The applicable window displays.

Make the necessary modifications.

Click **Save**.

## Delete Request Closure Code

Click the line that contains the item to delete.

From the *Select Actions* drop-down list, choose **Delete**.
