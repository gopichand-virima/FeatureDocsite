# Network Interfaces

Click **Network Interfaces**.

! Add Network Interface

Click **Add**. The Network Interface dialog box displays.

Click **New Network Interface.**

Enter the applicable field information.

When all selections/entries are made, click **Save**.

Edit Network Interface

Click **Existing Network Interface**.

Enter the search criteria to locate the existing device.  *See also* [Advanced Search](/common_topics/advanced_search_ng.mdx)*.*

Select the applicable device from the results list.

When all selections are made, click **Save**.
