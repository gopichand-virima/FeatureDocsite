# Self Service Overview

Self Service portal empowers end users to submit requests, access knowledge base articles, and resolve common issues without IT intervention.

## Key Features

- User-friendly service catalog
- Knowledge base access
- Request tracking
- Automated workflows
- Mobile accessibility
