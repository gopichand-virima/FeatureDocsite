# Kubernetes Nodes

<div class="NumList1" madcap:autonum="1. ">

While viewing an asset record, click **Components**.

</div>

<div class="NumList2" madcap:autonum="2. ">

Click **Kubernetes Nodes**.

</div>

<div class="Ind50">

<span class="image placeholder" original-image-src="../../../../Resources/Images/Components/kubernetes-nodes-new.png" original-image-title="" style="width: 487px;height: 128px;"></span>

Add Kubernetes Node

<div class="NumList1" madcap:autonum="1. ">

Click **Add**. The Kubernetes Nodes dialog box displays.

</div>

<div class="Ind50">

<span class="image placeholder" original-image-src="../../../../Resources/Images/Components/kubernetes-nodes-add-new.png" original-image-title="" style="width: 374px;height: 118px;"></span>

</div>

<div class="NumList2" madcap:autonum="2. ">

Click **New Kubernetes Node**

</div>

<div class="NumList2" style="font-style: normal;" madcap:autonum="3. ">

Enter the applicable information in each field.

</div>

<div class="NumList2" madcap:autonum="4. ">

When all entries are made, click **Save.**

</div>

Edit Kubernetes Node

<div class="NumList1" madcap:autonum="1. ">

Click **Existing Kubernetes Node**.

</div>

<div class="Ind50">

<span class="image placeholder" original-image-src="../../../../Resources/Images/Components/kubernetes-namespaces-existing-new.png" original-image-title="" style="width: 377px;height: 257px;"></span>

</div>

<div class="NumList2" madcap:autonum="2. ">

Enter the search criteria to locate the existing device*.*

</div>

<div class="NumList2" madcap:autonum="3. ">

Select the applicable device from the results list.

</div>

<div class="NumList2" madcap:autonum="4. ">

When all selections/entries are made, click **Save**.

</div>
