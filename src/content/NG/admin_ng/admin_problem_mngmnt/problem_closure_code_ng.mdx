# Problem Closure Code

Use this function to configure closure codes with descriptions for the Closed state in the Problem Management function.

In the main window, click **Admin > Problem Management > Problem Closure Code**. The Problem Closure Code window displays.

## New Problem Closure Code

From the *Select Actions* drop-down list, choose **New Problem Closure Code**. The New Problem Closure Code window displays.

Enter a *Name* and *Description*.

Select the **Resolution Type**:

- Root Cause Eliminated
- Workaround Implemented
- Known Error Documented
- Duplicate
- Not a Problem
- Cannot Reproduce

Click **Add**.

## Edit Problem Closure Code

Navigate to the window containing the record to edit.

Click the line containing the record. The applicable window displays.

Make the necessary modifications.

Click **Save**.

## Delete Problem Closure Code

Click the line that contains the item to delete.

From the *Select Actions* drop-down list, choose **Delete**.
