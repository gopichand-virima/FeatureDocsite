# New Incident

Use this function to create a new incident.


Roles with relevant permissions as per the Role Access of Incident can View/Create/Edit/Delete/Resolve an Incident.


All the fields and its values (which are pre-defined) can be customized. - HOW?

From the navigation pane, select **ITSM > Incident Management**. The Incident Management window displays.
