# SLA: Attachments

Use this function to add attachments for a service level agreement.

- In the main window, select **Admin > Other > Service Level Agreement**.
- Select the record to manage.
- Follow the instructions below for the applicable action.

!


### Add Attachment

- Click theAttachmentstab.
- Click**New Attachment**. The *Attach File dialog box* displays.

!

- In the *Upload Document* field, clickBrowseto select the applicable file.
- In the *Description* field, enter information about the attachment.
- Click**Upload**. When you return to the Attachment window, the added item displays.


### Download Attachment

- Click theAttachmentstab.
- Locate the row containing the attachment to download.
- Click the **Download** icon. The attachment downloads.


The process for downloading the attachment is based upon your individual system settings, such as the selected browser.


### Delete Attachment

Deleting is a permanent action and cannot be undone. Deleting may affect other functionality and information in the application such as data in configured reports, fields in windows, selectable options, etc. Therefore, be sure to understand the potential effects before making a deletion.


- Click theAttachmentstab.
- Click the **Delete** icon.
- In the Confirmation window, Type **Delete** and click **Delete**.
