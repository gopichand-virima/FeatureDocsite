# Peripheral Devices

 --- -------------------------------------------------------------------------------------------------------------
   This function is only available with certain asset types. Therefore, the relevant tab may not be displayed.
 --- -------------------------------------------------------------------------------------------------------------

 -- ----- ------------------------------------------------------
 1. While viewing an asset record, click **Components**.
 -- ----- ------------------------------------------------------

 -- ----- -------------------------------
 2. Click **Peripheral Devices**.
 -- ----- -------------------------------

::: {.Ind50
![peripheral devices 470x87](../../../Resources/Images/Components/peripheral-devices_470x87.png)

::: Ind50
![peripheral device add 427x120](../../../Resources/Images/Components/peripheral-device-add_427x120.png)

::: Ind50
![peripheral device existing 476x164](../../../Resources/Images/Components/peripheral-device-existing_476x164.png)

::: Ind50
![peripheral device existing new 473x322](../../../Resources/Images/Components/peripheral-device-existing-new_473x322.png)*.*
 -- ----- ----------------------------------------------------------------------------------------------------------------------------------------------------------------------

 -- ----- -----------------------------------------------------
 3. Select the applicable device from the results list.
 -- ----- -----------------------------------------------------

 -- ----- -----------------------------------------------
 4. When all selections are made, click **Save**.
 -- ----- -----------------------------------------------

## View and Edit the Peripheral Device Details

 -- ----- --------------------------------------------------------------------------------------------------------------------------------------------------------------------
 1. While viewing the list of existing devices, select the device to edit. The windows refreshes and the details for the device displays. These details are view only.
 -- ----- --------------------------------------------------------------------------------------------------------------------------------------------------------------------

 -- ----- -----------------------------
 2. To view the supporting CIs:
 -- ----- -----------------------------

 -- ----- --------------------
 a. Click the CI item.
 -- ----- --------------------

::: Ind75
![supporting cis](../../../Resources/Images/Components/supporting-cis.PNG)

::: Ind75
The Configuration Items window displays.

 -- ----- ----------------------------------------------------------------------------------------------------
 b. Click **Components**. *Refer to* [CI Components](CI-Comp-List.mdx){style="font-style: italic;"}*.*
 -- ----- ----------------------------------------------------------------------------------------------------

 -- ----- ---------------------------------------------------------------------------------------------------------
 3. To add a contact, navigate to the Contacts section and click **Add**. The Contacts dialog box displays.
 -- ----- ---------------------------------------------------------------------------------------------------------
