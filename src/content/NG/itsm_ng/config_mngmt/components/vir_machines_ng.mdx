# Virtual Machines

 --- -------------------------------------------------------------------------------------------------------------
   This function is only available with certain asset types. Therefore, the relevant tab may not be displayed.
 --- -------------------------------------------------------------------------------------------------------------

 -- ----- ------------------------------------------------------
 1. While viewing an asset record, click **Components**.
 -- ----- ------------------------------------------------------

 -- ----- -----------------------------
 2. Click **Virtual Machines**.
 -- ----- -----------------------------

::: Ind50
![virtual machines new 623x170](../../../Resources/Images/Components/virtual-machines-new_623x170.png)

## Add a New Virtual Machine

 -- ----- ----------------------------------------------------------
 1. Click **Add**. The Virtual Machines dialog box displays.
 -- ----- ----------------------------------------------------------

::: Ind50
![virtual machines add new 474x213](../../../Resources/Images/Components/virtual-machines-add-new_474x213.png)

 -- ----- --------------------------------
 2. Click **New Virtual Machine.**
 -- ----- --------------------------------

 -- ----- -----------------------------------------
 3. Enter the applicable field information.
 -- ----- -----------------------------------------

 -- ----- ------------------------------------------------------
 4. When all selections/entries are made, click **Save**
 -- ----- ------------------------------------------------------

## Modify Existing Virtual Machine

 -- ----- -------------------------------------
 1. Click **Existing Virtual Machine**.
 -- ----- -------------------------------------

::: Ind50
![virtual machines existing new 473x324](../../../Resources/Images/Components/virtual-machines-existing-new_473x324.png)

 -- ----- ----------------------------------------------------------------------------------------------------------------------------------------------------------------------
 2. Enter the search criteria to locate the existing device.  *See also* [Advanced Search](../../../Common%20Topics/Advanced-Search.mdx){style="font-style: italic;"}*.*
 -- ----- ----------------------------------------------------------------------------------------------------------------------------------------------------------------------

 -- ----- -----------------------------------------------------
 3. Select the applicable device from the results list.
 -- ----- -----------------------------------------------------

 -- ----- -----------------------------------------------
 4. When all selections are made, click **Save**.
 -- ----- -----------------------------------------------

## Delete Virtual Machine

While viewing the list of existing devices, select the device to delete, and click the **Delete** button.
