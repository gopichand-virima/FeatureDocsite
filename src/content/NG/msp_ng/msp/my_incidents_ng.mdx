# My Incidents

Use this function to view and manage incidents.

 -- ----- ---------------
 1. Log into MSP.
 -- ----- ---------------

 -- ----- ---------------------------------------------------------------
 2. Click **My Incidents**. A list of all incidents is displayed.
 -- ----- ---------------------------------------------------------------

::: Ind50
![my incidents main old](../Resources/Images/MSP/my-incidents-main-old.PNG)

 -- ----- -------------------------------------------------------------------------------------------------------------------------------------------------------------------
 3. To view the details for an incident, click the applicable line item. *Refer to* [Incidents](/itsm/incident_mngmt/incidents_6_1.mdx){style="font-style: italic;"}*.*
 -- ----- -------------------------------------------------------------------------------------------------------------------------------------------------------------------
