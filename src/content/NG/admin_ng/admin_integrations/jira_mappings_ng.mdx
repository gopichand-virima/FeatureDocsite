# Jira Asset Mappings

Use this function to view and add Jira mapping information.

 --- --------------------------------------------------------------------------------------------------
   There must be a [Jira Credential](Jira-Credentials.mdx) configured before a mapping can be done.
 --- --------------------------------------------------------------------------------------------------

In the main window, click **Admin \> Integrations \> Jira Asset Mappings**. The Jira Asset Mapping window displays.

![JIra Asset Mappings 6.1](../Resources/Images/Admin - Integration/JIra_Asset_Mappings_6.1.png)

[[[Closed]{.image .placeholder .MCDropDown_Image_Icon original-image-src="../../Skins/Default/Stylesheets/Images/transparent.gif" original-image-title="" height="12" width="16" Jira Mapping](javascript:void(0)){.MCDropDownHotSpot .dropDownHotspot .MCDropDownHotSpot_ .MCHotSpotImage aria-expanded="false"}]{.MCDropDownHead .dropDownHead}

 -- ----- ------------------------------------------------------------------------
 1. From the *Select Actions* drop-down list, choose **New Jira Mapping**.
 -- ----- ------------------------------------------------------------------------

::: Ind50
![JIra Asset New Mappings 6.1](../Resources/Images/Admin - Integration/JIra_Asset_New_Mappings_6.1.png)

 -- ----- --------------------------------------------------------------------------------------------------------------
 2. By default, the *Property* selected is Blueprint. Select the *Operator* and the applicable Virima Blueprint.
 -- ----- --------------------------------------------------------------------------------------------------------------

 -- ----- -------------------------------------------------------------------
 3. Select the Jira *Object Schema*, *Object Type*, and *Child Type*.
 -- ----- -------------------------------------------------------------------

 -- ----- ----------------------------------------------------------------------------------------------------------------------------------------------------------------------
 4. *Is Enabled* indicates the status of the mapping (true/false). A blueprint copies from Virima to Jira only if the mapping has the \"Is Enabled\" status set to True.
 -- ----- ----------------------------------------------------------------------------------------------------------------------------------------------------------------------

 -- ----- ------------------------------------------------
 5. To add a Jira Property Mapping, click **Add**.
 -- ----- ------------------------------------------------

 -- ----- ---------------------------------------------------------------------------------------
 6. Select the *Virima Property*, *Jira Property*, *Value* and if a *Correlator* applies.
 -- ----- ---------------------------------------------------------------------------------------

 -- ----- ----------------
 7. Click **Add**.
 -- ----- ----------------

[[[Closed]{.image .placeholder .MCDropDown_Image_Icon original-image-src="../../Skins/Default/Stylesheets/Images/transparent.gif" original-image-title="" height="12" width="16" Mappings](javascript:void(0)){.MCDropDownHotSpot .dropDownHotspot .MCDropDownHotSpot_ .MCHotSpotImage aria-expanded="false"}]{.MCDropDownHead .dropDownHead}

 -- ----- ---------------------------------------------------------------------
 1. From the *Select Actions* drop-down list, choose **Clone Mapping**.
 -- ----- ---------------------------------------------------------------------

 -- ----- -------------------------------------------------------------
 2. Use the checkboxes to select the records you want to clone.
 -- ----- -------------------------------------------------------------

 -- ----- -----------------------------------------------------------------
 3. The selected mappings will be duplicated and added to the list.
 -- ----- -----------------------------------------------------------------

[[[Closed]{.image .placeholder .MCDropDown_Image_Icon original-image-src="../../Skins/Default/Stylesheets/Images/transparent.gif" original-image-title="" height="12" width="16" Metadata](javascript:void(0)){.MCDropDownHotSpot .dropDownHotspot .MCDropDownHotSpot_ .MCHotSpotImage aria-expanded="false"}]{.MCDropDownHead .dropDownHead}

 -- ----- ------------------------------------------------------------------------
 1. From the *Select Actions* drop-down list, choose **Re-Sync Metadata**.
 -- ----- ------------------------------------------------------------------------

 -- ----- ------------------------------------------------------------------------------
 2. If the re-sync is successful, a confirmation message displays. Click **OK**.
 -- ----- ------------------------------------------------------------------------------

[[[Closed]{.image .placeholder .MCDropDown_Image_Icon original-image-src="../../Skins/Default/Stylesheets/Images/transparent.gif" original-image-title="" height="12" width="16" Mappings](javascript:void(0)){.MCDropDownHotSpot .dropDownHotspot .MCDropDownHotSpot_ .MCHotSpotImage aria-expanded="false"}]{.MCDropDownHead .dropDownHead}

 -- ----- ----------------------------------------------------------------------
 1. From the *Select Actions* drop-down list, choose **Reset Mappings**.
 -- ----- ----------------------------------------------------------------------

 -- ----- --------------------------------------------------------------
 2. If the reset is successful, a confirmation message displays.
 -- ----- --------------------------------------------------------------

[[[Closed]{.image .placeholder .MCDropDown_Image_Icon original-image-src="../../Skins/Default/Stylesheets/Images/transparent.gif" original-image-title="" height="12" width="16" Jira Mapping](javascript:void(0)){.MCDropDownHotSpot .dropDownHotspot .MCDropDownHotSpot_ .MCHotSpotImage aria-expanded="false"}]{.MCDropDownHead .dropDownHead}

 --- -----------------------------------------------------------------------------------------
   Not all functions include an option to edit record line items shown in the main window.
 --- -----------------------------------------------------------------------------------------

 -- ----- -------------------------------------------------------
 1. Navigate to the window containing the record to edit.
 -- ----- -------------------------------------------------------

 -- ----- -------------------------------------------------------------------------------------
 2. Click the line containing the record. The applicable window or dialog box displays.
 -- ----- -------------------------------------------------------------------------------------

 -- ----- -----------------------------------
 3. Make the necessary modifications.
 -- ----- -----------------------------------

 -- ----- -------------------------------------------
 4. Click **Save** or **Add**, as applicable.
 -- ----- -------------------------------------------

[[[Closed]{.image .placeholder .MCDropDown_Image_Icon original-image-src="../../Skins/Default/Stylesheets/Images/transparent.gif" original-image-title="" height="12" width="16" Jira Mapping](javascript:void(0)){.MCDropDownHotSpot .dropDownHotspot .MCDropDownHotSpot_ .MCHotSpotImage aria-expanded="false"}]{.MCDropDownHead .dropDownHead}

 --- --------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------
   Deleting is a permanent action and cannot be undone. Deleting may affect other functionality and information in the application such as data in configured reports, fields in windows, selectable options, etc. Therefore, be sure to understand the potential effects before making a deletion.
 --- --------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------

 -- ----- --------------------------------------------------
 1. Click the line that contains the item to delete.
 -- ----- --------------------------------------------------

 -- ----- ------------------------------------------------------------------------------------------------------------------------------------------------------
 2. From the *Select Actions* drop-down list, choose **Delete**. If a confirmation message is displayed, type **Delete** and click **Delete**to confirm.
 -- ----- ------------------------------------------------------------------------------------------------------------------------------------------------------
