# Details Tab

When you select a specific record from the **AD User Import Logs**, the system opens the **Details view**. This page provides a full summary of the chosen AD import activity.

**Key Information displayed**:

- **Name**: A user-defined name for the AD configuration.
- **Type of Import**: Specifies whether the import was for All Users or a specific filtered group.
- **Triggered At**: The exact date and time when the import process started.
- **Completed At**: Timestamp showing when the import was successfully finished (blank if still in progress).
- **Time Taken**: Total duration of the import job.
- **Edited Users**: Number of existing user records updated during this import.
- **New Users**: Number of users added as part of the import.
- **Total Users**: The complete number of user records processed.
- **Description**: A text field for system notes or administrator comments about the import (e.g., "Imported successfully" or "No changes detected").
