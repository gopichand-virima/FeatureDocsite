# Docker Images

 --- -------------------------------------------------------------------------------------------------------------
   This function is only available with certain asset types. Therefore, the relevant tab may not be displayed.
 --- -------------------------------------------------------------------------------------------------------------

 -- ----- ------------------------------------------------------
 1. While viewing an asset record, click **Components**.
 -- ----- ------------------------------------------------------

 -- ----- --------------------------
 2. Click **Docker Images**.
 -- ----- --------------------------

::: Ind50
![docker images new 470x136](../../../Resources/Images/Components/docker-images-new_470x136.png)

## Add a New Docker Image

 -- ----- -------------------------------------------------------
 1. Click **Add**. The Docker Images dialog box displays.
 -- ----- -------------------------------------------------------

::: Ind50
![docker images add new 371x213](../../../Resources/Images/Components/docker-images-add-new_371x213.png)

 -- ----- -----------------------------
 2. Click **New Docker Image.**
 -- ----- -----------------------------

 -- ----- -----------------------------------------
 3. Enter the applicable field information.
 -- ----- -----------------------------------------

 -- ----- --------------------------------------------
 4. When all entries are made, click **Save**.
 -- ----- --------------------------------------------

## Modify Existing Docker Image

 -- ----- ----------------------------------
 1. Click **Existing Docker Image**.
 -- ----- ----------------------------------

::: Ind50
![docker image existing new 381x257](../../../Resources/Images/Components/docker-image-existing-new_381x257.png)

 -- ----- ----------------------------------------------------------------------------------------------------------------------------------------------------------------------
 2. Enter the search criteria to locate the existing device.  *See also* [Advanced Search](../../../Common%20Topics/Advanced-Search.mdx){style="font-style: italic;"}*.*
 -- ----- ----------------------------------------------------------------------------------------------------------------------------------------------------------------------

 -- ----- -----------------------------------------------------
 3. Select the applicable device from the results list.
 -- ----- -----------------------------------------------------

 -- ----- -----------------------------------------------
 4. When all selections are made, click **Save**.
 -- ----- -----------------------------------------------

## Delete Docker Image

While viewing the list of existing devices, select the device to delete, and click the **Delete** button.
