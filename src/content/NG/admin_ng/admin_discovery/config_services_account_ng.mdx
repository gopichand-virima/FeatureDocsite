# Configure Services Account

Use this function to configure service accounts with admin privileges on both the host being scanned and the server where the application is being installed.

Open the **Services.msc** file.

Double-clickon **Virima Discovery Application**. The Virima *Discovery Agent Properties dialog box* displays.

Provide the Administrator credential information as shown.

Click **OK**Â to restart the application service.


The Discovery Application is packed within a Java runtime environment. However, Discovery can be pointed to a different JRE throughSettings>Java Configuration.
