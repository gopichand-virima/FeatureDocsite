# Reports - API Integration

API documentation for programmatic report generation and data export.

## API Endpoints

### Run Report
```
POST /api/v1/reports/run
```

Execute a saved report programmatically.

### Get Report Data
```
GET /api/v1/reports/{reportId}/data
```

Retrieve report results in JSON format.

### Export Report
```
GET /api/v1/reports/{reportId}/export
```

Export report in specified format (PDF, Excel, CSV).

### Create Custom Report
```
POST /api/v1/reports/custom
```

Create a new custom report definition.

## Authentication

```
Authorization: Bearer YOUR_API_TOKEN
```

## Request Example - Run Report

```json
{
  "reportId": "rpt-asset-inventory",
  "parameters": {
    "dateRange": {
      "start": "2025-10-01",
      "end": "2025-10-31"
    },
    "filters": {
      "assetType": "laptop",
      "status": "deployed"
    }
  },
  "format": "json"
}
```

## Response Example

```json
{
  "status": "success",
  "data": {
    "reportId": "rpt-asset-inventory",
    "executionId": "exec-12345",
    "generatedAt": "2025-11-10T10:30:00Z",
    "recordCount": 523,
    "results": [
      {
        "assetId": "asset-001",
        "assetTag": "LAP-001",
        "model": "Dell Latitude 7420",
        "assignedTo": "john.doe@company.com"

    ]
  }
}
```

## Export Formats

Supported formats:
- `json`: JSON data
- `pdf`: PDF document
- `xlsx`: Excel spreadsheet
- `csv`: Comma-separated values
- `pptx`: PowerPoint presentation

## Webhook Events

- Report generation completed
- Scheduled report delivered
- Report execution failed
- Data threshold exceeded
