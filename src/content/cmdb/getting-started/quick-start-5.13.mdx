# CMDB - Quick Start Guide

Get started with the CMDB module quickly and efficiently.

## Prerequisites

- CMDB module access permissions
- Understanding of your IT infrastructure
- Asset inventory data (optional but recommended)

## Quick Start Steps

### Step 1: Access CMDB
1. Log in to Virima
2. Navigate to CMDB from the main menu
3. Review the dashboard overview

### Step 2: Add Configuration Items
1. Click "Add CI" button
2. Select CI type (Server, Application, Network Device, etc.)
3. Fill in required attributes
4. Save the configuration item

### Step 3: Create Relationships
1. Select a configuration item
2. Click "Manage Relationships"
3. Choose relationship type (Runs on, Depends on, etc.)
4. Link to related CIs

### Step 4: View Dependencies
1. Select any CI
2. Click "View Dependency Map"
3. Explore the visual relationship diagram
4. Analyze impact and dependencies
