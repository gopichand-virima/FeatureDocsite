# Installed Software

This function is only available with certain asset types. Therefore, the relevant tab may not be displayed.


Use this function to add and manage the software installed on a host.

Components are categorized based on publishers. When the Discovery application detects more than one application from the same publisher on a host, they are grouped under Installed Software. A single instance of a publisher is grouped as **Uncategorized**.

!

!


### Add Installed Software

- In the main window, select **ITSM > Configuration Management > CMDB**.
- Search for and select the applicable asset record.
- When the asset record window displays, click the **Components** tab.
- Click the **Installed Software** tab.

!

[!](../../../../Resources/Images/ITSM/installed-software-new.png)

### Software Details

- Click the tab for an available software, then click **Add**. The Installed Software dialog box displays.

! !

- Do one of the following:
- Click **New Installed Software** and complete all the fields. All fields are required.
- Click **Existing Installed Software** and make any applicable changes.
- When all selections/entries are made, click **Save**.
- Enter the *Software Name*, *Software Version*, and *Asset Name*.
- Select the *Software Install Date*.
- When all selections/entries are made, click **Add**.


### Edit Installed Software

- In the selected record configuration window, click **Components**.
- In the secondary group of tabs, click **Installed Software**.
- From the list of records, select the applicable software to edit.
- Make the applicable edits on the appropriate tab.


### Delete Installed Software

Deleting is a permanent action and cannot be undone. Deleting may affect other functionality and information in the application such as data in configured reports, fields in windows, selectable options, etc. Therefore, be sure to understand the potential effects before making a deletion.


While viewing the list of existing records, select a record(s) to delete, and click the **Delete** button.
