# Configure Services Account

Use this function to configure service accounts with admin privileges on both the host being scanned and the server where the application is being installed.

<div class="numlist1" madcap:autonum="1. ">

Open the **Services.msc** file.

</div>

<div class="numlist2" madcap:autonum="2. ">

<span class="span_1">Double-click </span><span class="span_2">on **Virima Discovery Application**</span><span class="span_1">. The Virima *Discovery Agent Properties dialog box* displays.</span>

</div>

<div class="Ind50">

<span class="image placeholder" original-image-src="../Resources/Images/Discovery/discovery-agent-properties.png" original-image-title=""></span>

</div>

<div class="numlist2" madcap:autonum="3. ">

Provide the Administrator credential information as shown.

</div>

<div class="numlist2" madcap:autonum="4. ">

Click **OK** to restart the application service.

</div>
