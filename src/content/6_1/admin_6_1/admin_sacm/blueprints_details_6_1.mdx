# Blueprint: Details

Use this function to add details for a blueprint.

<div class="NumList1" madcap:conditions="Default.OldUI" madcap:autonum="1. ">

In the Blueprints window, from the *Select Actions* drop-down list, choose **New Blueprint**.

</div>

<div class="NumList2" madcap:conditions="Default.OldUI" madcap:autonum="2. ">

Select an existing record. A new window displays with the **Details** tab selected.

</div>

<div class="NumList2" style="font-style: normal;" madcap:conditions="Default.OldUI" madcap:autonum="3. ">

Complete the fields according to the information below.

</div>

<div class="NumList2" style="font-style: normal;" madcap:conditions="Default.OldUI" madcap:autonum="4. ">

Refer to [Searching](../Common%20Topics/Searching.htm) for details on locating items.

</div>

<div class="NumList2" madcap:conditions="Default.OldUI" madcap:autonum="5. ">

When all selections/entries are made, click **Save**.

</div>
