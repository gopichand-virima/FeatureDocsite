# CI: Installed On

Use this function to view what software is installed on what system.

<div class="NumList1" madcap:autonum="1. ">

From the navigation pane, select **ITSM /> Configuration Management /> Configuration Items**. The Configuration Items window displays.

</div>

<div class="NumList2" madcap:autonum="2. ">

Select a record in the list. A new window opens and the **Details** tab displays.

</div>

<div class="NumList2" madcap:autonum="3. ">

Click the **Installed On**Â tab. A list of all settings on which the software is installed displays.

</div>

<div class="NumList2" madcap:autonum="4. ">

To view the details for an asset, click on the asset name. You are returned to the Configuration Items window.

</div>

<div class="NumList2" madcap:autonum="5. ">

To view the location for a selected asset, click **View Map**.

</div>
