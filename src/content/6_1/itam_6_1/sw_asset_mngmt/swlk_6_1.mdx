# Software License Keys

Use this function to configure a license key that applies to a specific software asset.

 -- ----- ---------------------------------------------------------------------------------------------------------------------------------------------
 1. From the navigation pane, select **ITAM \> Software Asset Management \> Software License Keys**. The Software License Keys window displays.
 -- ----- ---------------------------------------------------------------------------------------------------------------------------------------------

::: Ind50
![sw asset main new 635x88](../../Resources/Images/ITAM/sw-asset-main-new_635x88.png)

 -- ----- -----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------

In the *Select Actions* field, choose one of the following: [New](SWLK-New.mdx), [Delete](../../Common%20Topics/Delete-Remove.mdx){.MCTopicPopup .MCTopicPopupHotSpot .a}, [Export](/discovery/import_data_files/export_6_1.mdx){.MCTopicPopup .MCTopicPopupHotSpot .a}, or [Import](/common_topics/import_6_1.mdx){.MCTopicPopup .MCTopicPopupHotSpot .a}.
