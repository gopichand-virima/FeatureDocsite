# Discovery Scan - API Integration

API documentation for automating discovery operations.

## API Endpoints

### Trigger Scan
```
POST /api/v1/discovery/scan
```

Initiate a discovery scan programmatically.

### Get Scan Status
```
GET /api/v1/discovery/scan/{scanId}/status
```

Check the status of a running scan.

### Get Discovered Assets
```
GET /api/v1/discovery/assets
```

Retrieve newly discovered assets.

### Manage Credentials
```
POST /api/v1/discovery/credentials
```

Add or update discovery credentials.

## Authentication

```
Authorization: Bearer YOUR_API_TOKEN
```

## Request Example - Trigger Scan

```json
{
  "profileId": "profile-123",
  "scanType": "network",
  "scope": {
    "ipRanges": ["192.168.1.0/24"],
    "credentials": ["cred-456", "cred-789"]
  }
}
```

## Response Example

```json
{
  "status": "success",
  "data": {
    "scanId": "scan-abc123",
    "status": "running",
    "startTime": "2025-11-10T10:30:00Z",
    "estimatedCompletion": "2025-11-10T11:00:00Z"
  }
}
```

## Webhooks

Configure webhooks to receive notifications:
- Scan completion
- New assets discovered
- Scan errors
- Credential failures
