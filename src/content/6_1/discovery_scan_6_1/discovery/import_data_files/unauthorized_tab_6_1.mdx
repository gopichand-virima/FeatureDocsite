# <span class="Strong">Unauthorized Tab</span>

Lists new records that do not currently exist in the CMDB. These will be added as new CIs once approved.

<span class="image placeholder" original-image-src="../../Resources/Images/Discovery/Import-Data-Files/Import%20Data%20Files_1_17.png" original-image-title=""></span>

- <span class="Strong" style="font-weight: bold;">Action Buttons:</span>

  - <span class="inline-comment-marker">"**Move to CMDB**" (to approve and move assets into the CMDB)</span>

  - <span class="inline-comment-marker">"**Delete**" (to remove selected assets)</span>

- Click any unauthorized CI to view details of the im<span class="inline-comment-marker">ported assets.</span>

  <span class="image placeholder" original-image-src="../../Resources/Images/Discovery/Import-Data-Files/Import%20Data%20Files_1_18.png" original-image-title=""></span>

# <span class="Strong">Layout and Sections</span>

1.  <span class="Strong" style="font-weight: bold;">Left Sidebar</span>

- <span class="Strong" style="font-weight: bold;">Primary Details:</span> Shows key identifiers and metadata for the asset, including:

  - **Asset ID** (e.g., IMP001273)

  - **Blueprint** (e.g., Windows Server)

  - **Created On** and **Last Modified On** timestamps

  - <span class="Strong" style="font-weight: bold;">Authorization status</span> – A badge indicating <span class="Strong">This CI is unauthorized</span>:

    - <span class="Emphasis" style="font-weight: bold;">Unauthorized</span> = the CI is new or not yet moved to CMDB; typically surfaced from the <span class="Emphasis">Unauthorized</span> tab in import views.

  - **Tags** if any (with an editable pencil icon)

- <span class="Strong">**Owner**:</span> The system assigns the logged-in user as the CI owner when importing a CI.

- <span class="Strong" style="font-weight: bold;">Import Details:</span> Information about the import process, such as:

  - File Name (e.g., Import+CI's.xlsx)

  - Status (e.g., Processed)

  - Uploaded By (e.g., Lakshmi Virima)

  - Uploaded On (timestamp)

  - Authorization status – A badge indicating This CI is unauthorized.

2.  <span class="Strong" style="font-weight: bold;">Main Content Area</span>

- <span class="Strong">**Header**:</span> Displays the asset identifier (e.g., IMP001273).

- <span class="Strong" style="font-weight: bold;">Action Buttons:</span>

  - <span class="Strong">Move to CMDB</span> (to promote the asset to the CMDB)

  - <span class="Strong">Rescan Now</span> (to trigger a new scan of the asset)

- <span class="Strong" style="font-weight: bold;">Tabbed Navigation:</span> Provides access to different information categories:

  - Details (active tab)

  - Components

  - Log On Events

  - Tasks

  - Comments

  - Attachments

  - Updates

3.  <span class="Strong" style="font-weight: bold;">Details Tab (Main Content)</span>

- <span class="Strong" style="font-weight: bold;">Asset Primary Information</span>

  - <span class="Strong" style="font-weight: bold;">Asset ID</span> – the imported ID (IMP prefix).

  - <span class="Strong" style="font-weight: bold;">Asset Name</span> – Asset name from the file.

  - <span class="Strong" style="font-weight: bold;">Host Name</span> – The hostname of the asset.

- <span class="Strong" style="font-weight: bold;">Hardware and Network</span>

  - <span class="Strong" style="font-weight: bold;">IP Address</span> – last/primary IP from the import.

- <span class="Strong" style="font-weight: bold;">Life Cycle Information</span>

  - <span class="Strong" style="font-weight: bold;">Acquisition Date</span> – date/time of acquisition provided during import.

- <span class="Strong" style="font-weight: bold;">Network Adapter Details</span>

  - <span class="Strong" style="font-weight: bold;">MAC Address</span> – network adapter MAC captured in the import.
