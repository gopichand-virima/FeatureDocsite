# My Requests

Use this function to displays the status of requests that are created by the currently logged in user.

+---+--------------------------------------------------------------------------------+
|   | ::: Bullet1 |
| Column 1 | Role access for Request is required. |
| --- | :---: |
| | |
| | ::: Bullet1 |
| | A Request can be canceled, but this must be done before the Request is closed. |
| | ::: |
+---+--------------------------------------------------------------------------------+

In the navigation pane, select **Self-Service \> My Requests**. The resulting window displays a list of all service catalog requests (if any).

![my requests main old](../Resources/Images/Self-Service/my-requests-main-old.PNG)

To view the related request information, under Request ID, click the **hyperlink**. The related [Request](/itsm/request_fulfillment/requests_6_1.mdx) window displays.
