# New Change Request

Use this function to create a change request.

|     |                                                                                                                                   |
|-----|-----------------------------------------------------------------------------------------------------------------------------------|
|     | Roles with relevant access as per role access of Change and RFC can view, add, edit, approve, reject, resolve and close a change. |

<div class="NumList1" madcap:autonum="1. ">

From the navigation pane, select **ITSM /> Change Management**. The Changes window displays.

</div>

<div class="Ind50 step-media">

<span class="image placeholder" original-image-src="../../Resources/Images/ITSM/Change%20Management/change-mngmnt-main-new.PNG" original-image-title="" style="width: 796px;height: 158px;"></span>

</div>

<div class="NumList2" madcap:autonum="2. ">

Click **New**.

</div>

<div class="NumList2" madcap:autonum="3. ">

Complete the fields according to the table below.

</div>

<div class="NumList2" madcap:autonum="4. ">

When all selections/entries are made, click **Add**.

</div>
