# Edit Software License Key

Use this function to edit the configured values for a software license key.

From the navigation pane, select **ITAM > Software Asset Management > Software License Keys**. The Software License Keys window displays.

Select a record to edit.

Modify any of the following information:

- [Details](/itam/sw_asset_mngmt/swlk_details_ng.mdx)
- [Tasks](/itsm/release_mngmt/tasks_ng.mdx)
- [Comments](/discovery/comments_ng.mdx)
- [Attachments](/discovery/attachments_ng.mdx)

When all selections/entries are made, click **Save**.
