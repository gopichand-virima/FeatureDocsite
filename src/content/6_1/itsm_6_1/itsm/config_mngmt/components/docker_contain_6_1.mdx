# Docker Containers

<div class="NumList1" madcap:autonum="1. ">

While viewing an asset record, click **Components**.

</div>

<div class="NumList2" madcap:autonum="2. ">

Click **Docker Containers**.

</div>

## Add a New Docker Container

<div class="NumList1" madcap:autonum="1. ">

Click **Add**. The Docker Containers dialog box displays.

</div>

<div class="Ind50">

<span class="image placeholder" original-image-src="../../../Resources/Images/Components/docker-containers-add-new.png" original-image-title="" style="width: 384px;height: 171px;"></span>

</div>

<div class="NumList2" madcap:autonum="2. ">

Click **New Docker Container.**

</div>

<div class="NumList2" style="font-style: normal;" madcap:autonum="3. ">

Enter the field information.

</div>

<div class="NumList2" madcap:autonum="4. ">

When all entries are made, click **Save**.

</div>

## Modify Existing Docker Container

<div class="NumList1" madcap:autonum="1. ">

Click **Existing Docker Container**.

</div>

<div class="Ind50">

<span class="image placeholder" original-image-src="../../../Resources/Images/Components/docker-container-existing-new.png" original-image-title="" style="width: 373px;height: 254px;"></span>

</div>

<div class="NumList2" madcap:autonum="2. ">

Enter the search criteria to locate the existing device.  *See also* <a href="../../../Common%20Topics/Advanced-Search.htm" style="font-style: italic;">Advanced Search</a>*.*

</div>

<div class="NumList2" madcap:autonum="3. ">

Select the applicable device from the results list.

</div>

<div class="NumList2" madcap:autonum="4. ">

When all selections are made, click **Save**.

</div>

## Delete Docker Container

While viewing the list of existing devices, select the device to delete, and click the **Delete** button.
