# Accessing the Azure AD User Import Logs

1. From the left navigation pane, go to **Discovery Scan > Azure AD User Import Logs**.

1. The **AD User Import Logs** page opens.

1. This shows a list of records.

  - **Key Information displayed**:
  - **Name** – The name used to identify the import (for example, Azure AD).
  - **Started On** – The date and time the import process began.
  - **Completed On** – The date and time the import process completed.

2. Use the **filter fields** at the top of each column to search by name or narrow results by date range.
