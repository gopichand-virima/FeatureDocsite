# Blueprints: History

Use this function to view changes, modifications and actions for the selected blueprint.

 --- --------------------------------------
   This information can only be viewed.
 --- --------------------------------------

 -- ----- ---------------------------------------------------------------------------------------------
 1. In the main window, select **Admin \> SACM \> Blueprints**. The Blueprints window displays.
 -- ----- ---------------------------------------------------------------------------------------------

 -- ----- -------------------------------------------------------------------------------------
 2. Select the record for the blueprint record you want to view. A new window displays.
 -- ----- -------------------------------------------------------------------------------------

 -- ----- ------------------------------------------------------------------
 3. Click the **History** tab.  The historical information displays.
 -- ----- ------------------------------------------------------------------
