# Access Problem Management

The Problem Management module helps identify and address the root causes of recurring incidents. This guide explains how to access and navigate Problem Management.

## Accessing from Main Menu

1. Log in to your Virima instance
2. Navigate to **ITSM** module from main navigation
3. Select **Problem Management** from the dropdown menu

## Quick Access Options

### Dashboard Widgets
Add Problem Management widgets:

1. Navigate to your **Dashboard**
2. Click **Add Widget**
3. Select widgets:
   - **Open Problems**: Active problem records
   - **My Problems**: Problems assigned to you
   - **Known Errors**: Documented workarounds
   - **Problem Analysis**: Trends and patterns

### Creating Problems
Quick create from anywhere:

1. Click **+** icon in top navigation
2. Select **New Problem**
3. Complete required fields
4. Click **Submit**

## Navigation Structure

Problem Management includes:

- **Problems**: All problem records
- **Known Errors**: Documented problems with workarounds
- **Root Cause Analysis**: Problem investigation tools
- **Problem Dashboard**: Metrics and trends
- **Related Incidents**: Incident-problem relationships

## User Permissions

Access varies by role:

### Problem Manager
- Full access to all problems
- Create and manage problem records
- Perform root cause analysis
- Close problem records
- Access all reports

### Problem Analyst
- Create problem records
- Investigate assigned problems
- Document findings
- Propose solutions
- View related data

### Standard Users
- View known errors
- Link incidents to problems
- View problem status
- Access workarounds

Contact your System Administrator for role changes.
