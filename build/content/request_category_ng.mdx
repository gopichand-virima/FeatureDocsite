# Request Category

Use this function to create and manage request categories that help classify and organize service requests.

In the main window, click **Admin > Request Management > Request Category**. The Request Category window displays.

## New Request Category

From the *Select Actions* drop-down list, choose **New Request Category**. The New Request Category window displays.

Enter a *Name* and *Description* for this category.

Select a **Parent Category** if this is a sub-category (optional).

To add an image/icon for this category:

- Click **Upload**
- Search for and select an image
- Click **OK**

Configure category properties:

**Requires Approval**: Check if requests in this category require approval.

**Default Assignment Group**: Select a group that typically handles this category.

**Auto-Fulfill**: Check if requests can be automatically fulfilled without manual intervention.

**Estimated Fulfillment Time**: Set typical completion timeframe for planning purposes.

When all selections/entries are made, click **Add**.

## Common Request Categories

Examples of request categories you might configure:

- **Access Requests**: User access and permissions
- **Hardware Requests**: Equipment and devices
- **Software Requests**: Software licenses and installations
- **Information Requests**: Documentation and information
- **Account Management**: User account creation, modification, deletion
- **Service Requests**: Standard IT services
- **Facilities Requests**: Office and facilities-related requests

## Edit Request Category

Navigate to the window containing the record to edit.

Click the line containing the record. The applicable window displays.

Make the necessary modifications.

Click **Save**.

## Delete Request Category

Deleting is a permanent action and cannot be undone. Ensure no requests are currently using this category before deletion.

Click the line that contains the item to delete.

From the *Select Actions* drop-down list, choose **Delete**. If a confirmation message is displayed, take the appropriate action.
