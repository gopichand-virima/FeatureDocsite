# Key Information Displayed

- [Name]{.Strong}: A user-defined name for the Azure AD configuration.

- [Triggered At]{.Strong}: The exact date and time when the import process started.

- [Completed At]{.Strong}: Timestamp showing when the import was successfully finished (blank if still **in progress**).

- [Time Taken]{.Strong}: Total duration of the import job.

- [Edited Users]{.Strong}: Number of existing user records updated during this import.

- [New Users]{.Strong}: Number of users added as part of the import.

- [Total Users]{.Strong}: The complete number of user records processed.

- [Description]{.Strong}: A text field for system notes or administrator comments about the import (e.g., "Imported successfully" or "No changes detected").
