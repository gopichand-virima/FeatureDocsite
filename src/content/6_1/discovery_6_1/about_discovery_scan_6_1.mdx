# About Discovery Scan

This module contains the following functions:

- [Dashboard](/dashboards/dashboard_overview_6_1.mdx) contains a list of all the **recent** scans and any reports that have been added to the window. In this window, [new content](/dashboards/dashboard_contents_6_1.mdx) can be added and [customized](/discovery/dashboard/dashboard_customization_6_1.mdx).
- [Run a Scan](/discovery/run_a_scan/run_a_scan_6_1.mdx) identifies the devices connected to and communicating in a network and allows for the configuration of a [scan frequency](/discovery/scans_frequency_6_1.mdx).
- [Recent Scans](/discovery/recent_scans/scans_recent_6_1.mdx) displays successfully completed scans and creates a new scan to run.
- [Scheduled Scans](/discovery/scans_scheduled_6_1.mdx) enables the execution of scan functionality according to a configured date and time.
- [IPAM Networks](/discovery/ipam_networks_6_1.mdx) enables the planning, tracking, and managing the information associated with an Internet Protocol address existing in the monitored network environment.
- [Discovered Items](/discovery/discovered_items_6_1.mdx) displays the details for a successful scan.
- [Import from AWS](/discovery/import_aws_6_1.mdx) identifies information in Amazon Web Services (AWS).
- [Import from AZURE](/discovery/import_azure_6_1.mdx) identifies information in AZURE, a cloud service owned by Microsoft.
- [Import Data Files](/discovery/import_data_file_6_1.mdx) imports details of CI, CI relations, and asset details from an Excel spreadsheet.
- [Imported Assets](/discovery/imported_assets_6_1.mdx) displays the results of data imported during the data files import operation.
- [AD User Import Logs](/discovery/ad_user_import_logs_6_1.mdx) displays scanned results and the details for imported AD users.
- [Azure AD User Import Logs](/discovery/azure_ad_user_import_logs_6_1.mdx) displays scanned results and the details for imported Azure AD user.
- [LDAP User Import Scans](/discovery/ldap_user_import_scans_6_1.mdx) displays scanned results and the details for imported LDAP Users.
