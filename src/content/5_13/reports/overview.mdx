# Reports Overview

Reports module provides comprehensive reporting and analytics capabilities across all Virima modules with customizable dashboards and visualizations.

## Key Features

- Pre-built report templates
- Custom report builder
- Interactive dashboards
- Scheduled report delivery
- Export to multiple formats
