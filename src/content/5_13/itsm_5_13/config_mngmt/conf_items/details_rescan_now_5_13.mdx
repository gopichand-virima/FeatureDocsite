# Rescan Now

Use this function to rescan selected assets.

While viewing the **Details** tab, click the **Rescan Now** button. The Rescan Now dialog box displays.


The IP Address for the selected asset is pre-populated and cannot be changed.


! !

In the *Choose a client* field, click the drop-down and select an available client.

In the *Client* field, click the drop-down list and select an available client.

In the *Type of Scan* field, click the drop-down and select the type of scan. *For details on the types of scans available, refer to [Run a Scan](/discovery/run_a_scan/run_a_scan_5_13.mdx).*

When all selections/entries are made, click **Scan**. A confirmation message displays and the window refreshes and displays the [Recent Scans](/discovery/recent_scans/scans_recent_5_13.mdx) information.
