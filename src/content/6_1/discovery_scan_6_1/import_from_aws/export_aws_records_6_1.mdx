# Exporting AWS Records

Exports data for the selected imported AWS record as an Excel spreadsheet and sends an email notification when the spreadsheet is ready for download.

- From the **Select Actions** drop-down list, choose **Export**.

 ![Import From AWS 1 12 676x249](../../Resources/Images/Discovery/Import-From-AWS/Import From AWS_1_12_676x249.png)

- Click **Continue.**

- The system downloads an[ Excel (.xlsx) file of]{.inline-comment-marker} the list with the details,

 - **Unique ID** -- The Virima import record ID (e.g., AWS000578).

 - **Account Id** -- The \"Account Id\" column displays the value entered for any credential type (Account Id, Organization Unit Id, or Root Id).

 - **Credential** -- The credential set used (e.g., aws, AWS_ACCOUNT).

 - **Imported By** -- The user who [initiated the import (e.g., ]{.inline-comment-marker}[Admin Virima]{.inline-comment-marker_1}[).]{.inline-comment-marker}

 - [Manual Import: The logged-in user who initiated a manual import.]{.inline-comment-marker}

 - [Scheduled Import: The logged-in user who initiated a scheduled import.]{.inline-comment-marker}

 - **Name** -- The AWS service or resource type imported (e.g., nat, load, AWS Batch, ECS, AWS EKS6).

 - **Imported On** -- The timestamp when that record was imported (as displayed in the UI).
