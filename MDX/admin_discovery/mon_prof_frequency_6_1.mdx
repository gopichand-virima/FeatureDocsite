# Monitoring Profile: Frequency

- In the *Monitoring Frequency* field, click the *Current Time Settings* drop-down list and select the applicable frequency (for example, every 30 seconds, every minute and so forth).

!

- Check to select the required **Time** (Second, Minute, Hour) and **Date** (Day, Month, Weekday) for scan functionality.
- Click the options to be monitored, either **System** or **File**.
- Click the links below to complete the other sections on this window.
- When all selections/entries are made, click **Add**.


If required information is missing (such as the monitoring frequency), an Alert Message displays. Resolve the error and then attempt to add or save the information again.
