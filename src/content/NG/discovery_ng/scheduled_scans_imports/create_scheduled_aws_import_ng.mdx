# Creating a Scheduled AWS Import

1. From the **Select Actions** dropdown, choose **New Scheduled AWS Import**.

2. A configuration dialog opens.

**Name:** Enter a descriptive name for the scheduled import.

**Credential:** Choose the AWS credential configured in **Admin > Discovery > Credentials**.

**Select Services:** Check the AWS services you want to import (e.g., EC2, RDS, S3, Lambda, VPC).

**Import Frequency:** Select how often the import should run (e.g., Every Day, Every 2 Days, Weekly).

**Active:** Check this box to activate the schedule.

**Timezone:** Select the appropriate timezone.

**Schedule Timing:** Configure the execution time (hour, minute, second).

1. Click **Add** to save the schedule.

The new scheduled job appears in the **Scheduled Scans / Imports** list.
