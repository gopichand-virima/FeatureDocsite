# Delete Report

Use this function to delete a report from the applicable entities (for example, My Dashboard) and archive the report.

From the navigation pane, select either **Reports > Ad Hoc Reports** or **Reports > Canned Reports.** The Reports window displays.

In the list, select one or more reports to delete.

From the *Select Actions* drop-down list, choose **Delete**.

In the Confirmation Message window, Type **Delete** and click **Delete**.


**For System Administrators.** *Refer to* **Admin > Other > Reports** *to also perform this function.*
