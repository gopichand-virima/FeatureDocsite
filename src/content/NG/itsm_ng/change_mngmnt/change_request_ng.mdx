# New Change Request

Use this function to create a change request.

 --- -----------------------------------------------------------------------------------------------------------------------------------
   Roles with relevant access as per role access of Change and RFC can view, add, edit, approve, reject, resolve and close a change.
 --- -----------------------------------------------------------------------------------------------------------------------------------

 -- ----- ----------------------------------------------------------------------------------------------
 1. From the navigation pane, select **ITSM \> Change Management**. The Changes window displays.
 -- ----- ----------------------------------------------------------------------------------------------

 -- ----- ---------------------------------------------------
 2. Complete the fields according to the table below.
 -- ----- ---------------------------------------------------

 -- ----- ------------------------------------------------------
 3. When all selections/entries are made, click **Add**.
 -- ----- ------------------------------------------------------
