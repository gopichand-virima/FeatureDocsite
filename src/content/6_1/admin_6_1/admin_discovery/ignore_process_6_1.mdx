# <span style="background-color: #ffffff;">Ignore Process</span>

Use this function to ignore back-end processes when creating relationships.

<div class="NumList1" madcap:conditions="Default.OldUI" madcap:autonum="1. ">

On the main window, select **Admin /> Discovery /> Ignore Process**. The Ignore Process Names dialog box displays.

</div>

<div class="Ind50">

<span class="image placeholder" original-image-src="../Resources/Images/Admin%20-%20Discovery/ignore-process-names.PNG" original-image-title="" madcap:conditions="Default.OldUI" style="width: 387px;height: 121px;"></span>

</div>

<div class="NumList1" madcap:conditions="Default.NewUI" madcap:autonum="1. ">

In the main window, click the **gear** icon and select **Discovery /> Ignore Process**.

</div>

<div class="Ind50" madcap:conditions="Default.NewUI">

<span class="image placeholder" original-image-src="../Resources/Images/Admin%20-%20Discovery/icon-ignore-adm-process.PNG" original-image-title=""></span>

</div>

<div class="Ind50" madcap:conditions="Default.NewUI">

The Ignore Process Names dialog box displays.

</div>

<div class="Ind50" madcap:conditions="Default.NewUI">

<span class="image placeholder" original-image-src="../Resources/Images/Admin%20-%20Discovery/ignore-process-names-new.png" original-image-title="" madcap:conditions="Default.NewUI" style="width: 385px;height: 191px;"></span>

</div>

<div class="NumList2" madcap:autonum="2. ">

Enter the name(s) of the process to ignore. Separate multiple process names with a comma.

</div>

<div class="NumList2" madcap:conditions="Default.OldUI" madcap:autonum="3. ">

Click **Add**.

</div>

<div class="NumList2" madcap:conditions="Default.NewUI" madcap:autonum="4. ">

Click **Save**.

</div>

<div class="NumList2" madcap:conditions="Default.NewUI" madcap:autonum="5. ">

In the confirmation window, click **OK**.

</div>
