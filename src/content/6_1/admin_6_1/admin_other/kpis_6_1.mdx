# KPIs

A Key Performance Indicator (KPI) is a measurable value that demonstrates the defined objectives effectively. It provides a reporting dashboard reflecting insights for the current performance.

Use this function to view the currently defined KPIs.

<div class="NumList1" madcap:conditions="Default.OldUI" madcap:autonum="1. ">

In the main window, click **Admin /> Others /> KPIs**. The KPIs window displays.

</div>

</div>

<div class="NumList2" madcap:conditions="Default.OldUI" madcap:autonum="2. ">

From the *Select Actions* drop-down list, choose an action: <a href="../Common%20Functions/Delete-Remove.htm" target="_popup">Delete</a> or <a href="../Common%20Functions/Export.htm" target="_popup">Export</a>.

</div>

<div class="NumList2" madcap:autonum="3. ">

To edit an existing KPI, click the applicable line item.
