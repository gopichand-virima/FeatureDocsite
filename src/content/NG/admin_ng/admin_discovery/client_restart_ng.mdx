# Restart Client

 --- ------------------------------------------------------------------
   In order to restart a client, the state must be set to \"true.\"
 --- ------------------------------------------------------------------

 -- ----- ----------------------------------------------------------------
 1. From the main window, select **Admin \> Discovery \> Client.**
 -- ----- ----------------------------------------------------------------

 -- ----- ------------------------------------------------------
 2. In the Client window, select the applicable client.\

 -- ----- ------------------------------------------------------

 -- ----- -------------------------------------
 3. Click the **Discovery Agents** tab.
 -- ----- -------------------------------------

 -- ----- ----------------------------------------------------------------------
 4. From the *Select Actions* drop-down list, choose **Restart Client**.
 -- ----- ----------------------------------------------------------------------

::: {.Ind50
![conf msg restart client old 242x105](../Resources/Images/Discovery/conf-msg-restart-client-old_242x105.png)

 -- ----- ----------------------------------------------------------------------------------------------
 5. In the Confirmation dialog box, click **Yes** (to restart) or **No** (to cancel the action).
 -- ----- ----------------------------------------------------------------------------------------------
