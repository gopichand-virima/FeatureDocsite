# Stockroom

Use this function to define a place and create a record for the existing stock across the organization. It also facilitates designated users to check on the stock and its type.

The benefits of stockroom configuration include the following:

- Recording and ordering off of stock and its information.
- Easy integration with the finance department.
- Better forecasting and planning.
- Easy report generation.


A [Stockroom Type](/itam/stockroom_type_6_1.mdx) should be created before adding a stockroom.


In the navigation pane, select **ITAM > Stockroom**. The Stockroom window displays.

!

!


### New Stockroom

- From the *Select Actions* drop-down list, choose **New Stockroom**. The New Stockroom window displays.

!

- Enter a *Display Name*.
- If the stockroom is in a different location, select *Is External*.
- Click **Add** for the following fields, then search for and select the applicable information:
- *Assignment Group*
- *Location*
- *Manager*
- *Stockroom Type (refer to [Stockroom Type](/itam/stockroom_type_6_1.mdx))*
- Click **Add**.


### Edit Stockroom

Not all functions include an option to edit record line items shown in the main window.


- Navigate to the window containing the record to edit.
- Click the line containing the record. The applicable window or dialog box displays.
- Make the necessary modifications.
- Click **Save** or **Add**, as applicable.


### Delete Stockroom

Deleting is a permanent action and cannot be undone. Deleting may affect other functionality and information in the application such as data in configured reports, fields in windows, selectable options, etc. Therefore, be sure to understand the potential effects before making a deletion.


- Click the line that contains the item to delete.
- From the *Select Actions* drop-down list, choose **Delete**. If a confirmation message is displayed, type **Delete** and click **Delete**to confirm.
