# Monitoring Profile: Details

Use this function to add details for a monitoring profile.

<div class="NumList1" madcap:autonum="1. ">

<span class="span_1">On the main window, select </span><span class="Strong_2" style="font-weight: bold;">Admin</span><span class="span_1" style="font-weight: bold;"> /> </span><span class="Strong_2" style="font-weight: bold;">Discovery</span><span class="span_1">** /> Discovery Monitoring Profile**. The Discovery Monitoring Profile window displays.</span>

</div>

<div class="NumList2" madcap:autonum="2. ">

Select an existing record. A new window displays with the **Details** tab displayed, by default.

</div>

<div class="Ind50" madcap:conditions="Default.OldUI">

<span class="image placeholder" original-image-src="../Resources/Images/Admin%20-%20Discovery/discovery-mon-prof-details.PNG" original-image-title="" madcap:conditions="Default.OldUI"></span>

</div>

<div class="Ind50" madcap:conditions="Default.NewUI">

<span class="image placeholder" original-image-src="../Resources/Images/Admin%20-%20Discovery/discovery-mon-prof-details-new.PNG" original-image-title=""></span>

</div>

<div class="NumList2" madcap:autonum="3. ">

In the *Name* field, enter a name for this profile.

</div>

<div class="NumList2" madcap:autonum="4. ">

In the *Description* field, enter information to identify this profile.

</div>

<div class="NumList2" madcap:autonum="5. ">

In the *Client* field, click the drop-down list and select the applicable client.

</div>

<div class="Bullet2">

If the applicable client is not shown, click the link to check the client's status.

</div>

<div class="NumList2" madcap:autonum="6. ">

In the *IP Address* field, enter the IP Address for the client. The IP range can be a single IP, multiple IPs (separated by a space), CIDR Notation or IP with Range.

</div>

<div class="NumList2" madcap:conditions="Default.OldUI" madcap:autonum="7. ">

When all selections/entries are made, click **Add**.

</div>

<div class="NumList2" madcap:conditions="Default.NewUI" madcap:autonum="8. ">

When all selections/entries are made, click **Save**.

</div>
