# Custom BSM Views

Use this function to customize the display properties of Business Service Management (BSM).

<div class="NumList1" madcap:conditions="Default.OldUI" madcap:autonum="1. ">

In the main window, select **Admin /> SACM /> BSM View**. The BSM View window displays.

</div>

<div class="Ind50" madcap:conditions="Default.OldUI">

<span class="image placeholder" original-image-src="../Resources/Images/Admin%20-%20SACM/bsm-view-old.PNG" original-image-title="" style="width: 680px;height: 114px;"></span>

</div>

<div class="NumList2" madcap:conditions="Default.OldUI" madcap:autonum="2. ">

From the *Select Actions* drop-down list, choose an action: <span style="background-color: #ffffff;">[New BSM View](Custom-BSM-VIews-New.htm)</span>, <a href="../Common%20Topics/Delete-Remove.htm" target="_popup">Delete</a> or <a href="../Common%20Topics/Export.htm" target="_popup">Export</a>.

</div>

<div class="NumList2" madcap:conditions="Default.NewUI" madcap:autonum="3. ">

To create a new view, click [New BSM View](Custom-BSM-VIews-New.htm).

</div>
