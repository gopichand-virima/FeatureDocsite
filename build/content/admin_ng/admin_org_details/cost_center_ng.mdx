# Cost Center

Use this function to define the cost centers.

In the main window, click **Admin > Organizational Details > Cost Center**. The Cost Center window displays.

### New Cost Center

- From the _Select Actions_ drop-down list, choose `New Cost Center`.
- Enter the _Name_, _Account Number_, and _Code_ for the cost center.
- To associate a _Manager_, _Location_ or _Parent_, click **Add** then search for and select the applicable item.
- Specify the _Valid From_ and _Valid To_ dates for this cost center to be active.
- When all selections/entries are made, click **Add**.

### Edit Cost Center

Not all functions include an option to edit record line items shown in the main window.

- Navigate to the window containing the record to edit.
- Click the line containing the record. The applicable window or dialog box displays.
- Make the necessary modifications.
- Click **Save** or **Add**, as applicable.

When editing an existing item, other options may be available, such as adding [Tasks](/itsm/release_management) or [Comments](/application_overview/comments).

The options change based upon the function; therefore, not every option may be shown.

### Delete Cost Center

Deleting is a permanent action and cannot be undone. Deleting may affect other functionality and information.

- Click the line that contains the item to delete.
- From the _Select Actions_ drop-down list, choose **Delete**. If a confirmation message is displayed, take appropriate action.

### Export Cost Center

Use the Export function to export the data shown in the main window.

- From the _Select Actions_ drop-down list, choose **Export**.
- Select the desired format (.csv or .xlsx).
- The file downloads to your local system based upon your browser settings.
