# **Rescan**

- Select at least one record with an IP address.
- From the **Select Actions** menu, click **Rescan**.
- If any CI without an IP address is selected, the system will alert: **"Please select a CI with an IP address."**
- A rescan dialog box opens and user should select the following to initiate rescanning,
  - Choose the **client** (discovery server)
  - Choose the **scan type** (e.g., Deep Host Scan)
  - Click**Scan**
- The system initiates a discovery scan for the selected records.
- A new scan run is logged under **Recent Scans**. Refresh to view updated scan status.
- Scan results appear in **Discovered Items**, not **Imported Assets**.
- The original imported asset record remains unchanged unless manually deleted or moved to CMDB.
