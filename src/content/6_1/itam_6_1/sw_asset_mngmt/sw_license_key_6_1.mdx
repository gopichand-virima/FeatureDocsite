# Software License Keys

Use this function to configure a license key for a specific software asset.

In the navigation pane, select **ITAM /> Software Asset Management /> Software License Keys**. The Software License Keys window displays.

<span class="image placeholder" original-image-src="../../Resources/Images/ITAM/sw-lic-key-main-old.PNG" original-image-title=""></span>

<span class="image placeholder" original-image-src="../../Resources/Images/ITAM/sw-lic-key-main-new.PNG" original-image-title=""></span>

New Software License Key

<div class="NumList2" madcap:conditions="Default.OldUI" madcap:autonum="1. ">

In the *Select Actions* drop-down list, choose **New**. The Software License Key dialog box displays.

</div>

<div class="Ind50" madcap:conditions="Default.OldUI">

<span class="image placeholder" original-image-src="../../Resources/Images/ITAM/sw-lic-key-dialog-box-old.PNG" original-image-title=""></span>

</div>

<div class="NumList1" madcap:conditions="Default.NewUI" madcap:autonum="1. ">

Click **New**. The Software License Key dialog box displays.

</div>

<div class="Ind50" madcap:conditions="Default.NewUI">

<span class="image placeholder" original-image-src="../../Resources/Images/ITAM/sw-lic-key-dialog-box-new.PNG" original-image-title=""></span>

</div>

<div class="NumList2" madcap:conditions="Default.NewUI" madcap:autonum="2. ">

Enter the *Asset Name* and *License Key* for the software asset.

</div>

<div class="NumList2" madcap:conditions="Default.NewUI" madcap:autonum="3. ">

Click **Add**.

</div>

Edit Software License Key

The following information and sub-functions are shown in this window:

<div class="Bullet1">

**Used On**. Lists the assets with the corresponding Software License key.

</div>

<div class="Bullet1">

**ITSM**. Displays and allows for the creation of [Incidents](../../ITSM/Incident-Mngmt/Incidents.htm), [Requests](../../ITSM/Request-Fulfillment/Requests.htm), [Changes](../../ITSM/Change-Mngmnt/Changes.htm), [Problems](../../ITSM/Problem-Mngmt/Problems.htm), [Known Errors](../../ITSM/Problem-Mngmt/Known-Errors.htm), [Releases](../../ITSM/Release-Mngmt/Releases.htm), and [Knowledge](../../ITSM/Knowledge-Mngmt/About-Knowledge-Mngmnt.htm).

</div>

<div class="Bullet1">

**Relationships**. Relates the software asset to an existing *Source* and *Target* and view the Business Service Map.

</div>

<div class="NumLista" madcap:autonum="a. ">

Click the **Add Source** or **Add Target** button. The Add Relation dialog box displays.

</div>

<div class="Ind75" madcap:conditions="Default.OldUI">

<span class="image placeholder" original-image-src="../../Resources/Images/ITAM/sw-lic-key-add-relation-old.PNG" original-image-title=""></span>

</div>

<div class="numListb" madcap:autonum="b. ">

Select the blueprint, and search for the applicable asset.

</div>

<div class="numListb" madcap:autonum="c. ">

In the *Choose Relation* field, click the drop-down list and choose the relation.

</div>

<div class="numListb" madcap:autonum="d. ">

Click **Add**.

</div>

<div class="Bullet1">

**SLA**. Configures response and resolution times for the Software License Key if marked as Incident, Requests, Changes, or Problems for resolution. Specify the *Response* and *Resolution Time* as well as the *Support* Levels. Then, configure the *Escalate* parameters.

</div>

<div class="Ind50" madcap:conditions="Default.OldUI">

<span class="image placeholder" original-image-src="../../Resources/Images/ITAM/sw-lic-key-sla-old.PNG" original-image-title=""></span>

</div>

<div class="Bullet1">

**Maintenance**. Set of activities defined to conserve an asset in its original condition. *Refer to [CI: Maintenance](../../ITSM/Config%20Mngmt/Config%20Items/Config%20Item%20Details/Config-Items-Maintenance.htm).*

</div>

<div class="Bullet1">

**Audits**. Analyzes the product or its process to assess compliance with defined specifications or the standards of the deliverable.  
  
To add an Audit:

</div>

<div class="NumLista" madcap:autonum="a. ">

Click the **Add Audit** button.

</div>

<div class="Ind50" madcap:conditions="Default.OldUI">

<span class="image placeholder" original-image-src="../../Resources/Images/ITAM/sw-lic-key-audit-old.PNG" original-image-title=""></span>

</div>

<div class="NumListb" madcap:autonum="b. ">

Enter a *Name* and *Description*.

</div>

<div class="NumListb" madcap:autonum="c. ">

In the *Auditor* field, click **Add,** then search for and select a user to function as the Auditor.

</div>

<div class="NumListb" madcap:autonum="d. ">

Select the *Recurring Frequency*.

</div>

<div class="NumListb" madcap:autonum="e. ">

Enter the *Duration (in days)* to indicate the length of time for the audit.

</div>

<div class="NumListb" madcap:autonum="f. ">

When all selections/entries are made, click **Add**.

</div>
