# Import From AWS

Use this function to enable users to import the required AWS service into the Discovery application.

From the *Select Actions* drop-down list, choose **Import**. The Import AWS dialog box displays.

In the *Name* field, enter a name for the import action.

In the *Credential* field, click the drop-down list and choose the appropriate credential to use to leverage the AWS service.

Select all the required services present to be imported from the AWS label.

Click **Add** to update the AWS import in the application.
