# Testing AD Configuration

1. Open the AD Configuration line item.

2. From the **Select Actions** drop-down list, Select [Test AD Configuration.]{.Strong}

1. ![AD User Import Logs 1 4](../../Resources/Images/Discovery/AD-User-Import-Logs/AD User Import Logs_1_4.png)

3. If you have set the values of the attributes correctly as per your organization\'s AD structure, then the connection will be a success. Otherwise, it will fail.

1. ![AD User Import Logs 1 5](../../Resources/Images/Discovery/AD-User-Import-Logs/AD User Import Logs_1_5.png)
