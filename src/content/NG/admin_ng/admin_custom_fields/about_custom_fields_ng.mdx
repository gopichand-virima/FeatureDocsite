# About Custom Fields

The Custom Fields module allows administrators to extend Virima's data model by adding custom attributes to various record types.

Custom Fields enable organizations to capture additional information specific to their business needs without modifying the core application.

In the main window, click **Admin > Custom Fields**. The Custom Fields window displays.

## Overview

Custom Fields provide:

- Extension of standard data model
- Custom data capture for any module
- Flexible field types and validations
- Conditional field display logic
- Integration with workflows and reports
- Import/export of custom field data

## Supported Field Types

**Text Fields**:

- Single-line text
- Multi-line text (text area)
- Rich text editor
- Formatted text

**Numeric Fields**:

- Integer
- Decimal
- Currency
- Percentage

**Date/Time Fields**:

- Date only
- Date and time
- Time only
- Duration

**Selection Fields**:

- Dropdown (single select)
- Multi-select dropdown
- Radio buttons
- Checkboxes

**Reference Fields**:

- User lookup
- Group lookup
- Asset reference
- Custom table reference

**Special Fields**:

- URL/Hyperlink
- Email address
- Phone number
- File attachment
- Image upload
- Barcode/QR code
- Geolocation

## Field Properties

**Basic Properties**:

- Field name
- Field label
- Field type
- Description
- Help text

**Validation**:

- Required field
- Unique values
- Min/max length
- Min/max value
- Regular expression pattern
- Custom validation rules

**Display**:

- Field order
- Read-only
- Hidden
- Conditional visibility
- Default value
- Placeholder text
