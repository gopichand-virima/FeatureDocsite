# Creating a Scheduled Intune Import

1. From the **Select Actions** dropdown, choose **New Scheduled Intune Import**.

2. Configure the following:

**Name:** Provide a name.

**Credential:** Select the Intune credential.

**Import Frequency:** Set how often the import runs.

**Active:** Enable the schedule.

**Timezone and Timing:** Specify when the job should execute.

1. Click **Add**.
