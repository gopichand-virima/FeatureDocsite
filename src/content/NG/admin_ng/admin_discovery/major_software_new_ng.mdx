# New Major Software

Use this function to add information for a new software.

 -- ----- ----------------------------------------------------------------------------------------------------------
 1. In the main window, select **Admin \> Discovery \> Major Software**. The Major Software window displays.
 -- ----- ----------------------------------------------------------------------------------------------------------

 -- ----- -------------------------------------------------------------------------------------
 2. In the main window, click the gear icon and select **Discovery \> Major Software**.
 -- ----- -------------------------------------------------------------------------------------

 -- ----- -----------------------------------------------------------------------------------------------------------------------------------
 3. From the *Select Actions* drop-down list, choose **New Major Software**. A new window displays with the **Details** tab selected.
 -- ----- -----------------------------------------------------------------------------------------------------------------------------------

 -- ----- ---------------------------------------------------------------------------------------------------
 4. Click the **New Major Software** button. A new window displays with the **Details** tab selected.
 -- ----- ---------------------------------------------------------------------------------------------------

 -- ----- -----------------------------------------------
 5. Complete the tabs according to the following:
 -- ----- -----------------------------------------------

::: Bullet2
[Details](Major-Software-Details.mdx){.MCTopicPopup .MCTopicPopupHotSpot .a}

::: Bullet2
[Tasks](/itsm/release_mngmt/tasks_6_1.mdx){.MCTopicPopup .MCTopicPopupHotSpot .a}

::: Bullet2
[Comments](/discovery/comments_6_1.mdx){.MCTopicPopup .MCTopicPopupHotSpot .a}

::: Bullet2
[Attachments](/discovery/attachments_6_1.mdx){.MCTopicPopup .MCTopicPopupHotSpot .a}

 -- ----- ------------------------------------------------------
 6. When all selections/entries are made, click **Add**.
 -- ----- ------------------------------------------------------
