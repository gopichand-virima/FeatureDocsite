# Testing AD Configuration

1.  Open the newly added Azure AD Configuration line item.

2.  From the Select Actions drop-down list, Select <span class="Strong" style="font-weight: bold;">Test Azure AD Configuration.</span>

<span class="image placeholder" original-image-src="../../Resources/Images/Discovery/Azure-AD-Users-Import-Logs/Azure%20AD%20User%20Import%20Logs_6.png" original-image-title=""></span>

3.  If you have set the values of the attributes correctly as per your organization's Azure AD structure, then the connection will be a success. Otherwise, it will fail.
