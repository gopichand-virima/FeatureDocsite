# CI: Relationships

Use this function to manage the relationships between sources and targets. The Discovery application detects the Connected To, Runs On, Communicates With, Installed On, Component Of, Physically Connected To, Attaches and Used By relationships during a scan.

|     |                                                                       |
|-----|-----------------------------------------------------------------------|
|     | Refer to [CMDB Process Jobs](CMDB-Process-Jobs.htm) for more details. |

<div class="NumList1" madcap:conditions="Default.OldUI" madcap:autonum="1. ">

In the navigation pane, select **ITSM /> Configuration Management /> CMDB**. The Configuration Items window displays.

</div>

<div class="NumList1" madcap:conditions="Default.NewUI" madcap:autonum="1. ">

In the navigation pane, select **Configuration Management /> CMDB**. The Configuration Items window displays.

</div>

<div class="NumList2" madcap:autonum="2. ">

Select a record in the list. A new window opens and the **Details** tab displays.

</div>

<div class="NumList2" madcap:autonum="3. ">

Click the **Relationships** tab. A list of all existing relationships displays.

</div>

<div class="Ind50">

<span class="image placeholder" original-image-src="../../../../Resources/Images/ITSM/add-relationship.PNG" original-image-title="" madcap:conditions="Default.OldUI"></span>

</div>

<div class="Ind50">

<span class="image placeholder" original-image-src="../../../../Resources/Images/ITSM/add-relationship-new.PNG" original-image-title="" madcap:conditions="Default.NewUI"></span>

</div>

<div class="NumList2" madcap:autonum="4. ">

Choose any of the following: **Add Source**, **Add Target**, or **<a href="Business-Service-Map.htm" style="font-weight: normal;">Business Service Map</a>**.

</div>

<div class="NumList2" madcap:autonum="5. ">

To remove any relationship, select it in the list, and click **Delete**.

</div>

## Add Source or Add Target

<div class="NumList1" madcap:autonum="1. ">

Click the **Add Source** or **Add Target button**. The Add Relation dialog box displays.

</div>

<div class="Ind50">

<span class="image placeholder" original-image-src="../../../../Resources/Images/ITSM/add-relation-dialog-box.PNG" original-image-title="" madcap:conditions="Default.OldUI"></span>

</div>

<div class="Ind50">

<span class="image placeholder" original-image-src="../../../../Resources/Images/ITSM/add-relation-dialog-box-new.PNG" original-image-title="" madcap:conditions="Default.NewUI"></span>

</div>

<div class="NumList2" style="color: #000000;" madcap:conditions="Default.OldUI" madcap:autonum="2. ">

From the drop-down list, select either the **Source** or **Target**.

</div>

<div class="NumList2" style="color: #000000;" madcap:conditions="Default.NewUI" madcap:autonum="3. ">

From the drop-down list, select the applicable blueprint from the available list.

</div>

<div class="NumList2" style="color: #000000;" madcap:autonum="4. ">

In the *Search* field, enter the search criteria to locate the corresponding asset.

</div>

<div class="NumList2" style="color: #000000;" madcap:conditions="Default.OldUI" madcap:autonum="5. ">

From the *Choose Relationship* drop-down list, select the relationship the Source or Target will have such as Assigned To, Belongs To, and so forth.

</div>

<div class="NumList2" style="color: #000000;" madcap:conditions="Default.NewUI" madcap:autonum="6. ">

From the *Relation* drop-down list, select the relationship the Source or Target will have such as Assigned To, Belongs To, and so forth.

</div>

<div class="NumList2" style="color: #000000;" madcap:conditions="Default.NewUI,Default.OldUI" madcap:autonum="7. ">

When all selections are made, click **Add**.

</div>
