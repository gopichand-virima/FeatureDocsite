# Email Preferences

Configure your email notification settings and preferences within Virima.

## Overview

Email Preferences allow you to control when and how you receive email notifications from the Virima platform. Customize your notification settings to stay informed without being overwhelmed.

## Accessing Email Preferences

### Navigate to Preferences
1. Click your **User Profile** icon
2. Select **Preferences** or **Settings**
3. Click **Email Preferences** or **Notifications**
4. Configure your settings
5. Click **Save** to apply changes

## Notification Types

### System Notifications
- Account updates and security alerts
- System maintenance announcements
- Version updates and new features
- Platform status changes

### Workflow Notifications
- Task assignments
- Approvals required
- Workflow completions
- Status changes

### Activity Notifications
- Comments and mentions
- Record updates
- Team collaboration
- Shared items

## Notification Settings

### Frequency Options
Choose how often you receive notifications:
- **Immediate**: Receive emails as events occur
- **Daily Digest**: One summary email per day
- **Weekly Digest**: One summary email per week
- **Disabled**: Turn off specific notification types

### Notification Categories
Configure preferences for each category:
- Incidents assigned to you
- Changes awaiting approval
- Asset updates
- Discovery scan results
- Report completions
- CMDB changes

## Email Format

### HTML vs. Plain Text
- **HTML Email**: Rich formatting with images and links
- **Plain Text**: Simple text-only format
- Choose based on email client compatibility

### Content Preferences
- Include record details in email
- Show only summary information
- Attach relevant documents
- Include quick action links

## Advanced Settings

### Custom Rules
Create custom notification rules:
- Set conditions for notifications
- Define priority-based alerts
- Create module-specific rules
- Configure team-based notifications

### Time-Based Settings
- Set quiet hours (no notifications during specified times)
- Configure timezone preferences
- Weekend and holiday settings
- Business hours only notifications

## Managing Email Volume

### Reduce Email Overload
- Consolidate similar notifications
- Use digest format for non-urgent items
- Disable notifications for low-priority items
- Set filters for specific criteria

### Priority Notifications
- Mark critical notification types
- Ensure important alerts are received immediately
- Filter by urgency and impact
- Customize sound and visual alerts

## Mobile Notifications

### Mobile Email Settings
- Optimize for mobile devices
- Push notification integration
- Mobile app notification preferences
- SMS alerts for critical items

## Unsubscribe Options

### Opt-Out Management
- Unsubscribe from specific notification types
- Maintain critical system notifications
- Re-subscribe at any time
- Manage subscription preferences

## Testing Email Settings

### Send Test Email
1. Configure your preferences
2. Click **Send Test Email**
3. Verify email receipt and formatting
4. Adjust settings as needed

## Best Practices

- Review preferences regularly
- Use digest format for high-volume notifications
- Keep critical alerts enabled
- Set quiet hours for work-life balance
- Test settings after changes
- Document team notification standards
- Train new users on preferences
