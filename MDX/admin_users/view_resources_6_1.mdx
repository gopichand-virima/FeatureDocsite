# View Resources

Use this function to view details for a user such as tasks performed, hours spent on the task, status of a task (such as *Active*, *Started*, and *Not Started*), and dates on which tasks occurred.

- In the main window, select **Admin > Users > Users**. The Access Management window displays.
- From the *Select Actions* drop-down list, choose **View Resources**. A new window opens and provides the relevant details.

!

- To toggle between Projects and Miscellaneous Task, use the drop-down list.
- To view users who have reported, click **Reported Users**.
- To close the window, click the **X** in the right upper corner of the window.
