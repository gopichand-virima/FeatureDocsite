# CI: Maintenance

Use this function to schedule a maintenance task for a CI such as reboot, shutdown or run a specific command.

<div class="NumList1" madcap:conditions="Default.OldUI" madcap:autonum="1. ">

From the navigation pane, select **ITSM /> Configuration Management /> Configuration Items**. The Configuration Items window displays.

</div>

<div class="NumList1" madcap:conditions="Default.NewUI" madcap:autonum="1. ">

From the navigation pane, select **Configuration Management /> Configuration Items**. The Configuration Items window displays.

</div>

<div class="NumList2" madcap:conditions="Default.OldUI,Default.NewUI" madcap:autonum="2. ">

Select a record in the list. A new window opens and the **Details** tab displays.

</div>

<div class="NumList2" madcap:conditions="Default.OldUI" madcap:autonum="3. ">

Click the **Maintenance** tab. The Add Maintenance Window displays.

</div>

<div class="NumList2" madcap:conditions="Default.NewUI" madcap:autonum="4. ">

Click the **New CI Maintenance** button. The Add Maintenance Window displays.

</div>
