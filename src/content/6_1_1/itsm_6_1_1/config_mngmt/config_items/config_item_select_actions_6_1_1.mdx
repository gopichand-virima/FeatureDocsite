# Select Actions

Use this function to perform an action on the selected asset.

From the *Select Actions* drop-down list, choose any of the following actions.


The drop-down list actions change based on the type of blueprint. Therefore, not all actions may be listed below as blueprints may be added or changed from time to time.

- [Enable/Disable Editing](/common_topics/enable_disable_editing_6_1_1.mdx)
- [Delete](/common_topics/delete_remove_6_1_1.mdx)
- [Get Config File](/itsm/config_mngmt/config_items/config_item_details/details_get_config_file_6_1_1.mdx)
- [Merge CI](/itsm/config_mngmt/config_items/config_item_details/details_merge_ci_6_1_1.mdx)
- [Process ADM](/common_topics/process_adm_6_1_1.mdx)
- [Rescan Now](/itsm/config_mngmt/config_items/config_item_details/details_rescan_now_6_1_1.mdx)
