## Left Navigation Pane

The left navigation pane includes the following:

::: Bullet1
[My Dashboard](My%20Dashboard/My-Dashboard-Old.mdx) displays data content information with a customization option.

::: Bullet1
[Discovery Scan](Discovery/About-Discovery-Scan.mdx) manages scanning activities (running, scheduling, etc.) and importing data (such as from AWS and Azure).

- |
| Column 1 | *ITAM* displays when CMDB and other modules (such as hardware assets and contract and vendor management) are enabled. |
| --- | :---: |
| | |
| | ::: Bullet1 |
| | *ITSM* displays when CMDB, Incidents/Changes/Problems and other ITSM modules are enabled. |
| | ::: |
+---+-----------------------------------------------------------------------------------------------------------------------+

::: Bullet1
[Vulnerability Management](Vulnerability%20Managment/Vulnerability-Management.mdx) identifies, evaluates, and reports security vulnerabilities in CIs present inside CMDB.

::: Bullet1
[Reports](/reports/reports_6_1.mdx) generates both canned and custom ad hoc reports.

## Admin Window

The administrative functions include the following:

::: Bullet1
[Organizational Details](Admin%20-%20Org%20Details/About-Org-Details.mdx) specifies business-related details such as departments, cost centers, locations, designations, operational hours, and holidays.

::: Bullet1
[Discovery](Admin%20-%20Discovery/Admin-Discovery.mdx) configures clients, defines credentials, defines major software, manages processes, and other discovery-related functionality (scan configuration, monitoring profile, and application mapping).

::: Bullet1
[SACM](Admin%20-%20SACM/Admin-SACM.mdx) handles blueprints, relationship types, confidence configuration, and software-related functions.

::: Bullet1
[Users](Admin%20-%20Users/Admin-Users.mdx) creates users, roles, groups, time tracking reports and allows for AD import/authentication, and Azure AD and SAML configuration.

::: Bullet1
[Incident Management](Admin%20-%20Incident%20Mngmnt/About-Incident-Mngmnt.mdx) defines escalations, categories, error codes, the incident model, priority calculator, states/sub states, and timescale.
