# Regular Scan

The <span class="Strong">Regular Scan</span> section conducts a full network scan and manages the IP address ranges to include or exclude from the scan.

<span class="image placeholder" original-image-src="../../Resources/Images/Discovery/IPAM-Networks/IPAM%20Functions%20Overview_2.png" original-image-title=""></span>

Fields and settings for configuring a <span class="Strong">Regular Scan</span>:

- <span class="Strong">Name:</span> Provide a name for the scan.

- <span class="Strong">Probe:</span> Select the type of probe.

- <span class="Strong">Client:</span> Select the client to scan (ensure the client is operational).

- <span class="Strong">IP Range:</span> Specify the IP range to scan (e.g., <span class="Strong">192.168.34.0/24</span>).

  - <span class="Strong">Exclude IP Range:</span> Define specific IP ranges to exclude from the scan.

  - The maximum number of IPs allowed per scan is 1024 (Subnet /22). You can enter IPs in various formats:

    - Single or multiple IPs from different subnets: e.g., <span class="Strong">192.168.48.1 192.168.49.2</span>.

    - Multiple IPs from the same subnet: e.g., <span class="Strong">192.168.48.1,2,3,4</span>.

    - A range of IPs in the same subnet: e.g., <span class="Strong">192.168.48.1-50</span>.

    - A single subnet: e.g., <span class="Strong">192.168.48.0/22</span>.

    - Multiple subnets: e.g., <span class="Strong">192.168.48.0/23 192.168.49.0/23</span>.

    - Combination of subnet and IPs: e.g., <span class="Strong">192.168.48.0/23 192.168.48.1</span>.

  - The total number of IPs should not exceed 1024.

- <span class="Strong">Location (Optional):</span> Specify the scan location.

- <span class="Strong">Scan Type:</span> Choose between <span class="Strong">Immediate</span> or <span class="Strong">Scheduled</span> scan.

  - <span class="Strong">Immediate:</span> The scan starts immediately.

  - <span class="Strong">Scheduled:</span> The scan runs at a scheduled time.

- <span class="Strong">Send Scan Report To (Optional):</span> Provide an email address for receiving scan results.

After the scan, details will appear in <span class="Strong">Recent Scans</span>. Upon a successful scan, results will move to <span class="Strong">Discovered Items</span> for further management.
