# Audits

The <span class="Strong" style="font-weight: bold;">Audits</span> tab in the CMDB allows users to define and manage audit schedules for Configuration Items (CIs). Audits ensure the <span class="Strong">integrity and accuracy</span> of CI data through periodic checks.

<span class="image placeholder" original-image-src="../Resources/Images/CMDB%20-%20Left-Nav/Configuration%20Management%20Database_65.png" original-image-title=""></span>

- Click on <span class="Strong" style="font-weight: bold;">New Audit</span> to add new audit.

  <span class="image placeholder" original-image-src="../Resources/Images/CMDB%20-%20Left-Nav/Configuration%20Management%20Database_66.png" original-image-title=""></span>

<!-- -->

- Enter the details, then click <span class="Strong" style="font-weight: bold;">Add</span> to save the new audit.

- Select one audit and click <span class="Strong" style="font-weight: bold;">Delete</span> to delete the audit.

<span class="Strong" style="font-weight: bold;">Key functionalities:</span>

- <span class="Strong" style="font-weight: bold;">Audit name</span>: Assign a meaningful label to indicate the audit's purpose (e.g., "detail check," "OS validation").

- <span class="Strong" style="font-weight: bold;">Add description</span>: Describe the audit's scope or objective.

- <span class="Strong" style="font-weight: bold;">Assign auditor</span>: Designate a user responsible for executing or reviewing the audit. Multiple users can be assigned.

- <span class="Strong" style="font-weight: bold;">Set recurring frequency</span>: Choose how often the audit should recur (daily, weekly, monthly, etc.).

- <span class="Strong" style="font-weight: bold;">Duration (in days)</span>: Specify the audit's validity duration or completion window.

<span class="Strong" style="font-weight: bold;">Example scenario:</span>

For an audit named <span class="Strong" style="font-weight: bold;">“detail check”</span>:

- It will run <span class="Strong" style="font-weight: bold;">daily</span>.

- It has a <span class="Strong" style="font-weight: bold;">duration of 7 days</span>.

- The assigned auditor is <span class="Strong" style="font-weight: bold;">Auditor Name</span>.

- The audit's description is <span class="Strong" style="font-weight: bold;">"check details."</span>

This setup enables timely verification of asset configurations and promotes regular CI hygiene practices.
