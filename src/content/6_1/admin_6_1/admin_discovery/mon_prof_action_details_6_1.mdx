# Monitoring Profile: Action Details

Use this function to specify what how the system handles changes found for an item being monitored. For example, if a configuration file is being monitored, and a change is detected, the action may be to create an incident report or send a notification to a user.

<div class="numlist1" madcap:conditions="Default.OldUI" madcap:autonum="1. ">

In the main window, click **Admin /> Discovery /> Discovery Monitoring Profile**.

</div>

<div class="NumList2" madcap:conditions="Default.NewUI" madcap:autonum="2. ">

In the main window, click the **gear** icon and select **Discovery /> Discovery Monitoring Profile**.

</div>

<div class="NumList2" madcap:conditions="Default.OldUI" madcap:autonum="3. ">

From the *Select Actions* drop-down list, choose **New**. The window updates.

</div>

<div class="NumList2" madcap:conditions="Default.NewUI" madcap:autonum="4. ">

Click **New**.

</div>

<div class="NumList2" madcap:autonum="5. ">

Navigate to the **Action Details** section.

</div>

<div class="Ind50" madcap:conditions="Default.OldUI">

<span class="image placeholder" original-image-src="../Resources/Images/Admin%20-%20Discovery/discovery-mon-prof-action-details-old.PNG" original-image-title=""></span>

</div>

<div class="Ind50" madcap:conditions="Default.NewUI">

<span class="image placeholder" original-image-src="../Resources/Images/Admin%20-%20Discovery/discovery-mon-prof-action-details-new.PNG" original-image-title=""></span>

</div>

<div class="NumList2" madcap:autonum="6. ">

In the *Target Module* field, click the drop-down list and select the required ITSM module.

</div>

<div class="NumList2" madcap:autonum="7. ">

In the *Create From Model* field, select the template to be used for the selected module.

</div>

<div class="NumList2" madcap:autonum="8. ">

In the *Property* fields, click the drop-down list, select the applicable property, and then select a matching value (required).

</div>
