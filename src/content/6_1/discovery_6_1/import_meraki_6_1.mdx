# Import from Meraki

Use this function to import Meraki devices such as Access Points, Switches, Firewalls, and Routers, which can be managed inside the Meraki dashboard.

In the navigation pane, select **Discovery Scan > Import from Meraki**. The Import from Meraki window displays.

From the *Select Actions* drop-down list, choose **Import**. The Import From Meraki dialog box displays.

Enter a *Name*.

In the *Credential* field, click the drop-down list, and select the applicable Meraki credential.

When all selections/entries are made, click **Add**.

Edit Import from Meraki

Use this function to edit the values of the configured parameters and update the information in the application.

In the navigation pane, select **Discovery Scan > Import from Meraki**.

Select the record to edit. The Edit Report window displays the configured details into the following categories:Â **All**, **Authorized**, and **Unauthorized**.

- *Authorized* represents all assets that are already present inside CMDB.
- *Unauthorized* represents new assets.

Click the *Log* tab to view all activity related to this import including issues discovered.

To move a record CMDB, do the following:

Select a record.

Click **Move to CMDB**.

In the Alert Message window, click **Continue** to move the record. The process runs in the background and no further action is required.
