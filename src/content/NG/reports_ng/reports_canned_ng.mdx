# Canned Reports

Use this function to generate canned reports for which formatting has been applied, making them presentable for display and distribution.

In the navigation pane, select **Reports \> Canned Reports**. The Reports window displays.

![canned report main old](../Resources/Images/Reports/canned-report-main-old.PNG)

![arrow border](../Resources/Images/arrow-border.PNG)

[[[Closed]{.image .placeholder .MCDropDown_Image_Icon original-image-src="../../Skins/Default/Stylesheets/Images/transparent.gif" original-image-title="" height="12" width="16" Canned Report](javascript:void(0)){.MCDropDownHotSpot .dropDownHotspot .MCDropDownHotSpot_ .MCHotSpotImage aria-expanded="false"}]{.MCDropDownHead .dropDownHead}

From the *Select Actions* drop-down list, choose [New Canned Report](Reports-New-Canned.mdx).

To create a new report, click [New Report](Reports-New-Canned.mdx).

[[[Closed]{.image .placeholder .MCDropDown_Image_Icon original-image-src="../../Skins/Default/Stylesheets/Images/transparent.gif" original-image-title="" height="12" width="16" Canned Reports](javascript:void(0)){.MCDropDownHotSpot .dropDownHotspot .MCDropDownHotSpot_ .MCHotSpotImage aria-expanded="false"}]{.MCDropDownHead .dropDownHead}

 --- -----------------------------------------------------------------------------------------
   Not all functions include an option to edit record line items shown in the main window.
 --- -----------------------------------------------------------------------------------------

 -- ----- -------------------------------------------------------
 1. Navigate to the window containing the record to edit.
 -- ----- -------------------------------------------------------

 -- ----- -------------------------------------------------------------------------------------
 2. Click the line containing the record. The applicable window or dialog box displays.
 -- ----- -------------------------------------------------------------------------------------

 -- ----- -----------------------------------
 3. Make the necessary modifications.
 -- ----- -----------------------------------

 -- ----- -------------------------------------------
 4. Click **Save** or **Add**, as applicable.
 -- ----- -------------------------------------------

[[[Closed]{.image .placeholder .MCDropDown_Image_Icon original-image-src="../../Skins/Default/Stylesheets/Images/transparent.gif" original-image-title="" height="12" width="16" Canned Reports](javascript:void(0)){.MCDropDownHotSpot .dropDownHotspot .MCDropDownHotSpot_ .MCHotSpotImage aria-expanded="false"}]{.MCDropDownHead .dropDownHead}

 --- --------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------
   Deleting is a permanent action and cannot be undone. Deleting may affect other functionality and information in the application such as data in configured reports, fields in windows, selectable options, etc. Therefore, be sure to understand the potential effects before making a deletion.
 --- --------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------

 -- ----- --------------------------------------------------
 1. Click the line that contains the item to delete.
 -- ----- --------------------------------------------------

 -- ----- ------------------------------------------------------------------------------------------------------------------------------------------------------
 2. From the *Select Actions* drop-down list, choose **Delete**. If a confirmation message is displayed, type **Delete** and click **Delete**to confirm.
 -- ----- ------------------------------------------------------------------------------------------------------------------------------------------------------
