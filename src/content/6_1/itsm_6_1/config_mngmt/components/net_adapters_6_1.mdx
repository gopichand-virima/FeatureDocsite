# Network Adapters

<div class="NumList1" madcap:autonum="1. ">

While viewing an asset record, click **Components**.

</div>

<div class="NumList2" madcap:autonum="2. ">

Click **Network Adapters**.

</div>

Add a New Network Adapter

<div class="NumList1" madcap:autonum="1. ">

Click **Add**. The Network Adapter dialog box displays.

</div>

<div class="Ind50" madcap:conditions="Default.OldUI">

<span class="image placeholder" original-image-src="../../../Resources/Images/Discovery/network-adapter-add-new.PNG" original-image-title="" style="width: 473px;height: 151px;"></span>

</div>

<div class="Ind50" madcap:conditions="Default.NewUI">

<span class="image placeholder" original-image-src="../../../Resources/Images/Components/network-adapters-new-add.PNG" original-image-title="" style="width: 473px;height: 207px;"></span>

</div>

<div class="NumList2" madcap:autonum="2. ">

Click **New Network Adapter.**

</div>

<div class="NumList2" madcap:autonum="3. ">

Enter the *MAC Address*, *Network Adapter Name*, *IP Address*, and *Asset Names*. These are all required fields.

</div>

<div class="NumList2" madcap:autonum="4. ">

Click **Add**.

</div>

Modify Existing Network Adapter

<div class="NumList1" madcap:autonum="1. ">

Click **Existing Network Adapter**.

</div>

<div class="Ind50" madcap:conditions="Default.OldUI">

<span class="image placeholder" original-image-src="../../../Resources/Images/Discovery/network-adapter-modify-existing.PNG" original-image-title="" style="width: 452px;height: 154px;"></span>

</div>

<div class="Ind50" madcap:conditions="Default.NewUI">

<span class="image placeholder" original-image-src="../../../Resources/Images/Components/network-adapters-new-existing.PNG" original-image-title="" style="width: 458px;height: 310px;"></span>

</div>

<div class="NumList2" madcap:autonum="2. ">

Enter the search criteria to locate the existing adapter.  *See also* <a href="../../../Common%20Topics/Advanced-Search.htm" style="font-style: italic;">Advanced Search</a>*.*

</div>

<div class="NumList2" madcap:autonum="3. ">

Select the applicable adapter from the results list.

</div>

<div class="NumList2" madcap:autonum="4. ">

When all selections are made, click **Save**.

</div>

Edit the Network Adapter Details

<div class="NumList1" madcap:autonum="1. ">

While viewing the list of existing adapters, select the adapter to edit. The windows refreshes and the details for the adapter displays.

</div>

<div class="NumList2" madcap:autonum="2. ">

Make any necessary changes to the existing information. The changes are automatically saved. *Refer to* <a href="CI-Comp-List.htm" style="font-style: italic;">CI Components</a>*.*

</div>

<div class="NumList2" madcap:autonum="3. ">

To add a contact, navigate to the Contacts section and click **Add**. The Contacts dialog box displays.

</div>

<div class="Ind50" madcap:conditions="Default.OldUI">

<span class="image placeholder" original-image-src="../../../Resources/Images/Components/network-adapaters-contacts.PNG" original-image-title="" style="width: 345px;height: 173px;"></span>

</div>

<div class="Ind50">

<span class="image placeholder" original-image-src="../../../Resources/Images/Components/network-adapaters-contacts-new.png" original-image-title="" madcap:conditions="Default.NewUI" style="width: 356px;height: 274px;"></span>

</div>

<div class="NumLista" madcap:autonum="a. ">

In the *Choose Relation* field, click the drop-down list and select how this contact should be related to this asset.

</div>

<div class="numListb" madcap:autonum="b. ">

In the search field, enter the search criteria.

</div>

<div class="numListb" madcap:autonum="c. ">

Select a contact from the results list, moving it to the selected list.

</div>

<div class="numListb" madcap:autonum="d. ">

When all selections are made, click **Save**.

</div>
