# Rescan Now

Use this function to rescan selected assets.

 -- ----- ---------------------------------------------------------------------------------------------------------
 1. While viewing the **Details** tab, click the **Rescan Now** button. The Rescan Now dialog box displays.
 -- ----- ---------------------------------------------------------------------------------------------------------

 --- -------------------------------------------------------------------------------
   The IP Address for the selected asset is pre-populated and cannot be changed.
 --- -------------------------------------------------------------------------------

::: {.Ind50
![rescan now dialog box 375x168](../../../Resources/Images/ITSM/rescan-now-dialog-box_375x168.png)

::: {.Ind50
![rescan now dialog box new 368x277](../../../Resources/Images/ITSM/rescan-now-dialog-box-new_368x277.png)

 -- ----- -------------------------------------------------------------------------------------
 2. In the *Choose a client* field, click the drop-down and select an available client.
 -- ----- -------------------------------------------------------------------------------------

 -- ----- ---------------------------------------------------------------------------------
 3. In the *Client* field, click the drop-down list and select an available client.
 -- ----- ---------------------------------------------------------------------------------

 -- ----- ---------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------
 4. In the *Type of Scan* field, click the drop-down and select the type of scan. *For details on the types of scans available, refer to [Run a Scan](../../../Discovery/Run-a-Scan/Run-a-Scan.mdx).*
 -- ----- ---------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------

 -- ----- ---------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------
 5. When all selections/entries are made, click **Scan**. A confirmation message displays and the window refreshes and displays the [Recent Scans](../../../Discovery/Recent-Scans/Scans-Recent.mdx) information.
 -- ----- ---------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------
