# Software License Key: Details

Use this function to view and modify the details for a software license key.

 -- ----- ----------------------------------------------------------------------------------------------------------
 1. While viewing the Software License key window, from the *Select Actions* drop-down list, choose **New**.
 -- ----- ----------------------------------------------------------------------------------------------------------

 -- ----- --------------------------------------------------------------------------------------------------
 2. Select an existing record. A new window displays with the **Details** tab displayed, by default.
 -- ----- --------------------------------------------------------------------------------------------------

 -- ----- ----------------------------------------------------------------------------
 3. Make the applicable changes to the Asset Primary Information and Contacts.
 -- ----- ----------------------------------------------------------------------------

 -- ----- -------------------------------------------------------
 4. When all selections/entries are made, click **Save**.
