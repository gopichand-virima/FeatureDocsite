# Incident Error Code

Use this function to define an error message for Incident Management to be displayed for incorrect configuration scenarios.

In the main window, click **Admin /> Incident Management /> Incident Error Code**. The Incident Error Code window displays.

<span class="image placeholder" original-image-src="../Resources/Images/Admin%20-%20Incident/incident-error-code-main-old.PNG" original-image-title=""></span>

In the main window, click the **gear** icon and select **Incident Management /> Incident Error Code**.

<span class="image placeholder" original-image-src="../Resources/Images/Admin%20-%20Incident/icon-incident-error-code.PNG" original-image-title=""></span>

The Incident Error Code window displays.

<span class="image placeholder" original-image-src="../Resources/Images/Admin%20-%20Incident/incident-error-code-main-new.PNG" original-image-title=""></span>
