# User Groups

Use this function to configure groups among the existing application users by defining their precise roles.

The following information should be configured before creating a new user group:

- Manager (required)
- Department (optional, but recommended)
- Role (optional, but recommended)


[Role Access](/admin_other/role_access_6_1.mdx) permissions must be set before adding a user group.


In the main window, click **Admin > Users > User Groups**. The User Groups window displays.

New User Group

From the *Select Actions* drop-down list, choose **New Group**. The New User Group window displays.

Complete the *Name*, *Group Email*, and *Description* fields.

For the *Department*, *Manager*, and *Role* fields, click the **Add** button, then search for and make the applicable selections.


In order to add a User Group, the Manager must be a member of the [Department](/admin_org_details/departments_6_1.mdx) being selected or an error message displays.


To add an image, click the **Upload** button and select the applicable image.

When all selections/entries are made, click **Add**.

Member Information for a Group

Once you add a group, you can add one or more members to it.

Click on a group in the list.

In the Details window, click the **Members** tab. The Members window displays.

Click **New User** and search for and select the applicable member(s)*.*

When all selections are made, click **Save**.

Edit User Group Delete User Group

To delete an existing user group:

Click the checkbox for the applicable group.

From the *Select Actions* drop-down list, choose **Delete**.

In the confirmation window, Type **Delete** and click **Delete**.
