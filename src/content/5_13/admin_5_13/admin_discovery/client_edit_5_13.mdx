# Manage Clients

Use this function to edit details or delete a client.

## Edit Client Details

In the main window, select **Admin > Discovery > Client**. The Client window displays.

In the main window, click the **gear** icon and select **Discovery > Client**. The Client window displays.

Select a client record to edit. The Client Details window displays.

Modify any of the following information:

- [Details](/admin_discovery/client_details_5_13.mdx)
- [Discovery Agents](/admin_discovery/client_discovery_agents_5_13.mdx)
- [Tasks](/itsm/release_mngmt/tasks_5_13.mdx)
- [Comments](/discovery/comments_5_13.mdx)
- [Attachments](/discovery/attachments_5_13.mdx)

When all selections/entries are made, click **Save**.
