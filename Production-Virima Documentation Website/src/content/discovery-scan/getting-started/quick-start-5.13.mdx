# Discovery Scan - Quick Start Guide

Start discovering your IT infrastructure in minutes.

## Prerequisites

- Network access to target systems
- Valid credentials for scanning
- Firewall permissions configured
- Discovery module permissions

## Quick Start Steps

### Step 1: Configure Credentials
1. Navigate to Discovery Scan > Credentials
2. Click "Add Credential"
3. Enter credential details (SNMP, WMI, SSH, etc.)
4. Test and save credentials

### Step 2: Define Scan Scope
1. Go to Discovery Scan > Scan Profiles
2. Create new scan profile
3. Specify IP ranges or cloud accounts
4. Select credential sets to use

### Step 3: Run Discovery
1. Click "Run Scan" on your profile
2. Monitor scan progress in real-time
3. Review scan results
4. Approve discovered assets

### Step 4: Schedule Regular Scans
1. Edit your scan profile
2. Set schedule (daily, weekly, monthly)
3. Configure scan time windows
4. Enable automated asset updates
