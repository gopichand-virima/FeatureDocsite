# Designations

Use this function to categorize users.

In the main window, click **Admin \> Organizational Details \> Designations**. The Designations Window displays.

![designations old 662x183](../Resources/Images/Admin - Org Details/designations-old_662x183.png)

![arrow border](../Resources/Images/arrow-border.PNG)

[[[Closed]{.image .placeholder .MCDropDown_Image_Icon original-image-src="../../Skins/Default/Stylesheets/Images/transparent.gif" original-image-title="" height="12" width="16" Designation](javascript:void(0)){.MCDropDownHotSpot .dropDownHotspot .MCDropDownHotSpot_ .MCHotSpotImage aria-expanded="false"}]{.MCDropDownHead .dropDownHead}

To create a new designation:

 -- ----- ------------------------------------------------------------------------------------------------------------
 1. From the *Select Actions* drop-down list, choose **New Designation**. The New Designation window displays.
 -- ----- ------------------------------------------------------------------------------------------------------------

![New Designation](../Resources/Images/Admin - Org Details/New_Designation.png)

 -- ----- -----------------------------------------------------------------
 2. Enter a *Name* and *Description*. The *Name* field is required.
 -- ----- -----------------------------------------------------------------

 -- ----- ----------------
 3. Click **Add**.
 -- ----- ----------------

[[[Closed]{.image .placeholder .MCDropDown_Image_Icon original-image-src="../../Skins/Default/Stylesheets/Images/transparent.gif" original-image-title="" height="12" width="16" Designation](javascript:void(0)){.MCDropDownHotSpot .dropDownHotspot .MCDropDownHotSpot_ .MCHotSpotImage aria-expanded="false"}]{.MCDropDownHead .dropDownHead}

To edit an existing designation:

 -- ----- -------------------------------------------------------
 1. Click a record in the list. The Details tab displays.
 -- ----- -------------------------------------------------------

 -- ----- -----------------------------------------
 2. Modify the *Name* and/or *Description*.
 -- ----- -----------------------------------------

 -- ----- --------------------------------------------
 3. View, add, or modify any of the following:
 -- ----- --------------------------------------------

::: Bullet2
[Tasks](/itsm/release_mngmt/tasks_6_1.mdx){.MCTopicPopup .MCTopicPopupHotSpot .a}

::: Bullet2
[Comments](/discovery/comments_6_1.mdx){.MCTopicPopup .MCTopicPopupHotSpot .a}

::: Bullet2
[Attachments](/discovery/attachments_6_1.mdx){.MCTopicPopup .MCTopicPopupHotSpot .a}

 -- ----- -------------------------------------------------------
 4. When all selections/entries are made, click **Save**.
 -- ----- -------------------------------------------------------

[[[Closed]{.image .placeholder .MCDropDown_Image_Icon original-image-src="../../Skins/Default/Stylesheets/Images/transparent.gif" original-image-title="" height="12" width="16" Designation](javascript:void(0)){.MCDropDownHotSpot .dropDownHotspot .MCDropDownHotSpot_ .MCHotSpotImage aria-expanded="false"}]{.MCDropDownHead .dropDownHead}

To delete an existing designation:

 -- ----- ----------------------------------------------------------------
 1. Highlight the designation to delete. The Details tab displays.
 -- ----- ----------------------------------------------------------------

 -- ----- --------------------------------------------------------------------------------------------------------------------------------------------------
 2. FFrom the *Select Actions* drop-down list, choose **Delete**. If a confirmation message is displayed, type **Delete** and then click **Delete**.
 -- ----- --------------------------------------------------------------------------------------------------------------------------------------------------
