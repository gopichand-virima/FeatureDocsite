# About Runbook

Use this function to create and manage runbooks for standard procedures.

A runbook is a set of standardized documents, references and procedures that explain common recurring IT tasks. Runbooks are created to provide guidance for dealing with various operational scenarios.

Runbooks can include:

- Standard operating procedures
- Troubleshooting guides
- System maintenance procedures
- Emergency response procedures
