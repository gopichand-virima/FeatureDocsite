# Logs

The **Logs** tab provides detailed execution logs for the selected scan. It includes:

- Timestamps for each scan step
- Discovery activities per host or subnet
- Any errors or warnings encountered
- Successful connections, failed authentications, or unreachable hosts

Use this log to troubleshoot scan failures, credential issues, or connectivity problems.
