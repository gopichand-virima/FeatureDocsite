# Release State

Use this function to add a new state defining the status for the Release being worked by an assignee.

In the main window, click **Admin /> Release Management /> Release State**. The Release State window displays.

<span class="image placeholder" original-image-src="../Resources/Images/Admin%20-%20Release/release-state-main-old.PNG" original-image-title=""></span>

In the main window, click the **gear** icon and select **Release Management /> Release State**.

<span class="image placeholder" original-image-src="../Resources/Images/Admin%20-%20Release/icon-release-state.PNG" original-image-title=""></span>

The Release State window displays.
