# AD User Import Scans: Details

Use this function to view AD User Import Scan information.


No edits can be made to this information. It is for viewing purposes only.


In the main window, select **Discovery Scan > AD User Import Scans**. The AD User Import Scans window displays.

Select an existing record. A new window displays with the **Details** tab selected. The information displayed is view only.
