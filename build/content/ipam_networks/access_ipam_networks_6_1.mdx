# Accessing IPAM Networks

In the navigation pane, select **Discovery Scan > IPAM Networks**. The IPAM Networks window displays.

The IPAM Networks page shows a comprehensive list of all discovered subnets and their associated IP address information. This centralized view enables IT administrators to efficiently manage IP address allocation, track usage, and identify available addresses across the network infrastructure.
