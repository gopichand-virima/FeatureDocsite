# Custom BSM Views

Use this function to customize the display properties of Business Service Management (BSM).

- In the main window, select **Admin > SACM > BSM View**. The BSM View window displays.

!

- From the *Select Actions* drop-down list, choose an action: [New BSM View](/admin_sacm/custom_bsm_views_new_6_1.mdx), [Delete](/common_topics/delete_remove_6_1.mdx) or [Export](/discovery/import_data_files/export_6_1.mdx).
- To create a new view, click [New BSM View](/admin_sacm/custom_bsm_views_new_6_1.mdx).

## Edit BSM View

Not all functions include an option to edit record line items shown in the main window.


- Navigate to the window containing the record to edit.
- Click the line containing the record. The applicable window or dialog box displays.
- Make the necessary modifications.
- Click **Save** or **Add**, as applicable.
