# Rescan Now

Use this function to rescan selected assets.

<div class="NumList1" madcap:autonum="1. ">

While viewing the **Details** tab, click the **Rescan Now** button. The Rescan Now dialog box displays.

</div>

|     |                                                                               |
|-----|-------------------------------------------------------------------------------|
|     | The IP Address for the selected asset is pre-populated and cannot be changed. |

<div class="Ind50" madcap:conditions="Default.OldUI">

<span class="image placeholder" original-image-src="../../../Resources/Images/ITSM/rescan-now-dialog-box.PNG" original-image-title="" style="width: 375px;height: 168px;"></span>

</div>

<div class="Ind50" madcap:conditions="Default.NewUI">

<span class="image placeholder" original-image-src="../../../Resources/Images/ITSM/rescan-now-dialog-box-new.PNG" original-image-title="" style="width: 368px;height: 277px;"></span>

</div>

<div class="NumList2" madcap:conditions="Default.OldUI" madcap:autonum="2. ">

In the *Choose a client* field, click the drop-down and select an available client.

</div>

<div class="NumList2" madcap:conditions="Default.NewUI" madcap:autonum="3. ">

In the *Client* field, click the drop-down list and select an available client.

</div>

<div class="NumList2" madcap:autonum="4. ">

In the *Type of Scan* field, click the drop-down and select the type of scan. *For details on the types of scans available, refer to [Run a Scan](../../../Discovery/Run-a-Scan/Run-a-Scan.htm).*

</div>

<div class="NumList2" madcap:autonum="5. ">

When all selections/entries are made, click **Scan**. A confirmation message displays and the window refreshes and displays the [Recent Scans](../../../Discovery/Recent-Scans/Scans-Recent.htm) information.

</div>
