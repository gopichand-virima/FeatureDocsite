# Hard Drives

## <span id="WindowsPatches"></span>

Displays a list of all hard drives installed on this asset and provides the functionality to update Windows patches.

<div class="NumList2" madcap:autonum="1. ">

Click the **Hard Drives** tab.

</div>

<div class="Ind50">

<span class="image placeholder" original-image-src="../../../../Resources/Images/Components/hard-drives-old.PNG" original-image-title="" madcap:conditions="Default.OldUI"></span>

</div>

<div class="NumList2" madcap:autonum="2. ">

To view the hard drive component details, click a line item. The Component Details information box displays. This information is view only.

</div>
