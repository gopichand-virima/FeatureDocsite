# Blueprints: Comment

Use this function to add, edit and delete comments.

!


### Adding a Comment

- Click the **Comments** tab.
- In the Comments window, click **New Comment**.

!

- Enter the applicable text.
- To add any documents, click **Attach**, and search for the attachment.
- Click **Add** to save the comment and any attachments.
- To leave the comment window without adding any comments or attachments, click **Cancel**.


### Editing a Comment

- Click the **Comments** tab.
- Locate the comment to edit.
- Click the **pencil** icon. The comment field displays.

!

- Edit the comment, as necessary.
- If applicable, attach any files. *Refer to **Adding an Attachment**.*
- Click **Save**. The new text replaces the existing text.


### Deleting a Comment

- Click the **Comments** tab.
- Locate the row containing the comment to delete.
- Click the **Delete** icon
- In the confirmation window, Type **Delete** and click **Delete**.
