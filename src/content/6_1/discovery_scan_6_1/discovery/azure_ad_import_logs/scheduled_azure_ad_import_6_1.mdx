# Scheduled Azure AD Import

After Azure AD configuration, follow these steps:

1.  Click the **Schedule Import icon** to the right of the selected line item.

2.  The **Schedule Azure AD Import** dialog box displays.

<span class="image placeholder" original-image-src="../../Resources/Images/Discovery/Azure-AD-Users-Import-Logs/Azure%20AD%20User%20Import%20Logs_10.png" original-image-title=""></span>

3.  Select <span class="Strong" style="font-weight: bold;">Enable Schedule Import</span> to turn on the import function.

4.  <span class="inline-comment-marker">Select </span><span class="Strong" style="font-weight: bold;">Reflect AD Account Status</span><span class="inline-comment-marker"> to match user login status with Azure AD.</span>

5.  Select <span class="Strong" style="font-weight: bold;">Activate User Account</span> to activate all users.

6.  Select <span class="Strong" style="font-weight: bold;">Send Activation notification</span> to the applicable users when the import is complete.

- <span class="Strong" style="font-weight: bold;">Reflect AD Account Status</span> – Enable to match user login status with Azure AD.

- <span class="Strong" style="font-weight: bold;">Activate User Account</span> – Enable to make imported users active by default.

- <span class="Strong" style="font-weight: bold;">Send Activation Notification</span> – Enable to email users upon account activation.

6.  In the <span class="Strong" style="font-weight: bold;">Azure AD Import Frequency</span> field, click the drop-down list and select the applicable import frequency, such as <span class="Strong" style="font-weight: bold;">Every Day, Every 2 days</span>, and so forth.

7.  In the schedule area, select <span class="Strong" style="font-weight: bold;">Timezone</span> and the appropriate schedule timing.

8.  When all selections/entries are made, click <span class="Strong" style="font-weight: bold;">Add</span>. (If you are editing the existing information, the Add button changes Update.)

9.  After import completion, view import records in the <span class="Strong" style="font-weight: bold;">Azure AD User Import Logs</span>.
