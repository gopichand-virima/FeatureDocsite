# Exporting Recent Scans

Users can export the list of recent scans to an Excel spreadsheet for external review, reporting, or archival.

1. Select one or more scan records.

2. From the **Select Actions** dropdown, choose **Export**.

1. A confirmation dialog will appear. Click **Continue**.

2. The system will generate an **Excel (.xlsx) file** and **email a download link** to the logged-in user.

**Exported Data Includes:**

- Scan Name
- Scan Type
- Status (Running, Completed, Failed)
- Client
- Started On
- Completed On
- Duration
- Schedule (On Demand / Scheduled Scan)
