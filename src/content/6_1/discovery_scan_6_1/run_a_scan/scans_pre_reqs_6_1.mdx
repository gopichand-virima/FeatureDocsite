# Prerequisites for Scan

[Credentials are required to scan the ]{.span_5}[Target Host]{.Strong_4}[ and can be entered using the methods described below.]{.span_5}

 --- ----------------------------------------------------------------------------------------------------------------------
   Added and imported credentials are encrypted and stored on the machine where the Discovery Application is installed.
 --- ----------------------------------------------------------------------------------------------------------------------

### [Method 1]{.Strong_4}[: ]{.span_5}[Through Virima Discovery Admin]

*Refer to* [Credentials](/admin_discovery/credentials_6_1.mdx){style="font-style: italic;"}*.*

### [Method 2]{.Strong_4}[: ]{.span_5}[Through the Discovery Application using the Application Interface]

 -- ----- ---------------------------------------------------------------------------
 1. [Right-click the ]{.span_5}[**Virima Discovery Application**.]{.Strong_4}
 -- ----- ---------------------------------------------------------------------------

 -- ----- ---------------------------------------------------------
 2. [Click ]{.span_5}[**Run as Administrator**.]{.Strong_4}
 -- ----- ---------------------------------------------------------

 -- ----- -----------------------------------------------------------------------------------------------------------------------------------------------------
 3. [Click ]{.span_5}[Start]{.Strong_4}[ and wait until it shows ]{.span_5}[that the Discovery Agent is Running.]{.Strong_4}
 -- ----- -----------------------------------------------------------------------------------------------------------------------------------------------------

 -- ----- -----------------------------------------------------------------------------
 4. [From the Credentials drop-down list, select **Add Credential**. ]{.span_5}
 -- ----- -----------------------------------------------------------------------------

![scan prereq credentials 471x216](../../Resources/Images/Discovery/scan-prereq-credentials_471x216.png)

::: Ind50
The Add new credential window displays.

![discovery agent add credential 472x416](../../Resources/Images/Admin - Discovery/discovery-agent-add-credential_472x416.png)

 -- ----- ---------------------------------
 5. [Complete the fields.]{.span_5}
 -- ----- ---------------------------------

 -- ----- -------------------------------------------------------
 6. When all selections/entries are made, click **Save**.
 -- ----- -------------------------------------------------------

### Method 3: Load the Credentials from an Excel File

To load the credentials through this method, credentials must be configured in an Excel file, as shown in the figure below.

![credential inventory excel sample 671x252](../../Resources/Images/Admin - Discovery/credential-inventory-excel-sample_671x252.png)

 -- ----- ---------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------
 1. [Store the credential file in this location: ]{.span_5}[[[\
 C:\\Program Files\\Virima Discovery Application\\Config]{style="font-size: 10pt;"}]{style="font-family: 'Courier New';"}]{.Strong_3}[[[ as ]{style="font-size: 10pt;"}]{style="font-family: 'Courier New';"}]{.span_5}[[[Credentials.xls]{style="font-size: 10pt;"}]{style="font-family: 'Courier New';"}]{.Strong_3}[[[ or ]{style="font-size: 10pt;"}]{style="font-family: 'Courier New';"}]{.span_5}[[[Credentials.xlsx]{style="font-size: 10pt;"}]{style="font-family: 'Courier New';"}]{.Strong_3}

 -- ----- ---------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------

 -- ----- ---------------------------------------------------------------------------
 2. [Right-click the ]{.span_5}[**Virima Discovery Application**.]{.Strong_4}
 -- ----- ---------------------------------------------------------------------------

 -- ----- ---------------------------------------------------------
 3. [Click ]{.span_5}[**Run as administrator**.]{.Strong_4}
 -- ----- ---------------------------------------------------------

 -- ----- ---------------------------------------------------------------------------------------------------------------------------------------------------------
 4. [Click ]{.span_5}[Start]{.Strong_4}[ and wait for the indication that the ]{.span_5}[Discovery agent is Running.]{.Strong_4}
 -- ----- ---------------------------------------------------------------------------------------------------------------------------------------------------------

 -- ----- -----------------------------------------------------------------------------------------------------------------------
 5. [In the Discovery Agent window, click the **Credentials** drop-down list and select **Import Credentials**.]{.span_5}
 -- ----- -----------------------------------------------------------------------------------------------------------------------

 --- ----------------------------------------------------------------------------------------------------------------------
   The added/imported credentials are encrypted and stored on the machine where the Discovery Application is installed.
 --- ----------------------------------------------------------------------------------------------------------------------
