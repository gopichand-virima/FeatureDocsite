# Problem Model

Use this function to configure and populate information that can be re used for defining change requests. It can be used to define commonly occurring changes raised by Service Desk Members to skip and reduce the time required to fill in forms.

In the main window, click **Admin > Problem Management > Problem Model**. The Problem Model window displays.

New Problem Model

From the *Select Actions* drop-down list, choose **New Problem Model**. The Problem Model window displays.
