# Export

To export uploaded file records,

1.  Select the file(s) to export.

2.  From the <span class="Strong" style="font-weight: bold;">Select Actions</span> menu, click <span class="Strong" style="font-weight: bold;">Export</span>.

    <span class="image placeholder" original-image-src="../../Resources/Images/Discovery/Import-Data-Files/Import%20Data%20Files_1_26.png" original-image-title=""></span>

<!-- -->

3.  A confirmation dialog appears, indicating that the selected records will be exported to an Excel file.

    <span class="image placeholder" original-image-src="../../Resources/Images/Discovery/Import-Data-Files/Import%20Data%20Files_1_27.png" original-image-title=""></span>

<!-- -->

4.  <span class="inline-comment-marker">Click </span><span class="Strong" style="font-weight: bold;">Continue</span><span class="inline-comment-marker">.</span>

5.  An email is sent to the requesting user’s email address (the one currently signed in).

    1.  <span class="Strong" style="font-weight: bold;">Download Link: </span>The email contains a download link to a ZIP file with the exported data.

    2.  The exported ZIP file follows this naming format:  
        ImportDataFiles/<hh.mm.AM/PM/>-/<MM.DD.YYYY/>.zip  
        (For example: ImportDataFiles10.15.AM-10.08.2025.zip)

    3.  <span class="Strong" style="font-weight: bold;">Columns Included in the Export:</span>  
        The exported file contains the following columns:

        1.  <span class="Strong" style="font-weight: bold;">File Name:</span> The name of the uploaded file (e.g., Import+CI's.xlsx)

        2.  <span class="Strong">**Status**:</span> The processing state of that upload (Processed, Processing)

        3.  <span class="Strong" style="font-weight: bold;">Uploaded By:</span> The display name of the user who uploaded the file

        4.  <span class="Strong" style="font-weight: bold;">Uploaded On:</span> The local timestamp when the file was uploaded (e.g., 10/08/2025, 10:15:47 AM)
