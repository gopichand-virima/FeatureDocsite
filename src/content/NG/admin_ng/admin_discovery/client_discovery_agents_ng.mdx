# Discovery Agents

[Use this function to view the total number of discovery agents installed for the corresponding Client, remotely install a new discovery agent, or restart the client.]{.span_1}

 -- ----- -----------------------------------------------------------------------------------------
 1. In the main window, click **Admin \> Discovery \> Client**. The Client window displays.
 -- ----- -----------------------------------------------------------------------------------------

 -- ----- -------------------------------------------------------------------------------------------------------------------------
 2. Select the client for which you want manage discovery agents. A new window displays with the **Details** tab displayed.
 -- ----- -------------------------------------------------------------------------------------------------------------------------

 -- ----- ---------------------------------------------------------------------------------------------------------------------------------------------------------------------------
 3. Click the **Discovery Agents** tab.![discovery agent 6.1](../Resources/Images/Admin - Discovery/discovery-agent-6.1.PNG)
 -- ----- ---------------------------------------------------------------------------------------------------------------------------------------------------------------------------

 -- ----- -----------------------------------------------------------------------------------------------------------------------------------------
 4. To trigger a scheduled scan on the remote machine where an agent is installed, refer to [Agent Schedules](Client-Agents-Schedules.mdx).
 -- ----- -----------------------------------------------------------------------------------------------------------------------------------------

 -- ----- -------------------------------------------------------------
 5. From the *Select Actions* drop-down list, choose an action:
 -- ----- -------------------------------------------------------------

::: {.Bullet2
[Build an agent](Client-Build-Agent.mdx)

::: {.Bullet2
[Remote Install](Client-Remote-Install.mdx) to add a new Discovery Agent.

::: {.Bullet2
[Restart the Client](Client-Restart.mdx)

::: {.Bullet2
[Scan Discovery Agents](Client-Scan.mdx)

 -- ----- -----------------------------------------------------------------------------------------------------------------
 6. **Update Agents**. To update the agents with the latest version, select an agent, then click **Update Agents**.
 -- ----- -----------------------------------------------------------------------------------------------------------------

 -- ----- ------------------------------------------------------------------------------------------------------
 7. **Schedule**. To schedule a scan on the remote client, click **Schedule** and specify a time period.
 -- ----- ------------------------------------------------------------------------------------------------------

 -- ----- -------------------------------------------------------------------------
 8. **Delete Schedule**. To delete the schedule, click **Delete Schedule**.
 -- ----- -------------------------------------------------------------------------

 -- ----- ---------------------------------------------------------------------------------------------------------------
 9. **Delete/Uninstall**. To uninstall or remove, select an agent, click **Uninstall**, then click **Uninstall**.
 -- ----- ---------------------------------------------------------------------------------------------------------------
