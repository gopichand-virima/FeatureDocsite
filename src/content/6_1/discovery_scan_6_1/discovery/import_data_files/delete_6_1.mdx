# Delete

To delete uploaded file records,

1.  Select one or more files from the list.

2.  From the <span class="Strong" style="font-weight: bold;">Select Actions</span> menu, click <span class="Strong" style="font-weight: bold;">Delete</span>.

    <span class="image placeholder" original-image-src="../../Resources/Images/Discovery/Import-Data-Files/Import%20Data%20Files_1_24.png" original-image-title=""></span>

<!-- -->

3.  If no files are selected, an alert appears: <span class="Emphasis" style="font-weight: bold;">"Please select at least one record."</span>

    <span class="image placeholder" original-image-src="../../Resources/Images/Discovery/Import-Data-Files/Import%20Data%20Files_1_25.png" original-image-title=""></span>

<!-- -->

4.  <span class="inline-comment-marker">Confirm the deletion and Click on </span><span class="Strong" style="font-weight: bold;">Delete</span><span class="inline-comment-marker">.</span>

    1.  <span class="inline-comment-marker">Deleting a file from Import Data Files will also remove the associated imported assets only if they have not yet been promoted to the CMDB.</span>

5.  <span class="inline-comment-marker">Click </span><span class="Strong" style="font-weight: bold;">Cancel </span><span class="inline-comment-marker">to cancel the delete.</span>
