# Edit Purchase Order Line Items

This feature enables editing the configured values of the existing contract, and update, and then save the same in the application.

Click to open required PO Line Item. The PO Line Item page displays.

The Stockroom displays the following information:

- Details
- Tasks
- Comments
- Attachment
- History
