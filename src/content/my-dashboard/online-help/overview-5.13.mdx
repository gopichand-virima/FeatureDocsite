# My Dashboard - Online Help Overview

Welcome to the My Dashboard module documentation for Virima version 5.13.

## About the Virima Modules and Functions

The All Modules window contains quick access to selected functions. From this main window, all functions are reached either through:
- The **Navigation pane** (down the left side of the window)
- The **Admin menu** (in the upper right corner of the window)

![Virima Application Overview](figma:asset/f9381b35c98ccaabf8b82fb583cf3f28e2b6db92.png)

### Home Page/All Modules Window

The home page provides quick access to the most frequently used modules, such as favorite functions and dashboards.
- **Adding a Favorite**: Locate an icon for the applicable section inside the box with the text, "Drag here to add."
- **Deleting** a Favorite: Locate the icon in the Favorites section. Then, click and drag the icon above the Favorite section until the text "Drag here to delete" inside a red box displays.

### Left Navigation Pane

The left navigation pane includes the following:
- **My Dashboard** displays data content information with a customization option.
- **Discovery Scan** manages scanning activities (running, scheduling, etc.) and importing data (such as from AWS and Azure).
- **ITAM** (asset management functions) and **ITSM** (service management functions) availability is based upon the subscription.

> **Note:** The menu heading displayed - either ITAM or ITSM - is based upon the subscription.

- **Vulnerability Management** identifies, evaluates, and reports security vulnerabilities in CIs present inside the CMDB.
- **Self Service** creates catalog items and shows incidents and requests for the current user.
- **Project and Program Management** defines project aspects and managing related projects.
- **Risk Register** provides a tool for managing risk and compliance.
- **Reports** has both canned and custom ad hoc reports.
- **MSP** displays a list of clients and includes a Dashboard, Reports, My Incidents, and Administrative functions (Client, Users, Mailbox Monitoring Configuration, and Email Templates). It also provides access to the Virima application.

### Admin Window

The administrative functions include the following:
- Configuration displays setting details, such as departments, cost centers, locations, designations, operational hours, and holidays.

## Overview

My Dashboard provides a centralized view of your IT infrastructure, allowing you to monitor key metrics, track incidents, and access critical information at a glance.

## Key Features

- **Customizable Widgets**: Add, remove, and arrange widgets to create your personalized dashboard
- **Real-time Monitoring**: View live data updates for infrastructure health and performance
- **Quick Actions**: Access frequently used functions directly from the dashboard
- **Multi-metric Views**: Monitor multiple KPIs simultaneously

## Getting Started

To begin using My Dashboard:
1. Log in to your Virima portal
2. Navigate to My Dashboard from the main menu
3. Customize your dashboard layout using the widget selector
4. Configure data refresh intervals and widget preferences
