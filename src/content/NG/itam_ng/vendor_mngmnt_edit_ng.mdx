# Edit Vendor Management

This feature enables editing the configured values of the existing contract, and update, and then save the same in the application.

 -- ----- --------------------------------------------------------------------------------
 1. Click to open required Vendor Management. The Edit vendor Management displays.
 -- ----- --------------------------------------------------------------------------------

Product Models:

This feature enables to update of product details manufactured by the corresponding application.

::: Ind50
![vendor mngmnt edit product 791x212](../Resources/Images/ITAM/vendor-mngmnt-edit-product_791x212.png)

New Product Model:

::: Ind50
![vendor mngmnt edit newprod 797x115](../Resources/Images/ITAM/vendor-mngmnt-edit-newprod_797x115.png)

 -- ----- ---------------------------------------------------------------------------
 1. Enter the name of the product manufactured by the vendor in the Name box.
 -- ----- ---------------------------------------------------------------------------

 -- ----- -------------------------------------------------------------------------------
 2. Enter unique numerical code provided for the product in the Model Number box.
 -- ----- -------------------------------------------------------------------------------

 -- ----- ---------------------------------------------------------------------
 3. Enter the name of the product category in the Model Categories box.
 -- ----- ---------------------------------------------------------------------

 -- ----- -------------------------------------------------------------
 4. Click Add to update the product details in the application.
 -- ----- -------------------------------------------------------------

 -- ----- -----------------------------------------------------------------------------------------------------------------
 5. Click Cancel to discard the actions, and in continuation no product details will be updated in the application.
 -- ----- -----------------------------------------------------------------------------------------------------------------
