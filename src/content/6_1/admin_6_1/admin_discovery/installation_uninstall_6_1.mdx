# Uninstall the Discovery Application

The best and safest way to uninstall the existing application is by using a Windows uninstaller.

<div class="NumList1" madcap:autonum="1. ">

Go to Windows <span class="Strong_4" style="font-weight: bold;">Control Panel</span><span class="span_5" style="font-weight: bold;"> </span>and select <span class="Strong_4">**Programs and Features**.</span>

</div>

<div class="NumList2" madcap:autonum="2. ">

<span class="span_5">Select </span><span class="Strong_4" style="font-weight: bold;">Virima Discovery Application.</span>

</div>

<div class="NumList2" madcap:autonum="3. ">

<span class="span_5">Click </span><span class="Strong_4">**Uninstall**.</span>

</div>

<div class="NumList2" madcap:autonum="4. ">

<span class="span_5">Navigate to the installed location and delete the directory </span><span class="Strong_3">Virima Discovery Application</span><span class="span_5"> to remove all the temp and log files.</span>

</div>
