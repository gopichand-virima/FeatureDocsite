# View IP List

Click on any network row to view the complete list of IP addresses within that subnet.

The IP List view displays:
- **IP Address**: The individual IP address
- **Status**: Used, Unused, or Reserved
- **Host Name**: Associated hostname (if discovered)
- **MAC Address**: Physical address of the device
- **Last Seen**: Timestamp of last detection
- **DHCP Lease**: Lease information if dynamically assigned
- **Device Type**: Type of device using the IP (if identified)
- **Comments**: Additional notes or documentation

Users can filter, sort, and export the IP list for reporting and analysis purposes.
