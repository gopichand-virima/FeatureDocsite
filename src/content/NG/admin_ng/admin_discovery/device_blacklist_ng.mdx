# Device Blacklist

Use this function to manage devices that should be excluded from discovery scans.

In the main window, click **Admin > Discovery > Device Blacklist**. The Device Blacklist window displays.

## New Device Blacklist Entry

From the *Select Actions* drop-down list, choose **New Device Blacklist**. The New Device Blacklist window displays.

Enter the device information to exclude:

- IP Address or IP Range
- MAC Address
- Hostname

Enter a *Description* or *Reason* for the exclusion.

When all selections/entries are made, click **Add**.

## Edit Device Blacklist

Navigate to the window containing the record to edit.

Click the line containing the record. The applicable window displays.

Make the necessary modifications.

Click **Save**.

## Delete Device Blacklist

Click the line that contains the item to delete.

From the *Select Actions* drop-down list, choose **Delete**.
