# Port Configuration

Use this function to create a relationship between a process and host if, during the Discovery process, it is found running on a specific port.

On the main window, select **Admin > Discovery > Port Configuration for Process**. The Port Configuration window displays.

!

!


### New Port Configuration Process

- In the *Select Actions* drop-down list, choose **New Port Configuration for Process**.

!

- Enter the *Process Name* and *Port*.
- In the *Relationship Type* field, click the drop-down list and select the applicable relation type.
- Click **Add**.
- If the user clicks Add without filling any required field (Process Name, Port, or Relationship Type) in the Port Configuration for Process window, the system displays an error.


### Edit Port Configuration Process

Not all functions include an option to edit record line items shown in the main window.


- Navigate to the window containing the record to edit.
- Click the line containing the record. The applicable window or dialog box displays.
- Make the necessary modifications.
- Click **Save** or **Add**, as applicable.


### Delete Port Configuration Process

Deleting is a permanent action and cannot be undone. Deleting may affect other functionality and information in the application such as data in configured reports, fields in windows, selectable options, etc. Therefore, be sure to understand the potential effects before making a deletion.


- Click the line that contains the item to delete.
- From the *Select Actions* drop-down list, choose **Delete**. If a confirmation message is displayed, type **Delete** and click **Delete**to confirm.
