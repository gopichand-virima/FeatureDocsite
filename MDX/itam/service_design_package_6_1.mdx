# Service Design Package

Use this function to manage service-related documents that contain Service Portfolio documents for catalog items.

In the navigation pane, select **ITAM > Service Design Package**. The Service Design Package window displays.

!

!


### New Service Design Package

- From the *Select Actions* drop-down list, choose New Service Design Package. The New Service Design Package window displays.
- Click **New**. The New Service Design Package window displays.
- Complete the tabs, referring to the information below. If applicable, click the **Upload** button to import text.
- On the *Requirements* tab, enter or upload information for the **Business Requirements**, **Service Applicability**, and **Service Contacts**.
- On the *Service Design* tab, enter or upload information for the **Service Functional Requirements**, **Service Level Requirements**, **Service and Operational Management Requirements**, and **Service Design and Topology**.
- On the *Organizational Readiness Assessment* tab, enter or upload the applicable information.
- On the *Service Lifecycle Plan*, enter or upload information for **Service Program**, **Service Transition Plan**, **Service Transition Plan**, **Service Operational Acceptance Plan**, **Service Acceptance Criteria**, and **Service Acceptance Criteria**.
- When the relevant information is added, click **Add**.


### Edit Service Design Package

Not all functions include an option to edit record line items shown in the main window.


- Navigate to the window containing the record to edit.
- Click the line containing the record. The applicable window or dialog box displays.
- Make the necessary modifications.
- Click **Save** or **Add**, as applicable.


### Delete Service Design Package

Deleting is a permanent action and cannot be undone. Deleting may affect other functionality and information in the application such as data in configured reports, fields in windows, selectable options, etc. Therefore, be sure to understand the potential effects before making a deletion.


- Click the line that contains the item to delete.
- From the *Select Actions* drop-down list, choose **Delete**. If a confirmation message is displayed, type **Delete** and click **Delete**to confirm.
