# Network Adapters

This function is only available with certain asset types. Therefore, the relevant tab may not be displayed.


- In the main window, select **ITSM > Configuration Management > CMDB**.
- Search for and select the applicable asset record.
- When the asset record window displays, click the **Components** tab.
- Click **Network Adapters**.

! !

!


### Add Network Adapter

- Click **Add**. The Network Adapter dialog box displays.

! !

- Click **New Network Adapter.**
- Enter the *MAC Address*, *Network Adapter Name*, *IP Address*, and *Asset Names*. These are all required fields.
- When all selections/entries are made, click **Save**.
- When all selections/entries are made, click **Add**.


### Edit Network Adapter

- Click **Existing Network Adapter**.

! !

- Enter the search criteria to locate the existing adapter*.*
- Select the applicable adapter from the results list.
- When all selections are made, click **Save**.


### Edit Network Adapter Details

- While viewing the list of existing adapters, select the adapter to edit. The windows refreshes and the details for the adapter displays.
- Make any necessary changes to the existing information. The changes are automatically saved. *Refer to* [CI Components](/itsm/config_mngmt/config_items/config_item_components/component_tab_options_6_1.mdx)*.*
- To add a contact, navigate to the Contacts section and click **Add**. The Contacts dialog box displays.

! !

- In the *Choose Relation* field, click the drop-down list and select how this contact should be related to this asset.
- In the search field, enter the search criteria.
- Select a contact from the results list, moving it to the selected list.
- When all selections are made, click **Save**.


### Delete Adapter

Deleting is a permanent action and cannot be undone. Deleting may affect other functionality and information in the application such as data in configured reports, fields in windows, selectable options, etc. Therefore, be sure to understand the potential effects before making a deletion.


- Click the line that contains the item to delete.
- From the *Select Actions* drop-down list, choose **Delete**. If a confirmation message is displayed, type **Delete** and click **Delete**to confirm.
