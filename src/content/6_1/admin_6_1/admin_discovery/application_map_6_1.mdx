# <span style="background-color: #ffffff;">Discovery Application Map</span>

Virima Application mapping is the process of discovering and identifying the interactions and communication between Virima applications and their underlying hardware infrastructure in a visual manner. This map provides an up-to-date, real-time dynamic blueprint of your application architecture of the communication between agents, applications, the Discover server and the Virima web application.

<span style="color: #000000;">In the main window, click **Admin** and then select</span><span style="color: #000000;"> </span>**<span style="color: #000000;">Discovery /> Discovery Application Map</span>**.

The application map displays.

<span class="image placeholder" original-image-src="../Resources/Images/Admin%20-%20Discovery/Application_Map_6.1.png" original-image-title=""></span>

The icons provide the following functionality:

<div class="Bullet1">

**Filter**. Filters the information in the window by type (such as IP Address).

</div>

<div class="Bullet1">

**Maximize/Minimize**. Resizes the window.

</div>

<div class="Bullet1">

**Load View**. Loads a saved view.

</div>

<div class="Bullet1">

**Save View**. Saves the currently shown view.

</div>

<div class="Bullet1">

**Save Image**. Downloads the currently shown view in image format.

</div>

Use the **Zoom** slider to adjust the focus of the window.

To reposition an item, click and drag it to the new location.
