# [Discovered Items: Updates]{style="background-color: #ffffff;"}

Use this function to view updates to discovered items.

 --- --------------------------------
   This information is view only.
 --- --------------------------------

 -- ----- --------------------------------------------------------------------------------------------------------------
 1. In the navigation pane, select **Discovery Scan \> Discovered Items**. The Discovered Items window displays.
 -- ----- --------------------------------------------------------------------------------------------------------------

 -- ----- --------------------------------------------------------------
 2. Click **Updates**. The list of all updated actions displays.
 -- ----- --------------------------------------------------------------

 --- -----------------------------------------------------------------------------------------------------------------
   The number shown after \"Updates\" indicates the number of changes made (described under the *Changes* column).
 --- -----------------------------------------------------------------------------------------------------------------
