# Azure AD Configuration Property Descriptions

|                                                                 |                                                                                                                   |
|-----------------------------------------------------------------|-------------------------------------------------------------------------------------------------------------------|
| <span class="Strong" style="font-weight: bold;">Property</span> | <span class="Strong" style="font-weight: bold;">Description</span>                                                |
| <span class="Strong">Name</span>                                | A user-defined label to identify this Azure AD configuration among multiple setups.                               |
| <span class="Strong">Client ID</span>                           | The unique identifier assigned to your Azure application. Used by Azure AD to recognize the application.          |
| <span class="Strong">Tenant ID</span>                           | The unique identifier of your Azure Active Directory tenant. Specifies the directory where authentication occurs. |
| <span class="Strong">Secret Key</span>                          | A confidential string generated in Azure, used as a password for application authentication.                      |
| <span class="Strong">Certificate</span>                         | A digital certificate (with private key) uploaded to Azure, used as a secure authentication method for the app.   |
