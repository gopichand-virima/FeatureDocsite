# New Requested Items

This feature enables the addition of required items to be requested with appropriate details.

Choose New Requested Item option from Select Actions drop down. The New requested Item page displays.

Enter the total number of the requested item types in the Requested item Number box.

Click Add to update Configuration Item details.

Click Add to update the name of the user-initiated for the requested item.

Click Add to update the name for the Requested For (purpose for requesting the corresponding item).

Instructions:

Enter the search criteria in the given text box, and then click search icon.

The results are displayed as per the given search criteria.

Select a required value, and then click Save.

Click Cancel to discard the action, and in continuation no value will be selected.

Reference : Click the link for https://wiki.virima.com/wiki/doku.php?id=advanced\_search\_page for Advanced Search instructions// Unsure if this text had to be added.

Click Opened On text box to select request item initiation date.

Click the Due Date text box to select the last date to provide the requested item.

Select the stage name of the request process from the Stage dropdown.

Select the state name of the request process from the State dropdown.

Enter the total quantity of the requested Item in the Quantity box.

Enter the name of the catalog item in the Catalog Item box.

Click the Estimated Delivery Date text box to select a date for delivering requested items.

Check the Backordered option if the corresponding service cannot be fulfilled at the current time. it affect the amount of time it takes before the customer eventually receives the ordered product/service.

Click Add to update Requested Item as per given details.
