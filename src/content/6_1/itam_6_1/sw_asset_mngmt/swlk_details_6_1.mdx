# Software License Key: Details

Use this function to view and modify the details for a software license key.

<div class="NumList1" madcap:autonum="1. ">

While viewing the Software License key window, from the *Select Actions* drop-down list, choose **New**.

</div>

<div class="NumList2" madcap:autonum="2. ">

Select an existing record. A new window displays with the **Details** tab displayed, by default.

</div>

<div class="NumList2" madcap:autonum="3. ">

Make the applicable changes to the Asset Primary Information and Contacts.

</div>

<div class="NumList2" madcap:autonum="4. ">

When all selections/entries are made, click **Save**.

</div>
