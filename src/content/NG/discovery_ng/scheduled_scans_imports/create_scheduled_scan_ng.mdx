# Creating a Scheduled Scan

1. From the **Select Actions** dropdown, choose **New Scan Schedule**.

2. A configuration dialog opens.

**Name:** Enter a name for the scheduled scan.

**IP Range:** Specify the target IPs or IP range (single IP, multiple IPs, CIDR, or range).

**Choose a Client:** Select the discovery client (server) that will run the scan.

**Type of Scan:** Choose a scan type (e.g., Deep Host Scan, Basic Host Scan, Port Scan).

**Scan Frequency:** Select how often the scan should run (e.g., Every Day, Every Week, Monthly).

**Active:** Check this box to activate the schedule.

**Timezone:** Select the timezone for scheduling.

**Schedule Timing:** Configure the exact time (hour, minute, second) when the scan should execute.

1. Click **Run** or **Add** to save the schedule.

The scheduled scan will now execute automatically according to the defined frequency and timing.
